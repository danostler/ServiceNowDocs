# UIB API reference

Use UI Builder \(UIB\) APIs in UIB client applications.

-   **[api - UI Builder](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/api-reference/ui-builder-api-reference/apiAPI.md)**  
The api API provides methods that you can use when developing client scripts in the UI Builder.
-   **[helpers - UI Builder](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/api-reference/ui-builder-api-reference/helpersAPI.md)**  
The helpers API provides general functionality that is common across page scripts, eliminating the need to write scripts for simple functionality such as opening and closing a modal.
