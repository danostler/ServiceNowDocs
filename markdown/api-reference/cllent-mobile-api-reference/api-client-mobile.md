# Client mobile API reference

Use Cabrillo JS to build custom mobile experiences inside of ServiceNow AI Platform native mobile apps.

-   **[Cabrillo JS constants](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/api-reference/cllent-mobile-api-reference/cabrillo-constants.md)**  
These are the constants used by Cabrillo JS.
-   **[Cabrillo JS data types](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/api-reference/cllent-mobile-api-reference/cabrillo-datatypes.md)**  
These are common data types used by Cabrillo JS APIs.
-   **[cabrillo - Client](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/api-reference/cllent-mobile-api-reference/cabrillo-namespace.md)**  
Cabrillo JS is a client side JavaScript API for accessing capabilities inside ServiceNow native mobile applications.
-   **[cabrillo.attachments - Client](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/api-reference/cllent-mobile-api-reference/cabrillo-attachments-namespace.md)**  
The name space for Cabrillo JS attachment functions. This enables adding and viewing attachments.
-   **[cabrillo.camera - Client](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/api-reference/cllent-mobile-api-reference/cabrillo-camera-namespace.md)**  
Cabrillo JS camera functions to provide camera access for bar code scanning.
-   **[cabrillo.flowControl - Client](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/api-reference/cllent-mobile-api-reference/cabrillo-FC-namespace.md)**  
Cabrillo JS functions for managing the flow of a web view.
-   **[cabrillo.message - Client](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/api-reference/cllent-mobile-api-reference/cabrillo-messgage-namespace.md)**  
Cabrillo JS functions to display messages in the native UI.
-   **[cabrillo.modal - Client](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/api-reference/cllent-mobile-api-reference/cabrillo-modal-namespace.md)**  
Cabrillo JS functions for presenting web content inside of native modals.
-   **[cabrillo.navigation - Client](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/api-reference/cllent-mobile-api-reference/cabrillo-navigation-namespace.md)**  
Cabrillo JS functions for forward and backward navigation.
-   **[cabrillo.nowapp - Client](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/api-reference/cllent-mobile-api-reference/cabrillo-nowapp-namespace.md)**  
The name space for Cabrillo Now App functions. This enables you to interact with screens inside of ServiceNow AI Platform® native mobile apps.
-   **[cabrillo.viewLayout - Client](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/api-reference/cllent-mobile-api-reference/cabrillo-viewLayout-namespace.md)**  
Cabrillo JS functions to provide access to native UI elements like buttons and spinners.
-   **[m\_form - Client](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/api-reference/cllent-mobile-api-reference/m_formClientAPI.md)**  
Provides methods to get and set input values on input form screens, as well as display messages.
-   **[MobileScriptIncludeCaller - Client](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/api-reference/cllent-mobile-api-reference/MobileScriptInclCallerClientAPI.md)**  
Provides methods to call script includes from Mobile UI Rule Actions.
