# Get started with Mobile SDK - iOS

You must complete the tasks outlined in this section to prepare your development environment to support writing iOS applications that integrate with the ServiceNow Mobile SDK.

-   Download and install the ServiceNow Mobile SDK libraries. For detailed instructions, see [Download and install the Mobile SDK libraries](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/api-reference/developer-guides/mobsdk-ios-down-install-sdk-libs.md).
-   Install and configure the Mobile SDK scoped application. For detailed instructions, see [Install and configure the Mobile SDK scoped application](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/api-reference/developer-guides/mobsdk-ios-install_config-sdk_plugin.md).
-   Configure your project. For more information, see [Configure your project](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/api-reference/developer-guides/mobsdk-ios-config-project.md).
-   Activate and configure the Mobile SDK on your ServiceNow instance. For details, see [Configure certificate and create an OAuth Entity](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/api-reference/developer-guides/mobsdk-ios-config-certificate.md).
-   Configure guest users. For details, see [Configure guest user access](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/api-reference/developer-guides/mobsdk-config-guest-access.md).

After all of these tasks are completed, you can then start creating mobile applications that leverage the significant resources on your ServiceNow instance.
