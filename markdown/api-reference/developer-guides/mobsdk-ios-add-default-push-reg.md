# Add a default push registration

Register the push notification so that all users of your application receive the push notifications by default.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **System Notification** &gt; **Push** &gt; **Push Application**.

2.  Locate and open the push notification that you created earlier.

3.  In the related list, select the **Push Default Registrations** tab, then select **New**.

4.  In **Notification**, enter `Virtual Agent Message Push Notification`.

    \[Omitted image "mobsdk-push-default\_reg.png"\] Alt text: Virtual Agent push notification

5.  Select **Submit**.
