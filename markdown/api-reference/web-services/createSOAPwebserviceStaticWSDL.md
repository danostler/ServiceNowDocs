# Create a scripted SOAP web service using a static WSDL

Follow these examples to create a scripted SOAP web service using a static WSDL.

-   **[Create a scripted web service using a static WSDL](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/api-reference/web-services/t_CreateStaticWSDLScriptWebService.md)**  
To use a static WSDL, create a scripted web service.
-   **[Create a static WSDL](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/api-reference/web-services/t_CreateAStaticWSDL.md)**  
Create a static WSDL with the required format to override the standard WSDL for your scripted web service.
-   **[Create a static WSDL script include](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/api-reference/web-services/t_CreateAStatickWSDLScriptInclude.md)**  
Create a script include to define the majority of the code used to process static WSDL requests.
-   **[Use the static WSDL](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/api-reference/web-services/r_UsingAStaticWSDL.md)**  
Load the static WSDL into a SOAP client to make requests to the SOAP web service.
