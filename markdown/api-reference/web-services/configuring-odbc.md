# Configuring the ODBC driver

After installing the ODBC driver, configure it to connect to your ServiceNow instance and to communicate through a proxy server, if applicable, and set properties to control ODBC behavior.

-   **[Configure the ODBC driver](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/api-reference/web-services/t_ConfigureTheODBCDriver.md)**  
Configure the ODBC driver to connect to your ServiceNow instance.
-   **[Configure the global DSN default](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/api-reference/web-services/t_ConfigureTheGlobalDSNDefault.md)**  
Configure the global default used by all newly created DSNs.
-   **[Create a new DSN](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/api-reference/web-services/t_CreatingANewDSN.md)**  
Use the ODBC driver and the ServiceNow data source to create an unlimited number of DSNs configured to connect with different instance URLs.
-   **[Specify a connection string](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/api-reference/web-services/specify-connection-string.md)**  
You can specify a connection string instead of defining a DSN.
-   **[Configure the logging level of the ODBC driver](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/api-reference/web-services/t_CnfgLoggingLevel.md)**  
Change the logging level of the ODBC driver.
-   **[Configure the ODBC driver for large data sets](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/api-reference/web-services/configure-odbc-large-data-sets.md)**  
You can set two ODBC driver properties to deal with errors you receive when using queries that return large amounts of data.
-   **[Configure ODBC to use proxy servers](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/api-reference/web-services/configure-odbc-proxy-servers.md)**  
The ODBC driver can be configured to route its HTTP SOAP requests via an HTTP proxy server.
-   **[Setting ODBC properties](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/api-reference/web-services/r_SettingODBCProperties.md)**  
The following properties customize connectivity and optimize the query behavior of the ODBC driver.
