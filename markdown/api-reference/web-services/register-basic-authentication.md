# Register Basic Authentication

Register basic authentication for your inbound API integrations to effectively monitoring their API usage on the **Inbound API Usage Dashboard**.

## Before you begin

Role required: oauth\_admin, web\_service\_admin

## Procedure

1.  Navigate to **All** &gt; **System Web Services** &gt; **Inbound API Integration Usage dashboard** &gt; **Registered integrations**.

2.  Select **New** on the Integration Provider Registries page or choose the existing record from the list.

    **Note:** The Integration Provider Registries page, shows all the inbound integration records.

3.  Specify the following fields to register the integration:
