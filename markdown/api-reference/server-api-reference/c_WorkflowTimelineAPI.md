# WorkflowTimeline- Global

The WorkflowTimeline script include provides methods to generate a view that shows a workflow context on the timeline.

## WorkflowTimeline-getItems\(\)

Generates the Ajax response for the timeline.

This method is called by the Prototype JavaScript Framework. See the AbstractTimelineSchedulePage script include for more details.
