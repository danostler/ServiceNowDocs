# Scripted REST API examples

Mutliple examples are available demonstrating how to create and use scripted REST APIs.

-   **[Scripted REST API example - script samples](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/api-reference/rest-api-explorer/r_ScriptedRESTServiceScriptExamples.md)**  
These examples demonstrate how to create various resource scripts for a scripted REST API.
-   **[Scripted REST API example - streaming vs object serialization](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/api-reference/rest-api-explorer/r_ScriptedRESTExampleStreamVsLO.md)**  
These examples demonstrate how to send a JSON response using streaming and using default object serialization.
-   **[Scripted REST API example - streaming file attachments](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/api-reference/rest-api-explorer/r_ScriptedRESTExampleStreamAttach.md)**  
This example demonstrates how to send an image attachment to a requesting user as a binary stream.
