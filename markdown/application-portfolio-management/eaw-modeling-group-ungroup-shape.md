# Convert a shape to a group shape

Group shapes enable you to combine multiple related shapes into a single container for better organization and clarity in diagrams. It’s useful for representing logical groupings such as a set of applications under a business capability, related processes under a value stream stage.

## Before you begin

Role required: sn\_apm.apm\_user

## About this task

You can convert all general shapes to group shapes, except shapes such as actor, sticky note, and junction.

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Architecture Workspace**.

2.  Open the Modeling page by selecting the Modeling icon \(\[Omitted image "icon-modeling-logo.png"\] Alt text: Modeling\).

3.  Create a blank diagram or open an existing diagram.

4.  Add a shape from the **General** shapes section.

5.  Select the **Convert to groupable shape** icon.

    The shape becomes a container shape. You can add other elements in the shape. The parent shape and child shapes behave as a single group.

    **Note:** To ungroup and make it as a normal shape, select **Revert to normal shape** icon.
