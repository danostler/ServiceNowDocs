# Working with the Setup page

As an Enterprise Architect, use the Setup page to configure Enterprise Architecture Workspace features and functionality.

-   **[View all application categories](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/application-portfolio-management/eaw-view-all-application-categories.md)**  
You can view the list of all application categories in the Enterprise Architecture Workspace.
-   **[View all application category groups](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/application-portfolio-management/eaw-view-all-application-category-groups.md)**  
You can view the list of all application category groups in the Enterprise Architecture Workspace.
-   **[View all application families](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/application-portfolio-management/eaw-view-all-application-families.md)**  
You can view the list of all application families in the Enterprise Architecture Workspace.
-   **[Working with indicators](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/application-portfolio-management/eaw-manage-indicators.md)**  
Manage capability and application indicators, and regenerate capability and application indicator scores on-demand in Enterprise Architecture Workspace.
-   **[View all scoring profiles](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/application-portfolio-management/eaw-view-all-scoring-profiles.md)**  
You can view the list of all scoring profiles in the Enterprise Architecture Workspace.
-   **[View information data domains](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/application-portfolio-management/eaw-view-info-data-domains.md)**  
View the list of all data domains configured in the Enterprise Architecture Workspace.
-   **[View all demand-actions](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/application-portfolio-management/eaw-view-demand-actions.md)**  
View all preconfigured demand actions in the Enterprise Architecture Workspace that help you enhance the capability of the applications.
-   **[View all certification policies](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/application-portfolio-management/eaw-view-all-cert-schedules.md)**  
View the list of all certification policies and manage them in the Enterprise Architecture Workspace.
-   **[Working with the document templates](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/application-portfolio-management/document-templates-for-ea.md)**  
Document Templates define how architectural documents are generated from diagrams and architectural artifacts in Enterprise Architecture Workspace.
