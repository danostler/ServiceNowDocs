# Create profile indicator form

You can use indicators within many scoring profiles, which generate indicator scores unique to that scoring profile.

## Create profile indicator form fields

<table id="table_eql_wth_tzb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Profile

</td><td>

Name of the application scoring profile.

</td></tr><tr><td>

Used in CI score calculation

</td><td>

Option for using the application indicator in calculating the application score.

</td></tr><tr><td>

Evaluate With In Scoring Profile Applications

</td><td>

Option for considering the business applications tied to the selected scoring profile in the evaluation of scores.

 Clearing the check box entails evaluation of all business applications within the enterprise or across all scoring profiles.

</td></tr><tr><td>

Indicator

</td><td>

Name of the application indicator.

</td></tr><tr><td>

Domain

</td><td>

The domain to which this indicator belongs.

</td></tr><tr><td>

Weightage

</td><td>

Numeral value for the indicator. The weight provided in the application score profile for an indicator contributes to the total score of the application.All indicator weight within a scoring profile must add up to 100.

</td></tr></tbody>
