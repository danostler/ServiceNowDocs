# Entity configuration form

The entity configuration form helps you to configure entities that can be used in the Enterprise Modeling and Visualization.

## Entity configuration from fields

<table id="table_tvq_dzn_1cc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Name of the entity.

</td></tr><tr><td>

Entity

</td><td>

Table name of the entity.

</td></tr><tr><td>

Display field

</td><td>

Name of the field to be displayed in the Enterprise Modeling and Visualization.

</td></tr><tr><td>

Diagram action

</td><td>

Action name in the diagram.

</td></tr><tr><td>

Insert allowed

</td><td>

Select to allow creating a new entity from the diagram.

</td></tr><tr><td>

View

</td><td>

Name of the form view for the entity. This is the view that opens in the side panel when an entity is opened in the Enterprise Modeling and Visualization diagram. For example, the views available for a business application are Default view, Business Application view, Business Applications view, Business Application ReadOnly view.**Note:** If this field is empty, the Default view gets applied for the entity.

</td></tr></tbody>
