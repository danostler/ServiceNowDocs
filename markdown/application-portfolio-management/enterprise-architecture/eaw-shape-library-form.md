# Shape library form

You can add new shapes to the shapes library as per your requirement. Use these shapes to create the diagrams in Enterprise Modeling and Visualization.

## Shape library form fields

|Field|Description|
|-----|-----------|
|Name|Enter a name for the shape library.|
|Application|Name of the application that the shape library is associated with. This field is auto-populated as Global and can't be edited.|
|Active|Select the check box to show the shape library in the Enterprise Architecture Workspace.|
|Domain|Name of the domain. Use the look up icon to select a domain name.|
