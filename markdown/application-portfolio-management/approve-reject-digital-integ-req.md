# Approve or reject a digital integration request

As an Enterprise Architect user, approve or reject a digital integration request submitted by Enterprise Architecture \(formerly APM\) users.

## Before you begin

The user must be part of the Enterprise Architect Group.

Role required: sn\_apm.apm\_analyst

## Procedure

1.  Navigate to **All** &gt; **Service Desk** &gt; **My Approvals**.

2.  Select the digital integration request that you want approve or reject.

3.  Select **Approve** or **Reject**.
