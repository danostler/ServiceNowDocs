# Add a data classification group - Legacy

Create a data classification group to categorize data classifications.

## Before you begin

Role required: sn\_apm.apm\_admin

## Procedure

1.  Navigate to Data Classification Group \[cmdb\_data\_classification\_group\] table.

2.  Select **New**.

3.  On the form, fill in the fields.

    For field information, see [Data classification group form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/application-portfolio-management/data-class-group-form.md).

    The applied tags are displayed in various colors on the top of the form.

4.  Select **Submit**.
