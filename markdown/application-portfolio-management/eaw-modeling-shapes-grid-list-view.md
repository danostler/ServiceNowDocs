# Switch to list or grid view of shapes panel

Switch between List View and Grid View in the shapes panel according to your modeling needs.

## Before you begin

Role required: sn\_apm.apm\_user

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Architecture Workspace**.

2.  Open the Modeling page by selecting the Modeling icon \(\[Omitted image "icon-modeling-logo.png"\] Alt text: Modeling\).

3.  Create a new diagram or open an existing diagram.

4.  Select the **Switch to list view** or **Switch to grid view** icon on the diagram page to switch between the list or grid view of the shapes.
