# Add a data classification - Legacy

Create a data classification to apply it to an information object. Effectively control the data used by the business applications.

## Before you begin

Role required: sn\_apm.apm\_admin

## Procedure

1.  Navigate to Data Classifications \[cmdb\_data\_classification\] table.

2.  Select **New**.

3.  On the form, fill in the fields.

    For field information, see [Data classification form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/application-portfolio-management/data-classification-form.md).

    The applied tags are displayed in various colors on the top of the form.

4.  Select **Submit**.
