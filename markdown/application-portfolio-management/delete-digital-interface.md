# Delete a digital interface

Delete a digital interface that you no longer need.

## Before you begin

Role required: sn\_apm.apm\_analyst

## Procedure

1.  Navigate to **All** &gt; **Enterprise Architecture** &gt; **Application Portfolio** &gt; **Digital Interfaces**.

2.  Open a digital interface number by clicking it.

    **Note:** You cannot delete a digital interface if it has any digital integration associated with it.

3.  Click **Delete**.

4.  Confirm the deletion.
