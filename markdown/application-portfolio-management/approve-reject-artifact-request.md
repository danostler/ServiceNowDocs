# Approve or reject an artifact version request - Legacy

As an Enterprise Architect, create, edit, and approve or reject architectural artifacts version requests that are submitted by other users.

## Before you begin

Role required: sn\_apm.apm\_analyst

## About this task

When a user requests an artifact, an email notification is received by the approver.

## Procedure

1.  Navigate to **All** &gt; **Service Desk** &gt; **My Approvals**.

2.  Select the artifact request that you want approve or reject.

3.  Select **Approve** or **Reject**.


## Result

The requester receives an email notification for the approval or rejection.
