# Download an artifact version - Legacy

Download an approved artifact version to view the architectural diagram in it.

## Before you begin

Role required: sn\_apm.apm\_user

## Procedure

1.  Navigate to **All** &gt; **Architectural Artifacts** &gt; **Versions**.

2.  Select the architectural artifact version that you want to download.

3.  Click **Download Artifact**.

    **Note:** You cannot download an unapproved artifact version or an artifact version that is a URL. If you try to download these, you get an error message.


## Result

An approved version of the document is downloaded.
