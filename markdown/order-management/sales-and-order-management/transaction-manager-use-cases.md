# Transaction Manager: Use cases

View a list of example use cases for ServiceNow CPQ Transaction Manager.

-   **[Transaction Manager use case: Apply parent line discounts to child lines](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/order-management/sales-and-order-management/transaction-manager-use-case-apply-parent-line-discounts-to-child-lines.md)**  
Transaction Manager can help manage transactions whose configurable products have many child and grandchild transaction line items.
-   **[Transaction Manager use case: Calculate prices for line-level fields](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/order-management/sales-and-order-management/transaction-manager-use-case-calculate-prices-for-line-level-fields.md)**  
Transaction Manager can apply rules to line-level fields that dynamically calculate prices according to changes to the field, such as when products are added or removed.
-   **[Transaction Manager use case: Calculate the total of line-level prices](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/order-management/sales-and-order-management/transaction-manager-use-case-calculate-the-total-of-line-level-prices.md)**  
Transaction Manager can include a determination rule that calculates the sum of line-level net prices so that it can be stored in a header-level field.
-   **[Transaction Manager use case: Display LIG line number on line detail](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/order-management/sales-and-order-management/transaction-manager-use-case-display-lig-line-number-on-line-detail.md)**  
How to display the Line Item Group line number in the Line Detail.
-   **[Transaction Manager use case: Document generation with Google Docs](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/order-management/sales-and-order-management/transaction-manager-use-case-document-generation-with-google-docs.md)**  
Generate documents from Transaction Manager by using Google Docs and Google Apps Script.
