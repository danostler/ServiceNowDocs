# Account Details form

The Accounts Details form enables you to create or review account details for a customer or for a partner.

<table id="table_qrh_qrq_4nb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Name of the enterprise or customer.

</td></tr><tr><td>

Number

</td><td>

Unique system-assigned account number, starting with a prefix ACCT.

</td></tr><tr><td>

Primary Contact

</td><td>

Name of the primary contact person for the account.

</td></tr><tr><td>

Parent Account

</td><td>

Name of the parent account, if any, for this account. -   For example, if you are setting up an account that is a national branch of a multi-national corporation, the parent account is the one for the multi-national corporation.
-   If there isn't a parent account, leave this field empty.

</td></tr><tr><td>

Registration Code

</td><td>

Registration code, if any, assigned to the account.

</td></tr><tr><td>

Customer

</td><td>

Option that designates if this account is for a customer.

</td></tr><tr><td>

Partner

</td><td>

Option that designates if this account is for a business partner.

</td></tr><tr><td>

Website

</td><td>

URL of the website for this account, if any.

</td></tr><tr><td>

Phone

</td><td>

</td></tr><tr><td>
