# Recall a quote approval request

Cancel a pending quote approval request to change a quote before resubmitting it for advanced approval.

## Before you begin

Role required: sn\_adv\_appr\_mgmt.approval\_request\_writer

## About this task

Recalling an approval request cancels any pending approval steps and returns the quote to the Draft state. Recall a quote approval request when changes are required after submission. For example, recall the request when quote details must be corrected or updated before the quote can proceed through the approval process.

## Procedure

1.  Navigate to **Workspaces** &gt; **CSM/FSM Configurable Workspace**.

2.  Select the List icon.

3.  Navigate to **Quotes** &gt; **All**.

4.  Select the quote with the approval request that you want to recall.

5.  Select the **Approvals** tab.

6.  Select **Recall approval request**.


## Result

The approval request is canceled, all pending approval steps are canceled, and the quote is returned to the Draft state. You can update the quote and resubmit it for advanced approval when ready.
