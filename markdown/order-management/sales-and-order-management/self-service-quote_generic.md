# Quote creation via Self-Service

The Quote Self-Service plugin \(com.sn\_quote\_self\_service\) enables users to create and manage quotes. Users can track quote status, browse catalogs, configure items, and apply discounts through a guided quote creation playbook.

## Key benefits of creating quotes via Self-Service

The Quote Self-Service application \(com.sn\_quote\_self\_service\) offers the following benefits:

-   A guided quote creation playbook simplifies the quoting process.
-   Self-service catalog browsing, product configuration, and quote building reduces quote turnaround time from days to minutes.
-   Users can directly edit, revise, apply discounts, and download without requesting changes through internal teams.
-   Audit trail and version history for revised quote prompting trust and transparency.

For information about the Quote creation via Self-Service fields, see [Quote creation via Self-Service fields](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/order-management/sales-and-order-management/quote-creation-fields-self-service.md).
