# Update validation settings in a matrix

Change the rule validation checks for a matrix. These checks run automatically when you save changes made to the matrix decision table.

## Before you begin

Role required: sn\_prd\_pm.product\_catalog\_admin or sn\_prd\_pm.product\_catalog\_manager

## Procedure

1.  In the CSM Configurable Workspace, select the **List** \[Omitted image "list-outline-24.svg"\] Alt text: view.

2.  Navigate to **Context Rule Management** &gt; **Rule Matrices**.

3.  Select the matrix to update.

4.  In the **Rule validations** field, change the rule validation.

    -   To add a validation rule, select the Search icon and select the rule validation.
    -   To delete a validation, select the X in the validation pill.
5.  Select **Save**.
