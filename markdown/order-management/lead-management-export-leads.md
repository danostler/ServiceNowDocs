# Export leads

You can export a quote as a JSON file in Lead Management and provide it to a customer.

## Before you begin

Role required: sales\_agent

## About this task

The export lead feature in Lead Management exports a list of all leads. Use the following steps to export leads.

## Procedure

1.  In the CSM Configurable Workspace, select the **List** \[Omitted image "list-outline-24.svg"\] Alt text: view.

2.  Navigate to **Leads** &gt; **All**.

3.  Select the **File Type**:

    -   Excel
    -   CVS
    -   JSON
    -   PDF
4.  Select the **Delivery Type**:

    -   Email - Enter a valid email address.
    -   Download - Downloads the export to a local drive.
    The list of leads is exported according to the parameters set.
