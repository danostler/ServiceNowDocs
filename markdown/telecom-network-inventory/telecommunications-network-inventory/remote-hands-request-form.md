# Remote Hands Request form

Describes the field information in the Remote Hands Request form.

<table id="table_htp_wkd_wrb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Account

</td><td>

Customer account for which the Remote Hands request is being raised.

</td></tr><tr><td>

Category

</td><td>

Type of Remote Hands service being requested. Available options include: -   **Power Usage Enquiry**: Allows users to request power usage details for a specific device
-   **Equipment Installation**: Allows users to request the installation of devices at a specified location.
-   **Equipment Restart**: Allows users to request a device restart
-   **Other Remote Hands**: If a user’s request does not fall under the listed categories, they can select Other Remote Hands and describe the requested service in the **Additional Information** field.

</td></tr><tr><td>

Contact

</td><td>

Contact assigned under the account

</td></tr><tr><td>

Install Base Item

</td><td>

Configuration item \(CI\) or device on which the action is to be performed.

</td></tr><tr><td>

Summary

</td><td>

Brief description of the request.

</td></tr><tr><td>

Additional Information

</td><td>

Detailed instructions or context related to the request.

</td></tr><tr><td>

Add attachments

</td><td>

Supporting documents, images, or reference files relevant to the request.

</td></tr></tbody>
