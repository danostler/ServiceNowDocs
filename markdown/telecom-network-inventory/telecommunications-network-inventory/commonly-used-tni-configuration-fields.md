# Commonly used network asset instance configuration fields

These fields, listed in alphabetic order, are common to some or most of the Configuration sections in the Telecommunications Network Inventory forms. The ServiceNow AI Platform uses them to configure individual network assets when you define your network asset instances.

<table id="table_htp_wkd_wrb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Comments

</td><td>

Free form text that is used to comment on a network asset. For example, `Duty tech is Rahul Dev`.

</td></tr><tr><td>

Dimensions

</td><td>

Physical dimensions of the network asset.

</td></tr><tr><td>

Distinguished name

</td><td>

Alternate name reference for the network asset that is based on the concatenated names and IDs from the other related network assets.

</td></tr><tr><td>

Manufacturer

</td><td>

Name of the network asset's manufacturer. Select the search icon \(\[Omitted image "search.png"\] Alt text: Search icon.\) and select a manufacturer code. To learn more, see [Create manufacturer and vendor codes](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/telecom-network-inventory/telecommunications-network-inventory/add-company-designate-manufacturer-tni.md).

</td></tr><tr><td>

Model number

</td><td>

Manufacturer's model number for this network asset.

</td></tr><tr><td>

Orientation

</td><td>

Physical orientation of the slots in this network asset:-   **--None--**

No specific physical slot orientation.

-   **Horizontal**

Horizontal slot orientation.

-   **Vertical**

Vertical slot orientation.


</td></tr><tr><td>

Network domain

</td><td>

Domain of ownership and responsibility for this network asset or connection. Select one of the following options:

-   **Mobility**

Represents wireless devices and connections.

-   **Telco**

Represent the edge or access networks.

-   **Core**

Represents the core network infrastructure.


</td></tr><tr><td>

Operation notes

</td><td>

Free-form operation note text for this network asset. For example, `Check diesel fuel for generator`.

</td></tr><tr><td>

POTS Number

</td><td>

Plain Old Telephone Service number that you are associating with this network site.

</td></tr><tr><td>

Product Model

</td><td>

Product model, if any, that is associated with this network asset. Select the search icon \(\[Omitted image "search.png"\] Alt text: Search icon.\) and select a product model.

</td></tr><tr><td>

Serial number

</td><td>

Assigned serial number for this network asset.

</td></tr></tbody>
