# Compute and Create Logical Connection form

The Compute and Create Logical Connection form enables you to create, review, and modify the connection details for a logical connection.

Description

</th></tr></thead><tbody><tr><td>

Starting network equipment where this connection is configured.

Start interface

</td><td>

Starting network interface where this connection is configured.**Note:** If this field is left empty, it’s automatically set by using the path computation and created logical connection.

</td></tr><tr><td>

Logical connection Model

</td><td>

Logical connection model where this connection is configured.**Note:** The topology connection models are not listed.

End site

</td><td>

Ending network equipment where this connection is configured.

</td></tr><tr><td>

</td><td>

Ending network interface where this connection is configured.**Note:** If this field is left empty, it’s automatically set by using the path computation and created logical connection.

Bandwidth

</td><td>
