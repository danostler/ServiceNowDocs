# Open site map from a network site form

Open Network site map from a network site form in the Telecommunications Network Inventory application. You can understand the geographical location of the network site and information about the connections.

## Before you begin

Role required: sn\_ni\_core.inventory\_admin, sn\_ni\_core.inventory\_agent

## Procedure

1.  Navigate to **Workspaces** &gt; **Network Inventory Workspace**.

2.  Click the list icon \(\[Omitted image "ni-workspace-list-icon.png"\] Alt text: List icon.\) and then go to **Network Sites** &gt; **All Sites**.

    You can view the sites that you manage by going to **Network Sites** &gt; **My Sites**.

3.  Open a network site that you want to see the details.

4.  Select **Network Site Map**.

    The network site form displays the **Network Site Map** button only if the site location has a valid latitude and longitude.


## Result

The Network site map page opens and displays your network site.
