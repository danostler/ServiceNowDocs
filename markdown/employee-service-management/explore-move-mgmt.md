# Exploring Workplace Move Management

Manage and fulfill employee requests with Workplace Move Management.

Enable employees to collaborate, communicate, and easily manage space changes from start to finish. As an employee, you can submit a request using the Workplace Service Portal. As a manager, you can make requests directly through the application.

As a manager, the automated process helps you to closely supervise the movement of employees and their belongings with minimal to no disruption. You can fulfill move requests through move cases and associated tasks. You can view the move requests raised by employees, and approve or reject a request anytime. The application enables you to identify seating conflicts while fulfilling move requests.

As an employee, raise a move request to move from one place to another using the Workplace Service Portal. Move to another space located in the same building or a different building. You can perform the following actions:

-   Raise a move request for yourself or for a co-worker.
-   Select a reason for move.
-   Select a date on which you want the movement to be implemented.
-   Specify the 'From' and 'To' location. You can also select a workplace directly on the floor map of the building.
-   Select equipment that must be moved along with the request.
-   Specify any extra information or instruction if required.

As an admin, the Workplace Move Management application enables you to perform the following actions:

-   Raise a move request for an employee directly using the application rather than using the Workplace Service Portal.
-   Monitor the move requests that are created in the application. Identify seating conflicts while processing move requests.
-   View or modify move requests or move tasks that are assigned to you, the ones that are currently open, and also all the move requests.
-   Automate and configure approvals for those move requests that require permissions.
-   Send email notifications to relevant stakeholders regarding the move requests when required.
-   Modify, approve, or reject a planned or ongoing move request.

## Workplace Central - Move management

Group multiple move requests under a move project and track their progress at a time using the Workplace Central Move Management workspace.
