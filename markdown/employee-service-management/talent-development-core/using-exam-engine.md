# Using Career Assessment

Use Career Assessment to create exams.

-   **[Create exam catalogs](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/talent-development-core/create-exam-catalog.md)**  
Create exam catalogs to be able to segregate exams based on category.
-   **[Create an exam](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/talent-development-core/create-exam.md)**  
Create an exam on Career Assessment.
-   **[Add exams to a learning path from Career Assessment](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/talent-development-core/create-quiz-path.md)**  
Create internal learning content to add exams as content items within a learning path from Career Assessment.
-   **[View an exam instance](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/talent-development-core/view-exam-instance.md)**  
View an exam instance to verify number of attempts, scores and completion status of an exam assigned to a user.
