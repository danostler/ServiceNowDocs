# Talent Aspiration types form

Administrators create aspiration types that are use by employees and managers.

|Field|Description|
|-----|-----------|
|**Active**|When selected, the aspiration type is available to be used.|
|**Application**|The application that's associated with this aspiration type.|
|**Description**|Details about the type of aspiration that helps guide you to the appropriate aspiration.|
|**Icon**|Select an icon that's displayed for each aspiration. You’re responsible to provide any icon files for your instance.|
|**Name**|A descriptive name of the aspiration type that helps employees identify which type they want to use.|
|**Order**|A numeric value that determines the order in which the **Aspiration type** is displayed in the user interface \(UI\).|
|**Tip**|Text that's display in the UI with each listed aspiration type.|
|**Title**|The UI text that's visible for each listed aspiration type.|
