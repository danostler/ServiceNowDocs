# View Talent Development Core licensed profiles

You can view a list of profiles that are licensed for Talent Development.

## Before you begin

Role required: sn\_egd\_core.admin

## About this task

The **Employee Profile** licenses users by the employee profile number. Each number is associated with a user. You can view the user name in the profile.

## Procedure

1.  Navigate to **All** &gt; **Employee Profile** &gt; **Application licensing** &gt; **Employee licensing**.

2.  Open an existing record.

    If you don't have any existing **Employee Application Access Criteria** records, you can create a record and view the matched profiles.

3.  In an existing **Employee Application Access Criteria** record, select the link above the **Criteria** field.

    The link shows the number of profiles that match the condition. If you only see the employee profile number, select the gear icon, and add the **User** field to the selected column.
