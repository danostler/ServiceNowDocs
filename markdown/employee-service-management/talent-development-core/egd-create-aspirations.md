# Create a Talent Development Aspiration

Create Aspirations to define and track professional and personal interests.

## About this task

**Note:**

When there are no role levels and no job profile configured, the Aspiration choice isn't available in the Your Career drop-down list.

## Before you begin

Role required: None

## Procedure

1.  From the Employee Center, navigate to **Your Career** &gt; **Aspirations**.

2.  Select **Create a new aspiration**.

3.  Select from the available **Aspiration Types**.

    If you don't see an aspiration type that matches your goal, you can select **Something else** to create a customized aspiration.

4.  Select **Next**.

5.  Select from the available **Interests**.

    If you selected **Something else**, you go directly to a form for creating your **Aspiration**.

6.  Fill in the form for creating an **Aspiration**.

    Some of the fields are filled in with your previous selections. Complete any empty or selectable fields.

7.  Select **Next**.

    You have the option to **Save as draft** so that you can take sometime to think about your aspiration before submitting it.

8.  Select who you want to share your aspiration with.

9.  Select **Submit**.


## What to do next

You can create a **Growth Plan** from your **Aspiration**. For details about creating growth plans, see [Create a growth plan](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/talent-development-core/egd-create-growth-plan.md)
