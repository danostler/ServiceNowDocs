# Proficiency level autofill configuration form

Form to create a configuration to autofill the proficiency for a skill.

<table id="table_j1w_jt2_1yb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Skill level type

</td><td>

The applicable skill level type for the configuration.

</td></tr><tr><td>

Role group

</td><td>

The applicable role group for the configuration.**Note:** The configuration is applied only to the selected role group. To indicate a global configuration that is applied to all role groups, leave the field blank.

</td></tr><tr><td>

Initial job level

</td><td>

The beginning job level for the configuration.

</td></tr><tr><td>

Initial proficiency

</td><td>

The beginning level of proficiency for the initial job level.

</td></tr><tr><td>

Active

</td><td>

Option to mark the configuration as active.

</td></tr><tr><td>

Order

</td><td>

The order in which to apply the configuration. For example, an order of 100 is applied first and an order of 200 is applied next.

</td></tr><tr><td>

Progression

</td><td>

The progression from one level to another. The options are:-   None - No progression is applied.
-   Incremental - Increasing order of progression.
-   Decremental - Decreasing order of progression.
-   Fixed - Fixed order of progression.

</td></tr><tr><td>

Progression type

</td><td>

The type of progression based on the option selected in the **Progression** field. The options are:-   Step - The progression increases or decreasing one step at a time across the job levels.
-   Spread - The progression is uniformly spread across the job levels.

</td></tr></tbody>
