# Create Talent Development Outcome Types

As an Talent admin you create outcome types that can be used in growth plans.

## Before you begin

Role required: sn\_egd\_core.admin

## About this task

An **Outcome Type** is used to categorize activities into logical outcomes.

## Procedure

1.  Navigate to **All** &gt; **Talent Development Core** &gt; **Growth Plans** &gt; **Outcome Types**.

2.  Select **New**.

3.  Complete the **Outcome Types** form.

    Read details about the different fields in [Talent Outcome Types reference](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/talent-development-core/egd-outcome-types-form.md)

4.  Select **Submit**.
