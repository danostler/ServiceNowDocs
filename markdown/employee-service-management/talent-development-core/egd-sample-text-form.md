# Talent Sample texts reference

Sample text is content that is used in different types of content in a selected document. Administrators create sample text templates.

|Fields|Description|
|------|-----------|
|**Active**|Indicates whether this sample text is available for use.|
|**Linked content**|Add links to the content that you want to add to the sample.|
|**Name**|A descriptive name for the content sample. The name of the sample helps administrators select with documents to add the content to.|
|**Order**|A numeric value that determines the display order of the sample text in the user interface \(UI\).|
|**Sample for**|Select a denouncement from the list where the sample text is used.|
|**Sample text**|The sample text that's used in the selected document.|
|**Table**|The table used to track where the sample text is used.|
