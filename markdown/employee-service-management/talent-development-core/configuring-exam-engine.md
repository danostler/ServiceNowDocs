## Allow RCAs

After you install Career Assessment, you might encounter Restricted Caller Access \(RCA\) approval messages requesting an update in the access request. Allow the following RCAs:

-   Target Human Resources Core
-   Target Journey Accelerator
-   Target Journey Designer
