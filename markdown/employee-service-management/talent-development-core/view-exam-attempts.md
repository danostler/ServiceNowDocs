# View number of exam instance attempts

View number of exam instance attempts by a user.

## Before you begin

Role required: sn\_exam.admin, sn\_exam.catalog\_admin, user

Only a user with the sn\_exam.catalog\_admin role who is configured as part of the admin list or is an owner of the catalog may view an exam instance within the associated catalog. Users assigned to the exam may also view associated exam instance attempts.

## Procedure

1.  Navigate to **All** &gt; **Career Assessment** &gt; **Exams** &gt; **Exam Instances**.

    The following modules are available based on the state of the instances.

    -   **All**: Displays exam instances in all states.
    -   **Assigned**: Displays exam instances that are yet to be attempted.
    -   **In Progress**: Displays exam instances that are in progress.
    -   **Complete**: Displays exam instances that are complete.
2.  Select the exam instance number that you want to view from the required module.

3.  Select the exam instance attempt number that you want to view.

4.  Review the fields of the form.

    For a description of the field values, see [Exam instance attempt form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/talent-development-core/exam-instance-attempt-form.md).
