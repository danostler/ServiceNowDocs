# Talent Outcome Types reference

Outcome types are created by administrators.

An **Outcome Type** indicates the desired outcome of a growth plan. Administrators define outcome types to help managers and employees create meaningful growth plans.

|Field|Description|
|-----|-----------|
|**Active**|Indicates whether this outcome type is available for use in a growth plan.|
|**Name**|The name of an outcome type that indicates what the achievement goal is for the growth plan.|
|**Order**|A numeric value that determines the display order of outcome types in the user interface \(UI\).|
|**Value**|A string that's used for tracking and grouping outcome types.|
