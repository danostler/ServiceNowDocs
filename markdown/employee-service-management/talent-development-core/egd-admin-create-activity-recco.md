# Create Talent Development Activity Recommendations

As an Talent admin you create **Activity Template Recommendations** that are used in growth plans.

## Before you begin

Role required: sn\_egd\_core.admin

## About this task

Controls whether the activity is **Required** or **Recommended** for the growth plan.

## Procedure

1.  Navigate to **All** &gt; **Talent Development Core** &gt; **Activity Template Recommendation**.

2.  Select **New**.

3.  Complete the **Activity Template Recommendation** form.

    Read details about the different fields in [Talent Activity Template Recommendation form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/talent-development-core/egd-activity-recommendation-form.md).

4.  Select **Submit**
