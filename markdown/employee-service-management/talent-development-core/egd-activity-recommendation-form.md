# Talent Activity Template Recommendation form

Administrators create activity template recommendation templates that employees and managers use to create career growth plans.

<table id="table_yfs_stj_twb"><thead><tr><th>

Fields

</th><th>

Description

</th></tr></thead><tbody><tr><td>

**Active**

</td><td>

When selected, the activity template is available to be used in a conversation.

</td></tr><tr><td>

**Activity template**

</td><td>

Select an **Activity template** from the list of template available to your organization.

</td></tr><tr><td>

**Availability**

</td><td>

-   Select **Recommended** to make the activity available as a choice to use, but it isn’t required.
-   Select **Required** to make the activity required to use.

</td></tr><tr><td>

**Description**

</td><td>

More detailed information about the activity. Detailed information can help managers and employees decide if they want to use this activity.

</td></tr><tr><td>

**Name**

</td><td>

The title is visible in the user interface \(UI\). A helpful title provides information to help make a choice whether to use the activity.

</td></tr><tr><td>

**Outcome type**

</td><td>

-   Select **Free form** if the activity template doesn't fit into the **Job growth** or **Skill development** categories.
-   Select **Job growth** if the activity is directly related to growth and development for your job.
-   Select **Skill development** for training that's related to a any skill you want to improve. Skill development can be job-related or for personal development.

</td></tr><tr><td>

**User criteria**

</td><td>

Select the criteria that must meet to have access to this talking point.

</td></tr></tbody>
