# Add skill to an achievement template

Associate a skill to an achievement credential template in Achievements and Credentials.

## Before you begin

Role required: sn\_lep.achievement\_manager

## Procedure

1.  Navigate to **All** &gt; **Achievements and Credentials** &gt; **Achievement Credential Templates**.

2.  Select an existing record.

3.  Click **New** in the Achievement Template Skills related list.

    On the form, fill in the fields:

<table id="table_ytt_hp5_c2c"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Credential template

</td><td>

Name of the credential template record that you want to associate a skill to.**Note:** Use the information icon,to view the credential template record

</td></tr><tr><td>

Skill

</td><td>

Option to select the skill you want to associate to the achievement template. **Note:** Use the search option to look up from a list of existing skills.

</td></tr></tbody>
</table>4.  Select **Submit**.
