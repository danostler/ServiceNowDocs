# Delete skills from skills profile

Employees can delete skills from the additional skills section in their skills profile.

## Before you begin

Role required: sn\_skills\_int.emp

## Procedure

1.  Navigate to **All** &gt; **Employee Center**.

2.  In the **Employee Center**, navigate to **Profile** &gt; **Career**.

3.  On the skills widget, select the menu icon that is in the skill pill and then select **Delete skill**.

    **Note:** After a skill is validated, the employee can’t delete the skill from the additional skills section of their skills profile.

4.  Select **Yes, delete skill** to delete the skill.
