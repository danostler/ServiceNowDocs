# Exam instance form

Description of the field values in the Exam Instance form.

|Field|Description|
|-----|-----------|
|Number|System-generated unique exam instance number.|
|Assigned to|Name of the user assigned to the exam instance.|
|Score|Latest marks obtained by the user.|
|Score percent|Percentage of latest marks obtained by the user.|
|Exam|Name of the exam associated with the exam instance.|
|State|Current state of the exam instance.|
|Result|Pass or fail status of the exam instance.|
|Current completed attempts|Number of attempts completed currently.|
|Reattempt time|Wait time before the next attempt.|
