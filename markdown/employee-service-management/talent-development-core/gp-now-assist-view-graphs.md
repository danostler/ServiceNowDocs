# View Growth plans related information in the Now Assist Admin console

From the Now Assist Admin console, view related information for Growth plans created with the help of Now Assist.

## Before you begin

Role required: admin

## About this task

To view Growth plans related information from the Now Assist Admin console, you must be assigned these three roles.

-   \[sn\_na\_analytics.viewer\]
-   \[platform\_analytics\_admin\]
-   \[sn\_td\_na.admin\]

## Procedure

1.  Navigate to **All** &gt; **User Administration** &gt; **Users**.

2.  Open the record for the user that you’re adding roles to.

3.  In the **Roles** related list, select **Edit**.

4.  Select the roles that you want to add.

5.  Select **Save**.
