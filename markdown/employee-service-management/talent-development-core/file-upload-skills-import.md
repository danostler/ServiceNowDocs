# Import skills through file upload

Import your industry specific skills data to your Skills Foundation application using a predefined skills import template so you can import your data in the required format.

## Before you begin

Role required: sn\_skills\_int.admin

## Procedure

1.  Navigate to **All** &gt; **Skills Workspace**.

2.  Select the Lists icon \(\[Omitted image "icon-list.png"\] Alt text: Lists icon\).

3.  Select **Import history** &gt; **Queue**.

4.  Select **Import**.

    You are redirected to the Playbook experience.

5.  Prepare the file to upload.

    1.  Select **download a template**.
    2.  Open the skills import template\(.xlsx file\) on your local system.
    3.  Enter the skill information in the appropriate columns.

        **Note:** The fields in the template are **Name**, **Level type**, **Description**, and **Source** \(non-required fields\).

    4.  Save the Excel file.
6.  Select **Attach file** to select the file for upload in excel or CSV format.

7.  Select your Excel data file.

8.  Select **Open**.

    **Note:** After the file is attached, you’ll have the options to download, rename the file, or delete the attached file.
