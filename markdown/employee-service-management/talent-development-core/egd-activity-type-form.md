# Talent Activity Types reference

Administrators create activity types that are used by managers and employees.

|Field|Description|
|-----|-----------|
|**Active**|Select **Active** to make the activity type available for use in Growth plans.|
|**Description**|The description displayed in the user interface \(UI\) for the activity type.|
|**Name**|The name of the activity type.|
|**Order**|A numeric value used to determine the display order of activity types.|
|**Value**|The value used to track the activity type.|
