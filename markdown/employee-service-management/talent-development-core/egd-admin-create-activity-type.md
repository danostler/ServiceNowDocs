# Create Talent Development Activity Types

As a Talent Development Core admin you create **Activity Types** that are used in growth plans.

## Before you begin

Role required: sn\_egd\_core.admin

## About this task

An **Activity Types** help to identify what an activity is intended to do.

## Procedure

1.  Navigate to **All** &gt; **Talent Development Core** &gt; **Activity Types**.

2.  Select **New**.

3.  Complete the **Activity Type** form.

    Read details about the different fields in [Talent Activity Types reference](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/talent-development-core/egd-activity-type-form.md).

4.  Select **Submit**
