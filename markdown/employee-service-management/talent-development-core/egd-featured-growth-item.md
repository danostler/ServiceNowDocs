# Talent Feature growth item reference

Administrators create featured growth items that are used by managers and employees.

**Note:** This form is dynamic. The fields that appear depend on the options you select as you complete the form.

<table id="table_nxd_j2j_twb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

**Active**

</td><td>

Option to indicate whether the feature growth item is available for use to the employees you specify in the **User criteria** field.

</td></tr><tr><td>

**Description**

</td><td>

Description that provides further details about the feature growth item you want to define.

</td></tr><tr><td>

**Image**

</td><td>

Image for the opportunity that you want to promote as a feature growth item.This field only appears when the **Type** field is set to **Opportunity**.

</td></tr><tr><td>

**Job requisition**

</td><td>

Job requisition number to identify the job role that you want to promote as a feature growth item.This field only appears when the **Opportunity type** field is set to **Internal Job**.

</td></tr><tr><td>

**Opportunity type**

</td><td>

Type of opportunity that you want to promote as a feature growth item to employees within your organization.This field only appears when the **Type** field is set to **Opportunity**.

</td></tr><tr><td>

**Order**

</td><td>

A numeric value that determines the order in which a feature growth item appears in the user interface \(UI\).

</td></tr><tr><td>

**Role level**

</td><td>

Title for the role that you want promote as a feature growth item to employees within your organization.This field only appears when the **Type** field is set to **Job Title**.

</td></tr><tr><td>

**Skill**

</td><td>

Skill that can be selected from the list of available skills for your organization.This field only appears when the **Type** field is set to **Skill**.

</td></tr><tr><td>

**Skill level**

</td><td>

Level of expertise in a skill that is required for the feature growth item you want to define.This field only appears when the **Type** field is set to **Skill**.

</td></tr><tr><td>

**Type**

</td><td>

Type of feature growth item. Options are as follows:-   Select **Job Title** to indicate the title of a job that you want to promote as a feature growth item. The job title that you want to promote appears in the **Role level** field.
-   Select **Skill** to promote a skill as a feature growth item that your employees can acquire.
-   Select **Opportunity** to define an opportunity type you want to promote as a feature growth item.

</td></tr><tr><td>

**User criteria**

</td><td>

Criteria to target specific groups of employees to whom the feature growth item is available. Groups, created by administrators, are logical containers composed of similar users.

</td></tr></tbody>
