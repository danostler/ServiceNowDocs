# Track reservations using the Event planner

As a reservation planner and a reservation viewer, the Event planner enables you to monitor the reservations that are requested in your workplace.

## Before you begin

Ensure that you have installed the Calendar component \(sn\_calendar\) of version 23.2.0 or later.

Role required: sn\_wsd\_rsv.reservation\_planner or sn\_wsd\_rsv.reservation\_viewer

## About this task

Check the number of reservations that are planned for the current day and if there are any extra services. The event planner also displays the reservations that are canceled for today and the reservations that are yet to be approved. The reservations are displayed with the following details by default:

-   Requester of the reservation
-   Location that is reserved
-   Building and campus of the location
-   Start and end time of the reservation
-   Current state of the reservation
-   Type of reservation

You can edit the columns to display the information according to your requirement.

You can view all the reservations that are currently active and also the reservations that are created by you or requested for you.

## Procedure

1.  Navigate to **All** &gt; **Workplace Central** &gt; **Workplace Central**.

    You can also open Workplace Central from the Employee Center directly. Navigate to **Workspaces** &gt; **Workplace Central**.

    The Workplace Analytics dashboard opens.

2.  On the left pane, select the **Event planner** icon \(\[Omitted image "event-planner-icon.png"\] Alt text: Event planner icon.\).

    The Event Planner dashboard opens.

3.  To view the current days reservation statues, do the following:

    1.  Go to the **Reservations** metrics section.

    2.  Select one of the following tiles to view as a list:

        -   **Planned reservations for today**
        -   **Reservations with extra services for today**
        -   **Canceled reservations of today**
        -   **Pending approval reservations**
4.  To view the active reservations, do one of the following:

    -   To view all the reservations, go to the **All Reservations** section.
    -   To view the reservation created by you or for you, go to the **My reservations** section.
    To view more reservations, select **View all**.

5.  To edit the columns of the list view, do the following:

    1.  Select the List actions icon \(\[Omitted image "evnt-plnnr-listactions-icon.png"\] Alt text: List actions.\).

    2.  Select **Edit columns**.

    3.  After you add the columns that you want to view, select **OK**.

6.  To refresh the list, select the Refresh icon \(\[Omitted image "evnt-plnnr-refreshiocn.png"\] Alt text: Refresh the list.\).

7.  To apply a filter on the list, select the Show filter icon \(\[Omitted image "evnt-plnnr-filterlist-icon.png"\] Alt text: Show filter option.\).

8.  To view all the reservations created in the workplace in a schedule view, select **Open schedule view**.
