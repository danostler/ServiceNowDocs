# Print reservation details

Print the details of a reservation created in your workplace.

## Before you begin

Confirm that you have installed the Calendar component \(sn\_calendar\) of version 23.2.0 or later.

Role required: sn\_wsd\_rsv.reservation\_planner or sn\_wsd\_rsv.reservation\_viewer

## Procedure

1.  Navigate to **All** &gt; **Workplace Central** &gt; **Workplace Central**.

    You can also open Workplace Central from the Employee Center directly by navigating to **Workspaces** &gt; **Workplace Central**.

2.  On the Workplace Analytics dashboard, select the Event planner icon \(\[Omitted image "event-planner-icon.png"\] Alt text: Event planner icon.\).

3.  View the current days reservation status in the Event Planner dashboard by doing the following:

    1.  Go to the Reservations metrics section.

    2.  Select one of the following tiles to view as a list:

        -   **Planned reservations for today**
        -   **Reservations with extra services for today**
        -   **Canceled reservations of today**
        -   **Pending approval reservations**
4.  View the active reservations by doing one of the following:

    -   View all the reservations by going to the All Reservations section.
    -   View the reservation created by you or for you by going to the My reservations section.
    View more reservations by selecting **View all**.

5.  Select the reservation that you want to print.

    The details of the reservation are displayed in a new tab.

6.  Select **Print reservation**.

    The workplace reservation report is available in both the Activity and Attachments sections.

    **Note:** The Print reservation feature is only available for single- and multi-child reservations and is not available for blocker reservations and multi-parent reservations.
