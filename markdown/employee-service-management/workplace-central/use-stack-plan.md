# Use Stack Plan

Use the stack plan to get a visual overview of the spaces in a building grouped by attributes like project or department. View key numbers like space count and capacity.

## Before you begin

To use the stack plan, verify that the Workplace Space Management installed.

Role required: sn\_wsd\_spcmgmt.space\_planner

## Procedure

1.  Navigate to **All** &gt; **Workplace Central** &gt; **Workplace Central**.

2.  On the side navigation, select the **Space Planning** module.

    The Space Planning workspace opens with the **Floor plan** tab selected by default.

3.  Select the **Stack plan** tab.

4.  In the **Search for campus** field, type the name of a campus, then select the campus from the type-ahead search results.

5.  From the **Select building** list, select a building.

6.  From the **View by** list, select a view by configuration.

    The spaces in the selected building are grouped based on the selected configuration. For example, if you select Department, the spaces are grouped based on their allocated department and the legend is updated.

7.  Use the stack plan based on your requirement.

    |------|-----|
    |**Zoom in or out on the stack plan**|Use the zoom options \[Omitted image "zoom-options.png"\] Alt text: on the stack plan.|
    |**View the KPIs and legend**|The details panel displays KPIs like the total space count and total capacity. The legend lists each group with its assigned color. The legend helps you identify what each color represents on the stack plan.|
