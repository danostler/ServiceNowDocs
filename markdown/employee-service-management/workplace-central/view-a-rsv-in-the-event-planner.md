# View reservation details in the event planner

Check details of a reservation created in your workplace.

## Before you begin

Ensure that you have installed the Calendar component \(sn\_calendar\) of version 23.2.0 or later.

Role required: sn\_wsd\_rsv.reservation\_planner or sn\_wsd\_rsv.reservation\_viewer

## Procedure

1.  Navigate to **All** &gt; **Workplace Central** &gt; **Workplace Central**.

    You can also open Workplace Central from the Employee Center directly. Navigate to **Workspaces** &gt; **Workplace Central**.

    The Workplace Analytics dashboard opens.

2.  On the left pane, select the **Event planner** icon \(\[Omitted image "event-planner-icon.png"\] Alt text: Event planner icon.\).

    The Event Planner dashboard opens.

3.  To view the current days reservation statues, do the following:

    1.  Go to the **Reservations** metrics section.

    2.  Select one of the following tiles to view as a list:

        -   **Planned reservations for today**
        -   **Reservations with extra services for today**
        -   **Canceled reservations of today**
        -   **Pending approval reservations**
4.  To view the active reservations, do one of the following:

    -   To view all the reservations, go to the **All Reservations** section.
    -   To view the reservation created by you or for you, go to the **My reservations** section.
    To view more reservations, select **View all**.

5.  Select the reservation that you want to view.

    The reservation opens in a new tab. The details of the reservation are displayed.

6.  Select any of the following tabs based on the details that you want to view:

    -   **Blocker Reservations**: To view the blocker reservations created before and after the reservations. This tab appears only if there are any extra services requested.
    -   **Cases**: To view the workplace cases created for the extra services requested in the reservation.
    -   **Tasks**: To view the workplace tasks created to fulfill the extra services requested in the reservation.
    -   **Requested Service**: To view the services requested with the reservation.
    -   **Approvers**: Approver of the reservation.
    -   **Invitees**: Invitees invited to the reservation.
    -   **Reservation Recordings**: Recordings of the reservation if a virtual meeting link was included.
7.  To open the reservation on the portal, select **Open reservation**.

    The reservation opens on the Employee Center portal.
