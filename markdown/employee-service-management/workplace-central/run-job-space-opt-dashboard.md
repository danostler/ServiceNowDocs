# Run the scheduled job for the Space Optimization dashboard

As an admin, run the scheduled job **Workplace Analytics Space Management report data generation** to calculate capacity and utilization for all spaces.

## Before you begin

Role required: admin

## About this task

This job should be run before viewing the dashboard for the first time. The job calculates capacity and utilization for all spaces and populates an internal table from which the dashboards are populated.

## Procedure

1.  Navigate to **All** &gt; **System Definition** &gt; **Scheduled Job**.

2.  Select **Workplace Analytics Space Management report data generation** from the list.

3.  Select **Execute Now** to run the job.
