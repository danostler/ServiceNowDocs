# Use Workplace Central

Perform workplace-related activities using the Workplace Central application.

To customize a page in the Workplace Central application, refer to the knowledge base article: [https://support.servicenow.com/kb?id=kb\_article\_view&amp;sysparm\_article=KB1639123](https://support.servicenow.com/kb?id=kb_article_view&sysparm_article=KB1639123).

-   **[Creating custom lists on Workplace Central](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-central/creating-custom-lists-wsd.md)**  
You can create custom lists using the Lists module on Workplace Central. You can create a custom list by using existing lists from Workplace Central, or by using a system table.
-   **[Working with Space Planning](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-central/working-with-space-planning.md)**  
Manage core functions like space hierarchy, neighborhoods, and user assignments.
-   **[Working with Space Optimization](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-central/working-with-space-optimization.md)**  
Perform space optimization-related actions such as creating scenarios, approving scenarios and more.
-   **[Working with Schedule Planning](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-central/working-with-schedule-planning.md)**  
Create and execute schedule plans to assign workspaces to your employees.
-   **[Working with Workplace Analytics](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-central/working-with-workplace-analytics.md)**  
As a workplace manager or administrator, use the Workplace Analytics dashboard to access key metrics for space optimization, maintenance management, lease administration, and Occupancy Insights to help you make effective decisions.
-   **[Working with Move management](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-central/working-with-move-management.md)**  
Create, monitor, and work on move projects created to schedule move requests. Work on the project using the move scheduling feature and monitor multiple move requests simultaneously.
-   **[Working with Event planner](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-central/working-with-event-planner.md)**  
Monitor all the reservations created in your organization from a single workspace. Create a reservation, open a reservation, schedule a reservation to another time or move to another location. Open the Location directory portal from the Event planner home and the Scheduled view.
-   **[Working with Case management](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-central/working-with-case-management.md)**  
Manage your workplace cases using the Case management workspace. Monitor the progress of the cases, update the cases and perform other actions.
-   **[Working with Kiosk Indoor Mapping](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-central/working-kiosk-ind-map.md)**  
Create indoor maps and configure map elements for Workplace Services Kiosk.
