# Edit a schedule plan

Edit a schedule plan to change the details, selected users, or selected spaces.

## Before you begin

You can only edit a schedule plan that is in the Draft state and assigned to you. If a schedule plan is not in the Draft state, you can only change the **Name**, **Assigned to**, and **Description** fields.

Role required: sn\_wsd\_spcmgmt.project\_owner

## Procedure

1.  Navigate to **All** &gt; **Workplace Central**, then select the **Schedule Planning** module.

    You can also open Workplace Central from the Employee Center by navigating to **Workspaces** &gt; **Workplace Central**.

2.  Select the **Schedule Planner** icon \(\[Omitted image "schedule-plan-icon.png"\] Alt text: Schedule Planner icon.\).

3.  From the Schedule Plans list, select the draft schedule plan that you want to edit.

4.  Change the Details, User selection, or Space selection based on your requirement.

    You can navigate to the required page by selecting **Back** or **Next**. Adding more users or spaces does not change the existing mappings.

    **Note:** Changing the Project, Schedule, Start date, or End date fields resets the User selection, Space selection, and Mapping pages.

5.  Navigate to the Mapping page, then select **Create Plan**.

    For more information about user-space mapping and validation, see [Create a schedule plan](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-central/create-schedule-plan.md).
