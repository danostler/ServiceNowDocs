# Run the scheduled job for the Maintenance Management dashboard

As an admin, run the scheduled job **Workplace Maintenance Case Historical Data Collection** to collect data for the past year that is used by the Maintenance Management dashboard.

## Before you begin

Role required: admin

## About this task

This job should be run before viewing the dashboard for the first time. It collects all the data required for the reports for the past year.

## Procedure

1.  Navigate to **All** &gt; **System Definition** &gt; **Scheduled Job**.

2.  Select **Workplace Maintenance Case Historical Data Collection** from the list.

3.  Select **Execute Now** to run the job.
