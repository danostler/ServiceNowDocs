# Install Career Conversations

Install the Career Conversations app to enable employees and managers to create, manage, and track conversations.

## Before you begin

Dependent plugins and apps for Career Conversations

-   [Manager Hub](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/hr-service-delivery/manager-hub-overview.md) 3.0.0
-   [Talent Development Core](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/talent-development-core/egd-landing-page.md)

-   See [Install the Talent Development Core application](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/talent-development-core/egd-core-install.md).


Role required: admin

## About this task

**Important:** You must install the Talent Development Core app before you install Career Conversations.

Career Conversations is dependent on features in Talent Development Core.

## Procedure

1.  Go to the ServiceNow Store and find the Talent Development Core application using the filter criteria and search bar.

    Under the **Request Install** button in the **Version** section, a list of available versions is displayed.

    **Note:** Select **Other App Versions** to view a list of all available versions of an application.

2.  Select a version from the list and select **Install**.

    In the Review Installation Details dialog box, any dependencies installed with your application are listed.

3.  If you're prompted, follow the links to the ServiceNow Store to get any additional entitlements for dependencies.

4.  If demo data is available and you want to install it, select the **Load demo data** check box.

    Demo data includes sample records that describe application features for common use cases. Load demo data when you first install the application on a development or test instance.

5.  Select **Install**.
