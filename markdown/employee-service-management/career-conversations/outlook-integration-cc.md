# Set up Microsoft Outlook integration in Career Conversations

Perform the following steps to set up an integration with Microsoft Outlook

-   **[Create a connection with Microsoft Exchange Online](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/career-conversations/set-up-ms-exhchange-cc.md)**  
Integrate the Career Conversations ServiceNow instance and Microsoft Exchange Online account by creating a custom OAuth application in Microsoft Exchange Online to authenticate ServiceNow requests.
-   **[Configure Microsoft Azure](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/career-conversations/ms-outlook-azure-cc.md)**  
Set up a authentication with Microsoft Azure to connect Microsoft Exchange Online.
-   **[Configure the webhook registry of Microsoft Exchange Online spoke in Career Conversations](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/career-conversations/webhook-registry-cc.md)**  
Configure callback URL for communication between the Career Conversations application and Microsoft Exchange Online calendar.
-   **[Set system properties for Microsoft Outlook integration](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/career-conversations/egd-admin-sys-properties-outlook.md)**  
Use system properties to enable integration between Career Conversations and Microsoft® Outlook® so you can simultaneously create and schedule conversations with the Career Conversations app.
