# Components installed with Career Conversations

Several types of components install with Career Conversations, including roles and tables.

**Note:** The Application Files table lists the components that are installed with this application. For instructions on how to access this table, see Find components installed with an application.

## Roles

|Application|Role title \[name\]|Description|
|-----------|-------------------|-----------|
|Career Conversations|Admin \[sn\_egd\_act.admin\]|Contains Talent Manager role \[sn\_egd\_act.manager\].|
|Career Conversations|Employee \[sn\_egd\_act.employee\]|Doesn't contain other roles.|
|Career Conversations|Manager \[sn\_egd\_act.manager\]|Contains Talent Employee role \[sn\_egd\_act.employee\].|

## Tables

Tables are used to track and maintain relationships among different ServiceNow applications and features.

|Application|Table label \[name\]|
|-----------|--------------------|
|Career Conversations|Action Item \[sn\_egd\_act\_action\_item\]|
|Career Conversations|Discussion \[sn\_egd\_act\_discussion\]|
|Career Conversations|Discussion Series \[sn\_egd\_act\_discussion\_series\]|
|Career Conversations|Suggested Talking Point \[sn\_egd\_act\_suggested\_talking\_point\]|
|Career Conversations|Talking Point \[sn\_egd\_act\_talking\_point\]|
