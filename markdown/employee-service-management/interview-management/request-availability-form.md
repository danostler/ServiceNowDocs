# Request scheduling preferences form

The request scheduling preferences form provides the field descriptions required to request an applicant or interviewer to share their scheduling preferences for upcoming interviews.

<table id="table_lwc_ndy_p2c"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Applicant

</td><td>

Applicant selected when requesting scheduling preferences.**Note:** This field is available only when you’re requesting an applicant's scheduling preferences.

</td></tr><tr><td>

Interviewers

</td><td>

Interviewers selected when requesting scheduling preferences. You can select the field to add other members from the hiring team to request their scheduling preferences.**Note:** This field is available only when you’re requesting an interviewer's scheduling preferences.

</td></tr><tr><td>

Duration

</td><td>

Duration for which the scheduling preferences is requested.

</td></tr><tr><td>

Select template

</td><td>

Predefined template to fill the **Subject** and **Message**.

</td></tr><tr><td>

Subject

</td><td>

Subject of the email notification to be sent to the applicant/interviewer requesting their availability.

</td></tr><tr><td>

Message

</td><td>

Body text of the email notification.

</td></tr></tbody>
