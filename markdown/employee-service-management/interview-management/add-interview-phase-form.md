# Create new interview phase form

The Create New Interview phase form provides the field descriptions required to define interview phases for a job requisition.

<table id="table_uhf_b2r_pfc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Job requisition

</td><td>

This is automatically set to the job requisition ID for which you're adding the interview phase.

</td></tr><tr><td>

Type

</td><td>

Type of the interview phase. The available options are:-   **Screening**
-   **Technical**
-   **Managerial**
-   **Executive**
-   **Cultural fit**

</td></tr><tr><td>

Title

</td><td>

Title of the interview phase you're setting up.

</td></tr><tr><td>

Order

</td><td>

Order in which this interview round should happen within the overall interview process.

</td></tr><tr><td>

Duration

</td><td>

Duration of the interview round.

</td></tr><tr><td>

Format

</td><td>

Format of the interview, whether it's a one-to-one or a panel round.

</td></tr><tr><td>

Channel

</td><td>

Mode of conducting the interview. The available options are **Zoom**, **Phone call**, and **In person**.

</td></tr><tr><td>

Interviewer roster

</td><td>

A primary list of interviewers to choose from, that can take the interview round.

</td></tr><tr><td>

Additional interviewer roster

</td><td>

A supplementary list of interviewers who can be added to the interview if the primary interviewers are unavailable or if additional perspective is needed.

</td></tr><tr><td>

Description

</td><td>

A short description of the interview phase.

</td></tr></tbody>
