# Add asset to workplace contract in Workplace Central

Add assets to the workplace contracts.

## Before you begin

Role required: sn\_wsd\_la.workplace\_central\_user, sn\_wsd\_la.manager or sn\_wsd\_la.admin

## About this task

Linking a contract with assets clarifies what the contract legally covers. The Assets Covered related list is shown only after you save or submit a workplace contract.

## Procedure

1.  Navigate to **Workspaces** &gt; **Workplace Central**.

2.  Select **Lease Administration**.

3.  Select a contract.

4.  Review the assets in the Assets Covered related list.

5.  If a desired asset covered by the contract is not available for selection, create it.

    1.  In the **Assets Covered** related list, select **New**.

    2.  Fill in the form related to the asset type.

    3.  Select **Save**.

6.  In the **Assets Covered** related list, select **Add** to add assets.

7.  Select the assets that is covered by the contract.

8.  Select **Add**.
