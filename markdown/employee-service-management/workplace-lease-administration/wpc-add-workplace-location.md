# Add workplace contract locations in Workplace Central

Add locations to the workplace contract for the locations the contract covers.

## Before you begin

Role required: sn\_wsd\_la.workplace\_central\_user, sn\_wsd\_la.manager or sn\_wsd\_la.admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Workplace Central**.

2.  Select **Lease Administration**.

3.  Select a contract.

4.  In the **Contract locations** related list, either select an existing location or add a new location.
