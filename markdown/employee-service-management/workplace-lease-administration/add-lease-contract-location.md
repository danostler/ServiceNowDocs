# Add a workplace contract location in Core UI

Add a location that is covered by the contract to a workplace contract.

## Before you begin

Role required: sn\_wsd\_la.manager or sn\_wsd\_la.admin

## Procedure

1.  Navigate to **All** &gt; **Workplace Lease Administration** &gt; **Workplace Contracts**.

2.  Select a contract.

3.  If the locations are not in the Contract locations related list, create them.

    1.  Select **New**.

    2.  Provide the location information in the Location form.

    3.  Select **Submit**.

4.  In the Contract locations related list, select the locations covered by the contract.

5.  Select **Save**.


## Result

The locations are added to the contract.
