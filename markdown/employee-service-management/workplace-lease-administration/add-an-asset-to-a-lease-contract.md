# Add assets to a workplace contract in Core UI

Add assets to a workplace contract, including software licenses, to specify what the contract legally covers.

## Before you begin

You must have an existing contract.

Linking a contract with assets clarifies what the contract legally covers. The Assets Covered related list is available only after you save or submit a workplace contract.

Role required: sn\_wsd\_la.manager or sn\_wsd\_la.admin

## Procedure

1.  Navigate to **All** &gt; **Workplace Lease Administration** &gt; **Workplace Contracts**.

2.  Select a contract.

3.  Review the assets in the Assets Covered related list.

4.  If a desired asset covered by the contract is not available for selection, create it.

    1.  Select **New**.

    2.  Select the type of asset to create.

    3.  Fill in the form related to the asset type.

    4.  Select **Submit**.

5.  In the Assets Covered related list, select **Edit**.

6.  Select the assets covered by the contract.

7.  Select **Save**.


## Result

The selected assets are added to the contract.
