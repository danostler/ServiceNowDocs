# Generate and manage AI‑suggested safety actions in Action planner

Generate AI-suggested safety actions in Action planner to quickly identify and create corrective measures for Health and Safety records.

## Before you begin

Verify that the Now Assist for Health and Safety application \(sn\_hs\_gen\_ai\) is installed.

**Important:** The **H&amp;S Action Planner** skill is turned on by default with the Now Assist for Health and Safety application. The skill will be automatically available to appropriate role users for the application. For more information, see .

Role required: sn\_ohs\_im.action\_writer

## About this task

The Action planner can generate AI-suggested actions for any Health and Safety record that has the **Actions** tab. The suggestions are produced by retrieving similar historical records and analyzing past actions taken in those cases, along with relevant knowledge base articles.

Review all AI-generated suggestions for accuracy before creating actions from them.

For more information on Action planner, see [Managing safety actions in the Action planner](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/health-and-safety-core/hs-action-planner.md).

## Procedure

1.  Navigate to **Workspaces** &gt; **Health and Safety Workspace**.

2.  Open a Health and Safety record, for example, a safety incident or risk assessment.

3.  Select the Action Planner icon \(\[Omitted image "health-safety-action-planner.png"\] Alt text: Action Planner icon\) from the contextual sidebar panel.

4.  Select **Suggest new actions** to generate AI-suggested actions for the record.

    The system searches for similar records, retrieves historical actions and relevant knowledge base articles, and then generates suggestions using a language model.

5.  Review the suggestion cards that appear in the **Suggested actions** section.

6.  Accept a suggestion and create an action from it.

    1.  On a suggested action card, select **Create action**.

        The inline **Edit action** form opens with pre-filled fields.

    2.  Review and update the fields as needed, for example, **Due date**.

    3.  Select **Create action**.

        The action is created, moves to the **Created actions** section, and is removed from **Suggested actions**.

7.  To dismiss a suggestion, select the X icon on the suggestion card.

    The card is removed from **Suggested actions**. The suggestion does not reappear for this record if you select **Suggest new actions** again.


## Result

Created actions:

-   The action appears as a card in the **Created actions** section of the action planner.
-   The action also appears in the list of actions in the **Actions** tab of the associated Health and Safety record.
-   The Action record form shows a message indicating that this action was suggested and created by AI.

Suggested actions:

-   All AI suggestions generated by the Action Planner are stored in the AI action suggestion \[sn\_ohs\_im\_ai\_action\_suggestion\] staging table.
-   Any suggestions that aren't dismissed or accepted for action creation remain in the **Pending** state. A scheduled job deletes these pending suggestions based on the configuration. For more information, see [Configure cleanup of pending AI action suggestions](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/now-assist-for-health-and-safety/hs-configure-deletion-pending-ai-action-suggestions.md).
