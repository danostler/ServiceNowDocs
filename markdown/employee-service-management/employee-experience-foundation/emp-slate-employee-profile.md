# Employee profile

The Employee profile enables you to search for colleagues, view profiles, and access contact information through an intuitive directory interface.

Employee profiles provide a centralized view of individual employee information, including basic details, organizational relationships, and contact information. Profiles are accessible through the org chart, search results, or direct navigation from the profile section.

\[Omitted image "es-emp-profile.png"\] Alt text: Profile of the employee

Each profile includes an org chart button that takes you directly to the position of that employee in the organizational hierarchy.

## Employee profile sections

Select the **Profile** option to find more information about the employee. The profile is organized into the following sections:

-   **About**

    Displays basic information such as job title, department, office location, and contact details. Employees viewing their own profile can edit the nickname, pronouns, and bio fields.

-   **Work details**

    Displays employment-specific information configured by the administrator in the overview UI configuration.

-   **Personal details**

    Displays non-sensitive personal information that the organization selects to surface on the profile.

-   **Team**

    Lists the manager, coworkers, and direct reports for the employee. Selecting a colleague in this section opens the profile for that person.


Each profile page includes an **Org chart** call-to-action that opens the org chart with the current profile employee selected. From the org chart, employees navigate up, down, or sideways through the reporting structure.

## Role-aware permissions

The profile applies role-aware permissions to control what each employee views and edits:

-   Personalize your profile and provide additional context with information such as Nickname, Pronouns, and Bio. You can edit this info based on the configurations that the administrator sets.
-   Employees have view-only access to profiles for colleagues profiles.

## AI-powered profile discovery

AI chat enhances profile discovery by responding to natural language questions about employees. You can ask questions like `Who is [employee name]?` and receive profile information with direct links to detailed employee pages.

## Profile configuration

Administrators can configure which fields appear in employee profiles through the Employee Center settings. This includes controlling the visibility of work details, personal information, and team relationships based on organizational requirements.
