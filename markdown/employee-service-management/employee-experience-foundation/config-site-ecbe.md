# Configure browser extension sites

Configure the Browser Extension for Employee Center in your desired websites for easy access to the Employee Center portal.

## Before you begin

Role required: sn\_ex\_brw\_ext.ecbe\_admin / sn\_ex\_brw\_ext.ecbe\_manager

## About this task

The Browser Extension for Employee Center lets you access your extension-supported Employee Center portal content via the browser extension.

**Note:**

-   If you try to access the browser extension from any random website, without configuring the extension for that website, you see an empty state message.
-   If you try to access content when none is configured for the Browser Extension for Employee Center, you see an empty state message.
-   When you enable Browser Extension for Employee Center it has a default HTTP response header, which is activated. For optimal user experience, you should have only one HTTP response header activated, at a time, on your portal.

## Procedure

1.  Navigate to **Employee Center** &gt; **Browser Extension** &gt; **Browser Extension Sites**.

2.  Select **New** to create a site that can access the Browser Extension for Employee Center.

3.  In the Configured Website form, fill in the fields, and select **Submit**.

    For a description of the field values, see [Website configuration form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/employee-experience-foundation/config-website-form.md).


## Result

You have set up the Browser Extension for Employee Center in your desired website.
