# Manage your calendar with the Calendar Management assistant

Use the Calendar Management specialized assistant for scheduling tasks. The assistant blocks time, reschedules meetings, checks attendee availability, and adds conferencing details from one scoped AI assistant.

## Before you begin

Before you start, verify the following:

-   The Calendar Management specialized assistant is available to you in the left panel.
-   Your calendar system is connected to the specialized assistant.

Role required: Employee

## About this task

The Calendar Management specialized assistant opens with suggested prompts, preloaded context, and connected tools. You start the scheduling task without describing the domain.

## Procedure

1.  In the left panel, select the **Calendar Management** specialized assistant.

2.  Review the suggested prompts or enter a natural-language request.

    For example: `Show my afternoon schedule and block 3:00 PM for compliance training`.

3.  Review the schedule that the assistant returns.

    The assistant presents the schedule in a scannable format with direct links to each event.

4.  Ask the assistant to modify a specific meeting by describing the changes.

    For example: `change the meeting time, add an attendee, or add conferencing details.`.

5.  Select a time slot from the options that the assistant displays.

    The assistant checks availability for every attendee and displays available time slots.

6.  Verify the new meeting details.

    The assistant displays the outcome in the conversation after the assistant commits the changes to the calendar.


## Result

The calendar updates with the new time block and rescheduled meeting. All attendees see the updated time, attendee list, and conferencing details in the calendar system.
