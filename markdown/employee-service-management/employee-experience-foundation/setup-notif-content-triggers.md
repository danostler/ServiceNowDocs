# Configure notification triggers

Configure the notification content triggers to determine how it’s displayed on the portal.

## Before you begin

Role required: notification\_provider\_admin

## Procedure

1.  Navigate to **Employee Center** &gt; **Portal Notifications** &gt; **Portal Notifications**.

2.  In the Notifications list, select the notification where you want to set the trigger.

3.  On the Notifications trigger conditions form, fill in the fields, and select **Submit**.

    For a description of the field values, see [Trigger conditions form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/employee-experience-foundation/notif-trigger-form.md).
