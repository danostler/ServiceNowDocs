# Exploring ServiceNow for Microsoft 365

The ServiceNow® for Microsoft Teams or Microsoft 365 applications extend the integration of the application to different business units. You can enable employees to more effectively request and receive service from within Microsoft Teams by using this integration.

ServiceNow for Microsoft Teams is available with IT Service Management, HR Service Delivery, Universal Request. It is also supported for various Agent experience platforms, like Meeting Extensions.

ServiceNow for Microsoft 365 is available with IT Service Management and HR Service Delivery.

**Important:**

-   From [IT Service Management for Microsoft 365](https://store.servicenow.com/sn_appstore_store.do#!/store/application/14eb9da8c3f310102986a81c8640dd08/2.6.7?referer=%2Fstore%2Fproduct%2F9389b69edbb8e810d27e581adc9619ca) \(version 2.6.7\), ServiceNow for Microsoft Teams is upgraded to ServiceNow for Microsoft 365.
-   From [HR Service Delivery for Microsoft 365](https://store.servicenow.com/sn_appstore_store.do#!/store/application/23364660c3b31010aab55b79c840ddc2/3.3.6?referer=%2Fstore%2Fproduct%2F9389b69edbb8e810d27e581adc9619ca) \(version 3.3.6\), ServiceNow for Microsoft Teams is upgraded to ServiceNow for Microsoft 365.

For upgrading existing Microsoft Teams capabilities to Microsoft 365 applications, see [Integrating ServiceNow with Microsoft 365 applications for Employee Experience](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/employee-experience-foundation/setup-sn-ms-teams-ms365.md).

## ServiceNow for Microsoft 365 integrations

ServiceNow for Microsoft 365 integration features are as follows:

-   Allows agents and employees to collaborate to resolve a ticket.
-   Enables agents to initiate a Microsoft Teams chat with an employee from a task record.
-   Enables agents to import the Microsoft Teams chat transcript back to the task record as a comment.
-   Enables employees with specific permissions to view Microsoft Teams chat from a task record.

-   **[Explore Microsoft Teams and Microsoft 365 integration for Employee Experience](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/employee-experience-foundation/c_employee_experience.md)**  
Offers help for employees to resolve their IT, HR, and UR related issues through Microsoft Teams or Microsoft 365. Employees can interact with the agents from within Microsoft Teams or Microsoft 365 to resolve their issues.
-   **[Explore Microsoft Teams integration for Agent Experience](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/employee-experience-foundation/c_agent_experience.md)**  
Allow the agents to work and manage the major incidents from Major Incident Management \(MIM\) workbench through Microsoft Teams.
