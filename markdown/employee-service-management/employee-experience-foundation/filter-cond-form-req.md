# Request filter condition form

The field descriptions provide a reference point when you're creating a Requests filter definition.

|Field|Description|
|-----|-----------|
|Table|Record table from where the filter condition options are sourced.|
|Filter config|Filter configuration under which the filter condition is being created.|
|Application|Application scope where the filter condition record resides.|
|Active|Option to enable the filter condition.|
|Filter|**Add filter condition**: Add a filter condition.|
|**Add OR Clause**: Add an OR clause to the filter condition.|
|**Add Sort**: Add a sorting clause to the filter condition.|
