# Action framework

The action framework includes some frequently used action items for quick access. You can use several default actions and create custom actions.

Using the action group framework, you can define these actions that a user can perform in a context.

-   Create actions and action groups for ready use as a widget.
-   Configure a group of actions that a user can perform.
-   Use the default actions and action groups or customize the actions to suit your business needs.
-   Manage the display preferences and user criteria.
-   Improve employee productivity with access to quick actions.

Here are one-line descriptions about the components in action framework.

-   **Action**: A single, defined step or operation within a workflow or process. For example, approve or reject.
-   **Action group**: A set of actions for organizing related actions within a flow or subflow. For example, approvals action group has a default **set of actions** Approve and Reject that you can use from the approval page.
-   **Action type**: A classification or category that specifies the kind of action to be performed. Type is used for easier identification and use.
-   **Action framework**: A structured approach or set of guidelines for organizing and executing actions to achieve specific business goals.

Using actions that are relevant to the user journey, you can complete workflows quickly. For example, you can use the default approve or reject actions to process the requests directly from the approvals page.

## Employee Profile action group

Employee Profile action group provides quick access to frequently used action items, such as, viewing the org chart, using email directly or calling. You can edit the default behaviors of the actions to match your requirements.

## Approvals experience action group

Approvals experience action group provides quick access to approve or reject the requests directly from the approvals page. You can modify the actions to suit your business requirements.

## Manager Hub action group

Manager Hub action group provides quick access to the action items available on the employee cards to view, call, or access email directly on the **Your team** page of the Manager Hub. You can customize the actions to suit your business requirements.
