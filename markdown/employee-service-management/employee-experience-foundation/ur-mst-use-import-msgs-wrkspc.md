# Import messages from Agent Workspace

As an agent, import a conversation with a user from Microsoft Teams. Append the conversation in the activity stream of the record in the Agent Workspace. Store all important messages related to a universal request in the ServiceNow instance.

## Before you begin

Role required: agent

## Procedure

1.  Navigate to **All** &gt; **Workspace Experience** &gt; **Workspaces** &gt; **Agent Workspace Home**.

2.  From the navigation list, select **Universal Request** &gt; **All**.

3.  Open the universal request you want to work on.

4.  Select **Import Messages**.

    You receive a prompt to log in to Microsoft Teams if you are not logged in.

5.  Click **Accept**.

6.  On the form, fill in the fields.

<table id="table_tjg_yp5_knb"><thead><tr><th align="left">

Field

</th><th align="left">

Description

</th></tr></thead><tbody><tr><td>

Show conversations

</td><td>

Option to show conversations. Choices are as follows:-   **Started from this record**: Lists all the communication tasks created for the record.
-   **List all conversations**: Lists all conversations by the logged-in agent.


</td></tr><tr><td>

Select a conversation

</td><td>

Conversation whose messages are imported for the record.

</td></tr><tr><td>

Import messages as

</td><td>

-   **Work notes**: The messages will be appended under Work notes.


</td></tr></tbody>

8.  Click **Import messages**.
