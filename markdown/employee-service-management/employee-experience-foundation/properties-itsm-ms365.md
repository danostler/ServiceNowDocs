# Properties installed with IT Service Management for Microsoft 365

The following properties are installed with the IT Service Management for Microsoft 365 plugin.

**Note:** The following properties are located in the System Properties \[sys\_properties\] table. To access the table, enter `sys_properties.list` in the navigation filter.

<table id="table_rg4_d2w_y1c"><thead><tr><th>

Property

</th><th>

Description

</th></tr></thead><tbody><tr><td>

sn\_now\_teams\_it.enable\_va\_comment\_notification

</td><td>

Use this property to enable the comment notifications sent by the Virtual Agent sys\_notification.-   Type: true\|false
-   Default value: true

</td></tr><tr><td>

sn\_now\_teams.enable\_va\_approval\_notification

</td><td>

Use this property to enable the approval notifications sent by the Virtual Agent sys\_notification.-   Type: true\|false
-   Default value: true

</td></tr></tbody>
