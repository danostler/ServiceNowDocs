# Configure static content

Static content helps create a central place for adding all the useful content for employees.

## Before you begin

Role required: Admin

## Procedure

1.  Navigate to **All** &gt; **Employee Center** &gt; **Static Content**.

2.  Click **New** or open an existing record.

3.  On the form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Title|Name of the record.|
    |Active|Check the box to activate.|
    |Rich text|Descriptive information for the static content.|
    |Available for|Option to make the static content available for only to select users, groups, roles, and more.|
    |Not available for|Option to make the static content unavailable for.|

4.  Click **Save**.


## What to do next

You can add the useful static content in the [Get support](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/employee-experience-foundation/apn-config-getsupport.md) section.
