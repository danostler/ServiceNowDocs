# Resolve a feedback task

Resolve a task created from a feedback submitted through the integrated experience and service feedback.

## Before you begin

Role required: esc\_admin

## Procedure

1.  Navigate to **Self-Service** &gt; **Employee Center** &gt; **My Tasks**.

2.  In the My tasks page, select **Feedback tasks** from the Filter by: options.

    You can select one or more of the following **Feedback task** types:

    -   **Experience**
    -   **In-Page**
    -   **Topic Page**
    -   **Workflow**
    **Note:** The **Task type** filter in My tasks can be enabled from the **To-dos filter categories**.

3.  In the **Open** tab, select the feedback task you want to resolve.

    The task details show up on the page including the **Feedback provided by**, **Scaled value**, and **Feedback definition** type.

4.  Select **Skip** or **Resolve and complete** to close an open feedback task.

    |Selection|Action|
    |---------|------|
    |**Skip**|The feedback task is skipped.|
    |**Resolve and complete**|The feedback resolve and complete window open to suggest steps to complete the task.|

5.  On the Resolve and complete window, fill in the fields.

    **Note:** A feedback notification is sent to the feedback provider when it's resolved.
