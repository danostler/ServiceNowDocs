# Work with org chart and profiles

Interact with employee profiles and organizational charts using conversational capabilities for efficient information discovery.

## Before you begin

Role required: none

## About this task

You can find employee information and navigate organizational structures through chat interactions.

## Procedure

1.  In the left navigation, select **Org Chart**.

    The org chart page displays your position in the organizational hierarchy with one level above and below.

2.  View additional organizational levels, select **Show More**.

    Levels associated to your position appear in the chart.

3.  Search and find a specific employee, enter their name in the search field.

    **Note:** The search function helps you locate employees across the organization without manually navigating through hierarchy levels.

4.  Try the following chat interactions to learn more about organization and employees.

    |Input|Response|
    |-----|--------|
    |`Who is [employee name]?`|Chat displays a widget with the employee basic information and a link to the full profile.|
    |`Show org of [employee name]`|Chat responds with information about the employee team size and organizational details.|
    |`Show [employee name]`|The org chart automatically updates to display the requested employee organizational context without requiring manual navigation. This applies only when already viewing the org chart.|
    |`Who reports to [manager name]?`|View relevant team information and navigation options.|
    |`Who is [employee name] manager?`|View relevant team information and navigation options.|

5.  Use the chart hierarchy and chat interactions to find employee organizational and profile details.


## Result

You can efficiently discover employee information and navigate organizational structures using natural language interactions.
