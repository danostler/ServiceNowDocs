# Install Employee Center for Microsoft Viva Connections application

Install the Employee Center for Microsoft Viva Connections application to enable users to easily access their task cards from Microsoft Teams.

## Before you begin

Role required: external\_app\_install\_admin

## Procedure

1.  Navigate to **System Application** &gt; **All Available Applications** &gt; **All**.

2.  In the search field, enter `Employee Center for Microsoft Viva Connections`.

    You can search for the application by its name or ID.

3.  Select **Install**.


## Result

A confirmation message appears on successful installation.
