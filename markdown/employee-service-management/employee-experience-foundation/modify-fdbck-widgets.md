# Modify the experience flow feedback widget

Customize the appearance of the experience flow feedback widget with the instance options.

## Before you begin

Role required: sn\_hr\_sp.esc\_admin

## Procedure

1.  Navigate to **All** &gt; **Employee Center**.

2.  From the Mega Menu, select a topic page.

3.  Press the control button and simultaneously select and hold \(or right-click\) to get the menu options.

4.  Select **Instance Options** and modify the following options.

## Result

The In page feedback widget is customized according to your preference.
