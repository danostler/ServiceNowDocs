# User experience dashboard

The user experience dashboard helps get a consolidated view of the feedback data. It provides a quick view of all types of feedback data in the User Experience Analytics dashboard.

The user experience dashboard shows feedback data gathered from different users at a glance. It provides data from the experience, service, and survey feedback.

If you're already familiar with the User Experience Analytics dashboard, track all your feedback data from different portals here.

The following are the different components of the experience and service feedback visible in the user experience dashboard.

## Experience feedback

See the following analytics insight for the experience feedback data.

-   **Page Id**: Page ID where the feedback is provided.
-   **Rating**: Rating grouped to understand the percentages of positive or negative feedback.
-   **Users**: User ID of the feedback provider.

    If the feedback provider chose to be anonymous, the user ID is displayed as Guest.

-   **Comment**: Percentage projection of the user sentiments collected through the ratings.

    **Note:** The comment insight is only available for the Employee Center portal not for the Employee Center Pro Kiosk and the Mobile Employee Service Portal.


## Experience feedback 5-point scale

See the following analytics insight for the experience feedback 5-point scale.

-   **Comment**: Percentage projection of the user sentiments collected through the ratings.
-   **Page Id**: Page ID where the feedback is provided.
-   **Rating**: Rating grouped to understand the percentages of positive or negative feedback.
-   **Users**: User ID of the feedback provider.

    If the feedback provider chose to be anonymous, the user ID is displayed as Guest.


## Service feedback

See the following analytics for the service feedback data.

-   **Page Id**: Page ID where the feedback is provided.
-   **Record Id**: A workflow tracking system where it shows whether the feedback is from a portal or an email.
-   **Rating**: Rating grouped to understand the percentages of positive or negative feedback.
-   **Users**: User ID of the feedback provider.

    If the feedback provider chose to be anonymous, the user ID is displayed as Guest.

-   **Topic Name**: Topic page where the feedback is provided.
-   **Comment**: Percentage projection of the user sentiments collected through the ratings.

## Survey feedback

See the following analytics for the survey feedback data.

-   **Page Id**: Page ID where the feedback is provided.
-   **Survey Name**: Name of the survey.
-   **User Id**: User ID of the feedback provider.

    If the feedback provider chose to be anonymous, the user ID is displayed as Guest.
