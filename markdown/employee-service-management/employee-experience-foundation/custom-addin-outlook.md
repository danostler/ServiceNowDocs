# Get a custom add-in for Microsoft Outlook

Get the custom add-in for your Microsoft Outlook client to use the features configured by your admin.

## Before you begin

The add-in is available only if the admin has deployed the add-in in the Microsoft admin center.

**Important:** ServiceNow Add-in for Microsoft 365 is only supported for Microsoft Outlook 2019 or later versions.

Role required: None

## Procedure

1.  Log in to the Microsoft Outlook client.

2.  From the Tools menu, select **Get Add-ins**.

3.  In the Office Add-ins window, select **Admin-managed**.

4.  Search for the add-in deployed by your admin, then select **Add**.

    For example, search for ServiceNow Add-in for Microsoft 365, then select **Add**.


## Result

The custom add-in is added to your Microsoft Outlook client. You can use it to request catalog items or open incidents and cases based on your emails.
