# Configure search source when AI search is disabled

In Employee Center, the AI search is enabled by default. In situations where the AI search is disabled, you can configure a search source from which to fetch search results.

## Before you begin

Role required: admin

## About this task

The search source configuration limits the search to the content associated with the taxonomy. Employee Center uses the Zing search engine if the AI search is disabled.

## Procedure

1.  Navigate to **All** &gt; **Service Portal** &gt; **Portals**.

2.  Click **Employee Center**.

3.  From the **Search Sources** related list, click **New** to add a search source.

4.  On the form, fill in the fields.

    For more information, see Define a search source.

5.  Click **Submit**.
