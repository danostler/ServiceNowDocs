# Catalog Mapping Configurations form

The Catalog Mapping Configurations form helps you customize mapping of emails and catalog items.

<table id="table_lsq_1yw_pcc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Catalog item

</td><td>

Catalog item that you want to map the email fields to.

</td></tr><tr><td>

Application

</td><td>

Application scope of the mapping configuration.This field is automatically set to the application scope of your instance.

</td></tr><tr><td>

Domain

</td><td>

Domain that you want to create the mapping configuration in.

</td></tr><tr><td>

UI script parser

</td><td>

UI script that contains custom logic for the mapping configuration.

</td></tr><tr><td>

Active

</td><td>

Option to activate the mapping configuration record.
