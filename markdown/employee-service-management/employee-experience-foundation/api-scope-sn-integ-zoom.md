# API permissions/scope for ServiceNow integration with Zoom

The list of the API permissions/scope requested by the ServiceNow Collaboration services for Zoom are as follows.

|Scope type|API scope|Description|
|----------|---------|-----------|
|User|user:master|View and manage a sub account’s user information.|
|user:read:admin|View all user information.|
|user:write:admin|View user information and manage users.|
|Chat|chat\_channel:read:admin|View all users’ chat channel.|
|chat\_channel:write:admin|View and manage all users’ chat channels.|
|chat\_message:read:admin|View all users’ chat messages.|
|chat\_message:write:admin|View and manage all users’ chat messages.|
|imchat:read:admin|View all users’ history and channels.|
|imchat:bot|Enable chatbot within Zoom Chat Client.|
