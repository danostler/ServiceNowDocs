# Customize Import Messages from Microsoft Teams UI action

Enables you to extend the **Import Messages from Microsoft Teams** UI action to the tables you require.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **ServiceNow for Microsoft 365** &gt; **UI Actions**.

2.  Open the existing **Import Messages from Microsoft Teams** UI action in a new window.

3.  On the **UI Actions** screen, Click **New**.

4.  On the **UI Action** form, fill the fields.
