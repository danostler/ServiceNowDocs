# Raise a request using the ServiceNow Add-in for Microsoft 365

Open a relevant catalog item or form to raise a request with pre-filled information from your email.

## Before you begin

The following procedure is for raising a request from Microsoft Outlook. You can use the add-in to raise a request from any Microsoft 365 app that is selected by the admin.

Role required: None

## Procedure

1.  Launch the Microsoft Outlook application.

2.  Select the email that you want to use to raise a request.

3.  Select the add-in, then select the button or menu item for your request.

    For example, select the ServiceNow Add-in for Microsoft 365, then select **Create incident**.

    You might have to login to the Employee Center with your ServiceNow credentials.

    The Employee Center portal opens in the activity tab in Microsoft Outlook.

4.  Fill in the form or the catalog item request, then select **Submit**.
