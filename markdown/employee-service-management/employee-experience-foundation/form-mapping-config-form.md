# Form Mapping Configurations form

The Form Mapping Configurations form helps you customize mapping of emails and forms.

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Table

</td><td>

Table that you want to map the email fields to.

</td></tr><tr><td>

Application

</td><td>

Application scope of the mapping configuration.This field is automatically set to the application scope of your instance.

</td></tr><tr><td>

Domain

</td><td>

Domain that you want to create the mapping configuration in.

</td></tr><tr><td>

Form View

</td><td>

Form view that you want to display when the user opens the form in the add-in.

</td></tr><tr><td>

UI script parser

</td><td>

UI script that contains custom logic for the mapping configuration.

</td></tr><tr><td>

Active

</td><td>

Option to activate the mapping configuration record.
