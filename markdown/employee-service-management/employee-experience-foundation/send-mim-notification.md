# Send a notification from Major incident management workbench

Send a Microsoft Teams notification to the stakeholders associated with a major incident.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Incident** &gt; **Major Incidents** &gt; **Open**.

2.  Open the relevant major incident.

3.  Select **View Workbench**.

4.  Select the **Communicate** tab.

5.  In the Communication Tasks form section, select **Add**.

    The communication task appears.

    \[Omitted image "compose-notification-mim-workbench.png"\] Alt text: Compose button for communication task.

6.  Select **Compose**.

7.  In the Compose modal screen, enter the details for the notification.

    1.  In the **To** field, enter the recipients associated with the communication task.

    2.  In the **Message** field, enter the message for the notification.

    3.  Select **Send**.

        \[Omitted image "compose-modal-mim.png"\] Alt text: Compose modal


## Result

The notification is sent to the recipients in Microsoft Teams. Recipients can select **View ticket details** to view the ticket in the ServiceNow instance.

\[Omitted image "notification-in-teams.png"\] Alt text: Notification in Microsoft Teams.
