# Create record with Azure GCC details

Create and update a new record in the sys\_properties.list table with the details of the Azure account of GCC customers.

## Before you begin

Role required: admin

## Procedure

1.  In the navigation filter, enter `sys_properties.list`.

    The System Properties \[sys\_properties\] table appears.

2.  Select **New**.

3.  In the **Name** field, enter *sn\_now\_teams.teams\_url*.

4.  In the **Value** field, enter the URL `https://gov.teams.microsoft.us`.

    **Note:** For DoD customers, enter the URL `https://dod.teams.microsoft.us`.

5.  Select **Update**.
