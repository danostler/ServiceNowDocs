# Upload manifest file in Microsoft Teams for Microsoft 365

Upload the manifest file in Microsoft Teams to integrate Microsoft 365 with your ServiceNow instance.

## Before you begin

Role required: admin

## Procedure

1.  Launch Microsoft Teams application.

2.  Navigate to **Apps** &gt; **Manage your Apps** &gt; **Upload an app**.

3.  Select the **Upload a custom app** option.

    **Note:**

    Ensure that ServiceNow for Microsoft 365 is allowed as a third-party app in Microsoft Teams. For information about allowing apps in Microsoft Teams, see the [Microsoft Teams documentation](https://learn.microsoft.com/en-us/microsoftteams/app-policies).

4.  Navigate to the location where the manifest file is downloaded and select the zip file.

5.  Select **Open**.

    The ServiceNow for Microsoft 365 app appears in the Apps screen.

    \[Omitted image "sn-ms-teams-ms365.png"\] Alt text: ServiceNow for Microsoft Teams app.

6.  Select the ServiceNow for Microsoft 365 app.

7.  Select **Add**.
