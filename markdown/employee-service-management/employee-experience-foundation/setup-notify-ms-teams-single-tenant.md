# Integrating Notify connector self-configured app with Microsoft Teams

Set up the Notify connector for Microsoft Teams using the self-configured app.

You might need to use the self-configured app setup due to your company or regulatory requirements.

-   **[Create an app in Microsoft Teams to enable making calls](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/employee-experience-foundation/create-app-ms-teams.md)**  
Create an app to make outbound calls from Microsoft Teams.
-   **[Manage permissions and authenticate the Notify connector app/bot in Microsoft Azure portal](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/employee-experience-foundation/manage-permissions-notify.md)**  
Assign permissions to users to be able to authenticate successfully and participate in conference calls in Microsoft Teams.
-   **[Create a Service user to make calls from Microsoft Teams](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/employee-experience-foundation/provide-app-access-policy-nc-st.md)**  
Create a service user role to be able to start online meetings on behalf of users in Microsoft Teams.
-   **[Create a Microsoft Teams application registry entry to connect the created app to ServiceNow instance](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/employee-experience-foundation/create-app-registry-entry.md)**  
Register your Microsoft Teams application with your ServiceNow instance for OAuth authorization.
-   **[Create a Connection &amp; Credentials alias for Microsoft Teams](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/employee-experience-foundation/create-connection-credential-aliases.md)**  
Create an alias to associate the connection information with the credential data required to complete the application integration in Microsoft Teams.
-   **[Update the records in the Microsoft Teams Configuration page](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/employee-experience-foundation/map-app-reg-cred-alias-ms-teams-config.md)**  
Update the created records in Microsoft Teams to make seamless calls.
-   **[Verify OIDC Application Registry creation for a self-configured app setup](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/employee-experience-foundation/verify-oidc-app-reg-creation-st.md)**  
Verify whether the OIDC Application Registry was created when Microsoft Teams Application Registry was created and mapped to the configuration during a self-configured app setup process.
