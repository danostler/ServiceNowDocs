# Simplified Navigation

Simplified navigation helps you design and configure an intuitive navigation experience and enrich information architecture and topic discovery experience.

Using the simplified navigation, you can design the primary navigation menu and configure the navigation setup based on employee needs and content priorities.

-   Provides a choice to configure left-side navigation
-   Simplifies navigation with a configurable flat menu.
-   Takes employees to destination with an easy hover or click.
-   Adapts across devices by transforming to bottom navigation for a consistent experience.

\[Omitted image "simplified-navigation-ec-overview.png"\] Alt text: Simplified navigation on left and at bottom

You can configure the simplified navigation in the following steps:

-   Streamline navigation setup from [Advanced Portal Navigation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/employee-experience-foundation/adv-portal-navigation-overview.md) by creating a record.
-   Select side navigation style and a set of predefined icons for consistency and usability.

**Note:** Migrating your existing navigation structure to simplified left navigation is easy.

## Simplified navigation: Useful tips and information

Use some of the following guidance and tips.

-   When the screen size is below 760 pixels, navigation switches to a mobile-friendly bottom view. Also, the header changes to a context menu view. The items that don't fit are accessible from the **More** menu.

    **Note:** The **More** menu can appear on both side navigation and bottom navigation.

-   Add topics up to two levels from the root-level. For example, when you add topics under Human Resources, you can only traverse from `Human Resources > Subtopic 1`.
-   In the left navigation, only the first level of child menu items appears. For deeper levels, hover or click back and forth for additional menus.

    \[Omitted image "simplified-left-nav-backforth.png"\] Alt text: How submenu navigation appears from the More menu

-   When a sub-menu contains many items, a scroll bar appears for navigation.
-   Left and bottom views are responsively adjusted as per available space.
-   For the root item without any child items or any active associated quick link, only one-level is visible on click.
-   When left navigation option is selected, the sync is supported up to one-level only.

    **Note:** For top navigation option, sync supports up to three levels of navigation.


For configuration steps, see [Configure simplified left navigation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/employee-experience-foundation/config-simplified-nav-as-menu.md).
