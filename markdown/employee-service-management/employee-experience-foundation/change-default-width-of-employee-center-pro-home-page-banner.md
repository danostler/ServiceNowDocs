# Modify banner width options

Modify the width of the Employee Center Pro banner widget, on the home page, to suit your preference.

## Before you begin

Role required: admin

## About this task

Configure the width of the Employee Center Pro banner widget, in one of the following sizes:

-   Fixed width view
-   Fluid width view

**Note:** **Custom HTML** options are not supported OOB on content experience widgets for **Banner content type**. The banner width configurable options are only available if you have Employee Center Pro version 25.0.3. The older versions only have the Fluid option available.

## Procedure

1.  Navigate to **All** &gt; **Service Portal** &gt; **Pages** &gt; **ec\_pro\_dashbaord**.

2.  Scroll down to the **Page Content** and select the **Employee Center Pro Homepage - Container 1**.

3.  Make the page editable.

4.  To keep the banner compressed, change the **Width** to **Fixed** and select **Update**.

    \[Omitted image "fixed-view.jpg"\] Alt text: The image shows the compressed home page banner provided with fixed view.

5.  To expand the banner to fit the screen, change the **Width** to **Fluid** and select **Update**.

    \[Omitted image "fluid-view.jpg"\] Alt text: The image shows a fit-to-screen banner provided with fluid width view.


## Result

Set the banner widget on the Employee Center Pro home page to your preferred width.
