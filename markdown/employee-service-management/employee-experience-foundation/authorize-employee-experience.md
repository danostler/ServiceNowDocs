# Authorize Employee Center for Microsoft Viva Connections

Authorize the Employee Center for Microsoft Viva Connections application to enable the display of the ServiceNow card in the Microsoft Viva dashboard.

## Before you begin

Role required:

-   ServiceNow instance: external\_app\_install\_admin
-   Microsoft Azure portal: Administrator role

The administrator role in Microsoft Azure can be as follows:

-   [Application Administrator](https://docs.microsoft.com/en-us/azure/active-directory/roles/permissions-reference#application-administrator)
-   [Cloud Application Administrator](https://docs.microsoft.com/en-us/azure/active-directory/roles/permissions-reference#cloud-application-administrator)
-   [Global Administrator](https://docs.microsoft.com/en-us/azure/active-directory/roles/permissions-reference#global-administrator)
-   [Privileged Role Administrator](https://docs.microsoft.com/en-us/azure/active-directory/roles/permissions-reference#privileged-role-administrator)

## Procedure

1.  Navigate to **All** &gt; **ServiceNow for Microsoft 365** &gt; **Install Azure apps**.

2.  Select the **Authorize** button in the **Employee Center for Viva Connections** entry.

3.  Select **Accept** to provide admin consent.


## Result

On successful authorization, the app is installed in the instance.
