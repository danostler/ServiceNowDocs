# Employee Center admin configurations

As the system admin, install Employee Center plugins, assign roles to users so they can configure Employee Center features, set the theme, header, and footer, and enable support for right-to-left languages.

The Employee Center application provides out-of-the-box portal pages with relevant widgets necessary to deliver an employee-centric portal experience.

To get started, follow these key steps to configure the portal:

1.  [Install plugins](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/employee-experience-foundation/install-plugins.md)
2.  [Assign roles](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/employee-experience-foundation/emp-center-personas.md)
3.  Configure the portal:
    -   [Customize Employee Center portal theme](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/employee-experience-foundation/customize-theme.md)
    -   [Employee Center global header](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/employee-experience-foundation/config-global-header.md)
    -   [Footer configuration](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/employee-experience-foundation/emp-center-footer-config.md)
    -   [Enable support for right-to-left languages](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/employee-experience-foundation/ec-direction-right-to-left-languages.md)
    -   Modify the portal pages, such as the home page:

        Navigate to **Service Portal** &gt; **Pages** to view all the pages.

        **Note:** The default Employee Center home page is `ec_home`.

    -   Modify the email notification subject and body, such as when a new task is assigned: [Modify email notification layout](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/employee-experience-foundation/activate-notification-template.md)
