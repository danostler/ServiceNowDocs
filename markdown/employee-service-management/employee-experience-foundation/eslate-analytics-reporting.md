# Analytics and reporting

Track user engagement, content performance, and AI assistant quality across the employee experience with built-in analytics tools.

## User Experience Analytics

Employee Slate includes User Experience Analytics \(UXA\) with no additional setup required. UXA captures behavioral data across every user session and surfaces it in a dedicated dashboard. Administrators and experience owners use the dashboard to understand employee engagement patterns across the portal.

UXA automatically captures the following interactions:

-   Every user session from start to end.
-   Page visits, including the home page and Tasks and requests.
-   Widget renders for example the Employee sidebar and org chart card.
-   Selections on specific widgets and page elements, beyond standard page views.

The UXA Overview Dashboard visualizes how employees navigate Employee Slate. Administrators use the dashboard to identify common navigation paths and exit points in the experience.

## UXA metrics

The UXA dashboard captures and displays the following metrics:

-   **Total active users**

    The number of distinct users who completed at least one session in the selected time period.

-   **New users**

    The number of users who accessed Employee Slate for the first time in the selected time period.

-   **Total sessions**

    The total number of sessions across all users in the selected time period.

-   **Average sessions per user**

    The mean number of sessions per active user in the selected time period.

-   **Average pages per session**

    The mean number of page views within a single session.

-   **Top pages**

    The most frequently visited pages, which include Employee Home Page, Canvas, 404 Page, Org chart, Tasks and requests, Profile, and Task pages.

-   **Users, sessions, events, and pages**

    Aggregate counts for each dimension, available as trend data over time.


**Note:** Department-level and business unit filtering for UXA data is not currently available. Check your entitlements to verify whether you have access to advanced UXA filtering when it is available.

## Employee comms analytics

Communications Analytics tracks the performance of announcements published through Employee Slate. This capability serves communications teams and stakeholders to measure the reach and engagement of company-wide or targeted content.

\[Omitted image "es-analytics-comms.png"\] Alt text: Comms analytics data and details

Communications Analytics provides the following metrics:

-   Impressions: the number of times an announcement appeared to a user.
-   Clicks: the number of times users selected the link in an announcement.
-   Top announcements: the announcements with the highest engagement in the selected period.
-   Breakdowns: totals and time-series values for each metric.
