# Agent Workspace for HR Case Management Guided Setup

The Guided Setup enables you to configure Agent Workspace for HR Case Management for your agents so that they can access HR cases, tasks and communications all from one location.

Agent Workspace for HR Case Management Guided Setup provides an introduction to starting your configuration. It also organizes configuration activities into categories. Complete the activities in each of the categories to customize the workspace for your human resources organization. Each category provides information like planning guidance, pre-setup steps, and links to useful information. Each category also provides links to the pages in your instance where you perform the configuration.

To access Agent Workspace for HR Case Management Guided Setup, navigate to **HR Case Management** and select Agent Workspace for HR Case Management Guided Setup. Select the **Get Started** button on the landing page to start the configuration.

In the Agent Workspace for HR Case Management Guided Setup page, you will see a list of different categories. Select the **Get Started** button under each category to start the configuration.

For more information on **Guided Setup**, see Using guided setup.
