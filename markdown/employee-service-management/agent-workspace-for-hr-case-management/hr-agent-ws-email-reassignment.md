# Send an email notification after reassignment

After you reassign an agent to an HR case, you can send an email to notify the agent.

## Before you begin

Role required: sn\_hr\_core.case\_writer

## Procedure

1.  Navigate to **All** &gt; **HR Case Management** &gt; **Agent Workspace for HR Case Management**.

2.  Select a case for which you want to send an email.

3.  Select \[Omitted image "magnifying-glass.png"\] Alt text: Search for Record icon next to the **Assigned to** field.

4.  Select an agent.

5.  Click the Open record icon \(\[Omitted image "icon-global-search.png"\] Alt text: Open Record icon\) next to the agent's name.

    The agent's details are opened in a new tab.

6.  In the Email field, click the Email icon\(\[Omitted image "envelope-outline.png"\] Alt text: Email icon\).

7.  Add a **Subject** line and compose the mail in the space provided.

8.  Click **Send**.
