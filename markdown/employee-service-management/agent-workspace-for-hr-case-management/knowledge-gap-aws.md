# Report a knowledge gap from a case in Agent Workspace for HR Case Management

Report a knowledge gap for a case to get a documented resolution for common HR cases.

## Before you begin

Role required: none

## Procedure

1.  Navigate to **All** &gt; **HR Case Management** &gt; **Agent Workspace for HR Case Management**.

2.  Select the **Lists** icon \(\[Omitted image "agent-ws-hr-list-icon.png"\] Alt text: Lists icon\).

3.  Select an HR case.

4.  Select the More icon \[Omitted image "more-icon.png"\] Alt text: More icon and select **Report knowledge gap** from the drop-down list.

5.  In the Knowledge feedback task form, enter a description in the **Description** field.

6.  Select **Save**.

7.  Request notifications of updates to an existing knowledge base article by selecting a knowledge base article from the **Outcome** field.

8.  Select **Save**.


## Result

A knowledge gap is created and is listed in the **Knowledge Feedback Tasks** related list on the case record page.
