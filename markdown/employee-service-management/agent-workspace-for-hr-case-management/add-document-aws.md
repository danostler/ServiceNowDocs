# Secure documents

Add an extra layer of security on documents that are linked to a case. Grant access to documents with sensitive or private data to groups, users, user criteria or roles.

## Before you begin

Role required: sn\_hr\_core.case\_writer

**Note:** Ensure that the Document Management \[com.snc.platform\_document\_management\] plugin is installed.

## Procedure

1.  Navigate to **All** &gt; **Agent Workspace for HR Case Management**.

2.  From Lists, select an HR case.

3.  Navigate to **More** &gt; **Documents**.

4.  Select the required document.

5.  Review the following tabs to perform the necessary actions.
