# Highlight fields in Agent Workspace for HR Case Management

Highlight fields in the workspace to call an agent's attention to them.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Workspace Experience** &gt; **Administration** &gt; **Highlighted Values**.

2.  Select **New**.

3.  On the form, fill in the fields.

5.  In **Highlighted Value Conditions**, select **New**.

6.  On the form, fill in the fields.

    For a description of field values, see [Highlighted value condition form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/agent-workspace-for-hr-case-management/highlighted-value-form.md).

7.  Select **Submit**.
