# View and download a signed contract document as a legal user

View and download a signed contract document as a legal user.

## Before you begin

Role required: sn\_lg\_ops.legal\_user and sn\_cm\_core.contract\_user

## Procedure

1.  Open the legal request either from the platform view or using Service Portal.

<table id="choicetable_uth_mkz_czb"><thead><tr><th align="left" id="d92257e55">

Method

</th><th align="left" id="d92257e58">

Actions

</th></tr></thead><tbody><tr><td id="d92257e64">

**Using platform view**

</td><td>

1.  Navigate to **All** &gt; **Legal Contracts** &gt; **All Contracts**.
2.  From the list, open a contract repository record.
3.  Navigate to the Contract Documents related list.
4.  Select the document name to view the details.
5.  Navigate to the Contract Document Revisions related list.
6.  View and download the contract document.
    -   If an external integration is active and published, select the link in the **URL** tab to open the external storage location to download the contract document.
    -   If an external storage integration is not available, select the document name to view the revision. From the **Manage Attachments** field, select the document to download it. You can also select the **view** option to view the document.


</td></tr><tr><td id="d92257e126">

**Using Service Portal**

</td><td>

1.  Navigate to **All** &gt; **Legal Service Portal**.
2.  Select the **My Requests** option on the header menu.
3.  Open the request you have submitted.
4.  In the **Legal Contracts** tab, select the contract repository number to view the document.
    -   For external storage: Contract document opens in new browser tab.
    -   For internal storage: Contract Document is downloaded.


</td></tr></tbody>
</table>
