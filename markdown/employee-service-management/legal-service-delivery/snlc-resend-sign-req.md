# Resend signature request

Resend signature request functionality enables contract fulfillers to manually trigger signature request when needed.

## Before you begin

Role required: sn\_cm\_core.contract\_fulfiller

## Procedure

1.  Navigate to your workspace.

2.  Open the contract request that is in Awaiting Signature status.

3.  Select **Resend signature request**.

4.  Select **Resend** on the confirmation screen.


## Result

Notification is resent to the current signatory.
