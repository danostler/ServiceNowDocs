# Configure fields to be displayed in Activity stream

Configure fields to be displayed in the Activity stream.

## Before you begin

Role required: sn\_lg\_ops.legal\_config

## Procedure

1.  Navigate to **All** &gt; **Legal Request** &gt; **Legal Requests** &gt; **All**.

2.  Select a legal request to customize Activity stream for.

3.  Select the activity stream icon \[Omitted image "List\_ActivityStreamIcon.png"\] Alt text: Activity stream icon.

4.  Select the filter activity icon \[Omitted image "playbook-filter.png"\] Alt text: Filter activity icon.

5.  Select **Configure available fields**.

6.  Select the fields from Available list and add to Selected list.

7.  Set the desired order.

8.  Select **Save**.

9.  Select **Update** to save the changes to the legal request form.
