# Reclassify contract documents in third-party contract review request as a legal fulfiller

As a legal fulfiller, reclassify the documents submitted in a third-party contract review request. For example, you need to change the contract document type from Non-disclosure Agreement to Partnership Agreement.

## Before you begin

Role required: sn\_lg\_cnt.contract\_fulfiller

## About this task

## Procedure

1.  Navigate to **All** &gt; **Legal Request** &gt; **Legal Counsel Center**.

2.  Select the List icon \(\[Omitted image "lsd-lcc-list-icon.png"\] Alt text: List icon\).

3.  Open the legal request for which you want to reclassify the documents.

4.  Select the contract request from the **Contract Requests** tab.

5.  In the **Contract Documents** tab, select **Reclassify**.

6.  In the **Contract type** drop-down list, select the contract type that you want to reclassify.

7.  In the **Reclassify to** drop-down list, select the contract type into which you want to reclassify.

    Only active contract types are displayed in the list.

8.  Add more information about the document in the **Work notes** field.

9.  Select **Reclassify** to update the classification.

10. Select **Save** to save the record.
