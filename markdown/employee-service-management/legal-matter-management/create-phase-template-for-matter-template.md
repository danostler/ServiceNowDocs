# Create a phase template for a matter template

Create a phase template for a matter template for resolving the matter in an organized way. When you apply a legal matter template while creating a matter, the associated phases from the template are added by default to the matter.

## Before you begin

Role required: sn\_lg\_matter.matter\_config

## Procedure

1.  Navigate to **All** &gt; **Legal Matter** &gt; **Templates** &gt; **All**.

2.  Open a matter template.

3.  In the Phases related list, click **New**.

4.  On the form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Number|System-generated phase template number.|
    |Name|Unique name for the phase template.|
    |Parent matter|Matter template to which the phase template belongs.|
    |Order|Number that determines the order in which the phase should be worked on to resolve the matter. A phase template with a lower-order number is higher in the queue to be worked on.|

5.  Click **Submit**.
