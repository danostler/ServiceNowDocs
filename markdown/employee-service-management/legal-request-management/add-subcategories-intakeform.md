# Add a subcategory for the intake form

Add a subcategory to an intake form of a practice area to organize legal requests and assignments through the intake form.

## Before you begin

Role required: sn\_lg\_ops.legal\_config

## About this task

**Note:** You can associate a subcategory with the same name to many intake forms. However, an intake form cannot have multiple subcategories with the same name.

## Procedure

1.  Navigate to **All** &gt; **Legal Administration** &gt; **Practice Areas**.

2.  Open a practice area.

3.  In the Intake Forms related list, open an intake form in which you want to add a subcategory.

4.  In the Subcategories related list, click **New**.

    If you are not able to see the **New** button in the current application scope, select the word here in the message at the top of the page to set the correct application scope for displaying the button.

5.  In the **Subcategories** field, enter a unique name of the subcategory

6.  Select the **Active** check box to indicate whether the subcategory is active.

7.  Click **Submit**.
