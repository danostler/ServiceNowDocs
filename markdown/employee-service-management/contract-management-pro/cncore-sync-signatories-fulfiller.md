# Resolve the failure to submit a self-serve contract

As a contract user, resubmit a contract request that was not submitted due to incomplete signatory information by updating and syncing signatory information.

## Before you begin

Role required: sn\_cm\_core.contract\_user

## Procedure

1.  Navigate to your workspace.

2.  Open the contract request.

3.  Review the information in the **Activity** stream to find the validation error.

4.  Select the **Signatories** tab.

5.  Select the signatory and update the information.

6.  Select **Save** to save the signatory details.

7.  Select the **Signatories** tab.

8.  Select **Sync signatories**.

    The contract document is updated with the new signatory details.

9.  Select **Save**.

10. Select **Resubmit**.


## Result

The contract request is resubmitted and the state changes from Draft to Work in progress.
