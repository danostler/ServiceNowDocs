# Submit obligation tasks

As an obligation user, work on obligation tasks to fulfill the obligation specified in the contract, and submit them for review.

## Before you begin

Role required: sn\_cm\_obligation.obligation\_user

## Procedure

1.  Open the obligation task from the email notification or Employee Center.

2.  On the menu bar, select **My Tasks**.
3.  Select the obligation task in the **Open** tab.

    **Note:** External storage is not available for Obligation Management.

3.  In the **Comments** field, add required comments.

4.  Select **Submit**.


## Result

The obligation task is sent for approval and the state of the task changes to Awaiting approval.
