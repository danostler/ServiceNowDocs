# Tracking an application record from the Hiring tab

Track an application on a job requisition and view its corresponding details as a hiring manager.

From the **Applications** tab of a job requisition, select an application record to view its details.

You can share your recommendation for the application with the recruiter by selecting **Select response** and then selecting the required option.

If you have already provided a response, you can add additional comments for the recruiter on an active application record by selecting **Add comment**.

The following information is available on the application record page:

-   The stages the application has gone through and the upcoming stages.
-   Upcoming activities section that displays the upcoming proposed and scheduled interviews in a chronological order. You can request the recruiter for a new time slot for a scheduled interview, by selecting the corresponding **Request new time** button.

The **Overview** tab displays the applicant details that include the basic information, professional experience, education information, and certifications, if any.

The **Interview** tab lists all the interviews scheduled with the applicant with details such as interview name, type, state, start time, and duration. You can select an interview from the list to navigate to the interview record page and view the interview details, attendee list, and feedback received for the applicant.

The **Feedback** tab displays the feedback received for the various interviews scheduled with the applicant. As a hiring manager, if you're one of the interviewers, you can select **Provide feedback** to [share your feedback](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/interview-management/provide-interview-feedback-hm-interviewer.md) with the recruiter. If needed, you can also edit your feedback.
