# Enable guided tours in Employee Relations

Enable guided tours in Employee Relations to get an overview of Employee Relations cases.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Guided Tour Designer** &gt; **Guided Tours**.

2.  Select the **Tour ID** beside **Details View of Employee Relations Case**.

3.  Select the **Active** check box.

4.  Select **Update**.

5.  Configure Auto Launch for the Employee Relations guided tour.

    1.  Navigate to **All** &gt; **Guided Tour Designer** &gt; **Configure Auto Launch**.

    2.  Select the **now/hr/agent/record** tile.

    3.  Select the **Auto Launch** toggle beside **Details View of Employee Relations Case** so it switches to the right.
