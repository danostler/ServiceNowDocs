# Configure time range data in tenure widgets

Use the Leader Hub Team Composition Configuration table to configure the time range data that appears in the following widgets: Range of years spent at the company and Time spent in current positions. These widgets appear on the Team details page.

## Before you begin

Role required: admin \[sn\_egd\_lh.admin\]

## About this task

Administrators can configure the following components for the data that appears in the previously mentioned widgets:

-   Up to six segments for each widget
-   The time range for each segment

## Procedure

1.  Navigate to the **All** menu, and enter `sn_egd_lh_team_composition_config.list` in the navigation filter.

    The Leader Hub Team Composition Configurations \[sn\_egd\_lh\_team\_composition\_config\] table appears.

2.  Select a record in the list to revise an existing configuration or select **New** to create a configuration.

3.  Select an option from the drop-down list associated with the **Tenure type** field to specify whether the segment is applicable to tenure in the position or the company.

    **Tip:** The option you select in this field determines the widget in which the segment appears. Select **Company** to create a segment that appears in the Range of years spent at the company widget. Conversely, select **Position** to create a segment that appears in the Time spent in current positions widget.

4.  Select a time unit from the drop-down list associated with the **Start unit** field to define whether the start of the time range is in months or years.

5.  Select a time unit from the drop-down list associated with the **End unit** field to define whether the end of the time range is in months or years.

6.  Enter the number of months or years at which the time range starts in the **Range start** field.

7.  Enter the number of months or years at which the time range ends in the **Range end** field.

8.  Indicate the sequential order in which your segment appears inside the widget by entering a numeric value in the **Display order** field.

    **Tip:** The segment you configure appears in the widget in the following order: from lowest **Display order** field value, to highest.

9.  Make your time range segment visible to leaders by selecting the check box associated with the **Active** field.

10. Select **Submit**.


## Result

The segment that you configured appears in one of the following widgets after the **LH get team composition tenure counts** scheduled job runs:

-   Range of years spent at the company
-   Time spent in current positions

## What to do next

1.  Repeat the steps in this task to create additional segments in the previously mentioned widgets.
2.  Verify that the **LH get team composition tenure counts** scheduled job runs weekly to update the data in the previously mentioned widgets. The calculated values that are generated by this scheduled job are also stored in the Leader Aggregate Values \[sn\_egd\_lh\_aggregate\_values\] table.
