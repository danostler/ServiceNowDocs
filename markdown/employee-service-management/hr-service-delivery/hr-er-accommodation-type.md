# Create accommodation types

Accommodation types help define and categorize accommodation requests.

## Before you begin

Role required: sn\_hr\_er.config\_manager

**Note:** The base system provides accommodation types, but you can add more.

## Procedure

1.  Navigate to **All** &gt; **Employee Relations** &gt; **Accommodation Types**.

2.  Select **New**.

3.  Fill in the form.
