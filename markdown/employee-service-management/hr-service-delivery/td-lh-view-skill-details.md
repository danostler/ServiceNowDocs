# View skill details

View skills relevant to your organization and metrics that you can use to identify critical skill gaps. This data can be used to hold managers responsible for developing talent within their teams.

## Before you begin

Role required: organizational leader \[sn\_egd\_lh.leader\]

## Procedure

1.  Navigate to **All** &gt; **Self-Service** &gt; **Employee Center**.

2.  From the Leader Hub drop-down list, select **Org skills**.

    The Org skills page appears.By default, skills are listed from lowest to highest skill score. Use the **Search** bar, and the **View** and **Sort by** filters to sort skills based on the criteria you choose from. Hover over the tool tip for a description of the column.

3.  Select a skill that you want to review.

4.  View the data on the Skill details page to obtain more information about the skill you selected.

    **Tip:** The Skill details page reflects the following information about the skill that you selected:

    -   Skill description
    -   Skill type
    -   Trend in the company \(rise or decline\)
    -   Roles that require the skill
    -   Skill score
    -   Skill compliance ratio
    -   Skill relevancy to the organization
    -   Average proficiency level
    -   Teams with the lowest and highest skill score
    -   Top experts \(individuals with the highest proficiency level\)
5.  Select **Schedule conversation** to schedule a conversation with the team lead for teams that needs attention or have the highest score.

    **Note:** This option is only visible if the Microsoft Outlook integration is enabled for Leader hub. For more information, see [Schedule a conversation form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/hr-service-delivery/schedule-conversation-lh.md).

6.  Select **Create a conversation** to create a conversation with the team lead for teams that needs attention or have the highest score.

    **Note:** This option is only visible if the Microsoft Outlook integration is not enabled for Leader hub. For more information, see [Create a conversation form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/hr-service-delivery/create-conversation-lh.md).
