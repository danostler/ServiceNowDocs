# Properties installed with Talent Profile

The system properties installed with the Talent Profile plugin are required to configure different functionalities in the Talent Profile environment.

These properties are available for Talent Profile.

**Note:** To open the System Properties \[sys\_properties\] table, enter `sys_properties.list` in the navigation filter.

<table id="table_fzd_dx3_tdc"><thead><tr><th>

Property

</th><th>

Description

</th></tr></thead><tbody><tr><td>

glide.ui.sn\_ta\_tp\_talent\_pool\_activity.fields

</td><td>

Talent pool activity formatting fields.-   Type: String
-   Default value: \*Attachments\*, \*EmailCorrespondence\*, \*EmailAutogenerated\*, number, pool\_name, owner, purpose, collaborate\_with, share\_with, description, comments

</td></tr><tr><td>

glide.ui.sn\_ta\_tp\_talent\_profile\_activity.fields

</td><td>

Talent profile activity formatter fields.-   Type: String
-   Default value: \*EmailCorrespondence\*, \*EmailAutogenerated\*, \*Attachments\*

</td></tr><tr><td>

sn\_ta\_tp.breached\_time.no\_of\_days

</td><td>

Number of days a talent pool can be inactive before it’s purged.-   Type: Integer
-   Default value: 2556
-   Other possible values:
    -   Ignore cache: Selected
    -   Read roles: admin, sn\_ta\_tp.talent\_profile\_admin
    -   Write roles: admin, sn\_ta\_tp.talent\_profile\_admin

</td></tr><tr><td>

sn\_ta\_tp.min\_admin\_count

</td><td>

Minimum number of active talent profile admin users. -   Type: String
-   Default value: 1
-   Other possible values:
    -   Ignore cache: Selected
    -   Read roles: sn\_ta\_tp.talent\_profile\_admin
    -   Write roles: sn\_ta\_tp.talent\_profile\_admin

</td></tr><tr><td>

sn\_ta\_tp.talent\_profile\_enabled

</td><td>

Option to opt in and out of the Talent Profile.-   Type: true \| false
-   Default value: true
-   Other possible values:
    -   Ignore cache: Selected
    -   Write roles: sn\_ta\_tp.talent\_profile\_admin

</td></tr><tr><td>

sn\_ta\_tp.talent\_profile\_last\_run\_at

</td><td>

Update frequency of the list of Talent Profiles.-   Type: String
-   Default value: YYYY-MM-DD 00:00:01
-   Other possible values:
    -   Ignore cache: Selected
    -   Private: Selected
    -   Write roles: sn\_ta\_tp.talent\_profile\_admin

</td></tr></tbody>
</table>
