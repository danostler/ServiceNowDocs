# Complete PIP assessment

As a manager, complete the PIP assessment indicating whether the PIP is successful.

## Before you begin

Role required: none

## Procedure

1.  Navigate to **All** &gt; **Employee Center**.

2.  Complete the PIP assessment task.

3.  If the PIP is successful:

    1.  Select **Yes** from the PIP successful list.
    2.  Click **Submit**.
4.  If the PIP is unsuccessful:

    1.  Select **No** from the PIP successful list.
    2.  From the PIP extended list, select **Yes** to indicate the extension date, or select **No** if you do not want to specify an extension date.
    3.  Click **Submit**.

## What to do next

When the PIP is marked unsuccessful, a PIP assessment case is created for the ER agent. The ER agent and the manager can discuss the next course of action for the employee.
