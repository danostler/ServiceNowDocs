# Employee passport

Enable employee's data export, which can be used by any other agency or department for tracking, compliance, and regulatory purposes.

This feature makes employee profile data, such as employment history and progression to be made available across different ServiceNow instances.

As an HR agent, you can extract the employment history of an employee by performing the following steps:

1.  Create a case for the employee with the HR service as Generate Employment History. A document template, Employment History, is automatically attached to the case.

    **Note:** HR Admin with the sn\_doc.writer role can configure the fields to be available in the Employment History document template.

2.  Use the **Generate Attachments** option in the document template to generate the employment history in a PDF, and attach it to the case.
