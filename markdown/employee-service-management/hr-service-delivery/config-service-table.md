# Create a service table configuration in Service Portfolio Management

Create a service table configuration for your Business Unit \(BU\). Associate your BU service mappings to the correct service table configuration.

## Before you begin

Role required: service\_admin

## Procedure

1.  Navigate to **All** &gt; **Service Portfolio Management** &gt; **CSDM Configuration** &gt; **Service table Config**.

2.  Click **New**.

3.  On the form, fill in the fields:

<table id="table_dc2_gzt_jzb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Name of the service.

</td></tr><tr><td>

Service table

</td><td>

BU table to which the service is mapped.

</td></tr><tr><td>

Mapping level

</td><td>

Mapping level associated with the Service table configuration: -   Service: Parent service
-   Service offering: Refines the parent service to a specific business need and performance level.


</td></tr><tr><td>

Application

</td><td>

Application to which the service table configuration is associated.

</td></tr><tr><td>

Active

</td><td>

Option to activate or deactivate the service table configuration.

</td></tr></tbody>
</table>4.  Click **Submit**.
