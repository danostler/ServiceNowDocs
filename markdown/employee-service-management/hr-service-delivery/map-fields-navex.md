# Map fields

Map ServiceNow fields with the fields that are pulled from NAVEX EthicsPoint. Enable an HR Agent to work on employee relation cases from a ServiceNow instance.

## Before you begin

Role required: sn\_hr\_navex\_ep.admin

## Procedure

1.  Navigate to **ER Navex Integration** &gt; **Administration** &gt; **Field Value Map**.

2.  Click **Get Navex Field Values**.

    All the fields from Navex EthicsPoint are pulled into ServiceNow instance.

    For complete details of fields that are imported, refer to this [KB article](https://support.servicenow.com/kb?id=kb_article_view&sysparm_article=KB1541716).

3.  Enter the required values in the **Value** field and **Primary** field for each of the record that is displayed.
