# Edit a generated document

Support rendering values of variables, scripts, and blocks in edit mode in Preview Document. View the content of variables, scripts, or blocks rather than only viewing the variable or block names.

## Before you begin

Role required: sn\_hr\_core.case\_writer

**Note:** This feature is available only when the document is generated from a template in the Document Template plugin.

## Procedure

1.  Navigate to **All** &gt; **HR Case Management** &gt; **All HR Cases** &gt; **Assigned to me**.

2.  Open an HR case that is assigned to you.

3.  Click **Preview Document**.

4.  In the Preview Document window:

    1.  Click **Edit Document**. Except for signature tokens, all other variables, scripts and blocks are rendered with their values.
    2.  Add text, any valid tokens, or signature tokens.

        **Note:** Invalid tokens will be removed while saving.

    3.  Click **Save changes**.
    The document is saved and sent as an attachment in document tasks that are created for participants.
