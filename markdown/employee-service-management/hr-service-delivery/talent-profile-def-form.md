# Talent profile definition form

The following table provides the field and its corresponding description. The fields are required to create a talent profile definition.

<table id="table_omn_dpv_tbc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Name of the talent profile definition.

</td></tr><tr><td>

Talent type

</td><td>

Types of talent profiles. There are four types available by default.

-   External
-   Employee
-   Alumni
-   Contingent

For more information on creating a talent type, see [Create a talent type](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/hr-service-delivery/create-implement-ta-tp.md).

</td></tr><tr><td>

Table

</td><td>

Table record from where the talent data is gathered.This field is automatically set depending on the choice of the **Talent type**.

</td></tr><tr><td>

Active

</td><td>

Option to enable the definition record.

</td></tr><tr><td>

Order

</td><td>

Order of preference for the definition.

</td></tr><tr><td>

Filter condition

</td><td>

Filter conditions to be met to be added to the definition.

</td></tr></tbody>
</table>
