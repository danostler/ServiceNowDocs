# HR Benchmarks

The HR Benchmarks application provides you with an insight into the industry averages of your peers. Use your Key Performance Indicators \(KPIs\) to compare your performance with global benchmarks, which contributes to improved performance for your organization.

## Key features

-   Percentile ranking to indicate your standing within your participating peer group.
-   Integration with ServiceNow Performance Analytics for daily data collection and drill down on KPI data.
-   Data filtering by industry, number of users, or geographical region.
-   Category-based roles to limit data access.
-   Ability to change KPI definitions to match customizations.
-   ServiceNow Benchmarks dashboard in ServiceNow Service Portal \(mobile friendly\).
-   Ability to download KPI reports in a PDF format.

## HR Benchmarks Dashboard

\[Omitted image "hr-benchmarks.png"\] Alt text: HR Benchmarks Dashboard

The KPIs in the HR Benchmarks application are performance analytics indicators that only collect the usage count data, for example, the number of closed HR cases in a month, based on the monthly aggregates. During data collection, the HR Benchmarks application does not consider any other details such as description of cases, or information about requests, changes, or applications.

The participating customer count for each cohort bucket in Industry Category, User Size, and regions aggregates are large enough to calculate monthly benchmarks values and to maintain full anonymity. To further verify data anonymity, the HR Benchmarks user interface enables you to use only one filter at a time.

**Important:** To learn more about the Benchmark features, see the  documentation.
