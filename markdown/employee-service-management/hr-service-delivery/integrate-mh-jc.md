# Integrate Manager Hub

Extend Manager Hub capabilities by integrating with other applications.

## Integration with Journey Accelerator

Use the Journey Accelerator templates to create and publish customized plans for key employee transitions. Create and access the journey plans from Manager Hub in Employee Center.

For this integration to work, you must install Manager Hub, Journey Accelerator, and Employee Center Pro applications.

## Integration with Journey Designer

Facilitate daily activities among various cross-functional processes and departments, and automate processes such as new hire onboarding.

For this integration to work, you must install Manager Hub, Journey Designer, and Employee Center Pro applications.

## Integration with Enterprise Employee Experience Pack

Administer and coordinate your employees return to the workplace in a manner that promotes your company's safety requirements. Click the Manage return to office link in the Manager Library tab in Employee Center.

For this integration to work, you must install Manager Hub, Enterprise Employee Experience Pack, and Employee Center Pro applications.

## Integration with Employee Relations COE

Report any misconduct or sensitive issues through the Employee Relations application, which provides a tighter security to protect employee privacy. Click the Report misconduct employee quick link in the Manager Library tab in Employee Center.

For this integration to work, you must install Manager Hub, Employee Relations, and Employee Center Pro applications.

## Integration with Human Resources Scoped App

Contact an HR personnel by clicking the Ask HR quick link in the Manager Library tab in Employee Center.

For this integration to work, you must install Manager Hub, Human Resources Scoped App application, and Employee Center Pro plugins.

## Integration with Content Experiences

Find useful articles and resources that help to expand your skills as a people leader. Use the Manager Library tab in Manager Hub in Employee Center.

Personalised and seasonally-based content from target communications and campaigns, knowledge or catalog items \(ServiceNow\), share point sites, or other third party applications can also be added in Manager Library.

Manager Library is available with Content Experiences application.

## Integration with Employee Center Core

Complete your tasks and approve any team requests from the Overview tab in Manager Hub in Employee Center.

Team tasks and requests appear only when the Employee Center Core application is installed.

## Integration with Proactive Prompts

Provide prompts in the Team Insights page in Manager Hub. Encourage managers or employees to take actions quickly.

For this integration to work, you must install Manager Hub, Proactive Prompts and Employee Center Pro applications.
