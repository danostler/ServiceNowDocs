# Add talent profiles to a Talent pool

Add similar talent profiles to a Talent pool to streamline communication to all the profiles during a particular hiring drive.

## Before you begin

Role required: sn\_ta\_hiring\_core.recruiter

## Procedure

1.  Navigate to **Workspaces** &gt; **Recruitment workspace**.

2.  Go to the **Talent pool** tab.

3.  In the Talent pool section, select the required pool where you want to add more talent profiles.

4.  In the Talent pool page, go to the **Talent** tab.

5.  Select **Add talent** to view the pop-up window with the list of talent profiles.

    **Note:** The button is only visible to the owner or the collaborator of the talent pool.

6.  Add talent profiles from either choosing All talent or a particular pool.

    -   To choose from all talent profiles, select **All talent** from the drop-down on the pop-up window.
    -   To choose from a particular pool, select your chosen pool from the drop-down on the pop-up window.
    **Note:** You see a pool in the drop-down list only if you're the owner of the pool, one of the collaborators, or the pool is shared with you.

7.  Select the talent profiles that you want to add to the current pool and select **Save**.
