# Transform maps

Transform maps specify data relationships between source table and target table.

For details on the transform maps, navigate to **ER Navex Integration** &gt; **Transform Maps**.

|Name|Description|
|----|-----------|
|ER Case Transform Map|Maps employee relation cases from the Case Staging \[sn\_hr\_navex\_ep\_case\_staging\] table to the Employee Relations Case \[sn\_hr\_er\_case\] table.|
|ER Involved Party Staging|Maps involved participants of cases from the Participant Staging \[sn\_hr\_navex\_ep\_participant\] table to the Involved Party \[sn\_hr\_er\_involved\_party\] table.|
|ER Evidence Transform Map|Maps employee relation case evidences from the Case File Staging \[sn\_hr\_navex\_ep\_file\_staging\] table to the Evidence \[sn\_em\_evidence\] table.|
