# Map Magnit tasks

Map an onboarding item name with HR task template to create an adhoc task for the contingent worker.

## Before you begin

Role required: sn\_hr\_magnit.admin

## Procedure

1.  Navigate to **All** &gt; **HR Magnit Integration** &gt; **Magnit Task Mapping**.

2.  On the form, fill in the fields:

    |Field|Description|
    |-----|-----------|
    |Onboarding item name|Onboarding item \(related to contingent worker\) that is pulled from Magnit application.|
    |HR task template|HR task template with which the onboarding item is mapped.|
    |Active|Option to activate the use of this mapping.|

3.  Click **Submit**.
