# Use Now Assist to open a job requisition

Use Now Assist in Virtual Agent to create or duplicate a job requisition for a vacancy that you want to fill in your organization.

## Before you begin

Make sure that you activate Now Assist in Virtual Agent version 4.0.1 or later as some Now Assist features might vary in earlier versions.

Role required: sn\_ta\_hiring\_core.hiring\_manager, sn\_ta\_hiring\_core.recruiter

## Procedure

1.  Navigate to **All** &gt; **Self-Service** &gt; **Employee Center**.

2.  Open the Virtual Agent chat window by selecting the chat icon \(\[Omitted image "icon-chat-va.png"\] Alt text: Chat icon for Virtual Agent\).

3.  Request to open a new job requisition in one of the following ways.

    -   Select **Show me everything** &gt; **New job opening request**.
    -   Enter a request in the chat to open a new job requisition.
4.  Select one of the following options and follow the Virtual Agent prompts.

## Result

A new job requisition is created.

**Note:** When a job request is created by a hiring manager, the recruiter can see the open request on the overview page of the Recruitment workspace.
