# Using HR Service Delivery Integration with Workday

Understand how you can use HR Service Delivery Integration with Workday to process employee information at ServiceNow.

-   **[Use transform maps](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/hr-service-delivery/transform-maps-w.md)**  
Transform maps specify data relationships between source table and target table.
-   **[View job tracker details from Workday](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/hr-service-delivery/w-job-tracker.md)**  
Track every pull that has been performed from ServiceNow to Workday either by schedule job or manual reconciliation process. View details of the job, such as when did the job begin and end, when did the records start loading into staging tables, or did the job run completely or fail in between.
-   **[View tasks from Workday](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/hr-service-delivery/view-to-dos-wd.md)**  
View details of the todos or tasks that are pulled from Workday into ServiceNow.
