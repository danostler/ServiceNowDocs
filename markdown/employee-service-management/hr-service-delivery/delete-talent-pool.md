# Delete a Talent pool

Delete a Talent pool when it has served its purpose or it’s no longer relevant.

## Before you begin

Role required: sn\_ta\_hiring\_core.recruiter

## Procedure

1.  Navigate to **Workspaces** &gt; **Recruitment workspace**.

2.  Go to the **Talent pool** tab.

3.  In the Talent pool section, select the Talent pool you want to delete.

4.  Select the ellipses on the Talent pool record page and select **Delete**.

5.  In the Delete talent pool pop-up window, select **OK** to confirm deleting the pool.

    **Note:** The delete functionality is available only for the owner of the Talent pool.


## Result

You have deleted the selected Talent pool.
