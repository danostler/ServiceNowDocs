# Employee Relations Performance Analytics Executive Overview

The HR Service Delivery Employee Relations \(ER\) Performance Analytics \(PA\) dashboard provides you with historical trends, data comparisons.

You must activate the Performance Analytics - Content Pack - Human Resources: Employee Relations Scoped App plugin.

The HR ER PA dashboard provides timely information for the following:

## Executive overview

Provides data on trends or areas of concern that are emerging and require a deeper analysis in order to create proactive strategies.

\[Omitted image "hr-er-ov1.png"\] Alt text: Executive Overview

## Case analysis

Provides case metrics and goals for ER agents and teams. Having this information can also help your ER teams plan their workload based on volume trends and the details on case data. This section provides:

-   The number of open ER cases from the current time and date.
-   The number of cases that have exceeded their Service Level Agreement \(SLA\).
-   The number of cases created for the week from the current time and date.
-   The number of restricted ER cases created from the current time and date.

## Investigation analysis

ER investigation leaders need to understand the quality of service in an ER investigation and be able to identify trends beyond individual cases. Comparing cases and trends in allegation data are important in this area. This section provides:

-   The number of ER cases by HR service. You can change how the data is displayed using the **Visualization** list.
-   The number of ER cases by geographical location.
-   The number of ER cases by department. You can change how the data is displayed using the **Visualization** list.
-   The number for each allegation type for open cases. You can change how the data is displayed using the **Visualization** list.
-   Data trends by allegation type.
-   The number of closed cases by allegation outcome.
