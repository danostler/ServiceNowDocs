# Create talent form

A new candidate profile is added by filling the following details in the form.

<table id="table_o44_nqd_xbc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

First name

</td><td>

Given name of the candidate.

</td></tr><tr><td>

Last name

</td><td>

Family name of the candidate.

</td></tr><tr><td>

Phone number

</td><td>

Contact number of the candidate.

</td></tr><tr><td>

Current job title

</td><td>

Existing job title of the prospective candidate.

</td></tr><tr><td>

Candidate type

</td><td>

Type of candidate. There are two candidate types available.

-   Prospect
-   Applicant

</td></tr><tr><td>

Consent status

</td><td>

Auto-generated consent status of the candidate.The consent status is changed when a change is requested from the candidate.

</td></tr><tr><td>

Email

</td><td>

Registered email address of the candidate.

</td></tr><tr><td>

Location

</td><td>

Location where the candidate is based or plans to join the company.

</td></tr><tr><td>

Current employer

</td><td>

Existing employer of the prospective candidate.

</td></tr><tr><td>

Attachment

</td><td>

Section to attach relevant documents or use or create different templates.
