# Set up a template for Total Rewards

Set up a template by adding sections to it. Templates enable you to display a customized Total Rewards view to employees in the configured Service Portal.

## Before you begin

Role required: sn\_hr\_workday\_adv.admin

## Procedure

1.  Navigate to **All** &gt; **Workday Advanced Use Cases** &gt; **Total Rewards** &gt; **Template Set up**.

2.  Click **New**.

3.  On the form, fill in the fields:

    |Field|Description|
    |-----|-----------|
    |Template name|Suitable name of the template.|
    |Sections|Sections that you want to include in the template, for example, Base Pay, Allowances, Incentive Pay, and Commission.|
    |Applies to|Template applies only to the selected user criteria. For example, the template is applicable only to employees whose Employment Type Is Full Time.|
    |Order|If more than one template is applied to an employee, only the template with a lesser order is displayed to the employee in the configured Service Portal.|
    |Domain|Reference to domain.|

4.  Click **Submit**.
