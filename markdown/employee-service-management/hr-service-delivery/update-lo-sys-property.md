# Configure learning objects for To-dos page

Add or remove learning objects that appear on the To-dos page as learning tasks using the **sn\_hr\_cornerstone.filtered\_learning\_objects** system property.

## Before you begin

Role required: sn\_hr\_cornerstone.admin

## Procedure

1.  In the navigation filter, enter `sys_properties.list`.

    The System Properties list is displayed.

2.  Select **sn\_hr\_cornerstone.filtered\_learning\_objects**.

3.  In the **Value** field, add or remove the learning object ID of the learning task you want on the To-dos page.

4.  Select **Update**.
