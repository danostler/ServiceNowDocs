# Configuring Talent profile

Install and configure the Talent profile application to get an aggregated view of different talent types at different point in the hiring journey.

-   **[Install Talent profile](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/hr-service-delivery/install-talent-profile.md)**  
You can install the Talent profile application \(sn\_ta\_tp\) if you have the admin role. The application includes demo data and installs related ServiceNow® Store applications and plugins if they are not already installed.
-   **[Set up Talent Definition](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/hr-service-delivery/talent-definition-setup.md)**  
Create a talent profile definition to opt in to Talent Profile.
-   **[Opt in to Talent Profile](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/hr-service-delivery/opt-in-ta-tp.md)**  
Opt in to the Talent Profile to access all the talent records, create talent pools, and add prospects to the job requisitions.
-   **[Generate talent profiles](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/hr-service-delivery/generate-tp.md)**  
Generate the talent profile creation after you opt in.
-   **[Create a talent type](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/hr-service-delivery/create-implement-ta-tp.md)**  
Create an implementation in the talent profile scripted extension point to add a new talent type.
-   **[Talent Profile management](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/hr-service-delivery/c_talent-manage.md)**  
Talent Profile management activities help keep data set up to date and clutter-free.
-   **[Set up a scheduled job for talent profile generation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/hr-service-delivery/setup-schd-job-tp.md)**  
Customize the frequency of the scheduled job to get an uninterrupted flow of new talent profiles.
