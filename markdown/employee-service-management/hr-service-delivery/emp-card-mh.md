# View employee cards in Skills Expectations section

View the score percentages of an employee based on an employee's validated skills and proficiencies, compared to their role-based skill requirements.

## Before you begin

Role required: sn\_mh.manager\_hub\_user

## Procedure

1.  Navigate to **All** &gt; **Employee Center**.

    The Employee Center portal opens.

2.  Open **Your team** &gt; **Skills**.

3.  In the Skills Expectations section, select \[Omitted image "viewmenu.png"\] Alt text: View menu icon on an employee card and select any of the following options:

Option

Validate skills

</td><td>

1.  Review the current skill proficiencies of an employee, and then select any skills you want to validate.
2.  Select **Validate selected skills**.

**Note:** For Score analysis, you must validate the skills.

</td></tr><tr><td>

View skills

</td><td>

Skill details page provides granular details of employee skills, proficiency levels, and gaps. For more information, see [Skill details of an employee](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/hr-service-delivery/skill-details-emp.md).

</td></tr><tr><td>

View profile

</td><td>

The employee profile page appears with details such as reporting structure, coworkers, designation of employee.

</td></tr></tbody>
</table>4.  In the Skills Expectations section, select **Score analysis** on an employee card to view the score percentage of the employee's validated skills.

    1.  Hover over the tool tip in the flyout to see criteria for skill proficiency levels.

    2.  Select **Assign learning** in the **Validated** skills section to view progress bars of skills that meet and don't meet expected proficient levels.

    3.  Select the Skill pill icon \(\[Omitted image "viewmenu.png"\] Alt text: View menu icon\) on a skill to perform one of the following:

        -   **View skill details**: View a detailed description of the skill.
        -   **Assign learning**: Assign a learning course to an employee. For more information, see [Assign learning form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/hr-service-delivery/assign-learning-form.md).
5.  To view all the employee cards, select **View all**.

6.  If you have team members reporting from different job functions, you can filter the skill distribution by job function.
