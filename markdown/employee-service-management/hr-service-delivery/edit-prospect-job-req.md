# Edit prospects in job requisition

Edit newly created prospect profiles within a job requisition until the prospect applies.

## Before you begin

Role required: sn\_ta\_hiring\_core.recruiter

## Procedure

1.  Navigate to **Workspaces** &gt; **Recruitment workspace**.

2.  Go to the **Job requisition** tab.

3.  Select the required job requisition.

4.  On the **Prospects** tab, select the talent profile to be edited.

    You can also select the talent profile from the All Talent section in the **Talent pool** tab on Recruitment workspace.

5.  Select **Edit candidate**.

    You can only edit a candidate profile until they've applied to the job requisition.

6.  Update the details, as required.

    For a description of the field values, see [Create talent form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/hr-service-delivery/new-talent-form.md).

7.  Select **Save**.
