# Variables supported in HTML document templates

Include customized variables in a document template in addition to the variables that are available in the **Select Variables** field in the HTML document template configuration page.

## Supported Variables

The following variables are supported in an HTML document template:

-   Date
-   Date/Time
-   Duration
-   Email
-   HTML
-   IP Address
-   List Collector
-   Lookup Multiple Choice
-   Lookup Select Box
-   Multi Line Text
-   Multiple Choice
-   Numeric Scale
-   Reference
-   Select Box
-   Single Line Text
-   URL
-   Wide Single Line Text
-   Yes / No
-   CheckBox
-   Requested For

## Unsupported Variables

The following variables are unsupported in an HTML document template:

-   Attachment
-   Break
-   Container End
-   Container Split
-   Container Start
-   Custom
-   Custom with Label
-   UI Page
-   Label
-   Rich Text Label
-   Masked
-   Multi Row Variable Set
