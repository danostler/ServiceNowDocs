# Create prospects in job requisitions

Create a prospect within a job requisition as a potential fit for the job opening.

## Before you begin

Role required: sn\_ta\_hiring\_core.recruiter

## Procedure

1.  Navigate to **Workspaces** &gt; **Recruitment workspace**.

2.  Go to the **Job requisition** tab.

3.  Select the required job requisition.

4.  Go to the **Prospects** tab in the job requisition record page.

5.  Select the **Create a prospect** button under **Add prospects**.

6.  On the Create New Candidate form, fill in the fields.

    For a description of the field values, see [Create talent form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/hr-service-delivery/new-talent-form.md).

7.  Select **Save**.


## Result

You have created a prospect within a job requisition.
