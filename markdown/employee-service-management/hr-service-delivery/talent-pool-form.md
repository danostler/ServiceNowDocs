# Talent pool form

The field descriptions provided are required to create a talent pool.

|Field|Description|
|-----|-----------|
|Number|Auto-generated number of the new talent pool.|
|Purpose|Short description of the purpose behind creating the talent pool.|
|Owner|Name of the owner of the talent pool.|
|Pool name|Name of the talent pool.|
|Collaborated with|Persons chosen to be collaborators in the talent pool.|
|Share with|Persons the talent pool is shared with.|
|Description|Detailed description of the requirement of the talent pool.|
|Attachments|Section to attach relevant documents or use or create different templates.|
