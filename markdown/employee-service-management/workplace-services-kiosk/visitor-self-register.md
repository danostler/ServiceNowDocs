# Self-register as a visitor

Register as a visitor by providing your details such as your name, purpose of the visit, and host.

## Before you begin

The self-registration flow must be configured and enabled by the admin. You can self-register only if you try to check-in and the kiosk device displays a `No visit found` page.

Role required: sn\_wsd\_kiosk.viewer

## Procedure

1.  On the error page, select **Register visit**.

2.  On the Register your visit page, enter the following details, then select **Continue**.

    -   Your email address
    -   Visit description
    -   Host name
    **Note:** The fields might change based on the initial requirements that are set by the admin.

3.  On the Complete your information page, enter the following details, then select **Continue**.

    -   First name
    -   Last name
    -   Phone number
    -   Job title
    -   Parking preferences
    -   License plate
    **Note:** The fields might change based on the additional requirements that are set by the admin.

4.  Take a picture for the check-in process.

    1.  On the Let's take a picture page, select **Continue**.

    2.  Select the camera icon to start the timer for a picture.

        You can use the time to adjust your position.

    3.  Select **Continue** to use the picture for your visit.

        You can also retake your picture by selecting **Retake**.

5.  Read the visitor policies, then select **Acknowledge**.

    Based on the configuration set by the admin, you might have to add a signature to acknowledge the policies.

6.  View the next steps, then select **Done**.
