# Kiosk configuration form

Use the Kiosk configuration form to create a kiosk device configuration.

<table id="table_pch_f4n_4bc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Name of the kiosk device.

</td></tr><tr><td>

Homepage configuration

</td><td>

Homepage configuration that must be linked to the kiosk device.

</td></tr><tr><td>

Managed by

</td><td>

User group that can manage the kiosk device.

</td></tr><tr><td>

Location

</td><td>

Location of the kiosk device. The company policies and the list of hosts are displayed based on the selected location.Ensure that the location is not a site or a region. The location must be a campus, a building, or a lower order location.

**Note:** To display employees in the list of hosts, the kiosk location must match with the Workplace Profile location of the employees.

</td></tr><tr><td>

Description

</td><td>

Description of the kiosk device.

</td></tr><tr><td>

Default language

</td><td>

Default language displayed on the kiosk device.

</td></tr><tr><td>

Single module mode

</td><td>

Option to enable only a single module for the kiosk configuration. If you have multiple modules linked to the configuration, the module with the lowest order is displayed on the kiosk.

</td></tr><tr><td>

Device ID

</td><td>

Auto-generated identifier for the kiosk device.
