# Customize the kiosk header

Customize the kiosk header by editing the CSS code of the Kiosk Header record.

## Before you begin

Role required: sp\_admin

## Procedure

1.  Navigate to **All** &gt; **Service Portal** &gt; **Portals**.

2.  From the Service Portals list, select the **Kiosk** record.

3.  From the **Theme** field, open the **Kiosk Theme** record.

4.  From the **Header** field, open the **Kiosk Header** record.

5.  In the **CSS** field, customize the code based on your requirement.

    For example, you can hide the kiosk header by setting the display of the `header-container` class to `none`.
