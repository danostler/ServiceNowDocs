# Create a flow for Workplace Services Kiosk

Create a check-in or check-out flow for the Workplace Service Kiosk.

## Before you begin

Role required: sn\_wsd\_visitor.admin

## Procedure

1.  Navigate to **All** &gt; **Workplace Visitor Management** &gt; **Administration**.

2.  Select either of the following modules based on your requirement.

    -   Check-in Flows
    -   Check-out Flows
    Based on your selected module, either the Kiosk Check-in Flow Configurations list or Kiosk Check-out Flow Configurations list opens.

3.  On the list, select **New**.

4.  On the form, fill in the fields.

    For a description of the field values, see one of the following topics based on the selected module.

    -   [Kiosk Check-in Flow Configuration form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/workplace-visitor-management/kiosk-check-in-config-form.md)
    -   [Kiosk Check-out Flow Configuration form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/workplace-visitor-management/kiosk-check-out-config-form.md)
