# Provider Space Type Mappings

This table maps workplace space types or locations with external ids provided by the space occupancy data providers.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Administration** &gt; **Provider Space Type Mappings**.

2.  On the form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Space Type|Workplace Service Delivery space type. For example, Rooms, Desks, meeting rooms, and so on|
    |Provider Space Type|Space type of the Provider. For example, Desks, Rooms, and so on.|
    |Provider|Provider name. For example: Metrikus.|

3.  Select **Submit**.
