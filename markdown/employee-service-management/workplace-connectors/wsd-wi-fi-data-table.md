# Wi-Fi data table

Retrieve the device name, access point, and connection date and time for a workplace location. The Workplace Connectors Wi-Fi data checks the occupancy of a workplace location.

## Before you begin

Role required: admin

**Note:** The application comes with seed data, or base system table records, that include pre-filled values for Wi-Fi data retrieved from Wireless Access Point \(WAP\) devices.

## Procedure

1.  Navigate to **All** &gt; **Workplace Connectors** &gt; **Wifi Access Point Data** &gt; **Wifi Data**.

2.  On the form, review the field details for Wi-Fi data.

    |Field|Description|
    |-----|-----------|
    |Access Point|Wi-Fi access point through which a device connects or disconnects.|
    |Device Name|The name of the device that is connected to Wi-Fi.|
    |Location|Workplace locations that are assigned to the Wi-Fi data provider. The location data shows the complete location hierarchy \(Region, site, campus, buildings, floors, and spaces\).|
    |Wap Client Name|The name of the client device connecting to a wireless access point.|
    |Event data|Time series data. Select this value to visualize the time-series data in different transforms. For more information, see .|
