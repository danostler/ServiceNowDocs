# Configure sensor data providers

Configure sensor data providers in Workplace Connectors for retrieving the workplace sensor data from third-party data providers.

## Before you begin

**Note:** If you’re using demo data or product data \(seed data\), the provider name field is pre-filled. In case, if you aren’t using the base system product data, you can create a provider data record. The seed data is available as part of the Metrikus Spoke integration.

Role required: admin \(sn\_wsd\_wc.admin\)

## Procedure

1.  Navigate to **All** &gt; **Workplace Connectors** &gt; **Administration** &gt; **Providers**.

2.  Select **New**.

    Enter a provider name. For example: Metrikus.

3.  Select **Submit**.
