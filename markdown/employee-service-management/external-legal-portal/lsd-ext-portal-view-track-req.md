# View and track requests submitted in External Legal Service Center

View and track requests submitted for a legal service from the External Legal Service Center.

## Before you begin

Role required: sn\_lg\_ext\_portal.ext\_user

## About this task

After you submit a request, the assigned group starts working on it. You receive an email when a comment is added to the request. The request is marked as complete once it is resolved.

## Procedure

1.  Navigate to the External Legal Service Center home page by accessing your instance URL and adding a /elp suffix.

    For example, https://example.com/elp.

2.  Log in with your user name and password.

3.  Navigate to **My Requests** from the header.

4.  Navigate to the Submitted request tab to see your requests.

    A list of submitted requests will be displayed with the State.

5.  Select a request to view the details.

    -   In the Activity tab, view the comments.
    -   In the Attachment tab, view the attached files.
    -   In the Request Details, view the summary and description.
6.  Post a comment in the request.

    1.  Navigate to the Activity tab.

    2.  Enter a comment in the Type your message here field.

    3.  Select **Post**.

7.  Cancel the request if it is no longer needed.

    1.  Select **Actions** &gt; **Cancel request**.

    2.  Enter the reason for cancellation.

    3.  Select **Confirm cancellation**.

    The request is canceled and you will receive an email notification for the cancellation.
