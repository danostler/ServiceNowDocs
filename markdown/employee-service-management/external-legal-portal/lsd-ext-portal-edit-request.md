# Edit requests in External Legal Service Center

Edit requests that are saved as draft in External Legal Service Center.

## Before you begin

You can only edit requests that are saved as draft. Once you submit a request, you cannot edit it.

Role required: sn\_lg\_ext\_portal.ext\_user

## Procedure

1.  Navigate to the External Legal Service Center home page by accessing your instance URL and adding a /elp suffix.

    For example, https://example.com/elp.

2.  Log in with your user name and password.

3.  Navigate to **My Requests** from the header.

4.  Navigate to the Drafts tab.

5.  Select a request.

6.  Select **Update Draft**.

7.  Edit the Summary description or Description.

8.  Work on the attachments.
