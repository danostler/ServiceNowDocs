# Create request from External Legal Service Center

Create a request for a legal service from the External Legal Service Center. You can submit a request or save it in draft to submit later.

## Before you begin

Role required: sn\_lg\_ext\_portal.ext\_user

## About this task

Verify you have registered with External Legal Service Center before submitting a request for a legal service. For more information, see [Register with External Legal Service Center](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/external-legal-portal/lsd-external-portal-register.md).

## Procedure

1.  Navigate to the External Legal Service Center home page by accessing your instance URL and adding a /elp suffix.

    For example, https://example.com/elp.

2.  Log in with your user name and password.

3.  From the Legal Services catalog, select the request type to be submitted.

4.  Add Summary description.

5.  Add Description.

6.  Add attachments.

    1.  Select Add icon \(\[Omitted image "lsd-ext-portal-add-attachment.png"\] Alt text: Add icon\).

    2.  Add attachments using any of the following methods.

    -   Select **Choose a file** and select the file to be attached.
    -   Drag files from your local computer into your browser window to attach them to the current record.
    -   From your local computer, copy the files that you want to attach. Click inside the Documents box and paste the files.
7.  Submit the request or save it as a draft.
