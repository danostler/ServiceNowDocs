# Configure HR Service Delivery for Healthcare

Plan and configure your HR Service Delivery for Healthcare.

## Configuration overview

First, activate the HR Service Delivery for Healthcare \[sn\_hr\_hc\] application from ServiceNow Store.

Activating HR Service Delivery for Healthcare also activates the following applications:

-   Journey Designer \[sn\_jny\]
-   Employee Center Pro \[sn\_ex\_sp\_pro\]
-   UI shared library \[sn\_ui\_lib\]
-   Document Templates \[sn\_doc\]
-   Content Experiences \[sn\_ca\]
-   Content Publishing \[sn\_cd\]
-   Employee experience taxonomy \[sn\_ect\]
-   Human Resources Scoped App: Core \[com.sn\_hr\_core\]
-   Employee Center - Pro \[com.sn\_ex\_employee\_center\_pro\]
-   User Criteria Scoped API \[com.glideapp.user\_criteria.scoped.api\]
-   Content Publishing \[com.sn\_content\_delivery\]
-   Healthcare Professional Data Model \[sn\_hc\_professional\]
-   Admin Experience Framework \[sn\_ace\]
-   Journey Accelerator \[sn\_ja\]
-   Professional Data Model \[sn\_professional\]
-   Employee Center \[sn\_ex\_sp\]
-   Employee Center Core \[sn\_hr\_sp\]
-   Employee Experience Foundation \[sn\_ex\_emp\_fd\]
-   Human Resources Scoped App: Lifecycle Events for Enterprise \[com.sn\_hr\_lifecycle\_ent\]
-   Signature Pad \[com.snc.signaturepad\]
-   Scoped Application Restricted Caller Access \[com.glide.scope.access.restricted\_caller\]
-   Employee Service Center \[com.sn\_hr\_service\_portal\]
