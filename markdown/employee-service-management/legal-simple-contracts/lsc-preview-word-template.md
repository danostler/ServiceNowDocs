# Preview a contract document for a Word document template

View the final form of a contract document with the content controls replaced with the values of the variables or return values of scripts.

## Before you begin

Role required: sn\_lg\_contracts.contracts\_config

## Procedure

1.  Navigate to **All** &gt; **Legal Administration** &gt; **Simple Contracts Admin** &gt; **Contract Templates**.

2.  Open a contract template.

3.  Select **Preview**.

4.  In the Preview Document window, select the Lookup using list icon \[Omitted image "lookup-using-list.png"\] Alt text: Lookup using list icon

5.  Select a legal contract request from the list for which you want to generate a sample contract document.

6.  Select **Generate Preview** and close the Preview document pop-up window.

7.  In the **Manage Attachments** field, select **View** to view the document.

8.  Select the document link to download it.

9.  Modify the contract template if you want to make any changes.
