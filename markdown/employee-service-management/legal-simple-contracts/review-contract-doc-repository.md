# Preview and download a signed contract document

Preview and download a signed contract document.

## Before you begin

Role required: sn\_lg\_ops.legal\_user

## About this task

When all signatories have signed a contract document, it is considered the final contract document and stored in the Legal Contract \[sn\_lg\_contracts\_repository\] table.

Access to a contract document is based on the following user roles and conditions:

-   A Contracts Fulfiller can create, view, and update contract documents.
-   Only a Contracts Administrator can delete contract documents.
-   Requesters can view only contract documents for which they submitted the legal request.
-   Users can view only contract documents for which they are included on the watch list of the corresponding legal requests.

## Procedure

1.  Navigate to **All** &gt; **Legal Request** &gt; **Legal Counsel Center**.

2.  Select the list icon \(\[Omitted image "lsd-lcc-list-icon.png"\] Alt text: List icon.\).

3.  In the **Lists** tab, under **Simple Contracts**, select **Legal Contracts** .

4.  In the list of legal contracts, select a company name to open the contract.

5.  Review the contract document and associated request details.

6.  Download the contract document.
