# Configure the Contractor Service Center portal theme

A theme is a collection of one or more style sheets \(CSS files\) that define a consistent look for a set of pages. Configure the default portal theme to apply your own theme to suit your branding and improve the visual design of the portal pages.

## Before you begin

Role required: admin

## About this task

Contractor Service Center application ships a default **CSC theme** that has the best visual design experience. The CSC theme provides a consistent experience across all portal pages.

## Procedure

1.  Navigate to **All** &gt; **Service Portal** &gt; **Themes**.

2.  Search for `CSC Theme` to use the default theme.

    **Note:**

    -   Use the default theme for a better visual and usability experience. However, you can select other themes to suit your needs.
    -   Remove the customized portal theme before an upgrade to see any default theme that might be released with a future version of Contractor Service Center.
3.  Configure the theme content to suit your requirement.

    1.  Navigate to **Service Portal** &gt; **Portals**.

    2.  Select **Contractor Service Center** to open the record.

    3.  Select **here** on the note to edit the record.

    4.  Select the **Preview this record** icon \[Omitted image "icon-preview.png"\] Alt text: preview icon for the **Theme** field and select **Open record**.

    5.  Select **here** on the note to edit the record.

    6.  Edit the **CSS variables** field to customize the text and the background colors of your portal.

        -   To create a portal theme and for a description of the field values to create a portal theme, see Create a portal theme.

5.  Select **Update**.
