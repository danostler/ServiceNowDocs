# Contractor service portal menu item form

Form field description for contractor service portal menu item.

<table id="table_ush_5nt_yfc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Menu

</td><td>

Name of the menu selected.

</td></tr><tr><td>

Application

</td><td>

Name of the application scope.

</td></tr><tr><td>

Title

</td><td>

Title for the new menu item.

</td></tr><tr><td>

Subtitle

</td><td>

Short description of the title for the new menu item.

</td></tr><tr><td>

Icon type

</td><td>

Indicates the type of menu item that is displayed in the menu.-   Icon card- enables the **Font icon** field.
-   Image card- enables the **Image** field.

</td></tr><tr><td>

Font icon

</td><td>

String field specifying the CSS class from FontAwesome, Glyphs or other loaded icon font. This class determines the icon displayed on the card.

</td></tr><tr><td>

Image

</td><td>

Reference field pointing to the db\_image table, which stores system-managed images.

</td></tr><tr><td>

Type

</td><td>

Specifies the type of URL target that is activated when you click on the card.-   Page option enables the **page** field.
-   URL option enables the **HREF/URL** field.
-   Catalog item option enables the **Catalog item** field

</td></tr><tr><td>

Page

</td><td>

Reference field to a service portal page.

</td></tr><tr><td>

HREF/URL

</td><td>

String field to add any URL.

</td></tr><tr><td>

Catalog item

</td><td>

Reference field to the catalog items table.

</td></tr><tr><td>

Order

</td><td>

Order in which the menu item appears on the portal.

</td></tr><tr><td>

Active

</td><td>

Active is checked by default for new instances and applies to the portals in the system.
