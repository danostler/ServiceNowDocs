# New member form

Field descriptions to add a new member to a group mapped to a location.

|Field|Description|
|-----|-----------|
|User ID|Unique identifier for the member.|
|First name|First name of the member.|
|Last name|Last name of the member.|
|Title|Member's job title. Search from predefined options.|
|Department|Department name to which the member belongs.|
|Password reset|Select this check box to require the member to change the password during the first login.|
|Locked out|Select this check box to lock the member out of the instance and terminate all their active sessions.|
|Active|Select this check box to make this member active.|
|Web service access only|Select this check box to designate this member as a non-interactive user.|
|Internal Integration User|Select this check box to designate this member as a service account.|
|Email|Member email address.|
|Language|Preferred language for the member.|
|Calendar integration|Choose the calendar integration option, such as outlook.|
|Time zone|Time zone in which the member is located.|
|Date format|Date format for the member's account.|
|Business phone|Member's business phone number.|
|Mobile phone|Member's mobile phone number|
|Photo|Attach a photo of the member, if appropriate.|
