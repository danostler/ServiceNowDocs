# Critical event portal

On-site responders use the critical event portal in Critical Event Management to conduct roll calls, coordinate individuals to the assembly point, and gain a complete view of the emergency.

When a critical event occurs, the on-site responders receive an email from an agent notifying them about the emergency at their location. This email contains a link to log in to the critical event portal.

In the portal, you can view the active critical events where you're a member of the responder group for the location of the event. Select the **Show Details** button on an active critical event to conduct roll calls, account for the impacted people, and view emergency details.

-   Conducting roll call

    You can view the list of impacted people in your location, manually add more people to the list, such as a visitor from another location, and set each person as **Needs Help** or **Accounted** using the three dot menu. For more details, see [Conduct a roll call as an on-site responder](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/critical-event-management/conduct-roll-call.md).

-   Emergency info panel

    The panel displays a summary of the emergency such as the state and severity of the event, the event category, the number of impacted people, and others.


When all impacted people in a critical event are accounted for, the on-site responder receives a notification from the agent to end the roll call.

On-site responders can use the critical event portal on a desktop, tablet, and mobile device.
