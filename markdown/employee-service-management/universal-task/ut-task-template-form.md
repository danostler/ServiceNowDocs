# Universal Task template form

Use the Universal Task template form to create a Universal Task template.

<table id="table_hhn_1my_s4b"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Name of this template.

</td></tr><tr><td>

Parent table

</td><td>

Table that this template applies to. By default, the template is applicable to all tables.

</td></tr><tr><td>

Short description

</td><td>

Description of the template. **Note:** Adding content to this field does not add that content to the **Short description** field of the forms that use this template.

</td></tr><tr><td>

Template

</td><td>

Content that automatically populates records that are based on this template. Select a field from the specified table in the left column. Then enter the data to automatically populate in the right column.

</td></tr><tr><td>

Application

</td><td>

Field that is automatically set based on the scope of your organization.

</td></tr><tr><td>

Active

</td><td>

Option for making the template available for use. A template must be active to be used.
