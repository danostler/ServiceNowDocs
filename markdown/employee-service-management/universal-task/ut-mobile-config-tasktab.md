# Display the universal tasks for a request in Now Mobile

Display all the universal tasks that are assigned for a particular request by simply configuring the **Tasks** tab in Now Mobile.

## Before you begin

Role required: admin

## Procedure

1.  In the navigation panel, enter `sys_sg_form_segment.list`.

2.  In the Form Screen Segments page, click **New**.

3.  Select the service-specific **My Request** form screen.

    To find your service-specific **My Request** form screen, select a **My request** form screen and then preview the record.

4.  In the embedded screen field, select **Tasks**.

5.  In the **Order** field, enter a number to indicate the order that you want the tab to appear in.

6.  Click **Update**.
