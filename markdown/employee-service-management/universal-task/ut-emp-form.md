For more information configuring a parent table, see [Configure Universal Task for your service](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/universal-task/config-ut-for-service.md).**Note:**

-   Only the tables that are configured for Universal Task with **Collect Employee Input** task type can be selected.
-   You can select a parent table only if you have write access to it.
