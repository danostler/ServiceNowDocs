# Configure Universal Task for your service

Configure the Universal Task application for your service so that you can create and assign tasks to employees.

## Before you begin

Role required: sn\_uni\_task.admin

## Procedure

1.  Navigate to **All** &gt; **Universal Task** &gt; **Administration** &gt; **Universal Task Configuration**.

2.  Click **New**.

3.  On the form, fill in the fields.

    For more information on form fields and descriptions, see [Universal Task Configuration form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/universal-task/unitask-config-form.md).

4.  Click **Submit**.
