# Upload the manifest file for a workplace employee

Upload the manifest file \(add-in for Microsoft Outlook\) for a workplace employee \(single user\). Employees can create workplace reservations using the Microsoft Outlook add-in file.

## Before you begin

Activated the add-in for Microsoft Outlook. For more information, see [Activate the add-in for Microsoft Outlook](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/workplace-reservation-management/add-outlook-addin.md).

Role required: admin

## Procedure

1.  Open Microsoft Outlook mailbox.

2.  Select an event in Outlook calendar.

3.  From the Tools menu on the Outlook ribbon, select **Get Add-ins**.

4.  In the Custom add-ins section, select **Add a custom add-in** &gt; **Add from file**.
