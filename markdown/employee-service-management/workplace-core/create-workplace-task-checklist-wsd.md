# Create a workplace task checklist

You can create a unique checklist for each workplace task template in Workplace Core. You can also quickly create a checklist from an existing checklist template.

## Before you begin

Role required: sn\_wsd\_core.admin or sn\_wsd\_core.workplace\_manager

## About this task

Ensure that you have the following:

-   A workplace task template already created in the application, if you are adding a checklist for a workplace task template.
-   A workplace task already created in the application, if you are adding a checklist for a workplace task.

You can also create checklists for individual workplace task records.

## Procedure

1.  Create a checklist to be included in all tasks using the same template or add a checklist to a specific task.

5.  Delete unwanted checklist items by selecting the delete item icon \(\[Omitted image "delete-checklist-icon.png"\] Alt text: delete item icon\).

6.  Change the order of checklist items by selecting the drag icon \(\[Omitted image "drag-checklist-icon.png"\] Alt text: drag icon\) and dragging the item to a different position in the list.

7.  Select **Update**.


## Result

-   If you created the checklist for a workplace task template, any task created using this workplace template will also contain this checklist.
-   If you created the checklist in a workplace task form, the checklist will be part of only that task.
