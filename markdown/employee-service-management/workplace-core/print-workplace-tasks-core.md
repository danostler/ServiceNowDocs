# Print workplace tasks

Workplace coordinators can print the workplace task details and the associated workplace activities that are created and scheduled in the workplace for each shift. The task reports contain reservation details, description, services, and all other relevant information for effective task management and resolution.

## Before you begin

Role required: sn\_wsd\_core.task\_reader

**Note:** Enable the Print task feature by installing the Document Templates plugin.

## Procedure

1.  Navigate to **All** &gt; **Workplace Core** &gt; **Workplace Tasks**.

2.  To view all the workplace tasks, select **All**.

3.  Select any workplace task to view the details.

4.  Select **Print task**.

    The downloaded document displays the workplace task and the associated service details.

    **Note:** The Print task button is visible to the sn\_wsd\_case.case\_reader role only.
