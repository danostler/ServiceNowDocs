# Manage QR codes

View, update, or print the already generated QR codes in the application. Before printing, you can change the conditions of already generated QR codes if required.

## Before you begin

Role required: sn\_wsd\_core.admin or sn\_wsd\_core.workplace\_manager

## Procedure

1.  Navigate to **All** &gt; **Workplace Core** &gt; **QR code management** &gt; **Manage QR codes**.

2.  To generate a new QR code, click **New**.

    For more information, refer to [Generate and print QR code for a space or room](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown).

3.  Select the QR code that you want to view or update.

4.  Do any of the following actions with the QR code:

    -   To update the QR code, change any field and click **Update**.
    -   To print the generated QR code, click **Print QR Codes**, and then click **download** to download the file.

        **Note:** The printed QR code is generated with the same file name. The last generated PDF file is replaced with the latest file.

    -   To delete the QR code, click **Delete**.
