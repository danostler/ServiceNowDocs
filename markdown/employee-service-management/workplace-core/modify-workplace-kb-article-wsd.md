# Modify a workplace knowledge base article

Update a knowledge base article at any time. Retire a published article or republish a retired article.

## Before you begin

Role required: sn\_wsd\_core.admin or sn\_wsd\_core.kb\_writer

## About this task

You can perform the following actions on a knowledge base article:

-   Publish the article to make it available to employees.
-   Update the article at any time if necessary.
-   Retire the article if it is no longer necessary.
-   Republish a retired article if necessary.
-   Delete an article.

A knowledge base article undergoes the following workflow when it is created:

-   Draft
-   Review
-   Published
-   Pending retirement
-   Retired

## Procedure

1.  Navigate to **All** &gt; **Workplace Safety Management** &gt; **Knowledge management**.

2.  Select **All articles**.

    You can also select any of the following options to find your knowledge base article.

    -   **Published articles**
    -   **Unpublished articles**
    -   **Retired articles**
3.  Select the knowledge base article that you want to modify.

4.  Edit the fields that you want to change and select **Update**.

5.  To retire, republish or a delete the knowledge base article, select any of the following options on the top right or at the end of the form.

    |Option|Description|
    |------|-----------|
    |**Update**|Select this option to update any changes made to the knowledge base article.|
    |**Retire**|Select this option to remove the knowledge base article from the portal.|
    |**Republish**|Select this option to republish a retired article.|
    |**Delete**|Select this option to delete the article from the application.|


## Result

The knowledge base article is modified with the changes.
