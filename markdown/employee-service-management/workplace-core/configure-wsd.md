# Configure Workplace Core

Install Workplace Core application. Provide your workplace data to configure spreadsheets and import workplace data. Modify your existing spreadsheets. Upload floor plans to provide floor maps for a selected floor.

Workplace Core administrators can install and configure the Workplace Core and application to enable workplace services for employees. It covers installation of Workplace Core application, preparing and importing workplace data, and enabling supporting capabilities that enable employees to interact with workplace services effectively. Administrators can set up workplace locations, spaces, and upload floor plans.

Workplace Service Delivery administrators can configure the following:

-   Workplace Core application and dependent plugins, if necessary.
-   Create campus, buildings, floors, and spaces.
-   Upload and maintain workplace floor plans.
-   Enable Now Assist in Virtual Agent to support conversational workplace services.

Installing Workplace Core sets up the Workplace Core data model and Workplace Service Delivery capabilities. After installation, administrators can create campuses, buildings, floors, and spaces for employees to make reservations. Workplace Core enables reservations, space recommendations, and workplace services to employees.

Workplace data can be created and maintained directly in the system or imported in bulk using configured spreadsheets, maintaining consistency and accuracy. Visual floor plans can also be uploaded and updated to reflect the physical layout of the workplace. This enables employees to view and select spaces using floor maps.

Enabling Virtual Agent provides employees with a conversational interface for workplace interactions, improving accessibility and self service capabilities.
