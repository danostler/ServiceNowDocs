# Configure a new legal hold notice template

Configure a legal hold notice template to align with your organizational requirements.

## Before you begin

Role required: sn\_lg\_ops.legal\_config

## Procedure

1.  Navigate to **All** &gt; **Document Templates** &gt; **All Document Template**.

2.  Select **New**.

3.  Select **HTML Document Template**.

4.  On the form, fill in the fields.

<table id="table_bw2_bgx_rfc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Name of the template.

</td></tr><tr><td>

Application

</td><td>

The application to which the template belongs, which in this case is Legal Hold Notification.**Legal Hold Notification**.

</td></tr><tr><td>

Table

</td><td>

Table associated with the type of template, which in this case is sn\_lg\_matter\_task. The table determines the available variables that can be used. **Note:** The tables that are listed are based on the application scope that is selected.

</td></tr></tbody>
</table>5.  Fill in the remaining fields as needed.

    For more information, see [Configure an HTML document template](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/hr-service-delivery/configure-HTML-doc-template.md).

6.  Select **Publish**.


## Result

A new legal hold notice template is created.
