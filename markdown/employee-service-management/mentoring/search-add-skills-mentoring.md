# Search and add skills

As a mentee, search for a skill name or a keyword to find a mentor who specialises in that skill to help you.

## Before you begin

Enroll as a mentee. For more information, see [Enroll as a mentor or mentee](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/mentoring/enroll-mentoring.md)

Role required: none

## Procedure

1.  On the mentee preferences wizard, answer the required questions.

2.  Use the **Search skills** option to add skills that a mentor can help you with.

3.  On the **Search for more skills** wizard, specify the proficiency level from one of the following using the slide bar.

    -   Unrated
    -   Novice
    -   Intermediate
    -   Advanced
    -   Expert
4.  Select **Add Skills**.


## Result

Selected skills are added to current skill preferences. You can go ahead and complete the mentee preferences wizard after adding skills.
