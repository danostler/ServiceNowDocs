# Add message for users during health screening

Add an optional custom message for people to read during their health screenings with additional information or guidelines.

## Before you begin

Role required: sn\_imt\_monitoring.monitoring\_admin

## Procedure

1.  Navigate to **All** &gt; **Employee Health Screening** &gt; **Information Message**.

2.  Click **New**.

3.  Enter a name for the message in the **Name** field.

4.  Select the users you want to receive the message in the **Audience** field.

    For example, you might have a message or guidelines specifically for users who work in the IT department.

5.  Provide the text for the message in the **Message** field.

    For example, you might want to advise users to remain in a certain location in the building.

6.  Click **Submit**.


## Result

Users in the selected audience will see this message when they complete their health screening.
