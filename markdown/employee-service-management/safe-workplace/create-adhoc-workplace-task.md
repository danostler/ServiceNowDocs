# Create an ad hoc workplace task

Create an ad hoc task in Workplace Core for unplanned activities. For example, if you notice that something isn't clean in an area or want the area to be cleaned again, you can create an ad hoc workplace task to get the cleaning done immediately.

## Before you begin

Role required: sn\_wsd\_core.admin or sn\_wsd\_core.workplace\_manager

## About this task

You can also access the workplace task form by navigating to **Workplace Core** &gt; **Workplace Tasks** &gt; **Create New**.

## Procedure

1.  Navigate to **All** &gt; **Self-Service** &gt; **Employee Center** &gt; **Workplace services** &gt; **Browse Workplace Services**.

2.  Select **Add a task**.

3.  On the form, fill in the fields.

5.  If you accessed the form from Workplace Service Portal, on the verification page, you can create another task or view this task record in a list.

    |Option|Action|
    |------|------|
    |**Create another task**|Select **Add another task**.|
    |**View the task record in a list**|Select **View all tasks**.|

6.  Add a checklist to the task to track the actions required to complete it.

    To add the checklist, open the task by navigating to **Workplace Core** &gt; **Workplace Tasks** &gt; **All**. For more information, see [Create a workplace task checklist](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-core/create-workplace-task-checklist-wsd.md).


## Result

The ad hoc task is added to the list of all tasks that you can view by navigating to **Workplace Core** &gt; **Workplace Tasks** &gt; **All**.
