# Assign delegated developer role to Universal Request administrator

For a Universal Request administrator \[sn\_uni\_req.ur\_admin\] to perform some platform duties, the delegated developer role must be added.

## Before you begin

Role required: admin with sn\_uni\_req.ur\_admin

-   **Universal Request administrator with delegated developer role**

    The Universal Request administrator with the delegated developer role can:

    -   Add and update script includes.
    -   Add and update ACLs.
    -   Configure forms.
    -   Configure related lists.

## Procedure

1.  Log in as admin.

2.  From the Filter navigator, enter **sys\_store\_app.list**.

    The **Store Application** list appears.

3.  Select **Universal Request**.

4.  Under **Related Links**, select **Manage Developers**.

5.  Ensure **Developers** is highlighted.

6.  Enter the name of the Universal Request administrator.

7.  Select all items \(green bar moves to the left\) to grant permission.

8.  Select **Save**.

9.  Log out and log back in to ensure that the changes take effect.
