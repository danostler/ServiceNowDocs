# Mapping configuration form

Use the **Mapping Configuration** form to configure your fields and field mapping for cross-departmental transfers.

|Field|Description|
|-----|-----------|
|Name|Enter the name of your mapping configuration|
|Active|Indicates if the configuration for the mapping is active or not|
|Application|The application scope that the service belongs to|
|Table|Select the service or department table you want to map|
|**Transfer Information**|
|Transfer template|Select the template that you want to populate for the ticket transfer.|
|**Associated Ticket Information**|
|Associated ticket template|Select the template that you want to populate for the associated ticket creation.|
|**Create UR Information**|
|Create UR template|Select the template that you want to apply to the universal request while a department ticket is created.|
