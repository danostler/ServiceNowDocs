# Enable Natural Language Understanding model for Universal Request topic

Apply the Natural Language Understanding vocabulary in the Universal Request application to help the system to understand human-expressed language.

## Before you begin

Role required: admin

To use the Universal Request topic for your chatbot conversations, select and enable the Natural Language Understanding \(NLU\) model. For more information, see .

## Procedure

1.  Navigate to **All** &gt; **Collaboration** &gt; **Virtual Agent** &gt; **General Settings**.

2.  On the **NLU Settings** tab, select **Enable NLU in Virtual Agent** to make it active.

3.  Select `ServiceNow NLU` in the **NLU service provider** drop-down list.

4.  Select **Save**.
