# Service Configuration form

Use the **Service Configuration** form to register your service to use the Universal Request application.

|Field|Definition|
|-----|----------|
|Service Name|Enter the name for your service.|
|Service Table|Select the application table that you want to use from the list.|
|Mapping Configuration|Select the mapping configuration that is defined for your service.|
|Application|The application scope that the service belongs to.|
|Service Set|Select the service set that your service belongs to.|
|Restrict requests|Select to mark all the requests created for the service as restricted.|
