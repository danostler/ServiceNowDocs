# Use the guided tour in Universal Request

Use the guided tour to navigate a sequence of interactive steps that provides an overview of Universal Request cases.

## Before you begin

Role required: sn\_uni\_req.universal\_request\_write, sn\_uni\_req.routing\_agent, sn\_uni\_req.sensitiveinfo\_agent

When you select a Universal Request case for the first time, the guided tour starts automatically.

Use this procedure the start the guided tour manually.

## Procedure

1.  Navigate to **All** &gt; **HR Case Management** &gt; **Agent Workspace for HR Case Management**.

2.  Go to **Lists** &gt; **Universal Request**.

3.  Select a universal request in any given categories.

4.  Select the **Show help** icon \(\[Omitted image "help\_question\_mark\_icon.png"\] Alt text: Show help icon\).​

5.  Select **Take a tour**.

6.  Select **Begin tour** in the Guided Tour pop-up window.

7.  Read the call outs to understand the interface of a Universal Request case.

8.  Select **Next** to progress through the tour.

    To end the tour at any point, select the **End Tour** option in the **Show help** menu option or select the **Close** icon in the pop-up window and confirm to end the tour.

9.  Select **Done** to complete the tour.
