# Application states for the applicant

An overview of the application states the way the applicants see it.

|State|Description|
|-----|-----------|
|Application submitted|The applicant has submitted the application.|
|In review|The application is in the review stage.|
|Interview|The applicant is in the interview process.|
|Offer received|The applicant has received an offer.|
|offer signed|The applicant has accepted the offer and has signed the offer letter.|
|Hired|The applicant is hired.|
|Not selected|The applicant isn’t selected for the job requisition.|
|Withdrawn|The applicant has withdrawn an application.|
