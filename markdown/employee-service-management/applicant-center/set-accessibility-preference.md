# Set accessibility preference

Enable keyboard accessibility for your profile to navigate through the applicant center portal using keyboard tabs.

## Before you begin

Role required: sn\_ta\_hiring\_core.external\_applicant or sn\_ta\_hiring\_core.internal\_applicant

## About this task

Users can tab through the platform UI without enabling accessibility. However, without accessibility enabled, tabbing goes from field to field and skips other items on a form, such as reference icons. For details, see [Using accessibility features](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/platform-user-interface/configure-user-experiences/keyboard-accessibility.md).

## Procedure

1.  Navigate to the Applicant Center.

2.  Select the avatar and select **Profile**.

3.  Turn on the Accessibility enabled toggle switch.
