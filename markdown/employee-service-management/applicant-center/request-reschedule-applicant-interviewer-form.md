# Request reschedule form

The request reschedule form provides the field descriptions required for an applicant or interviewer when requesting a recruiter to reschedule an interview.

<table id="table_jzp_4rm_s2c"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Select reason for reschedule

</td><td>

Reason you're requesting to reschedule the interview for.

</td></tr><tr><td>

Manually share slots

</td><td>

Option to manually add slots and share with recruiter.

</td></tr><tr><td>

Use my calendar

</td><td>

Option to enable recruiters to schedule interviews based on your calendar availability. **Note:** This option is available only for interviewers.

</td></tr><tr><td>

Select time zone

</td><td>

Time zone for which you're suggesting time slots.

</td></tr><tr><td>

Start date and time

</td><td>

Earliest date and time that you’re available for interviews. This field appears only when **Manually share slots** is selected.

</td></tr><tr><td>

End date and time

</td><td>

Last date and time of the period until which you're available. This field appears only when **Manually share slots** is selected.

</td></tr><tr><td>

All day

</td><td>

Option to share your availability for an entire day. The end date is automatically set to the start date that you select. This field appears only when **Manually share slots** is selected.

</td></tr><tr><td>

Recurring

</td><td>

Option to share your availability for recurring time period. This field appears only when **Manually share slots** is selected.

</td></tr><tr><td>

Repeat

</td><td>

Frequency of recurring availability you're sharing. The available options are **Daily**, **Weekly**, and **Monthly**. This field appears only when **Manually share slots** is selected.

</td></tr><tr><td>

Days

</td><td>

Days of the week when you're available. This field appears only when **Manually share slots** is selected, and then **Weekly** is selected from **Repeat**

</td></tr><tr><td>

Lasts

</td><td>

Select either **for** or **until** to specify until when your shared availability lasts. This field appears only when **Manually share slots** is selected

</td></tr><tr><td>

Number of days/weeks/months

</td><td>

Number of days/weeks/months you're sharing availability for. This field appears only when **Manually share slots** is selected, and then **for** is selected from **Lasts**.

</td></tr><tr><td>

Ends on this date

</td><td>

End date of the recurring availability you're sharing. This field appears only when **Manually share slots** is selected, and then **until** is selected from **Lasts**.

</td></tr><tr><td>

Add time slot

</td><td>

Add the selected time slots. This field appears only when **Manually share slots** is selected

</td></tr><tr><td>

Available time slots

</td><td>

Suggested time slots that recruiter can pick from when rescheduling the interview. You can edit or delete these time slots. This field appears only when **Manually share slots** is selected

</td></tr></tbody>
</table>
