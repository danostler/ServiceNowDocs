# Configure manager insights for Now Assist for HRSD

Configure the manager insights skill in Now Assist for HRSD to generate a summary of team data in Manager Hub.

## Before you begin

Role required: sn\_hr\_gen\_ai.admin

## Procedure

1.  Navigate to **All** &gt; **Now Assist Admin** &gt; **Skills**.

2.  Select **Employee**, then select **HRSD**.

3.  On the **Manager Insights** tile, select **Turn on**.

4.  In the Add user access section, select individuals or groups that can utilize this skill.

5.  In the role restrictions to skill section, select roles that have access to resources during skill execution.

6.  Select **Turn on**.
