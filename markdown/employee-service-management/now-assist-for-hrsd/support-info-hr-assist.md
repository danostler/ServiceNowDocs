# Supporting information for Now Assist for HRSD

Get a quick overview of the important information that is related to the Now Assist for HR Service Delivery \(HRSD\) application.

## Supported versions

Now Assist for HRSD is supported starting with Vancouver Patch 2.

## Supported language models

Use the Now LLM Service or any of the following supported third-party small or large language model with any Now Assist for HR Service Delivery \(HRSD\) skill or AI agent:

-   Azure OpenAI
-   AWS Anthropic
-   Google Gemini

Activate the Now Assist for HRSD store app \(sn\_hr\_gen\_ai\) to use the chat, case, and resolution skills.

The Now Assist for HRSD application has the following dependencies:

-   Human Resources Scoped App: Core \(com.sn\_hr\_core\)
-   Now Assist Platform \(com.sn.now.platform\)
-   NA-TD and NA-TA are dependent apps for NA-HR and should be installed for every NA-HR installation \(talent-core: sn\_egd\_core and interview-scheduling sn\_int\_schedule\)
