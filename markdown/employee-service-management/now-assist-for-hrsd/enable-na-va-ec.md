# Add Employee Center to the Now Assist in Virtual Agent display experience

Enable Now Assist in Virtual Agent to appear in the Employee Center so you can converse with AI agents during a process that's facilitated by an agentic workflow.

## Before you begin

Role required: virtual\_agent\_admin

## Procedure

1.  Navigate to **All** &gt; **Conversational Interfaces** &gt; **Assistants**.

    The Assistants page appears.

2.  Select **Manage assistants**.

3.  On the Now Assist in Virtual Agent \(default\) card, select **Edit**.

4.  In the Continue customizing this assistant widget, select **Go to display experiences**.

    The Display experiences section within the Settings page appears.

5.  Select **Employee Center** from the **Add portal** drop-down list.

    The Add chat experience dialog box appears.

6.  Select the **Enhanced chat** radio button.

7.  Select the **Allow the search bar to open into a full-page chat experience** check box.

8.  Select **Add**.

9.  Select **Save**.
