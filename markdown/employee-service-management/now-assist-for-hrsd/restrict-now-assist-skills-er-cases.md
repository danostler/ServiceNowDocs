# Restrict Now Assist capabilities for employee relations cases

Restrict the Now Assist capabilities for your employee relations cases and define the availability.

## Before you begin

-   Role required: sn\_hr\_gen\_ai.admin

## Procedure

1.  Navigate to **Now Assist Admin Console** &gt; **Employee**.

2.  Select the **Gen AI** feature that you want to restrict the behavior for.

3.  On the **Define Availability** tab, modify the condition and add **Task type is not Employee Relations Case**.
