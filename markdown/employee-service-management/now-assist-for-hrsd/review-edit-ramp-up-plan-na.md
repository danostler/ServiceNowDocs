# Review and edit onboarding ramp-up plans using Now Assist

Managers working for organizations using the Generate onboarding ramp-up plan agentic workflow can use Now Assist in Virtual Agent to review and edit personalized onboarding plans generated by a team of AI agents. Once the manager approves a plan that they reviewed, the corresponding plan is added to the new employee's onboarding journey.

## Before you begin

The Now Assist for HR Service Delivery \(HRSD\) \(sn\_hr\_gen\_ai\) plugin must be activated.

The Generate onboarding ramp-up plan agentic workflow must be activated and implemented into your organization's case or journey creation process.

The configuration tasks in the following topics must be completed:

-   [Set Onboarding ramp up trigger to use Employee Center portal](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/now-assist-for-hrsd/set-trigger-emp-center.md).
-   [Activate Onboarding ramp-up trigger](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/now-assist-for-hrsd/activate-trigger-rampup.md)
-   [Activate the associated Now Assist skills](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/now-assist-for-hrsd/turn-on-nowassist-skills.md)
-   [Add Employee Center to the Now Assist in Virtual Agent display experience](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/now-assist-for-hrsd/enable-na-va-ec.md)

Role required: manager

## About this task

A manager receives a notification from Now Assist in Virtual Agent after they access the Employee Center. The manager is prompted to review the ramp-up plan for a new employee who is onboarding in five days.

## Procedure

1.  Navigate to **All** &gt; **Self-Service** &gt; **Employee Center**.

    A notification from Now Assist in Virtual Agent appears.

2.  Select the Now Assist icon.

    Now Assist in Virtual Agent outputs a message that includes the following information:

    -   Name of the new employee
    -   Their start date
    -   The stages and tasks it generated to help the employee ramp up for their role
    -   A link to a draft of the onboarding plan
3.  Select the link provided by Now Assist in Virtual Agent to view the onboarding plan.

    The Journey details page appears with the stages created by the AI agent.

4.  Converse with Now Assist in Virtual Agent to convey the changes you want made to the ramp-up plan.

5.  Select the link provided by Now Assist in Virtual Agent to refresh the ramp-up plan with the changes you conveyed.

    The Journey details page is refreshed and reflects the changes you conveyed to the AI agent.

6.  Use Now Assist in Virtual Agent to convey that no further changes are needed after you're satisfied with the ramp-up plan you have reviewed.


## What to do next

You must publish the stages in the ramp-up plan that were added to the employee's onboarding journey. After you publish the stages that were added, the employee can view and complete the courses and tasks nested in the added stages.
