# Review offboarding knowledge transfer summaries using Now Assist

Departing employees working for organizations using the offboarding knowledge transfer plan generation agentic workflow can use Now Assist in Virtual Agent to review and approve AI-generated knowledge transfer summaries. Once the employee approves the summary, it’s shared with the manager and designated team members.

## Before you begin

The manager must have completed the knowledge transfer request for the departing employee. For more information about this process, see [Generate offboarding knowledge transfer plans using Now Assist](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/now-assist-for-hrsd/generate-knowledge-xfer-na.md).

Role required: employee

## About this task

When a manager completes a knowledge transfer request for a departing employee, the employee receives a notification from Now Assist in Virtual Agent after they access the Employee Center. The employee is prompted to review and approve the AI-generated knowledge transfer summary before it’s shared with the manager.

## Procedure

1.  Navigate to **All** &gt; **Self-Service** &gt; **Employee Center**.

    A notification from Now Assist in Virtual Agent appears.

2.  Select the Now Assist icon.

    Now Assist in Virtual Agent outputs a message that includes the following information:

    -   Name of the manager who requested the knowledge transfer
    -   Confirmation that a knowledge transfer has been requested
    -   A summary of the work produced by the departing employee for the specified time range, organized into categories with descriptions and links to view associated resources
    -   A link to view the knowledge transfer summary
    -   A prompt to confirm whether the knowledge transfer is complete so it can be shared with the manager
3.  Select the link provided by Now Assist in Virtual Agent to view the knowledge transfer summary.

    A knowledge transfer page for the departing employee appears and contains the following information:

    -   The time range for the knowledge transfer
    -   Key projects and resources organized by category, each with a description and associated resources
4.  Edit the information reflected in the knowledge transfer summary.

    |Option|Description|
    |------|-----------|
    |**Add a project**|Select the **Add a project** button to create a project with a title and description.|
    |**Edit a project**|Locate a project and select the corresponding **More actions** icon. Then, select **Edit** to update the project title or description.|
    |**Delete a project**|Locate a project and select the corresponding **More actions** icon. Then, select **Delete** to remove the project and its associated resources.|
    |**Add a resource**|Select the **Add a resource** button within a project to add a resource with the following information: resource name, type, URL, and an optional note.|
    |**Edit a resource**|Locate a resource and select the corresponding **More actions** icon. Then, select **Edit** to update the resource details.|
    |**Move a resource**|Locate a resource and select the corresponding **More actions** icon. Then, select **Move** to move the resource to a different project.|
    |**Delete a resource**|Locate a resource and select the corresponding **More actions** icon. Then, select **Delete** to remove the resource from the project.|

5.  In the Now Assist in Virtual Agent chat, enter `Yes` to confirm that the knowledge transfer is complete and ready to share with the manager.


## Result

The knowledge transfer summary is approved and shared with the manager. The manager is notified that the departing employee has completed the knowledge transfer review.

## What to do next

The manager can review the knowledge transfer summary approved by the departing employee and share it with successors who will assume their duties.
