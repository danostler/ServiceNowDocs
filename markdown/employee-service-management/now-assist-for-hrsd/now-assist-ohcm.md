# Now Assist for HRSD Virtual Agent topics

Use the Now Assist Virtual Agent topics in Employee Center to place requests to the configured HCM system, for example, view details of your time-off balance.

## Get time off balance

The Get time off balance Virtual Agent topic enables you to view your leave balance from the configured HCM system in Employee Center.

From Employee Center, open a chat window and type in key words, for example, leave balance. The Now Assist generative AI retrieves details from the HCM system and displays your time off balance.

## Get User Paychecks

The Get User Paychecks Virtual Agent topic enables you to view your payslip details in Employee Center.

From Employee Center, open a chat window and type in key words, for example, get payslips. After you provide the required details such as start date and end date, your payslip summary, in the form of pay period, gross pay, deductions, and net pay are displayed. A detailed breakdown of deductions is also displayed.

## Lookup Worker Profile

The Lookup Worker Profile Virtual Agent topic enables you to view the profile of a coworker.

From Employee Center, open a chat window and type in key words, for example, worker profile. After you have specified the details of coworkers, worker profile details of one or more users are displayed.

## Request Timeoff

The Request Timeoff Virtual Agent topic enables you to apply leave from Employee Center to the configured HCM system.

From Employee Center, open a chat window and type in key words, for example, request time off. After you have specified required details such as start date and end date, the time off request is sent to the configured HCM system.

## Add Expense

The Add Expense Virtual Agent topic enables you to submit an expense report from Employee Center to configured HCM system.

From Employee Center, open a chat window and type in key words, for example, add expense. After you have specified the required details such as type of expense, expense currency, expense amount, the expense report is submitted to the configured HCM system.

The Get Holiday Calendar Virtual Agent topic enables you to view your holiday calendar from the configured HCM system in Employee Center.

From Employee Center, open a chat window and type in key words, for example, holiday calendar. After specifying the calendar year, the holiday calendar appears in Employee Center.

## Get Total Rewards

The Get Total Rewards Virtual Agent topic enables you to view your rewards from the configured HCM system in Employee Center.

From Employee Center, open a chat window and type in key words, for example, rewards. Your rewards are displayed in Employee Center.

## Lookup Feedback

The Lookup Feedback Virtual Agent topic enables you to view feedback you have received from other employees in Employee Center from the configured HCM system.

From Employee Center, open a chat window and type in key words, for example, Lookup Feedback. Your feedback, if available, is displayed.

\[Omitted image "lookup-feedback.png"\] Alt text: Lookup feedback

## Lookup Benefit Enrolments

The Lookup Benefit Enrolments Virtual Agent topic enables you to view your benefit programs such as medical, 401\(k\) and so on, in Employee Center from the configured HCM system.

From Employee Center, open a chat window and type in key words, for example, Lookup Benefit Enrolments. Your benefit programs, if available, are displayed.

## Update personal details

You can update personal details in Virtual Agent using one of the following topics in Employee Center from the configured HCM system.

-   Update Address
-   Update Blood Type
-   Update Date of Birth
-   Update Email Address
-   Update Phone Number
