# SharePoint Online Search Connector scheduled jobs

Create a scheduled job to fetch each entity such as sites, drives, file metadata, and file content.

You can change the frequency of the scheduled jobs. The list of the scheduled jobs is as follows:

-   SharePoint Online Search Connector: Ingest Sites
-   SharePoint Online Search Connector: Ingest Drives
-   SharePoint Online Search Connector: Ingest Files
-   SharePoint Online Search Connector: Index Content
-   SharePoint Online Search Connector: Delete Content
-   SharePoint Online Search Connector: Renew Subscription
-   SharePoint Online Search Connector: Check Connector Status
-   SharePoint Online Search Connector: Incremental Pull
-   SharePoint Online Search Connector: Process Notification
-   SharePoint Online Search Connector: Ingest Permissions
-   SharePoint Online Search Connector: Insert New Users

**Note:** The default frequency for the scheduled jobs is set to maximize the performance.
