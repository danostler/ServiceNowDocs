# Choice new record form

Field description to add a new choice to create a new permit type.

|Field|Description|
|-----|-----------|
|Table|Field to select the table to create a new permit type.|
|Element|The column for which the choice is created.|
|Language|Language code for the label.|
|Sequence|This field determines the order in which the choice appears.|
|Inactive|Option to make the choice inactive.|
|Label|Label to appear in the choice list.|
|Value|Value related to the label.|
|Dependent value|Leave empty.|
|Hint|Leave empty.|
