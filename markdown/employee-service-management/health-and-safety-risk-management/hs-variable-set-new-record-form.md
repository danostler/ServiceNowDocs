# Variable set new record form

Field description to create a new variable set record.

|Field|Description|
|-----|-----------|
|Title|Title of a variable set. Appears if the **Display title** check box is selected.|
|Internal name|Variable set name for internal use.|
|Order|Order number for the variable set.|
|Type|Type of the variable set.|
|Application|Applications that can use this variable set.|
|Display title|Adds a title and an expandable header to the right of the variable set.|
|Layout|Set the layout display.|
|Description|Description of the variable set.|
