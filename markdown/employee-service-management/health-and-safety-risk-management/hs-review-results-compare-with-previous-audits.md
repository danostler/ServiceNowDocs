# Record safety audit results and compare with previous audits

Record the audit result and outcomes and compare them with previous audits to identify trends and track progress over time.

## Before you begin

To compare results with previous audits, the current audit must belong to an audit schedule that has at least one completed predecessor audit.

Role required: sn\_hs\_rm.safety\_audit\_manager or sn\_hs\_rm.safety\_audit\_writer

## About this task

The **Results** tab brings together the outcome and all associated records for an audit in a single view. Use it to record the audit result, review findings, observations, actions, and risk assessments. And if this audit is part of a recurring schedule, you can also bring in records from previous audits in the same schedule to compare results over time.

## Procedure

1.  Navigate to **Workspaces** &gt; **Health and Safety Workspace**.

2.  Select the risk management icon \(\[Omitted image "icon-risk-assessment.png"\] Alt text: Risk assessment icon\).

3.  In the **Audits** list, select **All** and select the audit.

4.  Select the **Results** tab.

5.  In the **Audit result** card, select the result and enter any final remarks.

6.  Review findings, observations, and, actions, risk assessments for the audit in their respective cards.

    Each card lists the records associated with the current audit.

7.  Select **Save**.

8.  Compare the current and previous audit results.

    1.  Select **Compare previous audits**.

        This button appears only when this audit belongs to an audit schedule.

    2.  In the **Select audits to compare** dialog box, select one or more previous audits from the current schedule.

        -   You can select multiple audits to compare across several iterations at once.
        -   Only the completed predecessor audits from the audit schedule appear in this list to compare.
    3.  Select **Select**.

        The records from all selected previous audits are added to the four record cards alongside the current audit's records.

    4.  To filter the results, for example, by date range or location, use the filter controls in the cards.


## Result

-   The audit result is recorded and all related records are visible in a single view.
-   If a comparison was applied, you can see how findings, observations, actions, and risk assessments have changed across audit iterations. For example, a finding that was non-compliant in a previous audit may now appear as compliant, confirming that the issue was resolved.
