# Add an action from survey response in safety inspection through Employee center

Learn to add actions to the survey responses on a safety inspection or audit survey to access safety issues discovered during a safety walk.

## Before you begin

Role required: sn\_hs\_rm.safety\_inspection\_writer

## About this task

The safety agent assigned to the survey can create actions on the survey. The actions must first be added to the findings related item of a safety inspection record. For information on adding action to findings, see [Add an action for a safety inspection or audit](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/health-and-safety-risk-management/hs-add-action-safety-inspection.md). Actions can also be created on the Now Agent app in offline mode provided all the inspection records are cached before going offline. For more information, see [Add actions for safety inspections survey in Mobile Agent app](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/health-and-safety/add-actions-for-safety-inspections-survey-in-mobile-agent-app.md).

## Procedure

1.  Navigate to **Employee Center** &gt; **Health and Safety** and select the **View safety inspections** quick link.

2.  Select **Open Survey** for one of the inspection record.

    -   A survey widget with a **Get started** button appears.
    -   An **Action** widget appears with the list of available actions in findings related item of a safety inspection record.
3.  Select the **Get started** button on the survey widget.

    The survey contains sections, and each section includes questions. When the response is added for the questions in the first section, it’s available in the **Action** widget to create an action for.

4.  Respond to the questions on the survey and then select the **Create action** button in the **Action** widget.

    Action can be created for each question in all the sections.

5.  On the **Create action form**, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Section|The sections of the survey that you have added response for.|
    |Question|The questions that you responded to in the selected section of the survey|

    For more information to complete the action form, see [Health and Safety action form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/health-and-safety-core/hs-action-form.md).

6.  Select **Create**.

    -   A new action is created and can be viewed in the action widget next to the survey.
    -   Select more actions\[Omitted image "more-actions-icon.png"\] Alt text: More actions icon Icon to edit or delete the actions on the action widget.
7.  Select **Submit**.
