# Configure permit-to-work

Configure permit type, permit checklist, checklist type, and edit a permit form.

Permit-to-work can be configured by an admin. An admin must assign the following roles to the permit coordinator \[sn\_hs\_rm.permit\_coordinator\] to enable them to configure permit-to-work.

-   catalog\_admin
-   decision\_rule\_author
-   personalize\_choices
-   personalize\_dictionary
-   snc\_internal
-   sn\_hs\_rm.permit\_coordinator

-   **[Configure a permit-to-work request](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/health-and-safety-risk-management/hs-configure-request-permit-to-work.md)**  
Edit existing questions on a permit to work form or add new questions.
-   **[Configure a permit type](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/health-and-safety-risk-management/hs-configure-permit-type.md)**  
Create a permit type to categorize risk and safety requirements for different types of work.
-   **[Configure a permit checklist](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/health-and-safety-risk-management/hs-configure-permit-checklist.md)**  
Create a permit checklist and link it to the permit type of a permit-to-work request.
