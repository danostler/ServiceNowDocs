# Properties installed with Workplace Service Delivery for Mobile

Several properties are available with Workplace Service Delivery for Mobile.

<table id="table_dqg_vzt_prb"><thead><tr><th>

Property

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Proximity search limit for spaces to browse near a person using Now mobile \(sn\_wsd\_mobile.mobile\_proximity\_search\_limit\)

</td><td>

This property enables you to set the search limit when an employee searches for a workplace near another user.-   Type: integer
-   Default value: **5**

</td></tr><tr><td>

Enable Map view option for WSD Mobile Reservations \[sn\_wsd\_mobile.enable\_map\_view\_for\_mobile\_reservations\]

</td><td>

This property shows the **Toggle map view** option while making a reservation from the mobile app.

</td></tr></tbody>
</table>**Note:** These properties are derived from the system properties \[sys\_properties.list\] table.
