# Mobile Wayfinding Administration

Configure map providers to enable Workplace Service Delivery for Mobile users to get directions to another workplace easily.

Mobile wayfinding functionality is available for mobile users.

**Note:** Indoor Mapping spatial map data and objects can be integrated in Workplace Service Delivery and viewed using responsive mobile.

-   Create a custom Map provider. Configure Indoor mapping as your map providers so that directions are displayed from one location to another.

    For more information, see [Configure a custom Map provider](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/workplace-service-delivery-for-mobile/configure-custom-map-provider.md).

-   Configure a custom Map screen. Configure map screens for wayfinding.

    For more information, see [Configure a custom map screen](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/workplace-service-delivery-for-mobile/create-custom-map-screen.md).

-   Indoor Navigation User profile: Use an employee's workplace profile for wayfinding. The Get directions option appears based on these conditions.

    -   Employee must have a floor map configured Indoor Mapping or Mappedin.
    -   Indoor wayfinding-My Reservation: Get directions to a location from the reservation details page. The specified location must have a floor map defined to view the **Get Directions** option. For more information, see [Get directions to a workplace](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/workplace-service-delivery-for-mobile/get-directions-to-a-workplace.md).
    Employees must have an ongoing reservation. If there is no reservation, the location specified in the employee's workplace profile is used.
