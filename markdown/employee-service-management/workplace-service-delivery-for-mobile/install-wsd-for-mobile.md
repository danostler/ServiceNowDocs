# Install Workplace Service Delivery for Mobile

Install the Workplace Service Delivery for Mobile application from the ServiceNow® Store.

## Before you begin

Complete the following setup instructions.

1.  Navigate to **Subscription Management** &gt; **Subscriptions** in your instance. The list displays the subscriptions that your organization has purchased.
2.  Verify that the following plugins are activated:
    -   Workplace Core
    -   Workplace Reservation Management

Use the following details when required:

-   Name of the application: Workplace Service Delivery for Mobile
-   ID of the application: sn-wsd-mobile

Role required: admin

## Procedure

1.  Activate and install plugin for Workplace Service Delivery for Mobile from the ServiceNow® Store.

2.  After installing the application, navigate to **System Applications** &gt; **All Available Applications** &gt; **All**.

3.  Find the application using the filter criteria and search bar.

    Search for the application by its name Workplace Service Delivery for Mobile\) or ID \(app-wsd-mobile\).

4.  In the Application installation dialog box, review the application dependencies.

    Dependent plugins and applications are listed.
