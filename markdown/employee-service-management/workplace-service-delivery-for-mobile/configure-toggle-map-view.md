# Configure the Toggle map view property

Configure the Toggle map view option on the Now Mobile app app while reserving a space.

## Before you begin

Role required: sn\_wsd\_rsv.admin

## Procedure

1.  Navigate to **All** &gt; **Workplace Service Delivery for Mobile** &gt; **Properties**.

2.  Enable the edit options on the Mobile Properties page.

3.  From the **Enable Map view option for WSD Mobile Reservations** drop-down list, select the following options as required:

    1.  Disabled: The **Toggle Map view** option is not available while making a reservation.

        \[Omitted image "wsd-toggle-view-disabled.png"\] Alt text: Toggle map view set to Disabled.

    2.  Enabled: The **Toggle Map view** option is visible and not selected \(checked\).

        \[Omitted image "wsd-toggle-view-enabled.png"\] Alt text: Toggle map view set to Enabled.

    3.  Default: The **Toggle Map view** option is available and selected \(checked\).

        \[Omitted image "wsd-toggle-map-view-default.png"\] Alt text: Toggle map view set to Default.

    4.  Required: The **Toggle Map view** option is available but set to read-only.

        \[Omitted image "wsd-toggle-map-view-required.png"\] Alt text: Toggle map view set to Required.

        **Note:** The **Toggle Map view** is not shown in this case. When a user proceeds with this option, map visualization \(Map view\) is used. If there is no map available for a building or a floor, users are not able to select a space \(as users can only select a space using a floor map\).

    For more information, see [Enable map view to create reservations](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-service-delivery-for-mobile/enable-map-view-mobile-reservations.md) and [Make a reservation using the Now Mobile app](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-service-delivery-for-mobile/make-a-reservation-on-mobile.md).
