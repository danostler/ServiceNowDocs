# Download manifest files

Download the manifest files as part of configuring and integrating WSD for Microsoft places application in Microsoft Teams.

## Before you begin

Role required: admin

Ensure to register the WSD for Microsoft places application in the Microsoft Azure portal.

For more information about registering the WSD for Microsoft places application, refer to [Register the WSD for Microsoft places application](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/workplace-service-delivery-integration-with-microsoft-places/installing-wsd-ms-places.md)

## Procedure

1.  Navigate to **Teams** &gt; **ServiceNow for Teams**.

2.  Click **New** to create a registry.

3.  All the fields are generated, enter a **Short name** and **Full name** according to your requirements.

4.  Click **Submit**.

5.  Click **Next**.

6.  Under the **Configure Virtual agent** tab, select the first option **Configure a ServiceNow instance to your Microsoft Teams tenant**.

7.  Click **Next**.

8.  Enter the **Application ID** and **Directory \(client\) ID**.

9.  Enter the **Tab name** and click **Next**.

    **Application Registry record** is created.

10. Click **Download Manifest**.


## What to do next

Follow up with modifying the .json file from the manifest zip file.
