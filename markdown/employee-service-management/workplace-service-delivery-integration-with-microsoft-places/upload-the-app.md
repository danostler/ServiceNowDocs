# Upload the WSD for Microsoft places application

Upload the WSD for Microsoft places application in Microsoft Teams.

## Before you begin

Role required: admin

Ensure to modify the manifest file \(.json file\)

For more information about modifying the manifest file, refer to [Modify the manifest file](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/workplace-service-delivery-integration-with-microsoft-places/modify-the-manifest-file.md)

## Procedure

1.  Log in to Microsoft Teams with the email address.

2.  Select **+ Apps**.

3.  Select **Manage your apps**.

4.  Under **apps**, click **Upload an app**.

5.  Select **Upload a customised app**, browse for the updated zip file, and upload.

6.  Click **Add** to integrate WSD for Microsoft places application in Microsoft Teams.


## Result

The WSD for Microsoft places application is integrated with Microsoft Teams.
