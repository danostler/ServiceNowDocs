# Managing Workplace Move Management requests

As an admin, you can create or edit workplace services and record producers related to move requests. You can create or edit move tasks to fulfill a move request case.

The **Request a desk change** service is a default workplace service in the Workplace Service Portal. Employees can use this service to submit a move request.

A workplace service, **Request an employee's desk change** is by default available in the application. The service can be accessed only by the admin or the workplace manager.

To create a move-related workplace service, refer to [Create a Workplace service](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-case-management/create-workplace-service.md). To configure and link the workplace service to a record producer, refer to [Configure a Record producer](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-case-management/wsd-create-record-producer.md).

As a workplace manager, the application enables you to also implement the actions that are available in the Workplace Case Management and the Workplace Core.

For example, in the Workplace Case Management application, you can do the following:

-   Create a move-related workplace service and link it to a record producer. You can also create a record producer if required.
-   Configure Case and Task templates for workplaces cases.
-   Create child move cases and tasks if required.

And in the Workplace Core application, you can do the following:

-   Create knowledge base articles for employees and publish them.
-   View the knowledge base articles created in the application and modify an article whenever required.

You can also group multiple move requests under a move project and track their progress at a time using the Workplace Central Move Management workspace. For more information, refer to [Move Management](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-central/move-mgmt.md).

-   **[Create a move request on behalf of an employee](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown)**  
Create a move request on behalf of an employee from the application directly instead of using the Workplace Service Portal.
-   **[Raise a move request for a space change](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/workplace-case-management/request-desk-change.md)**  
Create a move request to relocate to another space. Specify why you want to move and when you want to move. You can select a space that is located in your current building or at a different building. You can also select a desk directly on the floor map.
-   **[Update a move request workplace service](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/update-move-request-workplace-service.md)**  
As a workplace manager, modify a move request that is created in the Workplace Case Management application. You can add move tasks, edit a task template, add child move cases, and more.
-   **[Update a move-related workplace service record producer](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/view-move-request-record-producer.md)**  
Update a move-related workplace service record producer. Link a workplace service to one or more record producers.
-   **[Create a move-related knowledge base article](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/create-move-related-kb-article.md)**  
Create a knowledge base article so that you can provide move-related information or notices to employees.
-   **[Review move conflicts](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/review-conflicts.md)**  
You can review the reason behind a move request conflict and decide how to resolve the conflict.
