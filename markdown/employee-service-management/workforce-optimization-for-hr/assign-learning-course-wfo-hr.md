# Assign a course item to agents

Assign a course item to one or more agents and add a due date for them to complete the course.

## Before you begin

Role required: sn\_hr\_wfo.admin

## About this task

You can analyze the course items that were most taken by your team members. You can also see which courses were created in the last five days.

**Note:** You can close the status of a course item for any of your team members that you directly or additionally manage by selecting **Mark as complete** for that course item.

## Procedure

1.  Navigate to **All** &gt; **Workforce Optimization for HR** &gt; **Manager Workspace**.

2.  Select the coaching icon \(\).

3.  Select the **Learning** tab.

4.  Select **Learning Courses** &gt; **All Courses**.

    -   You can search courses based on keywords using the **Search courses** field or narrow down the display of the courses based on skills using the **Filter courses by skill** field.
    -   In the course item cards, you can view details of each course, including skills associated with the course. If a course has more than one skill, you can select the count next to a skill and view all skills. To remove a skill, select a skill and select **Remove**.
    -   You can close the status of a course item for any of your team members that you directly or additionally manage and by selecting **Mark as complete** for that course item.​
5.  Select the assign icon for the course item you want to assign to agents.

6.  In the **Assign learning course** screen, assign the learning course to groups or team members.
