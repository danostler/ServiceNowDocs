# Create a learning task

Create learning tasks for agents to keep track of their learning activities.

## Before you begin

Role required: sn\_hr\_wfo.admin

## About this task

Set learning task completion due dates to include or exclude weekends using the **exclude\_weekends\_on\_learning\_task\_due\_date** system property.

## Procedure

1.  Navigate to **All** &gt; **Workforce Optimization for HR** &gt; **Manager Workspace**.

2.  Click the coaching icon.

3.  Click the **Learning** tab.

4.  Click **Learning Tasks** and select **All Tasks**.

5.  Create a learning task.

    1.  Click **New**.

    2.  On the form, fill in the fields:
