# Modify the re-approval trigger

Modify which fields can trigger an approval request if it is updated.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **System Properties**.

2.  Select changes\_require\_reapproval\_fields \(sn\_ta\_hiring\_core.changes\_require\_reapproval\_fields\).

3.  Update the **Value** field with the field names.

    The following fields, if updated after receiving an approval, can re-trigger an approval request, as available out of the base system.

    -   Compensation range
    -   Location flexibility
    -   Preferred level of seniority
    -   Number of headcount
