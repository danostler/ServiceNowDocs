# Configure the fields to be duplicated in a job requisition

Configure which fields are duplicated when a recruiter duplicates a job requisitions.

You can configure which fields are duplicated with sn\_ta\_recruiter.duplicate\_allowed\_fields \(sn\_ta\_recruiter.duplicate\_allowed\_fields\) system property.

You must have the system administrator role to do this task.
