# Application states for the recruiter

An overview of the application states the way the recruiters see it.

|State|Description|
|-----|-----------|
|New|The applicant has submitted the application and is in **New** state in the recruiter's queue.|
|In review|The application is in review.|
|Interviewing|The applicant is in the interview process.|
|Offer sent|The applicant has received an offer.|
|Offer accepted|The applicant has accepted the offer.|
|Hired|The applicant is hired.|
|Rejected|The application is rejected.|
|Withdrawn|The applicant has withdrawn the application.|
