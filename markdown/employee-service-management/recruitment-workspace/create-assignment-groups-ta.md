# Create assignment groups

Create assignment groups to allocate job requisitions efficiently among the recruiters.

Create assignment groups, as a recruiter, to allocate job requisitions to a designated group of recruiters.

The process of assignment might be automated or manual. It serves the following purpose:

-   Allocate a type of job requisition to a group.
-   Automate the process of job requisition assignment.
-   Decrease the time spent on determining which requisition should be assigned to which recruiter.

Set up the assignment groups to view data on the recruitment dashboard in the Recruitment workspace overview page.
