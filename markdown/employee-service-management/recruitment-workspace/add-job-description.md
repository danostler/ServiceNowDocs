# Add a job description to a requisition

Add a job description to a requisition, using job description templates.

## Before you begin

Role required: sn\_ta\_hiring\_core.recruiter

## Procedure

1.  Navigate to **Workspaces** &gt; **Recruitment workspace**.

2.  Open the job requisition where you want to add a job description.

3.  Select the **Job description** tab to open the **Job description** field.

4.  In the Templates section, select a job description template from the **My Templates** tab.

5.  Update the content if required, when it's populated in the job requisition.

6.  Select **Save**.

    The Job descriptions are auto-populated from the job profiles too.


## What to do next

Send the requisition for approval. For more information, see [Submit a job requisition for approval](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/recruitment-workspace/submit-for-approval-job-req.md).
