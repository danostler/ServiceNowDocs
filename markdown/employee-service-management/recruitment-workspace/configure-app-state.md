# Configure application state

Add new job application states according to your organizational needs.

## Before you begin

Role required: admin

## Procedure

1.  Open the job application table\(sn\_ta\_hiring\_core\_job\_application\).

2.  Right-click and select **Configure** &gt; **Table**.

3.  In the **Columns** tab, search for and select the State column.

4.  In the **Choices** tab, select **New**.

5.  In the **Label** field, enter the name of the new application state.

6.  In the **Value** field, enter a numerical value for the new application state.

    **Note:** The value must be a unique number that determines the order of the state and enables proper sequencing. For example, to add a new state between two existing ones, enter a number that falls between their respective values.

7.  Select **Submit**.
