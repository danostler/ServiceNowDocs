# Create and manage a safety inspection in Mobile Agent app

Create a safety inspection using the Mobile Agent app without scheduling an inspection to manage your workplace inspection.

## Before you begin

Role required: sn\_hs\_rm.safety\_inspection\_writer

## About this task

Easily create new inspection records. Assign these inspection records to yourself or to a safety agent from the assigned groups. From an inspection record open and complete a related survey that is assigned to you. Add new findings or actions to the findings as necessary on the Mobile Agent app.

## Procedure

1.  Open the Mobile Agent app on your mobile device.

2.  In the navigation bar, tap **Home**.

3.  Select **My inspections** from the quick links to open a list of inspections.

4.  Select **Create new inspection** to add a new inspection.

    Alternatively, edit or delete the existing inspection record as needed from the **Details** tab of the inspection record.

5.  On the form fill, in the details.

    For more information for field description, see [Inspection schedule form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/health-and-safety-risk-management/hs-inspection-schedule-form.md).

6.  Select **Submit**.

    For more information on managing a safety inspection, see [Create and manage a safety inspection](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/health-and-safety-risk-management/hs-create-manage-safety-inspection.md)
