# Using Now Assist for workplace queries

Use the Now Assist panel to interact with intelligent QnA agents that respond to workplace-related queries using structured data from a Knowledge Graph.

## Before you begin

Role required: sn\_wsd\_rsv.manager

## About this task

When you submit a query in the Now Assist panel, the system automatically determines the query type and routes it to the appropriate domain-specific AI agent. If a query involves reservation data, the system routes the questions to the workplace reservation QnA agent.

## Procedure

1.  Navigate to the Next Experience Unified Navigation page.

2.  Select the Now Assist icon \[Omitted image "icon-ai-sparkle.png"\] Alt text:.

3.  Enter your reservation-related query.
