# Enable Now Assist panel

Enable Now Assist panel to activate conversational capabilities, map user intents to workflows, and link agents to relevant knowledge graphs for reservations, connectors, and utilization.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **AI Agent Studio** &gt; **Overview**.

2.  Select the Overview tab.

3.  Select Workplace Advisor QnA Agentic workflow.

4.  On the form, navigate to Select channels and status.

5.  In the Engage via the Now Assist panel section, make sure the **Display** toggle is turned on.

6.  Select **Save and test**.
