# Respond to a request for clarification for a Legal Invention Disclosure request

Provide information requested by the patent reviewer or fulfiller to complete a Legal Invention Disclosure request for review.

## Before you begin

Role required: sn\_lg\_ops.legal\_user

## About this task

If you have submitted a Legal Invention Disclosure with missing or incomplete information, the fulfiller will request clarification and you will receive an email notification.

## Procedure

1.  Navigate to **All** &gt; **Employee Center**.

2.  Select **My Requests**.

3.  In the search field, enter the Legal Invention Disclosure request number from the email notification and press Enter.

4.  Open the request by selecting the request name.

5.  Provide the required information.

    -   If you are being asked for clarification of the information you provided, enter the response to the query in the **Type your message here** field and select **Post**.
    -   If a field value is incorrect, select **Actions** &gt; **Edit Request** and correct the values in the fields.
6.  Select **Submit For Review** to send the request back to the fulfiller with the updated response.


## Result

The request is submitted back to the fulfiller for review.
