# Begin a discussion from a Talent Feedback request

You can choose to begin a discussion regarding a talent feedback request using the Discuss option.

## Before you begin

Role required: manager

## Procedure

1.  Navigate to **All** &gt; **Talent Feedback** &gt; **Feedback requests**.

2.  Select the feedback request you want to start a discussion for.

3.  Select the **Discuss** option.

4.  On the Start a Sidebar discussion form, fill in the information
