# Collect feedback using an agent

Use the Employee feedback collection AI agent to collect feedback and track feedback requests.

## Before you begin

Role required: manager

## Procedure

1.  Navigate to **All** &gt; **Employee Center** &gt; **Your team** &gt; **Team members**.

2.  Select the Now Assist icon to start a conversation with the agent.

3.  Converse with Now Assist in Virtual Agent to create a feedback request.

4.  Select the employee who you'd like to collect feedback for from the drop-down options suggested by the agent.

5.  Choose the type of feedback you want to collect from the feedback templates suggested by the agent and click **Submit**.

    **Note:** The feedback templates are set by the admin. For more information, see [Create feedback template using survey designer](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/talent-feedback/create-template-feedback-ai.md)

6.  Select the **See preview** option to see a preview of the questions that will be asked in the template you selected and click **Submit**.

    **Note:**

    -   You can choose to skip to see a preview of the questions by selecting the **Skip preview** option.
    -   You can also confirm your selection after reviewing the questions in the template by selecting the **Use this template** option or **Choose different template** option.
7.  Choose potential collaborators based on your organization's approved sources as suggested by the agent.

8.  Provide a time frame for the feedback to be submitted by when prompted by the agent.

9.  View a preview of the feedback request and end the chat with Now Assist in Virtual Agent.
