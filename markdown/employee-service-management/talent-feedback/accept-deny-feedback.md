# Decline a feedback request

You can decline a feedback request sent you if you do not wish to provide feedback for any reason.

## Before you begin

Role required: employee

## Procedure

1.  Navigate to **All** &gt; **Employee Center** &gt; **My tasks**.

2.  Select the feedback request that you want to view or submit feedback.

    **Note:** Feedback requests are categorized based on the requester. If there are multiple feedback requests from the same employee, you will see them one after the other.

3.  Select **Decline**.

4.  Add a reason for declination in the pop-up and select **Decline**.
