# Request skill feedback

You can request skill feedback from the skill feedback requests sent widget when there no existing feedback requests sent.

## Before you begin

Role required: manager

## Procedure

1.  In Employee Center navigate to **Your team** &gt; **Skills**.

2.  On the skill feedback requests sent widget, select **Request feedback**.

3.  On the **Request skill feedback** form, fill in the information.
