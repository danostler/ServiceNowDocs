# View skill feedback submitted

As a manager, view all the skill feedback submitted by employees.

## Before you begin

Role required: manager

## Procedure

1.  In Employee Center navigate to **Your team** &gt; **Skills**.

2.  On the skill feedback requests sent widget, select **View details** on any of the feedback listed.

3.  On the skill feedback form, view the information.
