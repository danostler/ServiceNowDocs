# Update or add text attributes

Add, edit, update, or remove text attributes in the CAD Editor.

## Before you begin

For example, if you want to change the name of a room, you can select the text and use the text edit panel to rename it.

Role required: admin

## Procedure

1.  Select the CAD Editor tab.

2.  Select the appropriate **Active layer** on the CAD editor can.

3.  Select the Text tool icon \(\[Omitted image "wsd-text-icon.png"\] Alt text: Text tool icon.\) from the menu.

4.  Select to add the text.

5.  The text edit panel opens.

    \[Omitted image "wsd-add-text-panel.png"\] Alt text: Text panel to add a text element.

6.  Add the text and close the panel.

    The text attribute is added to a selected layer on the CAD file.

7.  To edit an existing text, select the text to open the text edit panel and edit.

    The updated changes appear on the CAD Editor preview.

8.  Select **Remove** to remove a text.
