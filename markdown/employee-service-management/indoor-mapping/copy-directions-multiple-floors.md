# Copy direction graphs to multiple floors

Select a campus or building and copy floor direction graphs from a floor to a single floor or multiple floors in a building.

## Before you begin

**Note:** If a building has similar floor layouts across floors, for example floor 1, 2, 3, etc, and if you want to replicate directions from one floor to another, copy directions from a single floor to multiple floors on a building. This will ensure that all connectors are linked between all floors.

The destination floors that you are copying from must belong to the same building. Copying direction graphs from one floor to multiple floors helps to copy the direction graphs that are manually created. Users don't have to draw the graph on every floor if the building architecture and structure is similar.

Role required: map editors, map admin, map editor limited

## Procedure

1.  Navigate to **All** &gt; **Indoor Mapping** &gt; **Map Studio**.

2.  Select a campus.

3.  Select **Manage Workplace** &gt; **Directions**.

4.  Select a building on the Map Studio and click the More actions menu icon \(\[Omitted image "more-options-icon.png"\] Alt text: More actions menu icon.\).

5.  Select **Copy directions** from the list.

    1.  In the Copy Directions dialog box, complete the following:

        -   **From**: Select the floor from which you want to copy the directions from.
        -   **To**: The floors to which you want to copy the directions to. Click **Select All** to select all or multiple floors at one go.

            **Note:** The floors must belong to the same building. For example, if you want to copy directions from Building A, then the floors that you select to copy must belong to Building A.

6.  Select the floor in the **From** list and select a floor or click **Select all** to select multiple floors in a building.

7.  Click **Copy directions**.

    The floor directions are copied to selected floor or floors.

8.  Select **Clear floor directions** to clear existing directions for a selected building or floor.
