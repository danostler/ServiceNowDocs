# Manage CAD source files

Manage your CAD source files using the Autocad import tool in Indoor Mapping v1.8.0.

-   Customers using Indoor Mapping v1.8.0 can import and configure CAD source files using the Autocad import tool. By default, the Autocad import tool is enabled in the Map Studio.
-   Existing customers using Indoor Mapping v 1.7.0 and later version can migrate the CAD source files using the latest Autocad import tool in Indoor Mapping v1.8.0. If you don’t want to migrate to the latest version, disable the option to configure and import the latest Autocad processing tool in the Map Studio. For more information, see [Configure the AutoCAD import tool](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/indoor-mapping/enable-autocad-v2.md).

    If you have not installed Indoor Mapping v1.8.0, you can continue using the Autocad tool in Indoor Mapping v1.7.0 and later version.
