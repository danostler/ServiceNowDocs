# View editor content

Work with the View Editor content and customize the campus, buildings, layers, places, connectors, and direction modes properties.

## Before you begin

Role required: map admin, map editor, map editor limited

## Procedure

1.  Navigate to **All** &gt; **Indoor Mapping** &gt; **Map Studio**.

2.  Select a campus for which you want to customize the view editor content.

3.  From Map Studio, select **Manage Workplace** &gt; **View Editor**.

4.  Select a view that you want to edit.

    For example: Default.

    \[Omitted image "view-editor-content.png"\] Alt text: View editor content pane

5.  To edit a campus view content, click the Campus drop-down list to display the campus properties.

6.  Update the properties and decide how your campus will render on the map.

8.  Select **+Add building**.

9.  From the Add or remove content page, select the required buildings and floors.

10. After making the changes, click **Apply**.

11. Select **Layers** and from the drop-down list, select **+Add layer**.

12. From the Add or remove content page, select the layers that you want to see in your view.

13. Select **Apply**.

14. Select **Places** and from the drop-down list, select **+Add places**.

15. From the Add or remove content page, select the places you want to see in your view.

16. Click **Apply**.

17. Select **Connectors** and from the drop-down list, select **+Add connector**.

18. From the Add or remove content page, select the connectors you want to see in your view.

19. Click **Apply**.

20. Select **Direction Modes** and from the drop-down list, select **+Add direction modes**.

21. From the Add or remove content page, select the direction modes \(Default or Accessible\) that you want to see in your view.

22. Click **Apply**.
