# Draw a polyline or a polygon

Draw a polyline, polygon, or a geometrical shape using the Draw line tool available in the CAD Editor. Join lines to form a polygon or a shape.

## Before you begin

For example, if you want to create a geometrical shape to form a room, you can use the Draw Line tool to draw the vertices. Similarly, resolve missing polylines in a polygon or add a missing polygon by drawing a line and joining the nodes.

Role required: admin

## Procedure

1.  Select the CAD editor tab.

2.  Select **Active Layer**.

    Select a layer where you want to add the polyline, polygon, or a shape.

3.  Select the Draw line tool \[Omitted image "wsd-draw-line-icon.png"\] Alt text: Draw line icon. tool.

4.  Select a starting point or node and drag.

5.  Release the tool to create a line by selecting again on the last point that you’ve added.

6.  To create a shape, select the starting point.

    When you draw a polygon, you must select either the first point that you’ve created or select the last point or node to create a line.

7.  Select **Remove** to remove the element from the CAD file.
