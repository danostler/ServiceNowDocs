# Manage places

Edit campus and buildings to add places, icons, and place types that you want to display on the indoor maps.

## Before you begin

Place Types are used to define a physical location on your map, it can be a polygon or a point depending on if a place corresponds to a room or just a place in your map. Places are linked to place types and are used to categorize the different places on your map \(example: meeting room, office, printer, parks, staircase, elevators, etc\). Places also have markers showing the place title.

**Note:** Places can either be automatically generated by a CAD import or have to be manually created with a raster import. For more information, see [Import Raster files](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/indoor-mapping/import-raster-files.md) and [Import CAD files to Map Studio](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/indoor-mapping/CAD-file-import.md).

Role required: map admin, map editor, map editor limited

## Procedure

1.  Navigate to **All** &gt; **Indoor Mapping** &gt; **Map Studio**.

2.  Select a campus.

    The campus opens in the Map Studio home page.

3.  To edit and manage places, select a building.

4.  On the map menu, select **Manage Places**.

5.  Select the floor on which you want your places to display on your maps.

6.  Click **New point** or **New polygon** to start creating places and new points of interest.

7.  Select **New point**.

    \[Omitted image "create-new-point-raster.png"\] Alt text: create a new point

8.  Click on the map to position the point.

9.  Select **New polygon** to add polygons or points respectively.

10. Click on the map to draw a polygon for a place, and close the polygon by clicking on the blue point or dot.

    Polygons can be used when you do not have a building in campus.

11. Make sure you close the shape by completing or ending the creation back on the first point.

    On the form, fill in the fields

    |Field|Description|
    |-----|-----------|
    |Name|Label name to identify a specific place \(not visible to the end user\).|
    |Title|Title as it appears on the map.|
    |Place type|Place Types name. The place types are used to define a physical location on your map, it can be a polygon or a point depending on if this place corresponds to a room or just a place in your map \( cafeteria, booth, elevator, auditorium, etc\).|
    |Floor|Floor names. The places and place types are created for a selected floor.|
    |Add a Marker|Marker on the map. Add a marker to see the location pin on the map. Markers define the icon \(image\) of place and identifies a location on your map.|
    |Add an Entrance|Entrance to a room is used by the wayfinding application. If you add an entrance, your pathway stops at the entrance.|

12. Select a place \(configured using the CAD configuration page\) to open the place editing panel.

    Notice that the **Name**, **Title**, and **Floor** are read-only fields and cannot be edited. Only the **Place type** field can edited. A validation message is displayed to inform that the selected place was created using an Autocad file. These fields can be edited in the Autocad Configuration page \(Indoor Mapping v 1.8.0\). For more information, see [Autocad import tool](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/indoor-mapping/autocad-ovw-v2.md) and [Configure places](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/indoor-mapping/edit-places.md).

    \[Omitted image "wsd-place-created-with-autocad-config-msg.png"\] Alt text: Selected place in Map Studio showing created in Autocad message.

13. Click **Save**.

14. To switch from a point to a polygon, click the other options icon \(\[Omitted image "more-options-icon.png"\] Alt text: Other options\) on the Place Edit panel and select **Switch to a polygon**.

    When you select a place, the Place Edit panel shows the CAD file properties such as room, capacity, etc. The AutoCAD properties are stored in a dedicated table **sn\_map\_core\_place\_properties**. For more information, see [Compute CAD file properties to extract space or room surface area](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/indoor-mapping/compute-cad-properties.md).

    The Indoor Mapping application provides the ability to extract room dimension from CAD files and to use it in Workplace Service Delivery rooms and space measurements.
