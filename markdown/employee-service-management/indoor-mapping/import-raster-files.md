# Import Raster files

After creating your campus, buildings, and floor, upload and import the raster floor plan sources.

## Before you begin

For more information about raster source file requirements, [Raster file requirements](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/indoor-mapping/raster-requirements.md).

Role required: map admin, map editor, map editor limited

## Procedure

1.  Navigate to **All** &gt; **Indoor Mapping** &gt; **Map Studio**.

2.  Select a campus.

3.  Select **Manage workplace** &gt; **Floor and Floorplans**.

4.  Select the floor that you have created and click **Upload file**.

5.  Select the raster file that you want to import for the floor plan.

6.  Before uploading the raster file, you can edit the name of the source.

    By default, the name of the source is &lt;campus name-floor title-file name&gt;.

7.  Ensure that the correct floor name or level is selected.

8.  Click **Save**.

9.  After uploading the file \(uploading may take few minutes to complete\), you will see the preview of the floor plan at the top of the screen.

10. The next steps to configure the raster file are displayed.

11. Edit georeferences and use the two purple pins to set to scale your floor plan by dragging and moving the pins to fit the building shape on the World map.

12. Click on the pins, the color changes to pink and you can change the position.

13. Click and hold any of the pins to change its position.

    **Note:** It is important that you set your floor plan to scale as this will help in wayfinding and getting directions. The directions show the exact time and distance between points. For example: Point A and point B.

14. Select the layer type of the file that you are importing.

    Layer types must be configured beforehand while you are setting up your default values.

15. After completing Steps 11 and 12, select **Start Import** to import the raster source file.

16. After the floor plan is imported, click **Continue**.

    Alternatively, you can also navigate to **Manage Workplace** &gt; **Places**.

    You are redirected to Places.

    **Note:** Raster files are flat files and therefore, places have to be created manually.
