# Floor map import group form

You can specify details about the import group and select users who must be notified of the import progress.

<table id="table_azk_s24_tfc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Number

</td><td>

Identifier of the import group record.This field is auto-generated and can't be edited.

</td></tr><tr><td>

Name

</td><td>

Name of the import group.

</td></tr><tr><td>

Notify User

</td><td>

Option to notify the specified user about the import progress.

</td></tr><tr><td>

User to notify

</td><td>

System user who must be notified about the import progress.

</td></tr><tr><td>

Supervised AI execution

</td><td>

Option to specify whether the Automate map updates AI agentic workflow must be triggered manually or automatically.

</td></tr><tr><td>

AI Agent instructions

</td><td>

Special instructions for the AI agent in the Automate map updates AI agentic workflow.

</td></tr><tr><td>

State

</td><td>

State of the import process.This field is automatically set to Draft and can't be edited.

</td></tr><tr><td>

Remaining tasks

</td><td>

Number of tasks remaining in the import process.This field is auto-generated based on the Floor plan import tasks and can't be edited.
