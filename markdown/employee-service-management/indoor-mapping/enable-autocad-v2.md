# Configure the AutoCAD import tool

Configure and import your CAD source file to Map Studio.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Indoor Mapping** &gt; **Map Studio**.

2.  Select a Campus.

    For example: Demo Campus.

3.  Select **Continue** or navigate to **Manage Workplace** &gt; **Floor &amp; Floor plans**.

    This list of buildings for a selected campus is displayed.

4.  Select a building from the panel.

5.  Select a floor to upload your CAD source file.

    \[Omitted image "wsd-add-autocad-source-file.png"\] Alt text: Selected building showing floors and upload CAD file option.

6.  Select **Upload** file, if your CAD source isn’t uploaded and configured for a floor.

7.  Select **Configure** to configure and update to your CAD file.

8.  The Create a new source dialog box appears.

    \[Omitted image "wsd-enable-cadv2-option.png"\] Alt text: Option to enable the new CAD configuration option.

9.  Complete the following information:

    -   File: Name of the AutoCAD source file. For example: demo.dwg
    -   Name: Campus name is added and is pre-filled by default.
    -   Floor: Name of the floor where the CAD configuration changes have to be made or where the CAD file is to be imported. For example; Floor 3.
    -   Use the new Autocad processing tool: By default this option is enabled.

        **Note:** Disable the **Use the new Autocad processing tool** button if you want to continue using the Autocad tool provided in earlier versions of Indoor Mapping \(pre 1.8.0 version\). For more information, see [Import CAD files to Map Studio](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/indoor-mapping/CAD-file-import.md).

10. Review the CAD guidelines by selecting the **Click here** link.

11. Select **Save**.

    The CAD configuration page is displayed. For more information, see [CAD configuration](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/indoor-mapping/autocad-v2-config-layout.md).
