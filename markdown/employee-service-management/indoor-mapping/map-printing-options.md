# Map printing options

The map printing options provide configurations for the page format, font, icons, collision, legend, and timestamps.

|Field|Description|
|-----|-----------|
|Page Orientation|Orientation that you want to print the page in. The orientation can be landscape or portrait.|
|Page Size|Paper size that you want to use to print the map.|

|Field|Description|
|-----|-----------|
|Add page title|Option to display the title on the map.|
|Add timestamp|Option to display the print timestamp to the map.|
|Include legends|Option to enable the map legend.|
|Keep legends in a separate page|Option to move the legend to the second page of the print layout. This field appears only if **Include Legends** is selected.|
|Include directions|Option to display directions between the points selected in the location directory. This field appears only if your map contains directions between locations.|
|Include To and From info|Option to display the info card for the points selected in the location directory. This field appears only if **Include directions** is selected.|
|Replace space titles by numbers|Option to change the location names to numbers. Enabling this option modifies the legend with the numbers and associated location labels.|

|Field|Description|
|-----|-----------|
|Space label text size|Size of the location labels on the map.|
|Space marker size|Size of the location icons on the map.|
|Legend label text size|Size of the map legend text.|
|Legend label icon size|Size of the map legend icons.|
|Remove text anti-collision|Option to enable overlapping of location labels. You can enable this option and adjust the font size to avoid collision.|
|Remove icon anti-collision|Option to enable overlapping of location icons. You can enable this option and adjust the icon size to avoid collision.|
