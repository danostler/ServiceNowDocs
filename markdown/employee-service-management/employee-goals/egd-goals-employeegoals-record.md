# Employee Goals table

The records in the Employee Goals table contain goal management data. Each record is created when the Employee Goals application imports the corresponding goal data from your organization's human capital management \(HCM\) system.

The form that appears when you select a record in the Employee Goals table contains the following fields:

<table id="table_tj4_lqr_cbc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

**Name**

</td><td>

A name for the goal that was set by the employee.

</td></tr><tr><td>

**Owner**

</td><td>

The employee to whom the goal belongs.

</td></tr><tr><td>

**State**

</td><td>

One of the following options to describe the status of the goal in terms of its progression:-   **Draft**
-   **In Progress**
-   **Approved**
-   **Complete**
-   **Pending**
-   **Achieved**
-   **Not Achieved**

</td></tr><tr><td>

**Active**

</td><td>

A check box to indicate whether the goal is active.

</td></tr><tr><td>

**Archived**

</td><td>

A check box to indicate whether the goal is archived.

</td></tr><tr><td>

**Start date**

</td><td>

The date at which the goal starts.

</td></tr><tr><td>

**End date**

</td><td>

The date at which the goal ends.

</td></tr><tr><td>

**Shared with manager**

</td><td>

A check box to indicate whether the employee shared their goal with their manager.

</td></tr><tr><td>

**Description**

</td><td>

Free-form text to describe the goal in detail.

</td></tr><tr><td>

**Type**

</td><td>

One of the following options to indicate the goal type:-   **None**
-   **Business goal**
-   **Personal goal**

</td></tr><tr><td>

**Status**

</td><td>

One of the following colors to represent the status of the goal:-   **None**
-   **Green**
-   **Yellow**
-   **Red**

</td></tr><tr><td>

**Domain**

</td><td>

The domain associated with the employee specified in the **Owner** field.

</td></tr></tbody>
</table>
