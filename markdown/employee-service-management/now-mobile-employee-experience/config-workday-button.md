# Configure Workday approval functions for Now Mobile

Configure the approve and reject buttons for all the Workday approval task types. Complete all your approval tasks in the Now Mobile app.

## Before you begin

Role required: sn\_hr\_workday.admin

## Procedure

1.  Navigate to **All** &gt; **System Mobile** &gt; **Screens**.

2.  Select **About** for the correct approval type.

    Check the corresponding **Description** next to **About** to know the approval type.

3.  In the **Function instances belonging to screen** tab, turn the options to **True** as required.

    **Note:** The base system settings for a few approval types are set to false in the Now Mobile app. For more information, see [Workday approvals in Now Mobile](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/now-mobile-employee-experience/ec-workday-mobile.md).


## Result

Approve and reject buttons are displayed on the configured approval task types in mobile.
