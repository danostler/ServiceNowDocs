# Define people search as inactive

Configure whether users can search for other users in the system. By default, people search is enabled.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Now Mobile App** &gt; **Applet Launchers**.

2.  Open the page where you want to disable people search.

3.  In the **Search Configuration** field, select **Homepage Search - Catalog and Knowledge**.

4.  Click **Update**.


## Result

Your users can no longer see search results for other users in the system.
