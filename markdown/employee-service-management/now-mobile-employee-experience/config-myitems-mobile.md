# Configure My items

Set up My items on the Now Mobile® application to access all your tasks, requests, and other personal resources like assets and purchases from one place.

## Before you begin

Role required: sn\_hr\_sp.esc\_admin

## Procedure

1.  Navigate to **System Mobile** &gt; **Now Mobile App** &gt; **Applet Launchers**.

2.  In the Launcher screens page, select **Homepage**.

3.  In the Homepage screen, go to the **Body** tab and select **My items**.

4.  In the Launcher section mapping page, select **Preview this record** on the **UI section**, which should display **My items**.

5.  In the Icon section page, go to the **Icon section destinations** tab, select **New**, and create as many items you would like to display to on the My items section.

    Set up the icon sections, colors, destinations from the **Icon section destinations** tab. You can also use the Mobile App Builder to get a more holistic view while configuring. For more information, see .


## Result

You have set up My items on your Now Mobile® application.
