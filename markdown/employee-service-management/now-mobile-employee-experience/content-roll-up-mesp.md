# Configure content roll-up on topics for Now Mobile

Browse through the associated content on the parent topic and its sub-tree topics. Enable the feature to see a wider range of results for your search on the Now Mobile app.

## Before you begin

Role required: admin

## About this task

Browse content directly associated to the topic that you’re viewing along with the content on the sub-tree topics. With this feature, you can:

-   Find associated content on topic pages easily.
-   Search content with a broader result display with the results from the sub-tree topics.

## Procedure

1.  Navigate to **All** &gt; **System Properties** &gt; **All Properties** &gt; **sn\_ex\_sp.u\_allow.rollup.mesp**.

2.  Set the system property to **true**, and select **Save**.

    **Note:** Verify the value is set to true to enable the feature. The default value is set to false.
