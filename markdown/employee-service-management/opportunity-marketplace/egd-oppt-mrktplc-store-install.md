# Install Opportunity Marketplace

Opportunity Marketplace \(OPM\) is a key application that works as part of the Talent Development suite of applications. OPM improves employee engagement and retention with internal mobility.

## Before you begin

Opportunity Marketplace has the following dependencies. When you install OPM it activates or installs the following:

**Note:** Opportunity Marketplace installs the latest required versions of dependent apps and plugins.

-   Talent Development Core
-   Manager Hub
-   Skills Foundation
-   Finance Common Architecture
-   Hiring Core
-   Proactive Prompts
-   Talent Development Shared Library \[app-egd-shared-library\]

    **Note:** The shared library isn’t visible in the ServiceNow Store, but the latest required version is installed with OPM.


Role required: admin

## Procedure

1.  Go to the ServiceNow Store and find the Opportunity Marketplace \[sn\_opp\_market\] application using the filter criteria and search bar.

2.  Select a version from the list and select **Install**.

    In the Review Installation Details dialog box, any dependencies installed with your application are listed.

3.  If you're prompted, follow the links to the ServiceNow Store to get any additional entitlements for dependencies.

4.  If demo data is available and you want to include it, select the **Load demo data** check box.

    Demo data comprises sample records that describe application features for common use cases. Load demo data when you first install the application on a development or test instance.

5.  Select **Install**.
