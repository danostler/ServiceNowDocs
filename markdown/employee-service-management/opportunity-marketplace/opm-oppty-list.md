# Opportunity details and descriptions in Opportunity Marketplace

Admins can view detailed information about opportunities.

Here’s a list of the columns displayed by the base system for opportunities. You can modify which columns to display and the order they’re displayed by selecting the gear icon.

<table id="table_z1r_ybb_ccc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

**Title**

</td><td>

A name given to the opportunity by the opportunity owner.

</td></tr><tr><td>

**Opportunity type**

</td><td>

The base system opportunity type:

-   Gig
-   Project
-   Volunteer

 You can add custom opportunity types with names that are appropriate for your needs.

</td></tr><tr><td>

**State**

</td><td>

The **State** of the opportunity reflects the stage in the process where the opportunity is.

</td></tr><tr><td>

**Owners**

</td><td>

One or more owners of an opportunity can make updates and remove opportunities.

</td></tr><tr><td>

**User criteria**

</td><td>

User criteria gives access to groups of users for an opportunity.

</td></tr><tr><td>

**Start date**

</td><td>

The date the opportunity is published.

</td></tr><tr><td>

**End date**

</td><td>

The date the opportunity is closed.

</td></tr></tbody>
</table>You can update some of the fields. Double-click \(or use the keyboard shortcut\) to determine if it's editable.
