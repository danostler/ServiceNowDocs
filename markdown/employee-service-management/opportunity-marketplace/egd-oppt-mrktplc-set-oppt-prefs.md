# Set opportunity preferences

Set your preferences for opportunities so that your preferences match any opportunities you’re interested in.

## Before you begin

Role required: none

**Note:** Access is based on user criteria.

## Procedure

1.  From the Employee Center go to **Opportunity marketplace** &gt; **Other opportunities**.

2.  Select **Edit preferences**.

<table id="table_tlk_hgp_jcc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

**Which working style do you prefer?**

</td><td>

Select your preferred work style. You can select more than a single work style. Not selecting any work style indicates that you’re open to all work styles.

-   **Onsite**
-   **Remote**
-   **Hybrid**


</td></tr><tr><td>

**How many hours per week are you looking to commit?**

</td><td>

Enter the number of hours that you’re willing to work on an opportunity. The value for this field is for an entire work week.

</td></tr><tr><td>

**Which job levels are you interested in?**

</td><td>

Select all of job levels that you're interested in. You can select both individual contributor \(IC\) and manager \(M\) job levels.-   **IC1 - IC4**
-   **M1 - M2**


</td></tr><tr><td>

**Which work locations do you prefer?**

</td><td>

Select the field and then select a listed location. You can select more than one location.

</td></tr><tr><td>

**Save and close**

</td><td>

Save and close the preferences form to make your preferences immediately available.

</td></tr></tbody>
</table>3.  Select **Save and close** to save your selections and make the immediately available for opportunities.
