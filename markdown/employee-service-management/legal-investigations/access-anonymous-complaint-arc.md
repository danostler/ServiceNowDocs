# Access your anonymous report from the Anonymous Report Center portal

You can follow up on your anonymous report to view comments and updates from the legal fulfiller working on it or add comments.

## Before you begin

To access your anonymous report, you must have the report key and report number that was generated when you submitted the request.

Role required: None

## Procedure

1.  Open the Anonymous Report Center portal either via Employee Center portal or using a direct URL.

2.  If you're asked to provide credentials to log in, you can copy this URL and enter your instance details to access the Anonymous Report Center.

    For example: https://&lt;instance\_name&gt;/arc.

3.  In the Anonymous Report Center portal, click **Follow up on a report**.

4.  On the Follow up on a report page, enter the details.

    1.  In **Report Key**, enter the report key you get after submitting an anonymous report.

    2.  In **Report Number**, enter the report number you get after submitting an anonymous report.

5.  Enter the text verification code.

    **Note:** For the text verification code to work, you must install the HR Scoped Security plugin \(com.snc.hr.scoped\_security\).

6.  Click **Next**.

    The report is displayed in the Standard Ticket Page.
