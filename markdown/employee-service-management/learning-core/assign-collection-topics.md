# Assign topics to a collection

Add assigned topics to collections.

## Before you begin

Role required: sn\_lc.learning\_admin, sn\_lc.learning\_course\_catalog\_admin

## Procedure

1.  Navigate to **All** &gt; **Content** &gt; **Collection**.

2.  Open the collection that you want to add assigned topics to.

3.  On the Assigned Topics related list , select **Add**.

4.  Select a topic to be added as connected content to the collection.
