# Resume a course

Use this widget to easily access and resume any course that you may have left mid-way.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Employee Center** &gt; **My Learning**.

2.  In the Overview page, select the course, course collection, or path that you want to resume from the Continue where you left off widget.

    **Note:** This widget is only available when you have courses that are in the in-progress state.

3.  Select **Resume**.
