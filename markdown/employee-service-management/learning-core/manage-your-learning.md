# Using My Learning

Use My Learning in Employee Center to access your learnings and achievements.

-   **[Enroll in a course collection on My Learning](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/learning-core/enroll-course-lxp.md)**  
Enroll in a course collection to pursue courses that can enhance your skills.
-   **[Resume a course](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/learning-core/resume-course.md)**  
Use this widget to easily access and resume any course that you may have left mid-way.
-   **[Create a personal course collection](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/learning-core/create-personal-collection.md)**  
Create personal collections to include the courses and paths that you would like to access easily.
-   **[Create a Learning challenge](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/learning-core/learning-challenge.md)**  
Create a Learning challenge to challenge yourself to complete a particular number of courses within a time frame that you choose.
-   **[View recent activity](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/learning-core/view-recent-activity-lxp.md)**  
Review your recent learning activity with easy access to your assigned, saved, shared with, and history tabs.
-   **[Assign Learning to your team](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/learning-core/assign-learning-lxp.md)**  
As a manager, assign Learning to your team members to enhance their skills.
-   **[Manage Learning](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/learning-core/manage-learning-achievements.md)**  
Using Learning, learning administrators can manage user activities. To aid the users learning journey, learning admins can create achievements, define achievement rules, and monitor the overall learning activity of a user.
-   **[Skill sensing for learning content](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/learning-core/skill-sensing-for-learning-content.md)**  
Skill sensing is an automated approach to identify and extract skills from the title and description of a course so that the admin can associate the skills to a course.
