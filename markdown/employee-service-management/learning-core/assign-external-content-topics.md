# Assign topics to external content

Assign topics to external learning content in Learning.

## Before you begin

Role required:sn\_lc.learning\_admin, sn\_lc.learning\_course\_catalog\_admin

## Procedure

1.  Navigate to **All** &gt; **Content** &gt; **External Content**.

2.  Open the external learning course that you want to add assigned topics to.

3.  On the Assigned Topics tab, select **Add**.

4.  Select a topic to be added as connected content to the external learning course.
