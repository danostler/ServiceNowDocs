# Reporting Solutions for Learning Core

Learning Experience Dashboard provides a consolidated report on courses in each repository, catalogs with courses, and learning tasks status.

\[Omitted image "learning-exp-db.png"\] Alt text: Provides a consolidated report on courses in each repository, catalogs with courses, and learning tasks status.

## Learning Repository widget

Displays the number of courses associated with each of the source. Source indicates the third-party learning management system that has integrated with the ServiceNow application.

## Catalogs

Displays the number of catalogs associated with each of the source. Indicates how many number of courses from a particular source \(third-party learning management system\) are added to catalogs.

## Task Status

Displays the status of all the learning tasks within each source.
