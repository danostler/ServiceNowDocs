# Assign topics to internal content

Add assigned topics to internal learning content.

## Before you begin

Role required:sn\_lc.learning\_admin, sn\_lc.learning\_course\_catalog\_admin

## Procedure

1.  Navigate to **All** &gt; **Content** &gt; **Internal Content**.

2.  Open the internal learning course that you want to add assigned topics to.

3.  On the Assigned Topics related list, select **Add**.

4.  Select a topic to be added as connected content to the internal learning course.
