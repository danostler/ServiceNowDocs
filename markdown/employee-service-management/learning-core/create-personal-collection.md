# Create a personal course collection

Create personal collections to include the courses and paths that you would like to access easily.

## Before you begin

Role required: user

## Procedure

1.  Navigate to **All** &gt; **Employee Center** &gt; **My Learning**.

2.  Select the context menu icon \[Omitted image "context-menu-icon.png"\] Alt text: Context menu icon. on any course that is displayed on the screen.

3.  Select **Save to collection**.

4.  Select **Create New**.

5.  On the form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Collection name|Name of the new personal collection.|
    |Description|Brief description of the personal collection.|

6.  Select **Save**.
