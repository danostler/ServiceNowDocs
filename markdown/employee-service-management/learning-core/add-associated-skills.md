# Add associated skills

Add associated skills to external learning content in Learning.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Content** &gt; **External Content**.

2.  Open the external learning course that you want to add assigned topics to.

3.  On the **Associated skills** tab, select **New**.

4.  On the form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Name|Name of the skill.|
    |Description|Detailed description about the skill.|
    |Level Type|Level type to which the skill belongs.|
    |Active|Option to enable the skill.|

5.  Select **Submit**.

6.  On the **Associated skills** tab, select **Edit**.

7.  Add users who should have access to external learning content skills.
