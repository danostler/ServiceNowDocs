# Update similar tasks in bulk

Managers, mentors, and other journey participants with access to certain Journey designer task types are able to update similar tasks identified by Journey designer in a single action.

## About this task

There are four Journey designer task types available for bulk update:

-   Approval \[sysapproval.approver\]
-   Mark when complete \[mark\_when\_complete\]
-   Submit catalog item \[submit\_catalog\_item\]
-   Submit order guide \[submit\_order\_guide\]

**Note:** The completion workflow for the different task types is unique. The steps in this task are a general representation of how to complete a bulk update.

## Before you begin

Role required: you can use bulk update for any available Journey designer tasks that you have write permission for.

## Procedure

1.  From the Employee Center, navigate to **My Tasks**.

2.  Select a task to update.

    Similar tasks that are available for bulk update display a message indicating that there are other tasks that you can update at the same time.

3.  Select **Select more**

    When there are multiple tasks available, a list of those tasks is displayed. Task must be a similar type in the same journey type to be eligible for bulk update.

    You can select any or all tasks in the list of tasks available. You can work on only those tasks that need attention.

4.  Select **Continue**.

5.  Select **Next**.

6.  Complete the task.

    Depending on the task type, the completion of the task is unique to the task type.

    **Note:** Some tasks may be optional. Optional tasks can be skipped. Required tasks can’t be skipped.
