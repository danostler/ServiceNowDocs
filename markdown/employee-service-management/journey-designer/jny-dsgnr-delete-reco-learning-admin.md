# Delete recommended learning in the Journey designer app

Delete recommended learning for employee journey configurations.

## Before you begin

Role required: sn\_jny.admin

## About this task

Recommended learning added to a journey configuration can be deleted only by an administrator.

## Procedure

1.  Navigate to **All** &gt; **Journey designer** &gt; **Administration** &gt; **Manage Recommended Learning**.

2.  Select the check box next to the recommended course that you want to delete.

3.  Select **Delete** in the **Actions on selected rows** list.

4.  Select **Delete** to confirm that you want to delete the selected courses.
