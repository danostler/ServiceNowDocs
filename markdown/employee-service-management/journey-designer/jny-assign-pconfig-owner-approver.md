# Assign owners and approvers to a journey configuration

In Journey designer, administrators assign the initial owners and approvers to a plan configuration.

## Before you begin

Role required: sn\_jny.admin

## About this task

Co-owners have the same capabilities as owners.

**Note:** Approvers are optional. If no approver is selected, the approval process isn’t required.

## Procedure

1.  Navigate to **All** &gt; **Journey designer** &gt; **Administration** &gt; **Manage Plan Configurations**.

2.  Open a **Plan Configuration** to assign an owner and approver.

    Double-click the **Plan Configuration** to open it.

3.  Select the **Plan Config Owners and Approvers** tab.

4.  Select **New**

5.  Select the search icon for the **Owner field**.

6.  Start typing the name of the owner to search the list of users.

    You can click in the **Owner** field to view a list of recent selections.

    **Co-owners** and **Approvers** fields are available only after you select an **Owner**.

7.  Select the lock icon next to the **Co-owners** field.

8.  Select a co-owner.

9.  Select the lock icon next to the **Approvers** field.

10. Select **Submit**.
