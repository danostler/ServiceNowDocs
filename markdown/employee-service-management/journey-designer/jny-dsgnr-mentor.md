# Mentors can modify and manage journeys

Mentors with the appropriate permission can modify journeys and add other mentors.

## Before you begin

Role required: Modifier \[sn\_jny.writer\]

## Procedure

1.  From the Employee Center, select **Journeys**.

2.  Select the **Involved journeys** tab, and select the journey you want to update.

3.  Select **Edit Journey**.

    -   **Add a mentor**

        Select the **Mentor** edit icon to add additional mentors to the journey. Add mentors who have unique skills to share with the employee.

    -   **Add a stage**

        Select **Add a stage** to create a state in the journey. New stages can be helpful to provide additional training or resources from the original journey.

    -   **Add tasks**

        Select **Add tasks** to add additional tasks to an existing stage of a journey. New tasks give employees access to additional training and other resources.

4.  Select **Finish editing**.


## What to do next

With the correct permissions, Journey mentors can edit existing journeys. Mentors can:

-   Add Journey Accelerator tasks to a Journey.
-   Add Journey Accelerator stages to a journey.
-   Drag stages and tasks within a journey to change the order.
-   Add a mentor.
-   Publish the journey.
-   Add recommended learning and quick links.
