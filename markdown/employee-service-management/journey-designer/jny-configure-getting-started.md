# Configure the Getting started resources widget

Journey designer administrators can configure the Getting started resources widget to provide quick links that give access to internal and external resources. Administrators can provide links to knowledge base articles, catalog items, pages, or other pertinent information that your organization deems beneficial for an onboarding employee.

## Before you begin

Role required: admin \[sn\_jny.admin\]

## Procedure

1.  Navigate to **All** &gt; **Service Portal** &gt; **Pages**.

2.  Select **ID** from the drop-down list associated with the **Search** field.

3.  In the **Search** field, enter `*jny`.

    A list of pages associated with the Journey designer app appears.

4.  Select **jny\_journey\_onboarding\_details**.

    The Page form appears.

5.  Under Related Links, select **Open in Designer**.

    The Service Portal Designer displays the Journey designer Onboarding page.

6.  Select the **Edit instance options** icon associated with the Getting started resources widget.

    The Getting started resources dialog box appears.

7.  Select **Quick links**.

    For more information about quick links, see [Quick links](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/employee-experience-foundation/emp-center-quick-link-config.md).

    A drop-down list with quick links that are defined in the Quick Links \[sn\_ex\_sp\_quick\_link\] table appears.

8.  Select a quick link that you want to add to the Getting started resources widget.

9.  Select **Save**.
