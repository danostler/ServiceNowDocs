# Create and manage plan types

Create plan types to manage the top-level classification for stages and tasks in a Journey designer plan.

## Before you begin

Role required: admin \[sn\_jny.admin\]

## Procedure

1.  Navigate to **All** &gt; **Journey Designer** &gt; **Manage Plan Types**.

2.  Select **New**.

3.  In the **Title** field, enter a meaningful name for the type of plan that you are creating so that it is easily identifiable.

4.  After you have selected any header configuration you want to change, select **Submit**.
