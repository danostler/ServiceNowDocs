# Add Journey designer widget

The Journey designer widget is not on the Employee Center home page by default. An administrator must add it.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Service Portal** &gt; **Service Portal Configuration**.

2.  Select **Designer**.

3.  Select **Employe Center Homepage**.

4.  In the **Filter Widget** field, type `EC homepage Journey Tabs`.

5.  Drag the **EC homepage Journey Tabs** widget to the service portal page.

    Add the `EC homepage Journey Tabs` to any valid element on the page.


## What to do next

Verify that the widget is available in your organization's Employee Center portal.

1.  Login as a user that has permissions to view or modify journeys.
2.  Verify that the title **Team journeys** is visible.
