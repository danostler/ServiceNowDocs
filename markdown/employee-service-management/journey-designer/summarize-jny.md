# Summarize a journey with Now Assist for HR Service Delivery \(HRSD\)

Managers can instantly summarize the information that appears on the Journey detail page to acquire an overview of an employee's journey. This summary also reflects the blockers that are impeding the journey's progress and the steps required to resume the journey's progress.

## Before you begin

The Now Assist for HRSD \(sn\_hr\_gen\_ai\) plugin must be installed and the Journey Summarization for Managers skill must be activated.

Role required: manager \[sn\_jny.reader or sn\_jny.writer\]

## About this task

The following sections appear in the summary that is generated:

-   **Overview**

    This section provides the manager with the journey's progress by reflecting the percentage of the journey that is complete and whether there are any risks affecting this journey.

-   **Blocker**

    This section provides the manager with the overdue tasks that are blocking the journey's progress. This section also reflects the activity sets in which the overdue tasks are nested, and the activity sets that are affected by the overdue tasks.

-   **Next Steps**

    This section provides the manager with the next steps that either they, their employee, or another team member should take to resume the journey's progress.


## Procedure

1.  Navigate to **All** &gt; **Employee Center** &gt; **Journeys**.

2.  Select a journey for an employee on your team.

    The Journey detail page for the employee you selected appears.

3.  Select **Summarize** in the Now Assist can summarize this journey widget.

    A summary of the employee's journey is generated and includes the following information:

    -   An overview of the journey
    -   Blockers that are impeding the journey's progress
    -   The next steps that should be taken to resume the journey's progress
4.  Review the summary to acquire pertinent information about your employee's journey.
