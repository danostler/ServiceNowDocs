# Add task template to journey

Use task templates to quickly add pre-configured tasks to an employee's journey, such as scheduling a meeting or taking a learning course.

## Before you begin

-   Role required: Modifier \[sn\_jny.writer\]
-   [Create a journey](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/journey-designer/jny-dsgnr-create-journey.md)
-   [Create a task template](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/journey-designer/jny-dsgnr-create-task-template.md)

## Procedure

1.  From the Employee Center, navigate to **Journeys** &gt; **My team's journeys** and scroll to the **Journeys for my team** section.

2.  Select the Journey where to add the task template.

3.  Navigate to the stage where to add the task and click **Add tasks**.

4.  Select **Use task templates**.

5.  Select **Create task template**.

6.  Select from the available task templates and click **Use template**.
