# Review and approve plan configuration changes

Approvers receive approval tasks to track reviewing, approving, and rejecting changes to a plan configuration.

## Before you begin

Role required: assigned as approver for a plan configuration

## About this task

-   **Approver**

    Plan configuration approvers are assigned to a plan configuration by a ServiceNow administrator. Approvers are responsible for reviewing changes that were made by a plan configuration owner. An approver can choose to accept or reject the changes.

-   **Owner \(or co-owner\)**

    Changes plan configuration, and publishes the changes which initiate the plan configuration approval workflow.


## Procedure

1.  From the Employee Center service portal, navigate to **Journeys** page.

2.  Select **My Tasks**.

    Find the **Approve Request** task that you want to approve or reject.

    An approval request looks similar to this example.

    ```
    Approve Request
    
    Sales Employee Plan Configuration (Demo)
    
    Due in 7 days
    ```

3.  Review the changes.
