# Create a journey using Now Assist

Create and modify journeys for employees using Now Assist.

## Before you begin

Role required: manager

Install the Now Assist for HR Service Delivery \(HRSD\) plugin \(sn\_hr\_gen\_ai\).

## About this task

The user that creates a journey is the journey owner. Journey owners create, modify, and manage all aspects of a journey.

## Procedure

1.  Navigate to **All** &gt; **Employee Center**.

    The Employee Center portal opens.

2.  Select **Journeys**.

3.  Select **Create a new journey**.

    Journey templates have preconfigured journey configurations. Your organization administrator creates and manages journey configurations.

4.  Select **Build with Now Assist**, then select **Next**.

5.  On the form, fill in the fields.

Describe the journey that you want to create.Use a minimum of 60 characters to describe your journey. Select a pill to view sample directions for journeys.

Sample prompts that you can use include:

-   Create a personalized career development journey for Sarah Johnson, a junior software developer aiming to transition into a senior role: The journey should include stages such as "Identify skill gaps in leadership and advanced programming," "Enroll in targeted training programs like leadership workshops and advanced coding bootcamps," and "Conduct quarterly reviews with the manager to track progress and adjust goals." The journey should span over 6 months.
-   Design an onboarding journey for Sarah Lee, a new marketing manager: The journey should include stages like "Send a personalized welcome email with team introductions," "Provide training on the company's marketing tools and internal systems," "Attend the first strategy meeting to understand ongoing campaigns," and "Deliver the first project task, such as drafting a marketing campaign plan." The journey should be structured to span over 4 weeks and prioritize both acclimation and quick contributions.

    2.  Describe the journey that you want to create.

    3.  Select **Regenerate draft**.
