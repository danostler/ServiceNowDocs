# Install Journey designer

Install the Journey designer version 3 \[sn\_jny\] application to enable administrators and managers to create and manage various professional and personal events and journeys.

## Before you begin

**Note:** You must upgrade to Vancouver before you can install Journey designer.

Dependent plugins and apps for Journey designer:

-   Human Resources Scoped App: Core \[com.sn\_hr\_core\]
-   Human Resources Scoped App: Lifecycle Events for Enterprise \[com.sn\_hr\_lifecycle\_ent\]
-   Employee Center Pro 28.2.1 \[sn\_ex\_sp\_pro\]
-   Journey Accelerator 6.2.2 \[sn\_ja\]
-   Admin Experience Framework 2.1.3

Role required: admin.

## About this task

Journey designer must be installed as a store application.

**Important:** The Journey designer app also installs the Journey Accelerator \[sn\_ja\] \(JA\) application.

## Procedure

1.  Go to the ServiceNow Store and find the Journey designer \[sn\_jny\] application using the filter criteria and search bar.

    In the list next to the **Install** button, the versions that are available to you are displayed.

2.  Select a version from the list and select **Install**.

    In the Review Installation Details dialog box, any dependencies installed with your application are listed.

3.  If you're prompted, follow the links to the ServiceNow Store to get any additional entitlements for dependencies.

4.  If demo data is available and you want to install it, select the **Load demo data** check box.

    Demo data comprises sample records that describe application features for common use cases. Load demo data when you first install the application on a development or test instance.

    **Note:** Any demo data related to dependent applications must be updated when Journey designer is updated.

5.  Select **Install**.
