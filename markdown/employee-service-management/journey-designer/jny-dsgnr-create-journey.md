# Create a journey

Users that are managers and have Modifier \[sn\_jny.writer\] permission create journeys for employees and assign mentors to help with the journey.

## Before you begin

Role required: A user must manage other users to have access to create journeys.

## About this task

The user that creates a journey is the journey owner. Journey owners create, modify, and manage all aspects of a journey.

## Procedure

1.  From the Employee Center, select **Journeys**.

2.  Select **Create a new journey**.

    Journey templates have preconfigured journey configurations. Your organizations administrator creates and manages journey configurations.

3.  Select a journey template to use an existing journey configuration or select **Blank** to create your own journey.

4.  Complete the form to create a journey.

    |Field|Description|
    |-----|-----------|
    |This journey is for|Name of the employee that this journey is for.|
    |Journey name|Descriptive name of the journey.|
    |Journey mentor|Mentors that you can select. You can also set the mentors' view or edit permission. You can add up to 5 mentors.|
    |Journey description|Details about this plan.|

5.  Select **Create journey**.


## What to do next

Journey owners and mentors with permission can edit existing journeys.

-   Add Journey Accelerator tasks to a Journey.
-   Add Journey Accelerator stages to a journey.
-   Drag stages and tasks within a journey to change the order.
-   Add a mentor.
-   Publish the journey.
