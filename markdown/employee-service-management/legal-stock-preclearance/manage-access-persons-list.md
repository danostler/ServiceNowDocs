# Manage access persons list

Add users who have access to non-public information regarding the trade of securities of your organization.

## Before you begin

Role required: sn\_lg\_stock\_cp.stock\_config

## Procedure

1.  Navigate to **All** &gt; **Legal Administration** &gt; **Stock Preclearance Admin** &gt; **Access Persons**.

2.  To add a user to the access persons list,

    1.  Click **Add Users**.

    2.  In the Add User dialog box, select users from the list to add as access persons.

        The list shows only active users from the Users \[sys\_user\] table.

    1.  Scroll down to the Actions on selected rows menu and choose **Add Users to Access List**.

        The selected users are added to the Access Persons list.

3.  To modify a user's information in the Access Persons list,

    1.  Click the preview icon \(\[Omitted image "preview-icon.png"\] Alt text: Preview icon\) and click **Open Record**.

    2.  On the form, update the fields.

        |Field|Description|
        |-----|-----------|
        |User|Name of the user.|
        |Entry date|Date on which the user was added to the access persons list. This field is not available for modification.|
        |Exit date|Date the user is automatically removed from the access persons list.|
        |Sec 16 officer|Option for marking the user as a Section 16 officer.|

    3.  Click **Update**.
