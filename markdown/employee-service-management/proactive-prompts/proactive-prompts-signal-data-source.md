# Create a data source for a signal in Proactive Prompts

Specify where and how to get the data you want to use to send insights to users.

## Before you begin

Role required: sn\_pp.admin

## Procedure

1.  Navigate to **All** &gt; **Proactive Prompts** &gt; **Signal Data Sources**.

2.  Select **New**.

3.  On the Signal data source form, fill in the fields.

    For a description of the field values, see [Signal data source form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/employee-service-management/proactive-prompts/proactive-prompts-signal-datasource-form.md).

4.  Select **Submit**.
