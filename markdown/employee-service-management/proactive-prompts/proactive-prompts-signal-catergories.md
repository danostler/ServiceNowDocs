# Configure signal categories in Proactive Prompts

Create and configure categories to enable efficient search of prompts grouped under the same categories.

## Before you begin

Role required: sn\_pp.admin

## About this task

Signal categories are the logical groups used to label and divide the signals. Categories also enable you to pull category-specific prompts on the prompt widget.

## Procedure

1.  Navigate to **All** &gt; **Proactive Prompts** &gt; **Signal categories**.

2.  Select **New**.

3.  On the form, fill in the fields:

    |Fields|Description|
    |------|-----------|
    |Name|Name of the signal category.|
    |Description|A brief description of the signal category.|
    |Active|Option to activate the signal category record.|

4.  Select **Submit**.
