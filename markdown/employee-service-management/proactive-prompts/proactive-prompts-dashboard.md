# Proactive Prompts dashboard

Use the Proactive Prompts dashboard to view the usage analytics and key metrics of the prompts that are generated to provide actionable insights in the flow of work.

To access the dashboard, navigate to **All** &gt; **Proactive Prompts** &gt; **Administration** &gt; **Usage Analytics Dashboard**.

## End users and roles

Dashboard Viewer \(sn\_pp.dashboard\_viewer\) can view the dashboard with various prompt metrics for the selected time period.

\[Omitted image "proactive-prompts-dasboard.png"\] Alt text: Proactive prompts analytics dashboard

## Reports and indicators

|Title|Type|Description|
|-----|----|-----------|
|Completed prompts|Single score|Number of prompts marked as Complete by Proactive Prompts users.|
|Interactions|Single score|Number of interactions \(action clicks\) performed as a result of prompts.|
|Expired prompts|Single score|Number of prompts that expired after the time-out period. The number is further divided into Expired prompts, which expired with no interaction from the user, and Interacted and Expired prompts, which involved a user interaction at some point but later expired.|
|Dismissed prompts|Single score|Number of prompts viewed and dismissed by the users. The number is further divided into Dismissed prompts, which were not interacted with, and Interacted and Dismissed prompts, which involved a user interaction but that later were dismissed.|
|Trending actions|Donut|Actions preferred by users who interact with prompts.|
|Channels interacted|Donut|Channels in which users are interacting with prompts.|
|Prompts sent per channel|Horizontal bar|Number of prompts generated for each channel.|
|Actions taken per signal|Stacked column|Various actions taken by the users based on a signal prompt.|

To see a detailed view and in-depth analysis of the widget click it.

Use the **Filter by date** option to select a time period and view the analytics in the given time period.
