# Opting out of a prompt in Virtual Agent

Choose not to receive a specific prompt by opting out of the signal displayed in Virtual Agent.

## Before you begin

Role required: none

## Procedure

1.  Navigate to **All** &gt; **Employee Center**.

2.  Select the chat icon \(\[Omitted image "icon-chat-va.png"\] Alt text: Chat icon for Virtual Agent\) to start the Virtual Agent.

3.  Type `Show notifications`.

    All your notifications are displayed.

4.  Select the signal that you want to opt out of.

5.  Select the **Dismiss** option to withdraw from the signal.

6.  In the next set of actions, select **Opt-out** to unsubscribe from the signal.

    **Note:** The **Not relevant to me** action dismisses the current prompt and not the entire signal.

7.  Select **Yes** to confirm your choice.


## Result

You will not receive prompts from the selected signal. The opted-out prompts are moved to the Dismiss state.
