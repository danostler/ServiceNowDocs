# Retire or archive a product offering version

Move a published product offering version to a Retired or Archived state when it is no longer in use or being sold to your customers. When you retire a product offering version, you can't use it in a product catalog.

## Before you begin

Role required: admin

## About this task

You can maintain only one active version of the product offering. After a product offering is retired or archived, you can't do the following actions:

-   Update the product offering version.
-   Use it in a product catalog.

## Procedure

1.  Open CSM Configurable Workspace.

2.  Navigate to **List** &gt; **Offerings** &gt; **Product Offerings**.

    A list of product offering records is displayed.

3.  Select a record you want to modify.

4.  To view the previous versions that are related to this product offering, select **More** &gt; **Versions \(n\)**, where n represents the number of base and previous product offering versions.

    For detailed information about the field descriptions .

5.  From the Versions \(n\) tab, you can select one or more product offering versions to retire or archive.

    -   To retire a previous product offering version, select the check box next to the Number column, and then select **Retire**.
    -   To archive a previous product offering version, select the check box next to the Number column, and then select **Archive**.
