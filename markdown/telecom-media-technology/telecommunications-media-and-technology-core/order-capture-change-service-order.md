# Change a service order inventory record

As a provider, learn how to change a service order inventory record in the ServiceNow AI Platform for your consumers.

## Before you begin

Role required: order\_approver, order\_viewer, sn\_ind\_tmt\_orm.order-fulfillment\_agent, or sn\_ind\_tmt\_orm.order\_fulfillment\_manager

## Procedure

1.  Navigate to **Workspaces** &gt; **CSM/FSM Configurable Workspace**.

2.  To change a service order inventory record for an existing customer account, do the following actions:

    1.  From the **Configurable Workspace Lists** tab \[Omitted image "list-outline-24.svg"\] Alt text:, select **Service Orders**.
    2.  To select all orders, select **All** or to select an order, select **Open**.
    3.  Select **New**.
3.  When the Choose an order type dialog appears, select **Change a service** and select **Create**.

    \[Omitted image "order-capture-change-service-order-selector.png"\] Alt text: Choose an order type with the Change a service option highlighted.
