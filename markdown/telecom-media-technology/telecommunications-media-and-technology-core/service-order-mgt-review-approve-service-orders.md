# Reviewing service orders for fulfillment

Learn how your order fulfillment manager can review service orders for fulfillment by using the Order Management application. With this application, you can make sure that your service orders are fulfilled correctly.

## Service order review and approval tasks

After the ServiceNow AI Platform captures a service order, the service order manager selects and verifies the order, line items, and characteristics data, and then approves or rejects it. Learn how this process triggers the order fulfillment flow and creates the required tasks to fulfill the service order.
