# Review the domain resource orders that are associated with a service order

Review the domain resource orders that were generated for a selected service order line item so that you can ensure that all the related details are correct and complete.

## Before you begin

Role required: sn\_ind\_tmt\_orm.service\_order\_manager

## About this task

The domain resource orders manage the resources required for fulfillment of the services that you are rendering.

## Procedure

1.  To view the resource orders that are associated with an order line item, select the order line item.

2.  Select an associated resource order to review or create a resource order.
