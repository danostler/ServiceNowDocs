# Configure the graph questionnaire Now Assist for Telecommunications, Media and Technology \(TMT\)

Configure the graph questionnaire to add or modify the new queries to retrieve the knowledge graph data.

## Before you begin

Role required: Admin

## Procedure

1.  Type `sn_nowassist_skill_config_var.LIST` in the **All** search filter.

2.  Press enter.

3.  Search for `Queries` in the **Column name**.

4.  Open the record.

5.  Select **here** if you see the notice that says:

    This record is in the Now Assist for Telecommunications, Media and Technology \(TMT\) application, but Global is the current application. To edit this record, click **here**.

6.  Clear the **Read only** field.
