# Employee Slate for Now Assist configuration menu

Navigation sections available on the Employee Slate for Now Assist configuration summary page, and what each section contains.

|Field|Description|
|-----|-----------|
|Configuration Summary|Overview of the overall setup status, showing the percentage of sections completed and a progress card for each configuration section. Changes are recorded automatically.|
|Appearance|Parent section for visual configuration settings. Contains the Branding and theming subsection.|
|Branding and theming|Settings for the portal's visual identity, including colors, logos, and theme options.|
|Data sources|Parent section for connecting content sources to the portal. Contains the Internal sources and External sources subsections.|
|Internal sources|Settings for connecting internal data sources, such as knowledge bases and service catalog items, to the portal.|
|External sources|Settings for connecting external data sources to the portal to surface content from outside the instance.|
|Conversational assistant|Parent section for connecting a conversational assistant to the portal. Contains the Now Assist setup subsection.|
|Now Assist setup|Settings for integrating the Now Assist conversational assistant with the portal.|
|Tasks and requests|Parent section for configuring employee self-service interactions. Contains the Tasks, Requests, and AI preferences subsections.|
|Tasks|Settings for the tasks available to employees through the portal.|
|Requests|Settings for the request catalog items available to employees through the portal.|
|AI preferences|Settings that control how AI-assisted features behave within tasks and requests.|
|Canvas|Parent section for configuring the portal layout. Contains the Canvas configuration and Widget library subsections.|
|Canvas configuration|Settings for the layout and structure of the portal canvas, including page arrangement and display options.|
|Widget library|Collection of available widgets that can be added to the canvas to extend portal functionality.|
|Documentation|Parent section for reference materials. Contains the Documentation and references subsection.|
|Documentation and references|Links to supporting documentation and reference materials for the Employee Slate for Now Assist configuration.|
