# CBS Client Role Rule form

Field descriptions for the CBS Client Role Rule form, used to define the conditions that determine how the CBS requester role is assigned to employees.

|Field|Description|
|-----|-----------|
|Name|Name of the role rule.|
|Active|When selected, the role rule is active and applied to determine requester role assignment. This check box is selected by default.|
|Table|Table that the role rule applies to. This field is automatically set to User \[sys\_user\].|
|Condition|Filter conditions that determine which employees receive the CBS requester role. Use **Add Filter Condition** to add a condition, or **Add OR Clause** to add an alternative condition set.|
