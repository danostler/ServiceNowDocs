# Configure Employee Slate for Moveworks

Configure Employee Slate for Moveworks to provide a conversational interface for employee portal queries on CBS. Use the Configuration Summary page to complete setup across five sections and export the finished configuration.

## Before you begin

Role required: sn\_cbs.admin

## About this task

Set up the Employee Slate for Moveworks portal to get a conversational interface for all your employee portal-related queries on CBS.

The Configuration Summary page tracks setup progress across five sections: **Appearance**, **Conversational assistant**, **Tasks and requests**, **Canvas**, and **Documentation**. Each section displays its completion status and total number of configuration items. Complete all sections to finish the setup.

## Procedure

1.  Navigate to **Admin** &gt; **Admin Home** &gt; **Core Business Suite** &gt; ****.

2.  Select **Core Business Suite** card, and select **Configure**.

3.  In the Configure your product section, go to **Employee Experience**, and select **Get Started**.

4.  In the Employee Slate list, select **Employee Slate for Moveworks** to open it in **Product Hub**, and select **Start new setup**.

5.  In the Application Manger page, select **Install**.

    It provides a pop-message when the installation is complete and **Employee Slate for Moveworks** is ready to configure.

6.  In the Configuration console, select each section to set up **Employee Slate for Moveworks**.

    The Configuration Console page opens, showing the overall setup status and a card for each configuration section. The setup status displays the percentage of sections completed.

    \[Omitted image "es-moveworks-cbs.png"\] Alt text: Configuration Console displaying for Employee Slate for Moveworks configuration.

7.  On the Configuration Summary menu follow through the steps.

    For a description of the field values, see [Employee Slate for Moveworks configuration menu](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/core-business-suite/es-moveworks-cbs-fsummary.md).

8.  Select **Mark as configured** after finishing configuration of each section.


## Result

After all sections are complete, the setup status on the Configuration Summary page shows 100% configured. Select **Package and download** to export the configuration.
