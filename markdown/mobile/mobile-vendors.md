# Supported mobile device management vendors

Learn how to configure mobile device management software from supported vendors to distribute ServiceNow mobile apps.

For more information, select one of the listed supported vendors.

-   **[Workspace ONE UEM Mobile Device Management \(MDM\)](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/mobile/airwatch-mdm.md)**  
Use VMware Workspace ONE UEM to distribute and manage the ServiceNow mobile apps on user devices.
-   **[BlackBerry Unified Endpoint Management \(UEM\)](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/mobile/blackberry-uem.md)**  
Use BlackBerry UEM \(Unified Endpoint Management\) as a mobile device management system to distribute and manage the ServiceNow mobile app on user devices.
-   **[Citrix XenMobile](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/mobile/citrix-mdm.md)**  
Configure Citrix XenMobile to distribute and manage the ServiceNow mobile app on user devices.
-   **[IBM MaaS360 mobile device management \(MDM\)](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/mobile/IBM-MaaS360-mdm.md)**  
Use IBM Maas360 to monitor and manage smartphones, tablets, and other mobile devices from a web-based portal. Add the ServiceNow mobile app to the IBM Maas360 portal to monitor and manage distribution.
-   **[Intune mobile device management \(MDM\)](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/mobile/intune-mdm.md)**  
Microsoft Intune uses Azure to manage mobile devices and apps. With some additional configuration, you can manage the ServiceNow mobile app in Intune.
-   **[Jamf Pro mobile device management \(MDM\)](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/mobile/jamf-mdm.md)**  
Use Jamf Pro to distribute and manage the ServiceNow mobile app on user devices.
-   **[MobileIron Mobile Device Management \(MDM\)](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/mobile/mobileiron-mdm.md)**  
MobileIron is a cloud-based service you can use to manage applications for your users. Users can access apps from MobileIron while you manage and secure content on the network.
-   **[Approve the ServiceNow app for Google Play](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/mobile/approve-app-google.md)**  
Android for Work may not be able to find the ServiceNow application. If that is the case, the ServiceNow App might not be approved for your organization. You can approve the app by finding it in the Google Play store.
