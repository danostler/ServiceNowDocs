# BlackBerry Unified Endpoint Management \(UEM\)

Use BlackBerry UEM \(Unified Endpoint Management\) as a mobile device management system to distribute and manage the ServiceNow mobile app on user devices.

## Get the BlackBerry UEM registered versions of ServiceNow mobile apps

**Note:**

You can configure a proxy for mobile apps in the BlackBerry UEM. This proxy is used for network communication in BlackBerry Dynamics apps such as ServiceNow mobile BlackBerry and ServiceNow Agent BlackBerry apps.

This proxy functionality is not available on both mobile operating systems. However, currently it is only supported on ServiceNow iOS apps. ServiceNow Android apps do not support handling proxy communication set in the BlackBerry UEM.
