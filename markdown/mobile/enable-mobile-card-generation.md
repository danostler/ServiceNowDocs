# Enable the mobile card generation skill

Activate the mobile card generation skill to let Now Assist generate mobile components using existing web components as a reference.

## Before you begin

Role required: admin

Make sure that Now Assist for Creator and Mobile App Builder AI is enabled in the instance. For more information, see .

## Procedure

1.  Navigate to **All** &gt; **Now Assist Admin** &gt; **Skills**.

2.  Select the **Creator** category.

3.  Use the search bar to find the **Mobile card generation** skill.

4.  Select **Turn on** to open the Mobile card generation skill.

5.  Select which user roles will have access to the **Mobile card generation** skill.

6.  Select **Turn on** again to activate the **Mobile card generation** skill.


## Result

The **Use Now Assist** option is now available in the web to mobile conversion flow in Mobile App Builder.

## What to do next

Use the web to mobile conversion skill in Mobile App Builder to generate a native mobile component. For more information, see [Web to mobile component conversion](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/mobile/web-mobile-component-conversion.md).
