# Using Now Assist for Mobile

Use Now Assist for Mobile to enhance the mobile user experience.

-   **[Generate work order resolution notes on mobile](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/mobile/generate-work-order-res-notes.md)**  
Generate resolution notes in your Mobile app that summarize your work order task record.
-   **[Using standard chat on mobile](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/mobile/using-standard-chat-mobile.md)**  
Standard chat on mobile delivers an intuitive, text-based support experience within a static chat window, enhanced by AI-powered responses and integrated knowledge resources. You can interact with a Virtual Agent to resolve issues, access knowledge articles, and request catalog items all in one place.
-   **[Using enhanced chat on mobile](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/mobile/using-enhanced-chat-mobile.md)**  
Enhanced Chat on mobile offers a dynamic conversational support experience that integrates powerful AI-driven search and chat capabilities. You can seamlessly switch between multiple conversations, receive synthesized responses with inline citations, and access both knowledge articles and catalog items.
-   **[Using Now Assist web search on mobile](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/mobile/using-na-web-search-mobile.md)**  
Web search mode for Now Assist lets you perform an internet search from within Now Assist, showing external results from outside your organization.
-   **[Generate a record summary on mobile](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/mobile/generate-a-record-summary-on-mobile.md)**  
Generate a summary of a record in your Now mobile app using Now Assist.
-   **[Using the prominent action button](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/mobile/using-prominent-action-button.md)**  
Use the prominent action button to quickly access your Now Assist agent chat no matter where you are in the app.
-   **[Using Voice Input for mobile](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/mobile/na-using-voice-to-text.md)**  
The Voice Input feature allows users to easily transcribe spoken words into written text. Voice-to-text can be accessed directly from the search bar or Now Assist chat.
-   **[Using People Genius results on mobile](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/mobile/using-people-genius-results.md)**  
Search for people in your organization and view their profiles by entering their name into your search bar.
-   **[Using Mobile AI Voice Agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/mobile/using-mobile-ai-voice-agent.md)**  
Use Mobile AI Voice Agent to start voice conversations with an intelligent, AI-powered voice assistant.
