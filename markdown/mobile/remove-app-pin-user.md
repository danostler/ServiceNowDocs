# Remove your PIN from a mobile app

Remove the pin from a ServiceNow mobile app.

## Before you begin

Role required: admin

## Procedure

1.  Log into a ServiceNow mobile app.

2.  On the navigation bar, tap the **Settings**.

3.  On the settings page, tap **Security**.

4.  Tap **Disable app PIN**.

    Your app no longer configured with a security PIN.
