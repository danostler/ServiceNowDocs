# Add optional prompts to the ServiceNow AI Lens Launcher

Add optional prompts to the ServiceNow AI Lens launcher button.

## Before you begin

Role required: admin

## Procedure

1.  Set the `show_custom_instructions_field` button attribute to **True**.

2.  Select the **New** button in the **Button Options** tab.

3.  Set the **Type** to **Prompt**.

4.  In the **Translated value** field, enter the prompt text in the user's language to ensure localization.

5.  Select **Submit** to save your progress.
