# Mobile App Builder record screen

Use the Mobile App Builder record screen as the workspace where you perform all your mobile configurations.

The layout of the Mobile App Builder record screen provides instant access to any record connected to the root-level record, selected from the categories home screen. When selecting a record from the ServiceNow AI Platform configuration tree, the fields applicable to that record are displayed in the center configuration panel. When you select different records from the ServiceNow AI Platform configuration tree, the content in the configuration panel changes.

The Mobile App Builder record screen consists of the following sections:

-   ServiceNow AI Platform configuration tree: Displays your selected record in a hierarchical display.
-   Configuration panel: Displays the field types from the selected record in the ServiceNow AI Platform configuration tree.
-   Mobile interface example panel: Highlights the UI element you are configuring on your mobile device.
