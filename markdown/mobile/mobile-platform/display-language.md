# Languages on your mobile device

Select a language for your ServiceNow mobile app that may differ from the language defined for your mobile device.

**Note:** This capability is available from ServiceNow mobile app version 16.0.

For a list of supported languages, see [Localization on mobile devices](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/mobile/mobile-platform/localization-mobile-device.md).

## Sources of languages for ServiceNow mobile apps

Languages displayed in ServiceNow mobile applications are sourced from two areas: the ServiceNow mobile app’s language and the selected ServiceNow instance language.

-   The ServiceNow mobile app language is the main language setting for the mobile app. Examples are texts like the main screen titles \(such as **Settings**\), and local button titles \(such as the **Clear All** button on the filters screen or the **See All** option in record sections\).
-   The ServiceNow instance language is the selected setting on your ServiceNow mobile app instance. These localized components from the instance include things like field labels, web content, and other data stored on the server in the **Translated** field.

**Note:** As the ServiceNow mobile app language and the instance language may differ, there may be a blending of languages within your ServiceNow mobile app. For a clearer user experience, both language settings should be the same.

There's an additional display of content within your ServiceNow mobile apps, which is manually added by you and isn't affected by either the mobile app language setting or the instance language setting. Examples of these texts are the inputs you manually enter into areas like forms, records, and notes within the activity stream.

## Support of right-to-left languages

The display of right-to-left languages is supported in ServiceNow mobile apps. As of the Washington DC release, Arabic and Hebrew are supported right-to-left languages. Text as well as visual components are right-to-left aligned to mirror the more common left-to-right alignment.
