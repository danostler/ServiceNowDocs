# Filtering and sorting results in mobile

Use mobile filter capabilities within list, map, and calendar screens to help you search for specific data and streamline your results. Sorting lets you define how to view the filtered results.

## Before you begin

Role required: user

## Procedure

1.  Tap the filter icon \(\[Omitted image "filter-icon.png"\] Alt text: Filter icon\) to filter your records.

2.  Apply filtering and sorting options to your list by performing the following actions.

<table id="table_dxb_bs3_5pb"><tbody><tr><td>

Select the category or categories you want to filter. Your administrator decides which categories are available.**Note:** If only one category is defined, the instance does not present the option to select categories.

 To see the filter conditions and sorting options that are common between all the displayed categories, tap **All**.

</td><td>

\[Omitted image "mobile-filters-1.png"\] Alt text: Mobile filter categories.

</td></tr><tr><td>

Select **Sort by** to display a list of options by which you sort your filtered results.

</td><td>

\[Omitted image "mobile-filters-sort-by.png"\] Alt text: Mobile filter sorting options.

</td></tr><tr><td>

Select how you want to view your filtered results. Your administrator sets the default value. After you make your selection:-   Tap the back button \(\[Omitted image "back-button.png"\] Alt text: Back button to return to the previous screen.\) to return to the filtering options.
-   Tap **Apply** to apply your filtering and sorting selections.


</td><td>

\[Omitted image "mobile-filter-sort-menu.png"\] Alt text: Mobile filter sorting option menu.

</td></tr><tr><td>

For date fields, tap the date field to display a calendar interface. The following actions are available:

-   Swipe to the left or right to view previous or future months.
-   To select or de-select a date, tap the day.
-   To search a date range, tap a day, then tap a second day. The filter searches for dates matching that range.


</td><td>

\[Omitted image "mobile-filters-3.png"\] Alt text: Date selection interface for a mobile filter.

</td></tr><tr><td>

For fields where you must select a value rather than enter a value, the available options display as check boxes. Tap one or more options to select or de-select, and then tap **Apply**.

</td><td>

\[Omitted image "mobile-filters-4.png"\] Alt text: Check box options for filters.

</td></tr><tr><td>

Select **Apply**, after you define your filtering and sorting options.

</td><td>

\[Omitted image "mobile-filters-5.png"\] Alt text: Filter with selected conditions

</td></tr></tbody>
</table>3.  Bookmark your filter and sorting selections for easy access. See, [Mobile saved views](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/mobile/mobile-platform/saved-views-mobile.md).
