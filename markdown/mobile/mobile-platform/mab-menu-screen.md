# Mobile App Builder categories home screen

Use the Mobile App Builder categories home screen to select or create the most commonly used configuration options. Select category options like **Screens** or **Cards &amp; Icons** to create and edit records.

The Mobile App Builder home screen contains the following elements.

-   The left menu panel groups the most important mobile components into categories. These menu items are fixed and you cannot add or remove them from the list.

    Depending on your category selection, different record types are displayed in the main area of the page. Use the scroll bar or search option \(\[Omitted image "mab-search-in-category.png"\] Alt text: Search option for category results.\) to find your exact record. Alternatively, to create records for your selected category, select the **New** button \(\[Omitted image "mab-buttton-new-green-solid.png"\] Alt text: New button in categories home screen.\).

    **Note:** The **All mobile records** menu item is a tool to quickly search and access record types familiar to you.

-   A help menu icon \(\[Omitted image "mab-help-icon.png"\] Alt text: Mobile App Builder help icon\) with the following options:
    -   **Record configuration tour**: Provides an on-screen step-by-step guide demonstrating how to create records.

        **Note:** The Record configuration tour menu option is only available in the Mobile App Builder record screen.

    -   **Mobile App Builder overview**: Opens the Mobile App Builder tour modal window.
    -   **App preview instructions**: Opens a modal window with instructions to help users download mobile apps and log in to their instances.
    -   **Go to product documentation**: Opens the ServiceNow mobile documentation in a new browser.
    -   **Version**: Displays the Mobile App Builder version for use during troubleshooting.
-   An option to change the application scope is displayed at the top of the screen. Select the **Change scope** button \(\[Omitted image "mab-button-change-scope.png"\] Alt text: Change scope button in the categories home screen.\) to return to the application scope selection screen, and select an alternative application scope to work in.

    **Note:** To ensure that records are not saved in the wrong application scope, you cannot edit a cross-scope record if there are any unsaved changes.
