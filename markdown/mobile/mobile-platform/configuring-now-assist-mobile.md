# Configuring Now Assist for Mobile

Install and configure skills for Now Assist in Mobile.

-   **[Configure mobile resolution notes generation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/mobile/mobile-platform/configure-resolution-notes.md)**  
Generate the resolution notes that summarize work orders for mobile.
-   **[Enable Now Assist Agent skills for Virtual Agent on Mobile](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/mobile/mobile-platform/config-na-agent-skills-mobile-va.md)**  
Enable Now Assist Agent skills in Virtual Agent on Mobile to leverage the power of generative AI.
-   **[Configuring a prominent action button](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/mobile/mobile-platform/configuring-prominent-action-button.md)**  
Configure a prominent action button for quick access to Now Assist.
-   **[Configure a Lens launcher button](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/mobile/mobile-platform/configure-lens-launcher-button.md)**  
Add a ServiceNow AI Lens button to input form screens to enable users to quickly extract data from attachments and complete forms faster.
-   **[Configure Mobile AI Voice Agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/mobile/mobile-platform/config-mobile-ai-voice-agent.md)**  
Follow these steps to enable the Mobile AI Voice Agent feature for your mobile app users.
-   **[Configure Now Assist button in search](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/mobile/mobile-platform/configure-now-assist-button-search.md)**  
Add a conversational chat button to standard catalog search results in mobile. The chat button redirects users to a new Virtual Agent conversation for eligible catalog items.
