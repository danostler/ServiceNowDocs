# Mobile interface preview panel within a record screen

The mobile interface preview panel helps you to identify the UI element you’re working on in the configuration panel within your mobile device. Record types with preview support display a live preview of how the components will appear on a mobile device instead of showing examples.

Some records may include multiple UI elements. To view these UI elements, select the relevant tab in the example panel.

Live component previews are available for the following components:

-   List screens
-   Mobile web screens
-   Launcher screens
-   Input form screens
-   Analytics previews and chart screens
-   Navigation tabs
-   Mobile app configurations
-   UI rules
-   Cards and card templates
-   Legacy cards
-   Icons
-   Empty states and pop-ups
