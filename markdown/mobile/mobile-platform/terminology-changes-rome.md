# ServiceNow mobile terminology changes

Starting from the Rome version ServiceNow mobile has updated the mobile terminology. The updated wording helps you better understand which items you are configuring within a mobile app, how the configurations affect the on-screen view, and how the items relate to one another.

**Note:** The name changes apply from Rome and later versions. All versions earlier than Rome use the earlier terminology.
