# Enabling location tracking for users

Enable location tracking for specified users. The same tracking location settings apply to all the selected users on the instance.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Users**.

    A list of all your users on your current instance displays.

2.  Select the user who requires location tracking.

3.  In the User form, select the **Geolocation** tracked field.

4.  Select **Update**.

5.  Repeat these steps for each user who needs location tracking.
