# App PIN settings for mobile

Improve the security of your ServiceNow mobile apps by setting a security PIN.

## Using a PIN with your mobile app

A security PIN is a 6-digit code you must enter for added security. When this feature is enabled, you are prompted to enter your pin:

-   When logging into your app
-   When the app has been inactive for five minutes
-   When the app is closed using your operating systems force quit or force close option.
-   When accessing the security settings for the app

If your mobile device uses faceID, touchID, or similar biometric security, you can use biometric authentication in place of the PIN.

For information for administrators on PIN settings for your instance, see [Require an app PIN for the mobile app](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/mobile/mobile-platform/require-app-pin.md).
