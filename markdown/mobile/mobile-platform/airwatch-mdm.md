# Workspace ONE UEM Mobile Device Management \(MDM\)

Use VMware Workspace ONE UEM to distribute and manage the ServiceNow mobile apps on user devices.

Visit the following sections to view the steps to configure the ServiceNow mobile app for iOS and Android.

You should have access to Workspace ONE UEM before attempting any configuration.

-   **[Add the ServiceNow app for iOS to Workspace ONE UEM](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/mobile/mobile-platform/add-ios-airwatch.md)**  
Configure the ServiceNow app for iOS for Workspace ONE UEM distribution.
-   **[Add the ServiceNow app for Android to Workspace ONE UEM](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/mobile/mobile-platform/add-android-airwatch.md)**  
Configure the ServiceNow app for Android for Workspace ONE UEM distribution.
