## Mobile Agent

Mobile Agent is used by agents or the fulfiller persona. This app delivers out-of-the-box mobile experiences for the most common service desk agent workflows, which makes it easy for agents to triage, act on, and resolve requests on the go. [Mobile Agent app demo](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/mobile/mobile-platform/sg-agent-demo.md) offers a curated demonstration of the Mobile Agent app, with a full range of personas and the controls for each role.
