# Using Now Assist web search on mobile

Web search mode for Now Assist lets you perform an internet search from within Now Assist, showing external results from outside your organization.

\[Omitted image "na-web-search.png"\] Alt text: Virtual Agent displaying internet search results

Web search mode is available for both standard and enhanced chat conversations as an opt-in feature via Now Assist Virtual Agent assistant. Select the Start web search mode icon \(\[Omitted image "na-web-search-icon.png"\] Alt text:\) to enter into web search mode. When in web search mode, all queries will be answered via internet search results rather than internal company search results. A banner that indicates you're in web search mode appears and will disappear when you disable web search mode.

To exit out of web search mode and return to internal search results, do one of the following:

-   Type **Exit** in the search mode chat
-   Select the **End** web search icon \(\[Omitted image "na-web-search-icon.png"\] Alt text:\)
-   Select **End** in the web search banner \(enhanced chat only\)
