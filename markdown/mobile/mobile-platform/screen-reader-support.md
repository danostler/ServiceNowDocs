# Configuring screen reader support in mobile

Screen reader support \(known as VoiceOver for iOS devices and TalkBack for Android devices\) in mobile provides an audible description of what is displayed on your mobile screen. These can be elements like the content of mobile cards, the icons of the navigation bar, or the data within a list screen.

Users enable screen reader support by navigating to the settings area on their mobile device and selecting the appropriate options.

For ServiceNow mobile apps, the screen reader automatically processes text to be read from either hard-coded text or the **Translation** fields. However, some of the elements that the screen reader accesses doesn't contain meaningful content. An example of this are button names that are defined as an icon type. In such circumstances there is an option to add an alternative text value.
