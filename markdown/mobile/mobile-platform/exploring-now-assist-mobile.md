# Exploring Now Assist for Mobile

With Now Assist for Mobile, your users can generate case resolution notes to wrap up cases faster, generate answers to search questions, and augment the Virtual Agent experience.

## Overview

The following generative AI capabilities are available for mobile:

-   Case resolution notes generation can help agents wrap up cases faster by summarizing the case details.
-   Now Assist in Mobile Virtual Agent uses generative AI to deliver more direct, relevant, and conversational responses to questions.
-   Now Assist in AI Search on Mobile improves the search experience with the ability to generate or pick out relevant answers using Now Assist

## Skills

The Now Assist for Mobile application includes the generative AI skills and features that enable users to generate resolution notes.

\[Omitted image "resolution-notes.png"\] Alt text: a generated closure note is presented to the user

With Now Assist on mobile devices, agents can generate resolution notes that summarize work orders instead of writing them out manually. Agents can change the notes before they save them to the work order record.

\[Omitted image "na-answer.png"\] Alt text: the user's question of "what is spam" is answered with a knowledge article on spam

With Now Assist in Virtual Agent, users can enjoy a better experience that combines AI Search and chat. These skills can help resolve issues faster in Virtual Agent and lower the need for a live agent.

\[Omitted image "na-mobile-search.png"\] Alt text: the query of "get a new phone" is answered with a generated answer and links to sources

Now Assist in AI Search on Mobile combines the power of search with the Now LLM Service generative AI model to give you AI-generated or AI-chosen answers relevant to your question.
