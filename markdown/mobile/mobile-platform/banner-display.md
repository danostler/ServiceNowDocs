# Define notification banner display time

Define the length of time a banner notification is displayed at the top of your screen. Increasing the display time provides additional time to read a message before it disappears, and for the screen reader to read out the entire message.

## Before you begin

Role required: none

## About this task

Select the length of time a notification banner displays at the top of your screen. The default value is three seconds. If you select to use the voiceover feature throughout your mobile device, the default time automatically increases to 10 seconds.

## Procedure

1.  Log in to a ServiceNow mobile app.

2.  On the navigation bar, tap the **Settings** button.

3.  In the **Settings** page, tap **Preferences**

4.  In the **Preferences** page, tap **Banners**.

5.  In the **Banners** page, select the time for a notification to display before it disappears.
