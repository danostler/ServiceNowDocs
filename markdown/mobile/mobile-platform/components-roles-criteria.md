# Mobile components where user roles and user criteria permissions apply

Lists of mobile components that can be applied to user roles, user criteria, or to both these access control mechanisms.

## Mobile components that support both user roles and user criteria

|Component|Table|
|---------|-----|
|Native Client|sys\_sg\_native\_client|
|Launcher screen|sys\_sg\_applet\_launcher|
|Record section|sys\_sg\_item\_section|
|Icon section|sys\_sg\_navigation\_section|
|Legacy icon section|sys\_sg\_icon\_section|
|Analytics section|sys\_sg\_chart\_section|
|Media section|sys\_sg\_media\_section|
|Content section|sys\_sg\_content\_section|

|Details screen|sys\_sg\_details\_screen|

|Map screen|sys\_sg\_map\_screen|
