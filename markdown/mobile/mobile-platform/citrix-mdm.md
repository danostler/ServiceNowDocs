# Citrix XenMobile

Configure Citrix XenMobile to distribute and manage the ServiceNow mobile app on user devices.

Use the following steps to distribute the ServiceNow mobile app using the Citrix XenMobile service. You can also configure the mobile app for your default ServiceNow instance for the app.

You should have access to Citrix before attempting any configuration.

-   **[Add the ServiceNow app to Citrix XenMobile](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/mobile/mobile-platform/add-mobile-app-citrix.md)**  
Configure the ServiceNow for Cetrix XenMobile distribution.
-   **[Configure the ServiceNow app for the default instance](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/mobile/mobile-platform/config-app-default-instance-citrix.md)**  
Configure apps that support managed configuration by deploying an XML configuration file to users' iOS devices.
