# Selecting languages on your mobile device

There are two areas where you define the language for your ServiceNow mobile apps: the instance language setup and the mobile app language setup.

**Note:** For the optimum user experience, the same language should be used for the instance and the ServiceNow mobile app language.

For a list of supported languages, see [Localization on mobile devices](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/mobile/localization-mobile-device.md).

## Defining the ServiceNow instance language

You can change the instance language by selecting the relevant language from the drop-down field in the mobile app login screen. If this field isn't displayed, this means that your instance has been preconfigured for a specific language and can't be changed.

\[Omitted image "mobile-app-login-language.png"\] Alt text: ServiceNow mobile instance login screen with language selection.

Alternatively, you can change the instance language from your mobile device, after you log in. Navigate to **Settings** &gt; **Preferences** &gt; **Language** &gt; **Instance language**. \[Omitted image "instance-language-flow.png"\] Alt text: Flow of images showing the selections made from the Settings menu to select the instance language

## Defining the mobile app language

You can change the mobile app language in the ServiceNow mobile app itself. The process differs for Android and iOS applications as shown.

## Language page in your ServiceNow mobile apps

For both the iOS and Android workflows when changing the language for ServiceNow mobile apps you're directed to the Language Preferences page in the ServiceNow mobile app. The Language Preferences page contains the following fields.

**Note:** You can only select languages that are available on your instance.

 The **Preferred Language** is the ServiceNow mobile app language and can be configured from this page.

 The **Instance Language** is the selected setting on your ServiceNow mobile app instance. These localized components from the instance include things like field labels and web content.
