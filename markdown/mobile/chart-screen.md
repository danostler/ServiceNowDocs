# Chart screen

Chart screens display data visualizations created in the Analytics Center and are displayed in the analytics section of your launcher screen. Adding data visualizations helps users more easily identify trends and turning points through indicator scores and visual representation.

Use analytics sections to display charts and data visualizations. Charts give your managers or owners indications on trends or items that require their attention.

The chart screen supports the following report types:

-   Time Series
-   Bar
-   Pie and donut

To use a chart screen, you will first need to create a dashboard preview to link to your chart screen. For details on dashboard previews, see [Mobile dashboard preview](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/mobile/mobile-dashboard-preview.md).
