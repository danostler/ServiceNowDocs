# Create a PIN for a mobile app

Create a PIN for your mobile app.

## Before you begin

Role required: admin

## Procedure

1.  Log into a ServiceNow mobile app.

2.  On the navigation bar, tap the **Settings**.

3.  On the settings page, tap **Security**.

4.  Tap **Require an app PIN**.

5.  When prompted Enter a 6 digit PIN for your app.

    \[Omitted image "app-pin.png"\] Alt text: App PIN.

6.  When prompted, re-enter the same 6 digit PIN.

    Your app is now configured with a security PIN.

7.  Enable **Unlock with Fingerprint Sensor** to use your device's fingerprint biometric security instead of your PIN.
