# Grouped list screen

Grouped list records display a list of records that are grouped by a specific field.

<table id="table_ibt_f4x_nlb"><tbody><tr><td>

Use a grouped list to provide a view of a list of items grouped by a field. Groups lists can help present a large number of records in a easily navigable format. For example, you can display a list of incidents grouped by category or priority. You can configure grouped lists to navigate to a second list showing the records in the selected group.
