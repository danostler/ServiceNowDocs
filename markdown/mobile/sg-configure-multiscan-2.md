# Create UI parameters for your grouped input

Create UI parameters to contain the values that your users scan in with your grouped input.

## Before you begin

Role required: admin

## Procedure

1.  In the **UI Parameters** field, click the lookup icon \(\[Omitted image "reference-lookup-icon.png"\] Alt text: Lookup icon\) to display the UI parameters list.

2.  Click **New**.

3.  On the form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Name|Descriptive name of your UI parameter|
    |Parameter type|Type of parameter. Select **None**|
    |Input type|Type of entry for the parameter value. Select **QR/Barcode**|

4.  Click **Submit**.

5.  Repeat steps 1 through 4 to add additional parameters.

    For example, if you are creating a record to track hardware, you may include UI parameters for a serial number and a model number.

6.  Click **Submit**.
