# Change the fetch type for a screen

Change the fetch type of a screen to change when the app loads its data.

## Before you begin

Role required: admin

## About this task

Screen fetch types typically do not need to be changed. Be sure to test fetch type changes thoroughly before using in a production environment, as fetch type changes can have a significant affect on performance.

## Procedure

1.  Navigate to **All** &gt; **System Mobile** &gt; **Screens**.

2.  Open a screen.

3.  In the **Fetch Type** field, select a fetch type.

    If you are in the global scope, the record appear read-only. In this case, you can temporarily switch to the appropriate scope by clicking on the banner notification that appears at the top of the record.

    \[Omitted image "change-scope-warning.png"\] Alt text: Scope notification message.

4.  Click **Update**.
