# TISC integration with SIR Workspace

TISC integration with SIR automatically attaches the context on the observables within the security Incident workspace for the Security analysts. Any new observables from SIR can be sent to TISC for further analysis by CTI team using the UI actions provides on the screens.

Additionally, any enrichment results that are produced from integrations only available in SIR can also be pushed using the UI actions provided in the workspace.
