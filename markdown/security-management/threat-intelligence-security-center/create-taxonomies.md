# Creating Taxonomies

Create taxonomies for the data source records.

## Before you begin

Role required: sn\_sec\_tisc.admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Threat Intelligence Security Center** &gt; **Administration** &gt; **Taxonomies**.

2.  Click **New**.

3.  Enter the **Name**.

4.  Enable or disable the taxonomy **Status** using the form actions Enable or disable.

    **Note:** Disabled taxonomies will not be available for selection of any records where taxonomies are applicable.

5.  Go to **Taxonomy Values** section to add taxonomy values.

6.  Click **New** to open new taxonomy value form view and Enter the **Taxonomy** and **Value**.

7.  Click **Save**.

8.  For example, navigate to **Threat Intel Library** &gt; **Observables** to verify the taxonomies for the observables record.

9.  Navigate to **Tags and Taxonomies** section in any observable form page and select the newly created taxonomy.

    **Note:** You can also add tags and taxonomies from the list view of the Threat Intel library.
