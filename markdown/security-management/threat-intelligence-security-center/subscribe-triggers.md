# Subscribe Triggers

View the list of all the subscribed webhook triggers for the current webhook.

## Before you begin

Role required: sn\_sec\_tisc.admin

## Procedure

1.  Navigate to **All** &gt; **Threat Intelligence Security Center** &gt; **Administration**.

2.  Select **Webhooks Configurations** &gt; **Webhooks**.

    The Webhooks page displays.

3.  Select the webhook.

    The Subscribed Triggers section is displayed.

4.  Click **Subscribe** button on the Subscribed Trigger section.

    The list view shows all the enabled webhook triggers in the instance.

5.  Select the desired webhook triggers and click **Subscribe** on the Webhook Triggers dialog box.

    A confirmation message is displayed that the webhook is subscribed successfully to the selected trigger\(s\). From here, you can also directly create a new trigger and subscribe the webhook.

    **Note:** You can also **Unsubscribe** the webhook from the selected trigger\(s\).
