# Archive TISC related records

Use the Archive Related Records related list for TISC to add the related records to the archive rule.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **System Archiving** &gt; **Archive Rules**.

2.  Filter the tables names starts with sn\_sec\_tisc.

3.  Go to **Archive Related Records** related list section to view the TISC related records.
