# Roll up of MITRE Techniques from Artifacts to Case

When intelligence records are added to a case, all associated MITRE Techniques are automatically rolled up to the case level.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Threat Intelligence Security Center**.

2.  Click the **Threat Analyst Workbench** icon.

3.  Go to **Case Management** &gt; **All Cases**.

    All the cases are displayed.

4.  Select any case.

5.  Go to **Artifacts** tab.

    The associated artifacts are displayed as the related lists for that specific case.
