# Shodan integration

Shodan is a search engine that analyzes service banner information from connected devices all around the globe.

The Shodan integration performs enrichment on the following observables:

-   IP addresses
-   URLs
-   Domains

-   **[Configure and enable Shodan integration](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/security-management/threat-intelligence-security-center/shodan-integration.md)**  
Before you use Shodan integration, you must download it from the ServiceNow Store.
