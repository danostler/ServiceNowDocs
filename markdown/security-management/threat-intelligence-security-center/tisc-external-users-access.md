# External user access for case tasks

This section describes the external user access for case tasks.

## Case tasks assignment to external users

Case tasks are assigned to the users with the external role even though they do not have access to sn\_sec\_tisc.analyst role.

1.  Any external user assigned with a case task can only view case task module and will have the privilege to edit the case task.
2.  The external user can only view the case tasks assigned to that specific user or the group\(s\) the user belongs to.
3.  The external user should be able to reassign the task within the group the user belongs to.
