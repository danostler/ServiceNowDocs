# View RSS Feeds

Access and review RSS feed configurations to monitor external content sources or troubleshoot feed connectivity issues.

## Before you begin

Role required: sn\_sec\_tisc.admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Threat Intelligence Security Center**.

2.  Select the **Integrations** icon.

3.  Select **RSS** option.

    This table shows the configured RSS feeds within the base system.

    |Threat Feed|Description|URL|
    |-----------|-----------|---|
    |CISO2CISO.COM RSS Feed|RSS Feed data from CISO2CISO.com|[https://ciso2ciso.com/feed/](https://ciso2ciso.com/feed/)|
    |Google Project Zero|Google Project Zero|[https://rssmock.free.beeceptor.com/getGoogleProjectFeed](https://rssmock.free.beeceptor.com/getGoogleProjectFeed)|
    |Schneier on Security RSS Feed|RSS Feed data from Schneier on Security|[https://www.schneier.com/feed/atom/](https://www.schneier.com/feed/atom/)|
    |Krebs on Security RSS Feed|RSS Feed data from KerbsonSecurity|[https://krebsonsecurity.com/feed/](https://krebsonsecurity.com/feed/)|
    |We Live Security RSS Feed|RSS Feed data from welivesecurity|[http://feeds.feedburner.com/eset/blog](http://feeds.feedburner.com/eset/blog)|
    |JISA Softech RSS Feed|RSS Feed data from JISA|[https://www.jisasoftech.com/feed/](https://www.jisasoftech.com/feed/)|
    |Cisco Newsroom Security RSS feed|Latest Security News from cisco.|[https://newsroom.cisco.com/c/services/i/servlets/newsroom/rssfeed.json?feed=security](https://newsroom.cisco.com/c/services/i/servlets/newsroom/rssfeed.json?feed=security)|
    |HelpNet Security|Information security news with a focus on enterprise security.|[https://www.helpnetsecurity.com/feed/](https://www.helpnetsecurity.com/feed/)|
    |Security Affairs|Information on the digital forensics from top-tier media.|[https://securityaffairs.com/feed](https://securityaffairs.com/feed)|
    |Volexity RSS Feed|RSS Feed data of information security blogs from Volexity.|[https://www.volexity.com/blog/feed/](https://www.volexity.com/blog/feed/)|
    |Hackers News RSS Feed|RSS Feed data from The Hacker News|[https://feeds.feedburner.com/TheHackersNews?format=xml](https://feeds.feedburner.com/TheHackersNews?format=xml)|
    |Graham Cluley RSS Feed|RSS feed data from Graham Cluley|[https://grahamcluley.com/feed/](https://grahamcluley.com/feed/)|

4.  Select **Edit** to edit the feed to make the necessary updates.

5.  Select **Save** to apply the changes.
