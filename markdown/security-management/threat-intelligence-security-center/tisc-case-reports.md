# View Case Reports

View Case Reports.

## Before you begin

Role required: sn\_sec\_tisc\_analyst

## About this task

Case Reports provide a structured way to document and present intelligence tied to a specific case. Using templates designated with a Case report context, analysts can generate reports that automatically draw from the fields, related records, and data within a selected case.

## Procedure

1.  Navigate to **Workspaces** &gt; **Threat Intelligence Security Center** &gt; **Threat Intel Library**.

2.  Select **Reports** &gt; **Case Reports**

    This page displays the list of case reports.

3.  Select any report.

4.  Select **Download** to download the report.

5.  Select **Share** to share the report in email format.

6.  Enter the recipient and select **Send**.

    **Note:** You can also preview the report before sharing.
