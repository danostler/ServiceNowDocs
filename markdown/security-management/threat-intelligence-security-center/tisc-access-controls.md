# Access controls for a case

Allow access controls specific to a case.

|Role|Access|Description|
|----|------|-----------|
|sn\_sec\_tisc.restriction\_access\_manager|Read access|External Users added here will get access to view the case form but will not be able to edit any field in the case or perform any action.|
|Privileged access|External Users added here will get access to edit any field in the case form but will not be allowed to run orchestrations.|
|sn\_sec\_tisc.special\_access\_read\_manager|Read access|External Users added here will get access to view the case form but will not be able to edit any field in the case or perform any action.|
|sn\_sec\_tisc.special\_access\_write\_manager|Privileged access|External Users added here will get access to edit any field in the case form but will not be allowed to run orchestrations.|
