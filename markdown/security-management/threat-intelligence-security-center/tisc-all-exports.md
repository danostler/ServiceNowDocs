# View all exports

Use this section to view all the manually exported records.

## Before you begin

Role required: sn\_sec\_tisc.export

## Procedure

1.  Navigate to **Workspaces** &gt; **Threat Intelligence Security Center** &gt; **Imports/Exports**.

2.  Select **Exports** &gt; **All Exports**

    This page displays the export records.

3.  Open any record to view the status of the export process.
