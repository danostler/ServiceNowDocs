# Destroy Rules in TISC

View the destroy rules that are provisioned in the base system.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **System Archiving** &gt; **Archive Destroy Rules**.

    The list of destroy rules that are applicable for TISC are displayed. These destroy rules are different for every object type and are applicable independently.

2.  View the applicable destroy rules for TISC.

3.  Click on any rule to view the archive duration period.
