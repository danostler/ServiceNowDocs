# Check XML attachment property size

Verifies that the XML attachment property is sufficient for large files.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **System Properties** &gt; **Import Export**.

2.  Scroll down to **Import Properties** &gt; **XML Format** at the bottom of the page.

    \[Omitted image "XMLFileSize.jpg"\] Alt text: Maximum file size for import

3.  If necessary, change the value to 250 and click **Save**.
