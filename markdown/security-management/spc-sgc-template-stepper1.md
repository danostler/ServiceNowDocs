# Enter metadata

Fill in the metadata for your service graph connector.

## Before you begin

Role required: sn\_sec\_spc\_core.developer

## Procedure

1.  Navigate to **Workspaces** &gt; **Security Posture Control** &gt; **Connector builder**.

    The Custom API connectors are displayed.

2.  Select an existing connector record from the Connector name column to edit it.

3.  To create a record, select **New connector**.

    A tab opens in your browser and the required fields for the Enter metadata step are displayed.

4.  Fill in the fields.
