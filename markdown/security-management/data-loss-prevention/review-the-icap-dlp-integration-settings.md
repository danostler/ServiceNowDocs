# Review the ICAP DLP integration settings

Review the provider ICAP DLP integration settings and modify the default system properties to suit your environment.

## Before you begin

Role required: sn\_dlir.admin

## Procedure

1.  Navigate to **All** &gt; **ICAP DLP Integration** &gt; **Settings**.

2.  Review and modify the settings are required.

    |Property name|Description|
    |-------------|-----------|
    |**sn\_icap\_dlp.maximum\_retry**|Option to chose maximum number of retries to be performed for AWS Get object failures.|
    |**sn\_icap\_dlp.logging.verbosity**|Define the level of reporting the logs generated by the integration. The default value is set as 4.|
    |**sn\_icap\_dlp.enable\_file\_download**|Option to enable this property allows downloading the evidence file of the reported ICAP incident. By default, this option is enabled.|
    |**sn\_icap\_dlp.presigned\_file\_expiry\_time**|Specify the duration in seconds, for which the presigned file will be available for download or preview. The file will only be accessible within this time frame. This field is automatically set to 300.|
