# Configure the Symantec DLP  integration settings

Modify the  Symantec DLP  integration default system properties.

## Before you begin

Role required: sn\_dlir.admin

## Procedure

1.  Navigate to **Symantec DLP Integration** &gt; **Settings**.

2.  Modify the following settings as required.
