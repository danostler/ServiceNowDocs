# Configure DLP UI user instructions

Configure the system UI messages to add detailed user instruction headers as required.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **System UI** &gt; **Messages**.

2.  Search for **user\_instruction\_card\_heading**.

    By default, user\_instruction\_card\_heading anduser\_instruction\_modal\_window\_heading are the two UI message configurations provisioned to the users.

3.  Click on any of the user instruction to update or modify.
