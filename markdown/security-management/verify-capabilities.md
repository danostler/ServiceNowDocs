# Verify capabilities in ServiceNow Studio

Verify the capabilities that you have added using Now Assist for Security Incident Response integrations and update the scripts for any required changes.

## Before you begin

Role required: sn\_si\_int\_kit.integration\_creator

## Procedure

1.  On the Review Integration page, select **Open integration in studio**.

    The published integration opens in ServiceNow Studio.

2.  Select your application.

3.  Select **Action**.

    All the flow actions for the application are listed.

    1.  Select each action and verify it.

    2.  Make any necessary changes.

    3.  Select **Save**.

4.  Select **Capability Implementation**.

    1.  Verify each added capability.

    2.  Make any necessary changes.

    3.  Select **Update** to save the changes.

5.  Select **Script Include**.

    Lists the script include for each capability in the application. In each script include, the run\(\) method is the starting point.

    1.  Verify the script include for each capability.

    2.  Make any necessary changes.

    3.  Select **Update** to save the script include.
