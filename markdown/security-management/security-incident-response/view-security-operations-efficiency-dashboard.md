# View Security Operations Efficiency dashboard

Security operations center \(SOC\) managers can view overall efficiency metrics and measure the individual performance of the SOC team members in the organization.

## Before you begin

In the current release, the following report is not supported in the Security Operations Efficiency dashboard:

-   Security Incident stage Analysis - Workbench is not available

Role required: sn\_si.analyst \(Read\), sn\_si.manager or sn\_si.admin \(write\)

## Procedure

1.  Navigate to **Workspaces** &gt; **Security Incident Response**.

2.  Click **SIR Dashboards** icon, displayed on the left side of the workspace home page.

3.  Select **Security Operations Efficiency** from the drop down list.

    \[Omitted image "security-operations-efficiency-dashboard.png"\] Alt text: Security Incident Management dashboard

    View the Security Operations Efficiency dashboard for further action. For detailed information on Security Operations Efficiency dashboard, see [Security Operations Efficiency dashboard](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/security-management/security-incident-response/si-explorer-dashboard.md)
