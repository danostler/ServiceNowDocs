# Activate the Check Point NGTP integration

If you have not installed the application, follow the instructions to install it.

## Before you begin

Role required: admin

## Procedure

1.  Log in to the instance you want to install the application on.

2.  Navigate to **System Applications** &gt; **Applications**.

3.  Click the **Downloads** tab.

    A list of applications available for installation is displayed.

4.  Locate the Check Point Next Generation Threat Prevention application, select it, and click **Install**.

    Your application is automatically installed onto your instance.
