# View Security Incident Manager Overview dashboard

This dashboard provides managers with a high-level overview of the critical or high priority security incidents at a team level, SLAs that are about expire in 24 hours, and weekly incidents trends on the instance in the form of reports and graphical charts.

## Before you begin

Role required:

-   sn\_si.admin - Edit and view dashboard.
-   sn\_si.manager - View dashboard.

## About this task

You can analyze the critical or high priority security incidents, or SLAs that needs immediate attention. You can also visualize the weekly trends using the widgets in the Security Incident Manager Overview dashboard. The data or statistics for the weekly or monthly trends of your security incidents are collected from the incidents' tables that are defined by the organization. You can gain key insights by using the following charts:

-   **Team Critical Security Incidents**: This tile displays the number of critical security incidents for the team this week.
-   **Team High Security Incidents**: This tile displays the number of high security incidents for the team this week.
-   **SLAs expiring within 24 hours**: This tile displays the number of SLAs that will be expiring within 24 hours.
-   **Average Time to Identify**: This tile displays the average weekly time taken to identify security incidents.
-   **Average Time to Contain**: This tile displays the average weekly time taken to contain security incidents.
-   **Average Time to Contain Critical**: This tile displays the average weekly time taken to contain critical security incidents.
-   **Security Incident by CI Class, last 3 months**: This chart illustrates the trends of all the security incidents based on the CI Class for the last three months.
-   **Trend of All Security Incidents by Category**: This chart illustrates the trends of all the security incident based on category.
-   **Trend of All Security Incidents by Priority**: This chart illustrates the trends of all the security incident based on priority.

## Procedure

1.  Navigate to **All** &gt; **Workspaces** &gt; **Security Incident Response**.

2.  Click the **SIR Dashboards** icon on the left side of the workspace homepage.

3.  Select **Security Incident Manager Overview** from the drop-down list.

    \[Omitted image "security-incident-manage-dashboard.png"\] Alt text: Security Incident Manager Overview dashboard

4.  View the Security Incident Manager Overview for further action.
