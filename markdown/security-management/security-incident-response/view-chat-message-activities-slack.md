# View chat message activities in Slack

Chat message activity is recorded and displayed on the Collaboration tab of the Major Security Incident Management workspace.

## Before you begin

Role required: sn\_msi.workspace\_admin

## About this task

The chat message activity includes all the chat conversations occurred between the participants, different user groups. For more information, see [Manage Chat Channels communications using Slack](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/security-management/security-incident-response/collab-tab.md).
