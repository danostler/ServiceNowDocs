# View CISO Reporting Overview dashboard

This dashboard provides the Chief Information Security Officers \(CISOs\) with a high-level reporting overview of the security incidents and weekly incidents trends on the instance in the form of graphical charts. These charts help you effectively view and analyze how the Security operations center \(SOC\) performs.

## Before you begin

Role required: sn\_si.admin and sn\_si.ciso - Edit and view incidents.

## About this task

The CISO Reporting Overview dashboard provides key insights about the security incidents on a weekly basis, and you can visualize the weekly trends using widgets. You can gain key insights on the CISO Reporting by using the following charts:

-   **New Security Incidents this Week**: This tile displays the report of the number of new security incidents this week.
-   **Security Incidents Closed this Week**: This tile displays the report of the number of security incidents closed this week.
-   **New Security Incidents \(Running 7 days\)**: This tile displays the report of the number of new security incidents which are running for a week.
-   **Security Incidents Closed \(Running 7 days\)**: This tile displays the report of the number of new security incidents closed but are running for a week.
-   **Average Time to Identify**: This tile displays the report of the average weekly time taken to identify security incidents.
-   **Average Time to Contain**: This tile displays the report of the average weekly time taken to contain security incidents.
-   **Average Time to Contain Critical**: This tile displays the report of the average weekly time taken to contain critical security incidents.
-   **Weekly New Security Incidents**: This chart illustrates the report of the weekly new security incidents.
-   **Weekly Closed Security Incidents**: This chart illustrates the report of the weekly closed security incidents.
-   **Security Incident Closed Codes**: This chart illustrates the report of the closed codes of the security incidents.

## Procedure

1.  Navigate to **All** &gt; **Workspaces** &gt; **Security Incident Response**.

2.  Click the **SIR Dashboards** icon on the left side of the workspace homepage.

3.  Select **CISO Reporting Overview** from the drop-down list.

    \[Omitted image "ciso-report-dashboard-sir.png"\] Alt text: CISO Reporting Overview dashboard

4.  View the CISO Reporting dashboard for further action.
