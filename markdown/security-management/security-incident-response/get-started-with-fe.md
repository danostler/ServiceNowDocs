# Get Started with File Explorer

Review the following information before you start working with File Explorer Component.

Review the following information before you start establishing the connection with Microsoft SharePoint, creating folder and file actions, and creating folder templates for the file explorer component.

Microsoft SharePoint spoke, version 2.0.2 for ServiceNow® IntegrationHub.

Use **sn\_sp\_spoke** plugin ID to search and install Microsoft SharePoint spoke.

Installing this plugin provides flows and actions to automate collaboration, document management, and user access rights in Microsoft SharePoint.

</td></tr><tr><td>

File Explorer Component v1.1.0

</td><td>

Use **sn\_file\_explorer** plugin ID to search and install File Explorer Component v1.1.0

Installing this plugin provides a graphical user interface to browse the file system, which is available in the file explorer core.

</td></tr><tr><td>

File Explorer Core for Security Operations v1.1.1

</td><td>

Use **sn\_fe** plugin ID to search and install File Explorer Core.

Installing this plugin provides base infrastructure to integrate with any third-party file hosting provider.

</td></tr><tr><td>

Microsoft SharePoint File Explorer Connector for Security Operations v1.1.1

</td><td>

Use **sn\_fe\_sharepoint** plugin ID to search and install Microsoft SharePoint File Explorer Connector.

Installing this plugin will manage the Microsoft SharePoint Files/Folders using File Explorer.
