# Link to Major Security Incident

Link a security incident to a major security incident as child security incident\(s\).

## Before you begin

Role required: sn\_si.analyst.

## Procedure

1.  Navigate to **Workspaces** &gt; **Security Incident Response Workspace**.

2.  Open any security incident.

3.  Click **Link to Major Security Incident**.

    \[Omitted image "link-si-to-msi.png"\] Alt text: Link incident

4.  Select the incident\(s\) to link to a major security incident.

5.  Click **Link**.

6.  The incident record is linked as a child incident of the major security incident.

    After the incident is linked as a child incident to a major security incident, **Major Security Incident** tag is applied and displayed on the form banner.
