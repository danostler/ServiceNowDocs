# Preview the Report Template

Use the Preview functionality to preview your design-time report template to generate and share the report with other stakeholders during run time from the Major Security Incident Management workspace.

## Before you begin

Role required: sn\_msi.workspace\_manager

## Procedure

1.  Navigate to **Major Security Incident Management** &gt; **MSI Administration** &gt; **Status Report Setup**.

    The MSI Status Report Setup page displays.

2.  In the Report Template section, select **Configure**.

3.  Select the desired Report Template.

4.  Select the Report Type as **Email** or **PDF** based on your requirement.

5.  Select the **Save and Preview Report** action.

    The Preview Report dialogue box displays.

6.  Select a major security incident using the Search option to preview a report with the selected template.

7.  Select **Preview Report**.

    The report generates a preview according to the selected template configuration. If you selected the Report Type as **Email**, then the preview is in email template format. The status reports in email format are mobile-friendly.
