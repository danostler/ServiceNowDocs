# View Context Sensitive Analytics - SI dashboard

With this dashboard, managers and analysts can view the open security incidents, the average age of open Security Incidents, the average close time of security incidents, the percentage of security incidents that were opened and closed on the same day, and the percentage of the incidents that were not updated in the last 5 days and 30 days.

## Before you begin

Role required: sn\_si.analyst \(Read\), sn\_si.manager or sn\_si.admin \(write\)

## Procedure

1.  Navigate to **All** &gt; **Workspaces** &gt; **Security Incident Response**.

2.  Click **SIR Dashboards** icon, displayed on the left side of the workspace home page.

3.  Select **Context Sensitive Analytics – SI** from the drop down list.

    \[Omitted image "context-sensitive-analytics-si.png"\] Alt text: Context Sensitive Analytics - SI

4.  View the Context Sensitive Analytics – SI dashboard for further action.
