# Duplicate a report template in Security Incident Response

Duplicate a report template to edit or reuse an existing template to create a new template.

## Before you begin

**Note:** You have to create a copy of the Executive Summary and Analyst Status Reporting templates to reuse them.

To reuse the Executive Summary and Analyst Status Reporting, you have to create a copy of these and then

Role required: sn\_si.admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Security Incident Response Workspaces** &gt; **Administration**.

2.  Select **Reports**, and then **Report Templates**.

    The Report Templates page appears with all the templates available in the base system.

3.  Select the template.

4.  Select **Duplicate**.


## Result

The contents of the selected template are duplicated and copy of the selected template is available in the list of templates. The duplicate template is in the Draft state. You can customize the contents, and then rename the template as required.
