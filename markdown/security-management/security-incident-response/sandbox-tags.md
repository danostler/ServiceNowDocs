# Tag security incidents with the Sandbox submission status

Tag security incidents with the Submission Initiated and Submission Complete tags.

When you select the **Display tags** option in the [Sandbox submission configuration](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/security-management/security-incident-response/setup-sandbox-submission-configurations.md), tags are displayed to provide the status of a file or URL submission. Checking the status this way is useful because the sandbox analysis may take several minutes to process.

The display tags are called submission initiated, submission completed, and submission failed.
