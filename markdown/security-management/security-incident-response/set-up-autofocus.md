# Activate and configure Palo Alto Networks - AutoFocus integration

The Integration Configuration feature allows you to quickly activate and set up third-party security integrations, including Palo Alto Networks - AutoFocus. Before you can use the Palo Alto Networks - AutoFocus, you must download it from the ServiceNow Store.

## Before you begin

Role required: sn\_si\_admin

## Procedure

1.  [Download an application from the ServiceNow Store for the first time](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/security-management/security-operations/download-app-first-time.md).

2.  When the installation is complete, access Palo Alto Networks support site and obtain the API Key.

3.  In your instance, navigate to **Security Operations** &gt; **Integration Configuration**.

4.  In the Palo Alto Networks AutoFocus card, click **Configure**.

5.  Enter \(or paste\) the **API Key** you acquired from the Palo Alto Networks support site.

6.  Click **Submit**.
