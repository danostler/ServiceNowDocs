# View the Proofpoint Analytics Dashboard

The Proofpoint Analytics Dashboard provides details about VAP users and top clickers.

## About this task

Users mostly targeted for security attacks are considered VAP users. Employees who frequently click on suspicious and phishing links are considered top clickers.

## Before you begin

Role required: sn\_si\_admin

## Procedure

1.  Navigate to **All** &gt; **Workspaces** &gt; **Security Incident Response Workspace**.

2.  Select **SIR Dashboards**.

3.  Select **Proofpoint Analytics Dashboard** from the drop-down list.

4.  View information about VAP users.

    1.  View the list of VAP users by selecting the **VAP Users** tab.

    2.  View the details of a VAP user by selecting the user name.

5.  View information about top clickers.

    1.  View the list of top clickers by selecting the **Top Clickers** tab.

    2.  View the details of a top clicker by selecting the user name.
