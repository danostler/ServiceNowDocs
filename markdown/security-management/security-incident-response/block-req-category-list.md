# CrowdStrike Block Request Category List

Field descriptions for CrowdStrike Block Request Category List form.

| | |
|---|---|
|Hash Category|Defines whether this category list is used for allowing or blocking hashes|
|Source|Integration source associated with this CrowdStrike category list|
|Tag Type|Indicates whether the applied tag represents an allow or block action|
|Display Tag|Controls whether the tag is displayed on the observable and related security incidents|
|CrowdStrike Tag For Observable|Security tag applied to the observable in CrowdStrike|
|Active|Indicates whether this block request list is currently active|
|Create Change Request|Determines whether a Change Request is automatically created when this category is applied|
|Change Request|Change Request generated to activate or modify this category list|
|Require Approval|Indicates whether approval is required before adding or removing observables|
|Description|Additional details or clarification regarding the purpose of this category list|
