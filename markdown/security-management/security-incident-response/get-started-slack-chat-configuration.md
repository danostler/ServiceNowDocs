# Get started with Slack chat connector configuration

Review the following information before you start working with Slack as a connector application.

Make sure that the following plugins are installed, before you start establishing, activating, and validating Slack set up connections.

<table id="table_a2v_qm1_rpb"><thead><tr><th>

Name

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Slack Chat Connector for Security Incident Management

</td><td>

Use **sn\_slack\_connector** plugin ID to search and install Slack Connector for Major Security Incident Management.

Installing this plugin allows you to track the chat activities using the collaboration component specific to Slack.

</td></tr><tr><td>

Slack spoke

</td><td>

Installing this plugin establishes a spoke connection with Slack. For more information, see [Slack Spoke](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/integrate-applications/integration-hub/slack-spoke.md)

</td></tr></tbody>
