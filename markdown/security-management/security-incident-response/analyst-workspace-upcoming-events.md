# View upcoming tasks

The Security Analyst can view the upcoming tasks related to the Security Incidents and Response Tasks that are due, overdue or breached the defined SLA.

## Before you begin

Role required: sn\_si.analyst

## Procedure

1.  Navigate to **Workspaces** &gt; **Security Incident Response Workspace** &gt; **Upcoming**.

2.  Click **Today** tab to view the tasks that are due on the same day.

3.  Select **Tomorrow** tab to view the tasks that are overdue to the next day.

    The upcoming section lists the security incidents or response tasks that are overdue based on the due date and that breached the SLA definition.

4.  Click **Refresh** icon to refresh the upcoming section list.

    For example, if you click on the link, **More than 10 security incidents with breached SLA** then the list of breached SLA incidents are displayed in a new tab within the workspace.

    **Note:** You can perform New, Assign, Export, and Delete operations from any list view.
