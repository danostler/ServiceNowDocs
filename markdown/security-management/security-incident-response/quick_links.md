# Working with quick links

Add quick links for easy reference.

## Before you begin

Role required: sn\_si.analyst

## About this task

You can add and modify any useful quick links and bookmark those links that helps in the incident resolution.

## Procedure

1.  Navigate to **Workspaces** &gt; **Security Incident Response Workspace** &gt; **Quick links**.

2.  Click **Add Quick Link** icon to add the links.

3.  Enter the link to the URL and Label.

4.  Click **Save Link**.

    \[Omitted image "add-quick-links.png"\] Alt text: Add quick links

    The new link gets added below the quick links section.

5.  **To modify or delete the links:**

    1.  Click **Edit Link** icon.

    2.  Modify the desired link or display text which is the label of the URL.

    3.  Click **Save**.

    4.  Click **Done**.

        \[Omitted image "modify-quick-links.png"\] Alt text: Modify quick links

    5.  Use the upward and downward arrows to move the position of the links.

    6.  Click **Delete** to delete the links while editing that were bookmarked.
