# Activate MS Teams as a chat provider

Activate Microsoft Teams by validating the Microsoft Teams Connection.

## Before you begin

Role required: sn\_msi.workspace\_admin

Once the connection is activated and validated, an information message is displayed that the Provider connection is validated successfully. You can modify the channel configuration by adding and removing users, and possibly adding more channels or renaming existing channels.

## Procedure

1.  Navigate to **Major Security Incident Management** &gt; **Chat Provider Configuration**.

2.  Select **Microsoft Teams record**.

3.  Click **Validate Connection**.

    A confirmation message displays that, Provider connection is validated successfully.

4.  Select **Active** check box to activate the provider record.

    A confirmation message displays that the provider connection is validated successfully.

5.  **Save** the form after you activate the chat provider configuration.

    With this connection between Microsoft Teams and ServiceNow AI Platform Instance is successfully established.
