# Assign post incident review roles

You can target questions to specific pre-defined groups by assigning roles to Post Incident Review \(PIR\) categories.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Assessments** &gt; **Metric Definition** &gt; **Types**.

2.  Under the **Types** column, search for a **Post Incident Review** record.

3.  Open the record.

4.  Assign one or more roles for this category of questions.

5.  Select **Update**.

6.  Navigate to **User Administration** &gt; **Users**.

7.  Choose a user

8.  Add one or more metric type roles to the user record.

    **Note:** The roles must correspond to roles assigned in the Post Incident Review category.
