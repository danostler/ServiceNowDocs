# View Security Analyst Overview dashboard

With this dashboard, security analysts can view security incidents summarized based on analysts’ critical priority work, high priority work, security Incidents that are assigned to the analyst, tasks assigned to the analysts, and incident count.

## Before you begin

By default, the Security Analyst Overview dashboard is displayed for the users with sn\_si.analyst access.

Role required: sn\_si.analyst \(Read\), sn\_si.manager or sn\_si.admin \(Write\)

## Procedure

1.  Navigate to **Workspaces** &gt; **Security Incident Response**.

2.  Click **SIR Dashboards** icon, displayed on the left side of the workspace home page.

3.  Select **Security Analyst Overview** from the drop down list.
