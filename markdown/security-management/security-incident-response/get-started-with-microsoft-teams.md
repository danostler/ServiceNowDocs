# Get started with Microsoft Teams

Review the following information before you start working with Microsoft Teams as a connector application.

Make sure that the following plugins are installed, before you start establishing, activating, and validating Microsoft Teams set up connections.

<table id="table_a2v_qm1_rpb"><thead><tr><th>

Name

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Chat integration with Security Incident Management

</td><td>

Use **sn\_chat\_collab\_core** plugin ID to search and install Microsoft Teams Chat.

Installing this plugin allows you to track the chat activity for third-party applications such as Microsoft Teams \(or Slack and other third-party chat applications\).

</td></tr><tr><td>

Microsoft Teams Chat Connector for Security Incident Management

</td><td>

Use **sn\_ms\_teams\_connector** plugin ID to search and install Microsoft Teams Chat Connector for Major Security Incident Management.

Installing this plugin allows you to track the chat activities using the collaboration component specific to Microsoft Teams.

</td></tr><tr><td>

Microsoft Teams Graph spoke

</td><td>

Installing this plugin establishes a Graph spoke connection with Microsoft Teams. For more information, see [Microsoft Teams Graph spoke](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/integrate-applications/integration-hub/msteams-spoke.md)

</td></tr></tbody>
