# Add groups to a role

Assign groups to product-specific roles in the Security Exposure Management Workspace. Only explicit assignments are managed through this interface.

## Before you begin

Role required: admin

## About this task

Use this procedure to assign groups to roles for products configured in the Security Exposure Management Workspace. Only groups explicitly assigned to the role are managed here. Groups that inherit the role through hierarchy or other roles are excluded. The Users and Groups page displays products and roles based on installed plugins and feature configuration.

## Procedure

1.  Navigate to **Workspaces** &gt; **Security Exposure Management Workspace**.

2.  In the navigation pane, select **Administration**.

3.  Under **Others** section, locate **Users and Groups**.

4.  Select **Review**.

5.  Select the appropriate product.

6.  Locate the role to which you want to assign groups.

7.  Select the **Groups** option for that role.

8.  In the **Edit groups** window, select groups from the Collection list and move them to the Groups list.

9.  Close the modal to save the changes.


## Result

The group count updates automatically when the modal is closed. Only groups explicitly assigned to the role through this interface are displayed and managed here.
