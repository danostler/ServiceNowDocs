# Add users to a role

Assign users to product-specific roles using the interface in the Security Exposure Management Workspace.

## Before you begin

Role required: admin

## About this task

Use this procedure to explicitly assign users to product-specific roles. Only explicit role assignments made through this page are managed here. Role inheritance through other roles or group membership is not modified.

## Procedure

1.  Navigate to **Workspaces** &gt; **Security Exposure Management Workspace**.

2.  In the navigation pane, select **Administration**.

3.  Under **Others** section, locate **Users and Groups**.

4.  Select **Review**.

5.  Select the appropriate product.

6.  Locate the role to which you want to assign groups.

7.  Select the **Users** option for that role.

8.  In the **Edit groups** window, select groups from the Collection list and move them to the Users list.

9.  Close the modal to save the changes.

    **Note:**

    -   Only users explicitly assigned to the role are managed here.
    -   Users who inherit the role through another role or group are not shown.

## Result

The users are assigned to the role. The user count updates automatically when the modal is closed.
