# Define object-object relationships

Define relationships between SDOs, except the indicator object.

## Before you begin

Role required: sn\_ti.admin

## Procedure

1.  Navigate to **All** &gt; **Threat Intelligence** &gt; **IoC Repository** &gt; **Object-Object-Relationships**.

2.  Click **New**.

3.  Complete the fields in the form as appropriate.

    |Field|Description|
    |-----|-----------|
    |**Source Object**|Select and define the source object.|
    |**Source Object Type**|Defines the type of source object.|
    |**Target Object**|Select and define the target object.|
    |**Target Object Type**|Defines the type of target object.|
    |**Relationship Type**|A description that provides more details and context about the relationship type.|
    |**Source**|Specifies the threat source from which this record is created.|
    |**Source ID**|Uniquely identifies the source object.|

4.  Click **Submit**.
