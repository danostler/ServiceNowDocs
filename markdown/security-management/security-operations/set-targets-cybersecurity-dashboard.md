# Set targets

Set targets in days to remediate the vulnerabilities, security incidents, and misconfigurations.

## Before you begin

Role required: admin

## Procedure

1.  Select **Targets** on the widget or select the **Targets** icon on the KPI Details page.

2.  Select the **For everyone** button.

3.  In the **New value** field, enter the number of days in which you want to achieve the target.

4.  Select the target start date.

5.  Select the target review date.

6.  Select **Update**.
