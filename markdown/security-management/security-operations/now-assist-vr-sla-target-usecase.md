# Vulnerability Response AI agent collection for the Analyze vulnerability remediation status agentic workflow

The tools and AI agents for the Analyze vulnerability remediation status agentic workflow.

## Analyze vulnerability remediation status agentic workflow overview

Use the Analyze vulnerability remediation status agentic workflow from the Now Assist panel to help you:

-   Gain insights into your compliance metrics and statistics for how well you’re meeting remediation target dates on vulnerable item \(VIT\) records.
-   View your monthly VIT record remediation totals and identify missed targets.
-   Break down remediation data on VITs by **Severity**,**Assignment group**, **Configuration item** and **Vulnerability** for your monthly Service Level Agreement \(SLA\) compliance reviews.

If you want to modify this agentic workflow, you can [duplicate](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/intelligent-experiences/enable-ai-experiences/clone-aia-usecase.md) it, adjust the settings to suit your specific needs, and activate the duplicated version of the agentic workflow.

## Agent used in the Analyze vulnerability remediation status agentic workflow

The Remediation Compliance Insights Provider is used by the Analyze vulnerability remediation status agentic workflow.
