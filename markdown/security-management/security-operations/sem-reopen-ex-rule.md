# Reopen an exception rule

Reopen an exception rule that has been rejected, but you want to resubmit. Reopening the rule moves it to the Draft state.

## Before you begin

Role required: sn\_vul.vulnerability\_admin

## About this task

You can then update and resubmit the rule for approval.

## Procedure

1.  Navigate to **Workspaces** &gt; **Security Exposure Management Workspace**.

2.  Select **Administration** in the navigation pane.

3.  Select **Review** on the **Exception rules** tile.

4.  Find the rule that you want to reopen and click **Reopen**.

5.  Update the form and resubmit it for approval.
