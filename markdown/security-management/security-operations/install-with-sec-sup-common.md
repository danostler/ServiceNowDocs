# Components installed with Security Support Common

Several types of components are installed with Security Support Common. They provide common functionality for use across the various security applications, such as Security Incident Response.

## Locating and viewing installed components

See [View components installed with Security Support Common](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/security-management/security-operations/installed-with-secops-common-NEW.md) more information about locating and viewing installed components.

-   **[View components installed with Security Support Common](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/security-management/security-operations/installed-with-secops-common-NEW.md)**  
Several types of components are installed when you activate the Security Support Common plugin including but not limited to tables, user roles, and modules.
