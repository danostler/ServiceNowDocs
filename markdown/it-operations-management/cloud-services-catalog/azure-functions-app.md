# Microsoft Azure Functions App Catalog items

Cloud Services Catalog Microsoft Azure Function App with pay-as-you-go pricing benefit.

This is a fully functional sample catalog item that enables users to request a Windows instance of any instance type from Microsoft Azure.

<table id="simpletable_jff_kzy_nxb"><thead><tr><th>

Order

</th><th>

Features

</th></tr></thead><tbody><tr><td>

Order a function app to build and deploy small, event-driven functions without managing infrastructure. It will offer a server-less computing experience with pay-as-you-go pricing.

</td><td>

This catalog item deploys a NodeJS basic "hello world" application onto the provisioned Microsoft Azure function app. The versions of NodeJS supported are the same as the ones supported in Azure portal.

 The Windows admin password is obtained from Microsoft Azure at the time of creation.

</td></tr></tbody>
</table>**Note:**

Tag \(key-value\) is assigned to all deployed resources. Example: VM, network, storage as present in the stack. The key-value is updated only in the cmdb\_key\_value table, not in public cloud at this time. This will be fixed in future releases to update the tags in cloud.
