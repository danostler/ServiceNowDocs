# Application fingerprint suggestion form

When viewing suggestions based on application fingerprints, verify the information on the Application fingerprints suggestion form.

<table id="table_tty_5wd_b1c"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Discovery Pattern name

</td><td>

The name of the pattern.

</td></tr><tr><td>

Suggested CMDB CI Class

</td><td>

The name of the CMDB CI class. This field is pre-populated, but as the administrator you can modify it if necessary.

</td></tr><tr><td>

Suggested Classifier Rule

</td><td>

The name of the process excluded from discovery using the Ignore action.

</td></tr><tr><td>

Accuracy Percent

</td><td>

A percentage, based on a validation algorithm, that checks the quality of the prediction. Pick suggestions with higher accuracy.

 You can’t modify this field.

</td></tr><tr><td>

Suggested Group Name

</td><td>

The name ITOM Visibility generates for the suggestion. This field is pre-populated, but you can modify it if necessary.

</td></tr><tr><td>

Process Count

</td><td>

The number of discovered processes that this suggestion is based on.

</td></tr><tr><td>

Extended Group Name

</td><td>

The top 10 key words that provide more information about the process. Predictive Intelligence extracts these key words from the process.

 This field is pre-populated and you can’t modify it.

</td></tr></tbody>
