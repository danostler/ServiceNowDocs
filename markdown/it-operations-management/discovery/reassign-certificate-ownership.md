# Reassign Certificate Ownership

View a list of all certificates and their respective owners, and reassign ownership for certificates.

## Before you begin

Role required: pki\_admin or admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Certificate Management**.

2.  Select **Certificate records**.

3.  Select the certificate you want to reassign.

4.  Select the Search for Record \[Omitted image "icon-search-sow.png"\] Alt text: Search icon. icon in the **Assigned to** field.

5.  Select a new user to assign the certificate.

6.  Select **Save**.


## Result

The certificate is reassigned to a new owner. The new owner receives email notifications to attest or reject ownership of the certificate.
