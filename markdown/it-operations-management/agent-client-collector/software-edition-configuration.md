# Software edition configuration fields

Description of the software edition configuration fields in use when identifying the edition of Windows software.

<table id="table_dgm_5pv_g3c"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Registry operation

</td><td>

-   **Read value**: Reads the values stored in the specified registry key.
-   **Read subkey**: Reads the key stored in the registry path.

</td></tr><tr><td>

Registry path

</td><td>

Windows registry path indicating where the software is registered.

</td></tr><tr><td>

Product

</td><td>

Software product for which the edition needs to be determined.

</td></tr><tr><td>

Active

</td><td>

Option to activate edition detection for the indicated product.

</td></tr><tr><td>

Priority order

</td><td>

Indicates the hierarchy of values that take precedence when multiple editions are discovered.
