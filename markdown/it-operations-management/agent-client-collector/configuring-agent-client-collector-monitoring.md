# Configuring Agent Client Collector Monitoring

Plan and configure your Agent Client Collector Monitoring implementation.

To configure Agent Client Collector Monitoring, follow these steps:

1.  [Configure a MID Server to work with Agent Client Collector Monitoring](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-operations-management/agent-client-collector/configure-mid-acc-monitoring.md).
2.  [Choose and configure metrics to monitor](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-operations-management/agent-client-collector/configure-metric-monitors.md).
3.  [Configure the action level for anomaly detection](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-operations-management/agent-client-collector/configure-level-anomaly-detection.md).
4.  [Create a configuration settings rule](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-operations-management/agent-client-collector/create-config-overriding-rule.md).
5.  [Enable metrics collection and evaluation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-operations-management/agent-client-collector/acc-distributed-mid-cluster.md).

Depending on the checks and metrics you want to monitor, perform the following actions to enable full check activation and metrics collection:

1.  [Enable viewing HAProxy metrics](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-operations-management/agent-client-collector/configure-haproxy-metrics-stats-page.md).
2.  [Enable running of Varnish checks](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-operations-management/agent-client-collector/discover-varnish-ci.md).
3.  [Select the Azure policies to activate for metric collection](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-operations-management/agent-client-collector/select-policies-azure.md).
4.  [Create Azure policies for metric collection](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-operations-management/agent-client-collector/create-policies-azure.md).
