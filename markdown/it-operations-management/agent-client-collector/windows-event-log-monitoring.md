# Windows event log monitoring

Windows event log monitoring tracks, analyzes, and manages event logs generated by various Windows OS components.

The Windows OS Events policy includes the os.windows.check-event-log check, which measures the Windows event log against thresholds specified in the check's parameters. For details on the configurable check parameters, see [Windows event log filter parameters](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-operations-management/agent-client-collector/windows-event-log-parameters-tab.md).
