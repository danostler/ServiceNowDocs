# Nagios connector instance form

The Nagios connector instance form displays the fields you must fill in when creating a Nagios connector instance.

<table id="table_byp_mhz_zbc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Unique name for the Nagios metric connector instance.

</td></tr><tr><td>

Description

</td><td>

Brief explanation of the purpose or use of the connector instance.

</td></tr><tr><td>

Connector Definition

</td><td>

Connector definition to be used by the connector.Select **Nagios Metrics**.

</td></tr><tr><td>

Metrics database host

</td><td>

Host IP address of the host where the Nagios server collects metrics.

</td></tr><tr><td>

Metrics database credentials

</td><td>

Credentials required to access the Nagios database.

</td></tr><tr><td>

Metric collection

</td><td>

Option is pre-selected and read-only.

</td></tr><tr><td>

Metric collection schedule \(seconds\)

</td><td>

Time interval \(in seconds\) at which the connector retrieves metrics from the Nagios server.Default: 60 seconds.

</td></tr><tr><td>

Active

</td><td>

Option to activate the connector instance.

</td></tr></tbody>
