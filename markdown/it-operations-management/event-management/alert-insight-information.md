# Alert insight information

Alert insight aids faster alert triage, enabling a quicker way to find a solution and expose the probable root cause of the selected alert.

## Use alert insight to find a solution and help find the root cause of the alert

Results from data analysis provide further information about the alert, beyond that of the alert form. This aids faster alert triage, enabling a quicker way to find a solution and expose the probable root cause of the alert.

Information shown is from repeated alerts, similar alerts, incidents, problems, and change requests on the same CI, similar CIs, and other CIs that are related by CMDB relations or grouping.
