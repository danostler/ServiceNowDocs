# Machine learning solutions for Event Management

Build solutions for Event Management with Predictive Intelligence.

## Solution definition

<table id="table_vpq_m32_35b"><thead><tr><th>

Solution Definition

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Closed alert similarity

</td><td>

Closed alert similarity is a similarity solution definition that is used to recommend similar Alert records based on these fields: Description, Node, Resource, CI Type,Source, and Metric Name from the Alert \[em\_alert\] table. Results are visible to users on the Alert form and on the Similar Alerts tab in Workspace.

</td></tr><tr><td>

EM Alert Clustering Solution

</td><td>

EM Alert Clustering Solution is a clustering solution definition used in Event Management. Allows building clustering models enabling identification of common text patterns in alerts to create alert groups. Text clusters are based on these fields: Description, Metric Name, Configuration Item Class from the Alert \[em\_alert\] table.
