# Create a cloud native interface account configuration

Configure cloud accounts using the native interface of supported cloud providers like AWS AWS or Azure to enable seamless provisioning and integration.

## Before you begin

Role required: sn\_itom\_cam.cw\_admin

## Procedure

1.  Navigate to **All** &gt; **Cloud Workspace** &gt; **Configuration**.

2.  Select **New**.

3.  Enter the details in the **New Configuration** form.

    To fill the form fields, see [Cloud native interface configuration account fields](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-operations-management/cloud-account-management/cloud-native-interface-details.md).

4.  Select **Submit**.
