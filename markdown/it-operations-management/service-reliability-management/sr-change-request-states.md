# SRM change request states

Incidents can be in a few different states depending on how they are acted on.

Change requests can be manually created in SRM. Edit actions from the change request form using the state field in the form or using buttons in the change request record. Use the **Save** button in the form to keep any changes you've made. Changes to change requests in the list view are automatically saved.
