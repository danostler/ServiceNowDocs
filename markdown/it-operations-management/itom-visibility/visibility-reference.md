# ITOM Visibility reference

Reference topics provide additional information about mapping and fine-tuning application services using ITOM Visibility lists and forms.

-   **[Data collected by ITOM Visibility](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-operations-management/itom-visibility/data-collected-by-itom-visibility.md)**  
ITOM Visibility collects unique data for each type of device and stores it in dedicated tables, fields, and relationships.
-   **[ITOM Visibility and CSDM reference](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-operations-management/itom-visibility/itom-visibility-use-case-product-view.md)**  
ITOM Visibility consists of two ServiceNow products: Discovery and Service Mapping. These products are responsible for creating Configuration Items \(CIs\) in the CMDB and relating them. The goal of this product view is to help you to understand how ITOM Visibility works with the core CSDM framework.
-   **[Run options for discovery schedules](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-operations-management/itom-visibility/discovery-schedule-run-options.md)**  
Both horizontal and top-down discovery use these run options.
-   **[Plugins or applications installed with ITOM Visibility](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-operations-management/itom-visibility/plugin-app-itom-visibility.md)**  
Tables that list the plugins or applications that are installed with ITOM Visibility applications. When you update your application, any newly required application dependencies are installed.
-   **[Agent Client Collector \(ACC\) admin workspace agent actions](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-operations-management/itom-visibility/acc-admin-workspace-agent-actions.md)**  
When working in the Agent Client Collector \(ACC\) admin workspace, you can select actions to perform on the agent by selecting the **Agent actions** button.
-   **[Agent Client Collector \(ACC\) admin workspace information](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-operations-management/itom-visibility/acc-admin-workspace-information.md)**  
When selecting an agent in the ACC admin workspace, select from the displayed options to view information relating to the agent.
