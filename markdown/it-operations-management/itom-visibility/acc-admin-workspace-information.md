# Agent Client Collector \(ACC\) admin workspace information

When selecting an agent in the ACC admin workspace, select from the displayed options to view information relating to the agent.

<table id="table_d3r_3gg_gjc"><thead><tr><th>

Option

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Agent details

</td><td>

Details of the selected agent.

</td></tr><tr><td>

Diagnostic tests

</td><td>

Diagnostic tests that have run on the agent.Select **Run full diagnostics** to run a diagnostics test on demand.

</td></tr><tr><td>

Configured checks

</td><td>

The checks configured to run on the agent.

</td></tr><tr><td>

Disabled checks

</td><td>

Checks associated with the agent that are disabled.

</td></tr><tr><td>

Error messages

</td><td>

Error messages associated with the agent.Select the name of the error to open the error's details page. The page contains the following tabs:

-   Details: Details about the error.
-   Error Message Suggestions: Suggestions on how to fix the selected error.
-   Agents for error: The agents on which the selected error appears.

</td></tr></tbody>
