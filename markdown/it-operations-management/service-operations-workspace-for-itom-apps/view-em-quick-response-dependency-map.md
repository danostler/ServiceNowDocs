# View Dependency View map from quick response of an alert

View the **Dependency View** map for an alert from the **Quick Response** menu in Event Management.

## Before you begin

Role required: evt\_mgmt\_operator, evt\_mgmt\_admin

## Procedure

1.  Navigate to **All** &gt; **Event Management** &gt; **All Alerts**.

2.  In the **Number** column, select the alert for which you want to view the dependency view.

    The Alert form opens.

    \[Omitted image "event-alert-record.png"\] Alt text: The Alert form from All Alerts list in Event Management.

3.  Select **Quick Response**.

    The **Quick Response** menu opens.

    \[Omitted image "event-alert-response-dependency.png"\] Alt text: Event management alert record quick response menu.

4.  Under **Launch Application**, select **Dependency View**.

    The map opens in Service Operations Workspace.
