# Place an alert in the Maintenance state

Placing an alert into the Maintenance state indicates that there are issues with the alert's CIs that are being addressed.

## Before you begin

Role required: evt\_mgmt\_admin or evt\_mgmt\_operator

## Procedure

1.  Navigate to **Workspaces** &gt; **Service Operations Workspace**.

2.  From the navigation bar, select the Express list icon: \[Omitted image "express-list1.png"\] Alt text: Express list icon.

3.  Filter with **Active alerts**.

4.  Select an alert.

    \[Omitted image "maintenance-check-box.png"\] Alt text: Maintenance check box for an alert

5.  On the **Details** subtab, select the **Maintenance** check box to place the alert into the Maintenance state.

6.  Click **Save**.
