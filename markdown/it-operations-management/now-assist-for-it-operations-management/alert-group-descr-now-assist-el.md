# Save the alert analysis summary by Now Assist as alert group description in Express List

Save the human-readable summary generated by Now Assist in the Alert analysis as the description of an alert group.

## Before you begin

Role required: evt\_mgmt\_operator, evt\_mgmt\_admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Service Operations Workspace**.

2.  Select the Express List icon \(\[Omitted image "express-list1.png"\] Alt text: Express List icon.\) in the left navigation bar.

3.  In the Active alerts list, select the information icon \(\[Omitted image "info.png"\] Alt text: Information icon.\) to the left of an alert.

4.  On the preview panel Info tab, select **Analyze** in Alert analysis by Now Assist.

5.  Review the human-readable, AI-generated Alert analysis summary.

6.  Select **Save description** to save the summary content as the description of the alert group.

    The AI-generated summary content replaces the original description of the alert group.
