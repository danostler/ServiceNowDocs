# View an error analysis by Now Assist in Agent Client Collector

View an error analysis created by Now Assist using generative AI. Error analyses enable asking questions on a specific agent's error or error code.

## Before you begin

Role required: agent\_client\_collector\_admin

## Procedure

1.  Navigate to **All** &gt; **AI Agent Studio** &gt; **Create and manage**.

2.  Select the **Agent Client Collector \(ACC\) Diagnostic** entry.

3.  Locate the **Add AI agents that can perform these steps widget** and select the **Agent Client Collector \(ACC\) Diagnostic** entry.

4.  Enable the **Now Assist** option by sliding the **Display** toggle switch to the right.

    \[Omitted image "ai-agent-toggle-bar.png"\] Alt text: Display toggle bar

    Now Assist retrieves information on errors within agents.
