# Use LEAP playbooks in Service Operations Workspace SOW

Use LEAP playbooks in the SOW.

## Before you begin

Role required: LEAP admin

## Procedure

1.  Navigate to **All** &gt; **LEAP**.

2.  Select the required automation opportunity.

3.  [Generate LEAP playbooks](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-operations-management/aiops-leap-learning-enhanced-automation-playbooks/generate-playbooks.md).

4.  Open the SOW interface using the Workspaces menu.

5.  Select the required incident, and access the insights' panel.

6.  Select the playbook icon \[Omitted image "playbook-in-sow.png"\] Alt text: playbook icon in SOW insight panel.

7.  If a LEAP playbook is available for the selected incident, review the AI-filtered resolution steps and use them to resolve the issue.
