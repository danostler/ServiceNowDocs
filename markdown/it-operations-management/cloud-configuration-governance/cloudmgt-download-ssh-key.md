# Download an SSH key

If a resource on a stack requires login, then you must download the associated SSH key before you can access the resource.

## Before you begin

Role required: sn\_cmp.cloud\_service\_user

## Procedure

1.  On the **Cloud User Portal**, click **Activities**.

2.  In the **Activities** list, select **Manage** &gt; **Keys**.

3.  Click a key in the **SSH Key Pair** list and then click **Download Private Key** in the **SSH private key** section.
