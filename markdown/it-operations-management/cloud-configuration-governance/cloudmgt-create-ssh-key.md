# Create an SSH key

A stack might include a virtual machine that requires an SSH key for access. To request such a stack, you can either use a key pair that is assigned to you or you can generate another key.

## Before you begin

Role required: sn\_cmp.cloud\_service\_user

## Procedure

1.  On the Cloud User Portal, click **Activities**.

2.  In the **Activities** list, select **Manage** &gt; **Keys**.

3.  On the **SSH Key Pair** page, click **Generate Key Pair**, specify a **Name** for the key, and then click **OK**.


## Result

The system generates the key pair with the **Name** that you specified.
