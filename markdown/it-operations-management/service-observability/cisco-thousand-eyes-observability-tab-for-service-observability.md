# Cisco ThousandEyes Observability tab for Service Observability

Dashboard and charts on the Cisco ThousandEyes Observability tab of the Service Details page in the SOW.

## Synthetic Tests dashboard

This dashboard displays synthetic monitoring metrics for the selected service.

**Note:** If the dashboards have been customized, they might not reflect the information shown on this page.

|Chart|Description|Data source|
|-----|-----------|-----------|
|Latency|Network latency measurements over time.|Cisco ThousandEyes|
|Jitter|Variation in latency measurements over time.|Cisco ThousandEyes|
|Packet loss %|Percentage of network packets lost during transmission.|Cisco ThousandEyes|

|Chart|Description|Data source|
|-----|-----------|-----------|
|Throughput|HTTP throughput measurements showing data transfer rates.|Cisco ThousandEyes|
|Response Time|HTTP response time measurements for the monitored service.|Cisco ThousandEyes|

|Chart|Description|Data source|
|-----|-----------|-----------|
|Time to complete|Time required to fully load the monitored page.|Cisco ThousandEyes|
|Error Count|Number of errors encountered during page load tests.|Cisco ThousandEyes|
