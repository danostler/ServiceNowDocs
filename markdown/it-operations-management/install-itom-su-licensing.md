# Update ServiceNow ITOM/OT SU Licensing

Update the ServiceNow® ITOM/OT SU Licensing \[com.snc.itom.license\] application to ensure you use the latest licensing functionality. Updating the application installs related ServiceNow® Store applications and plugins if they are not already installed.

## Before you begin

-   Ensure you have an ITOM product installed on your ServiceNow instance for the ITOM/OT SU Licensing application to work. Installing the ITOM/OT SU Licensing application is not required, as the system automatically installs the application alongside any ITOM product.
-   Ensure that the application and all of its associated ServiceNow Store applications have valid ServiceNow entitlements. For more information, see [Get entitlement for a ServiceNow product or application](https://store.servicenow.com/$appstore.do#!/store/help?article=KB0030186).
-   Review the [ITOM/OT SU Licensing](https://store.servicenow.com/sn_appstore_store.do#!/store/application/4cce117e53b9301046dfddeeff7b12eb/1.1.0?referer=%2Fstore%2Fsearch%3Flistingtype%3Dallintegrations%25253Bancillary_app%25253Bcertified_apps%25253Bcontent%25253Bindustry_solution%25253Boem%25253Butility%25253Btemplate%26q%3DITOM%2520Licensing&sl=sh) application listing in the ServiceNow Store for information on dependencies, licensing or subscription requirements, and release compatibility.

Role required: admin

## About this task

You receive notifications when updates for ITOM/OT SU Licensing application are available on ServiceNow Store.

The following items are installed with ITOM/OT SU Licensing:

-   Scheduled jobs
-   Tables

For more information, see [Components installed with ITOM/OT SU Licensing](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-operations-management/components-installed-with-itom-su-licensing.md).

## Procedure

1.  Navigate to **All** &gt; **System Applications** &gt; **All Available Applications** &gt; **All**.

2.  Find the ServiceNow ITOM/OT SU Licensing application \(sn\_itom\_licensing\) using the filter criteria and search bar.

    You can search for the application by its name or ID.

3.  In the **Updates** tab, select the ServiceNow ITOM/OT SU Licensing application.

4.  Select **Proceed to update**.

    In the Install dialog that is displayed, any dependencies that are installed when the application is updated are listed.

5.  If you're prompted, follow the links to the ServiceNow Store to get any additional entitlements for dependencies.

6.  Select the version you want to upgrade to and select **Install**.
