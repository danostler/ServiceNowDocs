# General

The RPA Desktop Design Studio General components automate diverse use cases. Examples range from execute a database query to passing a variable.

-   **[Use the Confirm component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-general-messagebox-confirm.md)**  
Set a confirmation message that is shown in a dialog box requiring user inputs by using the Confirm component in the RPA Desktop Design Studio.
-   **[Use the Input component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-general-messagebox-input.md)**  
Set up a user input dialog with by using the Input component in RPA Desktop Design Studio.
-   **[Use the Input Password component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-general-messagebox-input-password.md)**  
Set up a password input form by using the Input Password component in RPA Desktop Design Studio.
-   **[Use the Show component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-general-messagebox-show.md)**  
Display custom messages as part of a Attended Robot automation by using the Show component in the RPA Desktop Design Studio
-   **[Use the Variable component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-general-variable.md)**  
Create a variable in RPA Desktop Design Studio.
-   **[Use the Try Catch component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-general-try-catch.md)**  
Catch an error or exception that occurs in a component in an automationand transfer control to the other components within the automation.
-   **[Use the Expression component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-general-expression.md)**  
Perform mathematical and logical operations with various operands and operators using the Expression component in RPA Desktop Design Studio. The Expression component supports multiple operators and functions for you to use.
-   **[Use the Find Window component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-general-findwindow.md)**  
Set the focus back on various windows that are inactive or running in the background as part of a Robotic Process Automation workflow in the RPA Desktop Design Studio.
-   **[Use the Parallel component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-general-parallel.md)**  
Execute multiple components in a sequence to perform multiple actions. The Parallel component exposes multiple threads that can execute multiple components as part of a Robotic Process Automation in the RPA Desktop Design Studio.
-   **[Use the Script component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-general-script.md)**  
Write and execute custom Visual Basic \(VB\) scripts by using the Script component in RPA Desktop Design Studio.
-   **[Use the SQL component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-general-sql.md)**  
Write custom SQL queries to perform various operations by using the SQL component in RPA Desktop Design Studio.
-   **[Use the Start App component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-general-start-app.md)**  
Start or launch an application or file by using the START APP component in RPA Desktop Design Studio.
-   **[Use the Format Text component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-general-format-text.md)**  
Format the text by using variable placeholders and line and tab variables with the Format Text component in RPA Desktop Design Studio. You can update the text and their positions easily by just updating the variable placeholders and the line and tab variables.
-   **[Use the Terminate component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-general-terminate.md)**  
Abort an automation by using the Terminate component in RPA Desktop Design Studio.
-   **[Use the Throw Error component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-general-throw-error.md)**  
Raise an exception during the automation process to indicate something incorrect has occurred by using the Throw Error component in RPA Desktop Design Studio.
-   **[Use the Wait For Time component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-wait-waitfortime.md)**  
Set a wait time by using the Wait For Time component in RPA Desktop Design Studio.
-   **[Supported functions for the Expression components](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/supported-expression-functions.md)**  
Learn about the various functions for the Expression components in RPA Desktop Design Studio.
-   **[Properties of the General components](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/Properties-general-components.md)**  
Learn about the properties of the General components in RPA Desktop Design Studio.
