# Add the ServiceNow RPA Chrome extension

Add the ServiceNow RPA Chrome extension to your Chrome browser to establish an interaction with the applications that are opened in this browser.

## Before you begin

Role required: none

## Procedure

1.  Navigate to [https://chrome.google.com/webstore/detail/servicenow-rpa-chrome-ext/bnaofpgjajbimmicdiipemhmheafhgkb](https://chrome.google.com/webstore/detail/servicenow-rpa-chrome-ext/bnaofpgjajbimmicdiipemhmheafhgkb).

2.  Select **Add to Chrome**.

3.  In the Add ServiceNow RPA Chrome Extension dialog box, select **Add extension**.

4.  To verify if the extension is added, do the following actions:

    1.  Select **Extensions**.

    2.  In the Extensions dialog box, select **Manage Extensions**.

    3.  Verify if the ServiceNow RPA Chrome Extension is enabled.

    4.  To enable it, slide the toggle switch.


## Result

You can see the ServiceNow RPA Chrome Extension on the Extensions page.

## What to do next

Restart the Chrome browser and the RPA Desktop Design Studio application.
