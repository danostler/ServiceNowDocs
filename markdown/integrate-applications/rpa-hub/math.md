# Math

The Math components in the RPA Desktop Design Studio perform simple mathematical operations. For example, find the highest integer or round off a decimal.

-   **[Use the Max component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-utilities-math-max.md)**  
Take two integers and find which one is larger as part of an automation Workflow by using the Max component in RPA Desktop Design Studio.
-   **[Use the Min component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-utilities-min-max.md)**  
Take two integers and find which one is smaller as part of an automation Workflow by using the Min component in RPA Desktop Design Studio.
-   **[Use the Round component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-utilities-math-round.md)**  
Round a decimal to the closest integer as part of an automation Workflow by using the Round component in RPA Desktop Design Studio.
-   **[Properties of the Math components](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/properties-math-component.md)**  
Learn about the properties of the Math components in RPA Desktop Design Studio.
