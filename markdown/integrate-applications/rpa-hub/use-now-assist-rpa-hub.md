# Using Now Assist for RPA Hub

If you're a RPA Desktop Design Studio user, you can create automations, activities, and automation logic additions from text instructions and preview options by using the Now Assist for RPA Hub application.

Perform the following tasks by using the RPA Desktop Design Studio application.

1.  [Create an automation with Now Assist](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/create-automation-now-assist.md).

    Create automations from text instructions and preview options by using generative AI.

2.  [Create an activity with Now Assist](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/create-activity-now-assist.md).

    Create activities from text instructions and preview options by using generative AI. Activities provide modularity to an automation project in RPA Desktop Design Studio.

3.  [Build an automation with Now Assist](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/build-automation-now-assist.md).

    Enhance the automation logic, starting from the components or from an empty design surface, that is based on text instructions by using generative AI.
