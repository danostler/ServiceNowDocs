# JSON

The JSON components in the RPA Desktop Design Studio perform different operations on JSON objects. For example, serialize a JSON object or get JSON object properties.

-   **[Use the DeserializeObject component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-utilities-json-deserializeobject.md)**  
Convert a string containing JSON data to a JSON object by using the DeserializeObject component in RPA Desktop Design Studio. You can also extract specific properties from the JSON object.
-   **[Use the GetProperties component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-utilities-json-getproperties.md)**  
Get the values of multiple properties from a JSON object by using the GetProperties component in RPA Desktop Design Studio.
-   **[Use the GetProperty component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-utilities-json-getproperty.md)**  
Get the value of a property from a JSON object by using the GetProperty component in the RPA Desktop Design Studio.
-   **[Use the JsonArrayToArray component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-utilities-json-jsonArrayToArray.md)**  
Convert a JSON object to an array of objects using the JsonArrayToArray component in the RPA Desktop Design Studio.
-   **[Use the SerializeObject component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-utilities-json-serializeobject.md)**  
Convert a JSON object to a string by using the SerializeObject component in RPA Desktop Design Studio.
-   **[Use the SetProperties component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-utilities-json-setproperties.md)**  
Set multiple properties within a JSON object by using the SetProperties component in the RPA Desktop Design Studio. You can also override a property within the JSON object.
-   **[Use the SetProperty component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-utilities-json-setproperty.md)**  
Set a property within a JSON object by using the SetProperty component in the RPA Desktop Design Studio. You can also override a property within the JSON object.
-   **[Properties of the JSON utilities](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/properties-utilities-json.md)**  
Learn about the properties of the JSON utilities in RPA Desktop Design Studio.
