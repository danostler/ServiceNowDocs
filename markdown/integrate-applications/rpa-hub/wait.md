# Wait

The Wait components in the RPA Desktop Design Studio enable you to detain or change the automation execution using the criteria provided in the component.

-   **[Use the Wait For Images component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-wait-waitforimgs.md)**  
Change the automation execution flow that is based on the image information on the screen by using the Wait For Images component in RPA Desktop Design Studio. You can also identify the state of a business application or an item that is based on an error or status icon.
-   **[Properties of the Wait components](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/wait-properties.md)**  
Learn about the properties of the Wait components in RPA Desktop Design Studio.
