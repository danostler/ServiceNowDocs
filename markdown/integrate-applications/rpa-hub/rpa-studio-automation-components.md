# Automation components

Design automations using various components in RPA Desktop Design Studio.

Make sure you're familiar with the basics of using a component, connector, activity, variable, and the various actions using automations. For more information, see the topics listed in [Building automations](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/rpa-studio-build.md) and [Using automations](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/rpa-studio-use.md).

-   **[List of components compatible with latest version](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/list-components-upgrade.md)**  
Learn about the list of components that are executed in their latest versions in the RPA Desktop Design Studio and robots, irrespective of the version they were used during the automation design.
-   **[Actions \(UI\)](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/actions-ui.md)**  
The Actions \(UI\) group of components work on and with browsers, browser elements, applications, and the desktop in the RPA Desktop Design Studio.
-   **[Conditions](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/conditions-components.md)**  
The Conditions is a group of components that you can use to automate decision making. These decisions are based on the value, input, or data matches with conditions or multiple values.
-   **[Connectors](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/connectors.md)**  
The Connectors provide specialized abilities, such as automating tasks with Microsoft Excel spreadsheets and Microsoft Word documents. Connectors are different from components because they expose multiple methods and they are used differently.
-   **[Data Modellers](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/data-modellers.md)**  
The Data Modeller creates templates by adding fields or data to PDF and XML documents. It also provides methods to classify the templates and use the fields or data created.
-   **[General](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/general-component.md)**  
The RPA Desktop Design Studio General components automate diverse use cases. Examples range from execute a database query to passing a variable.
-   **[ITSM connector](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/itsm.md)**  
The ITSM connector comprises two connectors: the Windows Management Instrumentation \(WMI\) and PowerShell. The WMI connector enables you to interact with the Windows Operating System to get important information and the PowerShell connector enables you to perform various actions on IT services.
-   **[Loops](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/loops.md)**  
The Loops provide a set of components that implement the concepts of loops, for each, and while as part of an automation Workflow in the RPA Desktop Design Studio.
-   **[RPA Hub](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/rpa-hub.md)**  
The components under RPA Hub enable interactions with the RPA Hub and ServiceNow platforms. The robots designed in RPA Desktop Design Studio can interact with various ServiceNow platform-related features. You can find components to do the same under this category.
-   **[Security](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/security.md)**  
The Security components in the RPA Desktop Design Studio perform data security operations such as Microsoft Data Protection API and data encryption.
-   **[ServiceNow](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/servicenow.md)**  
The ServiceNow components enable you to interact with ServiceNow platform features such as the Flow Designer, manage attachments from table records, and trigger embedded task automations.
-   **[Utilities](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/utilities.md)**  
The Utilities components in the RPA Desktop Design Studio enable diverse and wide-range useful operations. The utilities range from changing screen resolution to extracting zipped files.
-   **[Wait](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/wait.md)**  
The Wait components in the RPA Desktop Design Studio enable you to detain or change the automation execution using the criteria provided in the component.
