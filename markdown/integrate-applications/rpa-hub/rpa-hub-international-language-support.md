# Internationalization support for RPA Hub

You can customize the RPA Hub application for your international users by using a language plugin with the ServiceNow AI Platform. Internationalization is enabled for only for the RPA Hub instance on ServiceNow AI Platform, but not for thick client applications such as RPA Desktop Design Studio, Unattended Robot, and Attended Robot.

Internationalization support is automatically enabled and isn't configurable.

To view the full list of languages that are offered as ServiceNow AI Platform plugins and are supported in RPA Hub, see [Activate a language](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/platform-administration/system-localization/t_ActivateALanguage.md).
