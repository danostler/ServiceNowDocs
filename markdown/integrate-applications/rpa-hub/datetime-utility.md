# DateTime

The DateTime components in the RPA Desktop Design Studio enables different useful operations with date and time.

-   **[Use the Add component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-datetime-add.md)**  
Increase or decrease the time or date by the value you specify as part of a Workflow with the Add component in RPA Desktop Design Studio.
-   **[Use the Compare component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-datetime-format.md)**  
Compare two dates and time to find out whether the dates are the same by using the Compare component in RPA Desktop Design Studio.
-   **[Use the Format component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-datetime-compare.md)**  
Change the format of a date by using the Format component in RPA Desktop Design Studio.
-   **[Use the GetPart component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-datetime-getpart.md)**  
Get the value of a part of the date format by using the GetPart component in RPA Desktop Design Studio. For example, you might want to get the value of the month in the MM/DD/YYYY format.
-   **[Use the IsLeapYear component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-datetime-isleapyear.md)**  
Specify a year to find out whether it's a leap year by using the IsLeapYear component in RPA Desktop Design Studio.
-   **[Use the Now component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-datetime-now.md)**  
Get the current date and time of the system as part of an automation Workflow by using the component in RPA Desktop Design Studio.
-   **[Use the Today component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-datetime-today.md)**  
Get the current date of the system by using the Today component in RPA Desktop Design Studio.
-   **[Use the UtcNow component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-datetime-utcnow.md)**  
Get the current date and time of the system in the Coordinated Universal Time \(UTC\) format by using the UtcNow component in RPA Desktop Design Studio.
-   **[Properties of the DateTime components](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/util-datetime-prop.md)**  
Learn about the properties of the DateTime components in RPA Desktop Design Studio.
