# Directory

The Directory components in the RPA Desktop Design Studio perform standard operations on Windows directories. For example, rename a directory or get the list of files in a directory.

-   **[Use the CopyDirectory component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-directory-copydirectory.md)**  
Copy a directory to another location by using the CopyDirectory component in RPA Desktop Design Studio.
-   **[Use the CreateDirectory component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-directory-createdirectory.md)**  
Create a directory by using the CreateDirectory component in RPA Desktop Design Studio.
-   **[Use the CurrentDirectory component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-directory-currentdirectory.md)**  
Get the current working directory path and name by using the CurrentDirectory component in RPA Desktop Design Studio.
-   **[Use the DeleteDirectory component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-directory-deletedirectory.md)**  
Delete a directory by using the DeleteDirectory component in RPA Desktop Design Studio.
-   **[Use the DirectoryExists component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-directory-directoryexists.md)**  
Find out if a directory exists by using the DirectoryExists component in RPA Desktop Design Studio.
-   **[Use the GetFiles component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-directory-getfiles.md)**  
Get the names of the files in the directory and subdirectories by using the GetFiles component in RPA Desktop Design Studio.
-   **[Use the GetLastModifiedFolder component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-directory-getlastmodifiedfolder.md)**  
Get the name of the last modified folder by using the GetLastModifiedFolder component in RPA Desktop Design Studio.
-   **[Use the GetSubFolders component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-directory-getsubfolders.md)**  
Get the names of all subfolders under a directory by using the GetSubFolders component in RPA Desktop Design Studio.
-   **[Use the MoveDirectory component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-directory-movedirectory.md)**  
Move a directory and its contents to a new location by using the MoveDirectory component in RPA Desktop Design Studio.
-   **[Use the RenameDirectory component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-directory-renamedirectory.md)**  
Change the name of a directory by using the RenameDirectory component in RPA Desktop Design Studio.
-   **[Properties of the Directory components](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/util-directory-prop.md)**  
Learn about the properties of the Directory components in RPA Desktop Design Studio.
