# Dialog component properties

Refer to the descriptions of the properties of the Dialog components. You can update the properties, if required.

|Property|Description|
|--------|-----------|
|Delay Before Execution|The number of seconds to pause before running.|
|Delay After Execution|The number of seconds to pause after running.|
|Enable Timeout|Option to set a timeout value.|
|Timeout|Number of seconds that the operation ends after if not completed. To apply this value, you must set the **Enable Timeout** property.|
