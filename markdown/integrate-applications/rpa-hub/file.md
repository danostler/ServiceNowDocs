# File

The File components in the RPA Desktop Design Studio perform standard operations on the Windows files and directories. For example, copying files or retrieving temporary file names.

-   **[Use the AppendBytes component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-file-appendbytes.md)**  
Append bytes to a file as part of an automation Workflow by using the AppendBytes component in RPA Desktop Design Studio.
-   **[Use the AppendText component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-file-appendtext.md)**  
Add text to an existing text of a file as part of an automation Workflow by using the AppendText component in RPA Desktop Design Studio.
-   **[Use the ChangeExtension component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-file-changeextn.md)**  
Change the extension of a file by using the ChangeExtension component in RPA Desktop Design Studio.
-   **[Use the Combine component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-file-combine.md)**  
Combine two paths to create a single path with the Combine component in RPA Desktop Design Studio. For example, C:\\Users could be the first path and \\Data\\TestFolder could be the second path and both could be combined to create the path C:\\Users\\Data\\TestFolder.
-   **[Use the CopyFile component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-file-copyfile.md)**  
Copy an existing file to a new location by using the CopyFile component in RPA Desktop Design Studio.
-   **[Use the DeleteFile component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-file-deletefile.md)**  
Delete a file from the location you specify as part of an automation Workflow by using the DeleteFile component in RPA Desktop Design Studio.
-   **[Use the FileExists component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-file-fileexists.md)**  
Find out whether the file you specify exists as part of an automation Workflow by using the FileExists component in RPA Desktop Design Studio.
-   **[Use the GetDirectoryName component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-file-getdirname.md)**  
Get the directory name of the specified path as part of an automation Workflow by using the GetDirectoryName component in RPA Desktop Design Studio.
-   **[Use the GetExtension component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-utilities-file-getextension.md)**  
Get the extension of a file and know its type as part of an automation Workflow by using the GetExtension component in RPA Desktop Design Studio.
-   **[Use the GetFileName component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-file-getfilename.md)**  
Get the name of a file as part of an automation Workflow by using the GetFileName component in RPA Desktop Design Studio.
-   **[Use the GetFileNameWithoutExtension component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-utilities-file-getfilenamewithoutextension.md)**  
Get the name of a file without its extension as part of an automation Workflow by using the GetFileNameWithoutExtension component in RPA Desktop Design Studio.
-   **[Use the GetLastModifiedFile component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-utilities-file-getlastmodifiedfile.md)**  
Get the name and path to the last modified file in a directory by using the GetLastModifiedFile component in RPA Desktop Design Studio.
-   **[Use the GetPathRoot component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-utilities-file-getpathroot.md)**  
Get the name of the root folder or directory of a path by using the GetPathRoot component in RPA Desktop Design Studio.
-   **[Use the GetRandomFileName component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-utilities-file-getrandomfilename.md)**  
Get randomly generated file names and their extensions by using the GetRandomFileName component in RPA Desktop Design Studio.
-   **[Use the GetTempFileName component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-utilities-file-gettempfilename.md)**  
Get the names of temporary Windows files by using the GetTempFileName component in RPA Desktop Design Studio.
-   **[Use the HasExtension component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-utilities-file-hasextension.md)**  
Find out whether a file has an extension by using the HasExtension component in RPA Desktop Design Studio.
-   **[Use the IsPathRooted component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-utilities-file-ispathrooted.md)**  
Find out if the path you specify has a root folder by using the IsPathRooted component in RPA Desktop Design Studio.
-   **[Use the MoveFile component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-file-movefile.md)**  
Move a file from an existing location to a new location as part of an automation Workflow by using the MoveFile component in RPA Desktop Design Studio.
-   **[Use the ReadBytes component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-file-readbytes.md)**  
Read the specified number of bytes as part of an automation Workflow by using the ReadBytes component in RPA Desktop Design Studio.
-   **[Use the ReadText component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-file-readtext.md)**  
Return all text in a file as output by using the ReadText component in the RPA Desktop Design Studio.
-   **[Use the RenameFile component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-file-renamefile.md)**  
Rename a file and change the file type by using the RenameFile component in RPA Desktop Design Studio.
-   **[Use the WriteBytes component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-file-writebytes.md)**  
Write the specified byte array to a file as part of an automation Workflow by using the WriteBytes component in RPA Desktop Design Studio.
-   **[Use the WriteText component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-file-writetext.md)**  
Append or overwrite existing text to an existing file or create a file and write text to it with the WriteText component in the RPA Desktop Design Studio.
-   **[Properties of the File components](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/util-file-prop.md)**  
Learn about the properties of the File components in RPA Desktop Design Studio.
