# Loops

The Loops provide a set of components that implement the concepts of loops, for each, and while as part of an automation Workflow in the RPA Desktop Design Studio.

-   **[Use the Break component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-loop-break.md)**  
Break a loop at any time with the Break component in the RPA Desktop Design Studio.
-   **[Use the For Each component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-loops-foreach.md)**  
Traverse each item in a collection in the order of first to last using the For Each component in the RPA Desktop Design Studio. The collection can be an array, list, or a table.
-   **[Use the For Loop component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-loops-forloop.md)**  
Execute a code recurrently to perform tasks of repetitive nature with the For Loop component in the RPA Desktop Design Studio. The For Loop is a control flow component for specifying iterations.
-   **[Use the While component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/use-loops-while.md)**  
Execute a code until a specified condition is true using the While component in RPA Desktop Design Studio.
-   **[Properties of the Loops components](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/rpa-hub/loops-properties.md)**  
Learn about the properties of the Loops components in RPA Desktop Design Studio.
