# Zero Copy Connector for ERP data fabric tables

View external records from a data fabric table in lists and forms the same way you view records in a physical table.

## Access ERP data using table list view or GlideRecord scripts

The SAP ECC and SAP S/4HANA SAP S/4HANA connectors provide read-only access to data and metadata from SAP systems. A connection admin can set up a connection to SAP ECC or SAP S/4HANA SAP S/4HANA in the Workflow Data Fabric Hub Zero Copy Connector Hub and grant data stewards access to this connection. Data stewards can then use the established connection to create a data fabric table and map data from SAP ECC or SAP S/4HANA SAP S/4HANA. Lastly, access SAP data through the table list view or by using GlideRecord scripts.
