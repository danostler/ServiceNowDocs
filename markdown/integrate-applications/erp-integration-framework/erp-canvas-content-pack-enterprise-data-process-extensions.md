# Zero Copy Connector for ERP Enterprise Data Foundation data product process extensions

The Zero Copy Connector for ERP Enterprise Data Foundation data product contains process extensions that you may need when interacting with an SAP system.

## Enterprise Data Foundation data product process extensions

|Process Extension \(Subflow\) Name|Short Description|
|----------------------------------|-----------------|
|Create a new material|Create a new material with basic data and, optionally, sales data.|
|Search for a material|Search for a material.|
|Update material details|Update the description of an existing material and, optionally, update attributes such as sales data.|
