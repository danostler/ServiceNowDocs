# Zero Copy Connector for ERP SAP Foundation data product models

The Zero Copy Connector for ERP SAP Foundation data product contains models that you may need when interacting with an SAP system.

## SAP Foundation data product models

|Model Name|Short Description|Model types|Methods|
|----------|-----------------|-----------|-------|
|Address|Retrieve address details based on address number.|ERP|RFC/BAPI|
|Billing Block - List|Retrieve a list of billing blocks with descriptions. Billing blocks stop customer billing processes, for example, if the customer has credit or legal issues being resolved.|ERP|RFC/BAPI|
|Business Area|Retrieve a list of business areas.|ERP|RFC/BAPI|
|Company Code|Retrieve a list of company codes.|ERP|RFC/BAPI|
|Controlling Area|Retrieve a list of Controlling Areas.|ERP|RFC/BAPI|
|Country|Retrieve a list of countries.|ERP|RFC/BAPI|
|Currency|Retrieve a list of currency codes.|ERP|RFC/BAPI|
|Delivery Block - List|Retrieve a list of delivery blocks with descriptions. Delivery blocks stop outbound delivery creation for a sales order, for example, if the customer has credit limit issues.|ERP|RFC/BAPI|
|Incoterms|Retrieve a list of Inco terms.|ERP|RFC/BAPI|
|Language|Retrieve a list of languages maintained in the ERP system.|ERP|RFC/BAPI|
|Payment Terms|Retrieve a list of all payment terms.|ERP|RFC/BAPI|
|Plant|Retrieve a list of plants.|ERP|RFC/BAPI|
|Plant Location|Retrieve a list of locations within a plant.|ERP|RFC/BAPI|
|Purchasing Group|Retrieve a list of purchasing groups, along with communication details.|ERP|RFC/BAPI|
|Purchasing Organization|Retrieve a list of purchasing organizations.|ERP|RFC/BAPI|
|Sales Area|Retrieve a list of sales areas. \(Sales Area is a combination of Sales Organization, Distribution Channel, and Division.\)|ERP|RFC/BAPI|
|Storage Location|Retrieve a list of storage locations.|ERP|RFC/BAPI|
|Valuation Area|Retrieve a list of valuation areas.|ERP|RFC/BAPI|
