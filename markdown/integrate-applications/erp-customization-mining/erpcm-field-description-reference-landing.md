# ERP Semantic Mining field descriptions

Some tables of field descriptions in ERP Semantic Mining are too large to maintain in task topics. Find information on those large tables in this section.

**Important:** Starting with the Zurich release, ERP Semantic Mining is being prepared for future deprecation. It will be hidden and no longer activated on new instances but will continue to be supported.

-   **[ERP Semantic Mining task list field descriptions](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/erp-customization-mining/erpcm-task-list-details.md)**  
The task list in ERP Semantic Mining displays information on connection tasks for the ERP \(Enterprise Resource Planning\) system.
-   **[ERP Semantic Mining candidate list field descriptions](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/erp-customization-mining/erpcm-candidate-list-details.md)**  
The candidate list in ERP Semantic Mining displays information on the basic details for each candidate.
-   **[ERP Semantic Mining candidate scan results field descriptions](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/erp-customization-mining/erpcm-candidate-scan-results-descriptions.md)**  
On a candidate record in ERP Semantic Mining the **ERP scan results** tab displays information about the latest scan of the ERP \(Enterprise Resource Planning\) system.
-   **[ERP Semantic Mining candidate recommendations field descriptions](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/erp-customization-mining/erpcm-candidate-recommendations-field-descriptions.md)**  
On a candidate record in ERP Semantic Mining, the **Recommendations** tab displays information about suggested next actions for replatforming ERP \(Enterprise Resource Planning\) candidates.
-   **[ERP Semantic Mining similar candidates field descriptions](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/erp-customization-mining/erpcm-similar-candidates-field-descriptions.md)**  
On a candidate record in ERP Semantic Mining the **Similar candidates** tab displays information about similar candidates for replatforming ERP \(Enterprise Resource Planning\) apps.
-   **[ERP Semantic Mining snapshot field descriptions](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/erp-customization-mining/erpcm-snapshot-field-descriptions.md)**  
In ERP Semantic Mining, the **Configuration** page **Snapshots** tab displays information about import and export snapshots.
