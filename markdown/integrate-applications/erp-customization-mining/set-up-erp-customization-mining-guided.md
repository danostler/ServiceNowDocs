# Run Guided Setup for ERP Semantic Mining

Run the Guided Setup to configure ERP Semantic Mining.

## Before you begin

**Important:** Starting with the Zurich release, ERP Semantic Mining is being prepared for future deprecation. It will be hidden and no longer activated on new instances but will continue to be supported.

You must first download and install ERP Semantic Mining from the ServiceNow Store.

Role required: sn\_erp\_integration.erp\_admin, sn\_erp\_mining.erp\_admin

## About this task

For more information on using Guided Setup, see [Guided Setup](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/adoption-services/guided-setup.md).

Alternatively, you can configure ERP Semantic Mining without Guided Setup. For more information, see [Configure a new ERP Semantic Mining connection](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/erp-customization-mining/set-up-erp-customization-mining.md).

## Procedure

1.  Navigate to **All** &gt; **ERP Foundation** &gt; **ERP Semantic Mining** &gt; **ERP Semantic Mining Guided Setup**.

2.  Navigate to **All** &gt; **Zero Copy Connector for ERP** &gt; **Zero Copy Connector for ERP Home**.

3.  Complete the Guided Setup for Zero Copy Connector for ERP by selecting **Configure Zero Copy Connector for ERP**.

    Zero Copy Connector for ERP is required by ERP Semantic Mining to connect to systems.

    Follow the steps to configure Zero Copy Connector for ERP, and select **Continue**.

4.  Connect ERP Semantic Mining to the ERP system.

    1.  Select **Select the ERP system to connect**, follow the steps, and select **Continue**.

    2.  Select **Validate the connection status**, follow the steps, and select **Continue**.
