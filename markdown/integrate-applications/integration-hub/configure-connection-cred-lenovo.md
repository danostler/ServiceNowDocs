# Create a connection and credential for Lenovo spoke

Create a connection and credential for Lenovo to authenticate ServiceNow requests in Lenovo spoke.

## Before you begin

-   Request an Integration Hub subscription.
-   Activate the Lenovo spoke.
-   Role required: admin.

## Procedure

1.  Navigate to **All** &gt; **Connection and Credentials** &gt; **Connection and Credential Aliases**.

2.  Select **Lenovo**.

3.  From the **Related Links**, select **Create New Connection and Credential**.

4.  Change the default connection name for the connection to Lenovo Warranty API in the **Connection name** field.

    By default, this field is set to **Lenovo Connection**.

    **Important:** Don't change the default URL specified in the Connection URL field.

5.  In the **Client ID** field, enter the Client ID provided by Lenovo.

6.  Select **Create**.


## Result

The Lenovo connection is successfully created and listed in the Connections tab.
