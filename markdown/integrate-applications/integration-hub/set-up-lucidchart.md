# Create OAuth 2.0 Client in Lucidchart

Integrate your ServiceNow instance with your Lucidchart account. Create a custom OAuth application in Lucidchart to authenticate ServiceNow requests.

## Before you begin

-   Lucidchart account.
-   Role required: Lucidchart admin.

## About this task

Create a custom OAuth application in your Lucid account to enable the OAuth 2.0 authentication with the Lucidchart Diagramming spoke.

## Procedure

1.  Log in to your Lucid account \([www.lucid.app](https://lucid.app)\) using the admin credentials.

2.  In the left navigation, click **Admin**.

3.  Select **App Integration** &gt; **General**.

4.  In the Custom OAuth card, click **Settings**.

    The OAuth 2.0 Clients page is displayed.

5.  Enter a name for the client and click **Create OAuth 2.0 Client**.

6.  Click **Details** and view the Client ID and Client Secret details.

    The Client ID and Client Secret details are required to register the application as a third-party OAuth provider on your ServiceNow instance.


## Result

The custom OAuth application in your Lucid account is created and can be integrated with your ServiceNow instance.
