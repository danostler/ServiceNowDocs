# Configure group assignments

Configure group assignments for all the applications you want to deploy.

## Before you begin

Role required: admin

## Procedure

1.  Log in to [Microsoft Endpoint Manager admin center](https://endpoint.microsoft.com/) as an admin.

2.  Navigate to **Apps** &gt; **All apps**.

3.  Click the application you want to deploy.

4.  Click **Properties** under **Manage**.

5.  Under **Assignments**, perform assignments for the groups you want to deploy.
