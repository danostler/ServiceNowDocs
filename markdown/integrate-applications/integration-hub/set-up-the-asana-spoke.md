# Set up the Asana spoke

Configure the Asana spoke to integrate your ServiceNow instance with the Asana instance. The integration enables you to set up a flow and, when a trigger occurs, automate actions on the Asana instance. For example, set a trigger that triggers a flow every day at 13:00 to get the meta data about a workspace.

## Before you begin

-   Request an Integration Hub subscription.
-   Activate the Asana spoke.
-   Access to the Asana developer's console.
-   Role required: admin
