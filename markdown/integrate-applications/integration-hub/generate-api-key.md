# Generate API key

Generate an API key on the SAP Fieldglass tenant to enable your ServiceNow instance to connect to the SAP Fieldglass tenant. You provide the API key on the SAP Fieldglass spoke connection record and the SAP Fieldglass tenant authenticates your ServiceNow instance based on the API key.

## Before you begin

Role required: admin

Access to SAP Fieldglass tenant account.

## Procedure

1.  Log in to your SAP Fieldglass tenant account.

2.  Select Linked Accounts.\[Omitted image "sap-fieldglass-spoke-linked-account.png"\] Alt text: Linked Accounts link.

3.  On the Self-Service Dashboard, select the API Application Keys card.\[Omitted image "sap-fieldglass-spoke-application-keys-card.png"\] Alt text: API Application Keys card.

4.  Select **New**.\[Omitted image "sap-fieldglass-spoke-new-button.png"\] Alt text: New button.

5.  On the Create Application Key page, enter values.

    -   Application Name: Name of the application.
    -   Description: Custom description of the API key.
6.  Select **Create**.\[Omitted image "sap-fieldglass-spoke-api-key-details.png"\] Alt text: Create API Application Key page.

    The API key and other details are generated.\[Omitted image "sap-fieldglass-spoke-api-key-generated.png"\] Alt text: API key generated.
