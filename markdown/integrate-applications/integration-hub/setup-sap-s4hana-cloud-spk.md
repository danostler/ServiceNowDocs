# Setup the SAP S4 HANA Public Cloud spoke

Integrate the ServiceNow instance and SAP S4 HANA public cloud instance by using basic authentication to authenticate ServiceNow requests.

## Before you begin

-   Request an Integration Hub subscription.
-   Activate the SAP S4 HANA Public Cloud spoke.
-   Role required: admin.

## Procedure

1.  Navigate to **All** &gt; **Integration Hub** &gt; **Connection &amp; Credential Aliases**.

2.  Open the record, **SAP S4 HANA Cloud**.

3.  Click the **Create New Connection &amp; Credential** related link.

4.  On the form, fill in these values.

    |Field|Description|
    |-----|-----------|
    |Connection Name|Name to identify the connection record.|
    |Connection URL|URL to connect to SAP S4 HANA public cloud instance.|
    |User Name|User name to log in to the SAP S4 HANA public cloud instance.|
    |Password|Password to log in to the SAP S4 HANA public cloud instance.|

5.  Click **Create**.
