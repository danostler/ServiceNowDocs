# Components installed with client software distribution

Several types of components are installed with client software distribution \(CSD\).

-   **[Tables installed](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/integration-hub/csd-tables.md)**  
These tables are installed with the Client Software Distribution plugin \(com.snc.orchestration.client\_sf\_distribution\).
-   **[Plugins installed](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/integration-hub/csd-installed-plugins.md)**  
These plugins are installed with the Client Software Distribution plugin, if they are not already active.
-   **[Roles installed](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/integration-hub/csd-installed-roles.md)**  
These roles are installed with the Client Software Distribution plugin.
-   **[Script includes installed](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/integration-hub/csd-script-incl-installed.md)**  
These script includes are installed with the Client Software Distribution plugin.
-   **[Properties installed](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/integration-hub/csd-prop-installed.md)**  
Properties for client software distribution \(CSD\) manage the installation status of requested software deployments and configure scheduled Discoveries.
-   **[Flows and subflows installed](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/integration-hub/csd-flows-installed.md)**  
These flows and subflows are installed with the Client Software Distribution plugin.
