# Create an Atlassian Administration API Key

Create an Atlassian Administration API Key to authenticate ServiceNow requests.

## Before you begin

Role required: Organization admin

## Procedure

1.  Log in to [Atlassian Administration](http://admin.atlassian.com) with your admin credentials.

2.  Select your organization if you have more than one.

3.  Select **Settings** &gt; **API keys**.

4.  Select **Create API key**.

5.  On the Create an API key page, provide a name to identify the API key and the expiration date in the **Expires on** field.

    **Note:** By default, the key expires in one week from the creation date. If you’d like to modify the expiration date, select a new date in the **Expires on** field. You must select a date within one year of the creation date.

6.  Select **Create** to save the API key.

7.  Copy the values for your Organization ID and API key.

    **Note:** Save these values to use them later.

8.  Select **Done**.

    The key appears in your list of API keys. You can also revoke the API by selecting **Revoke** from the Actions column.
