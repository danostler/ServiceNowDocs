# Create child aliases for additional Docusign accounts

Create a child connection alias under the base alias for each account if your Docusign implementation includes multiple Docusign accounts.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Docusign** &gt; **Connection Aliases**.

2.  Select **New**.

3.  In the **Name** field, add a descriptive name for the account, for example, `Docusign HR account`.

4.  Save the record.
