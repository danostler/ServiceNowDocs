# Importing and streaming data in Integration Hub

Import data with Integration Hub - Import or stream data from your Apache Kafka environment with Stream Connect.

## Import data with Integration Hub - Import

Integration Hub - Import enables you to configure, run, and schedule your data imports all through a single simplified interface.

Integration Hub - Import consolidates multiple data integration capabilities into a single environment, eliminating the need to create and manage several forms throughout the platform. Its step-by-step experience guides you through the configuration of data sources, target tables, and data mapping. You can schedule your data imports or run them on demand.

## Stream data with Stream Connect

Stream Connect links your Kafka environment to your ServiceNow instance, enabling you to stream data between your instance and your external systems.

Stream Connect is integrated with Workflow Studio, so you can build flows that produce and consume Kafka events. Stream Connect also provides several consumers that enable you to stream and process data with scripts or by using your existing RTE or transform map configurations.
