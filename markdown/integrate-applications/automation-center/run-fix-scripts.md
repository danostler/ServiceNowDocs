# Run fix script to view demo data on Kanban board

To view automation requests from the demo data in your Kanban board, you must run the fix script.

## Before you begin

Role required: sn\_ac.automation\_technical\_user or sn\_ac.automation\_admin

## Procedure

1.  Navigate to **All** &gt; **System Definition** &gt; **Fix Scripts**.

2.  Open the Sync VTB Board.

3.  Select **Run Fix Script**.

    You can run the fix script in the background or just proceed. After the fix script is run, you can view the demo data automation requests in the Kanban board.
