# Automation Center Workspace

Automation Center Workspace provides an overall view of the performance and value of the KPIs for existing automations, executions, and new automation requests.

ServiceNow® Automation Center Workspace is a web-based platform with three dashboards: Overview, Executions, and Value.

-   The Overview dashboard includes the basic score cards, lists, and reports. It’s the default landing page of the Automation Center Workspace.
-   The Executions dashboard enables you to manage the health of automations in one central place​.
-   The Value dashboard provides summary information about the states of business goals and automations.

-   **[Automation Center Overview dashboard](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/automation-center/overview-dash.md)**  
The Automation Center Workspace landing page is the Overview dashboard, which contains basic score cards, lists, and reports.
-   **[Automation Center Executions dashboard](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/automation-center/executions-dash.md)**  
The Automation Center Executions dashboard helps you manage the health of automations in one central place. You can also​ import automation metrics and data from third-party providers regardless of the technology used.
-   **[Automation Center Value dashboard](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/automation-center/value-dash.md)**  
The Automation Center Value dashboard provides summary information about the states of goals and automations along with the actual and estimated cost and time saved for all active and future automations.
