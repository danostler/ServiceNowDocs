# Edit a task

Edit a task if you want to update the checklist.

## Before you begin

Role required: sn\_ac.automation\_admin

After the automation request is moved to the following states, the option to edit a task isn’t available:

-   Defer
-   Publish
-   Reject

## Procedure

1.  Navigate to **Workspaces** &gt; **Automation Center Workspace**.

2.  Select the list icon \(\[Omitted image "autocenter-list-icon.png"\] Alt text: List icon.\).

3.  On the **Lists** tab, under **Build**, select **All Automation Requests**.

4.  Open the automation request for which you want to delete a checklist.

    Ensure that the automation request already has a checklist.

5.  Select the Golive Checklist icon \(\[Omitted image "icon-golive.png"\] Alt text: golive checklist icon\) from the side panel.

6.  Select **Edit** or **Delete** from the details button for a task.

    If you select **Edit**, the Details form is displayed. Make the updates, and select **Save**.

    If you select **Delete**, select **Delete** on the confirmation window.
