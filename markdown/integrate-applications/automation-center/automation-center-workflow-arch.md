# Automation Center architecture

Learn about the Automation Center architecture.

## High-level architecture

Various components make up the architecture of the Automation Center application:

-   You install the Automation Center application from the ServiceNow Store.
-   Automation Center is a scoped application in the ServiceNow instance. It’s the command and control center for managing the automation goals, requests, tasks, resources, attributes, insights, automation, robots, and executions.
-   The automations, robots, and executions are populated from various sources such as ServiceNow® RPA Hub via flow and external vendors via REST calls.
