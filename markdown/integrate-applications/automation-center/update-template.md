# Update a template

Update your template if you have changed the checklist and want those changes to reflect in your template.

## Before you begin

Role required: sn\_ac.automation\_admin

After the automation request is moved to the following states, the option to update a template isn’t available:

-   Defer
-   Publish
-   Reject

## Procedure

1.  Navigate to **Workspaces** &gt; **Automation Center Workspace**.

2.  Select the list icon \(\[Omitted image "autocenter-list-icon.png"\] Alt text: List icon.\).

3.  On the **Lists** tab, under **Build**, select **All Automation Requests**.

4.  Open the automation request for which you want to update a template.

    Ensure that the automation request has a template.

5.  Select the Golive Checklist icon \(\[Omitted image "icon-golive.png"\] Alt text: golive checklist icon\) from the side panel.

6.  Make updates to the checklist.

    You can add more tasks to the checklist or edit the existing tasks. To add more tasks, follow the same steps as creating a checklist in [Create a checklist](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/automation-center/create-checklist.md). To edit or delete a task, see [Edit a task](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/automation-center/edit-task.md).

7.  Select **Update template** from the details button to ensure that your changes are reflected in your template.

    **Note:** The **Update template** option is available only for the templates that are created by you.

    \[Omitted image "golive-update.png"\] Alt text: Update a template

    The template is updated with your changes.
