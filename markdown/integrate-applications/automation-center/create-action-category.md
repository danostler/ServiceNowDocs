# Create an action category

Create an action category to associate with an action. Action categories help users to categorize the actions into different categories, which would make selecting categories easier.

## Before you begin

Role required: sn\_ac.automation\_technical\_user and sn\_ac.automation\_admin

## Procedure

1.  Navigate to **All** &gt; **Automation Center** &gt; **Build** &gt; **Action Categories**.

2.  Select **New**.

3.  On the Action Categories form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Name|An intuitive name for the action category.|
    |Application|Scope of the action category. This is auto-populated. Do not edit this field.|
    |Description|Description of the action category.|
    |Parent|Select an action category if you want to attach the new category with another one.|

4.  Select **Submit**.


## Result

The action category is created.
