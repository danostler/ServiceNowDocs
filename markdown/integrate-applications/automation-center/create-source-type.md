# Create automation source type

Define the kind of source type that you want to import and use in Automation Center.

## Before you begin

Role required: sn\_ac.automation\_technical\_user or sn\_ac.automation\_admin

## Procedure

1.  Navigate to **All** &gt; **Administration** &gt; **Automation Center** &gt; **Automation Source Type**.

    The Automation source types page is displayed with a list of existing automation source types.

2.  Select **New**.

    The Automation source types page is displayed.

3.  Provide a name and description.

4.  Select **Submit**.

    The Automation source types page is displayed with the newly created automation source type along with the other automation source types.
