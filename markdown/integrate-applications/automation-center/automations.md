# Working with automations

Create automations to automatically streamline your business processes, reduce the time that you spend on manual activities, and reduce your costs.

You can create automation goals, automation requests, and the tasks associated with them.

-   **[Create an automation goal](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/create-automation-goal.md)**  
Create an automation goal to manage the progress and estimate the outcomes of an automation request. One goal can be associated with multiple automation requests. But an automation request can be associated with only one goal.
-   **[Reinstate an automation goal](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/reinstate-automation-goal.md)**  
Reinstate an already achieved automation goal to restore and alter the scope of the automation.
-   **[Creating an automation request](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/create-automation-request.md)**  
An automation request helps you to automatically streamline your business processes, reduce the time that you spend on manual activities, and reduce your costs.
-   **[Create an automation task](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/create-automation-task.md)**  
Create an automation task to effectively manage the activities associated with the automation request. However, this is an optional step.
-   **[Create an automation resource](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/create-automation-resource.md)**  
Create an automation resource to associate a resource to an automation task. However, this is an optional step.
-   **[Move an automation request to the planned state](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/move-automation-request-to-planned.md)**  
After creating an automation request, move it to the **Planned** state before you work on it or create associated automation tasks or resources.
-   **[Move an automation request to the In progress state](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/move-automationrequest-to-inprogress.md)**  
As you begin the development for the automation request, move it to the **In progress** state. This state notifies anyone using Automation Center that the work for the automation is in progress.
-   **[Defer an automation request](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/defer-an-automation-request.md)**  
Defer an automation request from Automation Center if the priority of the request has changed and it is not urgent anymore.
-   **[Reject an automation request](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/reject-automation-request.md)**  
Reject an automation request from Automation Center if it's a duplicate of an existing automation or if it doesn’t meet the required criteria.
-   **[Publish an automation request](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/publish-automation-request.md)**  
Publish an automation request from Automation Center to make the request ready for execution.
-   **[Create an agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/create-agent.md)**  
Create an automation request agent to efficiently manage the tasks of the automation request without manual intervention.
-   **[Working with golive checklists](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/working-golive.md)**  
Create a golive checklist to automatically track items that must be reviewed before publishing the automation request. Every golive checklist is associated with an automation request.
-   **[Map automations with automation request](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/map-automation-to-req.md)**  
Map automations in Published state with an automation request to publish the automation request.
-   **[Update an automation attribute](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/create-attribute-ci.md)**  
Update an automation attribute to include the automation source and time savings information. This information is available in the Automation Center Value dashboard for analysis.
-   **[Enable granular integration for automations](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/granular-data-integration.md)**  
Enable granular integration for automations to capture cost saved and time saved at the transaction level. This feature makes the total cost-saved and total time-saved data analysis effective.
-   **[Manage and review automation executions](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/view-automation-executions.md)**  
Manage and review your automation executions, including their states, so that you can make sure that they are on track.
-   **[Review the bot processes](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/view-base-bot-process.md)**  
Review a list of available bot processes in Automation Center, so that you can make sure that they are on track.
-   **[View the available robots](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/view-robots-auto-center.md)**  
View a list of available robots in Automation Center for reference purpose.
-   **[Create an insight rule](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/create-insight-rule.md)**  
Create an insight rule to generate custom insights for an automation business process.
-   **[View automation insights](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/view-insights.md)**  
View automation insights to view the generated insights from the insight rules in Automation Center Executions dashboard.
