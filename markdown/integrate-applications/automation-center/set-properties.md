# Set Automation Center Properties

You must set the properties to define the default settings that are required for Automation Center.

## Before you begin

Role required: admin

You must be in the same scope in which the record exists.

## Procedure

1.  Navigate to **All** &gt; **Automation Center** &gt; **Administration** &gt; **Automation Properties**.

2.  If the form isn’t editable, ensure that the Application scope you are in is same as the one where the record is.

    If the form isn’t editable, a message is displayed. Follow the message to change the Application scope.

3.  On the form, fill in the fields.

    For a description of the field values, see [Automation Center properties](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/automation-center/automation-center-properties.md).

4.  Select **Save**.
