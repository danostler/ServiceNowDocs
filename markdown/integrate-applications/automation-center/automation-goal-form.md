# Automation goal form

Use the Automation goal form to create an automation goal in Automation Center.

<table id="table_pyx_mhd_4qb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Name of the automation goal.

</td></tr><tr><td>

Start date

</td><td>

Start date of the automation goal.

</td></tr><tr><td>

Target date

</td><td>

Target date of the automation goal.

</td></tr><tr><td>

State

</td><td>

State of the automation goal:-   **Pending**: The associated requests are in "New", "Deferred", or "Rejected" state, or there are no associated requests.
-   **Work in progress**: The associated requests are in "Planned", "In progress", or "Published" state.
-   **Achieved**: The associated requests are in "Published" or "Rejected" state with at least one request in "Published" state.
-   **Not achieved**: The target date isn’t met for the associated requests, and they are in "New", "Planned", "In progress", or "Published" state.

</td></tr><tr><td>

Owner

</td><td>

Owner of the automation goal.
