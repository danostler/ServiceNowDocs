# Exploring Automation Center

Learn about the Automation Center application key features, architecture, and workspace.

## Key features

-   Discover, prioritize, and manage automation requests from different sources through the full life cycle in Automation Center workspace.
-   Oversee and optimize automation performance versus business goals across multiple vendors.
-   Monitor and manage the health of the automation and prevent automation failures across multiple vendors.

Explore the following topics to learn more about Automation Center architecture and Workspace.

-   **[Automation Center architecture](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/automation-center-workflow-arch.md)**  
Learn about the Automation Center architecture.
-   **[Automation Center Workspace](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/automation-center-workspace-ui.md)**  
Automation Center Workspace provides an overall view of the performance and value of the KPIs for existing automations, executions, and new automation requests.
-   **[Automation Center Kanban board](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/kanban.md)**  
The Automation Center Kanban board provides an easy access to view and manage automation requests and automation tasks.
-   **[Action center](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/action-center.md)**  
Action Center provides an easy way to fix repeated issues that cause automations to fail in Automation Center.
-   **[Connection manager](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/conn-manager.md)**  
Connection Manager enables you to add data from a third-party tool into Automation Center tables by correctly mapping and testing the data import.
-   **[Migration accelerator](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/migration.md)**  
Migration accelerator enables seamless, low-effort, cost-efficient migration of UiPath and Blue Prism automations to ServiceNow RPA Hub. It’s a solution that minimizes downtime, maximizes reuse, and accelerates time-to-value.
