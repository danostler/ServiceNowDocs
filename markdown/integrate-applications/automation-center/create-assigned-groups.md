# Create assigned groups

Create assigned groups to limit the availability of the action to a specific group.

## Before you begin

Role required: sn\_ac.automation\_technical\_user and sn\_ac.automation\_admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Automation Center Workspace**.

2.  Select the Action Center icon \(\[Omitted image "action-center-icon.png"\] Alt text: Action Center icon\) on the left.

3.  Select an action.

    The details are displayed

4.  Select the **Assigned Groups** tab.

5.  Select **New**.

6.  Specify the group to which you want to assign permissions to view and execute the action.

7.  Select **Save**.
