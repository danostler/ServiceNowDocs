# Define source for automation

Define the source that you want to add for your automations.

## Before you begin

Role required: sn\_ac.automation\_technical\_user or sn\_ac.automation\_admin

## Procedure

1.  Provide details on the **Define source** form.

<table id="id_jw3_jvk_4tb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Source name

</td><td>

Provide a unique and intuitive name for your automation source.**Note:** Source name can contain only underscores and alphanumeric characters.

</td></tr><tr><td>

Description

</td><td>

Provide a description that details the purpose of the automation source.

</td></tr><tr><td>

Source type

</td><td>

Provide the name of the source from where you want to import data. For example, ServiceNow® Robotic Process Automation.If you want to create your own source type, select Add source type. On the form, provide a name, select an icon, and then select **Add**.

The source type is now available in the list of source types. Select it from the list.

</td></tr></tbody>
</table>2.  Select **Save and continue**.

    The **Configure data import** section is displayed with the **Schedule import** section made available to schedule the import of automation data.
