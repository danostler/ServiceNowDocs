# Creating a review request

Review request enable you to review the automations.

You can create an action request in two ways. You can create a request by using the ServiceNow® Service Catalog or from Automation Center.

-   **[Submit a review request from Service Catalog](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/review-service-catalog.md)**  
Submit a review request using the ServiceNow® Service Catalog form to request for reviews to review automations.
-   **[Create a review request from Automation Center](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/review-request-from-form.md)**  
Create a review request from Automation Center to request for reviews to review automations.
