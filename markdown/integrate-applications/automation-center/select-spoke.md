# Select a spoke

Choose a spoke to set up a connection. This is an optional step.

## Before you begin

Role required: sn\_ac.automation\_technical\_user or sn\_ac.automation\_admin

If your spoke isn’t already installed, ensure to install it first.

Proceed from the **Schedule data import** section. For information on scheduling data import, see [Schedule data import](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/schedule-import.md).

## Procedure

1.  Select a spoke.

    You can sort the available spokes alphabetically using the Sort by field. You can search a spoke by name in the Search field.

2.  Select **Save and continue**.

    If you want to skip this section, select **Skip**.

    The **Select action** section is displayed.
