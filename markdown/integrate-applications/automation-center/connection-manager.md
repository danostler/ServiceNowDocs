# Working with Connection manager

Connection Manager enables you to add data from a third-party tool into Automation Center tables by correctly mapping and testing the data import.

-   **[Create an automation source](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/create-auto-source.md)**  
Create an automation source to start importing data from any third-party tool.
-   **[Manage automation source](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/manage-auto-source.md)**  
You can perform other actions for an automation source after creating it.
