# Working with ServiceNow Spotlight feature

Use the ServiceNow Spotlight feature to auto-calculate automation request scores. Scores define the execution priority of the automation request. The higher the score, the higher the priority of the automation request.

-   **[Configuration Settings for ServiceNow Spotlight](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/spotlight-config.md)**  
To use the ServiceNow Spotlight feature, set the configuration settings.
-   **[Default ServiceNow Spotlight Criteria](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/default-spotlight-criteria.md)**  
ServiceNow Spotlight is available with default criteria. Each field provides a score based on the default criteria.
-   **[Working with ServiceNow Spotlight](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/use-spotlight.md)**  
Use the spotlight feature to auto-calculate the score of an automation request.
-   **[Set up ServiceNow Spotlight](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/setup-spotlight.md)**  
Set up ServiceNow Spotlight to start using and getting the benefits of this feature. The Spotlight feature automatically calculates the execution priority of the automation requests.
