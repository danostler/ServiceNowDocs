# Create an action request from Automation Center

Create an action request from Automation Center to request for actions to resolve issues causing your automations to fail.

## Before you begin

Role required: sn\_ac.automation\_business\_user, sn\_ac.automation\_technical\_user, or sn\_ac.automation\_admin

## Procedure

1.  Navigate to **All** &gt; **Automation Center** &gt; **Build** &gt; **All Automation Requests**.

2.  Select **New**.

3.  On the form, fill in the fields.

    For a description of the field values, see [Automation request form from Automation Center](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/automation-request-autocenter-form.md).

4.  Select **Submit**.

5.  Complete the additional fields on the form, depending on your role.

6.  Select **Update**.
