# Customize the layout for an automation request form

Change the layout of the automation request form from the Service Catalog to meet the requirements of your organization requirements.

## Before you begin

Role required: sn\_ac.automation\_admin

## About this task

In this task, you can change the layout of the automation request form generated by navigating to **All** &gt; **Automation Center** &gt; **Build** &gt; **Submit an Automation Request**.

## Procedure

1.  Navigate to **All** &gt; **Automation Center** &gt; **Administration** &gt; **Modify catalog for Automation Request**.

2.  On the form, if Global is the current application, select **here** in **To edit this record click here**.

3.  Make the required changes.

    For example, in the Variables related list, if you want to change the **Process Name** field to **Process**, double-click the **Process Name** field, make changes, and then select **Save**.

4.  Select **Update**.
