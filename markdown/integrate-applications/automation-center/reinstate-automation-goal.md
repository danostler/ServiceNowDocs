# Reinstate an automation goal

Reinstate an already achieved automation goal to restore and alter the scope of the automation.

## Before you begin

Create an automation goal. For more information, see [Create an automation goal](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/integrate-applications/automation-center/create-automation-goal.md).

Verify that the state of the automation goal is **Achieved**.

Role required: sn\_ac.automation\_business\_user

## Procedure

1.  Navigate to **All** &gt; **Automation Center** &gt; **Automation Center Home**.

2.  Select the list icon \(\[Omitted image "autocenter-list-icon.png"\] Alt text: List icon.\).

3.  On the **Lists** tab, under **Administration**, select **Automation Goals**.

4.  Open an achieved automation goal that you want to reinstate.

5.  Select **Reinstate**.

6.  In the **Target date** field, specify the target date.

    **Note:** If you don’t specify the target date and save the form, then when the goal is moved to the **Achieved** state, the target date is automatically set to one day later than the start date.

7.  Select **Save**.
