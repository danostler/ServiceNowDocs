# Create an export set from a list

You can quickly create an export set from a list.

## Before you begin

Role required: export\_set\_admin

## Procedure

1.  Navigate to a list of records.

2.  Perform the appropriate action for the list version.

    |Version|Action|
    |-------|------|
    |**List v2**|Right-click the list header and select **Export** &gt; **Export Set**.|
    |**List v3**|Open the list title menu and click **Export**, and then click **Export Set**.|

    The Export Set form appears. An export definition is created automatically based on the list.

3.  Configure the export set as needed.

4.  Click **Submit**.
