# Screen

The screen components in the RPA Desktop Design Studio enable capturing program windows and changing screen resolution.

-   **[Use the Capture component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-screen-capture.md)**  
Capture a screen or a window in the desktop or an application as part of an automation Workflow using the Capture component in the RPA Desktop Design Studio .
-   **[Use the CaptureWindow component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-screen-capturewindow.md)**  
Capture the image of an application window as part of an automation Workflow by using the CaptureWindow component in RPA Desktop Design Studio.
-   **[Use the ChangeResolution component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-screen-changeresolution.md)**  
Change the resolution of a user's system screen as part of an automation Workflow by using the ChangeResolution component in RPA Desktop Design Studio.
-   **[Use the GetResolution component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-utilities-screen-getresolution.md)**  
Get the resolution of the current screen by using the GetResolution component in RPA Desktop Design Studio.
-   **[Properties of the Screen components](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/util-screen-prop.md)**  
Learn about the properties of the Screen components in RPA Desktop Design Studio.
