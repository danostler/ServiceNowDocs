# Set up inbound connections

Inbound connections allow external systems to send data, events, or trigger actions in ServiceNow.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Workflow Data Fabric** &gt; **Connect Hub**.

2.  Click the **Connections and Credentials** tab.

3.  Click **Create** and select **Inbound Connection**.

4.  In the **System** field, select the external system that will send data.

5.  In the **Connector Type** field, select the type of connector that this incoming connection will use.

6.  In the **Spoke** field, select the spoke that will process the incoming data.

7.  In the **Endpoint Name** field, enter a name for the endpoint.

8.  In the **Authentication Method** field, select an authentication method and enter the authentication details for your chosen method.

9.  Click **Save**.
