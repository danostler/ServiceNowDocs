# View Connections

View the inbound and outbound connections for an external system.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Workflow Data Fabric** &gt; **Connect Hub**.

2.  Select the **External Systems** card.

    The **External Systems** tab displays the list of external systems \(custom and default systems\) currently available in Connect Hub.

3.  Select an external system and select **Connections**.

    The list of outbound connections available for the external system is displayed.

    **Note:**

    To view the inbound connections, click the **Inbound** tab,

4.  Click a connection to display the connection settings in a new **Connection** tab.
