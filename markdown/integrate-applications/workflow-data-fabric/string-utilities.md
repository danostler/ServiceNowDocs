# String utilities

The String components in the RPA Desktop Design Studio perform different operations on strings. For example, replace or convert string to upper or lower case.

-   **[Use the Base64Decode component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-utilities-stringutilities-base64decode.md)**  
Convert a base64 string to text by using the Base64Decode component in RPA Desktop Design Studio.
-   **[Use the Base64Encode component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-utilities-stringutilities-base64encode.md)**  
Encode a string with the base64 algorithm by using the Base64Encode component in RPA Desktop Design Studio.
-   **[Use the Compare component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-stringutil-compare.md)**  
Compare two string objects that are based on their ASCII values by using the Compare component in RPA Desktop Design Studio.
-   **[Use the Contains component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-stringutil-contains.md)**  
Check whether a body of text contains a specified text by using the Contains component in the RPA Desktop Design Studio.
-   **[Use the Format component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-utilities-stringutilities-format.md)**  
Provide a placeholder and a value in a body of text by using the Format component in RPA Desktop Design Studio.
-   **[Use the GetLength component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-stringutil-getlength.md)**  
Count the number of characters including spaces and special characters in a string by using the GetLength component in RPA Desktop Design Studio.
-   **[Use the GetHashCode component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-utilities-stringutilities-gethashcode.md)**  
Generate a HashCode from a string by using the GetHashCode component in RPA Desktop Design Studio.
-   **[Use the GetTextAfter component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-stringutil-gettextafter.md)**  
Find the string characters that are placed after a string by using the GetTextAfter component in RPA Desktop Design Studio.
-   **[Use the GetTextBefore component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-stringutil-gettxtbfr.md)**  
Find the string characters that are placed before a string by using the GetTextAfter component in RPA Desktop Design Studio.
-   **[Use the GetTextBetween component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-stringutil-gettxtbtwn.md)**  
Find the string characters that are placed between two strings by using the GetTextBetween component in RPA Desktop Design Studio.
-   **[Use the GetValueByRegex component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-utilities-stringutilities-getvaluebyregex.md)**  
Specify a regular expression and characters to find characters that match by using the GetValueByRegex component in RPA Desktop Design Studio.
-   **[Use the IsEqual component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-stringutil-isequal.md)**  
Check whether two string objects have the same value by using the IsEqual component in RPA Desktop Design Studio.
-   **[Use the IsNullOrEmpty component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-stringutil-isnullempty.md)**  
Determine whether the text is null or an empty string by using the IsNullOrEmpty component in RPA Desktop Design Studio.
-   **[Use the IsNullOrWhiteSpace component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-stringutil-isnullwhitespace.md)**  
Determine whether the text is null or contains only white-space characters by using the IsNullOrWhiteSpace component in RPA Desktop Design Studio.
-   **[Use the Join component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-stringutil-join.md)**  
Concatenate the elements of an array into a string by using the Join component in RPA Desktop Design Studio.
-   **[Use the Occurrence component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-utilities-stringutilities-occurrence.md)**  
Count how many times a character or word appears in a text body by using the Occurrence component in RPA Desktop Design Studio.
-   **[Use the RemoveWhiteSpace component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-stringutil-rmvwhtspc.md)**  
Remove any white spaces from a string by using the RemoveWhiteSpace component in RPA Desktop Design Studio.
-   **[Use the Replace component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-stringutil-replace.md)**  
Replace all occurrences of a word in a text with another word by using the Replace component in RPA Desktop Design Studio.
-   **[Use the Split component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-stringutil-split.md)**  
Divide a string into substrings by applying a specified separator. You can do this task by using the Split component in RPA Desktop Design Studio.
-   **[Use the SubString component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-stringutil-substring.md)**  
Retrieve a substring from a string by using the SubString component in RPA Desktop Design Studio. You can specify the start character position and length of the substring.
-   **[Use the ToLowerCase component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-stringutil-tolwrcase.md)**  
Convert an uppercase text to lowercase letters by using the ToLowerCase component in RPA Desktop Design Studio.
-   **[Use the ToUpperCase component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-stringutil-touprcase.md)**  
Convert a lowercase text to uppercase by using the ToUpperCase component in RPA Desktop Design Studio.
-   **[Use the Trim component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-stringutil-trim.md)**  
Remove the white-space characters at the beginning and end of a string by using the Trim component in RPA Desktop Design Studio.
-   **[Use the TrimEnd component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-stringutil-trimend.md)**  
Remove the white-space characters that you specify at the end of the text by using the TrimEnd component in RPA Desktop Design Studio.
-   **[Use the TrimStart component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-stringutil-trimstart.md)**  
Remove white-space characters at the beginning of the text that you specify by using the TrimStart component in RPA Desktop Design Studio.
-   **[Properties of the String Utilities components](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/util-stringutil-prop.md)**  
Learn about the properties of the String Utilities components in RPA Desktop Design Studio.
