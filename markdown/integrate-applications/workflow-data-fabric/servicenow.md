# ServiceNow

The ServiceNow components enable you to interact with ServiceNow platform features such as the Flow Designer, manage attachments from table records, and trigger embedded task automations.

For more information about Embedded Task Automation, see [Embedded Task Automation \(ETA\) in RPA Hub](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/embedded-task-auto-rpa.md).

-   **[Flow Designer components](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/flow-designer-components.md)**  
The Flow Designer components let you invoke flows, subflows, and actions from Workflow Studio and use them as part of a workflow. You can also use these components to determine a flow's output or execution status.
-   **[Attachments](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/attachments.md)**  
The Attachment components enable you to upload or download attachments from table records in the RPA Hub and get the metadata of the attachments.
-   **[Forms](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/forms_sn_rpa_studio.md)**  
The Forms components help you to trigger attended automations that are enabled with the Embedded Task Automation.
