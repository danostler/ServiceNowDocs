# Exploring Now Assist for Workflow Data Fabric \(WDF\)

With the Now Assist for Workflow Data Fabric \(WDF\) application, you can obtain information about setting up integrations.

## Now Assist for WDF overview

Use Now Assist for WDF to discover data fabric tables, connectors, and collectors, and request guidance on how to set up integrations.

Now Assist for WDF includes the oneExtend LLM skill that enables you to find information about collectors, connectors, data fabric tables, data products, and more.
