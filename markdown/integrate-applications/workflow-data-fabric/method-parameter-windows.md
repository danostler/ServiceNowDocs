# Parameters of the Windows connector methods

Learn about the parameters of the Windows connector methods in RPA Desktop Design Studio.

|Parameter Name|Description|Mandatory?|
|--------------|-----------|----------|
|Args|Arguments to pass as the initial values to start an application.|No|
|Form data field|Data to set in the form fields of the element.|Yes|
|Return \(Bitmap\)|Returns the screenshot image of the screen in the Bitmap format.|Not applicable|
|Path|Directory path where the initial files to start the application are stored.|Yes|
|Form element data|Returns the captured form element data.|Not applicable|

## Enter inputs for parameters

To enter inputs, see [Configure port properties](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/configure-input-port-properties.md).
