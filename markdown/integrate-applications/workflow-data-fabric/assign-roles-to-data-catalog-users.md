# Assign roles to Data Catalog users

Assign roles to control access to features, capabilities, and data in Data Catalog.

## Before you begin

Role required: admin

## About this task

For detailed information about roles that are installed with Data Catalog, see [Data catalog roles](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/data-catalog-roles.md).

## Procedure

-   Assign roles to users and groups using the Data Catalog feature.

    -   Assign a role to a user
    -   Assign a role to a group
