# Finding and accessing data assets

The Data Catalog provides a searchable view of data assets collected from connected systems. You can find assets by keyword, tag, domain, or glossary term, view metadata and lineage, and request access to data products and data interfaces.

The catalog aggregates metadata from multiple connected systems into a single location. Each asset record includes technical metadata collected automatically, along with any business context that data stewards have added — such as glossary terms, tags, and domain assignments.

When you find an asset, you can view its full details: schema, data lineage, related assets, and ownership information. For data products and data interfaces, you can submit an access request directly from the asset details page.
