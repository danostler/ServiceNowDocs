# Zip

The Zip components in the RPA Desktop Design Studio perform standard Zip utility operations. For example, extract or remove files.

-   **[Use the AddDirectory component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-zip-adddir.md)**  
Add the directory contents to an existing zip archive by using the AddDirectory component in RPA Desktop Design Studio.
-   **[Use the AddFile component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-zip-addfile.md)**  
Add a file to an existing zip archive by using the AddFile component in RPA Desktop Design Studio.
-   **[Use the AddFiles component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-zip-addfiles.md)**  
Add multiple files to an existing zip archive by using the AddFiles component in RPA Desktop Design Studio.
-   **[Use the CreateFile component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-zip-createfile.md)**  
Zip an existing folder or create a zip file by using the CreateFile component in RPA Desktop Design Studio.
-   **[Use the ExtractAll component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-zip-extractall.md)**  
Extract all the files from a zip archive to a specific location by using the ExtractAll component in RPA Desktop Design Studio.
-   **[Use the ExtractFile component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-zip-extractfile.md)**  
Extract a file from a zip archive to a specified location by using the ExtractFile component in RPA Desktop Design Studio.
-   **[Use the RemoveFile component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-zip-removefile.md)**  
Delete a specified file in a zip archive by using the RemoveFile component in RPA Desktop Design Studio.
-   **[Properties of the Zip components](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/util-zip-prop.md)**  
Learn about the properties of the Zip components in RPA Desktop Design Studio.
