# Methods and events of elements

The methods and events for various form designing elements are displayed in the Object Explorer pane.

-   **[Methods and events of the Button element](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/form-control-button.md)**  
The Button element in RPA Desktop Design Studio enables you to add one or more buttons on the form so that your user can click a button to perform an action.
-   **[Methods and events of the TextBlock element](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/form-control-textblock.md)**  
The TextBlock element in RPA Desktop Design Studio enables you to add a label on the form so that your user can view that label as read-only information.
-   **[Methods and events of the TextBox element](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/form-control-textbox.md)**  
The TextBox element in RPA Desktop Design Studio enables you to add a text box to a form so that your user can provide information within that text box.
-   **[Methods and events of the Date element](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/form-control-date.md)**  
The Date element in RPA Desktop Design Studio enables you to provide a date picker on the form. It enables the user to select a specific date on the form.
-   **[Methods and events of the Time element](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/form-control-time.md)**  
The Time element in RPA Desktop Design Studio enables you to provide a time picker on the form so that your user can select a specific time on the form.
-   **[Methods and events of the DateTime element](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/form-control-datetime.md)**  
The DateTime element in RPA Desktop Design Studio enables you to provide a date and time picker on the form so that your user can select a specific date and time on the form.
-   **[Methods and events of the CheckBox element](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/form-control-checkbox.md)**  
The CheckBox element in RPA Desktop Design Studio enables you to capture true or false, yes or no, or on or off inputs from users. It helps in decision-making, enabling or disabling features, and controlling workflow execution.
-   **[Methods and events of the RadioButton element](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/form-control-radiobutton.md)**  
The RadioButton element in RPA Desktop Design Studio enables you to select a choice from a group of choices.
-   **[Methods and events of the ComboBox element](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/form-control-combobox.md)**  
The ComboBox element in RPA Desktop Design Studio enables you to display the data in a drop-down combination box.
-   **[Methods and events of the ListBox element](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/form-control-listbox.md)**  
The ListBox element in RPA Desktop Design Studio enables you to display a list of items from which you can select one or more Items.
-   **[Methods and events of the DataGrid element](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/form-control-datagrid.md)**  
The DataGrid element in RPA Desktop Design Studio enables you to display and manipulate tabular data.
-   **[Methods and events of the PasswordBox element](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/form-control-passwordbox.md)**  
The PasswordBox element in RPA Desktop Design Studio is a text box that displays placeholder characters while you type a string.
-   **[Methods and events of the ImageControl element](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/form-control-imagecontrol.md)**  
The ImageControl element in RPA Desktop Design Studio enables you to display images in a form.
-   **[Methods and events of the ImageViewer element](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/form-control-imageviewer.md)**  
The ImageViewer element in RPA Desktop Design Studio enables you to open and view image files including JPEG, GIF, WMF, and other image formats in a form. You can also stretch and shrink images, rotate the images at different angles through all axis, and save them in different formats.
-   **[Methods and events of the TabControl element](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/form-control-tabcontrol.md)**  
The TabControl element in the RPA Desktop Design Studio enables you to add one or more tabs in your form.
-   **[Methods and events of the Web Browser element](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/form-control-webbrowser.md)**  
The Web Browser element in the RPA Desktop Design Studio enables you to host web pages and provides web browsing capabilities to your application.
-   **[Methods and events of the Form Renderer element](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/form-control-formreneder.md)**  
The Form Renderer element in RPA Desktop Design Studio enables you to build forms quickly with controls such as TextBox, ComboBox, CheckBox, RadioButton, ToggleButton, Date, Time, and PasswordBox.
