# View Connectors

View the list of connectors for an external system.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Workflow Data Fabric** &gt; **Connect Hub**.

2.  Select the **External Systems** card.

    The **External Systems** tab displays the list of external systems \(custom and default systems\) currently available in Connect Hub.

3.  Select an external system and select **Connectors**.

    The list of connectors available for the external system are displayed as cards.

4.  Click a connector card to display the connection settings in a new **Connector** tab.
