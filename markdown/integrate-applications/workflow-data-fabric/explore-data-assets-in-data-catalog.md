# Explore data assets in the data catalog

Find and explore catalog assets through search, filtering, and browsing.

Filter and sort data assets in the data catalog 

## Before you begin

Role required: WDF Consumer \(wdf\_consumer\)

## About this task

Discover data assets using search or browsing. Search looks across asset names, descriptions, tags, and other metadata fields. Browse lets you explore assets by source system, domain, or collection without specific search terms. \[Omitted image "data-catalog-home.png"\] Alt text: View of data assets in the data catalog

## Procedure

1.  Navigate to **All** &gt; **Workflow Data Fabric** &gt; **Workflow Data Fabric Home**.

2.  Select the Data catalog icon in the left sidebar.

3.  Browse the data assets available in the catalog.

4.  Apply filters to narrow the list of data assets.

    Filters help narrow large result sets to the most relevant assets. Apply multiple filters to create precise searches combining different attributes.

    1.  Review available filters in the left panel.

    2.  Select filter options to refine the list.

        Common filters:

        -   Asset Type: Table, Column, View, Dashboard, Report
        -   Source System: Filter by the originating data platform
        -   Domain: Organization or business area
        -   Tags: Applied classification tags
        -   Owner: Data owner or steward
5.  Change the sort order to organize the list by different criteria.

    From the sort dropdown menu, select a sort option:

    -   Relevance: Best matches first based on search keywords \(default\)
    -   Name: Alphabetical order \(A-Z\)
    -   Last Modified: Most recently updated first
6.  Find assets using search.

    1.  Enter keywords in the search bar.
    2.  Review suggested results as you type or press Enter to view all results.
    3.  Select any asset to view full details.
    Search tips:

    -   Use specific terms for better results \(for example, "customer orders" instead of "data"\)
    -   Search looks across multiple fields including names, descriptions, and business context
    -   Results display with key information including asset type, source system, and description
