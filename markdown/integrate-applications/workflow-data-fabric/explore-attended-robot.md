# Exploring Attended Robot

Learn about the Attended Robot features that enable you to run attended automations.

## Attended Robot overview

The ServiceNow® Attended Robot is a software agent that you install on a Windows machine. Attended Robot can execute attended automations that require human intervention.

Attended automations are typically triggered manually.

## Attended Robot benefits

-   User can start the automation by clicking a button that triggers a bot process on the same Window's machine.
-   Executes tasks in real time with a human in the loop for complex or subjective decisions as required.
