# Data Reader

The Data Reader components in the RPA Desktop Design Studio enable reading of data from different programs such as MS Excel and CSV.

**Note:** OLEDB 32 bit is not supported for the Data Reader components.

For details on the issues related to the Data Reader component, see [KB article KB1123553](https://support.servicenow.com/kb?id=kb_article_view&sysparm_article=KB1123553) and [KB article KB1123554](https://support.servicenow.com/kb?id=kb_article_view&sysparm_article=KB1123554).

-   **[Use the GetExcelSheetNames component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-datareader-getexcelsheetnames.md)**  
Get the names of the sheets in an Excel file as part of an automation Workflow by using the GetExcelSheetNames component in RPA Desktop Design Studio.
-   **[Use the QueryExcel component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-datareader-queryexcel.md)**  
Query data from a Microsoft Excel file with the QueryExcel component in RPA Desktop Design Studio.
-   **[Use the ReadFromCSV component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-datareader-readfromcsv.md)**  
Read data from a CSV file by using the ReadFromCSV component in RPA Desktop Design Studio.
-   **[Use the ReadFromExcel component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-datareader-readfromexcel.md)**  
Extract data from a Microsoft Excel file as a DataTable by using the ReadFromExcel component in RPA Desktop Design Studio.
-   **[Use the ReadFromXPS component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-datareader-readfromxps.md)**  
Read the data from an XPS file by using the ReadFromXPS component in RPA Desktop Design Studio.
-   **[Properties of the Data Reader components](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/util-datareader-prop.md)**  
Learn about the properties of the Data Reader components in RPA Desktop Design Studio.
