# Disconnect from an RPA Hub instance in Attended Robot

Disconnect anytime from an RPA Hub instance while using the Attended Robot.

## Before you begin

Connect to an RPA Hub instance. For more information, see [Connect to an RPA Hub instance from Attended Robot](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/connect-rda-instance-rpa.md).

Role required: none

## Procedure

1.  In the Attended Robot dialog box, select **Disconnect**.
