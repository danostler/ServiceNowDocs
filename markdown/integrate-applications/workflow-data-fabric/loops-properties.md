# Properties of the Loops components

Learn about the properties of the Loops components in RPA Desktop Design Studio.

|Property|Description|
|--------|-----------|
|Search|Option to enable a search for a property.|
|Delay After Execution|The number of seconds to pause after running.|
|Delay Before Execution|The number of seconds to pause before running.|
|Message|Custom message for the user.|
|Enable Timeout|Component times out after the specified time period.|
|Timeout|Custom time in seconds before the component times out. The default time is 60 seconds.|
|Delay for each execution \(Seconds\)|Custom delay in seconds before execution of each item in the object.|
|Delay for each Execution|Waiting time, in seconds, for each step performed until the ForLoop ends.|
|Increment By|Number, which increments, in the From and To boxes in the component.|
