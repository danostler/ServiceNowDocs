# Use Guided Tours in RPA Hub

Use Guided Tours in RPA Hub to get instructions on how to perform a task.​ Guided Tours use a series of steps that may span multiple pages. The tour includes links to detailed documentation.​

## Before you begin

You must do this task in the classic environment.

Role required: Role required: sn\_rpa\_fdn.rpa\_release\_manager or sn\_rpa\_fdn.rpa\_admin

## About this task

Guided Tours help train and onboard users within the RPA Hub user interface \(UI\). Each tour contains a series of interactive steps that help you complete a specific task or a process within the same browser window. If needed, administrators can create more tours. For more information, see .

When you visit a page that has a guided tour for the first time, a welcome screen appears. You can choose to either begin the tour or skip the tour.

The guided tour isn’t yet available when Next Experience is enabled.

In RPA Hub, guided tours are currently available for creating queues and configuring a bot process and associated assets.

## Procedure

1.  Navigate to **All** &gt; **Robotic Process Automation**.

2.  Select the feature that you want to configure.

    For example, select **Build** &gt; **Queues** to use the guided tour for creating queues.

3.  Select the **Show help** icon \(\[Omitted image "help-center-icon-rpa.png"\] Alt text: Show help icon\).

4.  Select **Pin Help Center menu** icon \[Omitted image "pol-nav-pin.png"\] Alt text: Pin Help Center menu to pin the Help Center side panel on the page.

5.  Select **Take a Tour**​.

6.  Take the necessary action and select **Next** to progress through the tour.

    For ending the tour at any point, you can select **End Tour** in the side panel or select the Close icon in the pop-up window and confirm to end the tour.

7.  Select **Complete** after the tour is ended​.
