# Data Modellers

The Data Modeller creates templates by adding fields or data to PDF and XML documents. It also provides methods to classify the templates and use the fields or data created.

-   To use the Data Modeller, see [Use the XmlTemplate component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/configure-xml-template.md).
-   To expose the methods, see [Use connector method](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/use-connector-method.md).
-   To use the methods, see [Use a component in RPA Desktop Design Studio](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/configure-components.md).

-   **[Use the XmlTemplate component](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/configure-xml-template.md)**  
Create templates from existing XML or PDF documents by adding custom data or fields to the documents.
-   **[View the XmlTemplate methods](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/view-xmltemplate-methods.md)**  
After configuring the XmlTemplate connector, you can expose its methods.
-   **[XmlTemplate connector methods](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/connector-xmlTemplate-methods.md)**  
The XMLTemplate methods classify the templates and get the fields added to the templates.
-   **[Parameters of the XmlTemplate connector methods](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/parameters-xmlTemplate-methods.md)**  
Learn about the various parameters of the XmlTemplate connector methods in RPA Desktop Design Studio.
-   **[Properties of the XmlTemplate methods](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/xmltemplate-methods-properties.md)**  
Learn about the properties of the XmlTemplate methods in RPA Desktop Design Studio.
