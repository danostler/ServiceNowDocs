# Configuring metadata collectors

Metadata collectors automatically import data asset information from external systems into Data Catalog.

Metadata collectors provide read-only access to external data systems and harvest information about data assets, making them searchable and discoverable in the catalog. Each collector type is designed for a specific platform and processes that platform's metadata structure. Configuration involves setting up source-system prerequisites and creating a collector record for each platform.
