# Monitoring relay connectivity

Monitor the connection health and registration status of private relays registered with Reverse Tunnel.

## Available monitoring options

After a private relay is registered and connected, use the relay record to verify connectivity and diagnose issues.

-   [Monitor relay connection health](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/monitor-relay-health.md)

    View real-time connection status, heartbeat data, and reconnection activity for a registered relay.

-   [Manage relay service endpoints through Reverse Tunnel](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/integrate-applications/workflow-data-fabric/manage-relay-services.md)

    Add or update the service endpoints assigned to a relay.
