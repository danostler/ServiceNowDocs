# Review the import set

After an import set completes, you can review the completed import and clean up import set tables. Even if no errors are reported by the progress indicator, review how your data is imported into the new import set.

## Before you begin

Role required: admin.

## Procedure

1.  Click **View imported data**.

    There are several fields that are visible in the table that were not part of the original data source, these are system rows that can be used to facilitate scripted import operations.

2.  Review the data.

3.  Return to the previous page.


## What to do next

Create a transform map to associate with this import set.
