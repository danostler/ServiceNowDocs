# Create a case with Disputes intake via Virtual Agent

Disputes intake via Virtual Agent streamlines the card dispute submission process for cardholders by providing a chat interface.

The Disputes intake via Virtual Agent skill in the Now Assist for Financial Services Operations \(FSO\) application gathers the case information from the customer’s responses, and presents dispute questionnaire questions to the customer in conversational language.

To create a new dispute case using Disputes intake via Virtual Agent, see [Submit a dispute case with Disputes intake via Virtual Agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/financial-services-operations/now-assist-for-financial-services-operations-fso/submit-dispute-case-disputes-intake-via-virtual-agent.md).
