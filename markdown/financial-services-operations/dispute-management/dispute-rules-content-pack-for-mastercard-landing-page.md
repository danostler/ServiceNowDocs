# Dispute Rules Content Pack for Mastercard

The ServiceNow® Dispute Rules Content Pack for Mastercard application provides your agents with questionnaires that they can use to intake dispute-related information for various dispute categories as per the Mastercard guidelines.

## Get started

The Dispute Rules Content Pack for Mastercard application can help you do the following tasks:

-   Incorporate a dispute questionnaire that is based on the Mastercard Chargeback Guide.
-   Provide a decision table to run chargeback eligibility rules that follow Mastercard's guidelines.
