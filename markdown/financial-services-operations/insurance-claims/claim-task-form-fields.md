# Claim Task form fields

This section describes the claim task form, which displays the information about the claims tasks, which are created and assigned to users to complete the work to resolve the insurance claims.

The task form includes the following fields.

<table id="table_ksk_khs_dcc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Number

</td><td>

Automatically assigned task number.

</td></tr><tr><td>

Parent

</td><td>

Number of the parent claim case.

</td></tr><tr><td>

Priority

</td><td>

Assigned priority of the task: -   1 - Critical
-   2 - High
-   3 - Moderate
-   4 - Low \(default\)

</td></tr><tr><td>

Status

</td><td>

Current status of the task.

</td></tr><tr><td>

Due date

</td><td>

Due date for this task.

</td></tr><tr><td>

Category

</td><td>

Category of the task:-   General
-   Report
-   Follow-up
-   Document
-   Closure
-   Validation

</td></tr><tr><td>

Assignment group

</td><td>

Assigned group.

</td></tr><tr><td>

Assigned to

</td><td>

Assigned user.

</td></tr><tr><td>

Short description

</td><td>

Brief summary of the task.

</td></tr><tr><td>

Description

</td><td>

Description of the work that needs to be done in order to complete the task.

</td></tr><tr><td>

Work notes

</td><td>

Information about the task, such as the steps taken toward the resolution.
