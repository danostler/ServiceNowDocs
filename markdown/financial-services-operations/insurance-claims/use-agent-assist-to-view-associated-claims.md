# Use Agent assist to view associated claims

Use the Agent assist contextual search to view associated claims and relevant knowledge articles.

## Before you begin

Role required: sn\_ins\_claim\_cml.fnol\_representative, sn\_ins\_claim\_cml.auto\_processor, sn\_ins\_claim\_cml.auto\_adjuster, sn\_ins\_claim\_cml.manager

## Procedure

1.  Navigate to **All** &gt; **Financial Services Operations** &gt; **Workspace**.

2.  Select the lists icon \(\[Omitted image "inline-data-vis-96px-list.png"\] Alt text: lists icon\).

3.  Open the task from the case playbook or the Commercial auto claim adjuster tasks list.
