# Review a customer interaction with Disputes intake via Virtual Agent

Retrieve a chat interaction between a customer and Disputes intake via Virtual Agent to review a transcript of the conversation and confirm responses.

## Before you begin

Role required: sn\_bom\_credit\_card.dispute\_agent, sn\_bom\_credit\_card.dispute\_manager, sn\_bom\_credit\_card.dispute\_viewer, sn\_bom\_credit\_card.contributor, sn\_bom.b2c\_agent, sn\_bom.b2b\_agent, sn\_bom.adjuster, sn\_bom.fnol\_representative

## Procedure

1.  Navigate to **All** &gt; **Financial Services Operations** &gt; **Workspace**.

2.  Select the lists icon \(\[Omitted image "inline-data-vis-list.png"\] Alt text: Lists icon.\).

3.  Open a card disputes service case.

4.  In the case, select the Interactions tab.

5.  In the Interactions list, select the interaction record you want to review.


## Result

The interaction record is displayed with the interaction transcript.

\[Omitted image "disputes-va-transcript.png"\] Alt text: The interaction transcript shows the conversation history between Virtual Agent and the customer.
