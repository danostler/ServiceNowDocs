# Work on an individual life insurance underwriting task

An underwriter can work on an individual life insurance underwriting task to evaluate the policy request and analyze risks for an insurance policy customer.

## Before you begin

Role required: sn\_ins\_indiv\_uw.underwriter

## About this task

The configured workflow automatically generates an underwriting task for an underwriting agent. The task is assigned to the underwriting operations team or an agent based on the assignment rules.

Use the case playbook that provides the activities and tasks required to research and complete the request.

## Procedure

1.  Navigate to **All** &gt; **Financial Services Operations** &gt; **Workspace**.

2.  Select the lists icon \(\[Omitted image "inline-data-vis-96px-list.png"\] Alt text: lists icon\).

3.  Open the task from the underwriting task list.
