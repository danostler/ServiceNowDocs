# Activate the Socure spoke from Integration Hub

Activate the Socure spoke plugin \(com.sn\_socure\_spoke\) from Integration Hub. Activating this plugin enables you to enhance your Workflow Studio experience with integration-specific content. You can use pre-built flows and actions to automate your integrations or create your own integration automation.

## Before you begin

Role required: admin

## Procedure

1.  To set up the Socure spoke, follow the instructions in Set up the Socure spoke.
