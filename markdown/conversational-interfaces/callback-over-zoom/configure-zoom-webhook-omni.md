# Configure a Zoom webhook for Zoom extension for Omnichannel Callback

Configure a Zoom webhook in ServiceNow to notify the Zoom extension for Omnichannel Callback app when certain events occur in Zoom.

## Before you begin

You must have copied the verification token displayed during the configuration of the Notify Zoom connector in Zoom.

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Zoom Spoke** &gt; **Zoom Webhook Registries**.

2.  Select **Zoom Callback Transcript completed** from the list.

3.  Paste the verification token for validating event notifications in the **Token** field.

4.  Select **Update**.
