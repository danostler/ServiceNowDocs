# Conversational Interfaces reference

Conversational Interfaces reference topics provide shared technical descriptions of features that apply to Virtual Agent and Agent Chat, such as the Interactions table.

-   **[Data management in Conversational Interfaces](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/conversational-interfaces/va-data-mgmt.md)**  
Use data management tools, such as table cleaners, to maintain Conversational Interfaces tables.
-   **[Interaction Insights table](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/conversational-interfaces/ci-interaction-insights-table.md)**  
User sentiment analysis prepares agents for the tone of incoming conversations. Business rules can be created based on this value and agents can take appropriate actions on the chat, whether it comes from Virtual Agent, another live agent, or from a queue.
