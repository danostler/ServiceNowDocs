# Edit a dashboard in the dashboard library

Edit an existing dashboard in the Related Dashboards page to update details such as the display name, description, or the associated dashboard itself.

## Before you begin

Role required: virtual\_agent\_admin

## About this task

You can edit an existing dashboard in the library to update its display name, description, or replace the dashboard with a different one. This allows you to swap out a dashboard without removing and re-adding it to the library. Editing a dashboard does not modify the underlying Platform Analytics dashboard content.

## Procedure

1.  Navigate to **All** &gt; **Assistant Designer** &gt; **Analytics** and select Related Dashboards in the left navigation.

2.  Locate the dashboard you want to edit and select **Edit** in the Actions menu.

    \[Omitted image "NAinVA-assistant-designer-analytics-edit-dashboard.png"\] Alt text: Edit Dashboard modal in Dashboard Library

3.  On the **Edit Dashboard** form, update the fields.

    |Field|Description|
    |-----|-----------|
    |Name|Name of the dashboard in Platform Analytics. Select a different dashboard to replace the current one, or keep the existing selection. This field is required.|
    |Display Name|Name that appears on the dashboard card in the library. Enter a name as per your requirements.|
    |Description \(Optional\)|Description of the dashboard you want to add to the library. Enter a brief description of the dashboard with its purpose.|
    |Owner|Displays the owner of the selected dashboard. This field is read-only.|
    |Updated by|Displays the user who last updated the dashboard. This field is read-only.|
    |Updated on|Displays the date when the dashboard was last updated. This field is read-only.|
    |Pin dashboard|Select the check box to automatically pin the dashboard to the Pinned dashboards section after adding it to the library.|

4.  Select **Save** to update the changes to the dashboard.

    Changes are immediately visible to users in the dashboard library.
