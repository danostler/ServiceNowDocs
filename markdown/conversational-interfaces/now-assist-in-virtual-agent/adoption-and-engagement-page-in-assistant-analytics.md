# Adoption and Engagement page in Assistant analytics

View how users are adopting assistants and engaging with them over time, helping you track growth, activity, and interaction patterns.

The Adoption &amp; Engagement dashboard page aggregates metrics related to user adoption and engagement, including average active users, new user growth, conversation volume trends, assist-to-execution trend, and channel distribution. The visual widgets enable administrators to monitor how assistants are being adopted, how frequently they are used, and how engagement evolves across different channels and time periods. These insights support targeted strategies to drive adoption and improve user engagement.

\[Omitted image "NAinVA-assistant-designer-analytics-adoption-engagement.png"\] Alt text: Adoption &amp; Engagement dashboard page in Assistant analytics.

The visualizations on the Adoption &amp; Engagement page help you with the following.

-   Track user growth and engagement trends to inform strategies for increasing assistant adoption and user retention.
-   Identify patterns in conversation volume and channel usage, enabling you to optimize assistant deployment.
-   Monitor assist-to-execution trends to assess how effectively assistants are using assists to deliver desired outcomes.

-   **Average Active Users**

    This area of the dashboard shows the average number of users actively engaging with at least one assistant in the selected period. This metric indicates the level of ongoing engagement and helps track the effectiveness of adoption initiatives.

    \[Omitted image "NAinVA-assistant-designer-analytics-adoption-engagement-avg-actv-users.png"\] Alt text: Average Active Users.

-   **New Users**

    This area of the dashboard shows the number of new users interacting with assistants for the first time in the selected date range. This metric highlights the growth in user base and helps you assess the effectiveness of adoption initiatives.

    \[Omitted image "NAinVA-assistant-designer-analytics-adoption-engagement-new-users.png"\] Alt text: New Users.

-   **Avg Conversations/User**

    This area of the dashboard shows the average number of conversations per unique user in the selected date range. It's calculated using the following formula: \(Total AI assistant conversations\)/\(Number of unique users\). This metric reveals how frequently users are interacting with assistants.

    \[Omitted image "NAinVA-assistant-designer-analytics-adoption-engagement-avg-conv-users.png"\] Alt text: Avg Conversations/User.

-   **Active Assistants**

    This area of the dashboard shows the number of active assistants as on the last date of the selected date range. This metric helps you in identifying the assistants which are currently driving engagement and contributing to adoption.

    \[Omitted image "NAinVA-assistant-designer-analytics-adoption-engagement-active-assistants.png"\] Alt text: Active Assistants.

-   **Conversation Volume Trend**

    This area of the dashboard shows the trend of conversation volumes over time, broken down by assistant. Hover over the chart to see total conversations by assistants and to view the % of conversations on that day relative to the total number of conversations in the selected date range. This metric helps you investigate periods of high or low activity to understand underlying causes and adjust engagement strategies.

    \[Omitted image "NAinVA-assistant-designer-analytics-adoption-engagement-conv-vol-trend.png"\] Alt text: Conversation Volume Trend.

-   **Assist to Execution Trend**

    This area of the dashboard shows the average number of assists consumed per execution or action on a given day. For example, if 100 assists were consumed to complete 50 executions, the assist to execution ratio is 2:1. This metric helps you assess how effectively assistants are using assists to deliver desired outcomes.

    \[Omitted image "NAinVA-assistant-designer-analytics-adoption-engagement-assist-exe-trend.png"\] Alt text: Assist to Execution Trend.

-   **Conversations per Channel**

    This area of the dashboard shows the distribution of conversations across different channels such as Teams, Slack, and so on.

    \[Omitted image "NAinVA-assistant-designer-analytics-adoption-engagement-conv-per-channel.png"\] Alt text: Conversations per Channel.

-   **New User Growth**

    This area of the dashboard shows the growth of new users in a selected date range. A new user is a user who has had their first conversation with an assistant in the selected date range. Track new user growth to evaluate the success of onboarding and adoption efforts.

    \[Omitted image "NAinVA-assistant-designer-analytics-adoption-engagement-new-user-growth.png"\] Alt text: New User Growth.
