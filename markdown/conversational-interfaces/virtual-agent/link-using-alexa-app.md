# Link your account using the Alexa app

Link your ServiceNow account using your Alexa mobile application.

## Before you begin

Role required: admin

## Procedure

1.  Download the Alexa app on your Android or iOS device.

2.  Log in with the same email address that you used for the Amazon developer account.

3.  In the lower-right area of the app, click **More**.

4.  Select **Skills &amp; Games**.

5.  Select **Your Skills** tab.

6.  Click **Dev**.

7.  Select the skill that you created.

8.  If the **enable skill** button does not appear, click **Disable Skill**.

9.  Click **Enable**.

10. Log in with the ServiceNow® instance credentials of any user of the instance.

    You should see that the account is linked.

    **Note:** Insert the user record in the provider\_user\_map record to complete the account linking.
