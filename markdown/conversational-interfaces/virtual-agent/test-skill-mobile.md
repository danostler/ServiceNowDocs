# Test a skill on the Alexa mobile app

Test your Alexa skill using the Alexa mobile app.

## Before you begin

Role required: none

## Procedure

1.  Log in to the Alexa mobile application using your email address that you used for logging in to the developer console.

2.  Speak the invocation name of your skill.

    For example, `alexa now virtual agent`.

    Alexa responds to the Virtual Agent and returns a greeting message.
