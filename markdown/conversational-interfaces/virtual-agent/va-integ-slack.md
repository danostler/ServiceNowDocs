# Integrating ServiceNow Virtual Agent with Slack

The Virtual Agent integration with Slack creates a one-to-one association between the ServiceNow instance that you are installing from and the Slack workspace in which the integration runs. By default, Virtual Agent supports one-click installation that integrates one Slack workspace with one ServiceNow instance.

-   **[Install the Conversational Integration with Slack](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/conversational-interfaces/virtual-agent/install-slack-from-store.md)**  
You can install the Conversational Integration with Slack \(sn\_va\_slack\) application if you have the admin role.
-   **[Integrate ServiceNow Virtual Agent with Slack](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/conversational-interfaces/virtual-agent/add-slack-snow-bot.md)**  
Add the Virtual Agent bot to your ServiceNow® instance to integrate with Slack.
-   **[Override Slack integration](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/conversational-interfaces/virtual-agent/override-slack-install.md)**  
Override Virtual Agent integration with Slack.
-   **[Manage the ServiceNow Virtual Agent integration with Slack](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/conversational-interfaces/virtual-agent/configure-va-slack.md)**  
Manage your Conversational Integration with Slack application, its branding, and customize the bot-specific messages in your ServiceNow instance after integration.
-   **[Remove ServiceNow Virtual Agent integration with Slack](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/conversational-interfaces/virtual-agent/uninstall-virtual-agent-slack.md)**  
Remove the ServiceNow Virtual Agent integration from your ServiceNow instance to disassociate the Slack app.
-   **[Integrating a self-configured bot with Slack workspace](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/conversational-interfaces/virtual-agent/va-integ-single-slack.md)**  
Enables ServiceNow® Virtual Agent to allow multiple ServiceNow instances to integrate with single Slack workspace. This integration also supports GCC deployments.
