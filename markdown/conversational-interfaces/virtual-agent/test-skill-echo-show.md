# Test a skill on the Alexa Echo or Echo Show screen device

Test your Alexa skill using the Alexa Echo or Echo Show screen device.

## Before you begin

Role required: none

## Procedure

1.  Log in to the Alexa Echo or Echo Show using your email address that you used for logging in to the developer console.

2.  Speak the invocation name of your skill or use the touch screen option to select the list option.

    For example, `alexa now virtual agent`.

    Alexa responds to the Virtual Agent and returns a greeting message.
