# Manage the self-configured bot integration with Slack

Manage your Conversational Integration with Slack application, its branding, and customize the self-configured bot-specific messages in your ServiceNow instance after integration.

## Before you begin

Roles required:

-   virtual\_agent\_admin and external\_app\_install\_admin or admin
-   Administrator for third-party applications \(Slack\)

## Procedure

1.  Navigate to **All** &gt; **Conversational Interfaces** &gt; **Settings**.

2.  In **General Settings** under **Channels and integrations**, select **View All**.

3.  Select **Manage** on the Slack tile.

    If you would like to integrate your ServiceNow instance with another Slack bot, select the Plus icon against the Self-configured bot or select the **Add integration** button on the Manage Slack channel page and select **Integrate with Self-configured bot**. For more information, see [Integrate your ServiceNow instance with a self-configured bot](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/conversational-interfaces/virtual-agent/integ-slack-self-config-bot.md) \[Omitted image "add-slack-integ-manage.png"\] Alt text: Manage Slack channel view in Conversational Interfaces general settings. The Add integration for self-configured bot button and Integrate with Self-configured bot option in the Add integration dropdown are highlighted.

    When you select **Manage**, you are redirected to the Manage Slack channel page where you can manage the bots.

4.  Select the Manage Bot icon \[Omitted image "manage-bot-icon.png"\] Alt text: Manage bot icon. and select **Manage Bot**.

    You are redirected to the Bot Configuration page for the selected bot.

5.  Select the **Configuration** tab to brand your Self-configured bot.

    For more information about branding your Self-configured bot, see [Configure branding for your self-configured bot](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/conversational-interfaces/virtual-agent/slack-brand-self-configured.md).

6.  Select the **Bot Messages** tab to create and edit a bot message.

    For more information, see [Create bot messages](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/conversational-interfaces/virtual-agent/create-bot-messages.md).
