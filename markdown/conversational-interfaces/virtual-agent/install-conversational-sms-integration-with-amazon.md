# Install the Conversational SMS Integration with AWS End User Messaging application

You can install the Conversational SMS Integration with AWS End User Messaging application \(sn\_sms\_aws\_adapter\) from the ServiceNow store and integrate it with AWS End User Messaging to enable customer support agent conversations via an SMS channel.

## Before you begin

Role required: Admin or virtual\_agent\_admin

## About this task

Ensure that the application and all of its associated ServiceNow store applications have valid ServiceNow entitlements.

## Procedure

1.  Navigate to **System Applications** &gt; **All Available Applications** &gt; **All**.

2.  Find the Conversational SMS Integration with AWS End User Messaging application \(sn\_sms\_aws\_adapter\) using the filter criteria and search bar.

    You can search for the application by its name or ID.

3.  Select the application and version from the list and select **Install**.

    In the Install dialog box, all dependent plugins are displayed.
