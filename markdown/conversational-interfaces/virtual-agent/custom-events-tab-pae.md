# Custom Events tab

Use the Custom Events tab to view the list of custom events created for Virtual Agent.

To use the Custom Events tab, you must have the Chat Analytics Viewer \(chat\_analytics\_viewer\) role.

\[Omitted image "dashboard-custom-events-dashboard-2-pae.png"\] Alt text: Virtual Agent Custom events tab.

The Custom Events tab shows the dates when custom events were triggered. If no one [created custom events](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/conversational-interfaces/virtual-agent/set-up-custom-events-pae.md), the list of custom events is empty. The maximum number of custom events displayed is 20.

Use the drop-down list to specify the custom event you want view analytics for.
