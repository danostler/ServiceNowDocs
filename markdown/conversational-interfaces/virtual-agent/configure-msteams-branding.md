# Configure branding for your ServiceNow Virtual Agent bot in Microsoft Teams

Enable users to configure branding for the published Microsoft Teams bot app, the ServiceNow® Virtual Agent. Users are provided with the flexibility to customize the default ServiceNow branding for the Microsoft Teams production bot name and icon.

## Before you begin

-   Ensure that you have the admin access to your Microsoft Teams tenant and Microsoft Teams admin center.
-   Ensure that Virtual Agent is installed on your instance.
-   Ensure that the Conversational Integration with Microsoft Teams store app is installed.

Role required: admin

## Procedure

1.  Log in to [Microsoft Teams](https://admin.teams.microsoft.com) admin center.

2.  Navigate to **Teams apps** &gt; **Manage apps**.

3.  In the list of apps, search for the Now Virtual Agent app using the Search bar and open it.

4.  Select **Customize** to customize the app.

5.  In the Customize panel, you can edit the app name, short description, color icon, outline icon, and accent color.

    **Note:** Follow the tool tips for file size and format restrictions for each field. You can also refer to [Microsoft documentation](https://learn.microsoft.com/en-us/microsoftteams/customize-apps) on further details on the app customization functionality.

6.  Select **Apply**.

    You will see the **Publish application** pop-up.

7.  Select **Publish**.

    **Note:** Please note that the changes you made to the app may take up to 24 hours to reflect.
