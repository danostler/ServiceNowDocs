# Topic Recommendations page

Use the Topic Recommendations page to run analysis reports, view recommendations, add topics to Virtual Agent, or link a recommendation to an existing topic.

## View all recommendations \(aggregate view\)

Topic Recommendations shows all results in this view by default. This view allows you to run or refresh analysis reports for all configured sources at one time. The configured sources for the recommendations are listed as hyperlinks. Click the link to open the settings for the report. If a recommendation has a New status, you can select it, and then click **Add to VA**.

\[Omitted image "tr-page-aggregate-view.png"\] Alt text: The aggregate view displays two full lists, regardless of report: one for full topic recommendations, and one for intent-only recommendations.

## View recommendations by source

You can view topic recommendations by their data source. This lets you run a new analysis or refresh an existing analysis for a single data source, rather than running all of them at once. Click **Settings** to view the report's settings. If a recommendation has a New status, you can select it, and then click **Add to VA**.

\[Omitted image "tr-page-source-view.png"\] Alt text: The source view groups recommendations by the data source for the analysis. An analysis report on the Incidents table would be broken out separately from reports on other tables.
