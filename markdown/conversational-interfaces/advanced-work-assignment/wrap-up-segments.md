# Wrap up segments

Define the wrap up segments.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Interaction** &gt; **Wrap Up segments**.

    The Interaction Wrap Up Segments screen appears.

2.  Do one of the following:

    -   To edit an existing segment, select the name of the segment.
    -   To create a new segment, select **New**.
3.  Fill in the fields:

    |Field|Description|
    |-----|-----------|
    |State| |
    |Opened on| |
    |Agent| |
    |Interaction| |
    |Wrap up configuration| |
    |Segmented| |
    |External| |
    |Segment table| |
    |External segment ID| |
    |Segment ID| |
    |Wrap up code| |
    |Notes| |
    |Timed out| |
    |Submitted on| |
    |Awaiting confirmation| |
    |Confirmed on| |

4.  Select **Update** if you are updating an existing segment or **Submit** if you are creating a new segment.
