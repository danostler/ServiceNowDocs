# Create a skill

Create a "German" skill.

## Before you begin

Role required: skill\_admin or admin

## Procedure

1.  Navigate to **All** &gt; **Skills** &gt; **All Skills**, and then click **New**.

2.  Enter the following information in the fields listed:

    -   **Name**: `German`
    -   **Active**: Selected
3.  Select **Submit**.
