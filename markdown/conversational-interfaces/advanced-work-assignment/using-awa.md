# Using Advanced Work Assignment

The following sections detail how awa\_manager or awa\_agent roles can use Advanced Work Assignment \(AWA\) to complete various tasks such as checking an agent's presence and capacity, checking unassigned work items, and creating SMS conversations.

-   **[Check for an agent's presence and capacity](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/conversational-interfaces/advanced-work-assignment/awa-agent-presence.md)**  
View a list of agents in Advanced Work Assignment, their presence status, and capacity for each of the agent's service channels to see which agent can handle a case. This list is filtered across different teams, channels in teams, and channels dashboards.
-   **[Check unassigned interactions](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/conversational-interfaces/advanced-work-assignment/awa-unassigned-interactions.md)**  
View a list of unassigned interactions in Advanced Work Assignment from chat and phone that are currently waiting in the queue and are not assigned to any agents. You can use this list to debug, implement advanced work assignments, or determine why certain work items are not being assigned.
-   **[Check unassigned task work items](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/conversational-interfaces/advanced-work-assignment/awa-unassigned-task-work-items.md)**  
View a list of unassigned task work items in Advanced Work Assignment, such as cases and incidents, that are currently waiting in the queue and are not assigned to any agents. You can use this list to debug, implement advanced work assignments, or determine why certain work items are unassigned.
-   **[Check work items and AWA events](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/conversational-interfaces/advanced-work-assignment/awa-work-items-all.md)**  
View a list of work items and AWA events.
-   **[Using Agent Affinity](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/conversational-interfaces/advanced-work-assignment/awa-agent-affinity-concept.md)**  
Agent Affinity is an Advanced Work Assignment feature that lets you assign work items by an agent's work history, related task, or account team affinity.
-   **[Agent initiated SMS conversation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/conversational-interfaces/advanced-work-assignment/agent-initiated-sms-store-app.md)**  
Customer Service Management supports additional messaging channel such as SMS which tracks interactions between the agent and customer.
-   **[Compose an SMS conversation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/conversational-interfaces/advanced-work-assignment/compose-sms-store-app.md)**  
An agent can compose an SMS to interact with a customer for resolving various issues. If the interaction is not messaging type, you see the **Compose SMS** option.
