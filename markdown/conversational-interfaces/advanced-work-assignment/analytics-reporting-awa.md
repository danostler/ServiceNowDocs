# Analytics &amp; Reporting Solutions

The following sections review advanced analytics and Platform Analytics Solutions. Advanced analytics equips you to view cards that represent the Advanced Work Assignment dashboards that you can access.

-   **[Using Advanced analytics to access AWA dashboards](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/conversational-interfaces/advanced-work-assignment/awa-advanced-analytics.md)**  
Advanced analytics enable you to view cards that represent the Advanced Work Assignment dashboards that you have access to. You can also see a dashboard picker, which lets you view the cards for all the dashboards in a group. You can then search and choose a dashboard from a list.
-   **[Operations dashboard](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/conversational-interfaces/advanced-work-assignment/operations-dashboard.md)**  
 Enable managers to monitor work items across all service channels so that their agents can better support customer needs.
-   **[Platform Analytics Solutions for Advanced Work Assignment](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/conversational-interfaces/advanced-work-assignment/awa-content-pack.md)**  
This Platform Analytics Solution contains preconfigured dashboards. These dashboards contain actionable data visualizations that help you improve your business processes and practices.
