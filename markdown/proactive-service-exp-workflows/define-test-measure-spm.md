# Define test measure definitions for a service test

Define unique test metrics, such as parameters and criteria to run the test on a given service during testing. These metrics help promote that the service meets the desired standards and requirements to provide a satisfactory user experience.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Service Test Management** &gt; **Test Groups** &gt; **All**.

2.  Select the test group that you want to open.

3.  In the Test Measure Definitions related list, define test measures for the service by selecting **New**.

4.  On the form, fill in the fields.
