# Harness integration with DevOps Change Velocity

Connect to your Harness instance to discover pipeline definitions and configure real-time notifications or polling to enable change traceability and automation.

## Get started

Use the workspace to onboard the Harness tool. Configure webhooks manually in Harness to send pipeline notifications to DevOps Change Velocity.
