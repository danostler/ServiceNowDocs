# Rules in Recommended Actions for ITSM

ITSM base system includes the following rules.

<table id="table_lr3_4tn_cyb"><thead><tr><th>

Rule

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Active incident

</td><td>

This will show all valid recommendations if the incident is active.

</td></tr><tr><td>

Active non-child non-MI Incident

</td><td>

This will show all valid recommendations if the incident is active, with no child incident, and which is not a major incident.

</td></tr><tr><td>

AI Search

</td><td>

This will allow you to search for AI powered recommendations across various records in the Service Operations Workspace, including Incident, Incident task, Problem, Problem task, Change Request, Change task, Interaction, and Request.

</td></tr><tr><td>

Major Incident with no Problem record

</td><td>

This will show all valid recommendations for the major incidents with no problem record.

</td></tr><tr><td>

High impact Problem with no known error article

</td><td>

Displays the Create known error article recommendation for a high-impact problem with no attached known error article.

</td></tr></tbody>
