# Mark the universal request as unrestricted in Service Operations Workspace

You can mark a restricted universal request as unrestricted if you identify that the request doesn't have any sensitive information.

## Before you begin

Role required: sn\_uni\_req.sensitiveinfo\_agent or service specific assignment group members

## Procedure

1.  Navigate to **All** &gt; **Workspaces** &gt; **Service Operations Workspace**.

2.  From the navigation list, select **List**.

3.  From the navigation list, select **Universal Request** &gt; **Open**.

4.  Navigate to the universal request that you want to mark as unrestricted.

5.  On the form, select the **Unrestrict** button.
