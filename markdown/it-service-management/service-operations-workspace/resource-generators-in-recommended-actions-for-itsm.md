# Resource generators in Recommended Actions for ITSM

ITSM base system includes the following resource generators.

<table id="table_b4k_3wn_cyb"><thead><tr><th>

Resource Generator

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Assignment group using Classification

</td><td>

The recommendation- **Assignment group \(Classification\)**,uses this resource generator to return field predictions in the Assignment group field of the current incident.

</td></tr><tr><td>

Configuration item using Classification

</td><td>

The recommendation- **Configuration item \(Classification\)**, uses this resource generator to return field predictions in the configuration item field of the current incident.

</td></tr><tr><td>

Incident Fields predictions TI

</td><td>

The recommendation- **Incident Fields value prediction \(TI\)**, uses this resource generator to return field predictions in the Assignment group, configuration item, and service fields of the current incident.

</td></tr><tr><td>

Open PRBs using CI &amp; Service

</td><td>

The recommendation- **Open PRBs \(CI &amp; service\)**, uses this resource generator to return problems with same CI &amp; Service, which can be linked to current incident.

</td></tr><tr><td>

Propose major incident using trend

</td><td>

The recommendation- **Propose Major Incident \(Trend\)**,uses this resource generator to return incidents which can be proposed as a major incident.

</td></tr><tr><td>

Resolved Incidents with same CI &amp; Service

</td><td>

The recommendation- **Similar Resolved incidents \(CI &amp; Service\)**, uses this resource generator to return resolved incidents with same CI &amp; Service, whose resolution can be linked to current incident.

</td></tr><tr><td>

Service using Classification

</td><td>

The recommendation- **Service \(Classification\)**, uses this resource generator to return field predictions in the Service field of the current incident.

</td></tr><tr><td>

Similar KBs using similarity

</td><td>

The recommendation- **Similar KB Articles \(Similarity\)**, uses this resource generator to return KB articles as recommendations which can be attached to the current incident.

</td></tr><tr><td>

Similar major incident using trends

</td><td>

The recommendation- **Similar Major Incident \(Trend\)**, uses this resource generator to return major incidents which can be linked to the current incident.

</td></tr><tr><td>

Similar open incidents using similarity

</td><td>

The recommendation- **Similar Open Incidents \(Similarity\)**, uses this resource generator to return open incidents which can be linked to the current incident.

</td></tr><tr><td>

Similar Open Incidents with same CI &amp; Service

</td><td>

The recommendation- **Similar Resolved incidents \(CI &amp; Service\)**, uses this resource generator to return open incidents with same CI &amp; Service.

</td></tr><tr><td>

Similar PRBs using similarity

</td><td>

The recommendation- **Similar Open PRBs \(Similarity\)**, uses this resource generator to return problems which can be linked to the current incident.

</td></tr><tr><td>

AI Search Resource Generator

</td><td>

The recommendation- **AI Search**, uses this resource generator to return AI-powered search results as recommendations.

</td></tr><tr><td>

Similar incidents using TI similarity

</td><td>

The recommendation- **Similar Incidents \(TI\)**, uses this resource generator to return similar records for incidents.

</td></tr><tr><td>

Similar resolved incidents using similarity

</td><td>

The recommendation- **Similar Resolved Incidents \(Similarity\)**, uses this resource generator to return resolved incidents whose resolution can be copied to the current incident.
