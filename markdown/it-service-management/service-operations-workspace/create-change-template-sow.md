# Create and propose a change template in Service Operations Workspace

Create, customize, and then propose a change template aligned to a change model.

## Before you begin

Role required: change\_manager

You must have a role with the following access with regard to the change model to modify or retire a change template:

-   Access to read
-   Access to modify

## Procedure

1.  Navigate to a list of changes.

    **Note:** The following lists are available:

    -   Open
    -   Closed
    -   All
2.  Create a change request by selecting **New**.

    The Create a change request page is displayed.

3.  View available change models by selecting **Model**.

4.  Select the model for which you want to create a change template.

5.  Select the More actions menu, and then select **Create template for selected model**.

    The Change Template page is displayed in UI16 module. For information on creating a template and providing template fields and policies, see [Create and propose a change template](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-service-management/change-management/create-change-template.md).


## What to do next

Modify information in existing change templates available in the Create a change request page by selecting **Edit template** from the More actions menu. You are redirected to the Change Template page in the UI16 module where you can modify the details as required. For more information, see [Modify a template](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-service-management/change-management/modify-or-retire-template.md).

Manage and retire change templates as needed. For more information, see [Retire a change template](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-service-management/change-management/retire-a-change-template.md).

Review change templates as approvers configured in the change model associated with the template. For more information, see [Review a change template](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-service-management/change-management/review-change-template.md).
