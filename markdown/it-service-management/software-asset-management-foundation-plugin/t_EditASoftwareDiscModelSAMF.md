# Manually normalize a SAM Foundation software discovery model

You can edit a software discovery model to manually normalize discovered software on the Software Discovery Models form so that it can be reconciled.

## Before you begin

Role required: sam\_user

## About this task

If the information automatically added to the software discovery model is incomplete, you can add the missing fields to manually normalize the software discovery model.

## Procedure

1.  Navigate to **All** &gt; **Software Asset** &gt; **Discovery** &gt; **Discovery models** and open a discovery model record.

2.  Fill in the **Publisher**, **Product**, and **Version** fields, as appropriate.

3.  Click **Normalize**.

    The normalization status is set to Manually Normalized.
