# Quick start tests for Vendor Management Workspace in configurable workspace

Validate that Vendor Manager Workspace still works after you make any configuration change such as apply an upgrade or develop an application. Copy and customize these quick start tests to pass when using your instance-specific data.

Vendor Management Workspace quick start tests require activating the Vendor Manager Workspace plugin \(com.snc.vlm.vmw\) and loading demo data.

|Test|Description|Release version|
|----|-----------|---------------|
|VLM: Create a Vendor|Track problems with the Vendor Manager Workspace plugin after upgrade. Create a new vendor.|New York|
