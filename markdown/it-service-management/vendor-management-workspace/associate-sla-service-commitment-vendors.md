# Associate SLAs and Service Commitments with Vendors

Create a vendor SLA and a service commitment and associate that with a vendor.

## Before you begin

Role required: sn\_vlm.vendor\_manager and sla\_manager

## Procedure

1.  Navigate to **Workspaces** &gt; **Vendor Management Workspace**.

2.  Select the List icon.

3.  Create an SLA definition.

    1.  Go to **Service Levels** &gt; **All Vendor SLAs**.

    2.  Select **New**.

    3.  Fill in the fields on the form.

        For information on field descriptions, see [Create an SLA definition](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-service-management/service-level-management/t_CreateAnSLADefinition.md).

    4.  Select **Save**.

4.  Create an SLA breakdown definition.

    1.  In the SLA definition that you've created, select the **SLA Breakdown definitions** tab.

    2.  Select **New**.

    3.  In the **SLA breakdown definition** field, select the definition.

    4.  Select **Save**.

5.  Create a service commitment.

    1.  In the SLA definition that you've created, select the **Service commitments** tab.

    2.  Select **New**.

    3.  Fill in the fields on the form.

    4.  Select **Save**.

    In the List menu, go to **Service Levels** &gt; **My Vendor SLAs** to view the vendor SLA and service commitment you've created.
