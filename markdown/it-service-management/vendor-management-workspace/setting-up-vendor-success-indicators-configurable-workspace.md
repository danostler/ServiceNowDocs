# Setting up Vendor Success Indicators in Vendor Management Workspace

Configure attributes that can give you meaningful insights when comparing top-performing vendors.

**Important:** You must install the Success Indicators \(com.snc.vendor.insights\) plugin separately from the ServiceNow Store. For more information, see [Activate Vendor Management Workspace](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-service-management/vendor-management-workspace/activate-vendor-management-configurable-workspace.md).

You can configure either tables or indicators to define attributes. The base evaluating table for the vendor success indicator is the Company \[core\_company\] table.

Use properties to configure the Vendor Success Indicators. For more information, see [Properties installed with Vendor Success Indicators](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-service-management/vendor-management-workspace/installed-w-vendor-manager-configurable-workspace.md).
