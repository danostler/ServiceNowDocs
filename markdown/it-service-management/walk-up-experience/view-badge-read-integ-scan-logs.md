# View Badge Scanner Integration scan logs

Monitor scanner-use error activity at a Walk-up Experience onsite location by viewing the scan logs. Each time a badge reader device triggers an error, a scan event is logged in the **Badge Scan Logs** module.

## Before you begin

Role required: sn\_badge.badge\_admin

## About this task

The **Badge Scan Logs** module only stores badge reader integration error messages. It does not display error messages for Walk-up Experience specific badge scans. You can fix the error and re-run the application to get the desired result.

Walk-up Experience badge scan errors can be found in the System Log.

## Procedure

1.  Navigate to **All** &gt; **Badge Reader Integration** &gt; **Badge Scan Logs**.

2.  Select an event in the Badge Scan Logs list to view the associated **Message**, **Badge Reader**, and **User Badge**.
