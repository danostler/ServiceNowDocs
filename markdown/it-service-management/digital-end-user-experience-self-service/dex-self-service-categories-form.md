# DEX Self-service categories form

The DEX Self-service categories form presents elaborate data on the form fields and their corresponding descriptions.

|Field|Description|
|-----|-----------|
|Title|Enter the title of the category.|
|Parent|Select the parent category if you want to make it a sub-category.|
|Application|View the application name as DEX Self-service here|
|Order|Enter the order number in which this category appears to the user in the Device health check experience.|
|Active|Select this check box to add this category. The DEX Self-service checks the performance of this category only if the check box is selected.|
