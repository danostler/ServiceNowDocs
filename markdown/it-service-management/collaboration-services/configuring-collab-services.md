# Configuring Collaboration services

Configure Slack as a communication channel to communicate with the users to work towards the resolution of incidents.

-   **[Install Collaboration services](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-service-management/collaboration-services/install-collaboration-services.md)**  
You can install the Collaboration services application \(sn\_tcm\_collab\_hook\) if you have the admin role.
-   **[Setting up Slack for communication](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-service-management/collaboration-services/setting-up-slack.md)**  
The Slack spoke requires creating a custom app on your Slack account to generate OAuth 2.0 tokens for the Slack spoke.
-   **[Configure Slack as a communication channel for a communication task](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-service-management/collaboration-services/configure-slack-for-comm-task.md)**  
Configure Slack as a mode of communication for a communication task to send direct message to the users.
-   **[Create slack channel for an incident](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-service-management/collaboration-services/create-slack-inci.md)**  
Create a private or a public slack channel to collaborate with support teams or send a communication to keep stakeholders informed.
-   **[View or join slack channel from an incident](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-service-management/collaboration-services/view-slack-channel.md)**  
View all slack channels associated with an incident to know if a required slack channel already exists before you create a new one. You can also import messages from a slack channel.
-   **[Set up slack for a user or group](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-service-management/collaboration-services/add-field-slack-icon.md)**  
Set up slack from an incident to communicate and work together in channels with a user or a group.
-   **[Open slack from an incident](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-service-management/collaboration-services/open-slack-from-inci.md)**  
Open slack from an incident and send message to a user or an assignment group to exchange incident information. You do not need to manually search for a user in slack to send a message.
