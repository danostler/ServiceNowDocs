# Migrate active records

Migrate active records to apply the new mappings to your active problem and problem task records.

## Before you begin

Role required: admin

## Procedure

1.  Click **Migrate Active Records**.

    If there are no active records to migrate, the migration job moves to the migrate inactive records stage.

    **Note:** Even if you close the migration window or navigate away from the migration job, the migration of records continues. If you click **Migrate Active Records**, the migration window displays the current progress.

    If any record fails to migrate, it is listed in one of the related lists:

    -   Failed Active Problem Updates
    -   Failed Active Problem Task Updates
    Verify the reason for their failure, resolve it, and migrate the records again.
