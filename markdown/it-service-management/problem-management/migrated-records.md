# Migrated

You have successfully activated the Problem Management Best Practice - Madrid - State Model plugin and updated your problem and problem task records to the new best practice states.

## Before you begin

Role required: admin

## Procedure

1.  View the records after migration in the related links.

    The records are listed in one of the following related lists:

    -   Problems by State
    -   Problem Tasks by State
2.  If any record failed to migrate, verify the reason for the failure, resolve it, and migrate the records again.

    The failed records are listed in one of the following related lists:

    -   Failed Active Problem Updates
    -   Failed Active Problem Task Updates
    -   Failed Inactive Problem Updates
    -   Failed Inactive Problem Task Updates
3.  Click **Finish** to return to guided setup to perform the clean-up tasks.
