# Complete a problem task

Resolve and complete the problem task.

## Before you begin

Role required: admin, problem\_admin, problem\_coordinator, problem\_manager, or problem\_task\_analyst

## Procedure

1.  Select **Complete**.

    \[Omitted image "pm-task-complete.png"\] Alt text: work in progress state

    The problem task enters the **Closed** state with **Close code** as **Complete**.

    **Note:** You can configure the Problem properties to reanalyze a task even after it is closed by selecting **Re-assess**. The state of the task changes from **Closed** to **Assess**. After all the problem tasks are completed or canceled, a notification is sent to the problem coordinator regarding the same.
