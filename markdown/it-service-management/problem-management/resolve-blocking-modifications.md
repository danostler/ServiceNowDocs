# Resolve blocking modifications

Blocking modifications that are detected in your instance are not compatible with the problem state model and you must resolve them before you activate the Problem Management Best Practice — Madrid — State Model plugin.

## Before you begin

Role required: admin

## Procedure

1.  Review and resolve all blocking modifications.

    For more information about modifications that block the problem state model plugin from being activated, see the [KB0819060](https://support.servicenow.com/kb_view.do?sysparm_article=KB0819060) article in the HI Knowledge Base.

2.  Click **Run Checks**.
