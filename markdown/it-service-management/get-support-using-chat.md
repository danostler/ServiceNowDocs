# Get chat support in Employee Center

Resolve user issues by requesting for support using chat.

## Before you begin

Role required: none

## Procedure

1.  Navigate to the Employee Center portal.

2.  Select the Open chat window icon.

    The Now Support chat window is displayed.

3.  Perform any of the following tasks.

**Use chat**

</td><td>

1.  In the **Chat** field, specify the issue.
2.  Use the Now Assist conversational experience to explain your issue. Now Assist generates self-serve troubleshooting steps and solutions for your issue.
