# Initiate a skill review for teams or groups

Start a skill review process for your groups or specific team members so that they can review and update their existing skills or to add new skills. Provide a due date for them to complete the skill review.

## Before you begin

Role required: sn\_wfo\_skillreview.manager

## Procedure

1.  Navigate to **Workspaces** &gt; **Manager Workspace**.

2.  Select the Coaching icon \(\[Omitted image "coaching-icon-uib.png"\] Alt text: Coaching icon\).

3.  Select the **Skills** tab.

4.  Select **Generate skill request**.

    You can save the generated skill request in **Draft** state and work on it at a later time.

5.  Fill in the fields on the **Initiate skill review** form:

<table id="table_igy_clf_2tb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Title

</td><td>

Name for the skill review.

</td></tr><tr><td>

Due Date

</td><td>

Date by which the team member or group must complete the skill review.

</td></tr><tr><td>

Assignment

</td><td>

-   **To my groups:** sends the skill review to the team members is all of your groups.
-   **Specific groups or team members:**sends the skill review to the specified groups or team members.


</td></tr><tr><td>

Skills to review

</td><td>

-   **Team member's existing skills:** request for team members to review their existing skill levels and update them.
-   **Additional skills:** request team members to add new skills.


</td></tr></tbody>
</table>6.  Select **Submit**.

    **Note:** You can cancel a skill review request that you have submitted for review.

    The agent will receive a notification to review the request.
