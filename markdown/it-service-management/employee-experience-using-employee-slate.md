# Employee experience in Employee Slate

You can enable an AI-first employee experience that brings search, requests, tasks, knowledge, and communications together in one intuitive destination. Employees can easily find information, manage requests and tasks, stay informed through communications and news, connect with teams, and get work done faster.

## Overview and key features of the Employee Slate portal

From the Employee Slate portal, an employee can perform the following actions:

-   Search for relevant information
-   Submit requests and incidents
-   Customize the canvas for quick reference of tasks
-   View assigned tasks and requests
-   Access the chat history of active and closed conversations
-   View notifications for tasks

**Note:** This employee experience is supported for the Employee Slate integration with both Now Assist and Moveworks.
