# Create custom record views

Create a UX custom form view to show any record for a given table in Digital Portfolio Management \(DPM\).

## Before you begin

Role required: sn\_dpm.dpm\_admin

## About this task

Create a UX custom form view to determine the view to show for a given table in DPM.

**Note:** Clear the cache to see the change results of this procedure.

## Procedure

1.  Navigate to **All** &gt; **Now Experience Framework** &gt; **Configuration Settings** &gt; **UX Form View Rules**.

2.  Select **New**.

3.  On the form, fill in the fields.
