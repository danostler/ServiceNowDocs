# Run the Oracle software counter for the legacy Oracle Process Pack

Running the Oracle software counter is a requirement to associate a software installation with an Oracle mapping.

## Before you begin

Role required: sam

## Procedure

1.  Navigate to **All** &gt; **Software Asset** &gt; **Reconciliation** &gt; **Software Counters**.

2.  Click the name of the counter.

3.  Click **Count Licenses**.

    You can view different types of software counter results.
