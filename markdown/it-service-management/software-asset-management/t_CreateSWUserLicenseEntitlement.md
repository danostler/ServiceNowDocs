# Create a software user license entitlement for the legacy Software Asset Management plugin

You can entitle a user to use one of your software licenses.

## Before you begin

Role required: asset

## Procedure

1.  Navigate to **All** &gt; **Asset** &gt; **Software** &gt; **User License Entitlements** and create a new record \(see table for field descriptions\).

    |Field|Description|
    |-----|-----------|
    |Display name|Read-only. Name used in record lists.|
    |Assigned to|User of the entitlement token.|
    |Licensed by|License granting this token.|
    |Cached|Read-only. Internal flag set and used by software counters logic.|

2.  Click **Submit**.
