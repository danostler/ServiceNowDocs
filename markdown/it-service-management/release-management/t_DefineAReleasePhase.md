# Define a release phase in Release Management

Define a release phase to represent the different stages of work required to complete your release.

## Before you begin

Role required: release\_v2\_admin

## Procedure

1.  Navigate to **All** &gt; **Release** &gt; **Releases**.

2.  Click the release number to which you want to add a release phase.

3.  In the **Release Phases** related list, click **New**.

4.  On the form, fill in the fields.
