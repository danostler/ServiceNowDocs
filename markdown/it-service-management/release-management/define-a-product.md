# Define a product in Release Management

Define a product in the Release Management application to facilitate managing a release process.

## Before you begin

Role required: release\_v2\_admin

## Procedure

1.  Navigate to **All** &gt; **Release** &gt; **Products**.

2.  select **New**.

3.  On the form, fill in the fields.

    |Field|Input value|
    |-----|-----------|
    |Number|Auto-generated unique identifier for the Product.|
    |Configuration Item|CI that corresponds to the product.|
    |Assignment Group|Group responsible for the product.|
    |Assigned To|user responsible for the product.|
    |Short Description|Brief description of the product.|
    |Description|Detailed description of the product.|
    |Work Notes|A journal field for logging notes about the product.|

4.  Click **Submit**.
