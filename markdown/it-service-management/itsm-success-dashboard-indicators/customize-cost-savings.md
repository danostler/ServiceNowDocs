# Create a cost savings indicator for ITSM Success Dashboard

Create a cost savings indicator in ITSM Success Dashboard to determine and display the time saved for the contributing indicators when agents work on an incident.

## Before you begin

Role required: sn\_sd.success\_dashboard\_admin

## Procedure

1.  Navigate to **All** &gt; **Success Dashboard** &gt; **Getting Started**.

2.  In the Customize Cost Savings card, select **For ITSM**.

3.  On the form, fill in the fields.
