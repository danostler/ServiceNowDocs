# Operational Success Admin Console

The Operational Success dashboard provides a consolidated view of the end-to-end performance of the IT processes that are implemented.

## Prerequisites

User Roles: You can assign different roles based on specific access requirements.

## Configure

Tabs and dashboards: Map a KPI category such as Incident, Problem, Change, Request, and so on as an Operational Success dashboard tab to view the ITSM processes performance in your organization. Use this option to map the Performance Analytics dashboard associated with these KPI categories.

## Advanced options

Review and configure the data visualization available in the base system for Incident, Problem, Change, Request, Interaction, Service catalog, Major Incident, Password Reset, and Walk-up Experience.
