# Maximise the benefits of using the ITSM Success Dashboard indicators

Use the ITSM Success Dashboard indicators to measure the performance of your ITSM implementation using the Key Performance Indicators \(KPIs\) defined.

-   **[Monitoring the KPIs of your ITSM implementation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-service-management/itsm-success-dashboard-indicators/monitor-kpi-for-itsm-imlementation.md)**  
View and monitor the ITSM Success Dashboard indicators to measure the performance of your ITSM implementation using the Key Performance Indicators \(KPIs\) defined by ServiceNow.
-   **[Share the Success Dashboard indicators](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-service-management/itsm-success-dashboard-indicators/share-itsm-success-dashboard.md)**  
Share the Success Dashboard indicators with your respective leadership teams, ITSM or HR to view the dashboard and measure the performance of their respective implementations.
-   **[Benchmarks for Success Dashboard indicators](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-service-management/itsm-success-dashboard-indicators/success-dashboard-benchmark.md)**  
You can view the global benchmark score for the Success Dashboard KPIs and trends along with comparative insight relative to the customers using Success Dashboard.
-   **[Estimated Cost Savings for Success Dashboard indicators](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-service-management/itsm-success-dashboard-indicators/estimated-cost-savings-sd.md)**  
Estimated Cost Savings provides the data to measure the success of the product by calculating the total amount saved by the agents for Virtual Agent, Knowledge, and Service Catalog.
-   **[View metrics for selected period in ITSM Success Dashboard indicators](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-service-management/itsm-success-dashboard-indicators/view-metrics-selected-period.md)**  
View the Key Performance Indicators \(KPIs\) metrics on the ITSM Success Dashboard for a selected period to analyze the performance.
-   **[Create target for the KPI in the Success Dashboard](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-service-management/itsm-success-dashboard-indicators/create-target-kpi-sd.md)**  
Create a target for a KPI to be on par with the benchmark score or the organizational targets.
