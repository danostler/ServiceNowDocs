# Configure Virtual Agent topics for ITSM Success Dashboard using self-service analytics

Configure the Virtual Agent \(VA\) topics for ITSM Success Dashboard indicators using the self-service analytics.

## Before you begin

Role required: admin

To update the live agent script include see, [Update the live agent script include](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-service-management/itsm-success-dashboard-indicators/update-live-agent-script-include.md).

## Procedure

1.  Navigate to **All** &gt; **Success Dashboard** &gt; **Getting Started** &gt; **Success Dashboard** &gt; **Virtual Agent deflection**.

2.  Select **Review and Configure**.

3.  Select **Review** tab.

4.  Select a deflection pattern under Virtual Agent deflection.

5.  Update the value in **Refresh Window** field.

    This is the refresh interval for the report to be executed.

6.  Select the fields under **Incidents considered**.

7.  Select the **Virtual Agent topics considered**.

8.  Select **Save**.
