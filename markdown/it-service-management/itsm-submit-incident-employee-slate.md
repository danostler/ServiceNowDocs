# Submit an incident in Employee Slate

Save time in submitting and resolving incidents.

## Before you begin

Role required: none

## Procedure

1.  Navigate to Employee Slate.

2.  Enter your request in the conversational interface.

    For example, `raise an incident that my vpn is not working`.

    \[Omitted image "ai-native-submit-incident-empslate.png"\] Alt text: Submit an incident in Employee Slate

3.  View the generated answers and select the relevant approach to submit an incident.

    \[Omitted image "ai-native-submit-incident-channel.png"\] Alt text: Select a channel to submit incident

4.  Fill the form and select **Submit**.

    \[Omitted image "ai-native-submit-incident-form-empslate.png"\] Alt text: Submit an incident form in Employee Slate
