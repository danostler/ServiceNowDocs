# Delete a card from the Desktop Assistant home page

Delete a card from the Desktop Assistant home page when you no longer need it.

## Before you begin

Role required: sn\_dex\_desktop.admin

## Procedure

1.  Navigate to **All** &gt; **Desktop Assistant** &gt; **Configuration** &gt; **Home**.

2.  Select the home page name from which you want to remove the card.

3.  In the **Desktop Assistant Section**, select the section name under which the card is positioned.

4.  In the **Desktop Assistant Card** column, hover over the card name and select the check box next to it.

5.  From the **Actions on selected rows** drop-down list, select **Delete**.

6.  In the Confirmation window, select **Delete**.
