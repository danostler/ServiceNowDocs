# Metric rules page

The metric rules page shows information about a specific rule, including its name, CI type, application name, alert severity, and alert status.

<table id="table_oz5_gmw_d1c"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Name of the metric rule.

</td></tr><tr><td>

Type

</td><td>

CI type, such as Saas/web, installed, or device, to which the metric rule is applied.

</td></tr><tr><td>

Application

</td><td>

Application name to which the metric rule is applied.If the CI type is Device, then the application name is marked as N/A.

</td></tr><tr><td>

Alert severities

</td><td>

Level of urgency or importance assigned to the alert.

</td></tr><tr><td>

Status

</td><td>

Current state of the alert rule, indicating whether it's active or inactive.

</td></tr><tr><td>

Action

</td><td>

Alert action for the metric rule that triggers the remedial action for the alert.

</td></tr><tr><td>

Updated

</td><td>

Date and time when the metric rule is last updated.

</td></tr></tbody>
