# Creating a metric rule

You can create a metric rule to establish criteria and thresholds for performance metrics, enabling timely alerts and proactive system management.

To create a metric rule, you must perform the following steps.

1.  [Specify metric rule CI](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-service-management/digital-end-user-experience-dex/specify-metric-rule-applications.md).
2.  [Define alert criteria](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-service-management/digital-end-user-experience-dex/define-alert-metric-criteria.md).
3.  [Add an alert action](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-service-management/digital-end-user-experience-dex/add-alert-action.md).
4.  [Define alert rule name and status](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-service-management/digital-end-user-experience-dex/define-alert-name-state.md).
