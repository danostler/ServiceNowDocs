# Managing DEX Agent policy

Create and manage agent policies for your organization based on the device criteria defined to manage the DEX metrics that are tracked.

You can perform actions on checks and policies to enhance the monitoring of your device. Policies contain Configuration Item \(CI\) that are monitored by Agent Client Collector \(ACC\) and checks that run on those CIs.
