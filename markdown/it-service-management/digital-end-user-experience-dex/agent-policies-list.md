# Agent policies list

The agent policies list shows information about each agent policy including its name, status, cron expressions, and updated time.

<table id="table_skx_qyg_52c"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Name of the policy.

</td></tr><tr><td>

Hierarchy

</td><td>

Policy hierarchical state.When the policy is part of a hierarchy, the values are Parent or Child. Otherwise, the value is None.

</td></tr><tr><td>

Active

</td><td>

Option to indicate that the policy is in use.The values are True or False.

</td></tr><tr><td>

Publish status

</td><td>

Publish status of the policy.The values are Draft, Ready to publish, Queued, and Published.

</td></tr><tr><td>

Interval\(sec\)

</td><td>

The frequency in seconds according to which the policy checks are executed.This field applies only to checks with an empty cron expression value.

</td></tr><tr><td>

Cron expressions

</td><td>

Expressions according to which the policy checks are executed.This field applies only to checks with an empty interval value.

</td></tr><tr><td>

Updated

</td><td>

Last time the policy was updated.

</td></tr></tbody>
