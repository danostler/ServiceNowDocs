# View upcoming shifts with On-Call Scheduling

Use the My work feature in the ITSM Mobile Agent app to view your upcoming shifts and request time off.

## Before you begin

Role required: itil, sn\_incident\_write

## About this task

Use the On-Call Scheduling applet to view your upcoming shifts and request time off. You can also view the members of your available on-call group.

## Procedure

1.  Open the ITSM Mobile Agent app.

2.  On the **On-call Schedule** tab, tap **Upcoming shifts**.

    All upcoming shifts for the current month are displayed. The shifts are grouped based on the type of events. One-day events are displayed at the top, followed by multi-day events and short-duration events that are sorted based on the time that the event was created.

    \[Omitted image "Upcoming-shifts-itsm-mobile.png"\] Alt text: Upcoming shifts.

3.  Tap a shift card to view the following shift details:

    -   Name
    -   Group
    -   Roster
    -   Start time
    -   End time
    -   Duration
    -   Type
4.  Swipe the shift card to the left and tap **Request time-off**.

5.  On the Request time-off form, enter the following details and tap **Submit** to complete the time-off request:

    -   Group
    -   Start date
    -   End date
    -   Proposed coverage
    -   Notes
