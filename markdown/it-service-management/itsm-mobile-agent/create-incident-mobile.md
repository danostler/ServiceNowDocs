# Create an incident from app landing page

Create an incident directly from your agent mobile app landing page with a single tap.

## Before you begin

Role required: itil, sn\_incident\_write, or admin

## Procedure

1.  On your agent mobile app landing page, select the \[Omitted image "create-incident-shortcut.jpg"\] Alt text: create incident shortcutto create an incident directly from the landing screen.

2.  Fill in the fields.
