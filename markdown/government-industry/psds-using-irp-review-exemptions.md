# Review a case for exemptions using Information Request Playbook

Review a case for exemptions.

## About this task

Several exemptions may apply to an information request case. .

## Before you begin

Role required: admin

## Procedure

1.  Select any exemptions that apply to the case, and select **Mark as Complete**.

2.  If no exemptions apply to the case, select **No exemption code\(s\) applicable**, then select **Mark as Complete**.
