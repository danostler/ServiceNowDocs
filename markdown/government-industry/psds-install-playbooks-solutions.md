# Install and configure playbooks and solutions in Public Sector Digital Services

Install one or more applications available for use with Public Sector Digital Services. You can then configure the features available for submitting requests and routing requests to agents and other entities.

The following applications are available with Public Sector Digital Services that enable you to create and use playbooks and other solutions as an end-to-end workflow for handling constituent applications and requests. For more information on installing and configuring each application, see the corresponding pages below.
