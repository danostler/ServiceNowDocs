# Configure the Merit Review Scoring Framework for a Grant Program in the Reviewer Service Portal

Define the scoring framework that a merit reviewer should follow as they review and score grant program proposals in the Reviewer Service portal. You can also select which knowledge bases or frameworks are displayed to a merit reviewer in the Reviewer Service portal.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Service Portal** &gt; **Portals**.

2.  Select the portal for which you want to display the desired merit review framework.

    Select the Reviewer Service Portal. For information on how to create a merit review framework, see [Create a custom Merit Review scoring framework for a Grant Program in the Reviewer Service Portal](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/government-industry/public-sector-digital-services/psds-config-gmp-create-scoring-framework.md).

3.  Select the desired scoring framework.

4.  Add a scoring rubric.

    A rubric is a Knowledge Article which needs to be added to this program so that the evaluators/reviewers know how to judge the proposals.

5.  In the Knowledge Bases related list, select **Edit**.

6.  On the Edit Members form, move the desired knowledge bases from the available items in the **Collection** column to the **Knowledge Bases List** column.

7.  Select **Save**.
