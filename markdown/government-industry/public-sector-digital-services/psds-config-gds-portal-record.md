# Configure the GOV.UK Design System Service Portal record

Use the portal record on the platform UI to change components of your GDS Service Portal.

## Before you begin

Role required: admin, sp\_admin

**Note:** Ensure the scope is set to the **GOV.UK Developer Toolkit** application.

## About this task

On the portal record page, you can change aspects such as portal title, URL suffix, homepage, login and registration pages, error pages, theme, menu headers, and catalog homepages. You can also define search sources and knowledge bases for the portal using the portal record.

\[Omitted image "psds\_uk\_gds\_portal\_record\_edit.png"\] Alt text: GDS Portal record edit view

## Procedure

1.  Navigate to **All** &gt; **Service Portal** &gt; **Portals** and select **UK Government Portal** to open the portal record.

    A page opens in your instance where you can configure the GDS Service Portal.

2.  Select the field you want to edit, and select the desired option using the catalog search menu for Page ID.

    Selecting the search icon \(\[Omitted image "search-icon-2.png"\]\) will open a page in your instance that displays a list of Page IDs.

3.  Select **Try it** to test out the edits your portal before publishing, or **Update** to save and publish your changes immediately.
