# Create an investigative case in Investigative Case Management

As an investigator \(case agent\), you can get started on an investigative case directly from the Investigative Case Management workspace, filling in as much supporting information as you have.

## Before you begin

Role required: icm.investigator, admin

## Procedure

1.  Navigate to the CSM Configurable Workspace and select **My active cases** &gt; **New**.

    Alternatively, you can create a new case using the CSM Configurable Workspace Lists menu by selecting **Lists** &gt; **Investigative Cases** &gt; **All** &gt; **New**.

2.  Select the case classification code from the dropdown menu to select the investigation type.

    By default, the workspace provides several generic classification codes that correspond to different investigative case types, but these may not correspond to your organization's internal codes. An admin can create and custom classification codes that more closely correspond to your investigative organization. For more information on this configuration, see [Modify case classification codes](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/government-industry/public-sector-digital-services/psds-config-icm-modify-classif-codes.md).

3.  Enter the case title and a summary of the case.

    The case summary is different from the case narrative.

4.  Enter the source of the case information.

    For example, if the case is being filed because of a constituent or witness complaint, select **Complaint**.

5.  Provide a brief description of the source.

6.  Fill in any routing information for the case, such as the division responsible for the case, and the Supervisory reviewer.

    The assigned office and originating office fields are auto-populated based off the logged in user. For more information on the case details form fields, see [Investigative Case Management case details form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/government-industry/public-sector-digital-services/psds-icm-ref-case-form.md).

7.  Select **Save** once you have filled in the details of the case.


## Result

A case has now been created and can be worked on through the Investigative Case Management workspace. You can add evidence and entities, assign team members, or link it to another case.
