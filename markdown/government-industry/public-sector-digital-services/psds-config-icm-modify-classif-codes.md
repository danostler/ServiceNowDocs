# Modify case classification codes

Create and modify case classification codes and titles to organize and categorize the investigation types your agency handles.

## About this task

When you select a classification code for a case, you can use the pre-defined case type classifications, or create your own. By default, the workspace provides the following generic classification titles and codes that correspond to different investigative case types:

-   13A, Aggravated Assault
-   23H, All Other Larceny
-   290, Destruction of Property
-   90D, Driving Under Influence
-   35A, Drug/Narcotic Violations
-   280, Embezzlement
-   26B, Fraud - Credit Card/ATM
-   26A, Fraud - False Pretenses
-   26C, Fraud - Impersonation
-   26F, Identity Theft
-   23D, Larceny - From Building
-   23A, Larceny - Pocket-picking
-   23B, Larceny - Purse-snatching
-   23C, Larceny - Shoplifting
-   90F, Weapon Law Violations

You can modify these existing classification codes into custom ones that more closely correspond to that of your investigative organization, or you can create new ones.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to the CSM Configurable Workspace and select **Investigative Classifications** &gt; **All**.

2.  Select the classification title you want to edit, or select **New**.

3.  On the form, fill in or update the fields with the desired classification code and title.

    \[Omitted image "psds-modify-classif-codes.png"\] Alt text: edit icm classification codes.

4.  Select **Save**.


## Result

The updated classification code or title should now appear in the dropdown menu when an investigator creates a new case.
