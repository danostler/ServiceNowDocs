# Add external reviewer groups in Grants Management

Add the people who will review and score the proposals for this program.

## Before you begin

Role required: sn\_gsm\_grnt\_mgmt.program\_manager, sn\_gsm\_grnt\_mgmt.grant\_director

**Note:** Reviewer teams can be updated throughout the duration of the program as needed.

## About this task

The Review Team activity allows the assignment of one or multiple groups who will review the grant applications to a Grant Program. At this stage, the Grant Program Manager selects the relevant groups, and each group can then be assigned to a single proposal during the intake/screening phase.

**Note:** Whenever a Review Group member is added, the users in that group are assigned to evaluate and score merit review tasks during the evaluation stage of the Proposal Playbook. If no specific Review Group is selected, all available Review Groups will appear in the **Conduct Merit Review** activity. Otherwise, only the selected Review Group will be shown.

## Procedure

1.  In the Review team activity, select **Add group**.

2.  Select the group.

    A merit review group must already have been configured for them to show up in this step. For more information on configuring external review groups, see .

3.  Select **Save**, then select **Mark Complete**.
