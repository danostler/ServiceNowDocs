# ICM Entity Management Location form

The Location form allows you to create a new location entity record within an investigative case, and capture as many identifiable details about the location as possible. Not all fields are mandatory, but a minimum amount of information is required for the record to be searchable within ICM.

| | |
|---|---|
|Location|Search for an existing record in the master index to auto-fill its information. If the desired location record has not yet been created, skip this and enter the details manually.|
|Name or short description|Name or short description of the location.|
|Type|Type of location. For a list of provided location types you can select when adding a location entity to an investigative case, see [Location types form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/government-industry/public-sector-digital-services/psds-icm-ref-location-types.md).|
|Additional information|Additional information about the location.|
|Location Format|Whether the location information will be entered as an address or as GPS coordinates.|
|Source from|Related case, Investigative task, or Other|
|Information Completeness|Not all fields are mandatory, but a minimum amount of information is required for the record to be searchable within ICM. The information completeness field indicates whether or not that minimum requirement is met. Displays as Partial or Complete.|
