# Evaluate an information request case for similar requests

Evaluate similar requests

## About this task

Reviews all cases in the org for similar cases. Browse through the 3 similar closed requests at the top of the page. Documents from similar closed requests are populated based on the case’s short description and description.

## Before you begin

Role required: admin

## Procedure

1.  Do one of the following actions depending on whether you have similar or linked completed requests.

    |Options|Steps|
    |-------|-----|
    |**If there are no similar or linked completed requests**|Select **Mark as Complete**.|
    |**If there are similar or linked completed requests**|Review them to see if any are relevant to the current request.|

2.  Once you have reviewed, select Mark complete.

    **Note:** It may take some time before the next activity loads. If it hasn’t loaded within 30 seconds, reload the web page.
