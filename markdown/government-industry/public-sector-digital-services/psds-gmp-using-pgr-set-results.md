# Set up a results letter using a template in Grants Management

Select which letter templates you want to use to notify applicants of their results for their proposals submitted for this program. These templates will be sent out once the final decisions have been made. The templates include an acceptance letter, a rejection letter, an ineligible letter, and a merit review summary.

## Before you begin

Role required: sn\_gsm\_grnt\_mgmt.program\_manager, sn\_gsm\_grnt\_mgmt.grant\_director

**Note:** Requires it to already be created. For info on how to create see [Create a Grants Program results letter template](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/government-industry/public-sector-digital-services/psds-config-gmp-configure-results-template.md)

## Procedure

1.  Navigate to **All** &gt; **CSM Configurable Workspace** &gt; **Lists**.

2.  Select **Grant Programs** &gt; **My grant programs** and select a grant program.

3.  Select which results letter templates should be used to notify applicants of their results for this program.

    Choose a letter template for Acceptance, Rejection, and Ineligible letters, and the Merit review summary. By default, the Grants Management Workspace provides generic results letter templates that can be modified to fit your organization's needs.

    You can choose the use the same template for multiple letter types.

4.  Select **Mark Complete**.
