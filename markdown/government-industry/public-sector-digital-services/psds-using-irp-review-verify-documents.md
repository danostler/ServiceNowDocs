# Verify uploaded documents in Information Request Playbook

Review documents.

## Before you begin

Role required: admin

## About this task

Several types of documents can be requested from the requester to be added in order to proceed with an information request case.

## Procedure

1.  In the contextual side panel, select the attachments icon \(paperclip icon\) to view all attached documents.

2.  Select an attachment to view the document.

3.  Review the files attached to the case, and confirm that the requester has included the required documents to validate their identity by selecting the boxes associated with the document type.

    For example, if the requester uploaded their ID or passport to verify their identity, select the checkbox next to **Personal identification, such as passport or driver’s license**.

4.  Document your verification work in the work notes.

5.  Once you have reviewed and verified all files attached to the case, select **Mark as Complete**.
