# Configure relationship types for an evidence record in Investigative Case Management

As an admin, you can add or remove the relationship types used to classify the relationship between a piece of evidence and another entity related to the case.

## About this task

For more information about relating entities to a case, see [Associate and link evidence to entities](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/government-industry/public-sector-digital-services/psds-using-icm-associate-evidence-entities.md).

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Task Relationships** &gt; **Relationship Types**.

2.  Select **New**.

3.  On the form, fill in the fields.

4.  Select **Submit**.


## Result

The relationship type definition is now created, and will display in the relationship type dropdown menu when an investigator associates an evidence record to an entity within a case.

## What to do next

An investigator can use this relationship type to define the relationship between an evidence record and an entity. For more information on how to link evidence to an entity, see [Associate and link evidence to entities](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/government-industry/public-sector-digital-services/psds-using-icm-associate-evidence-entities.md).
