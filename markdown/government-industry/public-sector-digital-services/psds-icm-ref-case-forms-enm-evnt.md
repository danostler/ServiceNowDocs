# ICM Entity Management Events form

The Event form allows you to create a new event entity record within an investigative case, and capture as many identifiable details about the event as possible. Not all fields are mandatory, but a minimum amount of information is required for the record to be searchable within ICM.

| | |
|---|---|
|Event type|Incident or Investigation|
|Name or short description|Name or short description of the event|
|Classification code|Displays only if **Incident** is selected. Pre-configured internal classification code for the type of incident.|
|Classification code text|Description of the classification code.|
|Activity type|Displays only if **Investigation** is selected. Type of Event. Can be an Interview, Canvass \(Neighborhood or Business\), Field Contact / Stop, Evidence Collection, Crime Scene Documentation, Surveillance. Undercover Operation, Arrest, Search, or Other.|
|Date|The date the event occurred.|
|Time|The time the event occurred.|
|Details|Details about the event.|
|Image|Optional image attachment.|
