# Browser support

Browser support varies for each version of the user interface \(UI\). Some features have additional browser requirements, which are noted in the appropriate documentation.

**Note:** Starting with the Washington release, Internet Explorer and [Internet Explorer mode in Edge](https://support.microsoft.com/en-us/microsoft-edge/internet-explorer-mode-in-microsoft-edge-6604162f-e38a-48b2-acd2-682dbac6f0de#:~:text=To%20turn%20on%20Internet%20Explorer,Restart%20Microsoft%20Edge.) are no longer supported.
