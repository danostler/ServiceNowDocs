# Release notes summaries for Australia features

Consolidated release notes information for new and updated Australia features.

## Learn about specific themes for multiple Australia products

In the Australia family, ServiceNow® features and products were introduced or updated. Every new or updated feature and product has its own individual release notes topic, which includes information such as upgrade tasks, new features, changed features, browser requirements, and other specifications. These product-specific release notes are listed in [Features and changes by product](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/release-notes/new-features-changes.md).

To help users look over different classes of information at a glance, each release notes section has its own summary topic. For example, the aggregates all browser requirements for features that were introduced or updated in Australia.
