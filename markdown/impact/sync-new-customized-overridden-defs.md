# Sync new, customized, overridden, definitions

You can sync overridden or custom definitions that your organization created or updated.

## Before you begin

Role required: Scan Engine Admin \(sn\_se.scan\_engine\_admin\).

## Procedure

1.  Navigate to **ALL** &gt; **Impact** &gt; **Definitions**.

2.  Select the definition you want to sync to open its details page.

3.  Select one of the following:

    -   **Sync Definition**: Syncs definitions to other instances through the Production instance. You can leave a note about the synchronized change.
    -   **Delete and Sync**: Deletes the definition and syncs the deletion to other instances through the Production instance. You can leave a note about the synchronized deletion.
    **Note:** You can also bulk sync definitions from the Definitions table. Select the box for each definition that you want to synchronize, and then select **Sync Definition** or **Delete and Sync**.
