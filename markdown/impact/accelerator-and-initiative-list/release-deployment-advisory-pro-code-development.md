# Release and Deployment Advisory for Pro-code Development

The Release and Deployment Advisory for Pro-code Development Accelerator provides guidance to assist pro-code developers in tools and techniques to release and deploy ServiceNow application configurations and customizations.

## Accelerator overview

This Accelerator provides you with a review and guidance on your release and deployment processes and methods for a selected set of applications including either ServiceNow sourced applications or custom-developed applications. Your Impact Squad provides guidance based on your responses to an intake questionnaire and a collaborative Discovery working session that reviews your current state for two selected applications.

**Note:** This Accelerator is available in Advanced and Total packages.

## What you get

**Important:** Your Squad will request that you complete a provided intake questionnaire in preparation for the Introduction Session as part of the Accelerator initiation.

The Accelerator activities are led by your Platform Architect \(PA\).

-   **Introductory customer session \(up to 180 minutes\)**
    -   Review the initiative approach and your objectives
    -   Establish clear expectations regarding the process and your participation, along with the level of detail for the deliverables
    -   Review the basics of the release and deployment methods including update sets and ServiceNow Studio used with Application Repository
    -   Confirm the selected applications
-   **Customer current state working session \(up to 180 minutes\)**

    Discuss the current state of your release and deployment methods for a select set of applications

-   **Customer recommendation session \(up to 120 minutes\)**
    -   Review leading practices related to release and deployment
    -   Review and discuss your tailored recommendations to enable you to more effectively release and deploy your applications
    -   Discuss the possible next steps to implement the recommendations and techniques to measure the results of your improvements
-   **Follow-up customer session \(optional upon customer request - up to 60 minutes\)**

    Opportunity for Q &amp; A related to the release and deployment guidance

-   **Release and Deployment Advisory deliverables**
    -   Workshop slides
    -   Guidance to improve your release and deployment processes
    -   Recommended next steps

Customers must provide the following supporting information:

-   Intake Questionnaire \(to be provided by the ServiceNow team\) completed by the customer at least one week prior to the Customer Working Session
-   Current release and deployment process for selected applications
-   Impact materials, such as:

    -   Product Adoption Roadmap
    -   Organization Chart related to selected applications

## Exceptions

This Accelerator does not include:

-   Defining release process specifications
-   Detailed release process analysisDetailed development guidance on ServiceNow Studio for application development
-   Source Control release review – request
-   Impact Expert Services advisory for further assistance
-   App Engine Studio and App Engine Management Center \(low-code development\)
-   Dev Ops – CI/CD, release automation
-   Citizen Development
-   Detailed review of technical governance processes for release and deployment
