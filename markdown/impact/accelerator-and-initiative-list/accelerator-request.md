# Accelerator request

Depending on the Impact package that you are subscribed to, you can request and consume a designated number of concurrent Impact Accelerators.

The process of requesting an Accelerator from IDI and Impact Store App is similar. When you request an Accelerator, you need to complete a request form and follow the on-screen guidance to complete the setup.

You can place a request for any of the available Accelerators and initiatives if you have upgraded to the latest versions of the application.

Some Accelerators require additional terms and conditions and are offered pursuant to the applicable [Impact Accelerator Description](https://www.servicenow.com/legal/servicenow-impact.html). For Accelerators that require consent, the terms and conditions must be accepted from the **Manage Permissions** menu. You can manage permissions using [Manage Initiative and Accelerator permissions](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/impact/accelerator-and-initiative-list/grant-ide-permissions.md)
