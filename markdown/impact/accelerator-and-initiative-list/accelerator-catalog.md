# Accelerator catalog

This is a complete Impact Accelerator catalog that combines the Architecture, Strategy, Technical, and Optimization sub-catalogs.

## Consumption model

According to your Impact package, Guided or Total, you may consume a fixed number of Accelerators simultaneously.

|Package|Total concurrent Accelerators|Concurrent consumption per sub-catalog| |
|Strategy|Architecture|Technical|Optimization|
|:-----:|:---------------------------:|:------------------------------------:|:---:|
|:------:|:----------:|:-------:|:----------:|
|Total|Seven|Two|Two|Two|One|
|Guided|One at a time across one catalog| |
|Strategic Value Add-On|Two per add-on|Two| | | |
|Platform Governance Add-On|Two per add-on| |Two| | |
|Platform Optimization Add-On|One| | | |One|

**Note:**

-   Refer to Release Notes and Upgrades to identify the most recently added or updated Accelerators.
-   The Impact team shall be scheduled based on resource availability.

## Accelerators available in each package

Accelerator outputs and formats have common standardized features, but may differ from package to package. Not all Accelerators are available for each package. The Accelerators and available feature levels are denoted by these symbols:
