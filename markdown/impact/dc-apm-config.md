# Configure Data Collection for APM

Configure Data Collection for APM.

## Before you begin

You must update Manual Indicators periodically \(monthly, for example\) using the scoresheet function.

Role required: admin, pa\_power\_user, pa\_admin, or pa\_data\_collector

## Procedure

1.  Navigate to **Performance/Platform Analytics &gt; Scoresheet**, and then select **Impact VM - Legacy APM systems run-rate**.

2.  Enter the data point in the relevant month cell.
