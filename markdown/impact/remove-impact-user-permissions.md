# Remove Impact user permissions

Remove permissions that you've assigned to an Impact user if that user should no longer have access to Impact.

## Before you begin

Role required: Impact admin

## Procedure

1.  Navigate to **Impact** &gt; **User Profile Menu** &gt; **Manage Users**.

2.  In the ServiceNow sites section, access the Impact Digital Experience tile.

3.  Select **View all users**.

4.  Enter the users name into the **Search users** field.

5.  Select **Actions** &gt; **Edit role\(s\)**.

6.  Edit the user role to **None**.

    **Note:** Refer to [Roles installed with Impact](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/impact/impact-roles.md) for details on which role to assign to the user for the appropriate access level in Impact.

7.  Select **Confirm**.

    A confirmation message displays then you are returned to the users page.
