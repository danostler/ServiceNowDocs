# Create a Value builder task

Manually create a Value builder task for publisher packs not being fully utilized in your instance.

## Before you begin

Role required: sam\_admin

## About this task

Value builder tasks are automatically created for all publisher packs indicated as incomplete in the Publisher pack utilization widget. However you can also manually create Value builder tasks.

## Procedure

1.  Navigate to **Software Asset Workspace** &gt; **Software asset success portal** &gt; **Value builder**.

2.  Click **New task**.

    The Create New SAM Value Builder Task page appears.

3.  On the form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |**Short description**|Short description of the task.|
    |**Description**|Detailed description of the task.|
    |**Instructions**|Any specific instructions related to the task.|
    |**Number**|Number assigned to the task.|
    |**Category**|The category that this task belongs to.|
    |**Success Goal**|The success goal that you want this task to align to.|
    |**State**|The current state of the task. By default, the state is **New**.|
    |**Assigned to**|Name of the user the task is assigned to.|
    |**Assignment Group**|The assignment group that the assigned to user belongs to.|
    |**Work notes**|Any additional notes pertaining to the task.|

4.  Click **Save**.

    The Value builder task is created and the state defaults to **New**.

5.  Click **Start**.

    The state of the task changes to **Work In Progress**.
