# View SSO users in workspace

View details of all users that have direct access to a SSO application in the Software Asset Workspace, but not through membership in a group.

## Before you begin

Role required: sam\_admin or sam\_user

## Procedure

1.  Navigate to **All** &gt; **Software asset** &gt; **Software Asset Workspace** &gt; **License operations**.

2.  Select **SSO users**.

3.  Select a user record to view additional details related to the user.
