# Subscription downgrade configuration condition

Determine various downgrade configuration conditions for SaaS applications and view the downgrade order based on the conditions.

|Field|Description|
|-----|-----------|
|Downgrade configuration|Name of the downgrade configuration that includes the source and destination subscription.|
|Condition|Defined downgrade configuration condition for the SaaS application.|
|Condition value|Value assigned to the downgrade configuration condition.|
|Products|Products for which the downgrade configuration condition is defined.|
|Active|Shows whether the downgrade configuration condition is active.|
