# View engineering license servers in workspace

View the list of license management servers that OpenLM or Open iT connect with to get data into your ServiceNow instance in the Software Asset Workspace.

## Before you begin

Information such as the name and type of the server, its current status, whether it's currently active, and the last connection time is displayed.

**Note:** Engineering license servers only appear if the Software Asset Management Professional for Engineering Applications \(com.sn\_samp\_eng\_app\) [plugin](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-asset-management/software-asset-management/t_RequSoftwareAssetMgmt.md) is activated.

Role required: sam\_admin or sam\_user

## Procedure

1.  Navigate to **All** &gt; **Software asset** &gt; **Software Asset Workspace** &gt; **License operations** &gt; **Engineering license server**.

2.  Select a license server record to view its details.
