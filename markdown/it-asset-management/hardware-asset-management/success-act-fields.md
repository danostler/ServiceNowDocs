# HAM Success Activity

Fields on the HAM Success Activity form help you create success activities to track the success of your created goals.

<table id="table_sps_zjg_fxb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Number

</td><td>

A unique identifier of the success activity.

</td></tr><tr><td>

Success Goal

</td><td>

The success goal that the activity is being associated to.

</td></tr><tr><td>

State

</td><td>

The current state of the success activity.Choose from the following options:

-   Open
-   Pending
-   Work in progress
-   Closed Complete
-   Closed Incomplete
-   Closed Skipped

</td></tr><tr><td>

Assignment group

</td><td>

The group this activity is assigned to.

</td></tr><tr><td>

Assigned to

</td><td>

A particular person in the assigned group.

</td></tr><tr><td>

Short description

</td><td>

A short description of the success activity.

</td></tr><tr><td>

Description

</td><td>

A detailed description of the success activity.

</td></tr><tr><td>

Work notes

</td><td>

Notes relating to the success activity.
