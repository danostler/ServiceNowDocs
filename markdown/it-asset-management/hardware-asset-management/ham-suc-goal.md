# HAM Success Goal Details

Fields on the HAM Success Goal Details form help you create success goals for tracking the success of the Hardware Asset Management application in your instance.

<table id="id_ybn_xrb_gxb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Number

</td><td>

A unique identifier of the success goal.

</td></tr><tr><td>

Status

</td><td>

Status of the success goal. By default, the success goal is in draft status.

</td></tr><tr><td>

Title

</td><td>

The title for the success goal.

</td></tr><tr><td>

Category

</td><td>

The category the success goal belongs to. Choose from the following options:

-   Automation
-   Inventory Improvement
-   Loss Reduction
-   Normalization
-   Process Improvement
-   Vendor/Contract Negotiation

</td></tr><tr><td>

Vendor

</td><td>

Vendor or publisher associated to the success goal.

</td></tr><tr><td>

Goal Type

</td><td>

Defines the metrics of your goal.-   Savings
-   Count

</td></tr><tr><td>

Goal Description

</td><td>

A detailed explanation of the success goal.

</td></tr><tr><td>

Group

</td><td>

The group the goal is to be assigned to.

</td></tr><tr><td>

Owner

</td><td>

The user from the assigned group the goal is assigned to.

</td></tr><tr><td>

Projected Start Date

</td><td>

The projected start date for the success goal.

</td></tr><tr><td>

Projected End Date

</td><td>

The projected end date for the success goal.

</td></tr><tr><td>

Actual Start Date

</td><td>

The actual start date of the success goal.

</td></tr><tr><td>

Actual End Date

</td><td>

The actual end date of the success goal.

</td></tr><tr><td>

Projected Savings

</td><td>

Anticipated savings from the success goal.This field appears only when **Goal Type** is selected as **Savings**.

</td></tr><tr><td>

Actual Savings

</td><td>

The actual savings from the success goal.This field appears only when **Goal Type** is selected as **Savings**.

</td></tr><tr><td>

Projected Count

</td><td>

Anticipated count from the success goal.This field appears only when **Goal Type** is set to **Count**.

</td></tr><tr><td>

Actual Count

</td><td>

The actual count from the success goal.This field appears only when **Goal Type** is is set to **Count**.

</td></tr><tr><td>

Achievement Description

</td><td>

A description of how the success goal was achieved.
