# View hardware asset requests submitted through the Zero Touch request flow

View the list of hardware asset requests of your employees submitted through the Zero Touch request flow in the Asset operations view.

## Before you begin

Role required: admin or asset

## Procedure

1.  Navigate to **Workspaces** &gt; **Hardware Asset Workspace** &gt; **Asset operations**.

2.  From the **Zero touch** list, select **Asset requests**.

    The list of asset requests is displayed. You can get the request details such as requested item, quantity, and stage.

3.  Select an asset request to view the details.
