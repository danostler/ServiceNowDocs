# Associate your catalog items with the Zero Touch request flow

Set the flow of your catalog item to Zero Touch request flow for fulfilling all the requests for that catalog item through an external provider.

## Before you begin

Role required: catalog\_admin

## Procedure

1.  Navigate to **All** &gt; **Service Catalog** &gt; **Catalog Definitions** &gt; **Maintain Items**.

2.  Select the catalog item.

3.  On the Catalog form, select **Zero touch request** in the **Flow** field.

4.  Select **Update**.
