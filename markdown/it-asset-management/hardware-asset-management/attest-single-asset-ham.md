# Create an attestation for a hardware asset

Validate a particular serialized hardware asset by creating an attestation for that asset.

## Before you begin

Role required: asset or inventory\_admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Hardware Asset Workspace** &gt; **Asset Estate**.

2.  Select the **Hardware assets** tab.

3.  Select the asset that you want to attest.

4.  Select the More Options icon \(\[Omitted image "more-actions-icon.png"\] Alt text: More options icon\) on the asset form.

5.  Select **Create attestation**.

    An attestation is created for the asset.

6.  View the asset attestation record details.

    1.  Navigate to the **Asset operations** view.

    2.  From the Attestation list, select **Attestations**.

    3.  Select the attestation record.
