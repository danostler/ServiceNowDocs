# Associate a catalog item with a remote record producer

Associate a service catalog item with your provider's remote record producer to send the order details to your provider for fulfillment.

## Before you begin

Role required: catalog\_admin

## Procedure

1.  Navigate to **All** &gt; **Asset** &gt; **HAM Zero Touch** &gt; **Catalog Item mappings**.

2.  On the form, fill in the fields.

<table id="table_lpj_2k1_kbc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Catalog item

</td><td>

Service Catalog item that you want to associate with the remote record producer.

</td></tr><tr><td>

Remote record producer

</td><td>

Remote record producer of the external provider.

</td></tr><tr><td>

Multiple quantity supported

</td><td>

Option to indicate that your provider's remote record producer supports multiple quantities for the catalog item.

</td></tr><tr><td>

Quantity variable

</td><td>

Variable to capture quantity from the list of variables defined for the catalog item.This field is editable only when the **Multiple quantity supported** check box is selected.

</td></tr><tr><td>

Active

</td><td>

Option to indicate that the catalog item mapping is available for use.

</td></tr></tbody>
</table>3.  Select **Submit**.
