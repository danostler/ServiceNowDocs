# Create a success goal category for hardware assets

Create a success goal category to associate it with the Hardware Asset Management success goal.

## Before you begin

Role required: ham\_admin

## Procedure

1.  Navigate to **Hardware Asset Workspace** &gt; **Asset operations** &gt; **Success goals** &gt; **Categories**.

2.  Select **New**.

3.  On the Success Goal Category form, fill in the category that you want to add and select the **Hardware asset category** check box.

4.  Select **Save**.


## Result

The category that you add gets listed as an option in the **Category** field while creating a success goal for Hardware Asset Management. For more information, see [Create a success goal for Hardware Asset Management](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-asset-management/hardware-asset-management/create-suc-goal.md).
