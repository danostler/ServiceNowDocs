# Request repair of a hardware asset using the Repair option

Request repair of a defective hardware asset in your stockroom directly through an option on the asset form.

## Before you begin

Role required: asset

## Procedure

1.  Navigate to **Workspaces** &gt; **Hardware Asset Workspace** &gt; **Asset Estate**.

2.  Select the **All assets** tab.

3.  Select an asset with State as **In stock** and Substate as **Defective** or **Pending repair**.

4.  Select the More Options icon \[Omitted image "more-actions-icon.png"\] Alt text: on the asset form.

5.  Select **Repair**.


## Result

A repair order with a repair order line in the **Troubleshoot** stage is created.
