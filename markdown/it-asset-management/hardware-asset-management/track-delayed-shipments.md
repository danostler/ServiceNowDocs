# Continue to track delayed shipments

Continue to track a stale hardware asset shipment that is delayed if the reason for the delay is known.

## Before you begin

Role required: inventory\_user \(read\),inventory\_admin, or asset\_integration\_admin

## Procedure

1.  Navigate to **All** &gt; **Hardware Asset Workspace** &gt; **Asset operations**.

2.  From the **Shipment** list, select **Shipments**.

3.  Select a shipment record that you want to track.

4.  On the Shipment details form, select the **Ignore stale check** check box.

5.  Select **Save**.


## Result

-   The **Stale shipment** check box on the Shipment details form is automatically cleared.
-   The tracking number of this shipment is sent to the carrier for tracking and this shipment record won’t be marked as stale again.
