# Create a checklist template for your Enterprise Asset Management work orders

Create a checklist template for your work orders that gets added as a work order template.

## Before you begin

Role required: sn\_eam.enterprise\_asset\_manager

## About this task

You can either copy from an existing checklist template and edit and add more checklist items or you can create a template.

## Procedure

1.  Navigate to **Enterprise Asset Workspace** &gt; **Work Management** &gt; **Checklist templates**.

2.  Select **New**.

3.  In the **Create checklist template** dialog box, fill in the fields.

## Result

The checklist template is created and appears in the Checklist templates list. After a checklist template is created, it can't be edited and can be used by other applications too.
