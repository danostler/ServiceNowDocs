# Manage Drop off tasks for enterprise assets using the Mobile Agent application

As an enterprise asset technician, manage the delivery of enterprise assets that you have in your personal stockroom to any other warehouse stockroom by creating Drop off tasks in the Mobile Agent application. The asset manager of the destination stockroom receives the assets that you dropped off, verifies the asset, and completes the Receive and Drop off tasks in the Mobile Agent application.

You can also manage Drop off tasks in the Enterprise Asset Workspace. For details, see [Delivering enterprise assets in your personal stockroom through Drop off tasks](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-asset-management/enterprise-asset-management/drop-off-receive-assets-eam.md).
