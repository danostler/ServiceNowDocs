# Create additional child assets

Create additional child assets for the model even if the assets are not defined in the model component.

## Before you begin

Role required: sn\_eam.enterprise\_admin

## Procedure

1.  Navigate to **Enterprise Asset Workspace** and select either **Enterprise asset estate** or **Enterprise model management** to access the **Child Assets** related list tab.

2.  Select **Add** to select assets from a stockroom.

    **Add** is visible only if the asset is in one of the following states:

    -   In Build
    -   In Use
    -   In Maintenance
    -   In Stock Defective
    -   In Stock Pending Repair
3.  Select a stockroom and assets in the Select child asset to add dialog box.

4.  Select **Add**.
