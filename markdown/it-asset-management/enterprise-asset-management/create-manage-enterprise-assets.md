# Create and manage enterprise assets

Create and manage enterprise assets throughout their life cycle.

-   **[Create a multi-component asset](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/enterprise-asset-management/create-multicomponent-asset.md)**  
Track and manage your assets by creating a multi-component asset from a multi-component model.
-   **[Create enterprise assets](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/enterprise-asset-management/create-asset-eam.md)**  
Create assets for specific model categories in the Enterprise Asset Management application.
-   **[Create linear assets](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/enterprise-asset-management/create-linear-asset.md)**  
Create linear assets to diversify your Enterprise Asset Management portfolio.
-   **[Create segments for linear assets](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/enterprise-asset-management/define-linear-segments.md)**  
Create segments for linear assets to assign work orders to the segments. You can also create maintenance plans for the segments.
-   **[Associate a discrete asset to a linear asset](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/enterprise-asset-management/associate-discreet-asset.md)**  
Associate a discrete asset to a linear asset in order to query and find these assets from the linear asset at a later point of time.
-   **[Find linear asset relationships](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/enterprise-asset-management/find-linear-asset-relships.md)**  
You can find relationships between linear assets such as overlapping, continuing, or intersecting assets.
-   **[Create pallet assets in the Enterprise Asset Workspace](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/enterprise-asset-management/create-pallet-asset-eam.md)**  
Create pallet assets to track and manage the pallets that can transport and store your enterprise, hardware, base, bundle, and consumable assets.
-   **[Update enterprise assets that have unknown enterprise models](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/enterprise-asset-management/updated-enterprise-assets-unknown-enterprise-model.md)**  
If an enterprise asset is associated with an unknown enterprise model, update it with a known enterprise model so that you can accurately categorize and track it.
-   **[Swap assets for parent multi-component asset](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/enterprise-asset-management/swap-assets-eam.md)**  
Swap child assets that are part of a multi-component asset. You can swap out only one child asset at a time.
-   **[Select assets for user-assembled asset](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/enterprise-asset-management/assemble-assets-eam.md)**  
Create a user-assembled asset by selecting assets from the parent asset's stockroom.
-   **[Release assets from the parent asset](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/enterprise-asset-management/release-assets-eam.md)**  
Release a child asset that is no longer required to be associated with its parent asset.
-   **[Create additional child assets](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/enterprise-asset-management/create-addon-asset.md)**  
Create additional child assets for the model even if the assets are not defined in the model component.
-   **[Create a TCO report in Enterprise Asset Workspace](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/enterprise-asset-management/create-eam-tco-report.md)**  
Create a TCO report to compare the actual or projected TCO of assets or to compare the actual or projected asset TCO with the benchmark cost of the enterprise model.
-   **[Create a TCO report source in Enterprise Asset Workspace](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/enterprise-asset-management/create-tco-report-source-eam.md)**  
Create a TCO report source to add report sources to a TCO report.
-   **[Add calibration attributes to an enterprise asset](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/enterprise-asset-management/add-calibration-attributes-enterprise-asset.md)**  
Add calibration attributes to an enterprise asset to specify and manage the calibrations that are required for the asset throughout its life cycle.
-   **[Clone a calibration attribute to an enterprise asset](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/enterprise-asset-management/clone-calibration-attributes-enterprise-asset.md)**  
Clone an existing calibration attribute to an enterprise asset to add a calibration attribute with similar settings.
-   **[Asset conditions in Enterprise Asset Management](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/enterprise-asset-management/asset-conditions-eam.md)**  
Assessing the condition of assets in your organization aids in effective maintenance planning, resource allocation, and managing the asset life cycle.
-   **[Create an asset group in Enterprise Asset Management](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/enterprise-asset-management/create-asset-groups-eam.md)**  
Create asset groups to create a logical group of entities that includes assets, locations, and users.
-   **[Create consumable assets in Enterprise Asset Workspace](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/enterprise-asset-management/create-consumables-eam.md)**  
Create consumables in the Enterprise Asset Workspace to track enterprise assets.
-   **[Use consumable assets in Enterprise Asset Workspace](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/enterprise-asset-management/create-eamconsumable-asset.md)**  
To use consumable assets, ensure that they are in the **In stock** state and **Available** substate.
-   **[Dispose consumable assets in Enterprise Asset Workspace](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/enterprise-asset-management/dispose-eamconsumable-assets.md)**  
Dispose of those consumable assets that are no longer required in your stockrooms.
-   **[Create an asset onboarding process](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/enterprise-asset-management/create-asset-onboarding-task-eam.md)**  
Create an asset onboarding process to walk you through the process of entering important information for assets.
-   **[Track assets using indoor maps in the Enterprise Asset Workspace](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/enterprise-asset-management/track-enterprise-assets-indoor-maps.md)**  
Use indoor maps to locate and track the enterprise assets, consumable assets, and pallets that are deployed across your organization. Each indoor map consists of an interactive interface that enables you to visualize where your assets are located within your campuses, buildings, floors, and places.
-   **[Use Advanced Shipment Notification in Enterprise Asset Management](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/enterprise-asset-management/asn-eam.md)**  
Use Advanced Shipment Notification \(ASN\) to automate and create enterprise asset records for when your assets are in transit.
-   **[View and add enterprise asset shipments](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/enterprise-asset-management/view-enterprise-asset-shipments.md)**  
Use the Shipments list to view all active enterprise asset shipments from a central location. If an existing shipment doesn’t appear on the list, you can add it manually.
-   **[Zebra MotionWorks RFID integration for Enterprise Asset Management](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/enterprise-asset-management/eam-zebra-rfid-integration.md)**  
You can integrate your ServiceNow instance with third-party Zebra MotionWorks location solutions to import and view real-time radio-frequency identification \(RFID\) location data for your enterprise assets.
-   **[View the asset hierarchy in the Enterprise Asset Workspace](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/enterprise-asset-management/view-asset-hierarchy-eam.md)**  
View the hierarchy details of a multi-component asset in a tree-like format in the Enterprise Asset Workspace.
-   **[View the total time worked on all tasks related to an enterprise asset](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/enterprise-asset-management/view-total-time-spent-on-asset-eam.md)**  
Use the Asset task time summary report to see the details of the time spent on all tasks for an asset and its child assets, grouped by task category.
-   **[Asset performance reports in the Enterprise Asset Workspace](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-asset-management/enterprise-asset-management/asset-performance-reports-eam.md)**  
Gain insights into the overall operational efficiency of your organization's enterprise assets by using the asset performance reports available in the Asset analytics view.
