# Socialsuite material topic fields

Fields added to the Material Topic \(sn\_esg\_material\_topic\) table for Socialsuite integration.

<table id="table_fkx_dz1_j3c"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Material Topic Source

</td><td>

Source system where the material topic was created. For topics imported from Socialsuite, this field displays **Socialsuite**.

</td></tr><tr><td>

Assessment Methodology

</td><td>

Methodology used for the materiality assessment in Socialsuite. The possible values are:-   • Double Materiality: Assesses both impact and financial materiality.
-   • Financial Materiality: Assesses financial impacts only.
-   • Impact Materiality: Assesses impact on society and environment only.

</td></tr><tr><td>

Reporting Period

</td><td>

Reporting period associated with the materiality assessment.

</td></tr><tr><td>

Reporting Period Start Date

</td><td>

Start date of the reporting period.

</td></tr><tr><td>

Reporting Period End Date

</td><td>

End date of the reporting period.

</td></tr><tr><td>

Impact Materiality Score

</td><td>

Score indicating the organization's impact on society and the environment. Scores range from 0-100.

</td></tr><tr><td>

Financial Materiality Score

</td><td>

Score indicating how sustainability issues affect the organization's financial performance. Scores range from 0-100.

</td></tr><tr><td>

Material Topic Score

</td><td>

Overall materiality score for the topic.

</td></tr><tr><td>

Topic Source

</td><td>

Additional source information for the material topic.

</td></tr><tr><td>

Materiality Decision

</td><td>

Decision outcome from the materiality assessment.

</td></tr></tbody>
