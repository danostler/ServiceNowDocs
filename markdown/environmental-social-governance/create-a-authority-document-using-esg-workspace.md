# Create an authority document using Operational Sustainability Workspace

Create an authority document that outlines the reporting obligations for ESG disclosures.

## Before you begin

Role required: sn\_esg.program\_manager

## Procedure

1.  Navigate to **All** &gt; **Operational Sustainability Management** &gt; **Operational Sustainability Workspace**.

2.  In the sidebar, select the Lists \[Omitted image "icon-lists.png"\] Alt text: icon.

3.  In the **Lists** tab, select **Disclosure requirements** &gt; **Authority documents**.

4.  On the Authority documents page, select **New**.

5.  On the form, fill in the fields.

    For a description of the details, see [New Authority Document form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/environmental-social-governance/new-authority-document-form.md).

6.  Select **Save**.


## Result

The authority document is created and the Overview page is displayed.
