# New Authority Document form

The New Authority Document form enables you to create a new authority document and define its properties, schedule, and associated citations.

<table id="table_FloorForm"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Number

</td><td>

Read-only field automatically populated with a unique identification number.

</td></tr><tr><td>

Name

</td><td>

Name of the document.

</td></tr><tr><td>

Source

</td><td>

A non-editable field with the source of the policy. For example, if the statement is from the UCF import, the source is UCF.

</td></tr><tr><td>

Source ID

</td><td>

Unique identification number used by the source to catalog this authority document.

</td></tr><tr><td>

Category

</td><td>

Category for this authority document.

</td></tr><tr><td>

Common name

</td><td>

Abbreviated name for the document.

</td></tr><tr><td>

Type

</td><td>

The document type. The available options are: -   Audit Guideline
-   Best Practice Guideline
-   Bill or Act
-   Contractual Obligation
-   International or National Standard
-   Not Set
-   Organizational Directive
-   Regulation of Statute
-   Safe Harbor
-   Self-Regulatory Body Requirement
-   Vendor Documentation

</td></tr><tr><td>

Version

</td><td>

Unique version number used by the source to identify this authority document.

</td></tr><tr><td>

Active

</td><td>

Option to make the authority document active. The default value is Active.

</td></tr><tr><td>

Description

</td><td>

Optional additional information about the authority document.

</td></tr><tr><td>

Url

</td><td>

URL of the stored authority document.

</td></tr><tr><td class="sub-head" colspan="2">

Schedule

</td></tr><tr><td>

Valid From

</td><td>

Date and time on which the policy becomes valid.

</td></tr><tr><td>

Valid To

</td><td>

Date and time on which the policy is no longer valid.

</td></tr><tr><td class="sub-head" colspan="2">

Settings

</td></tr><tr><td>

Functional domain

</td><td>

Functional domain for the authority document, for example, ESG.
