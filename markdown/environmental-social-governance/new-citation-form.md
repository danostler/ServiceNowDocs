# Create New Citation form

The Create New Citation form enables you to view and manage citation details, including source information, reporting requirements, and GRI or SASB category-specific fields.

<table id="table_FloorForm"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Name to identify this citation.

</td></tr><tr><td>

Source

</td><td>

Source of the policy. For example, a third-party provider.

</td></tr><tr><td>

Source ID

</td><td>

Unique identification number used by the source to catalog this authority document.

</td></tr><tr><td>

Type

</td><td>

Type of citation created. Use the optional value in this field for information for reports or to query for records of a specific type. The available values are:-   Core Topic
-   Process
-   Control Objective
-   Control
-   Supporting information

</td></tr><tr><td>

Reference

</td><td>

URL or document that the citation references.

</td></tr><tr><td>

Parent

</td><td>

Parent citation that this citation is associated with, if applicable.

</td></tr><tr><td>

Authority document

</td><td>

Parent authority document for this citation. Automatically provided when creating citations from the authority document form.

</td></tr><tr><td>

Active

</td><td>

Option to activate the citation. This field is automatically set to active.

</td></tr><tr><td>

Description

</td><td>

\(Optional\) Description of the citation.

</td></tr><tr><td class="sub-head" colspan="2">

Reporting

</td></tr><tr><td>

Reporting requirement

</td><td>

Specific reporting mandates for this citation, as defined by the authority document. For GRI and SASB citations, this field is automatically populated and read-only.

</td></tr><tr><td>

Supplemental guidance

</td><td>

Additional guidance on how to interpret and comply with the reporting requirements. For GRI and SASB citations, this field is automatically populated and read-only.

</td></tr></tbody>
