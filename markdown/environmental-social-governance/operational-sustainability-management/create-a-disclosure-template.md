# Create a disclosure template

Create your own unique Microsoft Word disclosure templates tailored to different types of disclosures. Utilize these templates that you create when you generate specific disclosures. The templates help to streamline the disclosure process by implementing customized templates for efficiency and consistency.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Operational Sustainability Management** &gt; **Administration** &gt; **Disclosure Templates**.

2.  Select **New**.

3.  On the form, fill in the fields.

<table id="table_lrv_yw4_brb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Disclosure type

</td><td>

Type of disclosure. The choices are as follows:-   **Annual report**
-   **ESG report**
-   **Regulatory and framework**
-   **Request for information**
-   **Ratings and indices**
-   **Supplier questionnaire**
-   **Others**


</td></tr><tr><td>

Template document

</td><td>

Document to add.**Note:** The file extension must be .docx.

</td></tr></tbody>
</table>4.  Select **Click to add...**.

5.  Select **Choose file**.

    1.  Select a file of your choice from your local system.

    2.  Select **Open**.

    3.  Select **OK**.

    **Note:** Ensure that the size of the file does not exceed 10 MB and only one template for each disclosure type is defined.
