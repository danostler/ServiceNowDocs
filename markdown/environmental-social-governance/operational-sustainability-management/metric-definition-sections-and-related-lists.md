# Metric definitions sections and related lists

The metric definition form includes sections and related lists that define the metric definition and display its attributes.

<table id="id_xqr_llg_q3c"><thead><tr><th>

Sections and related lists

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Details

</td><td>

Fields that define the metric, such as name, category, frequency, and data collection settings. The fields vary depending on the type of metric definition.

</td></tr><tr><td>

Entity type

</td><td>

Entity types associated with the metric definition. Displays the name, description, and entity conditions..**Note:** The Entity type related list is not displayed for calculated metric definitions.

</td></tr><tr><td>

Entities

</td><td>

Entities associated with the metric definition. Displays details such as name, owned by, and class.**Note:** This Entities related list is not displayed for the calculated metric definition.

</td></tr><tr><td>

Metrics

</td><td>

Metrics generated from this metric definition. Displays the name, associated entity, and active state of each metric.

</td></tr><tr><td>

Citations

</td><td>

Citations associated with the metric definition. Displays the reference, name, authority document, description, and compliance score.

</td></tr><tr><td>

Goals

</td><td>

Goals associated with the metric definition. Displays the name, state, category, start date, end date, owner, and status.

</td></tr><tr><td>

Targets

</td><td>

Targets associated with the metric definition. Displays the name, owner, state, status, progress, actual value, target value, and end date.

</td></tr><tr><td>

Metric definition data

</td><td>

Collected metric data records for this definition. Displays the collection frequency, interval end date, value, and state.**Note:** This related list is not displayed for calculated metric definitions.

</td></tr><tr><td>

Thresholds

</td><td>

Threshold rules applied to the metric definition. Displays the threshold type, valid from and until dates, red threshold, amber threshold, and target value. **Note:** This related list is displayed only for an automated metric definition and a manual metric definition. It isn’t displayed for the calculated metric definition.

</td></tr></tbody>
