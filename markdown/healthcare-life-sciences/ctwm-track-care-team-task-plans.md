# Track care team task plans

Once you’ve published a task plan, you can track progress of all healthcare orchestration cases and tasks from within the Healthcare Workspace.

## Before you begin

Role required: sn\_hco\_orc.plan\_author

## Procedure

1.  In the Healthcare Operations Workspace, navigate to **Task Plan Templates** &gt; **All Task Plan Templates**.

2.  Select the Task Plan template you wish to review.

3.  Navigate to **Template Items** to review all Healthcare orchestration cases, Healthcare orchestration tasks, Care team cases, and Care team tasks generated as a result of this plan.

4.  Select a case or task number to open a more detailed view.

5.  Navigate to **Task Plan Template Executions** to review all executions of this task plan.

    When a task plan is generated on a recurring basis, all history of these generations are represented here.

6.  Select an execution number for a more detailed view of that execution.
