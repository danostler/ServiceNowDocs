# Browse locations in Care Team Mobile

Browse locations in Care Team Mobile to view healthcare locations and create support requests for specific locations as needed.

\[Omitted image "hcls-cto-mobile-browse-locations.png"\] Alt text: Location browse functionality in Care Team Mobile.

You can browse locations using **Browse locations** from the Quick actions menu.

After selecting **Browse locations**, a list of all associated teams is then displayed.

Selecting into a team displays a detailed location hierarchy associated with that team.

You can select a location to view more detailed information about that location then optionally create a request for that location.

For more information about creating requests using **Browse locations**, see [Browse locations to create a support request in Care Team Mobile](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/healthcare-life-sciences/cto-mobile-create-request-browse-locations.md).
