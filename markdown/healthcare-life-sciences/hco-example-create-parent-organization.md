# Example - Create a parent organization \(HQ\)

Explore how a parent organization is created for Healthcare Operations, under which departments or similar sub-organizations can then be created and organized.

## Before you begin

Role required: admin

## About this task

Scenario: Armand, a ServiceNow administrator for a hospital, needs to set up a parent organization within Healthcare Operations. This parent organization will serve as the top-level entity under which various healthcare providers and facilities will be organized. Armand creates a parent organization in ServiceNow to ensure that all related entities inherit the correct organizational structure and permissions.

## Procedure

1.  Navigate to **All** &gt; **Healthcare organizations** &gt; **All**.

2.  Select **New**.

3.  Fill in the following fields.

    -   Name - Enter **Alectri Health**.
    -   Organization type - Enter **Healthcare Provider**.
4.  Select **Submit**.


## Result

A parent organization representing HQ, Alectri Health, has been created. This serves as the top-level entity for healthcare organizations within this organization.
