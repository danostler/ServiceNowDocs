# Report a medical device issue with Care Team Operations for Biomed

Report any issue with a medical device issue using the Report a medical device issue catalog item from the Care Team Operations portal.

## Before you begin

Role required: sn\_hco.care\_team\_member

## Procedure

1.  In the Care Team Portal, navigate to **Create a request** &gt; **Request biomed service** &gt; **Report a medical device issue**.
