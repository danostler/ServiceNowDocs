# Assign operational leader roles

Assign your operational leaders the correct roles so that they can leverage the Operational Rounding playbook to create task plan templates for care teams.

## Before you begin

Role required: admin

## About this task

Operational leaders create and fulfill task plan templates generates from the Operational Rounding playbook.

Operational leaders require two user roles:

-   sn\_hco\_orc.plan\_author **and**
-   sn\_hco\_orc.loc\_support\_agent\_manager **or** sn\_hco\_orc.loc\_support\_agent

For example, a supporting services department manager who needs to create task plan templates using the Operational Rounding playbook would have the sn\_hco\_orc.plan\_author and sn\_hco\_orc.loc\_support\_agent\_manager roles assigned to their user profile.

## Procedure

1.  Navigate to **All** &gt; **User Administration** &gt; **Users** then open a user record.

2.  In the **Roles** related list, select **Edit**.

3.  In the **Collection** list, select the desired roles, and then select **Add**.

4.  Select **Save**.
