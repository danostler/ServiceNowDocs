# Example - Associate a healthcare location with a healthcare organization

Explore how a healthcare location associated with an existing healthcare organization, so that users in the organization can select the location when creating support requests using Care Team Operations.

## Before you begin

Role required: admin

## About this task

**Scenario:**At Alectri Health Santa Clara Hospital, an ICU room is being used by both the cardiology and neurology departments. Armand, a ServiceNow administrator, needs to associate this room with multiple departments so care team members associated with these departments can select the location when creating support requests using care team operations.

## Procedure

1.  Navigate to **All &gt; Healthcare Operations &gt; Healthcare Organizations**.

2.  Select **Alectri Health Santa Clara Cardiology.**

    **Note:**This is the healthcare organization to be associated with a location.

3.  Navigate to the **Healthcare locations** related list.

4.  Click **Edit**.

5.  In the Collection panel, select **ICU Room**and use the right arrow to move it into the Healthcare Locations panel.

6.  Select **Save**.

7.  Repeat the above steps for **Alectri Health Santa Clara Neurology.**


## Result

The ICU room has now been associated with both Alectri Health Santa Clara neurology and cardiology departments and can be selected by care team members of both departments when creating support requests.
