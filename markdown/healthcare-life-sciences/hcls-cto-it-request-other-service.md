# Request other IT service with Care Team Operations for IT

Request IT service for non-EMR issues using the Request other IT service catalog item from the Care Team Portal.

## Before you begin

Role required: sn\_hco.care\_team\_member

## Procedure

1.  In the Care Team Portal, navigate to **Create a request** &gt; **Request IT service** &gt; **Request other IT service**.

2.  Fill in the following fields.
