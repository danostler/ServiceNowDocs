# Cancel appointments in a given date range

Cancel multiple appointment bookings at a location in a given date range.

## Before you begin

Role required: sn\_vaccine\_sm​.admin or sn\_vaccine\_sm​.manager

## Procedure

1.  Navigate to **All** &gt; **Vaccine Administration Management** &gt; **Administration** &gt; **Centers**.

2.  Open the vaccination center record that you want to modify.

3.  To cancel all appointments at the current location in a given date range along with subsequent doses, if any, click **Cancel Appointments**.

4.  A pop-up window appears that enables you to cancel the appointments by choosing the start date and the end date and click **Yes**.

    If the end date is empty, all appointments after the start date get canceled.
