# EMR Help data model tables

Tables installed with the EMR Help application enable you to submit requests on your ServiceNow® instance from an EMR System.

To learn more about the EMR Help data model, see [EMR Help data model](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/healthcare-life-sciences/emr-help/emr-data-model.md).

-   **[Remote request definition table](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/healthcare-life-sciences/emr-help/emr-remote-request-definition.md)**  
The Remote Request Definition \[sn\_ind\_rmt\_help\_request\_defn\] table stores the record producer and task type.
-   **[Remote request parameter table](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/healthcare-life-sciences/emr-help/emr-remote-request-parameter.md)**  
The Remote Request Parameter \[sn\_ind\_rmt\_help\_request\_param\] table stores all parameters that can be used across all remote request definitions.
-   **[Remote request data table](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/healthcare-life-sciences/emr-help/emr-remote-request-data.md)**  
The Remote Request Data \[sn\_ind\_rmt\_help\_request\_data\] table stores the captured parameter data associated with a task record.
-   **[EMR Request Data table](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/healthcare-life-sciences/emr-help/emr-emr-incident-data.md)**  
The EMR Request Data \[sn\_ind\_rmt\_help\_incident\_data\] table stores the captured parameter data associated with a task record.
-   **[Request configuration mapping table](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/healthcare-life-sciences/emr-help/emr-request-configuration-mapping.md)**  
The Request configuration mapping \[sn\_ind\_rmt\_help\_defn\_param\_data\_map\] table stores the association between the request parameter and the request definition.
