# Resolve a healthcare case using CSM/FSM Configurable Workspace

Use Workspace in a ServiceNow instance to work on healthcare cases created from an EMR system.

## Before you begin

Role required: sn\_ind\_rmt\_help.viewer

## Procedure

1.  Navigate to **Workspace** &gt; **CSM/FSM Configurable Workspace**.

2.  Select the **List** icon and then navigate to **Cases** &gt; **All**.

3.  Select the EMR case that you want to work on.

4.  Review and the case using the **Details** tab.

    **Note:**

    Information from the EMR request is captured in the **EMR session overview** contextual side panel.

5.  To close the EMR case, scroll to the **Closure Information** section.

6.  In the **Closure Information** section, complete the form fields.

7.  Select **Close Case**.
