# Browse locations to create a support request in Care Team Mobile

Use **Browse locations** in Care Team Mobile to browse and create support requests for specific locations.

## Before you begin

Role required: Team member, Team manager

## Procedure

1.  In Care Team Mobile, navigate to **Browse locations**.

2.  View all locations associated with your team.

3.  Select **Filters** to filter by team name and sort locations alphabetically or by name.

4.  Select a location to view more details about that location.

5.  Select **Assets** to display the list of assets associated with the location you are viewing.

6.  Select **Create a request** to create a support request for this location.

    The location information will automatically populate into the request.
