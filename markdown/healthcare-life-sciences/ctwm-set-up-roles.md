# Set up roles for care teams

Set up your care team agents and care team agent managers with the correct roles so care team cases and care team tasks can be fulfilled.

## Before you begin

Role required: admin

## About this task

In order to fulfill care team cases and tasks, the following roles should be assigned to care team members:

**Roles**

</td><td>

Care Team Agent Manager

</td></tr></tbody>
</table>## Procedure

1.  Navigate to **All** &gt; **User Administration** &gt; **Users** then open a user record.

2.  In the **Roles** related list, select **Edit**.

3.  In the **Collection** list, select the desired roles, and then select **Add**.

4.  Select **Save**.
