# Care Team Mobile overview

The Care Team Mobile landing page offers users a list of quick actions, recent updates on requests, and request tracking information.

\[Omitted image "hcls-cto-mobile-overview.png"\] Alt text: Care Team Mobile overview screen.

## Quick actions

The following quick actions are available directly from the Care Team landing page:

-   **Create a request**: Enables care team users to generate a support request for supporting departments.
-   **Scan asset**: Enables care team users to scan a QR code then generate a request with information about the associated medical device.
-   **Browse locations**: Helps care team users find and select healthcare locations they’re associated with to create requests specific to those locations.

## Recent updates

**Recent updates** displays all active requests related to you or your team. The most recently updated request displays first.

## Track requests

**Track requests** displays all requests associated with the user or their team organized by request type.

## Knowledge

**Knowledge** displays relevant knowledge articles.

## Saved

**Saved** displays all bookmarked requests for tracking purposes.

## Notifications

**Notifications** displays all notifications for the current user.

## Settings

**Settings** displays all setting configurations for the Care Team Mobile user experience.
