# Example - Create a department organization team

Explore how a department organization team is created under an existing department for Healthcare Operations.

## Before you begin

Role required: admin

## About this task

Scenario: Armand, a ServiceNow administrator for a hospital, needs to set up the Pediatrics department organization team at Pediatrics Alectri Santa Clata. These teams need to be associated with the specific department they are attached to. By correctly setting up department organizations under the parent department, this ensures that all related entities inherit the correct organizational structure and permissions.

## Procedure

1.  Navigate to **All** &gt; **Healthcare Operations** &gt; **Healthcare organizations** &gt; **All**.

2.  Select **New**.

3.  Fill in the following fields.

    -   Name - Enter **Pediatrics Alectri Santa Clara**.
    -   Organization type - Select **Organization team**.
    -   Parent organization - Select the parent department created in [Example - Create a department](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/healthcare-life-sciences/healthcare-and-life-sciences/hco-example-create-department.md), **Pediatrics Alectri Santa Clara**.
4.  Select **Submit**.


## Result

The General Pediatric Alectri Santa Clara department organization team has been created under the parent department Pediatrics Alectri Santa Clara. This setup ensures a streamlined and efficient organizational structure for managing various teams within the department.
