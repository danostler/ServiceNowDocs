# Configure Care Team Operations Case Creation Voice AI agent triggers

Change the trigger phrases used to call on the Care Team Operations Case Creation Voice AI agent via phone.

## Before you begin

Role required: admin

## About this task

By default, this agent is triggered when a user mentions Care Team Operations, CTO, or asks to submit a care team request. For example:

-   "I need help with CTO"
-   "I want to submit a Care Team Operations request"
-   "Care Team"

These phrases can be adjusted for your use case as needed.

**Important:** When updating these trigger phrases, use a unique value that is not used by any other installed agents to avoid those agents from erroneously being triggered.

## Procedure

1.  Navigate to **AI Agent Studio** &gt; **Create and manage** &gt; **AI Agents**.

2.  Select **Care Team Operations Case Creation Voice AI Agent**.

3.  In **Define the specialty**, navigate to **AI agent description**.

4.  Replace the existing instructions and examples with desired context.

5.  Select **Save**.


## Result

The Care Team Operations Case Creation Voice AI agent is now triggered on phone conversations based on the updated phrases and context you provided.
