# Create requests in the Care Team Portal

Use the Care Team Portal to create support requests for services from ancillary departments.

## Before you begin

Role required: sn\_hco.care\_team\_member or sn\_hco.care\_team\_manager

## About this task

Use the Care Team Portal, part of Healthcare Operations Core, to create support requests for ancillary departments. You can also leverage the Healthcare Operations Core case type to create your own record producers. For detailed instructions, refer to [Configuring Healthcare Operations Core](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/healthcare-life-sciences/healthcare-operations-core/hcls-cto-configuring.md).

If you have the Care Team Operations for Healthcare IT or Care Team Operations for Biomed plugins installed, the Care Team Portal includes pre-installed record producers. These plugins can be used to create preconfigured requests. For more information, see [Care Team Operations for Healthcare IT](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/healthcare-life-sciences/healthcare-and-life-sciences/hcls-cto-it-app.md) and [Care Team Operations for Biomed](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/healthcare-life-sciences/healthcare-and-life-sciences/care-team-operations-for-biomed.md).

## Procedure

1.  In the Care Team Portal, navigate to **Create a request**.

2.  Select the type of request that you want to create.

3.  Fill in details as needed and submit your request.


## Result

Your request is created and routed to the supporting department for fulfillment.
