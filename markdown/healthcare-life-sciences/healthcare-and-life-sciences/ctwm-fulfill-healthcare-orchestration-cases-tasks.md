# Fulfill healthcare orchestration cases and tasks

Fulfill the healthcare operations orchestration cases and tasks created from task plans.

## Before you begin

Role required: sn\_hco\_orc.loc\_support\_agent or sn\_hco\_orc.admin

## About this task

For more information on healthcare orchestration cases and tasks, see [Healthcare orchestration cases and tasks](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/healthcare-life-sciences/healthcare-and-life-sciences/ctwm-healthcare-orc-cases-task.md).

## Procedure

1.  In the Healthcare Orchestration Workspace, navigate to **Healthcare orchestration case** &gt; **All cases**.

2.  Update case details as needed.

3.  Select **Tasks** to view all associated healthcare orchestration tasks.

4.  Select a task and fulfill as needed.

5.  Use **Close case** or **Close task** to close out the case or task, respectively.
