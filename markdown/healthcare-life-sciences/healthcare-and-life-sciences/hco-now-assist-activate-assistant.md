# Activate AI Assistant for Care Team Portal

Enable AI Assistant functionality in the Care Team Portal to provide intelligent assistance and support for healthcare workflows.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Conversational Interfaces** &gt; **Assistants**.

2.  Select **Manage Assistants**.

3.  Navigate to **Now Assist in Virtual Agent \(default\)** and select **Edit**.

4.  Select **Edit chat features**.

5.  Navigate to **Display experiences**.

6.  In **Chat experiences**, select **Add Portal**.

7.  Select **Care Team Portal**.

8.  Select **Save**.


## Result

Now Assist in Virtual Agent is activated for use in the Care Team Portal.
