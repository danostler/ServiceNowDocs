# Report an EMR issue with Care Team Operations for IT

Report any issue with your EMR system using the Report an EMR issue catalog item from the Care Team Operations portal.

## Before you begin

Role required: sn\_hco.care\_team\_member

## Procedure

1.  In the Care Team Portal, navigate to **Create a request** &gt; **Request IT service** &gt; **Report an EMR issue**.

2.  Fill in the following fields.
