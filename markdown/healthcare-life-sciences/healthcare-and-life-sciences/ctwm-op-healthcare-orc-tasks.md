# Input Orchestration task details into the Operational Rounding playbook

Complete the **Orchestration task** playbook stage in the Operational rounding playbook.

## Before you begin

Role required: sn\_hco\_orc.loc\_support\_agent or sn\_hco\_orc.plan author

## Procedure

1.  In the **Operational rounding playbook**, navigate to the **Orchestration tasks** stage.

2.  Select **Add task** to define coordinator activities.

    For example, “Send unit communications” or “Review submitted assessments”.

3.  Set other fields as needed.

4.  Select **Continue**.
