# Configuring Care Team Operations for Healthcare IT

The following setup steps need to occur before using Care Team Operations for Healthcare IT.

## Configuration overview

1.  [Install Care Team Operations for Healthcare IT](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/healthcare-life-sciences/healthcare-and-life-sciences/cto-it-activate.md)

    You can install the Care Team Operations for Healthcare IT application \(sn\_cto\_hcit\) if you have the admin role.

2.  [Assign roles to Care Team Operations for Healthcare IT users](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/healthcare-life-sciences/healthcare-and-life-sciences/cto-it-set-up-roles.md)

    Ensure that the correct roles are assigned to users of Care Team Operations for Healthcare IT.

3.  [Create healthcare organizations for your healthcare IT support teams](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/healthcare-life-sciences/healthcare-and-life-sciences/cto-it-create-orgs.md)

    Create healthcare organizations to represent your fulfilling organizations for Healthcare IT cases.

4.  [Create a group for all location support agents in Care Team Operations for Healthcare IT](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/healthcare-life-sciences/healthcare-and-life-sciences/cto-it-create-support-agent-group.md)

    Create a group for location support agents with the sn\_cto\_hcit.loc\_support\_agent assigned so that users added to this group inherits the collection of roles for Care Team Operations for Healthcare IT.

5.  [Create a group for all location managers in Care Team Operations for Healthcare IT](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/healthcare-life-sciences/healthcare-and-life-sciences/cto-it-create-manager-group.md)

    Create a group for location managers with the sn\_customerservice.svc\_location\_manager assigned so that users added to this group will inherit the collection of roles for Care Team Operations for Healthcare IT.

6.  [Set up assignment groups for Care Team Operations for Healthcare IT](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/healthcare-life-sciences/healthcare-and-life-sciences/cto-it-assignment-groups.md)

    Associate assignment groups with your healthcare organizations to ensure cases are assigned to the correct healthcare organizations.

7.  [Modify state decision tables in Care Team Operations for Healthcare IT](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/healthcare-life-sciences/healthcare-and-life-sciences/cto-it-modify-decision-tables.md)

    Use Decision Builder to change the state mappings for cases and incidents in Care Team Operations for Healthcare IT.
