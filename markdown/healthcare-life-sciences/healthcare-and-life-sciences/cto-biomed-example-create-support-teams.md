# Example - Set up your biomed support department Team

Explore how a biomed support department is created to which support agents and managers can later be added in Care Team Operations for Biomed.

## Before you begin

Role required: admin

## About this task

Armand, a ServiceNow administrator, needs to set up a department organizational team for the Biomed support team of Alectri Health Santa Clara. Doing so will allow him to add users who are biomed support agents or managers later.

## Procedure

1.  Navigate to **All &gt; Healthcare Operations &gt; Healthcare Organizations &gt; All**.

2.  Select the **Alectri Health Santa Clara**.

    This is the hospital the department is a part of.

3.  Navigate to Child Organization and select **New.**

4.  Enter the following fields:

    1.  Name – Enter **Alectri Health Santa Clara** **Biomed Support**.

    2.  Organization type – Select **Hospital department.**

5.  Fill in the other fields as needed.

6.  Select **Submit**.


## Result

Armand has created a support department for the Biomed team at Alectri Health Santa Clara to which members can now be added for fulfilling support requests created from Care Team Operations.
