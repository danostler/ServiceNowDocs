# Enable Push Notifications for Care Team Mobile

Push notifications for Care Team Mobile must be enabled manually.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Push Application**.

2.  Select the **ServiceNow Mobile Application** record.

3.  Navigate to the **Push Default Registrations** related list and select **New**.

4.  Add the case type that you want push notifications enabled for.

    -   **Care Team Mobile HCIT Notifications**
    -   **Care Team Mobile Biomed Notifications**
    -   **Care Team Mobile EVS Notifications**
    -   **Care Team Mobile Facilities Notifications**
    **Note:** Repeat this step for other case types as needed.

5.  Select **Submit**.


## Result

Push notifications are enabled in Care Team Mobile for all specified case types.
