# Scan an asset to create a support request in Care Team Mobile

Use **Scan asset** in Care Team Mobile to generate support requests by scanning bar codes attached to assets.

## Before you begin

Role required: Team member, Team manager

## Procedure

1.  In Care Team Mobile, navigate to **Quick Actions** &gt; **Scan asset**.

2.  Scan the bar code attached to the asset to retrieve the asset tag.

3.  Select **Submit**.

    If no existing asset is found, a message is displayed to alert the user that no existing asset exists.

4.  Select the returned asset record.

    Any active requests, details, or activities associated with the asset are displayed.

5.  Select **Create a request** to create a support request for this asset.
