# Input Assigned organizations details into the Operational Rounding playbook

Complete the **Assigned organizations** playbook stage in the Operational rounding playbook.

## Before you begin

Role required: sn\_hco\_orc.loc\_support\_agent or sn\_hco\_orc.plan author

## Procedure

1.  In the **Operational rounding playbook**, navigate to the **Assigned Organizations** stage.

2.  Use the **Organizational selector** to check one of more units/departments across hospitals.

    **Note:** Use **Add all** to add all organizations to this plan.

3.  Select **Save**.

    The plan will generate **Care Team Cases** and any associated tasks for each selected organization.
