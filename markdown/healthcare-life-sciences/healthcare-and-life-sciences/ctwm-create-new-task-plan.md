# Create an Operational rounding task plan

Create a new task plan using the Operational rounding playbook.

## Before you begin

Role required: sn\_hco\_orc.loc\_support\_agent or sn\_hco\_orc.plan author

## About this task

## Procedure

1.  Navigate to **CSM/FSM Configurable Workspace** &gt; **Task Plan Templates** &gt; **All Task Plans** &gt; **New**.

2.  In **Create new task plan**, select **Operational Rounding**.

3.  Select **Create plan**.

4.  Complete the playbook stages that follow.
