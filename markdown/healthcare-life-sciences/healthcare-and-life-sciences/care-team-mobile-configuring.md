# Configure Care Team Mobile

Set up the Care Team Mobile application.

## Configuration overview

1.  [Activate Care Team Mobile](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/healthcare-life-sciences/healthcare-and-life-sciences/cto-mobile-install.md)

    Activate the Care Team Mobile plugin \[com.sn\_cto\_mobile\] if you have the admin role. This plugin includes demo data and activates related plugins if they aren’t already active.

2.  [Assign roles for Care Team Mobile users](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/healthcare-life-sciences/healthcare-and-life-sciences/cto-mobile-assign-roles.md)

    User access for Care Team Mobile is based on whether users have been assigned the Team Member or Team Manager role.

3.  [Enable Push Notifications for Care Team Mobile](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/healthcare-life-sciences/healthcare-and-life-sciences/cto-mobile-enable-push-notifications.md)

    Push notifications for Care Team Mobile must be enabled manually.
