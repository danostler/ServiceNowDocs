# Activate Care Team Mobile

Activate the Care Team Mobile plugin \[com.sn\_cto\_mobile\] if you have the admin role. This plugin includes demo data and activates related plugins if they aren’t already active.

## Before you begin

Role required: admin

The following plugins are activated with Care Team Mobile:

-   com.sn\_mab
-   sn\_mab\_api

## Procedure

1.  Navigate to **All** &gt; **System Applications** &gt; **All Available Applications** &gt; **All**.

2.  Find the \[com.cto\_mobile\] plugin using the filter criteria and search bar.

3.  If you're prompted, follow the links to the ServiceNow Store to get any additional entitlements for dependencies.

4.  Select **Install**.
