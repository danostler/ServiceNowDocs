# Cancel a submitted request in Care Team Mobile

To cancel a submitted request in Care Team Mobile, simply locate the request and select **Cancel case** from the action menu.

## Before you begin

Role required: Team member, Team manager

## About this task

\[Omitted image "hcls-cto-mobile-cancel-requests.png"\] Alt text: Cancel a request in Care Team Mobile.

## Procedure

1.  In Care Team Mobile, navigate to **Track requests**.

2.  Select the submitted request that you want to cancel.

3.  Select **Cancel case** from the Actions menu on the upper righthand corner of the screen.


## Result

The specified request has been canceled.
