# Plan summary stage

Review the task plan template you created using the Operational rounding playbook.

## Before you begin

Role required: admin

## Procedure

1.  In the **Operational rounding playbook**, navigate to the **Plan Summary** stage.

2.  Review all playbook stages as needed:

    -   Select a panel to display all field inputs for that stage.
    -   Update information within a playbook stage by selecting **Edit**.
    -   Select **Save** to save your updates or select **Cancel** to cancel any changes made.
3.  Select **Publish**to publish your task plan template.


## Result

Your task plan template is now published. Cases and tasks will now begin to generate.
