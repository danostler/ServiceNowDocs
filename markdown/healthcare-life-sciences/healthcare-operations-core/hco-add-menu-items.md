# Add menu items into the Care Team Portal with Healthcare Operations Core

Add more menu items into the Care Team Portal for easy user access.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Service Portals** &gt; **Portals**.

2.  Select **Care Team Portal**.

3.  Next to the **Main menu** field, select **Preview this record** and open the record.

4.  In the Menu Items related list, select **More**.

5.  In the Menu Items related list, select **New**.

6.  On the form, fill in the fields.

    For more information about the form fields and descriptions, see [Menu Item form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/employee-service-management/employee-experience-foundation/menu-item-form.md).

7.  Select **Submit**.
