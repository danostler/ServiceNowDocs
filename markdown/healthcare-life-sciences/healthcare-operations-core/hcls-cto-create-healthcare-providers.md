# Create healthcare providers

Create a healthcare provider and associate it with a parent healthcare organization and a common location.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Healthcare Operations** &gt; **Healthcare providers**.

2.  Select **New**.

3.  Fill in the following fields.

    **Note:**

    The Organization type field is automatically set to healthcare provider and is read only.

    |Field|Description|
    |-----|-----------|
    |Name|Name to identify the healthcare provider.|
    |Parent Organization|Parent organization associated with the provider.|
    |Manager|The manager associated with this healthcare provider.|
    |Phone|Phone number of the provider.|
    |Email|Email address associated with the provider.|
    |Description|A description of this healthcare provider.|
    |This is an internal location|Indicates that this healthcare provider is considered an internal location.|
    |Common Location|The common location associated with this healthcare provider.|

4.  Select **Submit**.
