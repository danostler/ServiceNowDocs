# Create healthcare departments

Create a healthcare department and associate it with a parent healthcare organization and a common location.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Healthcare Operations** &gt; **Healthcare departments**.

2.  Select **New**.

3.  Fill in the following fields.

    **Note:**

    The Organization type field is automatically set to healthcare department and is read only.

    |Field|Description|
    |-----|-----------|
    |Name|Name to identify the healthcare department.|
    |Parent Organization|Parent organization associated with the department.|
    |Manager|The manager associated with this healthcare department.|
    |Phone|Phone number of the department.|
    |Email|Email address associated with the department.|
    |Description|A description of this healthcare department.|
    |This is an internal location|Indicates that this healthcare department is considered an internal location.|
    |Common Location|The common location associated with this department.|

4.  Select **Submit**.
