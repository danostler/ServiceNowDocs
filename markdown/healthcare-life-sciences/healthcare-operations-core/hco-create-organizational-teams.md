# Create organizational teams in Healthcare Operations Core

Create a healthcare organization to associate with your healthcare department.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Healthcare Operations** &gt; **Healthcare organizations**.

2.  Select **New**.

3.  Ensure the **Organization type** field is set to **Organizational Team**.

4.  Fill in the following fields.

    |Field|Description|
    |-----|-----------|
    |Name|Name to identify the organizational team.|
    |Parent Organization|Parent organization associated with the organizational team.|
    |Manager|The manager associated with this organizational team.|
    |Phone|Phone number of the organizational team.|
    |Email|Email address associated with the organizational team.|
    |Description|A description of this organizational team.|
    |This is an internal location|Indicates that this organizational team is considered an internal location.|
    |Common Location|The common location associated with this organizational team.|

5.  Select **Submit**.
