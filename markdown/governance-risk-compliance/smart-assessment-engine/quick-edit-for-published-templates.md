# Quick edit for published templates

Quick Edit lets template managers make minor corrections to published templates without creating a version, enabling rapid fixes for typos and formatting issues while assessments remain in progress.

## Quick edit overview

Quick edit provides template managers the ability to modify published templates directly from the builder interface. When enabled at the template category level, the Quick edit feature enables authorized users to edit template details. These details include instructions, question text, descriptions, and response options, without requiring template republication or version creation.

Once enabled, the quick edit feature gives template managers access to inline editing across published templates, making it easy to correct typos and apply small, targeted updates.

All changes made through quick edit are tracked in an edit history, providing full transparency of modifications. Even when Quick Edit is turned off later, edit history remains available for transparency and compliance. The feature doesn’t imply unrestricted modification. Rather, it provides a safe mechanism for small, necessary updates while preserving the integrity of published templates.

## Benefits of quick edit

Quick edit in Smart Assessment Engine provides the following benefits:

-   Reduces friction by enabling quick fixes for typos, grammar, and minor formatting issues directly in published templates.
-   Avoids unnecessary creation of new template versions for small changes, streamlining template maintenance.
-   Enables corrections while assessments are actively in progress, confirming responders see accurate content without disruption.
-   Maintains a complete history of all quick edits, supporting compliance and governance requirements.
-   Improves operational efficiency for template managers.
