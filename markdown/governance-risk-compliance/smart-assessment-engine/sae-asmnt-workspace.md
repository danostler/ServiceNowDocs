# Accessing templates in the Assessment Workspace

You can see all your assessment template-related information on the Assessment Workspace landing page. You can review the published and unpublished assessment templates and create assessment templates by using the Smart Assessment Engine application.

## Smart assessment workspace

You can view all your assessment template-related information by navigating to **Workspaces** &gt; **Assessment Workspace**. If you have the sn\_smart\_asmt.assessment\_admin or sn\_smart\_asmt.template\_manager roles, you can create the assessment templates or update the unpublished assessment templates in the Assessment Workspace. A template manager's ability to create or update the assessment templates is determined by their category role. Any team member with the Template reader \[sn\_smart\_asmt.template\_reader\] role can view the specific templates for their category role. For more information on the assessment categories, see [Create an assessment template category](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/governance-risk-compliance/smart-assessment-engine/sae-asmnt-template-category-create.md). ## Creating assessment templates

You can create a template by selecting **New template** and filling out the Template details form. For more information, see [Create an assessment template](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/governance-risk-compliance/smart-assessment-engine/sae-asmnt-template-create.md) and [Using the template designer](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/governance-risk-compliance/smart-assessment-engine/sae-template-designer.md).
