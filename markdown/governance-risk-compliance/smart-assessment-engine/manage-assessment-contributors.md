# Manage assessment contributors

The assessment owner can add or remove contributors to an assessment, assign them either full assessment access or section-specific access based on their responsibilities and expertise. Contributors can work simultaneously on the assessment, enabling real-time collaboration and efficient progress.

## Before you begin

The Smart Assessment Collaboration plugin \(com.sn\_smart\_collab\) must be installed.

Role required: sn\_smart\_asmt.actor

## Procedure

1.  Navigate to your workspace and then to your list of assessments.

    Here’s an example of how to navigate to this list in the Compliance Workspace.

    1.  Navigate to **All** &gt; **Policy and Compliance** &gt; **Compliance Workspace**.
    2.  Select the tasks icon \[Omitted image "list-icon.png"\] Alt text:.
    3.  Select the **Control attestations** &gt; **Smart Assessments** on the left panel.
    4.  The smart assessments list is displayed.
2.  Select the assessments in which you want to add contributors.

    The assessment owner and admin can add additional contributors to the assessment.

3.  From the contextual sidebar, select the **Contributors** icon \[Omitted image "contributor-icon.png"\] Alt text:.

4.  Select the **Manage** button.

5.  In the **Manage contributors** window, to add contributors with full assessment access:

    1.  In the **Assessment access** section, select the **Select contributors** field.

    2.  From the dropdown list, select a contributor and repeat to add additional contributors.

    3.  Select **Save and close**.

    If no users appear in the **Select contributors** field, confirm the following:

    -   The users you're adding have the sn\_smart\_asmt.actor role.
    -   The **Smart Assessment Collaboration** plugin \(com.sn\_smart\_collab\) is installed and active.
6.  In the **Manage contributors** window, to add contributors with sectional access:

    1.  In the **Sectional access** section, select the **Assigned users** field for the **Section** you want to assign contributors to.

        If they have expertise across different areas, you can assign the same user to multiple sections.

    2.  From the dropdown list, select a contributor and repeat to add additional contributors.

    3.  Select **Save and close**.

7.  In the **Assessment access** or **Sectional access** section, select the remove icon \[Omitted image "remove-user.png"\] Alt text: next to the contributor's name you want to remove.

    In the **Sectional access** section, select **Assigned users** to display the remove icon \[Omitted image "remove-user.png"\] Alt text:.

8.  Select **Save and close**.
