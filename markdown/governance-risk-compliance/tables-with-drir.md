# Tables installed with Digital resilience incident reporting

The tables installed with Digital resilience incident reporting are listed in this section.

## Tables for Digital resilience incident reporting

|Table name|Description|
|----------|-----------|
|Action task \[sn\_grc\_case\_mgmt\_case\_task\]|Action tasks associated with the case.|
|Digital Resilience Incident \[sn\_grc\_inc\_rptg\_case\_task\]|Incident reporting case task.|
|Impacted area \[sn\_grc\_case\_mgmt\_impacted\_area\]|Impacted area for the case.|
|Related area \[sn\_grc\_case\_mgmt\_related\_area\]|Related area for the case.|
|Entities \[sn\_grc\_inc\_rptg\_entities\]|Entities used for Incident reporting.|
|Entities \[sn\_grc\_inc\_rptg\_sir\_entities\]|Entities used for Security incident reporting \(SIR\).|
|Due date configuration \[sn\_reg\_body\_mgmt\_agency\_profile \]|Due date configuration for regulatory agency profile.|
|Action Task Configuration \[sn\_reg\_body\_mgmt\_template\_configuration\]|Action task configuration for template configuration.|
