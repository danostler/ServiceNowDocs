# Update a GRC application previously downloaded from the ServiceNow Store

If you have previously downloaded an application from the ServiceNow Store and a new version is available, you can update it in your instance. When you update, the latest version of the application is installed from the application store along with the application package.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **System Applications** &gt; **All Available Applications** &gt; **All**.

    \[Omitted image "system-apps.png"\] Alt text: System applications

2.  Search for the application you want to update.

3.  Select the version you want to update to and click **Update**.

    Your application is automatically updated on your instance.
