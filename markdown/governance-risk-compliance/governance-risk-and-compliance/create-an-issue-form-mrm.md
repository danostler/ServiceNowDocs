# Create an issue form

Use the Create an issue form to capture validation findings or performance problems for a model.

See the following table for a description of the field values.

|Field|Description|
|-----|-----------|
|Name|Unique name for the issue. For example, `Model accuracy deviation`.|
|Description|Detailed information about the issue, including the context, observed behavior, and any relevant details.|
|Recommendation|Suggested action or solution to address the issue.|
|Priority|Importance level of the issue to help determine the order in which it should be resolved.|
|Assigned to|Individual responsible for resolving the issue.|
|Issue source|Origin or source from which the issue was identified. This field is automatically set to **Model Risk**.|
