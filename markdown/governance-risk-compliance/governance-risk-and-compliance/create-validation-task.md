# Create a validation task

Create a validation task by selecting a predefined validation template, assigning a validator, and setting a due date. This task defines the validation scope and responsibilities, preparing the model for formal validation and deployment.

## Before you begin

Role required: sn\_model\_risk\_mgmt.model\_governance or sn\_model\_risk\_mgmt.risk\_coordinator

## About this task

You can create a validation task for a model when it is in the Pre-deployment or Monitor stage.

## Procedure

1.  Navigate to **All** &gt; **Model Risk Workspace**.

2.  Select the list icon \[Omitted image "list-icon-mrm.png"\] Alt text:.

3.  Select the model for which you want to create a validation task.

4.  In the Pre-deployment state, select **Create task**.

5.  On the Model Validation dialog, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Validation checklist template|Predefined template that outlines the structure, questions, and scoring logic for the assessment. This field is auto-populated with the default assessment template.|
    |Validation assignee|Individual assigned to complete the validation task. This field is auto-populated with the assignee name selected as a validator in the stakeholder list. You can select only those users who are specified as stakeholders.|
    |Due date|Date by which the validation task must be completed.|
