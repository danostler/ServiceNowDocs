# Assessment metric type form

Use the Assessment Metric Type form to create an assessment questionnaire template instead of using the default template for Operational vulnerability.

## Assessment metric type form

For a description of the fields, see the following table:

|Field|Description|
|-----|-----------|
|Name|Name of the assessment template.|
|Assessment duration|Length of time allowed for completing the assessment from the time the assessment is generated.|
|Table|Table this assessment template applies to.|
|State|Status of the assessment. This field is automatically set to **Draft**. After creating the assessment template, ensure that you publish it and the state is set to **Published**. Or else, you cannot generate the instance.|
|Enforce condition|Option to indicate that the Enforce condition is active or inactive.|
|Roles|Roles that have read access to the template.|
|Scale factor|Scale value for all assessment results.|
|Condition|Filter conditions for the assessment template.|
|Description|Purpose of the assessment template and the reason for its creation.|
|Related lists|
|Metric categories|All metric categories for this assessment. For more information on the Metric category form, see .|
|Assessable records|All assessable records associated to the metric category. For more information on the Assessable records, see .|
