# Element variable record form

Use the Element variable record form to configure an element variable. Element variables are custom elements that are used in the Grid configuration. You can set up an element variable that is specific custom columns, which are required for a particular dependency of an element.

## Element variable record form

For description of the field values, see the table.

|Field|Description|
|-----|-----------|
|Model|Element definition \[sn\_bcm\_element\_definition\] reference. You can create multiple element variables for an element definition.|
|Type|Field class in the model table.|
|Label|Display name in the dependency group of the BIA dependency grid.|
|Column name|Name of the column in the table.|
|Application|Application that contains the record.|
|Active|Option to activate the element variable.|
|Read only|Option to render the element variable as read only.|
|Display|Option to display the element variable.|
|Enable reporting|Option to report the element variable.|
|Default Value|
|Default value|Description for the default value of the element variable.|
