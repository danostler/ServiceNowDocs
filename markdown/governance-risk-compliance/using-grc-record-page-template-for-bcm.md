# GRC record page template for BCM records

Beginning with the Australia release, the GRC records use the common GRC record page template. You can leverage the new variant of record pages to streamline page creation, simplify maintenance, and minimize the cost of page ownership.

With standard record page migration, you can configure consistent styling and a uniform look for all BCM records. You can also configure the header of the record to display additional information such as priority, due date, issue source, issue type, and so on.
