# Edge configuration form

Use the Edge configuration form to configure the Edge configurations settings such as the label, tooltip, default edge type for the edge of the selected node.

## Edge configuration new record form

|Field|Description|
|-----|-----------|
|Nexus map configuration|Nexus map configuration record for which the edge is being configured. For example, OpRes resilience map UI \(Bottom to Top\).|
|Node relationship configuration|Node relationship configuration for which the edge is being configured. For example, Entity ← Entity.|
|Source table|Source table for which the edge is being configured. For example, Entity \[sn\_grc\_profile\].|
|Target table|Target table for which the edge is being configured. For example, Entity \[sn\_grc\_profile\].|
|Relationship table|Relationship table for the source and target tables. For example, Entity hierarchy \[sn\_grc\_m2m\_profile\_profile\]|
|Active|Option to mark the Edge configuration as active.|
|Label|Label to be displayed for the edge configuration once the conditions are satisfied, for example, Class.|
|Tooltip|Tooltip to be used for the edge configuration, for example, Domain created by.|
|Default edge type|Default edge type for the edge configuration, for example, Solid.|
