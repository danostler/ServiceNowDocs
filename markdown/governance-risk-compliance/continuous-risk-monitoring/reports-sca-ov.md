# SCA Overview dashboard

The SCA Overview dashboard enables you to view and manage control assessments as they occur.

## Accessing the dashboard

Users with the following roles can access the dashboard:

-   CAM Administrator \(sn\_irm\_cont\_auth.admin\)
-   Executive Reader \(sn\_irm\_cont\_auth.executive\_read\)
-   Security Control Assessor \(sn\_irm\_cont\_auth.sec\_control\_assessor\)

To open the dashboard, navigate to **All** &gt; **Continuous Authorization &amp; Monitoring** &gt; **Analytics Dashboards** &gt; **SCA Overview**.

\[Omitted image "cam-sca-overview.png"\] Alt text: SCA Overview dashboard displaying the different reports.

## Reports on the SCA Overview dashboard

|Report|Description|
|------|-----------|
|Adequate Engagements|The number of engagements closed with an adequate result.|
|Satisfactory Engagements|The number of engagements closed with a satisfactory result.|
|Inadequate Engagements|The number of engagements closed with an inadequate result.|
|Controls by Engagement|The number of controls categorized by engagements.|
|Audit Task Breakdown|Audit tasks grouped or stacked, or both, by task type, assigned user, top task, or state.|
|POA&amp;M Breakdown|POA&amp;Ms grouped or stacked, or both, by engagement, state, or response.|
|Control Test Results|The number of completed control tests, broken down by overall control effectiveness rating.|
|Engagement Results|An overall count of audit engagements conducted for each entity. The chart is stacked to display the overall audit results for each entity.|
|Overdue Audit Tasks|List of open audit tasks that have exceeded the planned end date.|
