# Assign Continuous Authorization and Monitoring roles to users and groups

Assign roles to users and groups to prepare to use the CAM application.

## Before you begin

Role required: admin

## About this task



## Procedure

1.  Navigate to **All** &gt; **User Administration** &gt; **Users**.

2.  Select the name of a user.

3.  Select the **Roles** tab and then select **Edit**.

4.  Move the roles that you want to assign to the user from the Collection side to the Roles List, then select **Save**.

    Roles are describe in [CAM user roles](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/governance-risk-compliance/continuous-risk-monitoring/cam-roles-list.md).

5.  Repeat the process for each user or group.

    **Note:** If the Employee Operator plugin is installed, then each of these roles \(such as Authorization Official, Executive Reader, and Reader\) is classified as a lite role.
