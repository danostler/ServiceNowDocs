# Data Relationships Framework properties form

Use the Data Relationships Framework properties form to configure the relationship properties for Data Relationships Framework APIs.

## Data Relationships Framework properties form

<table id="table_wmt_yzg_rzb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

The logging level for Data Relationships Framework

</td><td>

Logging level for the Data Relationships Framework. Available options are:-   **Debug**
-   **Info**
-   **Warn**
-   **Error**

 The default value is Warn.

</td></tr><tr><td>

Maximum number of nodes to be processed

</td><td>

Maximum number of nodes to be processed when the Data Relationships Framework API is called.

 The default value is 10,000. For example, when the API is called, it displays the results for a maximum 10,000 nodes.

</td></tr><tr><td>

Maximum number of levels to be processed

</td><td>

Maximum number of levels to be processed when the Data Relationships Framework API is called.

 The default value is 5. For example, when the API is called, it processes maximum 5 levels of the relationships and does not go deeper than 5 levels.

</td></tr></tbody>
