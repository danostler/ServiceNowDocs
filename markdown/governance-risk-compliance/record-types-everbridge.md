# Import record types from Everbridge

Import the record types for your organization ID from Everbridge and use them to create contacts.

## Before you begin

Role required: BCM admin \(sn\_bcm.admin\)

## About this task

Record types are information stored in Everbridge which are required to create contacts.

## Procedure

1.  Navigate to **Business Continuity** &gt; **Everbridge notification Configuration** &gt; **Record Types**.

2.  Click **Import**.

    Record types for your organization ID with the category are imported from Everbridge. If successfully imported you get a message of confirmation, otherwise an error message.
