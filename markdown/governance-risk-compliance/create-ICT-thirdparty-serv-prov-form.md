# Create New ICT third-party service provider form

On the Create New ICT third-party service provider form, fill in the fields.

<table id="table_yrz_2qh_ddc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Type of code to identify the ICT service provider

</td><td>

Type of code that is used to identify the ICT third-party service provider. The available options are as follows:

-   **LEI**
-   **EUID**
-   **Country Code\_CRN**: Company Registration Number
-   **Country Code\_VAT**: Value-added Tax Identification number
-   **Country Code\_PNR**: Passenger Name Record
-   **Country Code\_NIN**: National Identification Number

</td></tr><tr><td>

Identification code of ICT third-party service provider

</td><td>

Identification code of the ICT third-party service provider.

</td></tr><tr><td>

Type of additional identification code to identify the ICT service provider

</td><td>

Type of additional identification code to identify the ICT third-party service provider. The available options are as follows:

-   **LEI**

: The last two digits of the LEI must be valid MOD 97-10 check digits per ISO 17442. The system rejects submissions with an invalid checksum.

-   **EUID**
-   **Country Code\_CRN**: Company Registration Number
-   **Country Code\_VAT**: Value-added Tax Identification number
-   **Country Code\_PNR**: Passenger Name Record
-   **Country Code\_NIN**: National Identification Number

</td></tr><tr><td>

Additional identification code of ICT service provider

</td><td>

Additional identification code to identify the ICT third-party service provider.

</td></tr><tr><td>

Type of person of the ICT service provider

</td><td>

Type of person of the ICT third-party service provider. The available options are as follows:-   **Legal person, excluding individual acting in a business capacity**
-   **Individual acting in a business capacity**

</td></tr><tr><td>

Country of the ICT third-party service provider’s headquarters

</td><td>

Country where the ICT third-party service provider’s headquarters is located.

</td></tr><tr><td>

Legal name of the ICT service provider

</td><td>

Legal name of the ICT third-party service provider.

</td></tr><tr><td>

Name of the ICT service provider

</td><td>

Name of the ICT third-party service provider. This field can list a commonly used brand name or trading name that the ICT third-party service provider operates under, which might not exactly match the legal name.

</td></tr><tr><td>

Currency

</td><td>

Currency of the country where the ICT third-party service provider’s headquarters is located.

</td></tr><tr><td>

Total annual expense or estimated cost of the ICT third-party service provider

</td><td>

Total annual expense or estimated cost of the ICT third-party service provider.

</td></tr><tr><td>

Is the third party the ultimate parent?

</td><td>

Option to confirm whether the third party the ultimate third-party. The available options are as follows:

-   **Yes**: The third party itself is the parent.
-   **No**: Default option.

</td></tr><tr><td>

Identification code of the ICT service provider’s ultimate parent undertaking

</td><td>

Reference field for the identification code of the ICT third-party service provider and name of the ICT third-party service provider.

 If the selection for the **Is the third party the ultimate parent?** field is **No**, choose the system populated identification code of the ICT third-party service provider other than the ultimate parent. If you choose the identification code of the ICT third-party service provider as that of the ultimate parent, an error message is displayed: `Identification code of the ICT third-party service provider and identification code of the ICT third-party service provider's ultimate parent must be different.`

 If the selection for the **Is the third party the ultimate parent?** field is **Yes**, the system auto-populates the identification code of the ICT third-party service provider.

</td></tr><tr><td>

Type of code to identify the ICT third-party service provider’s ultimate parent

</td><td>

Type of code that is used to identify the ICT third-party service provider’s ultimate parent. The available options are as follows:

-   **LEI**
-   **EUID**
-   **Country Code\_CRN**: Company Registration Number
-   **Country Code\_VAT**: Value-added Tax Identification number
-   **Country Code\_PNR**: Passenger Name Record
-   **Country Code\_NIN**: National Identification Number

</td></tr><tr><td>

Type of ICT services

</td><td>

Type of ICT services. The available options are as follows:-   **ICT project management**
-   **ICT Development**
-   **ICT help desk and first level support**
-   **ICT security management services**
-   **Provision of data**
-   **Data analysis**
-   **ICT, facilities and hosting services \(excluding Cloud services\)**
-   **Computation**
-   **Non-Cloud Data storage**
-   **Telecom carrier**
-   **Network infrastructure**
-   **Hardware and physical devices**
-   **Software licensing \(excluding SaaS\)**
-   **ICT operation management \(including maintenance\)**
-   **ICT Consulting**
-   **ICT Risk management**
-   **Cloud services: IaaS**
-   **Cloud services: PaaS**
-   **Cloud services: SaaS**

</td></tr></tbody>
