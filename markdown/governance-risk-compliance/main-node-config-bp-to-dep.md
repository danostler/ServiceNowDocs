# Business process to dependencies Main node configuration

The Business process to dependencies Main node configuration defines the relationships from a Business process to its dependencies.

For the Business process to dependencies Main node configuration, the source is OpRes CMDB and the table is Business Process \[cmdb\_ci\_business\_process\].

## Node relationship configurations

You can configure the node relationships such as the sequence, source table, relationship table, target table, and active flag in the Business process to dependencies Main node configuration as shown in the example.
