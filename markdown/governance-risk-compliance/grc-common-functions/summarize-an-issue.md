# Summarize an issue using Now Assist for Integrated Risk Management \(IRM\)

Generate an issue summary using the issue summarization skill for a concise overview of an issue record. Summaries help provide essential details for context and reduce the effort required to create summaries manually.

## Before you begin

Install the Now Assist for IRM application to generate issue summaries. For more information, see [Now Assist for Integrated Risk Management \(IRM\)](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/governance-risk-compliance/grc-common-functions/now-assist-for-irm.md).

**Important:** This Now Assist skill is turned on by default. The skill will be automatically available to appropriate role users for the application. For more information, see .

Role required: sn\_grc\_genai.issue\_user and sn\_irm\_gen\_ai.user

## About this task

The intelligence of Now Assist analyzes the issue records including the description, activity log, and remediation tasks. By using the issue summarization skill, you can generate a concise summary that captures the essential information that you need.

## Procedure

1.  Navigate to **Workspaces** &gt; **Risk Workspace**.

2.  Navigate to Issues and open any New, In progress, or Closed issue.

    **Note:**

    Check your entitlements to determine whether you have access to the issue summarization skill.

3.  Select the **Summarize the issue under the Overview** tab.

    A summary of all the important details, such as the issue definition, actions taken, remediation tasks, and many more is generated.

4.  After the summary is generated, view details about it, save it, expand, or collapse it, provide feedback, copy it, or regenerate it.

    The summaries that are generated by AI might miss key details. You might prefer to review the summary for accuracy and completeness before you proceed.
