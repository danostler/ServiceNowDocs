# Create an entity type

Create an entity type for the entities in your system. Entity types enable you to find and create new entities that match a set of filter conditions. You can create a hierarchy of the entity types within the entity classes.

## Before you begin

Roles required: sn\_grc.manager, sn\_risk\_workspace.IT\_risk\_manager, and sn\_risk\_workspace.operational\_risk\_manager

## Procedure

1.  Navigate to the desired workspace, go to the List view, and click **Entity types**.

2.  Click **New**.

    A Create New Entity Type page with the **Details** tab is displayed.

3.  In the **Details** tab, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Name|Name of the entity type.|
    |Compliance Score Percentage|Compliance score percentage assigned to the entity type.|
    |Active|Option to set the entity type as active.|
    |Description|Description of the entity type.|

4.  To save the entity type, click **Save**.

    When you save the entity type, the other related lists are displayed on the Entity type page. Update the related lists as applicable.

    -   Details
    -   Entities
    -   Entity Filters
    -   Risk Frameworks
    -   Risk Statements
    -   Policies
    -   Control Objectives
    -   Policy Exceptions
    -   Content References
    -   Privacy Assessments
    See [An entity in the workspace view](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/governance-risk-compliance/grc-common-functions/entities-in-risk-ws.md) for more information.

5.  To save the entity type, click **Save**.
