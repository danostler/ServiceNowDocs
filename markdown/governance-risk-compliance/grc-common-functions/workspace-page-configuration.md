# Workspace page configuration

A common record page can be used and configured within all the GRC applications.

A common record page template is introduced to customize tables in any GRC workspaces. You can configure the record page to show icons, separator, and stepper component. For more information on how to configure the workspace, see [Configure workspace page](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/governance-risk-compliance/grc-common-functions/configure-workspace-page.md) topic.

-   **[Configure workspace page](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/governance-risk-compliance/grc-common-functions/configure-workspace-page.md)**  
You can configure and customize your workspaces so that all the GRC applications have a similar appearance.
