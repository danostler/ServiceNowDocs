# Download and activate the Regulatory Agency Library plugin

You can activate the GRC: Regulatory Agency Library application if you have the admin role. This application includes demo data and provides you access to regulatory body profiles.

## Before you begin

In ServiceNow Store, you must confirm that you have entitlements \(or licenses\) to the application and its dependent applications.

Role required: admin

## Procedure

1.  Navigate to **System Applications** &gt; **All Available Applications** &gt; **All** &gt; ****.

2.  Find the Regulatory Agency Library application by using the filter criteria and search bar.

    You can search for the application by its name or ID. If you cannot find an application, you may have to request it from ServiceNow Store.

    Visit the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) website to view all the available apps and for information about submitting requests to the store. For cumulative release notes information for all released apps, see the [ServiceNow Store version history release notes](https://www.servicenow.com/docs/bundle/store-release-notes/page/release-notes/store/sn-store-release-notes.html).

3.  Select **Install**.

4.  In the Application installation dialog box, review the application dependencies.

    If your application requires other applications, activate them first if they aren’t already installed.

    Installing your application automatically installs the dependent applications if they aren’t installed already.

5.  If demo data is available and you want to load it, select **Load demo data**.

    Some applications include demo data, which are sample records that describe application features for common use cases. Load demo data when you first activate the application on a development or test instance.

    **Note:**

    If you don’t select the **Load demo data** check box for a store application during installation, demo data isn’t available to activate from the **Application Manager** later. For information on how to activate or reactivate demo data after the initial installation, see the [Workaround to install demo data if application is already installed \[KB0722909\]](https://support.servicenow.com/kb?id=kb_article_view&sysparm_article=KB0722909) article in the knowledge base.

6.  Select **Install**.
