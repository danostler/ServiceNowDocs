# Define the scripted variables

Define data for the scripted variables using the Template Configurations module for creating a report template in the Audit application. You can configure the data in either text or HTML format.

## Before you begin

Role required: sn\_grc\_doc\_design.developer

## Procedure

1.  Navigate to **All** &gt; **Audit** &gt; **Audit Report** &gt; **Template Configurations**.

2.  Open the template for which you want to create scripted variables.

3.  Select the **Scripted variables** tab.

    \[Omitted image "scripted-variable.png"\] Alt text:

4.  In the Scripted variables related list, select **New** to create a scripted variable.

    \[Omitted image "new-scripted-variable.png"\] Alt text:

5.  In the **Name** field, provide a descriptive name for the template.

6.  From the **Type** drop-down list, select either Text or HTML based on your requirement.

7.  Turn on the **Script** toggle to enable custom script for the scripted variable.

8.  Select **Submit**.

    The Scripted variables for the template appear in the related list.
