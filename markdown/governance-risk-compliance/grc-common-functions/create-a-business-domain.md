# Create a business domain

Create a business domain to define the context in which data is imported into a Microsoft Word document using Document designer.

## Before you begin

Role required: sn\_business\_domain.admin

## Procedure

1.  Enter `sn_grc_business_domain.LIST` in the filter navigator.

2.  Select **New**.

3.  Enter a name for the business domain in the **Name** field.

4.  Select **Submit**.


## What to do next

Configure role mappings for the business domain. For more information, see [Configure business domain roles for Document designer](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/governance-risk-compliance/grc-common-functions/configure-business-domain-roles-for-document-designer.md).
