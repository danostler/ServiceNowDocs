# Issue Summarization skill

Use the Issue Summarization skill to view a concise summary of an issue based on its life-cycle data. The skill analyzes the issue’s context and generates a short, readable overview that includes key details such as status, actions, and relevant metadata. This helps you quickly understand the issue without reviewing the full record.

## Understanding Issue Summarization skill

You can trigger draft AI-generated responses for smart assessment while responding, review the proposed answers, and choose whether to accept, modify, or discard them. This maintains human oversight and allows for customization as needed. After the responses are created an AI summary becomes available which provides a consolidated view of all AI-generated suggestions for an assessment.

When the AI generated responses are added the system ensures:

-   If a question has already been answered manually or through other means, the AI will not override the existing response. This ensures that user input is preserved and respected.
-   In collaborative assessments, each user can generate AI suggestions only once. These suggestions are visible only to the user who generated them. Other collaborators will not see the AI summary or suggestions generated by their peers.

## Benefits of Issue Summarization skill

-   Reduces time spent reviewing detailed issue records by providing concise AI-generated summaries.
-   Helps you quickly understand issue context, including key details and actions taken.
-   Improves clarity and standardization in how issue information is presented.
-   Enables faster decision-making by surfacing the most relevant information early in the process.
