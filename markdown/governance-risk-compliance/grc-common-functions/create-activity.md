# Create activity

Create a cybersecurity activity for a target.

## Before you begin

Policy and Compliance Management Role required: admin

NIST CSF Role required: sn\_irm\_nist\_csf.risk\_executive or sn\_irm\_nist\_csf.security\_officer or sn\_irm\_nist\_csf.control\_provider

## Procedure

1.  Navigate to **All** &gt; **NIST CSF** &gt; **Framework Profiling** &gt; **Create Activity** and create a new record \(see table for field descriptions\).

<table id="table_s3w_ftb_qhb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Number that identifies the activity.

</td></tr><tr><td>

Name of the target that has the activity associated with it.

</td></tr><tr><td>

-   Protect
-   Detect
-   Respond
-   Recover
-   Govern


</td></tr><tr><td>

Category

</td><td>

</td></tr><tr><td>

</td><td>

</td></tr></tbody>
</table>2.  Click **Submit**.

    **Note:** Every time a cybersecurity activity is created, the system validates the combination of target, function, and category for uniqueness.
