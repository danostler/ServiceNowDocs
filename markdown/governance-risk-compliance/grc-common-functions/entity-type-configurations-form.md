# Entity Type Configurations form

Use the Entity Type Configurations form to create an entity type configuration within the Entity Based Access application.

|Field|Description|
|-----|-----------|
|Configuration type|Type for entity access configuration such as Entity type.|
|Entity type|Name of the entity type.|
|Active|Option to enable the entity type configuration.|
|Users|Users that the access is configured for.|
|User groups|User groups that the access is configured for.|
|Entity user fields|User fields from the entity table \[sn\_grc\_profile\] that the access should be granted to.|
|Entity group fields|User group fields from the entity table \[sn\_grc\_profile\] that the access should be granted to.|
|Description|Description of the entity type configuration.|
