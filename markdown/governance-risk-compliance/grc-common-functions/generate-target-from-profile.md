# Generate target from profile or entity type

Generate a target record for a profile to track NIST RMF attributes.

## Before you begin

Role required: admin.

**Note:** Starting with version 10.1.0, the NIST RMF Use Case Accelerator will be supported only for customers who currently use the product. New and existing customers should consider using the GRC: Continuous Authorization Monitoring application. For details, [Continuous Authorization and Monitoring](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/governance-risk-compliance/continuous-risk-monitoring/grc-cam-landing-page.md).

Policy and Compliance Management

NIST RMF role required: sn\_irm\_nist\_rmf.risk\_executive or sn\_irm\_nist\_rmf.security\_officer

## Procedure

1.  Navigate to one of the following locations:

    -   **Policy and Compliance** &gt; **Scoping** &gt; **Profile Types**
    -   **Policy and Compliance** &gt; **Scoping** &gt; **My Profile**
    -   **Policy and Compliance** &gt; **Scoping** &gt; **All Profiles**
2.  Select and open Profile Type of Profile record.

3.  Check **Generate Target** and click **Update**.

4.  Identify the target:

    1.  Navigate to **NIST RMF** &gt; **Categorize** &gt; **Impact Analysis**.

    2.  Search for the target using the **Name** or **Profile** fields.

    3.  Confirm that the target appears in the list.
