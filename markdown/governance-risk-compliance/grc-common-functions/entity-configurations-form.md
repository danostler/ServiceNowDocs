# Entity configurations form

Use the Entity configurations form to configure access to the objects through an entity.

<table id="table_qkq_4dw_qbc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Configuration type

</td><td>

Type of entity access configuration, such as Entity.

</td></tr><tr><td>

Entity

</td><td>

Name of the entity that entity-based access is configured for.

</td></tr><tr><td>

Applicability

</td><td>

Option to define the applicability of the configuration. You can configure entity access for the selected entity only or you can configure entity access for the selected entity and its downstream entities. Available options are:-   **Selected entity only**
-   **Selected entity and downstream entities**

</td></tr><tr><td>

Active

</td><td>

Option to enable or disable the entity configuration.

</td></tr><tr><td>

Users

</td><td>

Users that the access is configured for.

</td></tr><tr><td>

User groups

</td><td>

User groups that the access is configured for.

</td></tr><tr><td>

Entity user fields

</td><td>

User fields on the Entity table \[sn\_grc\_profile\] that the access should be granted to.

</td></tr><tr><td>

Entity group fields

</td><td>

User group fields on the Entity table \[sn\_grc\_profile\] that the access should be granted to.

</td></tr><tr><td>

Description

</td><td>

Description of the entity configuration.

</td></tr></tbody>
