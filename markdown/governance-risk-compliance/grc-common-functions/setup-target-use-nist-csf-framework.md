# Set up target for NIST CSF framework

Set up a target record to use with NIST CSF framework after you've identified a target.

## Before you begin

NIST CSF Role required: none. sn\_irm\_nist\_csf.risk\_executive or sn\_irm\_nist\_csf.security\_officer

## Procedure

1.  Navigate to **All** &gt; **NIST CSF** &gt; **Framework Profiling** &gt; **Orient Targets**.

2.  Search for the target using the **Name** or **Profile** fields.

3.  Open the target record.

4.  Click the **Framework** field and select **NIST CSF**.

5.  Click **Update**.

6.  In the Targets list, open the record again.

7.  Verify the information in the Target form.

8.  In the NIST CSF section, fill out the fields \(see table for field descriptions\).

    |Field|Description|
    |-----|-----------|
    |Critical|Check box that indicates the target is focused on during cyber risks. NIST CSF recommends that organizations prioritize cybersecurity activities for critical targets|
    |Tier|Tier for a target that describes its capability to address cyber risks.|

9.  Click the **Activities** related list and validate the activities that are associated with the target.
