# Reference information for AI reporting assistant

Use this reference information to configure and manage AI reporting assistant functionality in Microsoft Word documents.

## AI reporting assistant properties

|Property|Description|
|--------|-----------|
|sn\_doc\_design\_ai.chart\_default\_color\_scheme|Name of the pa\_chart\_color\_schemes record used as the default chart color palette. The default value is Default.|
|sn\_doc\_design\_ai.table\_max\_columns|Maximum number of columns allowed in a generated table. The default value is 10.|
|sn\_doc\_design\_ai.table\_max\_rows|Maximum number of data rows included in a generated table, excluding the header row. The default value is 200.|

## AI reporting configurations

|Field|Description|
|-----|-----------|
|Business domain|The business domain to which the knowledge graph tags are mapped.|
|Knowledge graph tags|One or more knowledge graph tags used to improve AI reporting assistant query accuracy.|
|Active|Indicates whether the configuration is active.|
