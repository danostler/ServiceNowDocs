# Confidentiality configuration form

The Confidentiality configuration form is used for the configuration of the confidential records for your GRC tables.

<table id="table_pkk_zwt_d5b"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Table

</td><td>

Table that you want to add the Confidentiality feature in.

</td></tr><tr><td>

Field for confidentiality

</td><td>

Boolean field where you can enable for confidentiality in.

</td></tr><tr><td>

Field for Allowed users

</td><td>

Field where you can add the allowed users who can access the confidential records.

</td></tr><tr><td>

Field for Allowed groups

</td><td>

Field where you can add the user groups that are allowed to access the confidential records.

</td></tr><tr><td>

Populate allowed users from

</td><td>

Field where the allowed users are automatically filled in based on the list of columns in the GRC tables.

</td></tr><tr><td>

Populate allowed groups from

</td><td>

Field where the allowed user groups are automatically filled in based on the list of columns in the GRC tables.

</td></tr></tbody>
</table>**Note:** **Populate allowed users from** and **Populate allowed groups from** fields appear only when the **Auto-populate** field is selected.
