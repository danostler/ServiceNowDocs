# Assign an approval level for the approval configuration record

Assign one or more approval levels for the approval configuration record to support each step of approval.

## Before you begin

Role required: sn\_grc\_appr.admin

## Procedure

1.  Navigate to **All** &gt; **Approval Configurator** &gt; **Approval Configurations**.

2.  Click an approval configuration record in the list to set up the approval levels.

3.  Click **New** in the Approval Levels section.

4.  On the form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Name|Name of the approval level.|
    |Level|Stage of approval.|

    **Note:** You cannot have more than one approval level for an approval configuration with the same level.

5.  Click **Submit**.
