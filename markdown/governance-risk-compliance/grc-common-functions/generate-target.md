# Generate a target for an entity

Generate a target record for an entity to track NIST CSF attributes for that entity.

## Before you begin

Role required: admin, sn\_irm\_nist\_csf.risk\_executive or sn\_irm\_nist\_csf.security\_officer

## Procedure

1.  Generate a target for an entry.

    1.  Navigate to **Policy and Compliance** &gt; **Scoping** and select one of the following modules:

        -   **Entity Types**
        -   **My Entities**
        -   **All Entities**
    2.  Select one of the records.

    3.  From the Entity form, click **Generate target**.

        **Note:** When you click Generate target, it generates the target only if the entity has not been used for other frameworks, such as GDPR DPIA.

    4.  Click **Update**.

2.  Identify the target.

    1.  Navigate to **NIST CSF** &gt; **Framework Profiling** &gt; **Orient Targets**.

    2.  Search for the target using the **Name** or **Profile** fields.

    3.  Confirm that the target appears in the list.
