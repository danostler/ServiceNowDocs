# Activate the recommendation for similar control objectives skill

Activate and then configure the recommendation for a similar control objective skill under rationalization from Now Assist to generate recommendations for control objectives that are similar to the selected control objective.

## Before you begin

Role required: sn\_nowassist\_admin.nsa\_admin

## About this task

To recommend similar control objectives, predefined similarity parameters are used. The AI searches for similar names and descriptions to provide accurate recommendations. Once the skill is activated, the **Rationalize** button becomes available on the control objective detail page for users with the appropriate access.

## Procedure

1.  Navigate to **All** &gt; **Admin Center** &gt; **Now Assist Admin** &gt; **Features**.

2.  On the **Now Assist Features** tab, under the Technology workflow group, select **Risk &amp; Sustainability**.

    The Now Assist Skills for IRM page lists the available generative AI capabilities for Policy and Compliance.

3.  Under the Recommendation for similar control objective skill, select **Activate Skill**.

4.  Select **Save and Continue** on the **Similarity parameter** tab which is displayed in read-only mode.

    The default inputs specify the details that the Gen AI searches for, to provide recommendations. You can also review the **General Details** tab which is also in read-only mode if necessary.

5.  Select **Save and continue**.

6.  On the **Review and Activate** tab, select **Review and activate**.

    A skill activation success message is displayed.
