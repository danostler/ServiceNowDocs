# User roles for recommendation contexts

Users with different roles can read, create, or edit recommendation contexts for a regulatory alert.

|User role|Description|
|---------|-----------|
|sn\_reco\_template.recommendation\_config\_reader|Users with this role can read the recommendation configurations.|
|sn\_reco\_template.recommendation\_config\_writer|Users with this role can create and edit the recommendation configurations.|
|sn\_reco\_template.recommendation\_reader|Users with this role can read the recommendation templates.|
|sn\_reco\_template.recommendation\_writer|User with this role can create and edit recommendation templates.|
