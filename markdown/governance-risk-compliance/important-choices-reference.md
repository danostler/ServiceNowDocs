# GRC Choices form

Use the GRC Choices form to set up the rating of the assessment by updating its criticality and the order of the choice in the Important choices module.

## GRC Choices form

For a description of the field values, see the following table.

<table id="table_FloorForm"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Set

</td><td>

View that is set to Operational Resilience.

</td></tr><tr><td>

Name

</td><td>

Name of the assessment rating. For example, **1- Most Critical**.

</td></tr><tr><td>

Label

</td><td>

Label of the assessment rating. For example, **1- Most Critical**.

</td></tr><tr><td>

Choice category

</td><td>

Choice category. It is set to **Assessment Rating** by default.

</td></tr><tr><td>

Order

</td><td>

Field used for sorting the order of the assessment rating. Based on the Importance and Impact Assessment score, the application selects the corresponding rating.

\[Omitted image "or-rating-scales.png"\] Alt text: Rating scales.

</td></tr><tr><td>

Parent

</td><td>

Parent setting that is associated with the assessment rating. For example, the **Technology** pillar can be associated with the assessment rating.

</td></tr><tr><td>

Active

</td><td>

Indicates whether the configuration record is currently active and available for selection in the GRC Choice table.

</td></tr></tbody>
</table>
