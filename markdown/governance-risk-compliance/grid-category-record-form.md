# Grid category record form

Use the Grid category record form to configure a grid category such as Dependency Assessment for the grid configuration by using BCM UIB Workspace.

## Grid category record form

For description of the field values, see the table.

|Field|Description|
|-----|-----------|
|Name|Name of the grid category.|
|Code|Code for the grid category.|
|Enable element context|Option to enable the element context for the grid category.|
