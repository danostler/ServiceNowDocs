# Rating Scale New Record form

Use the Rating Scale New Record form to set up the default importance rating scale to define the parameters of the rating scale for your business services.

## Rating Scale New Record form

For a description of the field values, see the following table.

Field

Importance

</td><td>

Importance scale for the service. Available options are as follows:-   **1- Most critical**
-   **2- Somewhat critical**
-   **3- Less critical**
-   **4- Not critical**

</td></tr><tr><td>

Color

</td><td>

Color for the selected tolerance rating. Available options are:-   none
-   blue
-   brown
-   critical
-   gray
-   green
-   green-yellow
-   high
-   info
-   low
-   magenta
-   moderate
-   orange
-   pink
-   positive
-   purple
-   teal
-   warning
-   yellow

</td></tr><tr><td>

Order

</td><td>

Order for the defined importance and tolerance duration.

</td></tr><tr><td>

Minimum

</td><td>

Minimum score for the tolerance level of the disruption.

</td></tr><tr><td>

Maximum

</td><td>

Maximum score for the tolerance level of the disruption.

</td></tr><tr><td>

Impact tolerance

</td><td>

Impact tolerance duration in days as per the defined importance of the service.

</td></tr><tr><td>

Impact tolerance icon

</td><td>
