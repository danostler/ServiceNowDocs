# Set up the Document Template

Set up the Document Template \(HTML Template\) that is used for generating the PDF of the Operational vulnerability record.

## Before you begin

Role required: sn\_oper\_res.admin

## Procedure

1.  Navigate to **All** &gt; **Operational Vulnerability** &gt; **Document Templates**.

2.  To create the document template, select **New**.

3.  On the HTML Template record form, fill in the fields.

    For more information on the fields, see [HTML Template form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/governance-risk-compliance/html-template-form-op-vul.md).

4.  Select **Submit**.

5.  To insert date variables in the PDF, select **Insert Date**.

    You can use the button multiple times to add different types of dates based on the variables you choose.

6.  To insert the table of contents in the PDF, select **Insert Table of Contents**.

7.  To add blocks for advanced search, select **Add Blocks**.
