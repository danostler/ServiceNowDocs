# Request a policy extension through Service Portal

From the ServiceNow® Portal, anyone who has requested a policy exception can request an extension before the policy deadline.

## Before you begin

Role required: sn\_grc.business\_user, sn\_grc.business\_user\_lite

Add the My Policy Exceptions widget to the Service Portal index page. The widget is visible only if there is at least one policy exception in your list.

## Procedure

1.  Click **My Policy Exceptions** in the Service Portal.

2.  Open the policy exception for which you want to request for extension.

    The number of remaining extensions that you can request is displayed.

    Set the number of extensions allowed for a policy exception in the **Number of extensions allowed for a policy exception** field on the **Policy and Compliance Properties** page.

3.  In the **Request extension** section, enter the date until which you want to extend.

    A note displays the remaining number of extensions that you can request.

4.  Click **Request**.


## Result

The extension request is sent for approval to the approving authority.
