# Scenario analysis response task form

Use the Scenario analysis response task form in Operational Resilience Workspace to add the details about the response task that is associated with a scenario event.

## Scenario analysis response task form

For a description of the field values, see the following table.

|Field|Description|
|-----|-----------|
|Scenario Analysis Response panel|
|Number|Number of the scenario analysis response task such as SAR0001006. This field is auto-filled.|
|Role|Role of the assignee of the response task.|
|Instruction|Instruction for the assignee of the response task.|
|Impact panel|
|Impact duration|Impact duration of the scenario event in Days, Hours, Minutes, and Seconds.|
|Notes panel|
|Observations|Observations about the scenario event that are added by the assignee of the response task.|
|Gaps|Gaps or inconsistencies about the scenario event that are added by the assignee of the response task.|
|Recommendations|Recommendations about the scenario event that are added by the assignee of the response task.|
|Scenario analysis panel|
|Scenario analysis|Name of the scenario analysis.|
|Pillar|Pillars that are associated with the scenario analysis.|
|Description|Description of the scenario analysis.|
|Goal|Goal for performing the scenario analysis.|
