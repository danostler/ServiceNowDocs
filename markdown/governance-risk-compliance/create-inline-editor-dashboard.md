# Create an inline-editor dashboard

Create a dashboard using the inline-editor for standard analytical reporting that non-technical users can manage and share across pages and workspaces.

## Before you begin

Role required: sn\_bcm.manager, sn\_bcm.user

## Procedure

1.  Navigate to **Workspaces** &gt; **Business Continuity Workspace**.

2.  Select the option to create a new dashboard using the inline-editor.

3.  Drag and drop components onto the dashboard canvas.

    You can add the following components:

    -   Saved data visualizations
    -   New charts
    -   Lists
    -   Filters
    -   Images
    -   Headings
4.  Configure the dashboard settings to control permissions and sharing.

    The dashboard is in the Business Continuity Workspace dashboard picker based on the permissions you set.

5.  Save the dashboard.


## Result

You created an inline-editor dashboard that can be shared across pages and workspaces. Business users can manage this dashboard without technical knowledge.
