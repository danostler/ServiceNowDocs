# Main node configuration form

Use the Main node configuration form to create a Main node configuration record in the Data Relationships Framework.

## Main node configuration new record form

|Field|Description|
|-----|-----------|
|Name|Name of the Main node configuration record. For example, Service to dependencies.|
|Source|Source for the Main node configuration record. By default, it is OpRes CMDB.|
|Active|Option to indicate that the record is active. The value is **True** or **False**.|
|Main node|
|Table|Table that is used to set up the relationship, for example, Configuration Item \[cmdb\_ci\].|
|Conditions|Filter conditions used to set up the relationship for the configuration record. If you want a specific object from a table, you can set up the filter conditions accordingly.|
|Properties|
|Maximum number of nodes|Maximum number of nodes that can be displayed in the map.|
|Maximum number of levels|Maximum number of levels that are set to be displayed in the map.|
