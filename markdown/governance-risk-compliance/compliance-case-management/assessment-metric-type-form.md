# Assessment Metric Type form

Use the Assessment Metric Type form to create an assessment questionnaire template by using the GRC: Compliance Case Management application.

See the following table for a description of the field values.

|Field|Description|
|-----|-----------|
|Name|Name of the assessment template.|
|Assessment duration|Length of time allowed for completing the assessment from the time that the assessment is generated.|
|Table|Table that contains the records. This field is automatically set to **GRC core case \[sn\_grc\_case\_mgmt\_core\_case\]**.|
|State|State of the assessment. This field is automatically set to **Draft**.|
|Roles|Roles who can view the results and access records that are associated with this template.|
|Description|Purpose of the assessment template and the reason for its creation.|
|Introduction|Guidance text for business users who respond to the assessment.|
