# Create an Issue form

Use the Create an Issue form in GRC: Compliance Case Management to add an issue to a compliance case, report findings or observations or compliance deficiencies, or matters that require immediate attention from the respective owners.

Apart from the from fields, there are a few actions that you can perform on a Compliance Case record.

-   Attachment- select this icon in the side-bar to add any related attachments to the Compliance Case record.
-   Compose email- compose an email.
-   Discard draft- discard a draft version of the email.
-   Send email- send the email with all the details in it.
-   Create new email- Create a new email.
-   View drafts - view the existing drafts in the system.
-   Save- save the details of the compliance case.
-   360 view- view the compliance case in a 360 view with all the relationships.
-   Export PDF- export the compliance case as a PDF.
-   Delete- delete the compliance case.
