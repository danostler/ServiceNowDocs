# View Rule form

Use the View Rule form to define the workspace and default view rules in the GRC: Compliance Case Management application.

See the following table for a description of the field values.

<table id="table_FloorForm"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Name to identify the rule.

</td></tr><tr><td>

Table

</td><td>

Table that this view rule applies to. This field is automatically set to **Compliance case \[sn\_comp\_case\_compliance\_case\]**.

</td></tr><tr><td>

View

</td><td>

View option that you’re creating a rule for:-   **Default view**: Define the form view on the Classic environment.
-   **Workspace**: Define the form view on Workspace.

</td></tr><tr><td>

Experience Restricted

</td><td>

Option to enable this view rule to one or multiple workspaces.

</td></tr><tr><td>

Application

</td><td>

Name of the application that this view rule applies to. This field is automatically set to **GRC: Compliance Case Management**.

</td></tr><tr><td>

Execution Order

</td><td>

Order in which this view rule executes.

</td></tr><tr><td>

Active

</td><td>

Option to enable the view rule.

</td></tr><tr><td class="sub-head" colspan="2">

Conditions

</td></tr><tr><td>

Roles

</td><td>

Roles that this view rule applies to.

</td></tr><tr><td>

Conditions

</td><td>

Condition builder to set conditions for the view rule.

</td></tr><tr><td class="sub-head" colspan="2">

Form Settings

</td></tr><tr><td>

Hide details &amp; UI actions

</td><td>

Option to hide the **Details** tab in the related list.

</td></tr><tr><td>

Hide Journal Input Fields

</td><td>

Option to disable the journal fields on the Workspace forms.

</td></tr><tr><td>

Hide section navigation

</td><td>

Option to disable the jump-to section navigation on the Workspace forms.

</td></tr><tr><td>

Disable section collapsing

</td><td>

Option to disable the default form section collapse.

</td></tr><tr><td class="sub-head" colspan="2">

Form Tabs

</td></tr><tr><td>

Default Tab Order

</td><td>

Option to select the default tab order on the Workspace form.

</td></tr><tr><td>

Tab Order

</td><td>

Option to define the tab order on the Workspace form. This field appears only when you deselect the **Default Tab Order** option.

</td></tr><tr><td>

Default Tab Focus

</td><td>

Item that should be in focus when you open a record in the table that is specified in this rule.
