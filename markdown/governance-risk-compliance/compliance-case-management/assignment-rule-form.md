# Assignment Rule form

Use the Assignment Rule form to define the case assignment rules in the Compliance Case Management application.

See the following table for a description of the field values.

|Field|Description|
|-----|-----------|
|Name|Name of the assignment rule.|
|Application|Name of the application that this assignment rule applies to. This field is automatically set to **GRC: Compliance Case Management**.|
|Active|Option to enable the assignment rule.|
|Applies to|
|Table|Table that this assignment rule applies to. This field is automatically set to **Compliance case \[sn\_comp\_case\_compliance\_case\]**.|
|Conditions|Condition builder to set conditions for the assignment rule.|
|Assign to|
|User|User that the case or request is assigned to.|
|Group|Group that the case or request is assigned to.|
|Script|
|Script|Script to define the advanced assignment rule.|
