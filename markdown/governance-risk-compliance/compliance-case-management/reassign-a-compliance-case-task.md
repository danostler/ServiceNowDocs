# Reassign an Assessment type Action task

Reassign an Assessment type Action task to another user in the GRC: Compliance Case Management application. For example, a case analyst or an action task owner can reassign a compliance Action task to another Action task owner if an action task is wrongly assigned or isn’t accepted by the Action task owner.

## Before you begin

Role required: sn\_compliance\_ws.corporate\_compliance\_analyst

## About this task

As a case analyst, after a task is assigned to an action task owner, you can reassign the task to another user.

As an action task owner, you can also accept the task, start working on it, and then reassign the partially completed task to another user. When you reassign a partially completed Action task to another user, the data that you provide also gets copied and is available to the new Action task owner. The newly assigned task owner can modify the Action task based on that owner's requirements.

**Note:** You can only reassign the case tasks that have the **Type** field set to **Assessment**.

## Procedure

1.  Navigate to **Self Service** &gt; **Employee Center**.

2.  Access your task from the My tasks module.

3.  Open the task that you want to work on.

4.  In the Open section, select **Reassign**.

    The Reassign Action task dialog box appears.

5.  In the **Assignment group** field, select the assignment group.

6.  In the **Action task owner** field, select the owner.

7.  In the Reassign Action task dialog box, select **Reassign**.
