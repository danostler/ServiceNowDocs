# Add a related area to a compliance case

Add an area that is related to the compliance case or event that is reported in the GRC: Compliance Case Management application.

## Before you begin

Role required: sn\_compliance\_ws.corporate\_compliance\_analyst

## About this task

You can specify the related areas for a compliance case or reported event. Related areas include the policies, citations, control objectives, control, and risk events.

## Procedure

1.  Navigate to **All** &gt; **Policy and Compliance** &gt; **Compliance Workspace**.

2.  Select the list \[Omitted image "ws-list-icon.png"\] icon.

3.  From the list, select **All Cases**.

4.  Open the case that you want to add a related area to.

5.  Select the Related areas related list.

6.  Select **Add**.

7.  Select the related area type.

    Records related to the selected record type are available for your selection.

8.  Select the records from the available list.

    Modifying the **Select type** field clears the current selection.

9.  Select **Add**.

10. Select **Save**.
