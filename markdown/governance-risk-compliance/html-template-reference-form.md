# HTML Template form

If you have the sn\_oper\_res.admin role, use the HTML Template form to create an HTML template instead of using the default HTML template.

## HTML Template form

For a description of the field values on the form, see the following table.

Related lists

</td></tr><tr><td>

Header

</td><td>

Information about the header image. The options are as follows:-   **Header image**: Click to add a header image.
-   **Header image position**: Position the header image by using the following options:
    -   **Left**: Align the header image on the left of the page.
    -   **Center**: Align the header image in the center of the page.
    -   **Right**: Align the header image on the right of the page.
-   **Header image height**: Value of the header image height in pixels.

</td></tr><tr><td>

Footer

</td><td>

Information about the footer image. The options are as follows:-   **Footer image**: Click to add a footer image.
-   **Footer image position**: Position the footer image by using the following options:
    -   **Left**: Align the footer image on the left of the page.
    -   **Center**: Align the footer image in the center of the page.
    -   **Right**: Align the footer image on the right of the page.
-   **Footer image height**: Enter the value of the footer image height.
-   **Footer text**: Add a footnote on the document.
-   **Footer text position**: Position the footer text by using the following options:
    -   **Left**: Align the footer text on the left of the page.
    -   **Center**: Align the footer text in the center of the page.
    -   **Right**: Align the footer text on the right of the page.
-   **Vertical alignment**: Align the footer text by using the following options:
    -   **Above the footer image**: Align the footer text above the footer image.
    -   **Below the footer image**: Align the footer text below the footer image.

</td></tr><tr><td>

Page

</td><td>

Information about the page. The options are as follows:-   Page size: Size of the page. The available options are as follows:
    -   **A4**: A4 page size for the HTML template.
    -   **Letter**: Letter page size for the HTML template.
    -   **Legal**: Legal page size for the HTML template.
-   **Top/Bottom margin**: Top and bottom margin dimensions for the page in pixels.
-   **Left/Right margin**: Left and right margin dimensions for the page in pixels.

</td></tr></tbody>
