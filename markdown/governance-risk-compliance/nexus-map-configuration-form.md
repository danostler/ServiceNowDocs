# Nexus map configuration form

Use the Nexus map configuration form to define the UI configuration for the selected Main node configuration. It also helps you to define the node and edge configurations, determining how the Nexus map is displayed.

## Nexus map configuration new record form

|Field|Description|
|-----|-----------|
|Name|Unique name for the Nexus map configuration. For example, OpRes resilience map.|
|Workspace type|Desired Workspace for setting up the Nexus map. If the desired Workspace isn't available in the field, you can create a workspace record by accessing the "Nexus map configuration workspace type" module through the **All** menu.|
|Main node configuration|Main node configuration record displayed by default for which the Nexus map configuration is being configured.|
|Active|Option to indicate that the Nexus map configuration is active.|
|Short description|Short description of the Nexus map configuration.|
