# Create records in bulk

Create records in bulk from the Digital resilience third-party registers rather than creating one record at a time for single or multiple entities. You can save time and effort by working on multiple records at a time.

## Before you begin

Role required: sn\_oper\_res.admin

## About this task

Only Operational Resilience administrators can view the **Upload** option on the form and create records in bulk.

## Procedure

1.  Navigate to **Workspaces** &gt; **Operational Resilience Workspace** &gt; **Digital resilience third-party registers** &gt; **Excel download/upload requests**.

2.  Select the request type as **Upload**.

    **Note:** Only Operational Resilience administrators can view the **Upload** option and perform this task.

3.  Select an appropriate record that you want to work on.

4.  Download the default template.

5.  Follow the instructions in each template to fill out the columns in the template.

6.  Create an upload request for the record type and attach the completed template from your local hard drive.

7.  Submit the request.

8.  View the results to get an overview of the records that are uploaded successfully and the records with errors.

9.  Navigate to the spreadsheet to view the errors.

    1.  Correct the errors in the Microsoft Excel spreadsheet.

    2.  Upload the record again.

10. Repeat the steps for other records.

    The records are downloaded in bulk from the Digital resilience third-party registers.
