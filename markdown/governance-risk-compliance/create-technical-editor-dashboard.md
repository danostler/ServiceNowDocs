# Create a technical editor dashboard

Create a dashboard using the technical editor when you need advanced features like data resources, scripting, or custom user navigation.

## Before you begin

Role required: admin, ui\_builder\_admin

## About this task

Confirm that your dashboard requires technical features that are not available in the inline-editor, such as data binding, client scripting, or event handling.

## Procedure

1.  Navigate to UI Builder.

2.  Create a new dashboard using the technical editor.

3.  Configure data resources and add data visualization components to the dashboard.

4.  Use dynamic data binding for component properties to connect visualizations to your data sources.

5.  Add client scripting and event handling as needed.

    You can customize user navigation and implement complex interactions between dashboard components.

6.  Set the dashboard permissions and sharing options.

    The dashboard is displayed in the workspace dashboard picker based on the configured permissions.


## Result

You created a technical editor dashboard with advanced functionality. Follow similar design patterns to inline dashboards to maintain consistency in the dashboard picker view.
