# Phase form

To configure phases as part of recovery activities, use the Phase form to set up phases for recovery and event tasks in the BCM Configurable Workspace. You can then associate these phases with recovery and event tasks, enabling you to track their progression effectively.

## Phase form

For description of the field values, see the table.

|Field|Description|
|-----|-----------|
|Name|Unique name of the phase. For example, Ready for preparation.|
|Order|Unique order assigned to the phase. You can, however, manually adjust this order using inline editing if required, ensuring that the order remains unique across all phases.|
|Active|Option to mark the phase as active or inactive.|
