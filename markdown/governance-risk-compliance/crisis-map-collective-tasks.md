# Structured workflows for Crisis map

Use the Crisis map interface in the BCM Configurable Workspace to monitor and receive alerts about potential threats to your organization's business operations. You can then manage these alerts by initiating response workflows.

BCM program manager sets controls to customize the alerts, monitors assets within impacted areas, and initiates response workflows for crisis management.

-   For more information on managing and customizing the alerts in Crisis map, see [Manage alerts from the map interface](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/governance-risk-compliance/manage-alerts-in-crisis-map-interface-uib-ws.md) and [Set controls to customize the alerts](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/governance-risk-compliance/set-controls-and-customize-alerts-on-map-uib-ws.md).
-   For more information on layer clustering and monitoring assets within impacted areas, see [Select resource layer clustering](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/governance-risk-compliance/view-resource-layer-clustering-uib-ws.md) and [Monitor assets within the impacted areas](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/governance-risk-compliance/view-assets-at-risk-within-impacted-area.md).
-   For more information on initiating response workflows for crisis management, see [Initiate the response actions](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/governance-risk-compliance/notifystakeholders-initiate-Crisis-Management-workflows-uib-ws.md).

You can report a crisis from the Home page in the BCM Configurable Workspace. For information on viewing alerts, feeds, threats, see [Crisis map interface](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/governance-risk-compliance/threats-feeds-alerts-crisis-map.md). For administrative tasks, see [Setup for Crisis map](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/governance-risk-compliance/crisis-map-admin-tasks.md).

-   **[Enhanced Performance with UIB pages](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/governance-risk-compliance/crisis-map-migration.md)**  
The Crisis map application is now built with a UI Builder \(UIB\) page. Previously, the single-use component limited the ability to customize the user interface \(UI\), add filters, or modify the design in the Crisis map. By adopting the UIB page, you can gain full control over the implementation, enabling you to extend support for your specific use cases.
-   **[Manage alerts from the map interface](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/governance-risk-compliance/manage-alerts-in-crisis-map-interface-uib-ws.md)**  
Manage alerts from the Crisis map interface to identify threats and their geo-locations relative to your assets. You can open alerts for details, dismiss them when resolved, adjust the impacted area boundaries, or undo recent changes.
-   **[Set controls to customize the alerts](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/governance-risk-compliance/set-controls-and-customize-alerts-on-map-uib-ws.md)**  
Set controls in the Crisis map interface in BCM UIB Workspace. You can then customize the alerts for assets, locations, and so on.
-   **[Select resource layer clustering](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/governance-risk-compliance/view-resource-layer-clustering-uib-ws.md)**  
Select resource layer clustering in the Crisis map within the BCM Configurable Workspace to view assets or resources on the map.
-   **[Monitor assets within the impacted areas](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/governance-risk-compliance/view-assets-at-risk-within-impacted-area.md)**  
Monitor at-risk assets in the Crisis map within the BCM Configurable Workspace when alert threats appear near your business locations. This functionality enables you to protect resources in impacted areas and help prevent significant losses to your organization.
-   **[Initiate the response actions](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/governance-risk-compliance/notifystakeholders-initiate-Crisis-Management-workflows-uib-ws.md)**  
Initiate the response workflows for your recovery teams in BCM UIB Workspace. You can then notify the stakeholders and help them respond to the threat and take the necessary actions.
