# Node configuration form

Use the Node configuration form to configure the settings such as the primary label, icon, secondary label, tooltip for the selected node.

## Node configuration new record form

|Field|Description|
|-----|-----------|
|Table|Table associated with the node configuration.|
|Nexus map configuration|Nexus map configuration record displayed by default for which the Node configuration is being configured.|
|Primary label|Primary label to be displayed for the node, for example, Name.|
|Secondary label|Secondary label to be displayed for the node, for example, class.|
|Icon|Icon to be used for the node in the main node UI, for example, briefcase-outline.|
|Tooltip|Tooltip to be displayed when hovering over the node, for example, Name.|
|Context menu options|
|Configuration for set as main node|Nexus map configuration required to enable the "Make this primary" functionality at the node level. This configuration reflects the hierarchy flow for the node that is designated as primary.|
|Configuration for 360° view|360° configuration to open the 360 view of the record when you right-select on that particular node.|
|Context record|Context record to show or consider for the side panel and other context areas when you select that particular node.|
