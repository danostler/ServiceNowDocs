# Update existing records in bulk

Update existing records in bulk from the Digital resilience third-party registers.

## Before you begin

Role required: sn\_oper\_res.admin

## About this task

Only Operational Resilience administrators can view the **Upload** option and update existing records in bulk.

## Procedure

1.  Navigate to **Workspaces** &gt; **Operational Resilience Workspace** &gt; **Digital resilience third-party registers** &gt; **Excel download/upload requests**.

2.  Select the request type as **Download**.

3.  Select the records to be updated and submit the request.

4.  Download the Microsoft Excel spreadsheet to update your local hard drive.

5.  Follow the instructions in each template to fill out the columns in the template.

6.  Create an upload request for the record type and attach the completed template from your local hard drive.

    **Note:** Only Operational Resilience administrators can view the **Upload** option and perform this task.

7.  Submit the request.

8.  Review the results to see a summary of the successfully uploaded records and those with errors.

9.  Navigate to the spreadsheet to view the errors.

    1.  Correct the errors in the Microsoft Excel spreadsheet.

    2.  Upload the record again.

10. Repeat the steps for other records.
