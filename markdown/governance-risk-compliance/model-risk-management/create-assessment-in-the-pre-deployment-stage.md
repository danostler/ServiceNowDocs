# Create assessment in the Pre-deployment stage

Create a model risk assessment by verifying the details, assigning stakeholders, and create an assessment using the Model Risk Management application in the Pre-deployment stage.

## Before you begin

Role required: Role required: sn\_model\_risk\_mgmt.model\_governance

The Model Risk Coordinator \[sn\_model\_risk\_mgmt.risk\_coordinator\] can also create assessments when they are designated as a stakeholder.

## Procedure

1.  Navigate to **All** &gt; **Model Risk Workspace**.

2.  Select the list icon \[Omitted image "list-icon-mrm.png"\] Alt text:.

3.  From the list, navigate to **Models** &gt; **Pre-deployment**.

4.  Select the model for which you want to create an assessment task.

5.  In the Pre-deployment state, select **Create assessment**.

6.  On the Model risk task dialog, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Assessment template|Predefined template that outlines the structure, questions, and scoring logic for the assessment. This field is auto-populated with the default assessment template with category as Model Risk.|
    |Assessment assignee|Individual assigned to complete the assessment task. This field is auto-populated with the model owner’s name. You can select only those users who are specified as stakeholders.|
    |Due date|Date by which the assessment task must be completed.|

7.  Select **Create**.
