# Main node configuration new record form

Use the Main node configuration new record form to create a main node configuration record in the Data Relationships Framework.

## Main node configuration new record form

<table id="table_mzj_qpq_4pb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Name of the main node configuration record. For example, BCP.

</td></tr><tr><td>

Source

</td><td>

Source for the main node configuration record. Available options are:-   **BIA upstream dependency**
-   **BIA downstream dependencies**
-   **CMDB**
-   **Plan primary assets**
-   **Plan related assets**

</td></tr><tr><td>

Active

</td><td>

Option to indicate that the record is active.

</td></tr><tr><td>

Table

</td><td>

Table used to set up the relationship. For example, Plan \[sn\_bcp\_plan\].

</td></tr><tr><td>

Condition

</td><td>

Filter condition used to set up the relationship for the configuration record. For example, you can set the condition as **"Status in source" "is" "Valid"**.

</td></tr></tbody>
