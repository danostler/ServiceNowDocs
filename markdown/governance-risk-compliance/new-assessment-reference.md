# Assessment Metric Type form

Use the Assessment Metric Type form to create an assessment questionnaire template instead of using the default template. You can also use this form to request responses from the assessor.

## Assessment Metric Type form

For a description of the field values, see the following table.

|Field|Description|
|-----|-----------|
|Name|Name of the assessment template.|
|Roles|Additional user roles that can view the results and access records that are associated with this template. Users with the specified roles have read access to this template record and to the associated categories, metrics, assessable records and scorecards, category users, stakeholders, and decision matrices.|
|Active|Option to enable the assessment template.|
|State|Status of the assessment: **Draft** or **Published**. This field is auto-filled.|
|Description|Helpful information about the assessment template. Enter a clear description of the template and its purpose.|
