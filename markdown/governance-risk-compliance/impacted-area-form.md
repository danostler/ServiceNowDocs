# Impacted area form

On the Impacted area form, fill in the fields.

<table id="table_aqy_k2x_rvb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Title

</td><td>

Title of the impacted area.

</td></tr><tr><td>

Description

</td><td>

Brief description of the impacted areas or objects.

</td></tr><tr><td>

Impacted area

</td><td>

Areas or objects that are affected by the vulnerability or event. This field is available only when a value is selected from **Impacted area type**. For example, **Company: 3Com**.

</td></tr><tr><td>

Type

</td><td>

Type of the impacted area. For example, User or Company.

</td></tr><tr><td>

Impacted area type

</td><td>

Type of the impacted area. For example, the location of the event or vulnerability. The choices are as follows:-   **Entity**
-   **Location**
-   **User**
-   **Company**

 **Note:** You can also configure your own impacted area type.

</td></tr></tbody>
</table>
