# Set up the Vulnerability Assessment Template

Create the Vulnerability Assessment Template by defining the assessment parameters for Operational vulnerability. The assessment template is based on the Metric type assessment. It helps you to define important assessment details such as the assessment duration, associated tables, filter conditions, and so on.

## Before you begin

Role required: sn\_oper\_res.admin

## Procedure

1.  Navigate to **All** &gt; **Operational Vulnerability** &gt; **Vulnerability Assessment Template**.

2.  To create the Vulnerability Assessment \(Assessment Metric Type\) template, select **New**.

3.  On the Assessment Metric Type record form, fill in the fields.

    For more information on the fields, see [Assessment metric type form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/governance-risk-compliance/op-vul-assmt-metric-type-form.md).

4.  Use the Smart Assessment template that has been added to the vulnerability type.

5.  Select **Submit**.

6.  To copy an existing assessment template, select it from the list and select **Copy**.

7.  To update an existing assessment template, select it from the list and select **Update**.

8.  To delete an existing assessment template, select it from the list and select **Delete**.
