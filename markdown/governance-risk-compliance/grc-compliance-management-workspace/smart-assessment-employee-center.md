# Respond to attestations from the Employee Center

Respond to your attestations by logging in to the Employee Center.

## Before you begin

Role required: sn\_grc.business\_user, sn\_grc.business\_user\_lite

## About this task

As an attestation respondent of a control you can log in to the Employee Center portal and respond to the attestations.

## Procedure

1.  Navigate to **All** &gt; **Self-Service** &gt; **Employee Center**.

2.  Select **GRC tasks** in the **My active items** pane of the Home page.

3.  Select the new task in My to-dos section.

4.  Select the assessment instance link in the My pending tasks related list.

5.  Enter your response to the set of questions, and attach an evidence document.

6.  Select the **Next** button.

7.  Select the **Provide acknowledgement** option.

8.  Select the **Submit assessment** button.

9.  Select **Close** in the Submission Successful pop-up.

    All the questions are read only now that you have submitted the response.
