# Enable the control objective workflow

Enable the control objective workflow property to activate the review and approval process for changes to control objective records.

## Before you begin

Role required: sn\_compliance.admin

## About this task

The control objective workflow is disabled by default.

Disabling the property after it has been enabled reverts the form to its previous state. Workflow action buttons are hidden, and existing staging records become inaccessible through the UI, although they remain in the system for audit purposes.

**Note:** The control objective workflow property is available only when the GRC: Common Workspace Elements plugin is installed.

## Procedure

1.  Navigate to **All** &gt; **Policy and Compliance** &gt; **Administration** &gt; **Properties**.

2.  Locate the **Enable control objective workflow** property.

3.  Select the check box to enable the property, then select **Save**.
