# User interface changes for assessments based on GRC attestation

The Lists pane of the Compliance Workspace is updated with assessments link to navigate to the assessments. The attestation and classic attestation widgets are displayed in the home pages of the compliance manager, compliance analyst, and the IT compliance manager.

## Lists page of the Compliance Workspace

You can select the Assessments list option in the Control attestations list of the Compliance Workspace List view to view the list of assessments that are not in complete state.

## Attestation widgets in Compliance Workspace home pages

-   **Compliance Home page for the compliance manager**

    The home page for the compliance manager is updated with the Attestations widget. The existing attestation widget has been renamed as Classic attestations that has the data of the completed attestations for the control when the attestation method was Classic attestation.

    \[Omitted image "smart-assessments-comp-manager.png"\] Alt text: Compliance home page for the compliance manager.

-   **Compliance Home page for the compliance analyst**

    Similarly, the home page for the compliance analyst is updated with the Attestations widget. The existing attestation widget has been renamed as Classic attestations and has the data of the completed attestations for the control when its attestation method was Classic attestation.

-   **Compliance Home page for the IT compliance manager**

    So also the home page for the IT compliance manager displays the attestations data, and classic attestations if the control was formerly assessed by classic attestation method.


## Attestation widgets in the Overview pages of control objective and control

-   **Overview page of a control objective record**

    The overview page displays the Attestations widget with the open and overdue attestations for the last six months if the assessment method was Attestation.

-   **Overview page of a control record**

    Displays the Attestations widget with the open and overdue attestation details. If the control was previously assessed by the Classic attestation method, then the data of all completed attestations are displayed. You can also view the **Completed attestation** widget with its details.
