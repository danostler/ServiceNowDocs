# Primary origin form

On the Primary origin form, fill in the fields.

</th></tr></thead><tbody><tr><td>

Description

Related area

</td><td>

Type

</td><td>

Type of the related area. For example, any related risk event. The choices are as follows:-   **Citation**
-   **Policy**
-   **Control objective**
-   **Risk event**
