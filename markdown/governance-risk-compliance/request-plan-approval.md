# Request the plan approval

Request the plan approval from the analysis approver. You can then proceed with the scenario analysis in Operational Resilience Workspace.

## Before you begin

Role required: sn\_oper\_res.manager

## Procedure

1.  Navigate to **Workspaces** &gt; **Operational Resilience Workspace** &gt; **All scenario analysis** and select the scenario analysis that you’re working on.

2.  Select **Request plan approval**.

    As the plan approver, you can log into the scenario analysis and complete the process.

    The state of the scenario analysis is updated to **Pending plan approval**. The state and the owner of the scenario analysis are displayed in the header of the scenario analysis form.

3.  To approve, reject, or cancel the scenario analysis, double-click the State column.

4.  To reject the scenario analysis, update its state to **Rejected** and select **OK**.

5.  To cancel the scenario analysis, update its state to **Canceled** and select **OK**.

6.  To approve the scenario analysis in the **Pending plan approval** state, update its state to **Approved** and select **OK**.

    The state of the scenario analysis record is updated to **Analyze**.


## What to do next

As the next step, request analysis approval. For more information, see [Submit the scenario analysis and receive an approval](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/governance-risk-compliance/request-plan-approval-for-scenario-analysis.md).
