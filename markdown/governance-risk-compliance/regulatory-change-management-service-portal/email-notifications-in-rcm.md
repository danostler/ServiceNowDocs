# Email notifications in Regulatory Change Management

A number of email notifications are sent by the Regulatory Change Management application.

## Email notifications

The following table describes the recipient list of notifications and the associated conditions when a notification is sent.

|Notification|Recipients|
|------------|----------|
|When a regulatory feed is closed|Coordinator|
|When a regulatory feed is deferred|Coordinator|
|When a regulatory feed is assigned|Coordinator|
|Regulatory action task last day|assigned\_to \(comp user/risk user\)|
|Regulatory action task assigned|assigned\_to \(comp user/risk user\)|
|Regulatory change task assigned|assigned\_to \(reg change user\)|
|Regulatory change task approved|assigned\_to \(reg change user\)|
|Regulatory change task rejected|assigned\_to \(reg change user\)|
|Regulatory change task cancelled|assigned\_to \(reg change user\)|
|Regulatory change task reminder due today|assigned\_to \(reg change user\)|
|Source document import task assigned|assigned\_to \(reg change user\)|
|Source import document task approved|assigned\_to \(reg change user\)|
|Source import document task rejected|assigned\_to \(reg change user\)|
|Source import document task cancelled|assigned\_to \(reg change user\)|
|Import document task reminder due today|assigned\_to \(reg change user\)|
|Review regulatory change task|Approver|
|Risk assessment- Reassessment completed|Approver|
|Notify assessor|Assessor|
|Notify approver/approver changed|Approver|
|Assessor changed|Assessor|
|Notify assessor of assessment cancellation when the risk assessment methodolgy retires|Assessor|
|Notify assessment cancellation by owner|Assessor|
|Notify about upcoming due date|Assessor|
|Notify about due date passed|Assessor|
|Final notification for due date passed|Assessor|
|Smart assessment overdue|Assessor|
|Smart assessment is completed|Requester of assessment|
|Smart assessment created|Assessor|
|Smart assessment cancelled when alert is closed/cancelled|Assessor|
|Smart assessment cancelled by RCM admin|Assessor|
|Smart assessment reassigned|Assessor|
