# Regulatory Change Management dashboard

The  Regulatory Change Management dashboard provides the current view of the regulatory landscape.

**Note:** The Overview Dashboard in the Classic UI is no longer available from the Zurich release.

\[Omitted image "rcm-overview-dashboard.png"\] Alt text: Regulatory Change Management dashboard

## Overview of regulatory changes

For examples of how different people in your organization would use this dashboard, see these use cases.

|User|Dashboard use|
|----|-------------|
|sn\_grc\_reg\_change.manager |The RCM Manager can view the new regulatory Alerts and understand the state of the alerts and the different regulatory tasks. It also provides a calendar view of the regulatory tasks with their due dates.|

The Overview page provides a summary of the regulatory events and tasks, such as the following:

-   New Regulatory Alerts
-   New Regulatory Change Tasks
-   Overdue Regulatory Change Tasks
-   Critical Action Tasks
-   Regulatory Alert by State: Displays the alerts that are grouped by state, number, and percentage
-   Regulatory Change Tasks by State: Displays the tasks that are grouped by state, number, and percentage
-   Import Document Tasks by State: Displays the tasks that are grouped by state, number, and percentage
-   Action Tasks: Groups and stacks the action tasks by the State and Priority fields
-   Regulatory Change Tasks by Due Date: Displays a report of the regulatory change tasks
