# Source document alerts view

When you select a source document alert in the List view in the Compliance Workspace, it displays the details of the alert on a new page. The details include the title of the alert, provider of the alert, its state, and other important information in the Overview tab and Details tab.

The source document alerts page also displays the following tabs:

-   Overview tab
-   Details tab

See more information on the tabs.

Stepper component

</td><td>

Process flow stepper component for the source document alert that displays the following stages: -   New: When the source document type alert is in the **New** state, the highlighted stage in the stepper component is displayed as **New**.
-   In progress: When the state of the source document type alert is updated to **In progress**, the highlighted stage in the stepper component is displayed as **In progress**.
-   Closed: When the source document type alert is in **Closed** state, the highlighted stage in the stepper component is displayed as **Closed**.

</td></tr></tbody>
</table>|Field|Description|
|-----|-----------|
|Regulatory alert|Details of the alert such as title of the alert, citation associated with the alert, provider URL, regulatory body URL, name of the provider, type of the alert, and coordinator of the alert.|
|Dates|Details of the important dates associated with the alert such as source publication date, comments date, effective date, compliance date, and expiration date.|
|Activity journal|Text box for entering additional comments related to the alert.|
|Compose additional comments|Text box for entering additional comments related to the activity.|

Taxonomy side-panel: The Taxonomy side-panel displays the following taxonomy details for the source document type alerts:

-   Regulatory bodies
-   Content types
-   Jurisdictions
-   Themes
-   Sectors

**Note:** The taxonomy classes are displayed only if the records have any values. For example, if the regulatory alert does not have any values for the taxonomy class Jurisdictions, the taxonomy category Jurisdictions is not displayed in the taxonomy side panel.
