# Link Action tasks

Link Action tasks for existing impacted areas from within a regulatory change task or directly through the regulatory alert.

## Before you begin

Role required: sn\_grc\_reg\_change.user

## Procedure

1.  Navigate to **All** &gt; **Regulatory Change Management** &gt; **Compliance Workspace**.

2.  Go to **List** &gt; **Regulatory alerts** &gt; **All assigned alerts**.

3.  Select and open the required regulatory alert.

4.  On the **Regulatory tasks** tab, select and open the required regulatory change task.

5.  Navigate to the **Action tasks** tab on a regulatory change task.

6.  Select **Link action tasks**.

7.  Select the impacted areas for which you want to create action tasks.

8.  Select **Add**.

    \[Omitted image "link-action-tasks.png"\] Alt text: Create action tasks for impacted areas.


## Result

An action task is created for each of the selected impacted areas within the specific regulatory task.

Linking action tasks automates task assignment for each impacted area, ensuring accountability and streamlined execution of regulatory requirements across the organization.

## What to do next

After the action tasks are created, the tasks appear in the list for the owners. The owner works on the action tasks and closes them.
