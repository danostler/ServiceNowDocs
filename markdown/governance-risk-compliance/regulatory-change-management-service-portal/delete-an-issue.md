# Delete an issue

Delete any issue if it’s a part of a regulatory change task or a source document task if it’s no long required.

## Before you begin

Role required: sn\_grc\_reg\_change.manager, sn\_grc\_reg\_change.user

## About this task

Delete an issue that is no longer required.

## Procedure

1.  Navigate to **Application** &gt; **List** &gt; **Issues** &gt; **All issues**.

2.  Select the issue that you want to delete.

3.  Select **Delete** from the **More Actions** menu.

4.  Read the warning message in the Confirmation dialog and select **Delete**.

    The selected issue is deleted.


## Result

You can no longer view the deleted issue in the issues list.
