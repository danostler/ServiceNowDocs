# Download and install GRC: Regulatory Change Management

Download the GRC: Regulatory Change Management application from the ServiceNow Store and run it in your ServiceNow instance.

## Before you begin

Role required: sn\_grc\_reg\_change.manager, sn\_grc\_reg\_change.it\_admin, or sn\_grc\_reg\_change.admin

## Procedure

1.  Download the GRC: Regulatory Change Management application from the ServiceNow Store.

    Follow the instructions for [Download a GRC application from the ServiceNow Store for the first time](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/governance-risk-compliance/download-grc-first-time.md). Downloading the Regulatory Change Management application automatically installs the following required applications:

    -   GRC: Policy and Compliance Management
    -   GRC: Advanced Risk
2.  In your instance, navigate to **All** &gt; **System Applications** &gt; **All Available Applications** &gt; **All**.

3.  Locate GRC: Regulatory Change Management.

4.  Select an appropriate release in the Details section.

    You can select either the latest release or up to one of the two previous releases.

5.  Select **Install**.

    A new Application installation window is displayed.

6.  Confirm the details of the application and select **Install**.


## Result

The GRC: Regulatory Change Management application is installed on your ServiceNow instance.
