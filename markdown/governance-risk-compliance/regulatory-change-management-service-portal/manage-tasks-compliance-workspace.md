# Managing regulatory tasks in Compliance Workspace

You can perform various regulatory tasks in the Regulatory Change Management application in Compliance Workspace and receive granular information on the tasks and alerts using the data visualization widgets.

The Regulatory Change Management workflow and the Regulatory Change Management process flow provide information on the regulatory tasks that are performed by the Regulatory Change Management application users to comply with the regulatory changes. See the [Regulatory Change Management workflow](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/governance-risk-compliance/regulatory-change-management-service-portal/what-is-rcm.md) and [Regulatory process flow and tasks](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/governance-risk-compliance/regulatory-change-management-service-portal/reg-change-workflow-swimlane.md) for more information.

You can perform the following GRC: Regulatory Change Management configuration tasks in Compliance Workspace:

-   Assign the regulatory event alert to a Coordinator. For more information, see [Assign a regulatory event alert to a coordinator](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/governance-risk-compliance/regulatory-change-management-service-portal/perform-actions-on-reg-alerts-using-ws.md).
-   Assess the impact of the regulatory event alert, complete the assessment, and manage the regulatory change tasks associated with the alert. For more information, see [Assess the impact of a regulatory event alert](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/governance-risk-compliance/regulatory-change-management-service-portal/assess-impact-of-reg-change-using-ws.md) and [Manage the regulatory change tasks](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/governance-risk-compliance/regulatory-change-management-service-portal/manage-regulatory-change-tasks.md).
-   Assign the source document alert to a Coordinator. For more information, see [Assign a source document alert to a coordinator](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/governance-risk-compliance/regulatory-change-management-service-portal/manage-source-document-alerts-using-ws.md).
-   Manage the source document import tasks associated with the source document alert. For more information, see [Manage a source document import task](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/governance-risk-compliance/regulatory-change-management-service-portal/manage-source-doc-import-tasks-using-ws.md).
-   Manage the action tasks that are associated with the regulatory event alerts and the source document alerts. For more information, see [Create a new action task for the alert](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/governance-risk-compliance/regulatory-change-management-service-portal/create-action-task-using-ws.md) and [Complete the action task associated with the alert](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/governance-risk-compliance/regulatory-change-management-service-portal/create-action-task-src-document-alert.md).
-   Create an issue related to a regulatory task. For more information, see [Create or add an issue related to a regulatory task](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/governance-risk-compliance/regulatory-change-management-service-portal/create-an-issue-reg-change-comp-ws.md).
-   Map the taxonomy. For more information, see [Manage the taxonomy](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/governance-risk-compliance/regulatory-change-management-service-portal/manage-taxonomy-using-ws.md).
