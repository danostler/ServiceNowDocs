# Link Change tasks

Link Change tasks directly from the Impacted areas tab of a regulatory alert.

## Before you begin

Role required: sn\_grc\_reg\_change.user

## Procedure

1.  Navigate to **All** &gt; **Regulatory Change Management** &gt; **Compliance Workspace**.

2.  Go to **List** &gt; **Regulatory alerts** &gt; **All assigned alerts**.

3.  Select and open the required regulatory alert.

4.  Navigate to the **Impacted areas** tab on a regulatory alert.

5.  Select the impacted areas for which you want to link change tasks.

6.  Select **Link change tasks**.

7.  Select the regulatory change tasks that you want to link.

8.  Select **Confirm**.

    \[Omitted image "link-change-tasks.png"\] Alt text: Link change tasks


## Result

An action task is created for each selected impacted area and the specific regulatory change task. For example, if you select three impacted areas and two regulatory change tasks, then six action tasks are created.

Linking change tasks enables scalable task automation by generating action tasks for each impacted area and regulatory change task combination, confirming complete coverage and reducing manual effort.
