# Platform Analytics dashboards

Platform Analytics dashboards are reusable canvases that combine data visualizations with interactive filters and visual components, supporting bookmarking, certification, sharing, and advanced customization.

Platform Analytics dashboards are versatile tools for data analysis and presentation, designed to support various user needs and technical requirements.

## Types of dashboards

Platform Analytics dashboards are of two main types:

-   Inline editor dashboards, which are user-friendly and suitable for business users requiring basic visualizations.
-   Technical editor dashboards, which offer advanced features and are ideal for scenarios demanding custom user navigation and dynamic data binding.

Role-based access ensures that different users have the appropriate permissions, with internal users able to create dashboards, dashboard admins managing all dashboards, and UI builder admins developing advanced technical dashboards. When choosing between these types, organizations should consider factors such as the level of technical expertise available, the setup effort required, the need for advanced features, and the necessity of management capabilities for business users.

Both types of dashboards provide valuable insights, with inline-editor dashboards being perfect for quick, user-friendly analytical pages, and technical editor dashboards offering the flexibility and power needed for more complex data presentations.
