# Get entitlement for the BCM application

The first step in installing an application is to verify that the application or the product and its associated applications have valid ServiceNow entitlements.

## Before you begin

Role required: admin

Individuals performing entitlement must have a Now Support account and have permission to request applications for the instances under consideration.

## Procedure

1.  Navigate to the [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home).

2.  Select **Login** and log in using your Now Support credentials.

3.  Select the **ServiceNow Products** tab to view all available ServiceNow Store products.

    Integrations and other types of content are shown on the **Certified Apps** tab.

4.  Select the product name of the product that you are getting entitlement to.

    All applications associated with the product you are entitling are listed. When you have obtained entitlement to the core product, the applications listed under it are also entitled.

5.  Select **Opt-in** to verify that you have entitlement to the product and the applications listed.

    You are prompted to read and accept the ServiceNow Store terms and conditions.

6.  Select the check box and select **Accept**.

    **Note:** A check mark appears next to the product name and the **Manage Entitlements** button appears.

    The check mark indicates that you have subscribed to the product and its associated applications. It is time to manage your entitlements. This step is a one-time process for this product and any other products that you want to install in the future.

7.  You can select **Manage Entitlements** to change the instances affected by the applications to which you are entitled.

    After you have agreed to the ServiceNow Store Terms and Conditions and managed entitlements, you can entitle other products by selecting the plus sign \(+\) next to the product name.

    You are now ready to install the application.
