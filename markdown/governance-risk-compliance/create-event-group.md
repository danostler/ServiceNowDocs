# Create an event group for the scenario

Create an event group in the Operational Resilience application so that you can categorize the events that are defined in a scenario.

## Before you begin

Role required: sn\_oper\_res.admin

## About this task

Classify the events of a similar type into an event group for your scenario. For example, you can classify a natural event, such as flooding, into the Weather event group. Similarly, you can classify a power outage under the Energy Issues event group.

## Procedure

1.  Navigate to **All** &gt; **Operational Resilience** &gt; **Admin** &gt; **Event Groups**.

2.  Click **New**.

3.  In the **Name** field, enter the name for the event group.

4.  In the **Label** field, enter the label for the event group.

    The configuration of a sample event group is shown in the following example.

    \[Omitted image "event-groups.png"\] Alt text: Sample configuration for the GRC Choice New record form.

5.  In the **Order** field, enter the order for the event group and click **Submit**.

    For more information on the fields in the Event group form, see [GRC Choice New record form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/governance-risk-compliance/event-group-reference.md).
