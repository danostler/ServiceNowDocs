# Loss Scenario record form

Use the Loss Scenario record form in BCM UIB Workspace to configure a plan for an identified loss scenario.

## Loss Scenario record form

For description of the field values, see the table.

|Field|Description|
|-----|-----------|
|Scenario Name|Name of the impact scenario.|
|Elements impacted|Items or assets that are impacted because of the loss scenario.|
|Description|Short note about the loss scenario.|
