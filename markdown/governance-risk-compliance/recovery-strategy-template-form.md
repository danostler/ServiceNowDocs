# Recovery strategy template form

Use the Recovery strategy template form to define a reusable implementation profile that planners can apply to loss scenarios in business continuity plans.

## Recovery strategy template form

\[Omitted image "recovery-strategy-template-form.png"\] Alt text: Recovery strategy template form showing Name, Description, and Implementation Details section.

For description of the field values, see the table.

|Field|Description|
|-----|-----------|
|Name|Name of the recovery strategy template.|
|Description|Description of the recovery strategy.|
|Estimated time to implement|Estimated time to implement the recovery strategy.|
|Maximum duration of use|Maximum duration of use for the recovery strategy.|
|Estimated % of operations achieved|Estimated percentage of operations achieved for the recovery strategy.|
|Comments|Free-form comments shown to planners when this template is applied.|
