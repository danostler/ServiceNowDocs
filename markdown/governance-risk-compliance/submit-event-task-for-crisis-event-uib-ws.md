# Request an approval and approve the crisis event

Approve the crisis event in BCM UIB Workspace.

## Before you begin

Role required: sn\_bcm.planner, sn\_bcm.program\_manager

## Procedure

1.  Navigate to **Workspaces** &gt; **Business Continuity Workspace**.

2.  In the List view, navigate to **Crisis events** &gt; **Pending** and select the crisis event in the list.

3.  Log in as the approver of the event and select the Event in the List view.

4.  Verify that all activated plans are closed; close the open plans before submitting the event for an approval.

    **Note:** Submitting an exercise or actual event for approval when an activated plan is still in the **Open** state results in an error message: `All activated plans must be closed before submitting the event for an approval`.

    \[Omitted image "event-error-msg.png"\] Alt text: Error message.

5.  Select **Submit for approval**.

6.  Select **Approve**.

7.  Select **Save**.

8.  Select **Closed Complete**.

    You can update the state of the event to **Closed Complete**.

9.  Select **Closed Incomplete**.

    You can update the state of the event to **Closed Incomplete**.
