# Audit Workspace

Audit workspace is a single-pane view for audit supervisor and auditor to view the overall audit timeline and status, track budget and resources for engagements, trace high priority observations and issues, and monitor ongoing control testing and audit task progress.

## Highlights of the new Audit Workspace

The workspace provides a new user experience.

-   Default home page for the audit supervisor and auditor displays actionable insights and quick links.
-   Organized navigation menu that lists the modules based on each user's needs.
-   Re-imagined page layout with improved experience for practitioners, business users, and executives.
-   Record pages that provide holistic view and actionable insights specific to what is being viewed.
-   360° view to consider scores and ratings, and visualize data relationship across the enterprise for a record.

## Audit supervisor view

An audit supervisor can use the workspace to:

-   define plans and engagements
-   assign budget and resource to engagements
-   track the progress of engagements

## Auditor view

An auditor using the workspace can:

-   test effectiveness of controls
-   request evidence
-   create observations
-   complete all audit activities
