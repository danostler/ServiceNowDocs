# Approve a plan

Approve a plan if you're assigned as an approver after plan creation. You can also request that the plan be reviewed or request more information about the plan from the plan creator.

## Before you begin

Role required: sn\_audit.manager

## Procedure

1.  Navigate to **All** &gt; **Audit** &gt; **My Plan Approvals**.

2.  Select and open the plan you want to approve.

3.  Review the plan and click one of the following options:

    |Option|Description|
    |------|-----------|
    |**Approve**|Approve the plan that is created.|
    |**Request Revision**|Request for the plan to be reviewed or modified, if it is not appropriate.|
    |**Request Information**|Request more information about the plan.|
    |**Cancel**|Cancel the plan.|
