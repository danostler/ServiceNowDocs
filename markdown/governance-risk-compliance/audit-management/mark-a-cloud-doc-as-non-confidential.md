# Mark a cloud file as a shareable file

Select one of the non-sharable files and mark it as sharable. Once the cloud file is marked as shareable, it can be linked to another record, or it can be referenced by other records.​

## Before you begin

Role required: sn\_audit.admin, sn\_audit.manager, sn\_grc\_workspace.user

## Procedure

1.  Navigate to **All** &gt; **Audit** &gt; **Audit Workspace**.

2.  Select the tasks icon \(\[Omitted image "new-task-icon.png"\] Alt text: Tasks icon\) and select a record from the list.

3.  In the **Cloud files** related list, select the cloud document.

4.  Select **Mark as sharable**.

    When a cloud file is marked as a shareable document, it can be linked to other records.

    The cloud file is marked as shareable. It can be shared or referenced by other records​.
