# Matrix relationship form

The table gives a description of the field values for the Matrix relationship form.

|Field|Description|
|-----|-----------|
|Name|Name of the matrix relationship.|
|Source|Source of the current relationship.|
|Source table|Source table of the relationship.|
|Relationship|Relationship between the source and the target tables.|
|Target table|Target table of the relationship.|
|Filter condition|Conditions to be set on the target table to filter required field data.|
|Order|Order in which the columns are represented in the report.|
