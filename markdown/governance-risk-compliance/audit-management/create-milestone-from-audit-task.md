# Create a milestone from an audit task

Create and track an audit task as a milestone.

## Before you begin

Role required: sn\_audit.manager

## Procedure

1.  Navigate to **All** &gt; **Audit** &gt; **Milestones** &gt; **All Milestones**

2.  Open the Milestone that contains the audit task that you want to track.

3.  From the Audit Tasks related list, open the audit task you want to track.

4.  Click **Track as a Milestone**.

    The milestone is created and the task is added to the milestone.
