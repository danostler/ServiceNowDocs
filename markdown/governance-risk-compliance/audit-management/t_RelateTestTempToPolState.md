# Relate a test template to a control objective

Audit owners can create generic control test templates for a control objective, avoiding the creation of individual control test plans for every control.

## Before you begin

Role required: sn\_audit.admin or sn\_audit.manager

## Procedure

1.  Navigate to **All** &gt; **Audit** &gt; **Audit Testing** &gt; **Test Templates**.

2.  Open the test template record.

3.  Select a control objective and select **Update**.
