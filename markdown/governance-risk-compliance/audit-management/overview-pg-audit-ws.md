# Manage an engagement from the overview page

After you create and save an engagement with the basic information such as the description, dates, scope, and the objectives, you can see the overview page.

## Overview page of an audit engagement

Use the overview page to get the necessary details and monitor the state flow of the audit engagement in the stepper. You can

-   define and scope an audit engagement
-   assign team members to it
-   perform and review fieldwork
-   manage issues
-   complete the audit engagement using the different related items
