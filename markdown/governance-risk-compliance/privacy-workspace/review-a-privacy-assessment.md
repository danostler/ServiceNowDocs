# Review a privacy assessment

As a privacy analyst, review a privacy assessment after the responders submit the assessment. You can either close the assessment after a review or you can also request for revision if you determine that the assessment requires more information.

## Before you begin

Role required: sn\_privacy.analyst

## About this task

As a reviewer, you can also preview the assessment with the applicable risk statements, information objects, and control objectives.

## Procedure

1.  Navigate to **All** &gt; **Privacy Workspace** &gt; **Tasks**.

2.  Select and open the privacy assessment that you want to review.

3.  Select **View response**.

4.  Review the assessment responses.

5.  To request for more information from the responder, select **Request for revision**.

6.  To mark the assessment as complete, select **Update state**.

    1.  In the **State** field, select **Closed complete**.

    2.  In the Additional comments field, provide any additional information.

    3.  Select **Submit**.
