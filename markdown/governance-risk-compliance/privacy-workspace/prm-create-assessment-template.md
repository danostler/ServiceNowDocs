# Create a privacy case assessment template

Create an assessment and send those assessments to the privacy case task owners to collect their responses. The responses help the privacy case analysts to determine the breaches that have occurred due to the privacy case.

## Before you begin

Role required: sn\_privacy\_case.privacy\_case\_admin

## About this task

Instead of using the default template, you can create a questionnaire template for assessing privacy cases. After you set up the assessment template, it’s available in the case task section for the **Assessment** case type.

## Procedure

1.  Navigate to **All** &gt; **Privacy Case Management** &gt; **Assessment Templates**.

2.  Select **New**.

3.  On the Assessment Metric Type form, fill in the fields.

    For a description of the field values on the Assessment Metric Type form, see [Assessment metric type form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/governance-risk-compliance/privacy-workspace/prm-assessment-metric-form.md).

4.  Select **Submit**.


-   **[Assessment metric type form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/governance-risk-compliance/privacy-workspace/prm-assessment-metric-form.md)**  
Use the Assessment Metric Type form to create an assessment questionnaire template instead of using the default template.
