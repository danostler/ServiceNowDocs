# Send a privacy assessment from an entity

Send a privacy assessment to an entity owner to determine if there's personal data involved in the processing activities.

## Before you begin

Role required: sn\_privacy.manager or sn\_privacy.analyst

## About this task

You can send a privacy screening assessment from an entity if you do not have a processing activity created for that entity. After a processing activity is created for an entity, any further assessments can only be sent through the processing activity.

## Procedure

1.  Navigate to **All** &gt; **Privacy Management** &gt; **Privacy Workspace** &gt; **Scoping** &gt; **My entities**.

2.  Select and open the entity from which you want to send the assessments to the entity owner.

3.  Select **Initiate privacy assessments**.

    The Send privacy assessment dialog box opens.

4.  Provide the required information such as the template to use, the due date, and so on in the dialog box.

    The description of the assessment is automatically fetched from the description provided at the time of the assessment template configuration. You can modify the description as required.

5.  Select **Send**.
