# Export a privacy case as a PDF

Export a privacy case as a PDF to easily share it with various stakeholders.

## Before you begin

Role required: sn\_privacy\_case.privacy\_case\_analyst

## Procedure

1.  Navigate to **All** &gt; **Privacy Management** &gt; **Privacy Workspace**.

2.  Select the list \[Omitted image "ws-list-icon.png"\] icon.

3.  Select **All Cases**.

4.  Open the case that you want to export as a PDF.

5.  Select the More actions icon on the top right corner.

6.  Select **Export PDF**.

7.  Select the appropriate PDF template.

8.  Select **Export**.
