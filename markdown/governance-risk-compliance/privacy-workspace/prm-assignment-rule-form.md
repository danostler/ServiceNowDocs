# Assignment rule form

Learn about the fields on the assignment rule form. Use this form to define case assignment rules in the Privacy Case Management application.

## Assignment rule form

For a description of the field values, see the following table:

|Field|Description|
|-----|-----------|
|Name|Name of the assignment rule.|
|Application|Name of the application this assignment rule applies to. This field is automatically set to **GRC: Privacy Case Management**.|
|Active|Option to enable the assignment rule.|
|Applies To|
|Table|Table this assignment rule applies to. This field is automatically set to **Privacy case \[sn\_privacy\_case\_privacy\_case\]**.|
|Conditions|Conditions to be defined for the case type's assignment rule.|
|Assign to|
|User|User the case is assigned to.|
|Group|Group the case is assigned to.|
|Script|
|Script|Script to define advanced assignment rule.|
