# Cause and consequence form

Learn about the fields on the cause and consequence form. Use this form to define cause and consequence in the Privacy Case Management application.

On the view rule form, fill in the fields as appropriate.

</th></tr></thead><tbody><tr><td>

</td><td>

</td></tr><tr><td>

Cause status

Status of the cause. The choices are as follows:-   **Suspected**: Select this option, if the cause isn’t yet confirmed.
-   **Confirmed**: Select this option, if the cause is confirmed.

</td></tr><tr><td>

Primary cause

</td><td>

Option to select the cause as primary.

</td></tr><tr><td>

Cause description

</td><td>

Brief description of the cause.

</td></tr><tr><td class="sub-head" colspan="2">

Consequences and actions

</td></tr><tr><td>

Consequences

</td><td>

Consequences of the case or event. You can select multiple consequences for a reported case.

</td></tr><tr><td>

Actions taken

</td><td>

Any action or measures taken to address the case.

</td></tr><tr><td class="sub-head" colspan="2">

</td></tr><tr><td>
