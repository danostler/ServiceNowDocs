# Add a privacy regulation related to a case

Add the necessary regulations that are or can be impacted by the case that has been reported. In the ServiceNow platform, regulations are captured as authority documents. Adding the regulations to the case enables you to identify which regulations are breached or violated and prevents penalties and fines.

## Before you begin

Role required: sn\_privacy\_case.privacy\_case\_analyst

## About this task

After you add the regulations, you can analyse the case to determine which regulations are violated. You can then mark the reporting status of the regulations as reportable, non-reportable, or potentially reportable. By default, all the regulations present in the system are displayed and the status of the regulations is set to **To be determined**.

## Procedure

1.  Navigate to **All** &gt; **Privacy Management** &gt; **Privacy Workspace**.

2.  Select the Tasks icon \(\[Omitted image "icon-tprm-ws-tasks.png"\] Alt text: Electronic signature\).

3.  Select **Cases**.

4.  Open the case for which you want to add the regulations.

5.  Select the Regulations related list.

    1.  Select **Add**.

    2.  Select the regulations to add.

    3.  Select **Add**.

6.  To set the reporting status of a regulation, select and open the regulation.

7.  On the form, fill in the fields.
