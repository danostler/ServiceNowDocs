# Send a privacy assessment to multiple entities

Send the privacy screening assessment or the Privacy impact assessment \(PIA\) assessments to multiple entities to understand why and how personal data is being processed.

## Before you begin

Role required: sn\_privacy.manager

## Procedure

1.  Navigate to **All** &gt; **Privacy Management** &gt; **Privacy Workspace** &gt; **List** &gt; **Scoping** &gt; **Entity types**.

2.  Select the entity types that contain the entities for which you want to send the assessments.

3.  Select the entities related list.

4.  Select the check box against the required entities.

5.  Click **Send privacy assessments**.

6.  From the **Classification** field, select the type of assessment.

7.  From the **Assessment template** field, select the template you want to use.

8.  In the **Due Date** field, select the due date for completing the assessment.

9.  Click **Send**.


## Result

The assessment is sent to the entity owners.
