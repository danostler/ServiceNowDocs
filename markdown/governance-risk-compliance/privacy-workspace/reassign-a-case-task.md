# Reassign an assessment type case task

Reassign an **Assessment type** case task to another user. A case analyst or a case task owner can reassign a task to another case task owner, if required.

## Before you begin

Role required: sn\_privacy\_case.privacy\_case\_analyst, sn\_privacy\_case.privacy\_case\_business\_user

## About this task

There are three types of case tasks and they are described as follows.

-   **Assessment**: This type of case task requires the case task owner to take an assessment and provide their responses.
-   **Investigation**: This type of case task requires the case task owner to provide their observations.
-   **Others**: Like Investigation, this type of case task also requires the case task owner to provide their observations.

As a case analyst, after a task is assigned to a case task owner, you can reassign the task to another privacy business user. Possible reasons for reassignment can be that a case task is wrongly assigned or is not accepted by the case task owner and so on. You can reassign a task to privacy business users.

As a case task owner, you can also accept the task, start working on it, and then reassign the partially completed task to another user. When you reassign the partially completed case task to another user, the data that you have provided also gets copied over and is available when the new case task owner takes the assessment. This data can be modified by the new task owner.

**Note:** You can only reassign those case tasks that have the **Type** field set to **Assessment**.

## Procedure

1.  Navigate to **Self Service** &gt; **Employee Center**.

2.  Access your task from the **My tasks** module.

3.  Open the task that you want to work on.

4.  Select **Reassign**.

    The Reassign case task dialog box appears.

5.  In the **Assignment group** field, select the assignment group.

6.  In the **Case task owner** field, select the owner.

7.  Select **Reassign**.
