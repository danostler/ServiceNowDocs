# Add action tasks to a Personal Data Rights request

Create action tasks for a Personal Data Rights request and assign them to the appropriate task owners. Action tasks are tasks that are created to facilitate and complete a valid personal data rights request

## Before you begin

Role required: sn\_grc\_pdr.pdr\_admin

## Procedure

1.  Navigate to **All** &gt; **Personal Data Rights Workspace**.

2.  Open the request for which you want to add action tasks.

3.  Select the Action tasks related list.

4.  Select **New**.

5.  On the form, fill in the fields.

    For information on the form fields, see [New action task form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/governance-risk-compliance/privacy-workspace/new-action-task-form.md).

6.  Select **Save**.


-   **[New action task form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/governance-risk-compliance/privacy-workspace/new-action-task-form.md)**  
Use the new action task form to create action tasks for the appropriate owners.
