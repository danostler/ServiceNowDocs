# Create a breach factor type

Create categories to help responders to identify breach factors within specific categories. For example, incident nature is a breach factor type which has factors under it as Intentional and Malicious, or Intentional and not malicious, and Unintentional or inadvertent.

## Before you begin

Role required: sn\_privacy\_case.privacy\_case\_admin

## Procedure

1.  Navigate to **All** &gt; **Privacy Case Management** &gt; **Breach Assessment Configuration** &gt; **Breach Factor Types**.

2.  Select **New**.

3.  On the form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Label|Category to which the factor type belongs. For example, Incident nature, Recipient and so on.|
    |Name|Element of the breach factor.|
    |Active|Option to activate the factor.|
    |Application|Name of the application this factor applies to. This field is automatically set to **GRC: Privacy Case Management**.|
    |Description|Brief description of the breach factor type.|

4.  Select **Submit**.
