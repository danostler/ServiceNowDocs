# Work on a case task

As a case task owner, accept the task, provide the details requested by the case analyst, and submit it for a review. If a case task is assigned to an assignment group, and not to a specific user, then any user who is a part of the assignment group can accept the task and work on it.

## Before you begin

Role required: sn\_privacy\_case.privacy\_case\_business\_user

## About this task

If a case task is of type **Assessment**, a case requires the case task owner to take an assessment to determine losses incurred from a case. In this case, the task owner can accept the work and then take the assessment. If the case task is of type **Investigation** or **Others**, the case task owner can accept the task, provide their observations, and submit the case for review to the case analyst.

## Procedure

1.  Navigate to **Self Service** &gt; **Employee Center**.

2.  Access your task from the **My tasks** module.

3.  Open the task assigned to you.

4.  Select **Accept work** .

5.  In the confirmation dialog box, select **Accept work**.

6.  Depending on your case task type, perform the following steps.
