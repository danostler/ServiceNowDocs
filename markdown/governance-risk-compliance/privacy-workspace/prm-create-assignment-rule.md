# Create an assignment rule

Specify the conditions for assigning a privacy case to a particular user or group by creating assignment rules.

## Before you begin

Role required: sn\_privacy\_case.privacy\_case\_admin

## About this task

To determine the appropriate person or group to handle a privacy case, establish assignment rules that outline the necessary criteria. For instance, if the case pertains to data protection and information security, it should be assigned to the appropriate individual or team.

## Procedure

1.  Navigate to **All** &gt; **Privacy Case Management** &gt; **Case type**.

2.  Select the **Assignment Rules** related list on the Assignment Rule form.

3.  Select **New**.

4.  On the Assignment Rule form, fill in the fields.

    For a description of the field values on the Assignment Rule form, see [Assignment rule form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/governance-risk-compliance/privacy-workspace/prm-assignment-rule-form.md).

5.  Select **Submit**.
