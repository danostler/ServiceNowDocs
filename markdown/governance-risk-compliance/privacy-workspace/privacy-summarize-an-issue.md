# Use Issue summarization skill to summarize an issue

Use Issue summarization skill to summarize an issue, and obtain a quick context and awareness about an issue.

## Before you begin

Role required: sn\_prm\_gen\_ai.user

## About this task

**Important:** This Now Assist skill is turned on by default. The skill will be automatically available to appropriate role users for the application. For more information, see [Now Assist skills, agents, and agentic workflows on by default](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/intelligent-experiences/now-assist-skills/now-assist-skills-on-by-default.md).

The Issue summarization skill analyzes the issue records including the description, activity log, and remediation tasks, and generates a concise summary that captures the essential information that you need.

## Procedure

1.  Navigate to **Workspaces** &gt; **Privacy Workspace**.

2.  Select the List icon.

3.  Select All Issues.

4.  Open the issue for which you want to generate the tab.

5.  In the Issues tab, open the issue for which you want to summarize.

    **Note:**

    Check whether your role has the sn\_prm\_gen\_ai.user role to generate issue summarization.

6.  Generate a summary of the issue from the Overview tab by selecting **Summarize**.

    A summary of all the important details, such as the issue definition, actions taken, and remediation tasks are generated. Review the information for accuracy.

7.  After the summary is generated, you can take the following actions:

    -   To view details about the summary, select the information icon.
    -   To add the summary to the issue work notes, select **Share to work notes**. You can add or modify information before saving.
    -   Additionally, you also have the options to generate the summary again, expand or collapse the panel, provide feedback, or copy the information.
