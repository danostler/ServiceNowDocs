# Configure information object categories

Configure information object categories to classify information objects effectively. For example, attributes like iris scans and fingerprints are often referred to as biometric data, or email addresses and phone numbers can be grouped as contact information. Information object categories enable you to categorize these information objects under these broader classifications.

## Before you begin

Role required: sn\_privacy.admin

## Procedure

1.  Navigate to **All** &gt; **Privacy Management** &gt; **Privacy Workspace** &gt; **List** &gt; **Lists** &gt; **Privacy elements** &gt; **Information object category**.

2.  Select **New**.

3.  On the form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Name|Name of the information object category. For example, `Financial data`.|
    |Description|Description of the types of information objects the category contains.|
    |Active|Option to indicate if the category is active.|
    |Sensitive category|Option to indicate if the data is sensitive in nature.|
    |Functional domain|Domain of the category. This field is automatically set to **Privacy**.|

4.  Select **Save**.

    A new tab **Information objects** appears.

5.  Select the Information objects related list.

6.  Add the information objects that belong to the category.

7.  Select **Save**.
