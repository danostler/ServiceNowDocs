# Create a jurisdiction for a region

Create jurisdictions for regions for data breach notification obligations as each obligation is broken down by jurisdiction. Jurisdiction can be created two levels below a region.

## Before you begin

Role required: sn\_privacy\_case.privacy\_case\_admin. This role must contain the user\_admin role.

## Procedure

1.  Navigate to **All** &gt; **Privacy Case Management** &gt; **Breach Assessment Configuration** &gt; **Regions**.

2.  Open the region for which you want to add jurisdictions.

3.  Select the Locations related list.

4.  Select **New**.

5.  On the form, fill in the fields.

6.  Select **Submit**.
