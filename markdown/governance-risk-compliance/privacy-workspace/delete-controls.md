# Delete a control from a processing activity

Delete the controls that are no longer required in the processing activity.

## Before you begin

Role required: sn\_privacy.admin

## About this task

You can delete controls if the control objectives associated with those controls have been erroneously added. If you want to add new controls, you may want to delete existing controls.

**Note:** If a control is associated with any other entity or entity type, then you cannot delete the control. By deleting a control, you can only remove the association of that control with the processing activity.

## Procedure

1.  Navigate to **All** &gt; **Privacy Workspace** &gt; **Processing activities** &gt; **All processing activities**.

2.  Select and open the processing activity from which you want to delete controls.

3.  Click the Controls related list.

4.  Select the controls that you want to delete.

5.  Click **Delete**.


## Result

The selected controls are deleted from the processing activity.
