# New privacy assessment creation form

Use the fields on the new privacy assessment creation form to create a new privacy assessment.

## Create a new privacy assessment creation form fields

|Field|Description|
|-----|-----------|
|Privacy assessment|
|Number|Privacy assessment number. This field is automatically set.|
|Title|Name of the privacy assessment.|
|Requester|Name of the person requesting the assessment.|
|Assessment template|Template to be used for the assessment.|
|State|State of the assessment. This field is automatically set to **Draft**.|
|Priority|Priority of the assessment.|
|Description|Description of the assessment.|
|Assignment|
|Assignee|Name of the person who will take the assignment.|
|Reviewers|Names of the persons who will review the assessment. You can select multiple reviewers in this field.|
|Schedule|
|Due date|Date by which the assessment would be completed.|
|Accepted date|Date the assessment is accepted. This field is read-only and displays the date only when the task is accepted.|
|Created|Date the assessment is created. This field is read-only.|
|Task closure date|Date when the assessment task is closed. This field is read-only and displays the date only when the task is closed.|
|Summary|
|Observations|Observations about the assessment.|
|Activity|
|Work notes \(Private\)|Work notes for the assessment.|
|Additional comments \(Customer visible\)|Comments for updating this assessment.|
