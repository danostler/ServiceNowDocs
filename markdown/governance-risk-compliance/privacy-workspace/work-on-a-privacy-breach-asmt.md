# Work on a privacy breach assessment

Accept the work of assessing a privacy breach assessment and work on the assessment.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Self Service** &gt; **Employee Center** &gt; **GRC tasks** &gt; **My To-dos** &gt; **Data breach assessments**.

2.  Select and open the assessment that you want to work on.

3.  Select **Accept work**.

4.  In the Confirmation dialog box, provide any notes as necessary.

5.  Select **Accept work**.

6.  To add personal information artifacts, select **PI Artifacts**.

7.  In the Create PI Artifact dialog box, select the region and category.

    1.  Select **Create**.

    2.  Select **Save**.

8.  Select the PI artifact and fill in the details as required.

    1.  Select the Data elements related list and add the data elements to add.

    2.  Select **Update**.

    3.  Select the Jurisdictions related list and add the jurisdictions.

9.  To send the assessment for review, select **Request review**.

10. To confirm, select **Request**.
