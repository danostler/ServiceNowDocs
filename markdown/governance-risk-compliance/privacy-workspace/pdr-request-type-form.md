# Personal Data Rights request type form

Use the Request Type form in the Personal Data Rights application to categorize the personal data rights requests by their type.

The following table lists the descriptions of the field values.

|Field|Description|
|-----|-----------|
|Label|Label to be displayed for the request type. This field is automatically set to **Personal data rights request**.|
|Name|Name of the request type. This field is automatically set.|
|Category|Category of the request type. This field is automatically set to **Personal data rights request \[sn\_grc\_pdr\_request\]**.|
|Active|Option to select the request type as active or inactive.|
|Description|Description about this request type. This description is for the administrator to understand the request type.|
|State Model|
|State Model|Workflow states for the request type.|
|Action task state model|State model transition defined at Action task state model. The states change for the action task workflow according to the state model.|
|Inbound Email Configuration|
|Group email|Email address that automatically generates a request upon receiving an email.|
