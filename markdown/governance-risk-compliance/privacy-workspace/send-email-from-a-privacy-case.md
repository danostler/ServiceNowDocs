# Send an email from a privacy case

Send emails from a privacy case to seamlessly communicate with various stakeholders and regulators and gather more information about the case. All the emails sent and received are stored in the Privacy Case Management application.

## Before you begin

Role required: sn\_privacy\_case.privacy\_case\_analyst

## Procedure

1.  Navigate to **All** &gt; **Privacy Management** &gt; **Privacy Workspace**.

2.  Select the list \[Omitted image "ws-list-icon.png"\] icon.

3.  Select **All Cases**.

4.  Open the case that from which you want to send an email.

5.  Select the More actions icon on the top right corner.

6.  Select **Compose Email**.

7.  Draft the email.

8.  Select **Send Email**.
