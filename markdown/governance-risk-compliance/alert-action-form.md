# Alert Action form

Use the Alert Action form in BCM UIB Workspace to add details about the alert actions.

For the description of the field values, see the table.

<table id="table_zhl_v5g_dqb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Title

</td><td>

Title of the alert action.

</td></tr><tr><td>

Description

</td><td>

Detailed description of the alert action.

</td></tr><tr><td>

Name

</td><td>

Name of the action in the Crisis map dashboard.

</td></tr><tr><td>

Action name

</td><td>

Unique name for the alert action.

</td></tr><tr><td>

Run once

</td><td>

Option to indicate the execution of the action one or more times from the dashboard.

</td></tr><tr><td>

Type

</td><td>

Type of action to be executed. Available options are:-   **Task**
-   **Script**
-   **Flow**

</td></tr><tr><td>

Task type

</td><td>

Task type for the alert action. For example, Asset task, Catalog task.

</td></tr><tr><td>

Application

</td><td>

Read-only field. The field is used as a flag to filter the items relevant to crisis management from a common list of actions.

</td></tr><tr><td>

Active

</td><td>

Option to indicate that the alert action is active.

</td></tr><tr><td>

Order

</td><td>

Display order that you can follow on the dashboard.

</td></tr><tr><td>

Run automatically

</td><td>

Option for enabling the action to be executed automatically based on a condition.

</td></tr><tr><td>

Roles

</td><td>

User roles who can execute the task successfully.

</td></tr><tr><td>

Display Fields

</td><td>

Fields to be displayed in the response actions after the actions are executed.

</td></tr><tr><td>

Condition

</td><td>

Conditions to execute the actions.

</td></tr><tr><td>

Task script

</td><td>

Task script to be executed when the user selects an action control on the dashboard.
