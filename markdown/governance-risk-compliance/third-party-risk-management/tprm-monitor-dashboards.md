# Monitoring assessment data using TPRM dashboards

You can monitor and analyze assessment data at various levels in the Third-party Risk Management application using the Third-party insights dashboard and TPRM custom analytics dashboard. These dashboards provide you and your team with tailored insights and deliver relevant information at a glance, improving your decision-making process.

## Dashboards in Vendor Management Workspace

You can view the Dashboards page by navigating to **Workspaces** &gt; **Vendor Management Workspace** and selecting the dashboard page icon \[Omitted image "icon-tprm-ws-dashboard.png"\] Alt text:. From the dashboards page, you can view the Third-party insights and TPRM Custom Analytics dashboards, which are included as part of the base system. You also have the option to create more dashboards, customize them, and share them with others.

## Third-party insights dashboard

This dashboard is a versatile, blank template that includes instructions to help you customize it to your specific needs. You can add various elements such as data visualizations, bar charts, trend analyses, and score reports to create custom sections and data visualizations. TPR managers, TPR assessors, and third-party assessment reviewers can view this dashboard, while TPR managers have the additional ability to edit and share it.

\[Omitted image "tprm-tpi-dashboard.png"\] Alt text: Accessing the Third-party insights dashboard. For the text description, refer to the text that preceded this example.

## TPRM custom analytics dashboard

This dashboard is pre-configured with a default list showing all active questionnaire instances relevant to your role. The list has customizable columns, enabling you to filter and group data according to your preferences. TPR managers have full edit access to this dashboard, while TPR assessors and third-party assessment reviewers have read access only. This dashboard serves as a centralized hub for efficiently tracking and comparing assessment responses, making it easier for you to manage assessments.
