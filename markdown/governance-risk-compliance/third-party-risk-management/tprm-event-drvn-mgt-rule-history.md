# View the run history for event-driven management rules

When an event-driven management rule runs, it auto-generates and sends questionnaires and doc requests to third parties. You can view the current state \(running, successful, or recalled\) of all runs of rules. Select a record number to view details and to view the set of assessments generated by the rule.

## Before you begin

Role required: sn\_vdr\_risk\_asmt.vendor\_risk\_manager

## Procedure

1.  Use either of the following methods to start the process:

    -   Workspace: Select **Workspaces** &gt; **Vendor Management Workspace** and on the **Risk** tab, select the list icon \[Omitted image "icon-tprm-ws-list.png"\] Alt text:. In the **Event-driven management** list, select any option: **Active**, **Inactive**, or **All**.
    -   Legacy: Navigate to **All** &gt; **Third-party Risk Management** &gt; **Event-driven management** and then select any option: **Active**, **Inactive**, or **All**.
2.  Select the **Event-driven management history** tab in the workspace or view the Event-driven management history related list on the form.

    Each record of a run includes the following read-only values:
