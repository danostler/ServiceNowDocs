# Viewing information on fourth parties

The **Downstream suppliers** tab displays overview information on fourth parties that the third party engages with. On most reports, you can select an item to view the underlying data.

## Getting to the Downstream suppliers tab

1.  Select **Workspaces** &gt; **Vendor Management Workspace**.
2.  On the **Risk** tab, select \[Omitted image "icon-tprm-ws-home.png"\] Alt text: Home page icon..
3.  In the Third-party risk overview section, select any number in a box to open the list of third parties with that value.

4.  Select a third party name and then select the appropriate tab.

    \[Omitted image "tprm-ws-dd-list-third-parties.png"\] Alt text: Open a third party page.


## Information on the Downstream suppliers tab

The name, website, and DUNS number for the fourth party are listed.
