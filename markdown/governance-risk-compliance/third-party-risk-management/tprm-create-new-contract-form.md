# Create New Contract form

On the Create New Contract form, fill in the fields.

<table id="table_nfr_2g4_ycc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td class="sub-head" colspan="2">

Contract

</td></tr><tr><td>

Contract number

</td><td>

Unique identifier number for the contractual arrangement, for example, CNTR0009002. This field is auto-filled.

</td></tr><tr><td>

Contractual arrangement reference number

</td><td>

Internal reference number of the contractual arrangement assigned by the financial entity. It is used to identify the contractual arrangement between the financial entity and the direct ICT third-party service provider.

</td></tr><tr><td>

Name

</td><td>

Name and short description of the contract.

</td></tr><tr><td>

Type of contractual arrangement

</td><td>

Type of contractual arrangement that is identified by using one of the options in the corresponding drop-down list:-   **Standalone arrangement**
-   **Overarching arrangement**
-   **Subsequent or associated arrangement**

</td></tr><tr><td>

State

</td><td>

State of the contract. For example, Draft state.

</td></tr><tr><td>

Contract model

</td><td>

Type of contract being created such as a Service or a Purchase Agreement.

</td></tr><tr><td>

Currency

</td><td>

Code of the currency that is used to report the amount.

</td></tr><tr><td>

Annual expense or estimated cost for the past year

</td><td>

Annual expense or estimated cost \(or intragroup transfer\) of the ICT service arrangement for the past year.

</td></tr><tr><td>

Description

</td><td>

Detailed description of the contract.

</td></tr></tbody>
</table>
