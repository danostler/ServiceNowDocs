# Set up a request type for a provider

After you register a risk intelligence provider and service, you specify the available request types that you and your organization will use.

## Before you begin

Role required: sn\_vdr\_risk\_asmt.vendor\_assessment\_reviewer

## About this task

A request type is the name of the report available for the provider. Request types can be defined as ranging from Level 1 \(general risk intelligence\) to Level 3 \(granular risk investigation, such as hiring a private investigator\). There’s no limit to the level or description of the request as long as it makes sense to both you and the risk intelligence provider. For example, a risk intelligence report ordered from Black Kite that provides a high-level summary of cyber risk information could be classified as a Level 1 - General Cyber Risk Report request type.

## Procedure

1.  Navigate to **All** &gt; **Third-party Risk Management** &gt; **Risk Intelligence Provider Setup** &gt; **Request Types** and then select **New**.

2.  Select the provider, enter a request type, select **Submit**, and then repeat the process for each request type.

3.  Repeat the overall process to define a service for each additional type of score that the provider can return.


## Result

You can now request different types of risk scores from the risk intelligence provider.
