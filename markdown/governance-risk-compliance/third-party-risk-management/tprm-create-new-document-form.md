# Create new document form

On the Create new document form, fill in the fields for the document.

<table id="table_cyz_r34_ycd"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Unique name of the document.

</td></tr><tr><td>

Owner

</td><td>

Individual who owns a document for audit purposes.

</td></tr><tr><td>

Template

</td><td>

Option to determine whether the document is a template.This field isn’t available in the third-party portal.

</td></tr><tr><td>

Default version

</td><td>

Document version that is used for document download. By default, the latest version is the default version.This field isn’t available in the third-party portal.

</td></tr><tr><td>

Current version

</td><td>

Latest document version that is used for document download.This field isn’t available in the Vendor Management Workspace.

</td></tr><tr><td>

Sharing permissions

</td><td>

Option that enables the document creator or owner to configure permission controls for the document. You can specify a user role and one of the following permission types:-   Reader
-   Writer
-   Owner

This field isn’t available in the third-party portal.

</td></tr><tr><td>

Third party

</td><td>

Third party with which the document is associated.When a third party is provided, sharing permissions are enabled automatically in TPRM.

This field is required only if the document is associated with a third party. For example, internal documents may not need to be associated with a specific third party.

**Note:** When a document is created by a third-party contact in the third-party portal, this field is auto- filled and sharing permissions are enabled.

</td></tr><tr><td>

Department

</td><td>

Name of the department that created the document.For example, Customer Support, Development, Finance, HR, or IT.

This field isn’t available in the third-party portal.

</td></tr><tr><td>

Type

</td><td>

Type of document based on the content. For example, Guideline, Policy, Procedure, or Contract.

</td></tr><tr><td>

Classification

</td><td>

Classification of the document.You can specify one of the following classification types:

-   Public
-   Restricted
-   Confidential
-   Internal

</td></tr><tr><td>

State

</td><td>

The state the document is in: Draft, Submit, Review, or Complete.

</td></tr><tr><td>

Description

</td><td>

A brief description about the document.

</td></tr></tbody>
</table>
