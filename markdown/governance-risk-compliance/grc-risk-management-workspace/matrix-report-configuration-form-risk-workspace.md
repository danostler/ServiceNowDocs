# Matrix report configuration form

Use the Matrix report configuration form to configure the Matrix report in the Risk Workspace.

Matrix report registry reference.

</td></tr><tr><td>

Base table

Base table of the Matrix report registry.

</td></tr><tr><td>

Workspace

Workspace to which this report configuration applies. You must select **Risk Workspace**.

Display type

Type of display. The options are as follows:-   **Landing page**: Option to select if you want this configuration to apply to the Entity Risk Profile Matrix page.
