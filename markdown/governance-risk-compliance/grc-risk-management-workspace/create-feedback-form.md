# Create New Feedback form

Use the Create new feedback form to provide feedback on a record or fields in a record.

See the following table for a description of the field values.

<table id="table_l35_zwl_kkb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Name of the feedback.

</td></tr><tr><td>

Description

</td><td>

A brief description of the feedback.

</td></tr><tr><td>

Refers to ServiceNow record

</td><td>

Option to select a ServiceNow table record on which you want to provide feedback. Clear this option to select an external record.

</td></tr><tr><td>

Record type

</td><td>

Feedback integration configuration on which you want to provide feedback. This enables you to select the table defined in the configuration.**Note:** This field appears only when the **Refers to ServiceNow record** option is selected.

</td></tr><tr><td>

Priority

</td><td>

Priority of the feedback.

</td></tr><tr><td>

Record

</td><td>

Record on which you want to provide feedback. This enables you to select the records associated in the table defined in the configuration.

</td></tr><tr><td>

Watch list

</td><td>

Individuals to be notified regarding the feedback.

</td></tr><tr><td class="sub-head" colspan="2">

Reviewers

</td></tr><tr><td>

Reviewer group

</td><td>

Group that is providing the feedback.

</td></tr><tr><td>

Reviewer

</td><td>

Individual providing the feedback. This field is automatically set to your name. You can also provide feedback on behalf of someone.

</td></tr><tr><td class="sub-head" colspan="2">

Respondents

</td></tr><tr><td>

Respondent group

</td><td>

Group that is responding to the feedback.

</td></tr><tr><td>

Respondent

</td><td>

Individual responding to the feedback.

</td></tr><tr><td class="sub-head" colspan="2">

Response

</td></tr><tr><td>

Response

</td><td>

State of the response. This is a read-only field for a reviewer.

</td></tr><tr><td>

Respondent comments

</td><td>

Comments or explanations provided by the respondent for the feedback. This is a read-only field for a reviewer.

</td></tr><tr><td class="sub-head" colspan="2">

Activity

</td></tr><tr><td>

Work notes

</td><td>

Notes or information about the feedback. For example, problems associated with the record and its possible solution.

</td></tr><tr><td>

Additional comments \(Customer visible\)

</td><td>

Additional information about the feedback that you want to share with the customers.

</td></tr></tbody>
