# Create a risk response task in the Risk Workspace

Create a risk response task to define plan of actions, assign responsibilities, set priorities, and establish deadlines to ensure effective management of the assessed risks.

## Before you begin

Role required: sn\_risk\_advanced.ara\_assessor

## Procedure

1.  Navigate to **All** &gt; **Risk** &gt; **Risk Workspace**.

2.  Select the task \[Omitted image "task-icon.png"\] Alt text: Task icon. icon.

3.  Open a risk assessment for which you want to add a risk response task and navigate to the Risk response section.

4.  To create a risk response task, select **Create task** next to the selected risk response plan.

5.  On the form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Name|Unique name of the risk response task.|
    |Plan|Actions that are implemented to manage the identified risk. Actions include steps, methods, and resources required to manage the risk.|
    |Assigned to|Person responsible for managing the risk response task.|
    |Priority|Importance or urgency of the risk response task.|
    |Due date|Date by which the risk response task must be completed.|

6.  Select **Save task**.

Create multiple strategies with various action items for each risk response task. Action items help you to understand and focus on specific steps needed to manage risks.
