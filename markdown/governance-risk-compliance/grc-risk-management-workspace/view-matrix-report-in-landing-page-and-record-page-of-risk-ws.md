# View matrix report in the landing page and record page of Risk Workspace

Access and analyze the risk posture of your organization using entity-related data, such as risks, controls, KRIs, and events in a centralized, configurable grid-based view. The Matrix report reduces time spent switching views and helps risk managers assess data more easily, leading to more proactive and streamlined risk management.

## Before you begin

Role required: sn\_risk\_workspace.operational\_risk\_manager or sn\_risk\_workspace.IT\_risk\_manager or sn\_risk\_workspace.business\_op\_risk\_manager

Risk Workspace and Matrix report plugin should be installed to use the feature.

## About this task

## Procedure

1.  Navigate to **All** &gt; **Risk** &gt; **Risk Workspace**.

2.  Select the Entity risk profile matrix \[Omitted image "entity-risk-profile-matrix-icon-risk-ws.png"\] Alt text: Matrix report icon. icon.

    For the landing page display type matrix configuration, you can view the Matrix report in the Risk Workspace.

3.  Open the entity record for which you want to view the matrix report and select the **Matrix report** related list.

4.  Select the View related information \[Omitted image "view-related-info-icon.png"\] Alt text: View related information. icon next to the entity name to view the matrix report.

    \[Omitted image "entity-related-matrix-report-record-risk-ws.png"\] Alt text: Matrix report on the record page.
