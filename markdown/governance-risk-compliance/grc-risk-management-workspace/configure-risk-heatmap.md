# Configure risk heatmaps

Within a risk assessment methodology \(RAM\), configure the heatmap visualization for inherent and residual assessments. Use different heatmaps for different risk assessment methodologies.

## Before you begin

Role required: sn\_risk.admin

## About this task

Configure the heatmap using the prepackaged heatmap widgets. Map the configured heatmaps to a dashboard to monitor the risk posture of the RAM.

## Procedure

1.  Navigate to the dashboard where you want to add the heatmap.

2.  In the upper-right corner, change the selected application from Global to GRC: Advanced Risk.

3.  To add widgets, select the plus \[Omitted image "plus-icon.png"\] Alt text: Add widgets icon icon in the upper right.

4.  From the Add Widgets list, select one of the following.

    -   **Advanced inherent heatmap**.
    -   **Advanced residual heatmap** to configure the inherent heatmap or the residual heatmap respectively.
5.  If you want to reuse an existing filter for the RAM, then do the following:

    1.  Select **Interactive Filters** from the **Add Widgets** list

    2.  Select **Reference**

    3.  Select **Risk Assessment Methodologies**.

        Alternatively, you can create your own filter on the RAM table and add it on the dashboard tab.


## Result

The selected heatmap appears on the dashboard.
