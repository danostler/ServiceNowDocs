# Create Feedback from the record side panel

Create feedback on a record or fields in a record from the record side panel using the Advanced Risk application. The feedback enables you to recommend improvements related to data integrity, compliance, operational procedures, and other pertinent areas.

## Before you begin

Role required: sn\_rvw\_feedback.reviewer

## Procedure

1.  Navigate to **All** &gt; **Risk** &gt; **Risk Workspace**.

2.  Select the list \[Omitted image "list-icon-riskworkspace.png"\] Alt text: List icon. icon.

3.  Open the record on which you want to create a Feedback.

4.  In the side-panel, select the list \[Omitted image "side-panel-list-icon.jpg"\] Alt text: side panel list icon. icon.

5.  Select **New**.

6.  On the form, fill in the fields.

    For a description of the field values on the Create New Feedback form, see [Feedback Details form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/governance-risk-compliance/grc-risk-management-workspace/feedback-details-form.md).

7.  Select **Save**.

    The feedback is moved to the Draft state.

8.  To initiate a Sidebar discussion, select the More actions \[Omitted image "more-actions-vertical-icon.png"\] Alt text: More actions icon. icon on the feedback, and select **Discuss**.

9.  To add feedback items, select the **Add Item**.

10. On the form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Record context|Fields on the record on which you want to provide feedback.|
    |Comments|Observations or feedback on the selected context.|

11. Select **Save**.

12. Select **Send feedback**.

    The feedback is raised and assigned to the respondent. The feedback is moved to the Assigned state.
