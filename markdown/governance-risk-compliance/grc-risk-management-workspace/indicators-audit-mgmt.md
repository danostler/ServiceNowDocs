# GRC Audit Management plugin indicators

The GRC Audit Management plugin contains various indicators.

The Audit Management plugin includes the following indicators:

-   **Key controls not tested**

    Number of key controls that are not tested.

-   **Failed Controls**

    Number of controls whose control tests are ineffective.

-   **Open Audit Issue**

    Number of open audit issues

-   **Passed controls**

    Number of controls whose control tests are effective.

-   **Ongoing Control Test**

    Number of Ongoing Control Tests

-   **Failed control test quarterly**

    Number of failed control tests in a quarter.

-   **New controls**

    Number of new controls.
