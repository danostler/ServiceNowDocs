# Create a test plan in Risk Workspace

Create a test plan to document the control testing procedure. You can create a test plan from scratch or based on a test template to describe how a feature is to be tested.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Risk** &gt; **Risk Workspace** &gt; **Control testing** &gt; **Control test plans**.

2.  Click **New**.

3.  On the form, fill in the fields.
