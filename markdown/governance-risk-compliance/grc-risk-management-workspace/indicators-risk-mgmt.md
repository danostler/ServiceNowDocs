# GRC Risk Management plugin Performance Analytics indicators

The GRC Risk Management plugin contains various Performance Analytics indicators.

The GRC: Risk Management plugin includes the following Performance Analytics indicators:

-   **Risk without mitigating control**

    Number of risks without mitigating control.

-   **Active Risk**

    Number of Active Risks

-   **New risk**

    Number of risks which are in draft state.

-   **Risk without KRI's**

    Number of Risk's without KRI's.

-   **Orphan risks**

    Number of risks without risk statement.

-   **Risk not assessed**

    Number of risks which are not assessed.
