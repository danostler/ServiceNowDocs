# GRC Profiles plugin indicators

The GRC Profiles plugin contains various indicators.

The GRC: Profiles plugin includes the following indicators:

-   **Open remediation tasks**

    Number of open remediation tasks

-   **Today**

    Number of issues closed today.

-   **Today**

    Number of issues created today.

-   **By this year**

    Total number of issues that need to be resolved by this year.

-   **By this quarter**

    Total number of issues that needs to be resolved by this quarter.

-   **By this month**

    Total number of issues that need to be resolved by this month.

-   **All open issues**

    Number of open issues

-   **All Unassigned Issues**

    Number of unassigned issues.

-   **Remediation Task Aging Indicator**

    Remediation Task Age Distribution on the basis of state.

-   **All Open Issues QoQ**

    All Open Issues Quarter to Quarter

-   **Issues Aging Indicator**

    Issue Age Distribution on the basis of state.

-   **By this week**

    Total number of issues that needs to be resolved by this week.

-   **This quarter**

    Number of issues closed on this quarter.

-   **New issues**

    Number of new issues

-   **This month**

    Number of issues created this Month.

-   **By today**

    Total number of remediation task that needs to be completed by today.

-   **This week**

    Number of issues closed on this week.

-   **This week**

    Number of issues created this week

-   **By today**

    Total number of issues that need to be resolved by today.

-   **This quarter**

    Number of issues created in this quarter.

-   **By this year**

    Total number of remediation tasks that need to be completed by this year.

-   **By this month**

    Total number of remediation task that needs to be completed this month.

-   **By this week**

    Total number of remediation tasks that needs to be completed by this week.

-   **This month**

    Number of issues closed on this month
