# GRC Policy and Compliance Management plugin indicators

The GRC Policy and Compliance Management plugin contains various indicators.

The Policy and Compliance Management plugin includes the following indicators:

-   **\# of failed control indicators**

    Number of failed control indicators per quarter.

-   **Controls \(not retired\)**

    Number of non-retired controls.

-   **Control not Attested**

    Number of controls whose state is in draft or attest.

-   **Orphan controls**

    Number of controls without control objective

-   **Mandated Controls**

    Number of controls whose enforcement is mandatory.

-   **Key Controls**

    Number of key controls

-   **Controls without Test Plans**

    Number of controls without test plans.

-   **Exempted controls**

    Number of exempted controls.

-   **Controls without KCI's**

    Number of controls without indicators.

-   **Active Controls**

    Number of active controls.
