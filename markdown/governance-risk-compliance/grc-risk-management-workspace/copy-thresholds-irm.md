# Copy thresholds

Copy existing thresholds to quickly create similar threshold configurations for the same metric or metric definition.

## Before you begin

Role required: sn\_risk.admin or sn\_risk.manager

## About this task

The system validates the copied threshold configuration. If any validation errors occur \(such as duplicate colors\), you must resolve them before saving. Copied thresholds are created as inactive by default. To activate the copied threshold, select the Active check box before saving.

## Procedure

1.  Navigate to **All** &gt; **Risk** &gt; **Risk Workspace** &gt; **Metrics** &gt; **Metrics**.

2.  Select the metric that contains the threshold you want to copy.

3.  Select the Thresholds related list.

4.  Select the check box next to the threshold you want to copy.

5.  Select **Copy**.

6.  Review the pre-filled threshold information in the Metric Threshold form and modify the threshold fields as needed.

    For example, you can change the name, color, valid dates, or actions.

7.  Select **Save**.

    A new threshold is created for the metric with the configuration from the original threshold. You can now modify either threshold independently.
