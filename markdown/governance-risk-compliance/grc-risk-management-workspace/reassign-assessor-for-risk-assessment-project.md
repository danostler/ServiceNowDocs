# Reassign assessor for a risk assessment project

Reassign assessors for multiple in-progress risk assessment projects simultaneously to minimize disruptions during stakeholder transitions.

## Before you begin

Role required: sn\_risk\_advanced.risk\_asmt\_project\_manager

## Procedure

1.  Navigate to **All** &gt; **Risk** &gt; **Risk Workspace**.

2.  Select the list \[Omitted image "list-icon-riskworkspace.png"\] Alt text: List icon. icon.

3.  From the list, navigate to **Risk assessment projects**, and select **In progress**.

4.  Select the Risk Assessment projects for which you want to reassign assessors.

    Reassign project dialog box appears.

5.  In the Reassign project dialog box, select the assessor in the **Select assessor** field.

6.  Select **Confirm**.
