# Functional location form

The following table describes the field values for the Functional location form.

<table id="table_r1x_bkb_1gc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Entity name

</td><td>

Name of the functional location or equipment model entity that you’re creating.

</td></tr><tr><td>

Short description

</td><td>

Brief description of the functional location.

</td></tr><tr><td>

Parent

</td><td>

Name of the functional location, if any, that is the parent to this functional location. To change the parent, search for and select the entity that is a parent to the entity that you’re creating. This field is empty for the top-level parent entity, which has no parent.

</td></tr><tr><td>

Location

</td><td>

Location of the functional location or equipment model entity.

</td></tr><tr><td>

Level

</td><td>

Hierarchical level that is assigned from the selected equipment model template for data sorting and structuring purposes.

</td></tr><tr><td>

Type

</td><td>

Name of the level type that is assigned to the equipment model template level.

</td></tr><tr><td>

Short code

</td><td>

Short code that is assigned to this functional location.

</td></tr><tr><td>

Company

</td><td>

Name of the company that is associated with the functional location.

</td></tr><tr><td>

Template

</td><td>

Equipment model template that is assigned to the functional location you’re creating. This fields is automatically set to ISA 95 Default Template.

</td></tr><tr><td>

Process criticality

</td><td>

Process criticality of the functional location. Options are:-   1 - most critical
-   2 - somewhat critical
-   3 - less critical
-   4 - not critical

</td></tr><tr><td>

Assigned to

</td><td>

Assigned user who operates and handles this functional location.

</td></tr><tr><td>

Managed by

</td><td>

Assigned person who owns and is responsible for managing this record.

</td></tr><tr><td>

Schedule

</td><td>

Schedule that is assigned to this functional location.

</td></tr><tr><td>

Support group

</td><td>

Name of the group that supports this functional location.

</td></tr><tr><td>

Managed By Group

</td><td>

Name of the assigned group that owns and is responsible for managing this functional location.

</td></tr></tbody>
