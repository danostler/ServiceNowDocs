# Standard automation task form

The following table describes the field values for the standard automation task form.

<table id="table_rgr_3z3_1gc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Action type

</td><td>

The selected value should be Create standard task for Automation.

</td></tr><tr><td>

Origin

</td><td>

Standard that the task originates from.

</td></tr><tr><td>

Short description

</td><td>

Brief description of the task.

</td></tr><tr><td>

Priority

</td><td>

Priority of the task. Options are:-   1 - Direct
-   2 - This shift
-   3 - Today
-   4 - Within 7 days
-   5 - Within 30 days
-   6 - Unplanned

The default value is 6 - Unplanned.


</td></tr><tr><td>

Standard

</td><td>

Standard that is used to create a new standard task.

</td></tr><tr><td>

Functional location

</td><td>

Industrial facility work area where the task is completed. The location is defined using the ISA-95 standard.

</td></tr><tr><td>

Equipment

</td><td>

Machine to which the task relates.

</td></tr><tr><td>

Assignment group

</td><td>

Team or department responsible for completing the task.

</td></tr><tr><td>

Assigned to

</td><td>

User that the task should be assigned to.

</td></tr></tbody>
