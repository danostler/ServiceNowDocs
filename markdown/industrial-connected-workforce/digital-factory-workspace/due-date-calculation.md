# Due date calculation

Use this table to see how the due date for a task is calculated based on its priority.

|Priority|Due date|
|--------|--------|
|1 - Direct|Add one hour to the current time|
|2 - This shift|At the end of the current shift|
|3 - Today|At 23:59:59 of the current date|
|4 - Within 7 days|Add seven days to the current date|
|5 - Within 30 days|Add 30 days to the current date|
|6 - Unplanned|None|
