# Archive rules in Industrial Guided Tasks

Archive rules in the Industrial Guided Tasks are designed to manage and organize records, so that you can focus on current and relevant data.

The system automatically archives guided tasks that are inactive and that haven’t been updated in two years. The name of the rule that applies to guided tasks is "Archive inactive Guided Tasks after 2 years".

By default, the archiving rule is active in the ICW. You can manage the rule by going to **All** &gt; **System Archiving** &gt; **Archive Rules** and deactivating it as needed.

The archiving rules support data hygiene and system usability by making sure that outdated records don’t interfere with active workflows.

For more information on archive rules, see [Create an archive rule in Core UI](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-administration/t_CreateAnArchiveRule.md).
