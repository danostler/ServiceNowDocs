# Configuring the Access Simulator \(Take actions\)

Configure the take actions options to complete the role and group assignments that you wish to add after the simulation.

## Before you begin

Role required: admin, access\_analyzer\_admin

## Procedure

1.  Navigate to **All** &gt; **Access Analyzer** &gt; **Settings**.

    \[Omitted image "simulator-settings.png"\] Alt text: Settings page

2.  Select the toggle switch to enable.

    Read through the legal information before enabling the Access Simulator \(Take actions\).

3.  Select **Accept**.

4.  Select **Save** to save the configuration.
