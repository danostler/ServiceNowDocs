# Service Provider \(SP\) system properties

These system properties define how the instance interacts with the IdP as a Service Provider.

Follow the sequential process to define IdP as Service Provider.

-   [Set the instance URL for SAML](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-security/authentication/t_SetTheInstanceURL.md)
-   [Set the audience URL for SAML](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-security/authentication/t_SetTheAudienceURL.md)
-   [Set up a NameID policy for SAML](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-security/authentication/t_SetUpNameIDPolicy.md)
-   [\(Optional\) Enable providing an authentication context class for SAML](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-security/authentication/t_OptEnableProvidingAuthContxtClass.md)
-   [\(Optional\) Set keystore properties for signing logout requests for SAML](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-security/authentication/t_OptSetKeystorePropsSignLogoutReqs.md)
