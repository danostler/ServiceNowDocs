# Create an Azure AD test user

Create a test user on Azure AD.

## Before you begin

Role required: Azure admin

## Procedure

1.  On the Azure portal, navigate to **All** &gt; **Azure Active Directory** &gt; **Users** &gt; **All users**.

2.  Select **New user**.

3.  In the User properties, provide the following information:

    -   Name
    -   User name
    -   Password
4.  Click **Create**.
