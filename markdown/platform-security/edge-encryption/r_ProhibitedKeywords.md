# Prohibited keywords

The Edge Encryption proxy validates encryption rule scripts before saving the rule. Many JavaScript keywords aren’t allowed in encryption rule scripts.

|Keyword|
|-------|
|\_\_DIR\_\_|
|\_\_FILE\_\_|
|\_\_LINE\_\_|
|\_\_parent\_\_|
|\_\_proto\_\_|
|Error|
|eval|
|getClass|
|getPrototypeOf|
|Java|
|javax|
|javafx|
|JavaImporter|
|load|
|loadWithNewGlobal|
|new|
|Packages|
|Object|
|prototype|
|RegExp|
|setPrototypeOf|
|this|
|throw|
