# Set up an Edge Encryption user account

The Edge Encryption proxies connect to the instance as a user to obtain and update encryption configuration information. Create a user account for this purpose and give the edge\_encryption role to the user.

## Before you begin

The Edge Encryption plugin must be installed before you can assign the role.

Role required: admin

## Procedure

1.  On your ServiceNow instance, create a user account to be used by the Edge Encryption proxy applications.

2.  Assign the edge\_encryption role to the user.
