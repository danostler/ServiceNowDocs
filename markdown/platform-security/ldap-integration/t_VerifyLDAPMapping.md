# Verify LDAP mapping

After creating an LDAP transform map, refresh the LDAP data to verify the transform map works as expected.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **System LDAP** &gt; **Scheduled Loads**.

2.  Click your LDAP import job.

3.  Click **Execute Now**.
