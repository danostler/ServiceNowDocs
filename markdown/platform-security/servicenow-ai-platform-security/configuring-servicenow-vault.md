# Configuring ServiceNow Vault

Learn how to install and configure ServiceNow Vault.

## Configuration overview

-   [Install ServiceNow Vault](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-security/servicenow-ai-platform-security/install-vault.md)

    Install the ServiceNow Vault application and assign the required roles to users who need access to the Vault console.

-   [ServiceNow Vault roles](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-security/servicenow-ai-platform-security/vault-roles.md)

    Learn about the roles necessary to use ServiceNow Vault and elevate to them so that you can access the full range of Vault capabilities.

-   [Install Vault plugins](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-security/servicenow-ai-platform-security/install-vault-plugins.md)

    Install and verify the plug-ins needed to make the most of ServiceNow Vault.

-   [Install Now Assist for Vault](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-security/servicenow-ai-platform-security/configuring-now-assist-vault.md)

    Install Now Assist for Vault to make use of generative and agentic AI capabilities within ServiceNow Vault.


## Using guided setup to implement ServiceNow Vault

Guided setup provides a sequence of tasks that help you configure ServiceNow Vault on your ServiceNow instance. To open guided setup for ServiceNow Vault, navigate to **All** &gt; **Vault** &gt; **Vault console**.

For more information, see [Use guided setup for ServiceNow Vault](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-security/servicenow-ai-platform-security/use-vault-guided-setup.md).
