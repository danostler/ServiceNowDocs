# Data Discovery policy

Use Data Discovery policies to define what sensitive data patterns should be identified and how they should be handled.

The **Policy** tab of Data Discovery reviews current discovery policies. View additional details of a policy by selecting the **View details** button. Edit, delete, and switch the state of current policies by selecting the three dots \[Omitted image "dds-dot-icon.png"\] Alt text: Three dots expandable. To create a new policy see [Create new policy](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-security/data-discovery/dds-new-policy.md).
