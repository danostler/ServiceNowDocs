# User Trails

Displays audits of users in the ServiceNow® instance.

For a given user, User Trails show changes to:

-   Identity attributes
-   role membership
-   group membership

To access User Trails, navigate to **All** &gt; **System Security** &gt; **Identity and Access Audit** &gt; **User Trails**. The User Trails page displays the following information.

| | |
|---|---|
|User Name|Name of the user.|
|User Sys id|The User sys\_id for the audited record.|
|Source Table|Details about the Source Table where the audit was performed.|
|Action|Describes the audit action.|
|Created By|The user who made the change.|
|Created|The time and date the audit was performed.|
