# Campaign tasks form

Campaign tasks form enables you to add the recall campaign details.

<table id="table_rg5_n23_wfc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Number

</td><td>

Campaign task number that is automatically generated.

</td></tr><tr><td>

Assigned to

</td><td>

Name of the person to assign the campaign task.

</td></tr><tr><td>

Recall campaign

</td><td>

Recall campaign details that are fetched from [Recall campaign form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/manufacturing/mco-recall-campaign-form.md).

</td></tr><tr><td>

Priority

</td><td>

Priority information that is fetched from [Recall campaign form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/manufacturing/mco-recall-campaign-form.md).

</td></tr><tr><td>

State

</td><td>

State of the campaign task. Options are:-   Pending
-   Open
-   Work in Progress
-   Closed Complete
-   Closed Incomplete
-   Closed Skipped

</td></tr><tr><td>

Active

</td><td>

Active or inactive state of the campaign task.

</td></tr><tr><td>

Short description

</td><td>

Short description of the campaign task.

</td></tr><tr><td>

Description

</td><td>

Any information that you want to update for next phases.

</td></tr></tbody>
