# Stakeholders

Create the stakeholder form to update the stakeholder involved in the product quality investigation.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **Workspaces** &gt; **CSM/FSM Configurable Workspace** &gt; **List** &gt; **Product Quality Investigation** &gt; **Stakeholder**.

2.  Select **New**.

3.  On the Stakeholder form, fill in the fields.

    For a description of the field values, see [Stakeholder form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/manufacturing/mco-stakeholders-form.md).

4.  Select **Save**.
