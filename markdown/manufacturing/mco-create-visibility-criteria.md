# Create a visibility criteria

Create a visibility criteria so that a manufacturer can filter the dealer name for which the target sales promotion is applicable.

## Before you begin

Role required: manufacturing operations admin \(sn\_mfg\_cmn.manufacturing\_operations\_admin\) or sales promotion manager \(sn\_sales\_prm\_mgmt.sales\_promotion\_manager\)

## Procedure

1.  Navigate to **Workspaces** &gt; **CSM/FSM Configurable Workspace** &gt; **Lists** &gt; **Sales Promotion Management** &gt; **My Promotions** &gt; **Visibility Criteria**.

2.  Select **New**.

3.  On the form, fill in the fields.

Fields

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Sales promotion

</td><td>

</td></tr><tr><td>

</td></tr></tbody>
</table>4.  Select **Save**.
