# Corrective action form for PQI

The corrective action form enables you to add the details for PQI.

<table id="table_y3s_4v3_xhc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Number

</td><td>

Corrective actions number is automatically generated. The number starts with CRTV and incremented for every new report.

</td></tr><tr><td>

Remediation action plan

</td><td>

Select the remediation action plan from the list.

</td></tr><tr><td>

Issue

</td><td>

Select the record for which corrective action must be performed.

</td></tr><tr><td>

Total actual cost

</td><td>

Enter the actual cost of the corrective action.

</td></tr><tr><td>

Implementation date

</td><td>

Select the date when the corrective action must be implemented.

</td></tr><tr><td>

Validated date

</td><td>

Select the date when the actions were validated.

</td></tr><tr><td>

Priority

</td><td>

Choose the severity of the corrective action from the options:-   Critical
-   Major
-   Moderate
-   Minor

</td></tr><tr><td>

State

</td><td>

Choose the state of the corrective action:-   New
-   On hold
-   In progress
-   Closed complete
-   Closed canceled
-   Closed incomplete

</td></tr><tr><td>

Table

</td><td>

Select the table name for which corrective action must be performed.

</td></tr><tr><td>

Remediation item

</td><td>

Select the remediation item record associated with the action.

</td></tr><tr><td>

Assignment group

</td><td>

Select the organizational units or teams responsible for handling specific tasks, cases, or issues

</td></tr><tr><td>

Assigned to

</td><td>

Select the user to whom the corrective action has been assigned.

</td></tr><tr><td>

Short description

</td><td>

Enter a short note.

</td></tr><tr><td>

Description

</td><td>

Enter the detailed description of the corrective action.
