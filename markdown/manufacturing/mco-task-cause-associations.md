# Task cause associations

Create the task cause for the root cause analysis of the product non-conformance.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **Workspaces** &gt; **CSM/FSM Configurable Workspace** &gt; **List** &gt; **Product Non Conformance Case** &gt; **Root Cause Analysis Task** &gt; **Task Cause Association**.

2.  Select **New**.

3.  On the Task Cause Association form, fill in the following fields.

</table>4.  Select **Save**.
