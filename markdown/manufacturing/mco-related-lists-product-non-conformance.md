# Related lists for Non-conformance

In the workspace, you can use the product non-conformance related list to log the report.

|Related list|Description|
|------------|-----------|
|[Product non-conformance case task](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/manufacturing/mco-product-non-conformance-case-task.md)|Create Non-conformance case task that had issue.|
|[Root cause analysis task](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/manufacturing/mco-root-cause-analysis-task.md)|Create root cause analysis of the case.|
|[Correction actions](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/manufacturing/mco-correction-actions.md)|Create corrective actions.|
|[Containment actions](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/manufacturing/mco-containment-actions.md)|Create containment actions.|
|[Task SLA](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/manufacturing/mco-task-sla.md)|Create task SLA.|
|[Relationships](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/manufacturing/mco-relationships.md)|It is used to track similar or duplicate product non-conformance case.|

-   **[Product non-conformance case task](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/manufacturing/mco-product-non-conformance-case-task.md)**  
Create the non-conformance case task report for products that had an issue.
-   **[Root cause analysis task](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/manufacturing/mco-root-cause-analysis-task.md)**  
Create root cause analysis \(RCA\) to identify the underlying cause of an issue and implement effective, long-term solutions.
-   **[Correction actions](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/manufacturing/mco-correction-actions.md)**  
Create the correction actions for the non-conformance report.
-   **[Containment actions](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/manufacturing/mco-containment-actions.md)**  
Create a containment action report to address issues such as non-compliance and process gaps.
-   **[Task SLA](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/manufacturing/mco-task-sla.md)**  
Create one or more Service Level Agreement \(SLA\) definitions and use them to create an SLA record. This SLA record enables you to use an SLA system for your organization's task.
-   **[Relationships](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/manufacturing/mco-relationships.md)**  
Create the relationship between parent and child and assign the type of relationship.
