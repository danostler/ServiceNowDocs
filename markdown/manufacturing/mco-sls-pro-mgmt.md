# Sales promotion management

The Agent workspace enables the OEMs to create the promotion campaign for a specific set of customers.

|Tasks|Description|
|-----|-----------|
|[Create a promotion](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/manufacturing/mco-my-promotions.md)|Displays all the sales promotion claims that are assigned to the person logged in to the workspace.|
|[View all promotions](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/manufacturing/mco-all-promotions.md)|Displays all the sales promotion claims.|

-   **[Create a promotion](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/manufacturing/mco-my-promotions.md)**  
Create a sales promotions campaign. OEM agent can view all the sales promotion campaign created by them.
-   **[View all promotions](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/manufacturing/mco-all-promotions.md)**  
As an OEM agent, view all the promotions that you created.
