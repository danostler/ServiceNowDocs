# Using generative AI in Now Assist for Manufacturing Commercial Operations \(MCO\)

If you have an agent role, you can summarize the report details with the Now Assist for Manufacturing Commercial Operations \(MCO\) application.

## Skills reuse

By default, all skills exist in the global domain. When you use Now Assist in a domain-separated environment, users are only able to access data within their domain. For example, if a user uses the summarization skill, Now Assist only uses material that exists within the user's domain when generating that summary. Additionally, there is no co-mingling of data for domain-separated instances when using generative AI skills. The data resides only on the instance, and the shared services used for generative AI do not persist any requests \(prompts\) and responses.
