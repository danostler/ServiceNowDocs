# Cause action plans

Create a cause action plan for the remediation action.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **Workspaces** &gt; **CSM/FSM Configurable Workspace** &gt; **List** &gt; **Remediation Action Plan** &gt; **Cause action plan**.

2.  Select **New**.

3.  On the Cause Action Plans form, fill in the fields.

4.  Select **Save**.
