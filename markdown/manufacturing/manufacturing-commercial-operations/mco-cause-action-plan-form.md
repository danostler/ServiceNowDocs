# Cause action plan form

Cause action plan form enable you to add the cause action details for remediation plan.

|Field|Description|
|-----|-----------|
|Number|Cause action plan number is automatically generated. The number starts with IC and incremented for every new report.|
|External id|Select the external id which is used for mapping with external systems or imported data.|
|Short description|Enter a brief description of the issue cause.|
|Description|Enter a detailed description of the issue cause.|
