# My promotions

Agents workspace enables OEM to create the sales promotions campaigns. OEM agent can view all the sales promotion campaign created by them.

## Before you begin

Role required: manufacturing operations admin \(sn\_mfg\_cmn.manufacturing\_operations\_admin\) or sales promotion manager \(sn\_sales\_prm\_mgmt.sales\_promotion\_manager\)

## Procedure

1.  Navigate to **Workspaces** &gt; **CSM/FSM Configurable Workspace** &gt; **Lists** &gt; **Sales Promotion Management** &gt; **My Promotion**.

2.  Select **New**.

3.  On the My Promotion form, fill in the fields.

    For a description of the field values, see [My promotions form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/manufacturing/manufacturing-commercial-operations/mco-my-promotions-form.md).

4.  Select **Save**.

5.  Select **Create checklist**.

    For further instruction, see [Create a checklist template](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/manufacturing/manufacturing-commercial-operations/mco-create-checklist.md).

6.  Select **Publish**.

    Published promotions cannot be modified.

7.  To reuse the sales promotion details, select **Copy**.


-   **[Create a checklist template](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/manufacturing/manufacturing-commercial-operations/mco-create-checklist.md)**  
Agents workspace enables the manufacturer to create a checklist for verification.
