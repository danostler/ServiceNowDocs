# Reject all claims

Rejects all the claims job expenses raised by a dealer.

## Before you begin

Role required: claims agent

## Procedure

1.  Navigate to **Workspaces** &gt; **CSM/FSM Configurable Workspace** &gt; **Lists** &gt; **Repair**.

    **Note:** To learn how to create and navigate to the Review &amp; approve form, see [Create a repair claim](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/manufacturing/manufacturing-commercial-operations/mco-create-repair-claim.md).

2.  From the Activities menu, select **Review &amp; approve** from **Activities**.

3.  From the drop-down list, select **Reject all**.

    All the claims are rejected. the Claim Summary form is displayed.

4.  View the repair claim summary details, select **Claim Summary**.

5.  Select **Submit**.

6.  Add **Comment** and **Confirm**.
