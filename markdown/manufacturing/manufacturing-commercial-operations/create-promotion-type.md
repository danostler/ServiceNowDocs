# Create promotion type

As a OEM or manufacturer, create a sales promotion type that is applicable to a specific set of promotions.

## Before you begin

Role required: manufacturing operations admin \(sn\_mfg\_cmn.manufacturing\_operations\_admin\) or sales promotion manager \(sn\_sales\_prm\_mgmt.sales\_promotion\_manager\)

## Procedure

1.  Navigate to **Workspaces** &gt; **CSM/FSM Configurable Workspace** &gt; **Lists** &gt; **Sales Promotion Setup** &gt; **Promotion Types**.

2.  Select **New**.

3.  On the form, fill in the fields.
