# Repair form

Includes all the forms related to repair and recall.

-   **[Campaign tasks form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/manufacturing/manufacturing-commercial-operations/mco-campaign-tasks-form.md)**  
Campaign tasks form enables you to add the recall campaign details.
-   **[Channel partner form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/manufacturing/manufacturing-commercial-operations/mco-cp-form.md)**  
Use the fields on the channel partner form to manage and store information related to channel partners.
-   **[Corrective action form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/manufacturing/manufacturing-commercial-operations/mco-corrective-action-form.md)**  
The corrective action form enables the OEM to address the asset issue, either by eliminating or replacing the asset.
-   **[Corrective action charges form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/manufacturing/manufacturing-commercial-operations/mco_corrective_action_charges_form.md)**  
Corrective action charges form include the expenses related to addressing a non-conformance like labor charges.
-   **[Claim details form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/manufacturing/manufacturing-commercial-operations/mco-claim-details-form.md)**  
The Claim details form enables the dealer to enter the claim details for the repair work performed.
-   **[Install based item form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/manufacturing/manufacturing-commercial-operations/install-based-item-form.md)**  
An install base item refers to an application service configuration item. Each install base item references the configuration item record for the customer. If issues arise with a configuration item, customer service agents can trace which install base items that the case is affecting.
-   **[Internal business location form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/manufacturing/manufacturing-commercial-operations/mco-internal-business-location-form.md)**  
Internal Business Location \(IBL\) form to store information about an internal business location. This information includes the staff members assigned to that location, cases, sold products, and install base items created for customers.
-   **[MCO input attributes form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/manufacturing/manufacturing-commercial-operations/mco-input-atrributes_form.md)**  
MCO input attributes form.
-   **[Phase task form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/manufacturing/manufacturing-commercial-operations/mco-phase-task-form.md)**  
Phase task form enables you to add the details of the recall phase.
-   **[Recall campaign form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/manufacturing/manufacturing-commercial-operations/mco-recall-campaign-form.md)**  
The recall campaign form provides the original equipment manufacturer \(OEM\) with the required information to initiate a recall campaign.
-   **[Recall campaign phase form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/manufacturing/manufacturing-commercial-operations/mco-recall-campaign-phases-form.md)**  
The recall campaign phase form enables you to add the recall campaign phases details.
-   **[Repair claim form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/manufacturing/manufacturing-commercial-operations/repair-claim-form.md)**  
The repair claim form enables you to enter the warranty and recall claim details.
-   **[Pre-authorization form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/manufacturing/manufacturing-commercial-operations/mco-pre-auth-form.md)**  
The pre-authorization form enables you to enter the pre-authorization details.
-   **[Pre-authorization job details form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/manufacturing/manufacturing-commercial-operations/mco-pre-auth-job-details-form.md)**  
The pre-authorization job form enables you to enter the warranty, labor, and part details.
