# General form

General forms used in MCO.

-   **[Effectiveness form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/manufacturing/manufacturing-commercial-operations/mco-effectiveness-form.md)**  
This form enables you to add the effectiveness information.
-   **[Discuss form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/manufacturing/manufacturing-commercial-operations/mco-discuss-sidebar.md)**  
The Start a Sidebar discussion form enables you to start a discussion.
-   **[Notes form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/manufacturing/manufacturing-commercial-operations/mco-notes.md)**  
Notes form enables you to add the watch list details.
-   **[Resolution information form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/manufacturing/manufacturing-commercial-operations/mco-resolution-information-form.md)**  
This form enables you to add the report closing details.
