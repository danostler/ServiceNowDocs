# Create promotion questionnaire

Create a promotion questionnaire. Configure the MCO input set.

## Before you begin

Role required: manufacturing operations admin \(sn\_mfg\_cmn.manufacturing\_operations\_admin\)

## Procedure

1.  Navigate to **Workspaces** &gt; **CSM/FSM Configurable Workspace** &gt; **Lists** &gt; **Sales Promotion Setup** &gt; **Promotion Questionnaires**.

2.  Select **New**.

3.  On the form, fill in the fields.
