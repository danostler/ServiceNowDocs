## Navigating the Console pages

The Discovery Console for OT is organized into multiple pages navigated by the side navigation panel. Each page is designed to help you accomplish specific tasks. Every page contains information and step-by-step procedures for using the Discovery Console for OT to query and inventory in your OT environment.
