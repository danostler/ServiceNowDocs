# Create a user

Create a user account that can access the Discovery Console for OT.

## Before you begin

Role required: user

## Procedure

1.  Navigate to the Users page.

2.  Select the add icon \[Omitted image "add-icon-msi.jpg"\] Alt text:.

3.  On the form, fill in the following fields.

    |Field|Description|
    |-----|-----------|
    |Username|Username for the user account.|
    |Email|Email for the user's account.|
    |Password|Secure password for the user account.|
    |Confirm Password|Retype the password.|
    |Roles|Roles that the user account needs. If needed, you can select more than one role.|
