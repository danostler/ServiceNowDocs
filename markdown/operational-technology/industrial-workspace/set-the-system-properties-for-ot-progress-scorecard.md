# Set the system properties for the Operational Technology Progress Scorecard

Set the system properties for the Operational Technology \(OT\) Progress Scorecard so that you can enable the properties as needed.

## Before you begin

Role required: configure\_industrial\_workspace

## Procedure

1.  Navigate to **All** &gt; **Industrial Workspace Admin** &gt; **Workspace System Properties**.

2.  Edit the following properties as needed.

<table id="table_m2l_kgy_4bc"><thead><tr><th>

System property

</th><th>

Description

</th></tr></thead><tbody><tr><td colspan="2">

OT Progress Scorecard configuration

</td></tr><tr><td>

sn\_mfg\_common.otpsc.page\_title

</td><td>

The page title shown on the OT Progress Scorecard. The default is `OT Progress Scorecard`.

</td></tr><tr><td>

sn\_mfg\_common.otpsc.records\_per\_page

</td><td>

The number of records shown per page. The default is `10`.

</td></tr><tr><td>

sn\_mfg\_common.otpsc.low\_threshold\_color

</td><td>

A color to represent the low threshold in the OT Progress Scorecard.

</td></tr><tr><td>

sn\_mfg\_common.otpsc.medium\_threshold\_color

</td><td>

A color to represent the medium threshold in the OT Progress Scorecard.

</td></tr><tr><td>

sn\_mfg\_common.otpsc.high\_threshold\_color

</td><td>

A color to represent the high threshold in the OT Progress Scorecard.

</td></tr></tbody>
</table>3.  Select **Save**.
