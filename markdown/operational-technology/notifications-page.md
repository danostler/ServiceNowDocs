# Notifications page

The Notifications page shows the notifications available for your Discovery Console for OT Sensors.

You can also add a filter to sort and find Notifications related to the following categories.

-   End Date
-   Rule
-   Rule Type
-   Severity
-   Status

To filter the notifications, do the following.

1.  Select the plus icon \[Omitted image "filter-plus-icon.png"\] Alt text:.

    The **Select Filter Type** field opens.

2.  Select a filter from the drop-down menu.

\[Omitted image "notifications-page-filter.png"\] Alt text: Notifications filter

The Notifications are automatically sorted by the selected filter and displayed on the page.
