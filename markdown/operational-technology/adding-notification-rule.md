# Add a Notification rule

Create a Notification Rule.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to the Notification Rules page in your Console.

2.  Select the add icon \[Omitted image "add-icon-msi.jpg"\] Alt text:.

3.  On the form, fill in the following fields.

    |Field|Description|
    |-----|-----------|
    |Identification|Name and description for the Notification rule.|
    |Type|Rule type. You can choose from Console Disk Usage, Device Disk Usage, and Heartbeat.|
    |Parameters|Parameters to edit the utilization threshold as needed.|
    |Actions|Add an action for the notification rule from the available list. You can display a message, email, or phone contact. You can also send an email, SMS email, or Syslog. You can add multiple actions to your notification rule.|
    |Notifications|Settings to edit the severity, pop-ups, and suppression of time.|

    1.  Confirmation

        Confirm your selections in the previous sections.

4.  Select **Create Rule**.
