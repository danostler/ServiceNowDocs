# Import Sites

In SGC Central the next step is to import the Sites.

## Before you begin

Role required: admin

## Procedure

1.  Before importing assets, you must import sites.

    \[Omitted image "sgc-central-import-sites.png"\] Alt text: Import sites

2.  Open the **Site Scheduled Data Import** job; that is, select the site displayed under **Scheduled Data Import**.

3.  The Configure import schedule window opens for that site.

    \[Omitted image "sgcc-configure-import-schedule-2.png"\] Alt text: Configure import schedule

4.  Select the **Execute Now** button to manually import.

    \[Omitted image "execute-import-schedule.png"\] Alt text: Execute the import schedule

5.  Repeat these steps for additional sites.

6.  Select **Continue** to move to the next step.
