# Edit a user's information

Edit a User's information and keep the user account up to date.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to the Users page.

2.  Select the user account that you want to modify.

3.  Select **Edit**.

4.  Edit the available fields as needed.

5.  Select **Save**.
