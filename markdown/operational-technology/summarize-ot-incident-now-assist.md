# Summarize an Operational Technology incident by using Now Assist for Operational Technology Service Management \(OTSM\)

Quickly understand the Operational Technology \(OT\) incident context and respond to a user's inquiries by using the OT incident summarization skill in the Now Assist for Operational Technology Service Management \(OTSM\) application.

## Before you begin

Role required: sn\_ot\_incident\_write

## About this task

An OT incident summary provides you with a concise summary of an incident. The summary is based on the OT incident state and is generated from the following information:

-   Short description
-   Description
-   Work notes
-   Additional comments
-   Priority
-   Site
-   Equipment model entity
-   OT device
-   OT incident task

## Procedure

1.  Navigate to **All** &gt; **Industrial Workspace**.

2.  Open the Industrial Workspace list view.

3.  On the OT Incidents module, select one of the available lists.

4.  Open the OT incident record that you want to summarize.

5.  In the OT incident record, select the **Summarize** button.

6.  When you're finished summarizing an OT incident, you can add it to the work notes, expand or collapse it, provide feedback, copy it, or view information about it.
