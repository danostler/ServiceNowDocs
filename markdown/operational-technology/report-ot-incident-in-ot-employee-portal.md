# Report an OT incident in the Employee Center for OT

If you're not an Operational Technology \(OT\) incident user and need to report an OT incident, report the incident in the Employee Center for OT.

## Before you begin

Role required: cmdb\_ot\_viewer

## About this task

As a user without an OT incident role, you can still report an OT incident as long as you're assigned the **cmdb\_ot\_viewer** role. By reporting the OT incident in the Employee Center for OT, an OT incident record is created in the Industrial Workspace. This request is available on both desktop and the Now Mobile app experience.

## Procedure

1.  To report an OT incident on desktop, complete the following steps.

    1.  Navigate to **All** &gt; **Self-Service** &gt; **Employee Center**.
    2.  Select the **Industrial services** tab.
    3.  On the Industrial services page, select the **Report OT incident**.
    4.  On the form, fill out the fields.

        |Field|Description|
        |-----|-----------|
        |Short description|Brief description of the incident.|
        |Site|Site where the OT incident occurred.|
        |Equipment model entity|Affected equipment model entity. This field is optional.|
        |OT device|Affected OT device.|
        |Description|Detailed explanation of the incident. This field is optional.|
        |Add attachments|Attachments needed for the OT incident, if any.|

    5.  To save the incident as a draft and come back to it later, select **Save as draft**.
    6.  When you're ready to submit the incident, select **Submit**.
2.  To report an OT incident on the Now Mobile app, complete the following steps.

    1.  Open the Now Mobile app.
    2.  Select **Help center**.
    3.  Under Browse all topics, select **Industrial Services**.
    4.  Follow the same steps as you would in the desktop experience above to submit the request or save it as a draft.

## Result

When you report the OT incident, an overview of the incident is provided along with a link in the activity stream to view the incident in the Industrial Workspace.

**Note:** When you view the incident in the Industrial Workspace, you have read-only access and cannot edit the record fields. You can only add comments or work notes.

You can also view your request in the Employee Center by selecting **My Requests**. Viewing your request on the requests page lets you see any work notes or activities saved to your submitted request.
