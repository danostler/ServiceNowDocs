# Summarize a subflow

Generate a summary of a subflow with Now Assist for Creator.

## Before you begin

Role required:

-   now.assist.creator
-   flow\_designer or admin

## Procedure

1.  Navigate to **All** &gt; **Process Automation** &gt; **Workflow Studio**.

2.  On the homepage, select **Subflows**.

3.  Open the subflow that you want to summarize.

4.  From the More Actions menu, select **Subflow summary**.

5.  From the Subflow summary dialog box, select **Summarize**.

    \[Omitted image "subflow-summary-03.png"\] Alt text: An example summary of the Change - Implementation tasks subflow.


## Result

Now Assist for Creator generates a summary of the subflow. The subflow summary also displays its creation date.

## What to do next

\[Omitted image "subflow-summary-04.png"\] Alt text: Refresh summary option on the Subflow summary dialog box

You can regenerate the subflow summary by selecting the **Refresh summary** icon next to the summary creation date.
