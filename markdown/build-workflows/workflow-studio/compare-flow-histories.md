# Compare flow histories

Compare two flow histories to see what content was added, removed, and updated.

## Before you begin

Role required: flow\_designer or admin

## Procedure

1.  Navigate to **All** &gt; **Process Automation** &gt; **Workflow Studio**.

2.  On the homepage, select **Flows**.

3.  Open the flow whose flow history entries you want to view.

4.  Select **Edit flow**.

5.  From the sidebar view, select the **History** icon.

    The sidebar view displays a list of flow history entries.

6.  From the Flow history list, select the flow history entry that you want to use as the comparison starting point.

    By default, the flow compare uses the most current flow history entry as the comparison starting point.

7.  From the Flow History list, select another flow history entry that you want to compare to the currently selected flow history entry.

8.  From the More actions menu of the selected Flow history card, select **Compare Entry**.

    The flow comparison view opens with the starting history entry on the right side, and the compared flow history entry on the left side.

9.  Navigate through the flow history entries to see what content was added, removed, and changed.

10. Expand changed flow components by selecting anywhere in the row of the flow step.
