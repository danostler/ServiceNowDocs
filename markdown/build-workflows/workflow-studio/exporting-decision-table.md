# Export a Decision table from an Instance to another using update sets

Export a decision table from one instance to another using update sets. This process reduces the time required to recreate the table across different instances.

## Before you begin

Role required: admin

## Procedure

1.  Exporting the Decision table using the update set
2.  In the source instance, navigate to **All** &gt; **System Update Sets** &gt; **Local Update Sets**.

3.  From the list, select the decision table update set to export.

4.  Under **Related Links**, select **Export to XML**.

5.  Navigate to **All** &gt; **Decision Tables**.

6.  From the list, select the decision table you want to export.

7.  Select **Export** to export the decision table to an Excel workbook.

    The decision table is ready to be imported to target instance.
