# Flow history compare view

Compare two flow history entries in a side-by-side view. Use the step highlighting and change type icons to determine what flow components have been added, removed, and changed.

The flow history compare view enables you to perform these actions.

-   Compare two flow history entries side-by-side.
-   See flow components that were added to a flow history entry.
-   See flow components that were removed from a flow history entry.
-   See flow components that were updated in a flow history entry.
-   Select a different flow history entry on either side of the flow compare view.
-   Save a copy of a flow history entry as a new flow.
-   Restore a flow history entry as the current configuration of a flow.
-   Delete a flow history entry.

## Flow history compare view components

\[Omitted image "compare-flow-history-view-01.png"\] Alt text: Flow history compare view

The Flow history compare view consists of these components within a flow.

-   **1. Go back to flow view**

    Option to return to the flow editor view.

-   **2. Name of first history entry**

    Name of the first flow history entry as a unique date and time stamp.

-   **3. First flow history options**

    Options to delete, save as copy, and restore the first flow history entry.

-   **2. Name of second history entry**

    Name of the second flow history entry as a unique date and time stamp.

-   **3. Second flow history options**

    Options to delete, save as copy, and restore the second flow history entry.

-   **6. Changed flow components indicators**

    Visual indicators for changed flow components. Changed flow components are highlighted in blue and have the changed item icon. \[Omitted image "compare-history-changed-item-icon.png"\] Alt text: Changed items icon

-   **7. Removed flow components indicators**

    Visual indicators for removed flow components. Removed flow components are highlighted in red and have the removed item icon. \[Omitted image "compare-history-removed-item-icon.png"\] Alt text: Removed items icon

-   **8. Added flow components indicators**

    Visual indicators of an added flow component. Added flow components are highlighted in green and have the added item icon. \[Omitted image "compare-history-added-item-icon.png"\] Alt text: Added items icon


\[Omitted image "compare-flow-history-view-02.png"\] Alt text: Flow history compare view of changed items

-   **9. Expanded changed item view of first history entry**

    Configuration of inputs for the first history entry. Changed flow components are highlighted in blue and have the changed item icon. \[Omitted image "compare-history-changed-item-icon.png"\] Alt text: Changed items icon

-   **10. Expanded changed item view of second history entry**

    Configuration of inputs for the second history entry. Changed flow components are highlighted in blue and have the changed item icon. \[Omitted image "compare-history-changed-item-icon.png"\] Alt text: Changed items icon

-   **11. Input that changed in first history entry**

    Configuration settings of a changed input in the first history entry.

-   **12. Input that changed in first history entry**

    Configuration settings of a changed input in the second history entry.
