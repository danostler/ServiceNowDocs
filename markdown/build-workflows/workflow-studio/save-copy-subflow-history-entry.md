# Save a copy of a subflow's flow history entry

Create a copy of a flow history entry and save it as a new subflow.

## Before you begin

Role required: flow\_designer or admin

## About this task

You can save a copy of a flow history entry to preserve a previous subflow configuration or as a data backup.

## Procedure

1.  Navigate to **All** &gt; **Process Automation** &gt; **Workflow Studio**.

2.  On the homepage, select **Subflows**.

3.  Open the subflow whose previous configuration you want to save as a copy.

4.  Select **Edit subflow**.

5.  From the sidebar view, select the **History** icon.

6.  From the list of flow history entries, select the entry whose configuration you want to copy.

    Workflow Studio displays the configuration of flow history entry in the Flow History view.

7.  From the Flow History view, select **Save as copy**.

    \[Omitted image "subflow-history-save-copy-01.png"\] Alt text: Flow configuration for 5th June, 5:17AM with an option to Save as copy

8.  In **New subflow name**, enter the name of the copied subflow.

9.  In **Application**, select the application scope to copy the subflow.

10. From the Create a copy of this flow confirmation dialog box, select **Copy**.

    \[Omitted image "subflow-history-save-copy-02.png"\] Alt text: Create a copy of this subflow dialog box. The new name is Wait 3 day for email reply.


## Result

\[Omitted image "subflow-history-save-copy-03.png"\] Alt text: Workflow Studio tab displays new fsublow Wait 3 Days for Email Reply.

Workflow Studio displays the copied subflow configuration in a new tab.
