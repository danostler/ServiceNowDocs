# Save a playbook variant as a favorite

Save a playbook variant as a favorite for quick reference.

## Before you begin

Role required: admin or playbook.admin

## Procedure

1.  Navigate to **All** &gt; **Workflow Studio**.

2.  Open the playbook with the variant that you want to save as a favorite.

3.  Select the variant icon \(\[Omitted image "playbook-variants-icon.png"\] Alt text: variant icon\) to open the variant panel on the left.

4.  Hover over the variant in the variant panel, and select the kebab menu to open the action menu.

    \[Omitted image "search-favorite-playbook-variants.gif"\] Alt text: Setting the Manager variant as the favorite, and searching for the IC variant with the Quick Find search bar

5.  Set the variant as a **Favorite**.

    **Tip:** You can also just search for variants in the **Quick Find** search bar.
