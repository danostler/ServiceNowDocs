-   **[Manage Journey designer task updates in bulk](https://www.servicenow.com/docs/access?context=jny-dsgnr-bulk-task-update&family=washingtondc&ft:locale=en-US)**

Update similar tasks in similar journeys in a single action. Tasks must be of the same type and in journeys of the same type. The current task types available for bulk updates are:

    -   Mark when complete \[mark\_when\_complete\]

Enable journeys to support a change of manager.

-   **[Base instance demo data of Journey designer to support multiple journeys](https://www.servicenow.com/docs/access?context=parental-loa&family=xanadu&ft:locale=en-US)**

Leverage the parental leave guide shipped as part of Parental Leave demo data to enhance the employee experience.

-   **[Return to the Journey details page after a task is completed](https://www.servicenow.com/docs/access?context=jny-dsgnr-employee-journey-tasks&family=xanadu&ft:locale=en-US)**

Navigate back to the Journey details page after task completion, avoiding the need to go through the breadcrumbs.

-   **[Delete subsequent tasks when a journey is deleted](https://www.servicenow.com/docs/access?context=jny-dsgnr-mngr-mntr-quick-links&family=xanadu&ft:locale=en-US)**

Delete a journey and all the subsequent tasks at the same time.

-   **[Generate onboarding ramp-up agentic workflow](https://www.servicenow.com/docs/access?context=onboarding-ramp-up-plan-agentic-wf&family=zurich&ft:locale=en-US)**
