# Supported query operations

The following encoded query operators are supported for creating context objects.

-   "&gt;="
-   "!="
-   "&lt;="
-   "="
-   "&gt;"
-   "&lt;"
-   "IS\_NOT\_NULL"
-   "ISNOTEMPTY"
-   "NOT\_CONTAINS"
-   "EMPTYSTRING"
-   "STARTSWITH"
-   "ANYTHING"
-   "ENDSWITH"
-   "ISEMPTY"
-   "IS\_NULL"
-   "CONTAINS"
-   "NOT LIKE"
-   "BETWEEN"
-   "LIKE"
-   "DYNAMIC"
-   "NOT\_IN"
-   "IN"
-   "ON"
-   "INSTANCEOF"
