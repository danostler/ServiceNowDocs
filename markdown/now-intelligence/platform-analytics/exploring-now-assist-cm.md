# About Dashboard Summarization

The Dashboard Summary is your AI companion for providing instant insights into your dashboards.

## Dashboard Summary overview

The objective of the Dashboard Summary is to provide insights into your dashboards including highlights, sources of information on the dashboard, and what has changed since the last analysis.

\[Omitted image "explore-nacm.png"\] Alt text: A Now Assist Dashboard Summary

A Dashboard Summary is divided into the following sections:

-   **Data scope**

    The scope is either the entire dashboard or a dashboard tab.

    The Dashboard Summary element includes content for the entire dashboard. When the summary is generated by the Context Menu element and this element sits above the tabs on a dashboard, summaries are generated for the entire dashboard. If the context menu element is on an individual tab, the summary applies only to that tab.

-   **Summary**

    An AI-generated summary of the dashboard's content in a single paragraph.

-   **Key highlights**

    The information from the dashboard's elements that informs the summary. Each highlight has one or more citations and a notation of why the highlight matters. Select the citation to see which dashboard elements support the highlight.

-   **What changed**

    Details of the differences between the previous summary request and the latest request as long as the dashboard has the same configuration.

    **Note:** Due to data retention policies, if a summary is generated more than 45 days after the previous execution, the **What changed** section isn't shown.

-   **Refine**

    Enables you to shorten, elaborate, or change the tone of a generated summary.


## Dashboard Summary users

Any authenticated user can use the Dashboard Summary on a dashboard. When the plugin is installed, the associated skills are activated by default.

**Note:** Admins can restrict the users who can use the Dashboard Summary to specified roles. For more information, see [Configure the Now Assist context menu in Now Assist Experiences](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/now-intelligence/configure-db-summarization-skill-nacm.md).

## Dashboard Summary benefits

-   AI-Guided insights: Gain insights into dashboard content using AI-generated summaries.
-   Seamless integration with all Platform Analytics experience dashboards. Trigger the Dashboard Summary from any dashboard that contains it. Add it to any dashboard you own to make these insights available to your users.
