# Edit a conversion funnel

Edit conversion funnel data to collect or steps to include in your analysis.

## Before you begin

Role required: Users with the analytics\_viewer, portal\_analytics\_viewer, mobile\_analytics\_viewer, core\_ui\_analytics\_viewer, now\_experience\_analytics\_viewer roles can create, edit and delete funnels for the applications they have access to.

## Procedure

1.  Navigate to **Platform Analytics** &gt; **Usage Insights**.

2.  From the Applications list, select the application with the funnel you want to edit.

3.  Select **Analysis** &gt; **Conversion funnel**.

4.  Select the Actions menu \[Omitted image "more-actions-menu-icon.png"\] Alt text: Actions menu icon of the funnel you want to edit and choose **Edit**.

    \[Omitted image "uxa-funnel-edit.png"\] Alt text: Conversion funnel module with Actions menu and Edit option highlighted

5.  Change the funnel name or description, and edit or add steps as applicable.

6.  Select **Save** to update the funnel configuration.
