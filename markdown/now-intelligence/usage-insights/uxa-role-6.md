# Web analytics viewer \[web\_analytics\_viewer\]

View Usage Insights in the application navigator. Assigned users have viewer role permissions for Usage Insights for web applications.

## Contains Roles

List of roles contained within the role.

-   portal\_analytics\_viewer
-   core\_ui\_analytics\_viewer
-   now\_experience\_analytics\_viewer

## Groups

List of groups this role is assigned to by default.

None.

## Special considerations

None.
