# Working with improvement opportunities

Set improvement opportunities to find areas of improvement by which you could optimize a process. You can either select from a list of rules available to you or set your own rules. Based on the rules set by you, you can view your areas of improvement.

Create improvement opportunities for the following:

-   A process table: Improvement opportunities for a process table is set for the table. You can add them to the projects that are based on that table by adding it from the library.
-   A project: Improvement opportunities for a project are specific for the project.

There are two types of improvement opportunities:

-   Rule-based improvement opportunities: These improvement opportunities consist of one or more rule chains executed in a sequence. The records that match the logic of these specifications are classified as a match for the improvement opportunities.

-   Automated improvement opportunities: The improvement opportunities are displayed based on the default patterns that are already available for selection.


There are two UI locations from where you can create the improvement opportunities:

-   Classic view
-   Finding Builder

-   **[Set improvement opportunities for process tables](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/now-intelligence/process-mining/impr-opp-table.md)**  
Set rule-based or automated improvement opportunities for a process table from the Classic view or Finding Builder.
-   **[Setting improvement opportunity for projects](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/now-intelligence/process-mining/project-imp-opp.md)**  
Set improvement opportunities for projects to find areas of improvement by which teams could optimize a process. You can either select from a list of rules available to you or set your own rules. Based on the rules set by you, you can view your areas of improvement.
