# Run intent and activity analysis

Generate a intent and activity analysis on a node.

## Before you begin

Role required: sn\_process\_mining\_analyst, sn\_process\_mining\_power\_user, or sn\_process\_mining\_admin

## About this task

You can run an intent and activity analysis from Analyst workbench or Opportunity details tab. When running from Analyst workbench, it provides information about the node that is selected. When running from Opportunity details tab, it uncovers the details of the automation opportunities across end-to-end processes.

**Note:** Intent and activity analysis is available for a record count between 50 and 300K records.

## Procedure

1.  Navigate to **All** &gt; **Workspaces** &gt; **Process Mining Workspace**.

2.  From the list of projects, select a project.

3.  Select any node from Analyst workbench.

    A window is displayed with all the details.

    \[Omitted image "intent-activity.png"\] Alt text: Iintent and activity analysis

4.  From the **Investigate** section, select **Intent and activity analysis**.

    Alternately, select an improvement opportunity and then select **Intent and activity analysis** from the **Investigate** section.

    \[Omitted image "intent-ana-imp-opp.png"\] Alt text: Intent and activity analysis from Opportunity details tab

    A scheduled task starts.

5.  After the scheduled task is complete, select **View result**.

    The result is displayed.

    \[Omitted image "intent-activity-ana-result.png"\] Alt text: Intent and activity analysis result

    Use the thumbs up and thumbs down icons to provide feedback for the result that you have received.
