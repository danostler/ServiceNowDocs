# Activate work notes analysis skill

Activate the transition work notes analysis skill to be able to use Now Assist for Process Mining to analyse work notes.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Now Assist Admin** &gt; **Skills**.

2.  Select **Creator**.

3.  Select **Work Notes Analysis** from the list of Now Assist skills for Creator.

4.  Select **Turn on** for the Transition Work Notes Analysis skill.

    \[Omitted image "na-skills.png"\] Alt text: Transition work notes analysis skill activated

    The skill is activated for your instance.
