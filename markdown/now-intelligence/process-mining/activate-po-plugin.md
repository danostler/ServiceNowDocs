# Access Process Mining

Process Mining is available with ServiceNow AI Platform. With the free version, you can do only sample mining. You need a license to do full mining.

To purchase a subscription, contact your account manager.

**Note:** Before purchasing a subscription, you can evaluate the features using the Process Mining evaluation projects available for ITSM, CSM, and HR. For more information, see [Evaluation projects](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/now-intelligence/process-mining/evaluate-pm.md).

For information on components installed with Process Mining, see [Components installed with Process Mining](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/now-intelligence/process-mining/components-installed.md).

**Important:** You may explore content packs that provide you with pre-defined template for projects, process configuration, improvement opportunities, and so on. To download and import the content packs, see [Creating process configurations using content pack templates](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/now-intelligence/process-mining/content-pack-config.md).

If you plan to activate and explore Process Mining in a sub-prod or clone instance, then clone from production with copy audit and logs data. You can do this by deselecting the **Exclude audit and log data** when cloning. Taking this step ensures that audit logs are available to create a full process map using Process Mining.
