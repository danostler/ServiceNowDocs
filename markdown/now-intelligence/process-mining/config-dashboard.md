# Configure Process Mining map in PAR dashboard

Configure a Process Mining map to view the process graph in the PAR dashboard.

## Before you begin

Role required: sn\_process\_optimization\_power\_user or sn\_process\_optimization\_admin

## Procedure

1.  Navigate to your PAR dashboard.

2.  Configure a Process Mining map.

    1.  Select **Add new element**.

    2.  Select **Process Mining - Map** from the list.

    \[Omitted image "dashboard-pm.png"\] Alt text: Configure Process Mining map

3.  Configure a Process Mining project for the map.

    1.  Select a Process Mining project from the **Process mining project** list in the **Configuration** area.

    \[Omitted image "dash-pm-config.png"\] Alt text: Configure Process Mining project

4.  Customize the orientation of the outline of the process graph set for the current project workbench for nodes within the graph.

    For example, when using a vertically oriented widget, you might prefer the graph and star diagram to be displayed in a vertical layout rather than a horizontal one.

    \[Omitted image "dash-pm-orientation.png"\] Alt text: Graph orientation

5.  If you have configured filters in your dashboard, and you want the Process Mining map to reflect the applied filters, then select the **Follow filters** button.

    \[Omitted image "dash-pm-filter.png"\] Alt text: Filter

6.  Select **Save** and **Exit editing mode**.

    For more information about Platform Analytics \(PAR\) dashboards, see [Exploring Platform Analytics dashboards](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/now-intelligence/ac-elements.md).


## Result

The Process Mining map is available in the PAR dashboard.
