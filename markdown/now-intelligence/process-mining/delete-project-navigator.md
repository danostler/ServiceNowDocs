# Delete a project

Delete a project if you are not using it and want to clean up data. Deleting a project deletes the project's configurations and versions or projects it has generated. You can delete a project from the Process Mining Workspace.

## Before you begin

Role required: sn\_process\_optimization\_power\_user or sn\_process\_optimization\_admin

If you have the sn\_process\_optimization\_analyst role, you can delete only those projects that you have created.

## Procedure

1.  Navigate to **Workspaces** &gt; **Process Mining Workspace**.

2.  Select the ellipses beside the project that you want to delete.

3.  Select **Delete** from the list.

4.  Select **Delete** on the confirmation dialog box to delete the project.
