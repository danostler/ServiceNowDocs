# Request Process Mining for external data

Request Process Mining for external data application to import external data and work with it within ServiceNow® instance.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to the **All** &gt; **System Application** &gt; **All Available Applications** &gt; **All**.

2.  Find the application using the filter criteria and search bar.

    You can search for the application by its name or ID. If you cannot find an application, you may have to request it from the ServiceNow® Store.

3.  Select **Install**.
