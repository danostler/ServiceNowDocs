# Duplicate a visualization in the Visualization Designer

Duplicate a visualization so that you can use aspects of the source in a copy of the visualization.

## Before you begin

Role required: viz\_creator

## Procedure

1.  Navigate to **All** &gt; **Platform Analytics** &gt; **Library** &gt; **Data Visualizations**

2.  Select the visualization that you want to duplicate.

3.  From the More actions menu \(\[Omitted image "icon-vert-3dot-p.png"\] Alt text:\), select **Duplicate**.

4.  Give the duplicate a unique name and select **Duplicate**.


## Result

The visualization is copied with the new name and the suffix **- Copy**. All other aspects of the visualization remain the same.

**Important:** The new copy of the visualization is automatically saved.
