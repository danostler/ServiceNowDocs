# Platform Analytics UI artifacts

Core UI and Next Experience UI artifacts are located under Platform Analytics Administration. Users with the admin role can navigate from these components to the associated tables.

## Core UI artifacts

Navigate to **All** &gt; **Platform Analytics Administration** &gt; **Core UI Artifacts** to manage aspects of your Core UI content.

-   **Dashboards**

    Opens `pa_dashboards.list`. Select any dashboard to open its form.

    Select one or more entries and choose **Actions on selected rows** &gt; **Migrate dashboard** to migrate them to Platform Analytics experience dashboards.

-   **Widgets**

    Opens `pa_widgets.list`. Select any widget to open its form.

    Select one or more entries and choose **Actions on selected rows** &gt; **Migrate PA widget** to migrate them to Platform Analytics experience indicator visualizations.

-   **Reports**

    Opens `sys_report.list`. Select any report to open its form.

    Select one or more entries and choose **Actions on selected rows** &gt; **Migrate report** to migrate them to Platform Analytics experience data visualizations.

-   **Interactive Filters**

    Opens `sys_ui_hp_publisher.list`. Select any filter to open its form.

    Select one or more entries and choose **Actions on selected rows** &gt; **Migrate filter** to migrate them to Platform Analytics experience interactive filters.


## Next Experience UI artifacts

Navigate to **All** &gt; **Platform Analytics Administration** &gt; **Next Experience Artifacts** to manage aspects of your Next Experience content.

-   **Dashboards**

    Opens `par_dashboard.list`. Select any dashboard to open its form.

    Select one or more entries and choose **Actions on selected rows** &gt; **Rollback** to return them to Core UI.

-   **Data Visualizations**

    Opens `par_visualization.list`. Select any visualization to open its form.

-   **Filters**

    Opens `par_component_filter.list`. Select any filter to open its form.
