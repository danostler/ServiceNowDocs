# Edit a data visualization in the Visualization Designer

After you create a data visualization, you can return to it and edit the data visualization's appearance and other aspects.

## Before you begin

Role required: You can edit any data visualization you own unless it is a shared visualization, in the library. If the data visualization is in the library and you own it, you need the itil, report\_user, or viz\_creator role to edit it. You can edit any data visualization with the viz\_admin role or higher.

## Procedure

1.  Navigate to **All** &gt; **Platform Analytics** &gt; **Library** &gt; **Data Visualizations**, or open an in-line dashboard and select **Edit**.

2.  Select the visualization that you want to edit.

3.  In the Visualization Designer, select the Configuration icon \(\[Omitted image "icon-ac-config.png"\] Alt text: Configuration icon\) to edit header, data, and presentation options.

4.  Select the information icon \(\[Omitted image "icon-info-ac.png"\] Alt text: Info icon\) to edit name, description, certification, and requester information.
