# View KPI Details

From a list of indicators, select an indicator to open its KPI Details page.

## Before you begin

Role required: None. However, upgraded instances may require pa\_viewer.

## About this task

The following actions open KPI Details:

## Procedure

-   Select an indicator in the Platform Analytics indicator library, **Platform Analytics** &gt; **Library** &gt; **Indicators**.

    For an indicator to appear in that list, the **Show in library** setting in the Access Control tab of that indicator record must be selected \(true\).

-   Select a point on the graph of an indicator in a data visualization.

    While by default this opens KPI Details, the specific data visualization might be configured not to.

-   If a query in Analytics Q&amp;A returns an indicator, click on an indicator score to open the indicator in KPI Details.

-   In an application built with Next Experience Components, a component such as Indicator Scorecard may link to KPI Details.


## Result

The KPI Details page on the selected indicator opens.
