# Data collected by Task Mining

Task Mining collects workstation and user categories of data.

<table id="table_gb1_33d_zcc"><thead><tr><th>

Data category

</th><th>

Details collected

</th></tr></thead><tbody><tr><td>

Workstation user data

</td><td>

-   User login names
-   Domain names
-   IP addresses
-   Workstation names

</td></tr><tr><td>

Workstation user activity

</td><td>

-   Timestamps
-   Description of activities

</td></tr><tr><td>

Screen information

</td><td>

-   Name of screens
-   Possible personal information depending on the app

</td></tr><tr><td>

Screen content

</td><td>

-   Information displayed on a web screen
-   GUI objects
-   Possible sensitive data like contract numbers and client names
-   **Important:** Task Mining doesn't collect password field data.


</td></tr></tbody>
