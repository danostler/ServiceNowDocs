# Submit a data collection request

Enable data collection for workstation users for a specific time range by completing a data request from the Employee Center.

## Before you begin

Role required: sn\_tm\_core.analyst, sn\_tm\_core.power\_user, sn\_tm\_core.admin

## Procedure

1.  Navigate to **All** &gt; **Self-service** &gt; **Employee Center**.

2.  Search for and select **Data Request**.

3.  On the form, fill in the fields.

4.  |Field|Description|
|-----|-----------|
|Name|Name for the request.|
|Description|Description for the request.|
|Date from|Date to start collecting workstation user data.|
|Date to|Optional date to stop collecting workstation user data.|
|Workstation users|Name of the workstation users you want to collect workstation data from.|

5.  Select **Request**.

6.  On the confirmation form, select a user from the **Request for** menu to submit the request on behalf of another user.

7.  Select **Request**.


## Result

A data request goes to every manager of each workstation user for approval.

## What to do next

If you are an approver, [Approve workspace user data collection](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/now-intelligence/task-mining/approve-workspace-user-log-requests.md).

If you want to track the progress of data log requests, see [Follow data request progress](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/now-intelligence/task-mining/manage-data-log-requests.md).
