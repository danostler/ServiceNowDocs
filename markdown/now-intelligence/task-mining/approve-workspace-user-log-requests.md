# Approve workspace user data collection

Approve requests from Task Mining analysts to enable workstation user data collection.

## Before you begin

Role required: approver\_user or one of the Task Mining roles \(sn\_tm\_core.analyst, sn\_tm\_core.power\_user, sn\_tm\_core.admin\)

## About this task

A request for approval is sent to every manager of each workstation user that data collection was requested for.

## Procedure

1.  Navigate to **All** &gt; **Self-service** &gt; **My Approvals**.

2.  Select the request to see its details.

3.  Accept or reject the request.

    -   Enable data logging for the specified workstation users for the defined dates by selecting **Approve**.
    -   Reject the data logging request by entering a reason in the **Comments** field and selecting **Reject**.

        **Note:** A reason is required when rejecting a data request.
