# Task Mining key terms

Key terms used in Task Mining.

<table id="table_ptz_c54_hcc"><thead><tr><th>

Term

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Task Mining application

</td><td>

The interface for managing data collection settings and system behavior, which handles the processing of collected data and provides dashboards.

</td></tr><tr><td>

Task Mining agent

</td><td>

A service installed on a user's workstation that logs certain events from desktop applications.

</td></tr><tr><td>

Data processing server

</td><td>

A multi-tenant cloud component that receives data from the Task Mining agents deployed across user workstations.

</td></tr><tr><td>

Data request

</td><td>

A request made to a workstation user's supervisor to approve the collection of workstation user logs, user data, and user activity.

</td></tr><tr><td>

Workstation user log

</td><td>

The record of events and activities that occur when a task is performed, including workstation user data, workstation user activity, screen information, and screen content.

</td></tr><tr><td>

Workstation user data

</td><td>

Data includes workstation user login name, domain name, IP address, and workstation name.

</td></tr><tr><td>

Workstation user activity

</td><td>

Data includes timestamp and description of activity.

</td></tr><tr><td>

Screen information

</td><td>

Data includes the name of the screen and possible personal info depending on the app.

</td></tr><tr><td>

Screen content

</td><td>

Data includes GUI objects but not password fields.

</td></tr><tr><td>

Application categories

</td><td>

A breakdown of the workstation data collected, including activity type, application categories, application, and activity.

</td></tr><tr><td>

Masking

</td><td>

A data processing technique that replaces the original value of fields with altered values.

</td></tr><tr><td>

Anonymization

</td><td>

A data processing technique that removes or modifies personally identifiable information.

</td></tr><tr><td>

Categorization

</td><td>

Categorization rules determine how information from apps and windows is shown in analyses.

</td></tr><tr><td>

DataMart

</td><td>

A datamart is the mined data with categorization rules applied.

</td></tr></tbody>
