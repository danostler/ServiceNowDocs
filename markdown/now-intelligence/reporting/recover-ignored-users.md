# Reassess ignored users

When you address the ACL Assessment for Reports, you can ignore any individual user. View these users to consider their statuses again.

## Before you begin

Roles required: admin and security\_admin.

## Procedure

1.  Navigate to **All** &gt; **Reports** &gt; **ACL Assessment for Reports** &gt; **Ignored Reports and Users List**.

2.  Select the row with the Ignored User.

3.  From the **Actions on select rows** list, select **Delete**.

    \[Omitted image "rep-acl-reassess-ignored-users.png"\] Alt text: Ignored users and reports list with ignored user and Delete action selected


## Result

The ACL Assessment for Reports application includes the users you deleted from this report the next time you run the assessment.
