# Data visualization details

The Details pane gives you an overview of key information about the data visualization that you are currently editing.

Select the **Details** icon \[Omitted image "analytics-center-db-vd-details.png"\] Alt text: Details icon in the Visualization designer to see the details of the visualization you are editing. You can see the following information:

<table id="table_i1d_x3v_q5b"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

The name should be meaningful. If you have editing rights, you can change the name here.

</td></tr><tr><td>

Description

</td><td>

The description should be information that helps other users understand the nature and purpose of the visualization. If you have editing rights, you can change the description here.

</td></tr><tr><td>

Mark as certified

</td><td>

If you have a role that lets you certify a data visualization for use in your organization, certify or decertify the visualization here. For more information, see [Certify a data visualization in the Visualization Designer](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/now-intelligence/certify-dv-ac.md).

</td></tr><tr><td>

Requested by

</td><td>

If a user on the instance requested that you create or edit this visualization, you can name the person here. The user is considered a responsible user. The data visualization is shared with them, with editing rights.

</td></tr><tr><td>

Created by, Last updated by

</td><td>

The users who created and last updated this visualization

</td></tr><tr><td class="sub-head" colspan="2">

Usage

</td></tr><tr><td>

*n* dashboards

</td><td>

If the visualization is used on any dashboards, the number of dashboards displays here.

 The **View your dashboards** link opens a simple list of those dashboards that you created or were shared with you and that contain this data visualization. The number of dashboards listed can be smaller than the total number of dashboards that contain the visualization.

</td></tr><tr><td>

Shared with

</td><td>

Icons for every user who the visualization is shared with. Point to an icon to view the user's full name and the sharing details. For more information, see [Share a data visualization in the Visualization Designer](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/now-intelligence/share-dv-ac.md).
