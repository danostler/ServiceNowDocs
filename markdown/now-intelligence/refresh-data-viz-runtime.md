# Refresh a data visualization as a viewer

A viewer of a data visualization can refresh that data visualization without refreshing the page or having editing rights.

You can refresh a visualization through the visualization's More actions menu.

\[Omitted image "refresh-viz.png"\] Alt text: Refresh option in the More options menu of a data visualization.

**Note:** If the creator of a visualization has suppressed this option, you will not see Refresh under the More actions menu.
