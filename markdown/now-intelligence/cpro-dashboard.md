# Chief Procurement Officer \(CPRO\) Dashboard

The Chief Procurement Officer Dashboard provides a consolidated view of the enterprise procurement performance across spending, requisitions, sourcing operations, negotiations, and contract life cycle management.

The CPRO dashboard serves as a centralized workspace for tracking end-to-end procurement health, enabling leaders to optimize spend, manage requisition flow, ensure timely sourcing execution, and maintain contract discipline. It brings together critical indicators such as year-to-date spend, savings achieved, open requisitions, sourcing backlog, negotiation outcomes, and contract renewals or expiration.

This dashboard visualizes essential procurement insights including requisition aging, sourcing request trends, negotiation timeliness, department-level sourcing delays, contract renewal pipelines, and spend categorization. These metrics help procurement executives identify bottlenecks, improve cycle time, enhance spend governance, and maximize value from procurement operations.
