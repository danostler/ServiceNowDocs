# List widgets

List widgets show the scores of multiple indicators.

</td><td>

For several indicators, shows any of the following attributes:-   Scores at several points in time
-   Changes between the last two scores
-   % Change
-   Trends
-   Bullet charts

</td></tr><tr><td>

\[Omitted image "spider-icon.png"\] Alt text: Spider list widget icon

</td><td>

Shows a plot of the latest score of each indicator on its own axis, starting with 0 at the center.

</td></tr></tbody>
</table>## Permission to view indicators

On upgraded instances, you might need the pa\_viewer role to view List widgets. Furthermore, List widgets do not show indicators that you do not have permission to read.
