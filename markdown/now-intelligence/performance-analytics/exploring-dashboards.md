# Exploring Responsive dashboards

Learn more about dashboards with a sample workflow and reviewing the benefits it can provide for different users.

## Dashboards overview

Core UI Responsive Dashboards show reports, Performance Analytics indicator widgets, interactive filters, and other Core UI components of the story of the data on your instance.

Owners of Core UI responsive dashboards have the option to migrate these dashboards to Platform Analytics experience. For more information, see [Migrate dashboards that you own](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/now-intelligence/data-migration-migrate-dbs-you-own.md).

**Important:** Responsive dashboards cannot show Platform Analytics components such as data visualizations. For more information, see [Platform Analytics experience](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/now-intelligence/par-workspace.md).

## Benefits of Responsive Dashboards

|Benefit|Feature|Users|
|-------|-------|-----|
|View a list of dashboards and choose dashboards to see up-to-date information about your instance in the context of related data.|[Find a responsive dashboard](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/now-intelligence/performance-analytics/find-resp-db.md)|There are no specific permissions required to work with dashboards. You must have permission to view at least one dashboard to see the Dashboards Overview and the dashboard picker.|
|Create a dashboard to tell a story with current instance data that you can share.|[Create or configure a responsive dashboard](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/now-intelligence/performance-analytics/t_CreateADashboard.md)| |
