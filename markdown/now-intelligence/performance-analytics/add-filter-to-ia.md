# Add a filter to Interactive Analysis

Add a filter to show more refined information in your Interactive Analysis.

## Before you begin

Role required: none

You must have access to the list of records that you want to analyze.

## Procedure

1.  Navigate to the table that you want to analyze.

2.  Right-click on the header of the column you want to analyze and select **Launch Interactive Analysis**.

3.  Click the filter icon \(\[Omitted image "List\_FilterIcon.png"\] Alt text: filter icon\) and click **Add filters**.

    Use the **Search Filters** bar to limit the number of filters displayed.

    \[Omitted image "ia-add-filters.png"\] Alt text: Add filters panel

4.  Select the filter to add.

5.  Click **Apply Filters**.

    \[Omitted image "apply-filters.png"\] Alt text: Filters panel with Filter icon, Add Filters button, and Apply Filters button highlighted.


## Result

The new filter is applied to the interactive analysis. Click the information icon \(\[Omitted image "PreviewIcon.png"\] Alt text: Info icon\) to view the filters in the source filter summary.
