# Import a KPI Composer project

If you have an exported KPI Composer project, you can import it to your instance.

Prerequisites
:   [Create a KPI Composer project](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/now-intelligence/performance-analytics/create-kpi-composer-project.md)

## Before you begin

Role required: sn\_kpi\_composer.admin

## Procedure

1.  Navigate to **All** &gt; **KPI Composer** &gt; **Import project**.

2.  Select **Choose file** and navigate to the exported project in your file system.

3.  Select **Import**.

    If you already have a copy of the same project, the import overwrites it.


## Result

You now can open the project in KPI Composer.
