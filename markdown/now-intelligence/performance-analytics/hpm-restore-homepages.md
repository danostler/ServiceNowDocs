# Restore a homepage

Restore retired homepages as dashboards.

## Before you begin

You may find that the stakeholders of a retired homepage still want its information. You can restore that retired homepage as a responsive dashboard.

Role required: admin or dashboard\_admin.

## Procedure

1.  Navigate to **All** &gt; **Self Service** &gt; **Service Catalog**.

2.  Select **Can we help you**.

3.  Select **Restore Homepage**.

4.  Enter the name of the retired homepage or use the search icon \(\[Omitted image "icon-search.png"\] Alt text: lookup icon\) to find the retired homepage.

5.  Select **Submit**.


## Result

The retired homepage is converted to a dashboard. The Homepage Migration Status table shows these changes.

-   The Dashboard column is populated with the name of the new dashboard.
-   The entry in the State column for the restored homepage is changed to Closed complete.
-   The entry in the Decision column is changed to Restored.
