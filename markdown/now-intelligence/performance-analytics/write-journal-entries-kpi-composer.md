# Write journal entries for a project

Keep track of your KPI Composer project with journal entries

Prerequisites
:   [Create a KPI Composer project](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/now-intelligence/performance-analytics/create-kpi-composer-project.md)

## Before you begin

Role required: Anyone with access to a KPI Composer project, including view access, can write a journal entry.

## Procedure

1.  Click the journal icon \(\[Omitted image "kpi-composer-journal-icon.png"\] Alt text:\).

2.  Write a journal entry.

3.  Click **Post**.


## Result

All posted journal entries appear in descending chronological order.
