# Performance Analytics \(Indicator data sources\)

ServiceNow® Performance Analytics is an in-platform process optimization solution that utilizes indicators \(KPIs\) to answer key business questions. Performance Analytics can help increase quality and reduce the costs of service delivery.

## Get started

Performance Analytics tools including KPI Composer, KPI Details, and Spotlight, empower administrators and other stakeholders with real-time data to address current status and accurately evaluate trends.
