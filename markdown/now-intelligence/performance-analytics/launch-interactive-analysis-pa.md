# Launch Interactive Analysis

Launch Interactive Analysis from any list.

## Before you begin

Role required: none.

You must have access to the list of records that you want to analyze.

## Procedure

1.  Navigate to any list.

2.  Configure the columns that are displayed on the list.

    The columns that appear on the list when you launch Interactive Analysis determine which fields are included in the analysis. The included fields determine which **Group by** and **Stack by** options are available, and which interactive filters appear by default.

3.  Right-click the column header for a reference, choice, date/time, integer, or boolean field and select **Launch Interactive Analysis**.

    The column that you launch Interactive Analysis from is used as the default **Group by** value.

4.  Change how data is aggregated by selecting different values in the **Group by** and **Stack by** choice lists, or filter the data by applying one or more interactive filters.

5.  Drill down into a subset of the data by clicking a visualization, such as a bar in the bar chart or a cell in the heatmap.
