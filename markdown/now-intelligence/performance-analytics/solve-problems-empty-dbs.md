# Solve problems with empty dashboards

When a dashboard shows an empty page with an empty dashboard selector, the dashboard name may include special characters.

## Before you begin

This problem happens when the dashboard name includes tab or newline characters or ASCII characters that aren't letters or numbers.

Role required: admin

## Procedure

1.  Navigate to `pa_dashboards_list.do`.

2.  Edit the title fields of problem dashboards to remove any special characters.


## Result

The dashboard page shows the expected content and a populated dashboard selector.
