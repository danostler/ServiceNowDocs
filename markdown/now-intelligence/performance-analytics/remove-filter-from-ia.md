# Remove a filter from Interactive Analysis

You can remove a filter from Interactive Analysis and specify whether to remove the filter element from Group by and Stack by lists in the analysis.

## Before you begin

Role required: none.

You must have access to the list of records that you want to analyze.

## Procedure

1.  Navigate to the table that you want to analyze.

2.  Right-click on a column header and select **Launch Interactive Analysis** or open a dashboard that you own to show the interactive filters.

3.  Click the filter icon \(\[Omitted image "List\_FilterIcon.png"\] Alt text: filter icon\) to show the interactive filters.

4.  Point to the filter you want to remove and click the **Remove filter** icon.

    \[Omitted image "icon-remove-if.png"\] Alt text: remove filter icon


## Result

The filter is removed from the Interactive Analysis.
