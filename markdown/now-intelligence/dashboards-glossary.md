# Dashboard terms

Dashboards in Platform Analytics and the classic environment use terms that describe how data is organized, displayed, and shared.

## bookmark

A saved reference to a dashboard that you can access quickly from the Dashboards library. Bookmarked dashboards appear in a dedicated list in the library overview.

## category

A label assigned to one or more dashboards to support searching and filtering in the Dashboards library. Users with the analytics\_categories\_admin role create and manage categories. Dashboard owners apply categories to their dashboards.

## certification badge

An indicator on a dashboard that shows the dashboard is company-approved and recommended for use. Only users with the admin or dashboard\_admin role can certify a dashboard.

## dashboard

A canvas for organizing and sharing data visually. Dashboards contain data visualizations, filters, and other visual elements. Users can share dashboards with other users, groups, or roles to create a shared view of data.

## data visualization

A graphic representation of table, indicator, or other instance data placed on a Platform Analytics dashboard. Data visualizations replace reports and Performance Analytics widgets from the Core UI. Types include bar charts, dial charts, gauge charts, geomaps, heatmaps, pivot tables, single scores, and time series charts.

## element

A visual object placed on a Platform Analytics dashboard. Elements include data visualizations, dividers, filters, filter groups, headings, images, lists, Now Assist Context Menus, Process Mining maps, rich text, workbenches, and funnels.

## filter

A dashboard element that narrows the data shown in visualizations based on specified criteria. Filters can be placed on individual tabs or above the tabs so they apply to all tabs. Filter types include data filters and domain filters.

## filter group

A set of filters that can be applied simultaneously to multiple visualizations on a dashboard. Use a filter group when multiple filters apply to multiple visualizations.

## inline editor

The built-in dashboard editing tool in Platform Analytics. The inline editor enables users with edit rights to add, resize, and rearrange elements directly on the dashboard canvas without leaving the dashboard view.

## library

The central location in Platform Analytics where users access dashboards, data visualizations, and indicators. The library shows all content the user has permission to view, including content they own, content shared with them, and bookmarked content.

## tab

A named section within a dashboard used to organize related content. A dashboard can have multiple tabs. Filters placed above the tabs apply to all tabs, while filters placed on a tab apply only to that tab's content.

## top stage

The area of a tabbed dashboard that is visible regardless of which tab is active. Elements placed in the top stage, including filters, apply to content on all tabs.

## widget

An object added to a Core UI dashboard. Widget types include reports, Performance Analytics visualizations, interactive filters, and content blocks. Widgets are not available on Platform Analytics dashboards; on migrated dashboards, widgets are displayed in iframes.
