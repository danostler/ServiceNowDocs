# Certify a Platform Analytics dashboard

Certify a dashboard to indicate that it is company-approved and recommended for use.

## Before you begin

Role required: admin or dashboard\_admin

## Procedure

1.  Navigate to **All** &gt; **Platform Analytics** &gt; **Library** &gt; **Dashboards**

2.  Select the dashboard you want to certify.

3.  Select the information button \(\[Omitted image "icon-info.png"\] Alt text: Info button\) to open the Details panel.

4.  Select **Edit**.

5.  Select the toggle labeled **Mark as certified**.

6.  Select **Save**.


## Result

In the Dashboards library, the dashboard's entry is marked **Certified** in the Certified column.
