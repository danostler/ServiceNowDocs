# Execute an update set scan

Use update set scan to execute applicable checks against records that are affected by the update set.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **System Update Sets** &gt; **Local Update Sets**.

2.  Select the desired update set from the list.

3.  Click **Scan Update Set** from the **Related Links** list.
