# View import history

View your import history by accessing the **My Application Import History** module.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **System Applications** &gt; **My Application Import History**.

    The list of import history records in your instance shows up.

2.  Select one of the imports from the list for more information about that import.

    The form view of sys\_upgrade\_history record shows up.

    **Note:** To learn more about importing from Source Control, see ServiceNow Studio. Once the importing is done, use the **My Application Import History** module to view the import history.
