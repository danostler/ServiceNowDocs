# Browser settings and performance

Enable data compression through your web browser to avoid slow performance on pages with large data sets.

Web pages in your instance might include large data sets, especially in tables with many rows and columns. To improve speed on pages that require large amounts of data, most browsers can accept compressed data. Data compression reduces the size of data transmitted between the server and the client to improve transfer speed and bandwidth use.

Compression is enabled by default on all ServiceNow application servers and in many web browsers. Review your specific browser settings to verify that data compression is supported and enabled.
