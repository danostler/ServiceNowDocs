# Create an operational toggle

Create an operational toggle to maintain system run levels that throttle the application operation. Operational toggles enhance your users' experience by tailoring resource consumption to system run levels.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **System Run Level** &gt; **Operational Toggles**.

2.  Select **New**.

3.  On the Operational Toggles form, fill in the fields.

    For a description of the field values, see [Operational toggles form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-administration/platform-performance/operational-toggles-form.md)

4.  Select **Submit**.
