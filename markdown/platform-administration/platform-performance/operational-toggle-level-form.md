# Operational toggle level form

A description of the fields in the Operational Toggle Level form.

|Field|Description|
|-----|-----------|
|Toggle|The preexisting toggle that you want to create levels for.|
|Short Description|A brief description of the desired level's function.|
|Application|The scope that the toggle level will apply to. This could be either a specific application or the global scope.|
|Value|Behavior of the toggle level.|
