# Uninstalling applications

Application Manager can be used to uninstall applications that are no longer in use on your instance.

## Overview of using the Application Manager to uninstall

Some applications can be uninstalled from your instance to remove scripts, background jobs, and roles that are no longer needed. In some cases, it's possible to keep the associated fields and data after uninstalling the application that generated them.

**Note:** Some applications and plugins can't be uninstalled after they've been installed.

-   **[Uninstall an application](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-administration/application-manager/uninstall-application-app-mgr.md)**  
Uninstall an unused application to remove unnecessary or conflicting scripts, background jobs, and roles from your instance.
