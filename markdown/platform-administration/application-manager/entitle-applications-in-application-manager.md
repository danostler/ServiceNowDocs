# Entitle an application

Entitle an application from your ServiceNow AI Platform instance if the application has been purchased but not yet linked to the instance.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Admin Center** &gt; **Application Manager**.

2.  From the filter options, select the **Instance not entitled** indicator.

    **Note:** The **Instance not entitled** filter is only displayed if your instance has an application that has been purchased and not entitled.

3.  Select the menu icon on the application tile, and select **Entitle this instance**.

4.  In the Application entitlement dialog box, select **OK**.


## Result

The application is entitled and can be installed.
