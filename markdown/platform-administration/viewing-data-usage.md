# Analyzing data usage on your instance

View a summary of storage consumption and analyze data usage on your current instance over time in the Data Management Console.

## Overview of analyzing data usage

-   [Monitor cloud capacity in Subscription Management](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-administration/monitoring-cloud-entitlements.md)

    View a summary of your cloud storage entitlements and track storage capacity across all the instances associated with your account in Subscription Management.

-   [Data Management overview](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-administration/data-management-overview.md)

    Monitor the current size of the instance and track cloud storage consumption. For example, if you purchased shared storage for your instances, you can determine how much storage is consumed from the pool by your current instance and which tables are consuming the most amount of data.

-   [Viewing all tables](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-administration/data-management-tables.md)

    Access a list of the tables on your instance and gain insight into the growth of these tables over the last thirty days. View the number of active data management rules that are configured on these tables.

-   [Viewing data usage by table](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-administration/data-management-table-data.md)

    Access a summary of data usage over time for specific table. Determine which tables are driving volume to the current table and how much volume the current table is contributing to associated tables.

-   [Viewing table rules](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-administration/data-management-table-rules.md)

    Track the backlog of records in a table that meet the requirements of a data management rule but haven't been processed yet. View a summary of rule execution history on a table.

-   [Viewing rule activities](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-administration/data-management-rule-activities.md)

    Analyze a specific rule configured on a table. Track the backlog of records that meet the rule's conditions but haven't been processed yet, and view a detailed summary of the rule's execution history.
