# Exploring Admin Center

Admin Center gives a centralized control of the instance to the admins with easy access to the platform capabilities, applications, and configurable admin dashboards.

## Admin Center overview

Admin Center is a comprehensive platform management solution, offering access to platform capabilities, application discovery, and intelligent insights to drive informed decision-making.

## Admin Center users

Admin Center has the following users.

|Users|Description|
|-----|-----------|
|System administrator \[admin\]|Admin role is required for the access to all applications and features in the instance.|

|Benefit|Feature|Users|
