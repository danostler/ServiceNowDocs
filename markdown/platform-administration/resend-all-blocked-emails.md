# Resend all blocked emails

Unblock the bounce email address and resend all blocked emails.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **System Logs** &gt; **Bounce Email Addresses**.

2.  Select the record from the list.

3.  In the record, update the State field to **Unblock**.

4.  Select **Save**.

5.  To send all blocked emails, select **Resend All Blocked Emails** from **Related Links**.

6.  To resend a single or multiple blocked emails, select **Blocked Emails** tab.

    1.  Hover on the email and select the email or emails.

    2.  From the **Actions on selected rows** drop-down, select **Resend Blocked Emails**.
