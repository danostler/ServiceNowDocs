# Delete a custom notification

Delete custom notifications that you don't need any more.

## Before you begin

Role required: none

## Procedure

1.  From your user profile, navigate to **Preferences** &gt; **Notifications** and then select **Advanced Preferences**.

2.  Select the **Custom Notifications** tab.

3.  In the notification entry, select the more actions icon \( \[Omitted image "delete\_cn.png"\] Alt text: More actions icon\) and select **Delete**.
