# AI Search Admin console Home page

The AI Search Admin console Home page is where you can oversee all AI Search activity. You can set up and manage AI Search across applications, track activity, and get more information on the AI Search feature.

\[Omitted image "ais-admin-01.png"\] Alt text: The AI Search Admin console Home page with 7 callouts highlighted. For a text description, see the table that follows.

|Field|Description|
|-----|-----------|
|1. Navigation bar|Tabs to navigate between Home, Applications, and Shared Configurations pages.|
|2. Learn more about AI Search|Link that opens documentation for AI Search.|
|3. Launch|Button that opens Now Assist UI page where you can set up Now Assist in AI Search for various applications.|
|4. Set up and manage AI Search in your applications|Sample row of applications that have AI Search prebuilt. Select a given card to open the application's configuration view. Select **See all applications** to open the Applications page view.|
|5. Track AI Search activity|Panel showing a summary of AI Search configurations such as **Indexed documents by month** or **Queries by search profile**. Select the More actions icon \(\[Omitted image "more-actions-icon.png"\] Alt text: More actions icon.\) to refresh the view on a given configuration. Select **Go to full dashboard** to open the full dashboard forAI Search configurations.|
|6. Add features to AI Search|List of apps and plugins that expand AI Search capability. Select **Install** on a given app to begin the setup.|
|7. Introduction|Sidebar with AI Search overview video, Helpful Resources links to AI Search documentation and community forum, and FAQs.|
