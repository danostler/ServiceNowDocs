# Searching in AI Search

See how AI Search displays search results, refinement filter options, and Genius Result answers. Learn how to search using the AI Search query language. Understand when and how AI Search automatically resubmits queries.

**Note:** For more details on using AI Search features in global and workspace search, see [Using global search with AI Search for Next Experience](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-administration/ai-search/using-ais-next-experience-app.md).

-   **[AI Search query language](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-administration/ai-search/query-language-ais.md)**  
Learn how to construct search queries using terms, phrases, and AI Search query operators.
-   **[Viewing search results in AI Search](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-administration/ai-search/viewing-search-results-ais.md)**  
Learn about the components that AI Search uses to display results for search queries.
-   **[Change the sort order for your search results](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-administration/ai-search/sort-search-results-ais.md)**  
Choose the order in which AI Search displays results for your search. By default, the most relevant search results appear first. You can override this behavior and view search results ordered by their field values.
-   **[Automatic search query resubmission](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-administration/ai-search/auto-query-resubmission-ais.md)**  
When a search query returns too few results, AI Search automatically modifies and resubmits it with the goal of returning at least the desired number of results.
-   **[Fuzzy numeric search in AI Search](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-administration/ai-search/fuzzy-numeric-search.md)**  
Fuzzy numeric search lets you find indexed records by their Number field values without needing to enter alphabetic prefixes or leading zeroes.
-   **[Provide feedback on Genius Results](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-administration/ai-search/genius-result-feedback-ais.md)**  
Users can submit feedback for each Genius Result answer card, specifying whether it was helpful for their search or not. For cards from default Genius Result configurations, users can also select the reason they found an answer unhelpful. AI Search records feedback responses as search signals that you can use to assess the relevancy of your Genius Result configurations.
