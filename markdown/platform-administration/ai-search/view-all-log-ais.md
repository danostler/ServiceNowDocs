# View all AI Search log messages

Review AI Search events by viewing log messages.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **AI Search** &gt; **AI Search Logs** &gt; **All**.

    The Log \[syslog\] table appears with a **\[Message\] \[contains\] \[ais\]** filter condition applied.

2.  Use the list search, sort, filter, and group functions to locate messages of interest.
