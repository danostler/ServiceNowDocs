# Add features to AI Search

Improve AI Search functionality by adding extra features through supported applications. Install the applications to expand AI Search capabilities.

## Before you begin

Role required: ais\_admin

## About this task

Expand AI Search functionality by installing the AI Search Integration Hub spoke and the AI Search Data Labeling and AI Search for Next Experience applications. If any of these apps are already installed, their corresponding installation card no longer appears. If all three apps are installed, the Add features to AI Search section doesn't appear.

## Procedure

1.  Navigate to **All** &gt; **AI Search Admin** &gt; **AI Search Admin Home**.

2.  In the Add features to AI Search section, select Install for the application you want to add to AI Search.
