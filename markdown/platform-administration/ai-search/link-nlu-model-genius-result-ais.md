# Link an NLU model and intents to a Genius Result configuration

Specify the NLU model and intents to use for a Genius Result configuration with an NLU trigger condition.

## Before you begin

Role required: ais\_admin

## Procedure

1.  Navigate to **All** &gt; **AI Search** &gt; **Search Query Settings** &gt; **Genius Results**.

2.  Open the Genius Result configuration that you want to link an NLU model to.

3.  In the NLU Model related list, select **Link Existing**.

4.  On the AI Search Genius Result Configuration NLU Model Mapping form, fill in the fields.

5.  Select **Submit**.


## Result

The NLU model and its selected intents appear in the NLU Models related list.

## What to do next

For an NLU trigger condition to function, the NLU model must be published. You can verify NLU model status by navigating to **All** &gt; **AI Search** &gt; **Search Query Settings** &gt; **Search Intents**.
