# Export data to XML

Export data from a source instance to an XML file.

## Before you begin

Role required: admin

## Procedure

1.  Log in to the source instance \(the instance that should send the data\).

2.  Export a record or a list of records to XML.

**Export a record to XML**

</td><td>

1.  Navigate to the record that you want to export.
2.  Select the Additional actions menu \(\[Omitted image "Form\_MenuIcon.png"\] Alt text: Additional actions menu.\).
3.  Select **Export** &gt; **XML This Record\)**.


</td></tr><tr><td id="d272697e101">

**Export a list of records to XML**

</td><td>

1.  Navigate to the list of records that you want to export.
2.  Select and hold \(or right-click\) any column header in the list.
3.  Select **Export** &gt; **XML**.
4.  When the export completes, select **Download**.


</td></tr></tbody>
</table>
