# Generate a schema map

Generate a schema map to view different parts of the database schema.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **System Definition** &gt; **Tables &amp; Columns**.

2.  In the **Table Names** pane, select a table.

3.  Click **Schema map**.

    The schema map for the selected table opens in a separate tab or window.

    If required, you can print this map from the browser.
