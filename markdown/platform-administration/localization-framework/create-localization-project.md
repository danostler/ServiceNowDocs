# Create a localization project

Create localization projects to bundle multiple requested items under one localization task.

## Before you begin

Enable Localization Projects in Localization Framework settings.

Role required: localization\_manager

## Procedure

1.  Navigate to **All** &gt; **Localization Framework** &gt; **My Projects**.

2.  Select **New**.

3.  On the form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Number|Unique project number.|
    |State|Current state of the project. Automatically it is set to **Draft**.|
    |Assigned to|Person to whom this project is assigned to.|
    |Assignment group|Group of the assigned person.|
    |Short Description|Brief description of the project.|
    |Work Notes|Notes regarding the project.|
    |Additional comments \(Customer Visible\)|Notes that are visible to the customer.|

4.  Select **Submit**.
