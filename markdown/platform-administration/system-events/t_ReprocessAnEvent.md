# Reprocess an event

You can fire the event again for testing or diagnostic purposes.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **System Logs** &gt; **Events**.

2.  Open an event.

3.  Under **Related Links**, click **Reprocess Event**.

    The event returns to the event queue.
