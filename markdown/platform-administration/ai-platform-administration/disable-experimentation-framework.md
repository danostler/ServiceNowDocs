# Turn off experimentation framework

Turn off experimentation framework to opt out of all current and future experiments.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Experimentation Framework** &gt; **All Experiments**.

2.  Select **Disable Experimentation Framework**.

3.  Select **OK**.

    **Note:** You can re-enable experimentation framework at any time.
