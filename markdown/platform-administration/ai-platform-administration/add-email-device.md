# Add a new notification delivery channel

Add new channels for email, instant message, SMS, and voice to receive notifications and set a schedule for the channel for the notifications you want to receive.

## Before you begin

Role required: none

## Procedure

1.  From your user profile, navigate to **Preferences** &gt; **Notifications** and then select **Advanced Preferences**.

2.  Select the **Delivery Channels** tab.

3.  In the relevant delivery channel type, select **+Add**.

4.  In the dialog box, select the **Channel Info** entry.

5.  Enter a channel name and the relevant details.

6.  Select **Set Schedule** and select a schedule from the **Schedule**drop-down list to schedule notifications for the channel.

7.  Select **Save**.
