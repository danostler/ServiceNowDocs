# System Diagnostics

Review System Diagnostics to monitor the health and performance of your instance.

Use the following tools and modules to diagnose the health of your instance:

-   System Events and Jobs Dashboard

    -   [System Events dashboard](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-administration/ai-platform-administration/setup-event-monitoring-dashboard.md): Use the System Events dashboard to set the filters to display information about events on your instance.
    -   [Scheduled Jobs dashboard](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-administration/ai-platform-administration/scheduled-jobs-dashboard.md): Use the Scheduled Jobs dashboard to view the details of the completed jobs at the instance level.
    \[Omitted image "dashboard\_nav.gif"\] Alt text: Image showing dashboard gif

-   [Stats Tools](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-administration/ai-platform-administration/c_StatsToolsDiagnosticsPlugin.md): Use Stats tools for performance evaluation by accessing the system activities statistics.
