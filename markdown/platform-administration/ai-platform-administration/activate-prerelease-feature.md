# Activate a pre-release feature

Activate a pre-release feature on the instance so your users can try it out and provide feedback to the product team.

## Before you begin

Before activating a feature, review the feature details, including its description, any dependencies, and the support terms that apply to pre-release features.

Role required: admin

## About this task

When you activate a pre-release feature, you can test the feature and provide feedback before it reaches general availability. Accept the terms and conditions before activating the feature.

**Important:**

Features activate directly on your instance. Confirm that activation is appropriate for your environment before proceeding. Activation is per-feature. Activating one feature does not automatically activate others listed on the Feature Preview Program page. You can deactivate any feature at any time after activation.

## Procedure

1.  Access the Feature Preview Program using one of the following ways:

    -   Select the notification when a pre-release capability is added to your instance.
    -   Navigate to **All** &gt; **Feature Preview Program**.
    -   On the admin page, select **See all pre-release features** from the onboarding modal.
    If you received a notification that a new pre-release feature was added to your instance, you can select the notification to go directly to the Feature Preview Program page.

2.  Review the feature details for the pre-release feature to activate.

    The feature detail page includes a description of the feature and information about what it includes.

3.  Select **Activate**.

4.  Review the terms and conditions.

    The consent agreement applies to this feature only, you're not providing consent for any other features listed in the program.

5.  Select the check box to confirm that you have read the terms and conditions.

6.  Select **Accept and activate**.


## Result

The feature is now activated on your production instance. Your users can now access and try out the feature.

**Note:** If you have any queries or need clarifications about the pre-release feature, reach out to the product team.

## What to do next

After activating a feature, you can contact the product team directly to share feedback.

If you decide to stop using the feature, you can deactivate it at any time. For more information, see [Deactivate a pre-release feature](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-administration/ai-platform-administration/deactivate-prerelease-feature.md).
