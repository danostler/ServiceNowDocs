# Inbound email actions

Define an inbound email action and processing to script how the system responds to an inbound email.

-   **[Inbound email actions](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-administration/ai-platform-administration/c_InboundEmailActions.md)**  
Define an inbound email action to script how the system responds to an inbound email.
-   **[Inbound email action processing](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-administration/ai-platform-administration/inbound-action-processing.md)**  
The system determines which inbound actions to run by comparing the inbound email type and inbound action conditions to the incoming email message. Certain properties are available to set the reply and forwarding prefixes in the email subject lines that your instance recognizes when processing inbound emails.
-   **[Criteria for matching email to inbound actions](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-administration/ai-platform-administration/inbound-action-type-criteria.md)**  
The system matches incoming email to the conditions of the active inbound actions.
