# Rollback a data job

Use rollback to revert changes made by a normalization or transformation data job.

## Before you begin

Role required: admin or normalizer

## About this task

You can only rollback completed data jobs.

## Procedure

1.  Navigate to **Field Normalization** &gt; **Data Jobs** &gt; **All**.

2.  Select the completed data job to rollback.

    The system displays the data normalization job record.

3.  From **Related Links**, select **Rollback**.

    The values in the **Pending Values** Related List are removed and the data job **State** indicates that it was rolled back.
