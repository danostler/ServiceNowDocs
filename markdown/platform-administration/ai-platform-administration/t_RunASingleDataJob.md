# Run a single data job

Running the data job/jobs is the fourth step in transforming a field. This topic explains how to run a single data job.

## Procedure

1.  Start the Transform application data job to apply the transform to all the appropriate records in the database.

2.  Open the Transformation record and switch the **Mode** to **Active**.

3.  Select the **Data Jobs** Related List.

4.  Run the data job using one of the following methods.

    -   In the list of data jobs, select the check box of the jobs you want to run, and then select **Start** from the **Actions** menu.

    -   Click the link in the Created column to open the data job, and then click the **Start** Related Link.

    The **State** of the jobs turns to **Queued** as it runs and to **Completed** when it finishes successfully.
