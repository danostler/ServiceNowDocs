# Create and add assignment groups

Create and add assignment groups where you want users that are part of a group to be the recipients of the provider notification.

## Before you begin

Role required: admin or notifications provider admin

## Procedure

1.  Navigate to **All** &gt; **System Notification** &gt; **Provider** &gt; **Notification**.

2.  Select the notification.

3.  Select **Groups**.

4.  Select **New** or **Edit** to add assignment groups.

    -   Select **New** to add assignment groups.

        On the form, fill in the fields.

        |Field|Description|
        |-----|-----------|
        |Name|Name of the group.|
        |Manager|Name of the manager.|
        |Group email|Group email address.|
        |Parent|Name of the parent group.|
        |Description|Description of the group.|

    -   Select **Edit** and select the groups from the collection to add to the group list.
5.  Select **Submit** or **Save**.
