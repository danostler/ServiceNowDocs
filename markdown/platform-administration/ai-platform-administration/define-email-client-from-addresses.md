# Define email client from addresses

Set an allowable email address that is displayed in the From address of a message sent from the email client.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **Email Client** &gt; **Email Client From Address** and click **New**.

2.  In the **Display name**, enter the name to be displayed in the From field of the email client message.

3.  Enter a valid **Email address** for the From address in the email client.

4.  Select **Submit**.

    The From address is added to the Email Client From Address \[sys\_email\_client\_from\_address\] table.
