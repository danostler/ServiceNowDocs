# Scopes and roles for console items

Review the following information regarding the scopes and roles required for configuration for each console item in the Platform module.

## Scopes for console items

|Console item name|Scopes|
|-----------------|------|
|Branding|Global|
|Localization|Global|
|Identity management integrations|Global|
|Inbound email|Global|
|Outbound email|Global|
|Operational data|Global|
|AI Search|Global|
|Security Settings|Global|
|Assets|Global|

## Roles for console items configuration

|Console item name|Roles|
|-----------------|-----|
|Branding|admin|
|Localization|admin|
|Identity management integrations|admin for SSO, user\_admin and import\_admin for LDAP|
|Inbound email|email\_account\_admin|
|Outbound email|email\_account\_admin|
|Operational data|admin, user\_admin for Roles assignment|
|AI Search|admin|
|Security Settings|sn\_vsc.security\_center\_viewer, sn\_vsc.se|
|Assets|admin|
