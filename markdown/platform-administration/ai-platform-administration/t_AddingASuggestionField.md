# Suggestion field type

To help users find what they are looking for, you can add a suggestion field to any form.

## Before you begin

Role required: personalize\_dictionary

## Procedure

1.  Open the form to which you're adding a suggestion field.

2.  Add a new field or locate an existing field.

3.  Right-click the field label and select **Configure Dictionary**.

4.  In the **Choice** field, select **Suggestion**.

5.  Select **Update**.
