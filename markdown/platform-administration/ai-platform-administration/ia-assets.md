# Assets

Configure the hardware and consumables to setup the assets for your ServiceNow implementation.

## Hardware

Configure the assets with intuitive workflows.

|Field|Description|
|-----|-----------|
|Display name|Name of the hardware|
|Asset tag|Tag ID of the hardware|
|Model category|Category name of the hardware|
|State|State of the hardware|
|Assigned to|Name of the assigned person|
|Cost|Cost of the hardware|

Select **Add a hardware** if you want to add a hardware to configure. You can also export the list of hardware into an excel sheet or an available file type by selecting **Export**.

## Consumables

Manage assets with intuitive work flows and life cycle visibility.

|Field|Description|
|-----|-----------|
|Display name|Name of the consumable|
|State|State of the consumable|
|Substate|Detailed state of the consumable|
|Stockroom|Name of the location where the consumable has been stocked|
|Quantity|Available quantity of the consumable in the stockroom|
|Model category|Category of the consumable|

Select **Add a consumable** if you want to add a consumable to configure. You can also export the list of consumables into an excel sheet or an available file type by selecting **Export**.
