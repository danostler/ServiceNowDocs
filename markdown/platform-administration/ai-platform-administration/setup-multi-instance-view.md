# Set up Multi-Instance View

Configure Multi-Instance View to see consolidated activity across all connected instances.

## Before you begin

Role required: clone\_admin on primary instance.

## Procedure

1.  Navigate to **All** &gt; **Clone Admin Console** &gt; **Clone Home** &gt; **Configuration**.

2.  Select **Multi-Instance View** link.

3.  Enable the **Multi-Instance View** option to view consolidated activity across all connected instances

    Clear the option to disable the Multi-Instance View feature. Re-enabling Multi-Instance View feature will restore your previous settings without losing your configuration.
