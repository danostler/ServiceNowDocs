# Register an instance for cloning

Register your target instance before requesting your clone. You can also register a new target instance on the clone request page.

## Before you begin

Role required: clone\_admin

## Procedure

1.  Navigate to **All** &gt; **Clone Admin Console** &gt; **Home**.

2.  Navigate to the **Configurations** &gt; **Clone instances**.

3.  Select **New**.

4.  Fill in the form.

5.  Select **Save**.

    The target instance is configured.
