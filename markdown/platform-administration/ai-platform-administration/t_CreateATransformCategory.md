# Create a transform category

Create a transform category to group the transform definitions together.

## Procedure

1.  Navigate to **All** &gt; **Field Normalization** &gt; **Administration** &gt; **Transform Categories**.

2.  Click **New** in the **Transform Categories Related List**.

3.  Enter the **Name** of this category and a description.

4.  Select an **Order** for this category and save the record.

    The order determines the display order of categories in lists and forms. Two Related lists appear:

    -   **Field Types**: Click **Edit** to select an existing field type for this category or click **New** to create a new field type. The normalization field types provided are:

        -   Decimal
        -   Float
        -   Integer
        -   Numeric
        -   String
        -   URL
    -   **Transform Definitions**: Click **Edit** to select the transform definitions that are included in this category.
