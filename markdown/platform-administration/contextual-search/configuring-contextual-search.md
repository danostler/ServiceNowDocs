# Configuring contextual search

You can configure different components of contextual search.

-   **[Contextual search properties](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-administration/contextual-search/contextual-search-properties.md)**  
Contextual search properties are used to control the fields that you can search from and the number of results displayed per page.
-   **[Define a search context](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-administration/contextual-search/t_DefineSearchContext.md)**  
Define where agents can search for a particular resource. You can specify that the search resources are predefined or additional resources outside the system.
-   **[Configure table for a contextual search](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-administration/contextual-search/t_DefineContextualSearchForForm.md)**  
Configure the options for the contextual search that a table performs.
-   **[Filter configuration for contextual search](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-administration/contextual-search/filter-configuration-for-cxs.md)**  
Define filter configurations to map fields from the form to provide a more targeted search result set.
-   **[Configure search resource context properties](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-administration/contextual-search/t_ApplyConfigurationToSearch.md)**  
Configure the properties to limit the search to a specific catalog or knowledge base. For example, if you want to limit the knowledge search to an IT knowledge base or a service catalog search to software service catalog, you can configure the properties.
