# View a searcher

View the details of a searcher to know which search resources it searches and which search contexts use it.

## Before you begin

Role required: admin

## About this task

ServiceNow provides pre-defined searchers. You can't create, update, or delete searchers.

## Procedure

1.  Navigate to **All** &gt; **Contextual Search** &gt; **Searchers**.

2.  Edit a searcher record.

    \[Omitted image "ContextualSearcherConfiguration.png"\] Alt text: Searcher form showing Search Resources related list.

3.  In the Search Resources related list, view the resources searched by this searcher.

4.  In the Search Contexts related list, view all the search contexts that use this specific searcher.
