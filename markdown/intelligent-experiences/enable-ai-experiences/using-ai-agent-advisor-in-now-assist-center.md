# AI Agent Advisor in Now Assist Center

Use AI Agent Advisor in Now Assist Center to automatically discover automation opportunities in your instance and deploy AI agents to implement the automations.

Now Assist Center enables you to perform the following activities using AI Agent Advisor:

-   [View your automation opportunities](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/intelligent-experiences/enable-ai-experiences/now-assist-center-view-automation-opportunities.md)

    Review the automation opportunities that AI Agent Advisor has identified for your instance.

-   [Implement an automation opportunity from Now Assist Center](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/intelligent-experiences/enable-ai-experiences/activate-automation-opportunity-now-assist-center.md)

    Deploy a matched AI agent or a new agent to automate a resolution for an identified automation opportunity.

-   [Edit an AI agent from an automation opportunity](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/intelligent-experiences/enable-ai-experiences/now-assist-center-edit-opportunity-agent.md)

    Edit an AI agent that was created for an automation opportunity.


For more information, see [Using AI Agent Advisor in Now Assist Center](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/intelligent-experiences/enable-ai-experiences/now-assist-center-using-ai-agent-advisor.md).
