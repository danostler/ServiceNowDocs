# Now Assist Admin Settings

Configure general settings for all Now Assist applications from the **Settings** tab in the Now Assist Admin console.

As you begin to explore Now Assist, use **Settings** in the Now Assist Admin console to activate the plugins. See [Install Now Assist plugins](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/intelligent-experiences/enable-ai-experiences/install-now-assist-feature-plugins.md).

Turn on the Now Assist panel, review multi-language support, data sharing and processing, and Now Assist Guardian features, and view the Now Assist account settings. You can also manage the large language model integrations under **Settings**.

-   **[Configure multilingual service for Now Assist applications](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/intelligent-experiences/enable-ai-experiences/enable-dynamic-translation-for-now-assist-applications.md)**  
Turn on multilingual service for user-entered text with native translation or Dynamic Translation in Now Assist applications.
-   **[Data sharing and processes](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/intelligent-experiences/enable-ai-experiences/now-assist-data-sharing-and-processes.md)**  
Data sharing and processes topic description.
-   **[Now Assist Guardian](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/intelligent-experiences/enable-ai-experiences/now-assist-guardian.md)**  
Now Assist Guardian is built on the ServiceNow Small Language Model \(SLM\) and monitors generative AI interactions to detect offensive content, prompt injection attacks, and sensitive topics.
-   **[Manage AI models](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/intelligent-experiences/enable-ai-experiences/manage-large-language-models.md)**  
Access and select the LLM \(large language model\) provider used for various Now Assist skills. The selection impacts all the skills within the capability.
-   **[Review Now Assist account](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/intelligent-experiences/enable-ai-experiences/review-now-assist-account-information.md)**  
Review your Now Assist license details on the Account page of the Now Assist Admin console to make sure that you're up to date on what's available to you.
