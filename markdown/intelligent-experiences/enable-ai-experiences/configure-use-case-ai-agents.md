# Create an agentic workflow

Create an agentic workflow in AI Agent Studio so that AI agents can coordinate to solve complex problems.

## Before you begin

Role required: sn\_aia.admin

## About this task

Agentic workflows solve business problems by automating processes with agentic AI. In AI Agent Studio, you must define an agentic workflow and connect it with an AI agent to execute complex goals. These goals can involve variable data or other factors that traditional automation can struggle with.

## Procedure

1.  Navigate to **All** &gt; **AI Agent Studio** &gt; **Create and manage** &gt; **Agentic workflows** and select **New**.

2.  [Define your key requirements](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/intelligent-experiences/enable-ai-experiences/define-key-requirements.md).

3.  [Define the agentic workflow security controls](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/intelligent-experiences/enable-ai-experiences/define-sec-controls-aw.md).

4.  [Add a trigger to automatically invoke your agentic workflow if a specified event occurs.](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/intelligent-experiences/enable-ai-experiences/add-trigger-aw.md).

    If you want your agentic workflow to be used only in chats, you don’t need to add a trigger.

5.  [Determine the channels to invoke your agentic workflow.](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/intelligent-experiences/enable-ai-experiences/channels-access-aw.md).


## Result

An agentic workflow is created and can be seen in the agentic workflow list in the Create and manage page.

## What to do next

-   Move to the **Testing** playground to [test your new agentic workflow](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/intelligent-experiences/enable-ai-experiences/test-aia-use-case.md) using example utterances.
-   Review the [Platform agentic workflows](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/intelligent-experiences/enable-ai-experiences/platform-use-cases.md).
