# Model Context Protocol Server Console tools

Use the following tools that are available with Model Context Protocol Server Console by default.

|Name|Category|Description|
|----|--------|-----------|
|case\_summarization|AI Skill|Get a summary of details in a case record. For more information, see .|
|email\_recommendation|AI Skill|Get a recommended response to an email. For more information, see .|
|incident\_summarization|AI Skill|Get a summary of details in an incident record. For more information, see .|
|lookup\_case\_records|Scripted REST API|This API enables looks up case records for the case\_summarization tool.|
|lookup\_incident\_records|Scripted REST API|This API looks up incident records for the incident\_summarization tool.|
|lookup\_kb\_records|Scripted REST API| |
