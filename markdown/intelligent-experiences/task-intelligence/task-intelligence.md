# Task Intelligence

Task Intelligence uses machine learning to train solutions with your data and achieve important outcomes.

## Overview of Task Intelligence

Task Intelligence enables you to set up machine learning models to automate task creation, triage, and investigation. You can then track and assess the performance of your solutions to optimize their potential to help you work more efficiently and lowering the mean time to resolve \(MTTR\).
