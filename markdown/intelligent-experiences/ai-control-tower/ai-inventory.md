# AI asset inventory

AI asset inventory includes all the AI assets such as AI models, AI systems, prompts, datasets, and MCP servers that an organization can use.

## AI asset inventory view

-   **AI systems**

    An AI system can make predictions, recommendations, or decisions that impact physical or virtual environments, with outputs or behaviors that aren’t always predetermined by the developer or user. These systems are often trained using large amounts of structured or unstructured data and may function with different levels of autonomy, or none at all, to fulfill objectives set by humans.

-   **AI models**

    AI models consist of functions and algorithms that are trained and evaluated on datasets to generate predictions or take decisions.

-   **Prompts**

    Any data, including text or images, that is used to guide, instruct, or influence an AI model’s output. Prompts can also refer to an element in a process that requires user input in a field before moving to the output screen.

-   **Datasets**

    A dataset usually consists of records and fields organized into rows and columns. These datasets are stored within a database table.

-   **MCP Server**

    Provides external functionality by exposing tools for AI agents to use.


AI assets are presented in a list format and organized by display name, provider, vendor, managed by, lifecycle phase, state, status, and risk classification.
