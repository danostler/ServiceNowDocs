# Value tab in AI Control Tower

Use the Value dashboard page to gain insights into the value realized from multiple types of AI in your organization.

The Value dashboard page contains visualizations that help you assess the value of your AI assets. Creator skills are included in the Value dashboard page. Value metrics for creator skills are aggregated across non-production and production instances.

**Note:** Depending on your license, you have access to certain application features, generative AI skills, agentic workflows, and AI agents. For more information, see [ServiceNow product tiers](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/intelligent-experiences/enable-ai-experiences/ai-native-sku-overview.md).

\[Omitted image "value-tab-aict.png"\] Alt text: Value tab in AI Control Tower.

**Note:** Starting with the March 2026 release, the Value dashboard displays insights and value calculations only for managed AI assets within your selected time period. To view insights for unmanaged assets, reconfigure them as managed assets. For AI assets that were previously managed and later reconfigured as unmanaged, the system calculates values only through the date of their status change to unmanaged.

The indicators on the dashboard are based on the value templates associated with each AI asset. See [Using value templates](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/intelligent-experiences/ai-control-tower/using-value-templates.md) for more information.

-   **Top 5 AI systems by value**

    This area of the dashboard shows the five AI systems that generated most value \(in hrs\).

    \[Omitted image "aict-value-top-5-assets-by-value.png"\] Alt text: Top 5 AI systems by value.


-   **Productivity**

    This area of the dashboard shows the estimated productivity gains \(in hours\) in the selected period as a result of using various AI assets.

    \[Omitted image "aict-value-productivity.png"\] Alt text: Productivity indicator.

-   **Average AI users**

    This area of the dashboard shows the average unique users who used an AI asset in the selected date range.

    \[Omitted image "aict-value-avg-ai-users.png"\] Alt text: Average AI users.

-   **Top 5 AI assets by value \(in hrs\)**

    This area of the dashboard shows the top 5 AI assets that delivered the most value to the users in the selected date range.

    \[Omitted image "aict-value-top-5-assets-by-value.png"\] Alt text: Top 5 AI assets by value in hours.


## Daily updates

    This area of the dashboard shows the number of AI actions per each AI asset.

-   **Daily unique users**

-   **Top 5 AI systems by value**
