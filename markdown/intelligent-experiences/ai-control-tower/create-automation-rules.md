# Create an Automation rule

Create an Automation rule to set AI assets as managed.

## Before you begin

Role required: AI steward \[sn\_ai\_governance.ai\_steward\]

## Procedure

1.  Navigate to **AI Control Tower** &gt; **Configuration** &gt; **Controls** &gt; **Automation rules**.

2.  Select **+Create new**.

3.  Enter the rule according to your requirements in the Rule name field.

4.  Enter the description of the rule in the Description field.

5.  Select the Asset and define the conditions.

6.  Select the Activate the rule check box.


## Result

An automation rule is created for AI assets.
