# Cases

You can view all the assigned and unassigned AI cases in a list view in the AI Control Tower.

## Cases list view

You can view all the cases in a list view. You can perform actions such as search, filter, refresh, and personalize list features on the list view.

Select any case from the list, select **Edit-More Actions** to assign the case to yourself, or **Delete** or **Export** the selected case. You can also select **New** to create a new AI case.
