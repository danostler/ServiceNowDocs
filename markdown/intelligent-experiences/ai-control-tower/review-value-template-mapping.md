# Review a value template mapping

Review and approve value template mapping for an AI asset.

## Before you begin

Make sure that the state of AI asset is **Deployed**, the asset life-cycle phase is **New**, and the life-cycle status is **In review**.

Roles required: AI steward \[sn\_ai\_g overnance\_ai\_steward\] and AI asset owner \[sn\_ai\_asset\_mgmt.ai\_asset\_owner\]

## About this task

The AI steward reviews and validates the AI system's Value Templates to ensure:

-   **Accurate productivity metrics:** Hours saved are correctly calculated.
-   **Valid assumptions:** Time-saving estimates and acceptance rates are realistic.
-   **Complete data:** All inputs, calculations, and expected gains are properly documented.
-   **Proper alignment:** Asset template mappings are updated as needed.

## Procedure

1.  Navigate to **Workspaces** &gt; **AI Control Tower**.

2.  From the AI Control Tower, open the AI assets view.

3.  Select an AI asset and click **Start review**.

    The AI Asset Lifecycle tab gets open and one task for Value template review is created.

4.  Select the Value Template review and approval task and open it.
