# My overview tab in AI Control Tower

The **My overview** tab in the AI Control Tower home page displays all active assets to the asset owner and the AI stewards for assets that they manage or own.

## Explore My overview tab

The **My overview** tab displays the following widgets. You can drill down into the data on each widget.

-   **Assets**

    This area displays all the assets owned by the asset owner, organized into the following categories:

    -   Generative AI
    -   Agentic AI
    -   AI prompt
    -   AI dataset
    -   AI model
-   **Assets by lifecycle status**

    This area displays the lifecycle status of the assets, organized into the following categories:

    -   Deployed
    -   In review
    -   AI steward review
    -   Ready for deployment
    -   Rejected
    -   Cancelled
-   **AI cases by state**

    This area displays the active cases categorized into **New** and **Triage**.


The **Create an AI asset** button is added to the home page to track and manage the life cycles of your AI systems, AI models, Prompts, and Datasets.

## Tasks and cases list

The **My overview** tab displays a list of tasks and cases. You can view all the active tasks that are assigned by default under tasks.

The tasks list includes the following columns:

-   Number
-   Priority
-   Short description
-   Assigned to
-   State
-   Due date
-   Asset in review

The cases list includes the following columns:

-   Name
-   Number
-   Case analyst
-   Assignment group
-   State
-   Priority
