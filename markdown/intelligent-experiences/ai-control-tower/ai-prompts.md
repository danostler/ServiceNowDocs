# Prompts

Explore the prompts in detail with examples.

A prompt is a question or instruction provided to an AI system, indicating what you want it to accomplish. Prompts are commonly used in tools like ChatGPT or image generators.

Certain AI systems, such as GenAI systems, require user prompts to function. Monitoring prompts can help analyze user interactions with the AI and the types of outputs produced.

Examples:

-   Write a poem about the moon- AI writes a poem
-   Translate hello into Spanish- Hola
-   Show red dresses under $50- Filters e-commerce results
-   Summarize this email- Creates a short summary
