# Create labeling project from template

Templates enable you to create new labeling projects quickly by reusing pre-configured project settings.

## Before you begin

Role required: sn\_data\_kit.admin or sn\_data\_kit.analyst

## About this task

A saved template preserves:

-   Project layout and field configurations
-   Data field mappings
-   Input field definitions
-   Field properties

## Procedure

1.  Navigate to **All** &gt; **Now Assist Data Kit** &gt; **Home**.

2.  There are two ways to create a labeling project from a template.

    -   1.  From the Projects tab of the Data labeling section, select **Create project**.
2.  Select **Use template** and then find the template that you want to use.
3.  Select **Create project**.
    -   1.  From the Data labeling section, select the **Templates** tab.
2.  Find and select the template that you want to use.
3.  Select **Create project**.
3.  Fill in the fields for the project details.

    |Field|Description|
    |-----|-----------|
    |Project name|The name for the project.|
    |Description|A description for the project.|
    |Labeling instructions|Instructions for how the entries should be labeled.|
    |Add labelers|The users the labeling tasks will be assigned to.|

4.  Select **Continue**.

5.  Choose the data collection that you want to add to the labeling project.

6.  Select **Continue**.

7.  Add a preprocessor.

    1.  Display name

    2.  Subflow

    3.  Input source

8.  Select **Continue**.

9.  Select the project layout.

10. Select **Continue**.

11. Review the project summary and select **Publish**.
