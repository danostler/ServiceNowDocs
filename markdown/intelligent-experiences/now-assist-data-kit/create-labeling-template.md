# Create a labeling project template

You can create a template for labeling projects to quickly assign labeling tasks in a labeling project.

## Before you begin

Role required: sn\_data\_kit.admin

## Procedure

1.  Navigate to **All** &gt; **Now Assist Data Kit** &gt; **Home**.

2.  Create a project or select a project that has already been published.

3.  Select **Create template**.

4.  Give the template a name and select **Create**.

    Your template will now appear in the list of available templates.
