# Create a labeling project from scratch

Create a labeling project to review datasets, mark data quality, categorize information, and provide structured feedback on individual records.

## Before you begin

Role required: sn\_data\_kit.admin or sn\_data\_kit.analyst

## Procedure

1.  Navigate to **All** &gt; **Now Assist Data Kit** &gt; **Home**.

2.  From the Data labeling section, select **Create project**.

3.  Select whether to create a project from scratch or from a template and select **Create project**.

4.  Fill in the fields for the project details.

    |Field|Description|
    |-----|-----------|
    |Project name|The name of the project.|
    |Description|A description of the project.|
    |Labeling instructions|Instructions for how the entries should be labeled.|
    |Add labelers|The users the labeling tasks will be assigned to.|

5.  Select **Continue**.

6.  Choose the data collection that you want to add to the labeling project.

7.  Select **Continue**.

8.  Add a preprocessor.

    1.  Display name

    2.  Subflow

    3.  Input source

9.  Select **Continue**.

10. Select the project layout.

    1.  Select the number of columns for the layout.

    2.  For each column, select Add Component.

    3.  Add a component name and select a component type.

    4.  You can select the Read only component.

    5.  Select the input source.

    6.  Select **Add**.

11. Select **Continue**.

12. Review the project summary and select **Publish**.


## Result

After your labeling project is published, you can view the tasks related to that project by selecting it from the **Projects** tab.
