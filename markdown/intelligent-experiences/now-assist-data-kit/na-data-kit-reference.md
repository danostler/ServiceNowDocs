# Now Assist Data Kit reference

Reference topics for Now Assist Data Kit.

-   **[Now Assist Data Kit roles \(sn\_data\_kit.analyst\)](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/intelligent-experiences/now-assist-data-kit/data-kit-analyst-role.md)**  
This user can edit and save ground truth in Now Assist Data Kit. This role has read only access within Now Assist Data Kit.
-   **[Now Assist Data Kit roles \(sn\_data\_kit.admin\)](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/intelligent-experiences/now-assist-data-kit/data-kit-admin-role.md)**
