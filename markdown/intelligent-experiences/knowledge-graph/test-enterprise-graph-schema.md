# Test an Enterprise Graph schema

Promote functionality by entering a query and testing the Enterprise Graph schema.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Knowledge Graph** &gt; **Knowledge Graph Designer** &gt; **Enterprise Graph**.

2.  Select **Open your last view** to load and view the recently opened Enterprise Graph schema.

    The Enterprise graph schema​ opens in the Knowledge Graph canvas page.

3.  From the toolbar, select **Test**.

    You are navigated to Test Knowledge Graph Schema page.

    \[Omitted image "test\_enterprise\_graph.png"\] Alt text: Test Enterprise Graph schema

4.  In the Query section of the Test Knowledge Graph Schema window, enter your question.

5.  Select an LLM model from the following:

    -   **NowLLM**
    -   **Claude**
    -   **NowLLM-LTS**
    -   **Gemini**
    -   **GPT**
    \[Omitted image "gpt\_models\_eg.png"\] Alt text: Select LLM Model

6.  Select an Enterprise Graph mode.

    The options are:

    -   Enterprise Graph
    -   Enterprise Graph \(Small\)
    \[Omitted image "EG\_modes.png"\] Alt text: Select Enterprise Graph mode

7.  Select **Include conversation history** check box to add the any previous chats.

8.  Select **Anchor Tables** check box to add the Anchor tables.

9.  Select **Tags** check box to add the associated tags.

    In Enterprise Graph \(Small\) mode, tags are mandatory.

10. Select **Group response by table** to sort the results.

11. Select **Show column properties in response** to view the column properties along with associated data.

12. Select **Run Query**.
