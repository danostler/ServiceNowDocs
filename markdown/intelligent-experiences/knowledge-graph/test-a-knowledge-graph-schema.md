# Test a Knowledge Graph schema

Promote functionality by entering a query and testing the Knowledge Graph schema.

## Before you begin

Role required: kg\_admin

## Procedure

1.  Navigate to **All** &gt; **Knowledge Graph** &gt; **Knowledge Graph Designer**.

    The UI displays a list of all the Knowledge Graph schema on the landing page.

2.  From the list of Knowledge Graph schemas, select the Knowledge Graph that you want to test.

    The Knowledge Graph schema​ opens in the canvas.

3.  From the toolbar, select **Test**.

4.  In the Query section of the Test Knowledge Graph Schema window, enter your question.

5.  Select an LLM model from the following:

    -   **NowLLM**
    -   **Claude**
    -   **Gemini**
    -   **GPT**
6.  Select **Include conversation history** check box to add the any previous chats.

7.  Select **Group response by table** to sort the results.

8.  Select **Show column properties in response** to view the associated properties.

9.  Select **Run Query**.
