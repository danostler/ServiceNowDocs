# ServiceNow AI Lens skill

This skill provides generative AI capabilities to read, understand, respond, and act on visual data such as hand-written texts, images, and websites and take powerful actions to boost productivity.

This skill is a Platform skill that is enabled by the admin from the ServiceNow AI Lens skill card and enables users to perform the following actions:

-   Create or update ServiceNow records by auto-filling forms using the extracted data.
-   Preview the extracted data.
-   Provide instructions to extract data in a specific way or to get a specific output.
-   Gather insights from multiple images.
