# Assign a name to a cluster

Name your clusters to help identify and organize them.

## Before you begin

-   Create a clustering solution definition or use an existing one.
-   Role required: admin or ml\_admin

## Procedure

1.  Navigate to **All** &gt; **Predictive Intelligence** &gt; **Clustering** &gt; **Solutions**.

2.  Select a solution.

3.  Under the **Cluster Summary** related list, select a cluster ID.

4.  Enter a name in the **Cluster Name** field.

    \[Omitted image "cluster\_name.png"\] Alt text: Cluster Name field on the Cluster Summary form.

5.  Select **Update**.
