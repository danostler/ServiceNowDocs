# Administering Developer Sandboxes data

To define how data is populated in Developer Sandboxes, admins must configure Data Generation Profiles and templates that call the profiles.

Developer Sandboxes provides developers the ability to request, access, and manage individual sandboxes on top of the same underlying development instance. Developers can write and merge code and configuration changes without the risk of their changes getting over-written on the instance mid-development.
