# Navigation between records

Navigate between a customer update record and the customized object or the application file for the object.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **System update sets** &gt; **Local update sets**.

2.  Select an update set.

3.  To navigate to a customized object, such as the application menu record, select the **Show Related Record** related link.

4.  To navigate to the application file record for the object, select the **Show Application File** related link.

5.  To navigate to the current customer update record from a customized object or an app file, select and hold \(or right-click\) the form header and select **Show latest Update**.
