# Administer system update sets

Perform administration duties for system update sets in your instance.

-   [Grant access to the update set picker](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/application-development/system-update-sets/t_GrantAccessToTheUpdateSetPicker.md)

    Enable a non-administrative user to use the update set picker.

-   [Overwrite customizations during an upgrade](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/application-development/system-update-sets/t_OverwriteCustomizsDuringUpgrades.md)

    Specify which customized objects you want to replace during the next upgrade. By default, the upgrade process skips changes to customized objects.
