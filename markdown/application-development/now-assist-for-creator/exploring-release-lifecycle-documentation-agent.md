# Exploring release lifecycle documentation agent

Learn about how the release lifecycle documentation AI agent works to help you to manage application development at scale by generating useful, transparent documentation throughout the app lifecycle.

## Release lifecycle documentation AI agent overview

The release lifecycle documentation AI agent is an autonomous AI agent available with Now Assist for Creator. The AI agent enables you to generate readable, useful documentation across the application lifecycle, including release notes and update set descriptions. Using the release lifecycle documentation AI agent can help to improve traceability, governance, and release transparency through improved documentation, while also reducing the manual effort required to document changes.

## Release lifecycle documentation AI agent users

|User|Description|
|----|-----------|
|Developer|Quickly generate update set descriptions and release notes to provide greater visibility into the changes made.|
|Release manager|Review update set descriptions and release notes to understand the changes in a given deployment request or release.|
