# Exploring UI generation

Learn how UI generation uses AI-powered skills to help app developers, UI Builder admins, and low-code developers build experiences on the ServiceNow AI Platform® using natural language.

## UI generation overview

Now Assist for Creator enables the UI generation feature. With this feature, you can provide a description of the UI experience, and AI-powered Now Assist will help you get started. Users of all skill levels can benefit from UI generation, making it easier and more efficient to create engaging experiences.

## UI generation benefits

The following table summarises the key benefits of UI generation features and how they help different users work more efficiently.

|Feature|Benefits|Users|
|-------|--------|-----|
|[Experience Generation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/now-assist-for-creator/concept-experience-generation.md)|Improves UI quality and consistency by automating experience creation, reducing manual effort, and clearly identifying AI-generated experiences.|Developers, Administrators|
|[UI Builder Agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/now-assist-for-creator/uib-builder-agent-concept.md)|Reduces time spent searching for help by providing instant answers about UI Builder features and documentation.|Developers|
|[Client script summarization](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/now-assist-for-creator/client-script-summarization.md)|Saves time by automatically explaining what a client script does, making code easier to understand, review, and maintain.|Developers|
|[Data binding generation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/now-assist-for-creator/concept-data-binding-generation.md)|Reduces user errors, removes the need to memorize syntax, and speeds up data binding configuration.|Developers|
|[Event handler generation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/now-assist-for-creator/concept-event-handler-generation.md)|Simplifies event handler configuration by helping you use plain language instead of filling out forms manually. It manages routing, binds parameters, and sets destinations, so you can focus on creating your experience rather than the technical details.|Developers|
|[Module generation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/now-assist-for-creator/ai_module_creation_concept.md)|Speeds up application development by enabling module creation through natural-language descriptions instead of manual configuration.|Developers|

Explore the Now Assist capabilities available in UI Builder, from generating full experiences and modules to configuring data bindings and event handlers using natural language.

-   **[Experience Generation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/now-assist-for-creator/concept-experience-generation.md)**  
Experience Generation is a Now Assist capability in UI Generation that uses AI to generate fully functional, multi-page experiences from plain-English descriptions.
-   **[UI Builder Agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/now-assist-for-creator/uib-builder-agent-concept.md)**  
The UI Builder Agent is a Now Assist AI agent that assists low-code developers working in UI Builder by responding to questions, information requests, and page editing instructions.
-   **[Client script summarization](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/now-assist-for-creator/client-script-summarization.md)**  
Client script summarization uses Now Assist to create easy-to-understand explanations of client scripts within the UI Builder editor. This feature helps creators to learn what a script does without needing to read the complex code.
-   **[Data binding generation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/now-assist-for-creator/concept-data-binding-generation.md)**  
 Now Assist generates data bindings for component properties using natural language instructions, reducing the need to manually navigate data trees or construct formulas.
-   **[Event handler generation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/now-assist-for-creator/concept-event-handler-generation.md)**  
Now Assist event handler generation enables you to configure event handlers for UI components using natural language, eliminating the need for manual configuration of complex routing parameters and binding expressions.
-   **[Module generation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/now-assist-for-creator/ai_module_creation_concept.md)**  
Now Assist generates application modules in UI Builder workspaces using natural-language prompts.
-   **[General guidelines UI generation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/now-assist-for-creator/general-guidelines-ui-generation.md)**  
Use these general guidelines for UI generation to get better experiences.
