# Add a flow to a custom application with app generation

Add a flow to a custom application by asking Now Assist. Describe what you want the flow to do, or ask Now Assist for suggestions.

## Before you begin

Starting with the Australia release, app generation is being prepared for future deprecation. It will be hidden and no longer activated on new instances but will continue to be supported. Build Agent provides the latest experience for this functionality. For more information, see [Build Agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/building-applications/build-agent.md).

Role required: now\_assist\_panel\_user

**Note:** If you have users that only need to edit \(not create\) apps, they can be assigned the delegated\_developer or now\_assist\_panel\_user roles.

To add a flow to an application created using app generation, enable the Now Assist for Creator flow generation skill and confirm that you have a flow\_designer role. For more information, see [Flow generation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/now-assist-for-creator/flow-generation-landing.md).

## About this task

Use the app generation skill to add one or more flows to a custom application while creating or updating it. Specify exactly what you want in the flow, or ask Now Assist for suggestions.

**Note:** The following steps describe how to add a flow to an existing application. The process for adding a flow during application creation is similar.

## Procedure

1.  Navigate to **App Engine** &gt; **ServiceNow Studio**.

2.  Select the Now Assist icon.

    \[Omitted image "app-generation-task-initiation-xsr2.png"\] Alt text: Now Assist highlighted in banner.

3.  In the Now Assist panel, select **Update an app**.

    \[Omitted image "app-generation-task-update-app.png"\] Alt text: Now Assist panel with update an app button highlighted.

4.  Select the application to edit.

5.  Select **Submit**.

6.  Start the conversation and ask for a flow.

    \[Omitted image "app-generation-add-flow1.png"\] Alt text: Now Assist panel with add a flow request highlighted.

7.  Specify what you want or ask for suggestions.

    \[Omitted image "app-generation-add-flow2.png"\] Alt text: Now Assist panel with flow details requested highlighted.

    For more information about flows, see .

8.  Continue the conversation until the flow contains what you need.

9.  Preview and save the application to view the new flow.

    For more information, see [Review and edit applications built using app generation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/now-assist-for-creator/sns-app-gen-review-apps.md).
