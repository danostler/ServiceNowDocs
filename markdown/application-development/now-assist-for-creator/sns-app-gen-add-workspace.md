# Add a workspace to a custom application with app generation

Add a workspace to a custom application by asking Now Assist. Describe what you want in the workspace, or ask Now Assist for recommendations.

## Before you begin

Starting with the Australia release, app generation is being prepared for future deprecation. It will be hidden and no longer activated on new instances but will continue to be supported. Build Agent provides the latest experience for this functionality. For more information, see [Build Agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/building-applications/build-agent.md).

Role required: now\_assist\_panel\_user

**Note:** If you have users that only need to edit \(not create\) apps, they can be assigned the delegated\_developer or now\_assist\_panel\_user roles.

To add a workspace to an application using the app generation skill, enable the Now Assist for Creator experience generation skill. For more information, see [Create an AI-generated experience](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/now-assist-for-creator/generate-ui.md).

## About this task

Use the app generation skill to add a workspace to a custom application while creating or updating it. Specify what you want in the workspace, or ask Now Assist for recommendations.

**Note:** These steps describe how to add a workspace to an existing application. The process for adding a workspace during application creation is similar.

## Procedure

1.  Navigate to **App Engine** &gt; **ServiceNow Studio**.

2.  Select the Now Assist icon.

    \[Omitted image "app-generation-task-initiation-xsr2.png"\] Alt text: Now Assist highlighted in banner.

3.  In the Now Assist panel, select **Update an app**.

    \[Omitted image "app-generation-task-update-app.png"\] Alt text: Now Assist panel with update an app button highlighted.

4.  Select the application to edit.

5.  Select **Submit**.

6.  Start a conversation and ask for a workspace.

    \[Omitted image "app-generation-add-workspace1.png"\] Alt text: Now Assist panel with add a workspace request highlighted.

7.  Specify what you want in the workspace, or ask for suggestions.

    Now Assist can suggest a variety of elements to include in a workspace.

    \[Omitted image "app-generation-add-workspace2.png"\] Alt text: Now Assist panel with recommendations for a workspace listed.

8.  Continue the conversation until the workspace contains what you need.

    \[Omitted image "app-generation-add-workspace3.png"\] Alt text: Now Assist panel with elements specified for the new workspace.

9.  Preview and save the application to view the workspace.

    For more information, see [Review and edit applications built using app generation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/now-assist-for-creator/sns-app-gen-review-apps.md).
