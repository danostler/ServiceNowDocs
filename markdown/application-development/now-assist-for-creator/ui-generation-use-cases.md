# UI Generation use cases

Learn how UI Generation capabilities in Now Assist for Creator support developers in creating and configuring UI Builder experiences using natural language.

UI generation is a set of AI-assisted features available in Now Assist for Creator. Together, these features reduce the manual effort involved in building and maintaining ServiceNow UI experiences by enabling developers describe what they need in natural language and having Now Assist generate ready-to-use output.

-   **[Use case: Create an AI-generated experience](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/now-assist-for-creator/use-case-experience-generation.md)**  
Learn how to generate a multi-page UI experience from a natural language description.
-   **[Use case: Using UI Builder Agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/now-assist-for-creator/use-case-ui-builder-agent.md)**  
Learn UI Builder features and accelerate page configuration using conversational AI.
-   **[Use case: Summarize a client script using Now Assist](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/now-assist-for-creator/use-case-client-script-summarization.md)**  
Understand and edit client scripts.
-   **[Use case: Generate a data binding and formula](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/now-assist-for-creator/use-case-data-binding-generation.md)**  
Automate data binding configuration without memorizing formula syntax.
-   **[Use case: Configure an event handler with Now Assist](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/now-assist-for-creator/use-case-event-handler-generation.md)**  
Automate event handler configuration without writing manual binding expressions.
-   **[Use case: Generate modules using Now Assist](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/now-assist-for-creator/use-case-module-generation.md)**
