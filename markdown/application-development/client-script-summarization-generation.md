# Client script summarization

Use client script summary generation to get both a high-level summary and a detailed explanation of a client script.

## Before you begin

Turn on the Client script summarization skill.

Role required: ui\_builder\_admin

## Procedure

1.  Log in to an instance.

2.  Navigate to **All** &gt; **Now Experience Framework** &gt; **UI Builder**.

3.  Open an experience with a client script.

4.  Select a page to open the page editor.

5.  From the Data and scripts panel, open a client script.

    The Explain Code panel appears if you have the ui\_builder\_admin role and have enabled the Client script summarization skill.

6.  Select **Explain this code**.

    **Note:** If you change the script, a message in the **Explain code panel** notifies you of the update and asks whether the code should be explained again. Select **Explain again** to generate a new summary.


## Result

A concise summary, information about where the client script is triggered, and a detailed explanation of the script's functionality are displayed.
