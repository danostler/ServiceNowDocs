# Tools included with Now Assist for App Engine

Learn about the tools and applications included with Now Assist for App Engine.

Now Assist for App Engine enables users to leverage the generative and agentic AI capabilities of the ServiceNow AI Platform. With Now Assist for App Engine, you can create and activate custom skills using Now Assist Skill Kit, build custom AI agents using AI Agent Studio, generate synthetic data for testing skills and AI agents using Now Assist Data Kit, and more.

The following table highlights some of the ServiceNow AI Platform tools and applications that you can use to enhance custom applications with Now Assist for App Engine.

**Note:** Check your entitlements to determine whether you have access to the following tools and applications.

|Tool|Description|Resources for additional information|
|----|-----------|------------------------------------|
|AI Agent Studio|Create flexible, adaptive AI agents for App Engine use cases.||
|Now Assist Skill Kit|Develop custom skills that can be used in App Engine applications.||
|Now Assist Data Kit|Build and maintain datasets to evaluate your custom skills.||
|Document Intelligence|Incorporate generative AI capabilities into your flows and playbooks.||
|Virtual Agent|Create generative AI-powered chatbots.||

For a complete list of the tools and skills in Now Assist for App Engine, see the product listing in the [ServiceNow Store](https://store.servicenow.com/store/app/5d27eef41bd92a50396216db234bcb45).
