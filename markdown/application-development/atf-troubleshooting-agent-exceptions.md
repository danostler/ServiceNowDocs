# ATF troubleshooting agent design considerations

Leverage the full potential of ATF troubleshooting agent by following these design considerations.

## Exceptions

ATF troubleshooting agent doesn't support the following metadata:

-   Flows
-   Configurable workspace steps
-   Custom UI test steps
-   Custom test steps
