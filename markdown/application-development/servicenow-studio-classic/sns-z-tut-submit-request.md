# Submit a request

Submit a request in the Marketing Services workspace to test its functionality.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **App Engine** &gt; **ServiceNow Studio**.

2.  From the Navigator panel, open the **Marketing Services** workspace.

3.  Select **Preview**.

    Observe that Unassigned Tasks is **0**.

4.  Return to the platform.

5.  In the Filter Navigator, search for `portal`.

6.  Select **Service Portal Home**.

    The Service Portal opens in a new tab.

7.  Search for `marketing`.

8.  Select **Marketing Design Request** from the search results.

9.  Complete the **Marketing Design Request** form.

10. Select **Submit**.

11. Return to the **Marketing Services** workspace.

    Observe that Unassigned Tasks is now **1**.

12. Select **1** to view the new Marketing Design Request submission.
