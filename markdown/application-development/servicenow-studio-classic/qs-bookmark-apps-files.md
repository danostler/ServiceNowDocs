# Bookmark apps and files in ServiceNow Studio

Bookmark apps and app files in ServiceNow Studio so you can access them quickly from the Bookmarks tab in the Navigator panel.

## Before you begin

Role required: admin or delegated\_developer

## About this task

**Note:** Bookmarks and favorites are distinct ServiceNow capabilities. Bookmarks exist only within ServiceNow Studio, while favorites are accessible throughout your instance. Bookmarks do not automatically become favorites, and favorites do not automatically become bookmarks.

## Procedure

1.  Navigate to **All** &gt; **App Engine** &gt; **ServiceNow Studio**.

2.  Select the Add to collections icon \[Omitted image "sn-studio-bookmark-icon.png"\] Alt text: from one of the following locations.

    -   To bookmark an open app, select the Add to collections icon next to the app name.

        \[Omitted image "sn-studio-bookmark-app-as2.png"\] Alt text: Bookmark an app that is open in the canvas.

    -   To bookmark an app or file in the Navigator panel, hover over it and select the Add to collections icon.

        \[Omitted image "sn-studio-bookmark-file-as2.png"\] Alt text: Bookmark a file from the Navigator panel.

3.  Select the **Bookmarks** collection.

4.  Select **Save**.

5.  View your bookmarked apps and files in the **Collections** tab of the Navigator panel.
