# Add files to a collection in ServiceNow Studio

Group your apps, files and lists into folders and organize your work by stories, epics, or related use cases.

## Before you begin

Role required: admin or delegated\_developer

## About this task

Add files to one or more collections to group them for related work.

## Procedure

1.  Navigate to **All** &gt; **App Engine** &gt; **ServiceNow Studio**.

2.  In the File Navigator, select a file to add to a collection.

    \[Omitted image "sn-studio-add-to-collection.png"\] Alt text: Add a file to a collection.

3.  Select one or more collections to add the file to.

    \[Omitted image "sn-studio-collections-pane.png"\] Alt text: Select which collections you want to add your file to.

4.  Select **Save**.


## Result

The collection contains the file you added and is viewable from the Collections tab in the Navigator panel.
