# ServiceNow Studio quick start

Complete these tasks to learn the core workflows in ServiceNow Studio — from finding apps and files to packaging and deploying changes.

## Where do I start in ServiceNow Studio?

Complete these tasks to learn the core workflows in ServiceNow Studio. For a step-by-step app creation tutorial, see [ServiceNow Studio and Build Agent tutorial](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/servicenow-studio-classic/sns-ba-tutorial-landing.md).

|[Find an app or app file using the Navigator panel](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/servicenow-studio-classic/qs-find-app-app-file-using-navigator-panel.md)|Browse the Navigator panel taxonomy to locate files and metadata records by type.|

|[Bookmark lists in ServiceNow Studio](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/servicenow-studio-classic/bookmark-lists-in-sns.md)|Bookmark lists you use often so you can return to them without searching.|

|[Add files to a collection in ServiceNow Studio](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/servicenow-studio-classic/qs-add-files-to-a-collection.md)|Add apps, files, and lists to one or more collections.|
|[Access recently opened apps and app files in ServiceNow Studio](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/servicenow-studio-classic/qs-access-recently-opened-apps-files.md)|View and reopen apps and files you have worked on recently.|
|[Find info about an app in ServiceNow Studio](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/servicenow-studio-classic/qs-find-info-about-an-app-servicenow-studio.md)|Open the App details page to view app metadata and create files.|
