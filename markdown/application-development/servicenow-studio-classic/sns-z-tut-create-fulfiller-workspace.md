# Create the fulfiller workspace

Workspaces are a great option for departments. Workspaces can be tailored specifically for department needs and can simplify interacting with the platform.

## Before you begin

Role required: admin or delegated\_developer

## Procedure

1.  Navigate to **All** &gt; **App Engine** &gt; **ServiceNow Studio**.

2.  Select **Create** &gt; **File** &gt; **User Interface** &gt; **Workspace**.

3.  Select **Continue**.

4.  Select **Begin**.

5.  In the **Name** field, enter `Marketing Services`.

6.  Accept the other provided defaults for the workspace design and select **Continue**.

    \[Omitted image "sns-z-tut-workspace.png"\] Alt text: Accept the defaults in the Marketing Services workspace.

7.  Select **Done**.
