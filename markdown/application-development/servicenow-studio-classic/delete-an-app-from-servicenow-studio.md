# Delete an app from ServiceNow Studio

Delete apps that are no longer needed from ServiceNow Studio to remove them permanently from the instance.

## Before you begin

Role required: admin or delegated\_developer

## Procedure

1.  Navigate to **All** &gt; **App Engine** &gt; **ServiceNow Studio**.

2.  Open the application to delete.

3.  Select **App details** to open the app in the canvas.

4.  Select the more options icon \[Omitted image "sn-studio-more-options-icon.png"\] Alt text: More options icon, and select **Delete**.

5.  In the **Permanently delete this app?** modal, enter `Delete`.

6.  Select **Close**.


## Result

The app is permanently deleted from the instance and no longer appears in ServiceNow Studio.
