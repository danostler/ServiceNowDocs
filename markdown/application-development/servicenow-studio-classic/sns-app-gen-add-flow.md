# Add a flow to a custom application with Now Assist for app generation

Use Now Assist for app generation to add a flow to a custom application.

## Before you begin

Role required: now\_assist\_panel\_user

**Note:** If you have users that only need to edit \(not create\) apps, they can be assigned the delegated\_developer and now\_assist\_panel\_user roles.

To add a flow to an application in Now Assist for app generation, the Now Assist Flow Generation skill must be active and you must have a flow\_designer role. For more information, see .

## About this task

As you're creating or updating custom applications, use Now Assist for app generation to add one or more flows to your application. Specify exactly what you want in the flow or ask for suggestions.

**Note:** The following steps explain how to add a flow to an existing application, but the process to add a flow to an application that you're creating is similar.

## Procedure

1.  Navigate to **App Engine** &gt; **ServiceNow Studio**.

2.  Select the Now Assist icon.

    \[Omitted image "app-generation-task-initiation-xsr2.png"\] Alt text: Now Assist highlighted in banner.

3.  In the Now Assist panel, select **Update an app**.

    \[Omitted image "app-generation-task-update-app.png"\] Alt text: Now Assist panel with update an app button highlighted.

4.  Select the application to edit.

5.  Select **Submit**.

6.  Start the conversation and ask for a flow.

    \[Omitted image "app-generation-add-flow1.png"\] Alt text: Now Assist panel with add a flow request highlighted.

7.  Specify what you want or ask for suggestions.

    \[Omitted image "app-generation-add-flow2.png"\] Alt text: Now Assist panel with flow details requested highlighted.

    For more information about flows, see .

8.  Continue the conversation until the flow contains what you need.

9.  When you're ready, preview and save the application to view the new flow.

    For more information, see [Review and edit applications using Now Assist for app generation in ServiceNow Studio](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/application-development/servicenow-studio-classic/sns-app-gen-review-apps.md).
