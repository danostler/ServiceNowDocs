# ServiceNow Studio and Build Agent tutorial

Use Build Agent in ServiceNow Studio to develop reusable server‑side logic and create an application on the ServiceNow AI Platform, taking it from initial data modeling through testing and validation.

Complete this tutorial to learn how to use Build Agent for common development tasks and to apply practical patterns for data modeling, testing, and validation. This tutorial covers the following items:

-   Application creation, custom data modeling, and table design
-   Server-side business logic with script includes
-   Validation and testing using background scripts
-   Reusable utility patterns for real-world use cases

The full [ServiceNow Studio and Build Agent tutorial](https://servicenow-events-or-lab-guidebo.gitbook.io/knowledge-2026/knowledge-2026/lab8113-k26/overview/lab-overview) is hosted on Gitbook, and may require a login.
