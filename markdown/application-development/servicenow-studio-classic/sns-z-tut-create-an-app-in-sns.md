# Create an app in ServiceNow Studio

Build an app quickly in ServiceNow Studio, the new, innovative environment for development on the ServiceNow AI Platform.

## Before you begin

Role required: admin

## About this task

ServiceNow Studio provides developers with a unified tool for all application-building tasks, enabling fast innovation and the creation of business-critical workflow applications within a complete builder experience. Let’s get started!

## Procedure

1.  Navigate to **All** &gt; **App Engine** &gt; **ServiceNow Studio**.

2.  Select **Create** &gt; **App**.

3.  In the **Name** field, enter `Marketing Services`.

4.  Select **Continue**.

5.  Select **Continue** to accept the default admin and user roles.

    ServiceNow Studio will create default roles for admins and users for the Marketing Services application. Users assigned to these roles will be members of the department and will play a role in service fulfillment as fulfillers.

6.  In the Navigator panel, select the refresh icon \(\[Omitted image "wb-refresh-icon.png"\]\) to refresh the list of apps.

7.  Select the Marketing Services app from the list, and open it by selecting **App details**.

8.  Select the more options icon \(\[Omitted image "sn-studio-more-options-icon.png"\]\), then select **Show parent categories.**

    The **Show parent categories** option changes how the application files are grouped.
