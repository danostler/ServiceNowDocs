# Bookmark lists in ServiceNow Studio

Enable easy access to your favorite lists by bookmarking them in ServiceNow Studio.

## Before you begin

Role required: admin or delegated\_developer

## About this task

**Note:** Bookmarks and favorites are distinct ServiceNow capabilities. Bookmarks exist only within ServiceNow Studio, while favorites are accessible throughout your instance. When you bookmark lists in ServiceNow Studio, your bookmarks do not automatically become favorites. Favorites also do not automatically become bookmarks.

## Procedure

1.  Navigate to **All** &gt; **App Engine** &gt; **ServiceNow Studio**.

2.  Open the list you want to bookmark.

    A list in ServiceNow Studio can vary from the entire File Categories list to a list of just a few files.

3.  Bookmark the list at the top of the page or by accessing the more options icon.

    -   At the top of the list, select **Bookmark list** next to the **Create** button.

        \[Omitted image "sn-studio-bookmark-top.png"\] Alt text: Bookmark the list from the top of the page.

    -   Select the more options icon \[Omitted image "sn-studio-more-options-icon.png"\] Alt text:, and select **Bookmark list**.

        For example, you can open the list of all flows from the Automation file category and bookmark the list.

        \[Omitted image "sn-studio-bookmark-list.png"\] Alt text: Bookmark a list from more options menu in the Navigator panel.

4.  Enter a name for your bookmark in the **Add to bookmarks** dialog.

5.  Select **Add**.

6.  Select **Close**.

7.  Refresh the Navigator panel, and access your recently bookmarked lists on the **Bookmarks** tab.

8.  Edit the bookmark label by selecting the rename bookmark icon, entering a new name, and selecting **Save**.
