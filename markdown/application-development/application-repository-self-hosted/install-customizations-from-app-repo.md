# Install customizations from an application repository

Install your customization to a store application on an instance.

## Before you begin

Role required: admin

[Publish a customization of a store application to the application repository](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/application-repository-self-hosted/publish-customizations-app-repo.md). Check the entitlement type of the application to ensure that your instance is entitled to the application.

## Procedure

1.  Navigate to **System Applications** &gt; **All Available Applications** &gt; **All**.

2.  Find the application.

3.  Select a customization version to install.

    \[Omitted image "base-customization-versions.png"\] Alt text: Customizations versions

    **Note:** For scoped applications installed via plugins, the base system version is the ServiceNow release version that you are currently using \(Paris, Quebec, Rome, and so on\).

    \[Omitted image "scoped-app-edit-in-studio.png"\] Alt text: Edit a scoped application installed via plugin in Studio.

4.  Click **Install**.
