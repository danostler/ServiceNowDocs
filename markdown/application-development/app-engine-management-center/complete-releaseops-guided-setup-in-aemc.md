# Complete ReleaseOps guided set up in AEMC

Complete ReleaseOps guided setup in AEMC to configure your ReleaseOps ecosystem to handle your deployments.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to the Set up ReleaseOps and Multi-Instance Management page.

    \[Omitted image "releaseops-setup-page-as2.png"\] Alt text: Set up ReleaseOps and Multi-Instance Management page in AEMC

2.  In the **Complete ReleaseOps and MIM Guided Setup** step, select **Configure**.

3.  On the Setup History page, select **Start new configuration**.

    \[Omitted image "releaseops-setup-history.png"\] Alt text: Select Start a new configuration to begin setting up ReleaseOps.

4.  Enter a name for the configuration in the **Configuration name field**.

    \[Omitted image "releaseops-setup-configuration-name.png"\] Alt text: Enter a name and description for the configuration.

5.  Enter a description for the configuration in the **Description** field.

6.  Select **Next**.

    When you select **Next**, you're automatically redirected to the ReleaseOps guided setup page.

7.  [Complete ReleaseOps guided setup](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/releaseops/complete-guided-setup.md).


## What to do next

Complete ReleaseOps configuration in AEMC by [enabling the ReleaseOps system property](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/app-engine-management-center/enable-releaseops-system-property.md).
