# Configure ReleaseOps in AEMC

Complete ReleaseOps guided setup in AEMC to start using ReleaseOps for your deployments.

## Before you begin

Role required: admin

## Procedure

1.  On your production instance, navigate to **All** &gt; **App Engine** &gt; **Administration** &gt; **Guided Setup**.

2.  Select **Get Started**.

3.  On the App Engine Management Center guided setup page, select **Get Started** in the **Set up ReleaseOps and Multi-Instance Management** card.

    \[Omitted image "aemc-set-up-releaseops-as2.png"\] Alt text: Select Get Started to set up ReleaseOps and Multi-Instance Management.

4.  [Complete ReleaseOps guided set up in AEMC](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/app-engine-management-center/complete-releaseops-guided-setup-in-aemc.md).

5.  [Enable ReleaseOps system property](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/app-engine-management-center/enable-releaseops-system-property.md).

6.  [Run trust profile setup in AEMC](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/app-engine-management-center/run-trust-profile-setup-aemc.md).
