# Using AEMC

AEMC enables you to manage app development and deployments, all in one centralized location. Learn about how you can use AEMC to govern your organization's app development.

## Aims of AEMC

The App Engine Management Center \(AEMC\) is designed to empower App Engine admins with all of the tools needed to make governing app development easier. AEMC provides a unified experience for managing all stages of application development, from idea submission to deployment. Using AEMC, admins can manage requests for new apps and delegate development and collaboration permissions. Admins can also oversee the entire deployment process with insights into individual deployment requests, test suites, and scheduled deployments. The following list outlines the tasks that the AEMC can help you to accomplish:

-   Mange intake requests submitted by developers for new apps.
-   Manage app requests submitted by no-code developers from within Creator Studio.
-   Manage collaboration requests submitted by developers working on new apps.
-   Manage deployment requests for apps that are ready to be moved through the pipeline.
-   Schedule app deployments.
-   View insights about developers and custom apps within their ServiceNow environment.
