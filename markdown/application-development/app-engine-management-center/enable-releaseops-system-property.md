# Enable ReleaseOps system property

Enable the ReleaseOps system property to have AEMC fetch data from the environments configured in ReleaseOps.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to the Set up ReleaseOps and Multi-Instance Management page.

    \[Omitted image "releaseops-setup-page-as2.png"\] Alt text: Set up ReleaseOps and Multi-Instance Management page in AEMC

2.  In the **Enable ReleaseOps system property** step, select **Configure**.

3.  On the AEMC system properties page, select **Enable ReleaseOps**.

    \[Omitted image "releaseops-enable-system-property.png"\] Alt text: Select Enable ReleaseOps to enable ReleaseOps within AEMC.
