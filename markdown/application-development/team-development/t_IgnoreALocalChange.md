# Ignore a local change

Ignoring a local change helps to prevent a record from generating new versions in the Local Changes list.

## Before you begin

Role required: admin

## About this task

An ignored local change applies to the current version for the record. You can’t push ignored records to another instance.

## Procedure

1.  Navigate to **All** &gt; **Team Development** &gt; **Team Dashboard**.

2.  Filter the Local Changes list to show only the changes that you want to ignore.

3.  Select **Ignore All**.

4.  Review the Ignored list to confirm that the correct changes are ignored.
