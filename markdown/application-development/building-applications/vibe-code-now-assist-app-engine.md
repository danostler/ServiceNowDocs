# AI-assisted development with Now Assist for App Engine

Now Assist for App Engine enables you to embed AI functionality into an application that you built on the ServiceNow AI Platform.

While Now Assist for Creator skills and Build Agent help you build the app, Now Assist for App Engine helps you embed AI features into the experience so requesters and fulfillers have access to AI-powered interactions.

Now Assist for App Engine enables you to leverage relevant ServiceNow AI Platform capabilities to enhance custom apps with GenAI and AI agents. Some of its most important features are the following:

-   Now Assist Skill Kit, which you can use to create custom skills for ServiceNow products and Now Assist for App Engine. For more information, see [Now Assist Skill Kit](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/intelligent-experiences/now-assist-skill-kit/now-assist-skill-kit-landing.md).
-   AI Agent Studio, which enables you to create, manage, and test AI agents and agentic workflows. For more information, see [AI Agent Studio](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/intelligent-experiences/enable-ai-experiences/ai-agent-studio.md).
-   The custom app record summarization skill, which generates AI summaries for the contents of records in custom applications and tables. For more information, see [Custom app record summarization skill](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/application-development/now-assist-for-app-engine/custom-app-record-summarization-na-for-app-engine.md).

Common capabilities for Now Assist for App Engine include:

-   AI Search and knowledge summarization inside the app experience.
-   AI Agent Studio and agentic workflows that guide users and fulfillers through tasks.
-   Generative flow actions and in-app assistance that reduces manual effort.
-   Content-aware responses that adhere to platform security and data access models.

For more information on Now Assist for App Engine, see [Now Assist for App Engine](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/application-development/now-assist-for-app-engine/add-ai-to-custom-apps-with-now-assist-for-app-engine-enterprise.md).

With Now Assist for App Engine, you have access to Now Assist AI assets, such as Platform skills, AI agents, and agentic workflows, as well as the tools for developing custom AI assets using Now Assist Skill Kit and AI Agent Studio.

For a list of all the tools available in Now Assist for App Engine, see [Tools included with Now Assist for App Engine](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/application-development/now-assist-for-app-engine/tools-included-with-now-assist-for-app-engine.md).
