# Set runtime access to application tables

Set these access permissions to protect application tables at runtime.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **System Applications** &gt; **Applications**.

2.  Click the button for the application type you want to edit.

<table id="choicetable_bxr_ryb_br"><tbody><tr><td id="d106976e80">

**__Developed__**

</td><td>

displays applications created on this instance.

</td></tr><tr><td id="d106976e90">

**__Downloaded__**

</td><td>

displays applications downloaded on this instance.

</td></tr></tbody>
</table>3.  Click the application name or the **Edit** button for the application you want to work on.

4.  From the **Tables** related list, select the table whose access permission you want to set.

5.  From the **Application Access** section, set the runtime access permissions.

6.  Click **Update**.
