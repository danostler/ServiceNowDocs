# Test Agent access

Learn about the prerequisites required to access and use Test Agent.

Before you can use Test Agent, your instance must meet specific prerequisite requirements to support test generation, execution, and troubleshooting workflows.

Ensure that the following applications are installed:

-   Build Agent: See [Install Build Agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/building-applications/install-build-agent.md) for more information.

    **Note:** If you install Build Agent, Test Agent is installed automatically as part of Build Agent.

-   ATF Test Generator and Cloud Runner: Install the [ATF Test Generator and Cloud Runner](https://store.servicenow.com/store/app/e4292f6e1be06a50a85b16db234bcbc3) store application from ServiceNow store. You also need to set up the cloud user for seamless execution from the Build Agent interface. See  and  for more information.

These components provide the underlying capabilities required for Test Agent to function correctly.
