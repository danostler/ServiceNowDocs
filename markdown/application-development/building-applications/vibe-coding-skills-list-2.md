# ServiceNow Now Assist skills: Developer-oriented list

The ServiceNow AI Platform offers a range of skills that support efficiency in each development workflow.

If you are an experienced developer, this list can help you identify which tasks AI automates on your behalf. Each skill represents a previously manual task, a complex business rule, or a classification system that AI automates. Architect the data flow and governance for these skills, rather than writing the repetitive logic they replace.

See  for a complete list of Now Assist skills.

## Application and artifact creation

|Skill|Description|
|-----|-----------|
|App generation|Generates a starter ServiceNow application from a natural-language description.|
|Catalog item generation|Creates a service catalog item from intent.|
|Experience generation|Generates UI and experience components from a prompt.|
|Mobile card generation|Creates mobile UI cards for ServiceNow mobile experiences.|
|Create a demand|Creates demand records conversationally.|
|Story generation|Generates stories from higher-level planning items.|
|Target generation|Creates targets from planning or strategic context.|
|Write planning item|Generates planning items from text input.|
