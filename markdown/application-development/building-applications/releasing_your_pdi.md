# Releasing your PDI

Release your PDI to unassign the instance from you, reset to its original state, and reassign it to another Developer Program member.

## Before you begin

Role required: none

## About this task

If you no longer need your PDI or you are not going to be able to use it for an extended period of time, save your work and release the instance. When you release your PDI, the instance is unassigned from you, reset to its original state, and reassigned to another Developer Program member. When you request an instance or restore an instance, you will receive a PDI with a different instance name and URL.

## Procedure

1.  To release your PDI:

    1.  Open the **Account** menu to access the **My Instance** section.

    2.  Click the **Release instance** action.

    3.  In the **Your instance actions** dialog, read the details and select a reason for releasing your instance from the choice list.

        \[Omitted image "release-pdi.png"\] Alt text: Specify why you're releasing an instance

    4.  Click the **Release instance** button.
