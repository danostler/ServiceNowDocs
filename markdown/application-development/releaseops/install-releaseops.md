# Install ReleaseOps

Install ReleaseOps from the ServiceNow Store for the first time.

## Before you begin

Role required: admin

Because ReleaseOps involves a multi-instance ecosystem, you must complete this procedure on each instance in your ReleaseOps ecosystem \(for example, development, test, and production\).

## Procedure

1.  Open the [ServiceNow Store](https://store.servicenow.com/store) website.

2.  In the **Search** bar, enter `ReleaseOps` and select enter or return on your keyboard.

3.  Select the ReleaseOps tile.

4.  Select **Get**.

5.  Sign in with your ServiceNow user ID.

6.  Run the installation on your instance and accept all of the plugins.

    For more information about installing a ServiceNow® Store application on your instance, see [Install an application or plugin](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/platform-administration/application-manager/installing-applications-in-application-manager.md).


## Result

ReleaseOps is installed on your instance.
