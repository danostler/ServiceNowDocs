# Deployment requests in ReleaseOps

Deployment requests contain one or more update sets, and are contained in a release.

Deployment requests are the building blocks of moving code into production. They capture the content that a developer wants to be moved, including update sets. Deployment requests describe how and when the developer would like that code to go into production.
