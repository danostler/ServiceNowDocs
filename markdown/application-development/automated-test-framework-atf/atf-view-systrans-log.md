# View transaction data for automated test results

To help troubleshoot performance issues with automatic tests, you can inspect related records from the transactions log entry \[syslog\_transaction\] table.

## Before you begin

Role required: test\_admin or admin

## About this task

**Note:** The system may not be able to log some transactions with short durations.

## Procedure

1.  Navigate to related transaction record in the system transaction log.

    -   From the [Step results record](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/application-development/automated-test-framework-atf/atf-step-result-record.md), view transactions in the Step transactions related list.
    -   From the [Test Results record](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/application-development/automated-test-framework-atf/atf-test-results-record.md), view transactions in the Test Transactions related list.
