# Learning plan for getting started with App Engine Studio

ServiceNow provides citizen developers with several quick training modules for getting started building low-code apps in App Engine Studio \(AES\).

The ServiceNow Developers site provides numerous resources for developing apps in AES.

Use the following learning modules to quickly explore ways to build apps.

|Course|Description|
|------|-----------|
|[Build My First App Engine Studio Application](https://developer.servicenow.com/dev.do#!/learn/learning-plans/tokyo/citizen_developer/BAP_Objectives_BMFAESA_tokyo)|Create an application to track a collection using AES. The application will be secure, automated, and provide an engaging user experience.|
|[Create an App and Data Model](https://developer.servicenow.com/dev.do#!/learn/learning-plans/tokyo/citizen_developer/CAS_Objectives_CreateAppData_tokyo)|Create an application from scratch with AES, including the data model used by the application. The data model is a foundation for learning how to add user experiences, logic and automation, and security to your application in other learning modules.|
|[Create Record Producer User Experiences](https://developer.servicenow.com/dev.do#!/learn/learning-plans/tokyo/citizen_developer/CIT_Objectives_CatalogItem_tokyo)|Create record producer catalog items in AES. Users can submit a request or create a task using Service Catalog items in Service Portal.|
|[Create Workspace User Experiences](https://developer.servicenow.com/dev.do#!/learn/learning-plans/tokyo/citizen_developer/WSP_Objectives_Workspace_tokyo)|Create a workspace to report on and manage data in an application created with AES.|
|[Create Mobile User Experiences](https://developer.servicenow.com/dev.do#!/learn/learning-plans/tokyo/citizen_developer/MOB_Objectives_MobileExperience_tokyo)|Create and test a mobile experience in AES that enable users to access applications from anywhere.|
|[Automate Processes with Flows](https://developer.servicenow.com/dev.do#!/learn/learning-plans/tokyo/citizen_developer/app_store_learnv2_aescreateappfromscratch_tokyo_automate_processes_with_flows_objectives)|Create flows in AES to automate business processes. Flows reduce human error, improve efficiency, and make apps work better.|
|[Secure Apps and Data](https://developer.servicenow.com/dev.do#!/learn/learning-plans/tokyo/citizen_developer/app_store_learnv2_aescreateappfromscratch_tokyo_secure_apps_and_data_objectives)|Use AES to secure an application by creating roles, assigning the roles to users, and testing user access to application data.|
