# Get metrics by duration

Filter metrics in the Performance Analyzer by duration from the last fifteen minutes to the last seven days.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Next Experience Developer Tools** &gt; **Performance Analyzer**.

2.  In the Duration field, select a duration from the list.


## Result

The Performance Analyzer displays aggregate metrics for the selected duration.
