# Get application metrics

Filter metrics by application in the Performance Analyzer.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Next Experience Developer Tools** &gt; **Performance Analyzer**.

2.  From the Application field, select an application.


## Result

The Performance Analyzer displays page-route metrics for the selected application including total UI time, download time, and trends.
