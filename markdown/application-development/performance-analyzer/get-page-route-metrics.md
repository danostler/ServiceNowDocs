# Get page route metrics

Filter metrics for an application by page route in the Performance Analyzer.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Next Experience Developer Tools** &gt; **Performance Analyzer**.

2.  From the Application field, select an application.

3.  From the Page route field, select a page route.


## Result

The Performance Analyzer displays interaction metrics for the selected page route including total UI time and download time.
