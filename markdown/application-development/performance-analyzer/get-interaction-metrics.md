# Get interaction metrics

Filter metrics for an application's page route by interaction in the Performance Analyzer.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Next Experience Developer Tools** &gt; **Performance Analyzer**.

2.  From the Application field, select an application.

3.  From the Page route field, select a page route.

4.  From the Timestamp field, select a timestamp.


## Result

The Performance Analyzer displays a waterfall with details for each resource on the page during that interaction.
