# Configure Zoom and touchpoint integration

Integrate Zoom with touchpoint records to schedule virtual meetings and generate AI-powered meeting summaries.

This integration uses the Zoom spoke and a bi-directional webhook to connect Zoom meetings to touchpoint records. After the integration is set up, customer success managers can schedule Zoom meetings from a touchpoint and automatically generate summaries when meetings end.
