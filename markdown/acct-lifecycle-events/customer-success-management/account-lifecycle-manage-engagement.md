# Manage engagements

Configure metrics to calculate the health and risk score for an engagement.

-   **[Calculate the engagement health score](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/acct-lifecycle-events/customer-success-management/account-lifecycle-setup-health-defn.md)**  
Use the **Calculated** metric data source to determine the health score of an engagement.
-   **[Create a risk definition](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/acct-lifecycle-events/customer-success-management/account-lifecycle-setup-risk-defn.md)**  
Use the engagement risk definition to configure the metrics required to define the risk signals for an engagement.
-   **[Setup the color banding table](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/acct-lifecycle-events/customer-success-management/account-lifecycle-setup-color-banding.md)**  
Use the color banding table to set up how the engagement health score is visually represented in the Success Landing and Engagement pages.
