# Configure success blueprints

Configure success blueprint templates and the supporting elements that customer success managers use to define how success is measured for an engagement.

A success blueprint captures what success looks like for a customer, how it is measured, and the initiatives that will deliver it. Templates define a reusable set of success objectives and primary success outcomes for a specific product or solution. Customer success managers can apply these templates to engagements, or create blueprints manually when no template fits.
