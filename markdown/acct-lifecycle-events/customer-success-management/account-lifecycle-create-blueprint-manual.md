# Create a success blueprint manually

Create a success blueprint by adding success objectives, primary success outcomes, and success initiatives directly, without using a template.

Use the manual path when no success blueprint template is available for the product, or when the engagement requires objectives that aren't covered by an existing template.

To create the blueprint, complete the following tasks in order from the engagement page:

1.  Navigate to the **Success blueprint** tab and select **Create success blueprint**.
2.  Add a success objective that represents the value the customer expects from the products they purchased. See [Add a success objective](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/acct-lifecycle-events/customer-success-management/account-lifecycle-create-object.md).
3.  Add one or more primary success outcomes that measure the achievement of the objective. See [Add a primary success outcome](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/acct-lifecycle-events/customer-success-management/account-lifecycle-create-outcome.md).
4.  Add a success initiative that captures the activities and tasks the provider and customer complete to achieve each outcome. See [Add a success initiative](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/acct-lifecycle-events/customer-success-management/account-lifecycle-success-initiative.md).

After you completed these tasks, the success blueprint is complete. From the engagement page, you can do the following:

-   [Measure success rate](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/acct-lifecycle-events/customer-success-management/account-lifecycle-success-score-calc.md)
-   [Success initiative roadmap](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/acct-lifecycle-events/customer-success-management/account-lifecycle-success-roadmap.md)

-   **[Add a success objective](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/acct-lifecycle-events/customer-success-management/account-lifecycle-create-object.md)**  
Create a success objective that represents the value of the products sold to the customer.
-   **[Add a primary success outcome](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/acct-lifecycle-events/customer-success-management/account-lifecycle-create-outcome.md)**  
Create a primary success outcome that can be used to measure the achievement of a success objective.
-   **[Add a success initiative](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/acct-lifecycle-events/customer-success-management/account-lifecycle-success-initiative.md)**  
Create a success initiative with a planned set of internal or external tasks to support a primary success outcome.
