# Create a success blueprint

Create a success blueprint to define the success objectives, primary success outcomes, and success initiatives for an engagement.

The procedure for creating a success blueprint depends on whether a published template is available for the product. For an overview of the success blueprint feature and guidance on choosing between the two paths, see [Success blueprint](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/acct-lifecycle-events/customer-success-management/account-lifecycle-success-blueprint.md).

You can create a success blueprint in the following ways:

-   [Create a success blueprint from a template](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/acct-lifecycle-events/customer-success-management/account-lifecycle-create-blueprint.md)
-   [Create a success blueprint manually](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/acct-lifecycle-events/customer-success-management/account-lifecycle-create-blueprint-manual.md)

-   **[Create a success blueprint from a template](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/acct-lifecycle-events/customer-success-management/account-lifecycle-create-blueprint.md)**  
Create a success blueprint from a template to add success objectives and outcomes to an engagement.
-   **[Create a success blueprint manually](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/acct-lifecycle-events/customer-success-management/account-lifecycle-create-blueprint-manual.md)**  
Create a success blueprint by adding success objectives, primary success outcomes, and success initiatives directly, without using a template.
