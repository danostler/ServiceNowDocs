# Track an In-store operations case created from a store plan

Monitor the progress of an In-store operations case from the CSM/FSM Configurable Workspace.

## Before you begin

Role required: sn\_rtl\_instore\_ops.manager

## Procedure

1.  In **CSM/FSM Configurable Workspace**, navigate to **Retail cases**.

2.  Select **All**.

3.  Select an In-store operations case to monitor.

4.  Review the case details.

    You can edit and save case details, or select **Close case** to close the case as needed.
