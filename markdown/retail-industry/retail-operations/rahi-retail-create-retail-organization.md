# Create a retail organization

Create a retail organization to enable management of hierarchical stores and franchises that simplifies store associate management.

## Before you begin

Role required: sn\_rtl.retail\_admin

## About this task

## Procedure

1.  Navigate to **All** &gt; **Retail Service Management** &gt; **Retail Organization**.

2.  Select **All Organizations**.

3.  Select **New**.

4.  Fill in the following fields.
