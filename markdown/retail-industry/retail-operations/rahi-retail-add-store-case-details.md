# Add store case details

Log the store case details to the new HQ communications plan.

## Before you begin

Role required: sn\_rtl\_hq\_ops.agent, sn\_rtl\_hq\_ops.agent\_manager, sn\_rtl\_hq\_ops.location\_manager

## Procedure

1.  On the Store case form, select the appropriate priority from the **Priority** drop-down list.

2.  Enter the short description in the **Short Description**.

3.  Enter the description in the **Description**.

4.  Select **Save** to save the details.

5.  Select **Save &amp; Continue**.

    A new form appears where you must add a task for the store.
