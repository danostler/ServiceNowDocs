# Add HQ task details

Log the HQ task details to the new HQ communications store plan.

## Before you begin

Role required: sn\_rtl\_hq\_ops.agent, sn\_rtl\_hq\_ops.agent\_manager, sn\_rtl\_hq\_ops.location\_manager

## Procedure

1.  Select **Add**.

    A new HQ task form appears.

2.  On the HQ task form, select the required option from the **Priority** drop-down list.

3.  Enter the name of the agent, the agent manager, or the location manager in the **Assigned to**.

4.  Enter the subject in the **Subject**.

5.  Enter the description in the **Description**.

6.  Select **Continue**.

    A new prompt Continue without saving appears.

    1.  Select **Yes** to continue without saving the details.

    2.  Select **No** to remain in the same form.

7.  Select **Cancel** to discard the details.

8.  Select **Save** to save the details.

    The new HQ task added appears.

    1.  Select \[Omitted image "Form\_DeleteIcon.png"\] Alt text: to delete the new HQ task.

    2.  Select \[Omitted image "icon-pencil-ac.png"\] Alt text: to edit the HQ task details.

    3.  Select **Continue** to go next step.
