# Monitor a published store plan across multiple stores

Audit or review previously published schedules as needed using the CSM/FSM Configurable Workspace.

## Before you begin

Role required: sn\_rtl\_hq\_ops.agent, sn\_rtl\_hq\_ops.agent\_manager, sn\_rtl\_hq\_ops.location\_manager

## Procedure

1.  In **CSM/FSM Configurable Workspace**, navigate to **Store Plans**.

2.  Select **All**.

3.  Select an existing plan.

4.  Within the playbook, review the following steps:

    -   HQ cases
    -   HQ tasks
    -   Store cases
    -   Store tasks
    -   Affected stores
