# Manage store plans to generate cases and tasks

Manage store plans to plan and schedule generation of cases and tasks to simplify manual efforts.

-   **[Create a store plan from the workspace](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/retail-industry/retail-operations/create-plan-workspace.md)**  
The HQ manager creates a new plan for case and task generation.
-   **[Add HQ case details to the HQ communication plan](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/retail-industry/retail-operations/rahi-retail-add-hq-case-details.md)**  
Log the HQ case details to the new HQ communications store plan.
-   **[Add HQ task details](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/retail-industry/retail-operations/rahi-retail-add-hq-task-details.md)**  
Log the HQ task details to the new HQ communications store plan.
-   **[Add store case details](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/retail-industry/retail-operations/rahi-retail-add-store-case-details.md)**  
Log the store case details to the new HQ communications plan.
-   **[Add store task details](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/retail-industry/retail-operations/rahi-retail-add-store-task-details.md)**  
Log the store task details to the new HQ communications plan.
-   **[Add affected Retail stores](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/retail-industry/retail-operations/rahi-retail-add-affected-retail-stores.md)**  
Determine the affected Retail stores from the list of retail stores.
-   **[Schedule the store plan](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/retail-industry/retail-operations/rahi-retail_schedule-the-cases-or-tasks.md)**  
Schedule the store plan based on the requirement- immediate, one time, or recurring. Once the store plan gets scheduled, it automates the case and task generation which reduces manual work.
-   **[Review store plan details](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/retail-industry/retail-operations/review-store-plan-details.md)**  
Review the details logged for the store plan and make any updates in any of the sections, if required.
