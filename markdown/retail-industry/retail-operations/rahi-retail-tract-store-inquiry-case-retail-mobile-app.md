# Track a store inquiry case from the Retail mobile app

Track the status of a store inquiry case from the Retail mobile app. By tracking your case from this mobile app, you can immediately get visibility into the progress, issues, or status of your case without having to use a computer.

## Before you begin

Role required: sn\_rtl\_stre\_servcs.contributor

## Procedure

1.  In the Retail mobile app, select **My Store** &gt; **All cases** from the menu bar.

2.  Select the case number that you want to track.

    If the case number isn't available, search the case keywords by using a search bar on the home page or by using filters on the case list.
