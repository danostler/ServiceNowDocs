# Work on an In-store operations case

Track and fulfill an in-store operations case within Retail Mobile.

## Before you begin

Role required: sn\_rtl\_instore\_ops.manager or sn\_rtl\_instore\_ops.associate

## Procedure

1.  In **Retail Mobile**, select **My Work**.

2.  In **Cases**, select the store case you want to work on.

3.  From the **Actions** dropdown, select the following as needed:

    -   Assign case - Assign the case to someone else.
    -   Assign to me - Assign the case to yourself.
    -   Edit case - Edit the case details.
4.  In the Activity tab, add comments as needed.

5.  Select **Close case** to mark the case as complete.
