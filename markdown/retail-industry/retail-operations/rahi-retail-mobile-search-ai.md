# Search for assigned cases and tasks using AI Search in Retail Mobile

Use the Search function in Retail Mobile to find cases or tasks based on relevant keywords or descriptions.

## Before you begin

Role required: sn\_rtl\_instore\_ops.manager or sn\_rtl\_instore\_ops.associate

**Note:** This requires AI search to be enabled within the Retail Mobile instance. For more information on AI Search, see .

## Procedure

1.  In Retail Mobile, navigate to the **Search** bar.

2.  Enter a keyword or description related to a case or task.

3.  Select matching store cases and tasks as needed.
