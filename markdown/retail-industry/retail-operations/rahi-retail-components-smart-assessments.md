# Components installed with Smart Assessments for CSM

Smart Assessment support in Customer Service Management \(CSM\) and Field Service Management \(FSM\) workspaces provides a unified experience for creating, configuring, and viewing assessment templates and instances.

The following enhancements are available in these workspaces:

-   Smart Assessment functionality is available in Customer Service Management \(CSM\) and Field Service Management \(FSM\) workspaces.
-   Users can create Smart Assessment templates by using a modal dialog within the workspace experience.
-   The Template Overview page is available in CSM and FSM workspaces for defining and managing Smart Assessment questions.
-   Interoperability configurations enable the Template Overview experience from the Smart Assessment workspace to be reused in CSM and FSM workspaces.
-   The integration supports the complete Smart Assessment template creation workflow within CSM and FSM workspaces.
-   A dedicated Smart Assessment Instance record page variant is available in CSM and FSM configurable workspaces.
-   The Smart Assessment Instance record page enables users to view Smart Assessment instance details within the workspace experience.
