# Close a customer complaint case in workspace

Provide a resolution and close a customer complaint case in workspace.

## Before you begin

Roles required: sn\_rtl\_cs\_cmplnt.agent, sn\_rtl\_cs\_cmplnt.agent\_manager

## Procedure

1.  Navigate to **Lists** &gt; **Retail Customer Complaints** &gt; **All**.

2.  Select the case number to work on the case.

3.  Select **Close Case** by adding the following resolution details.

    |Fields|Description|
    |------|-----------|
    |**Resolution Code**|The code number of the resolution.|
    |**Resolution notes**|Note about the resolution.|
