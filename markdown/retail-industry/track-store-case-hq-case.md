# Track a store case from an HQ case

Track a store case from an HQ case to view the status.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **Store Plan** &gt; **All**.

2.  Select a store plan and then select **Store Plan** &gt; **All** &gt; **Track plan** &gt; **HQ case** &gt; **Store case**.

3.  Select the fields on the HQ case details page such as the store cases and tasks.

    |Fields|Description|
    |------|-----------|
    |**Store cases**|Displays the list of store cases created under the Store Plan.|
    |**Store case**|Displays the selected store case details.|
    |**Tasks**|Displays the associated store tasks.|
