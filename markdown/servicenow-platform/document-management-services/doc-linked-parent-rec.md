# Linking documents to a parent record

Documents can be linked to records referred to as "parent records." You can create a document version and reference it through a parent record. You can also view, edit, download, or delete document references in the parent record.

**Note:**

The terms "other record" and "parent record" are used interchangeably and refer to any records to which documents are linked.

You can use references to link a document record to any other table or record on the platform.

-   **[Enable viewing related documents in a parent record](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/servicenow-platform/document-management-services/enable-doc-related-list.md)**  
Add the Documents related list to the record to be able to link and view documents related to the record.
-   **[Create a document record in a parent record](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/servicenow-platform/document-management-services/create-new-doc-parent.md)**  
Add documents to a parent record so they are available for downloading, sharing, or viewing directly from the record.
-   **[Upload a document version in a parent record](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/servicenow-platform/document-management-services/upload-new-doc-version.md)**  
Upload a document version for each document in the parent record.
-   **[Download a document from its parent record](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/servicenow-platform/document-management-services/download-doc-parent-rec.md)**  
Download documents related to a parent record directly from the record.
-   **[Link existing documents to a parent record](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/servicenow-platform/document-management-services/link-doc-parent-rec.md)**  
Link documents that are in your repository to a parent record.
-   **[Unlink documents from a parent record](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/servicenow-platform/document-management-services/unlink-docs-parent-rec.md)**  
Remove and unlink documents from a parent record when they are no longer relevant.
-   **[Delete a document from a parent record](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/servicenow-platform/document-management-services/delete-doc-parent-rec.md)**  
Delete documents directly from a parent record.
