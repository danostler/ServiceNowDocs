# Create a document approval revision rule

Create a document approval revision rule for the document review and approval process.

## Before you begin

Role required: platform\_document\_management\_admin or admin

## Procedure

1.  Navigate to **All** &gt; **Documents** &gt; **Approval and Revision Rules**.

2.  On the form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Name|A unique identifier for the approval rule.|
    |Description|A short description of the approval rule.|
    |Document Conditions|A condition builder that determines which documents use this approval rule.|
    |Active|Activates the rule.|

3.  Select **Submit**.
