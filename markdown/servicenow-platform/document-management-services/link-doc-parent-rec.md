# Link existing documents to a parent record

Link documents that are in your repository to a parent record.

## Before you begin

Role required: none

You must have sufficient permissions to access the document.

## Procedure

1.  Navigate to the required record \(for example, Incident\).

2.  Select the **Documents** tab.

3.  Select **Edit**.

4.  Select the documents you want to link.

5.  In the **Actions on selected rows** drop-down menu, select **Link Documents**.
