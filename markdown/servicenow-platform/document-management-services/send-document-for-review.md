# Send a document version for review and approval

Send a document version for review and approval before publishing the document.

## Before you begin

Role required: none

## Procedure

1.  Navigate to **All** &gt; **Documents** &gt; **Documents**.

2.  Select the document record.

3.  Select the version and select **Submit Version**.
