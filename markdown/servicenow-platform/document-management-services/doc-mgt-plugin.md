# Document Management plugin

With the ServiceNow® Document Management plugin \(**com.snc.platform\_document\_management**\), you can create documents, manage large numbers of documents efficiently, and track each version. The plugin is independent, with no dependencies on existing plugins and has been built on the existing **sys\_attachment** framework.

## Overview

There are Document Management roles the system admin can assign to maintain the lifecycle of documents. The system admin installs the plugin and then assigns one or more of these roles to users.

The system admin installs the Document Management plugin \(**com.snc.platform\_document\_management**\).

The application contains four modules:

-   Documents
-   References
-   Lists
-   List Entries
