# Create provider permissions

Create provider permissions for multi provider documents.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Multi Provider Document** &gt; **Admin** &gt; **Permission Configurations**.

2.  Select **New**.

3.  On the form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Permissions|Name of the document.|
    |Description|A brief description about the provider permission.|
    |Provider details|Select the provider details.|
    |Attributes|Name and value of the attribute.|

4.  Select **Submit**.
