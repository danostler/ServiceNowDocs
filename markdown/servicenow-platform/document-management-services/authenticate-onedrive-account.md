# Authenticate Microsoft OneDrive account

Enable personal authentication for Microsoft OneDrive for Multi Provider Document Services Framework.

## Before you begin

Role required: sn\_personal\_auth.personal\_auth\_user

## Procedure

1.  Navigate to **All** &gt; **IntegrationHub** &gt; **Personal Authentication** &gt; **Personal Integrations**.

2.  Go to Microsoft OneDrive Spoke Credential tile, select **Authenticate**.
