# Cancel a document

Change the document state to Cancelled when no more changes will be made to the document.

## Before you begin

Role required: document\_management\_admin

or

Role required: document\_management\_user

## About this task

When canceling a document, there is no confirmation message, so ensure that you want to cancel the document.

## Procedure

1.  Navigate to the document record.

2.  Right-click the header bar and select **Cancel Document**.

    An info message confirms the cancellation:

    The **State** is set to **Cancelled**.
