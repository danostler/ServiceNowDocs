# Download a document from its parent record

Download documents related to a parent record directly from the record.

## Before you begin

Role required: none

## About this task

**Note:** To download a document, the version file type must be an attachment and not a URL.

## Procedure

1.  Navigate to the required record \(for example, Incident\).

2.  Select the **Documents** tab.

3.  Select the documents you want to download.

4.  In the **Actions on selected rows** drop-down menu, select **Download Document**.
