# Configure permissions

Configure permission for roles and groups for multi provider documents.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Multi Provider Document** &gt; **Admin** &gt; **Permission Configurations**.

2.  Select **New**.

3.  Select the Entity Type.

4.  Select the Entity id by selecting the search icon.

5.  Go to the Documents field, select the role from the document list.

6.  Select **OK**.

7.  Enter Provider group attributes Name and Value.

8.  In the Permissions field, select the permissions that you want to grant.

9.  Select **Submit**.
