# Check out a document

Documents in the Managed Documents application can only be revised by one user at a time. Check out the document to revise it.

## Before you begin

Role required: document\_management\_admin

or

Role required: document\_management\_user

## Procedure

1.  Navigate to the document record.

2.  Select the **Check Out Document** related link.

3.  Select **Download file** to download the current version of the document when it is checked out.

    \[Omitted image "MDOCCheckout.png"\] Alt text: Checkout

4.  Click **Check Out**.

    After checking out a document, you can edit the document and make any necessary changes.
