# Share documents with roles

As a Writer or Owner, assign access and permissions based on user roles.

## Before you begin

Role required: dm\_user\_role\_read

## Procedure

1.  Navigate to **Documents** &gt; **Documents**.

2.  Open the required document record.

3.  Select the **Sharing Permissions** check box.

4.  Select the **Role Permissions** tab.

5.  Select **New** and select the permission level from the drop-down list.

6.  Select the user role you want to assign permission to.

7.  Click **Submit**.
