# Deactivate a document

To change the document state to Inactive, deactivate the document. This is useful when changes to a document have been put on hold. An inactive document can be reopened at any time.

## Before you begin

Role required: document\_management\_admin

or

Role required: document\_management\_user

## Procedure

1.  Navigate to the document record.

2.  Right-click the header bar and select **Deactivate Document**.

    **Note:** To activate a document, right-click on the header and click **Reopen Document**.
