# Unlink documents from a parent record

Remove and unlink documents from a parent record when they are no longer relevant.

## Before you begin

Role required: admin

## About this task

**Note:** This action unlinks the document from the parent record but doesn’t delete it from the repository.

## Procedure

1.  Navigate to the record \(for example, Incident\).

2.  Select the **Documents** tab.

3.  Select the documents you want to unlink.

4.  In the **Actions on selected rows** drop-down menu, select **Remove**.
