# Create documents using document lists for a target table record

You can create and reference documents on a target table record through the DocumentListService API.

## Before you begin

See the DocumentListService API topic

Role required: admin

**Note:** You cannot create documents through Document Lists with the user interface \(UI\). It must be done with APIs.
