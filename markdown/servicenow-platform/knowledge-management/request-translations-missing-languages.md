# Request missing translations for available languages

Manually create translation tasks if an article does not have translation tasks created for all the languages you have configured.

## Before you begin

-   Role required: none
-   Verify that the **glide.knowman.translation.enable\_translation\_task** property is set to true.

## Procedure

1.  Navigate to **All** &gt; **Knowledge** &gt; **Articles** &gt; **Published**.

2.  Select an article from the list of published articles.

3.  Click the **Request translations** related link.

4.  Select the languages from the list that are unselected, which indicates they do not have associated translation tasks.

5.  Click **Submit**.


## Result

The translation tasks for the languages are created and listed in the Knowledge Translation Tasks tab.
