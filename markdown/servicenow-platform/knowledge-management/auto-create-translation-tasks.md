# Auto-create translation tasks

Auto-create translation tasks in specified languages to avoid having to manually create translation tasks after an article is published. The available languages are determined by the language plugins that have been installed on your instance.

## Before you begin

Role required: knowledge\_admin or admin

## Procedure

1.  Navigate to **All** &gt; **Knowledge** &gt; **Administration** &gt; **Knowledge Bases**.

2.  Select an existing knowledge base.

3.  Display the languages you have installed by clicking the lock icon to unlock the **Languages** field.

4.  Select the languages for which you want to create tasks from the drop-down list.

5.  Select **Auto-create translation tasks**.

6.  Click **Submit**.


## What to do next

Open a knowledge article and access the Knowledge Translation Tasks related list to view the associated translation tasks.
