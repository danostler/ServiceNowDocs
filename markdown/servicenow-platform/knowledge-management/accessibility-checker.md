# Accessibility checker in the toolbar

Accessibility checker in the toolbar enables you to identify and resolve accessibility related issues in a Knowledge article.

## Before you begin

Open the Knowledge article in which you want to use the Accessibility checker.

Role required: user with contribute access

## Procedure

1.  Navigate to **All** &gt; **Knowledge** &gt; **Articles** &gt; **All**.

2.  Select a Knowledge article.

3.  Select the **Accessibility checker** icon \[Omitted image "accessibility-checker-icon.png"\] Alt text: Accessibility checker icon in the toolbar of the article body.

4.  View accessibility-related issues in the **Accessibility Checker** window along with suggestions for resolution of each issue.

5.  Select **Repair** to implement the suggestion, or **Ignore**.

    If there are no accessibility issues in the article, then the **Accessibility Checker** window displays a success message:

    ```
    No accessibility issues detected
    ```
