# Queue workflow activity

The **Queue** activity places an active Notify call in a queue.

Add the **Queue** activity to a workflow on the Notify Call \[notify\_call\] table to put the current call on hold. This activity does not specify any input variables.

The queue that the call is added to is given a random ID.
