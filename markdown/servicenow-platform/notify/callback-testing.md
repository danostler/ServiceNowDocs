# Twilio-Direct callback testing

Make an outgoing call or send an SMS using Twilio and trigger callbacks to ServiceNow to test whether Twilio can reach the instance. You can also view when the last callback test was run, its status, and the test logs.

## Before you begin

Role required: notify\_setup\_admin, notify\_admin

## Procedure

1.  Navigate to **All** &gt; **Notify** &gt; **Administration** &gt; **Twilio Direct Configuration**.

2.  Click **Test Callbacks**.

3.  Select the SMS option and click **Initiate SMS**.

4.  Select the call option and click **Initiate Test Call**.

    **Note:** The callbacks can be tested only after a connection is established with Twilio.
