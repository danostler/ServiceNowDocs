# Disable Zoom meeting password

Disable Zoom meeting password so that you can join a Zoom meeting without any meeting password.

## Before you begin

Role required: notify\_admin or notify\_setup\_admin

## Procedure

1.  Navigate to **Notify** &gt; **Zoom** &gt; **Configuration**.

2.  Select **Do not set password**.

3.  Click **Update**.
