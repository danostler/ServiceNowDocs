# Initiate Notify Cisco Webex conference from MIM workbench

Initiate a Webex conference from the MIM workbench to work towards the resolution of the major incident.

## Before you begin

Role required: notify\_view, or major\_incident\_manager

## Procedure

1.  Navigate to **All** &gt; **Incident** &gt; **Major Incidents** &gt; **Open**.

2.  Open the relevant major incident.

3.  Select **View Workbench**.

4.  Select **Collaborate** tab.

5.  Select **Initiate** button.

6.  On the form, fill the fields.

<table id="table_ic1_ypm_1rb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Conference Bridge

</td><td>

Option to select the conference bridge for the meeting.

 The Conference Bridge will be selected as Webex only Webex is the default provider selector.

</td></tr><tr><td>

Add Participants

</td><td>

Option to add the users and the stakeholders to work towards the resolution of the ticket.

 -   **Users**: Enter the name of the user to include in the call.
-   **Group**: Enter the group of users to be included in the call.
-   **Email**: Enter a valid email of the participant to be included in the call.
 After adding the required stakeholders, click **Add to selected**.

</td></tr><tr><td>

Title

</td><td>

The default title is the ticket number appended with the short description.

</td></tr><tr><td>

Include a brief message for participants

</td><td>

Option to provide an opening message to the user before beginning the interaction.

</td></tr></tbody>
</table>7.  Select **Start**.

8.  Select **Open Link** in the dialog box that appears.

    You can also click the link from the workbench to begin the Webex conference call.
