# Manage Notify properties

Manage the Notify Webex Connector application properties to configure the meeting capabilities.

## Before you begin

Role required: notify\_setup\_admin

## Procedure

1.  Navigate to **All** &gt; **Notify** &gt; **Administration** &gt; **Notify Properties**.

2.  On the form, fill the fields.

<table id="table_esc_zcm_1rb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Enable Notify integration for Task table and extensions

</td><td>

Option to enable Notify integration for Task table and extensions.

 Select the check box.

</td></tr><tr><td>

Enable Notify integration for Incident Communications Management. Entering phone number in 'com.snc.iam.notify\_number' property is equivalent to setting this to true.

</td><td>

Option to use Notify Webex Connector for incident and major incident-related records.

 Select the check box.

</td></tr></tbody>
</table>3.  Select **Save**.
