# View a summary of map contents on the Overview panel

The Overview panel displays summary data for items on the map that are associated with the home node — counts and types of CIs and connections, discovery sources, and so on.

## Before you begin

Role required: sn\_cmdb\_user

## About this task

**Note:** The information on the Overview panel isn't affected by zoom settings or by filter settings even if nodes that are associated with the home node are not visible on the map.

## Procedure

1.  While working in a map, in the Search bar, search for and select the CI to set as the home node and then select the Overview icon \[Omitted image "icon-um-overview-panel.png"\].
