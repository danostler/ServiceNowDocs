# Accessing the IntegrationHub ETL dashboard in SGC Central

You can access the IntegrationHub ETL dashboard within the SGC Central view of the Service Graph Workspace or CMDB Workspace to manage ETL transform maps.

IntegrationHub ETL is a tool for creating and managing ETL transform maps to integrate third-party data into the CMDB or non-CMDB tables. Select **Go to IntegrationHub ETL Home** to view the IntegrationHub ETL home page.

To learn about the IntegrationHub ETL store app, see [IntegrationHub ETL](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/servicenow-platform/integration-hub-etl/integrationhub-etl.md).

## Access IntegrationHub ETL in SGC Central

The IntegrationHub ETL dashboard provides a centralized platform for monitoring ETL processes within the SGC Central view of the Service Graph Workspace or CMDB Workspace.

### Before you begin

Role required: admin

### Procedure

1.  Use one of the following methods to open SGC Central:

    -   Navigate to **Workspaces** &gt; **Service Graph Workspace**, and from the left navigation panel, select the Ingestion icon to open the SGC Central view.
    -   Navigate to **Workspaces** &gt; **CMDB Workspace** &gt; **SGC Central**.
2.  Select **IntegrationHub ETL**.

3.  In the ETL definitions section of the IntegrationHub ETL page, select a category to view more details about the selected ETL definition.

    |Category|Description|
    |--------|-----------|
    |Total|Total number of ETL definitions configured within the system.|
    |Active|Number of ETL processes that are currently active and running.|
    |Scheduled|Number of ETL processes that are scheduled for future execution.|

4.  To open the home page of the IntegrationHub ETL store app, select **Go to IntegrationHub ETL Home**.
