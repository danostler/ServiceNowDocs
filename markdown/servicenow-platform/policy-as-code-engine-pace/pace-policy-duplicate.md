# Duplicate a PaCE policy

Create a duplicate of an existing policy.

## Before you begin

Role required: sn\_pace.policy\_editor

## Procedure

1.  Navigate to **All** &gt; **Policies** &gt; **My Policies**.

    **Note:** To view all policies generated by admin/other users, navigate to **Policies** &gt; **All Policies**.

2.  Click the **Policy name** link for the policy that you want to duplicate.

3.  In the Details page, click the **More Actions** icon and select **Duplicate**.

4.  In the Duplicate Policy window, select the version of the policy you want to copy and click **Duplicate**.

    A new policy with the same parameters as the selected policy is created in an **Inactive** state. You must publish a version of the new policy before you can activate it.
