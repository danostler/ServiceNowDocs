# Map PaCE policies using Static Mapping

Use Static Mapping to map multiple PaCE policies against a record such as a change request.

## Before you begin

Role required: sn\_pace.mapping\_admin

## Procedure

1.  Navigate to **Mappings** &gt; **Static Mappings**.

2.  Select **Add**.

3.  Select a document type in the **Document Type** field.

4.  Select a policy in the **Policy** field or you can select the Search icon \[Omitted image "pace-search-icon.jpg"\] Alt text: Search icon. to view and select multiple policies, then select **Map**.

5.  When you're finished, select **Done**.


## Result

The map will show up in the Static Mappings list. You can delete a map on the list by selecting the box next to the Input, then select **Delete**.
