# Using Quizzes

You can use quizzes.

-   **[Set up and administer quizzes](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/servicenow-platform/ai-platform-capabilities/t_QuizProcess.md)**  
Set up and administer quizzes.
-   **[Data types for quizzes](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/servicenow-platform/ai-platform-capabilities/r_AvailableDataTypes.md)**  
You need to choose a data type for each quiz question. Various data types helps you collect and analyze different kinds of data.
-   **[Activate the quiz designer](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/servicenow-platform/ai-platform-capabilities/t_ActivateQuizDesigner.md)**  
Administrators can activate the Quiz Designer plugin.
-   **[View quiz reports](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/servicenow-platform/ai-platform-capabilities/t_QuizOverviewModule.md)**  
Use the Quiz Overview dashboard to view various reports on data such as Quizzes by State, Questions by Data Type, and Number of Correct Answers.
-   **[Quiz designer](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/servicenow-platform/ai-platform-capabilities/c_QuizDesigner.md)**  
The quiz designer provides a single interface that users with the assessment\_admin role can use to create, edit, and distribute quizzes.
-   **[Take a quiz](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/servicenow-platform/ai-platform-capabilities/t_CompleteaQuiz.md)**  
All available quizzes assigned to you appear in your personal assessment and survey queue. When you submit a quiz, the system stores your responses. If a quiz allows retakes, you can resubmit it as many times as you like until its due date.
-   **[View a quiz instance](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/servicenow-platform/ai-platform-capabilities/view-quiz-instance.md)**  
View an instance to verify that quiz instances were created, to check the state of a quiz instance, or to reassign a quiz instance. A quiz instance represents one questionnaire assigned to one user.
-   **[Add a metric category and metric in the question bank for quizzes](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/servicenow-platform/ai-platform-capabilities/add-questionbank-quiz.md)**  
Reuse the question categories \(metric categories\) and questions \(metrics\) added in the question bank for quizzes. You can add metric categories or metrics from the question bank to a quiz, or from the quiz to a question bank.
-   **[Create quizzes with forms](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/servicenow-platform/ai-platform-capabilities/c_CreateQuizzesWithForms.md)**  
As an alternative to the Quiz Designer, you can create a complete quiz using records in the Assessment application.
