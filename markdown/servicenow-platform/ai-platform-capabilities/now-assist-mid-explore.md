# Explore Now Assist agentic workflows for MID Server

The Now Assist for MID Server application uses generative AI to analyze errors and provide remediation procedures. The procedures collate technical information from multiple web sources to help you investigate the issues more effectively.

Analysis tools such as MID Guardian completely transforms the issue triage process and can help achieve faster, seamless problem management. Powered by generative AI, it uses a large language model \(LLM\) to generate human-readable alert titles and detailed descriptions based on enriched and actionable alert data. The AI-generated alert information facilitates quick alert triage and effective analysis for your team, which can lead to a substantial reduction in resolution time.
