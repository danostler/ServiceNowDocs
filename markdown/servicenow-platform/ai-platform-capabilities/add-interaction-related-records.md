# Add Interaction Related Records to HR Agent workspace

ServiceNow Voice for HR creates an interaction record for every call between an employee and an HR agent, which contain related lists of details relating to the employee and interaction. To maximize the full feature capability, add the Interaction record to the HR Agent workspace.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Interaction** &gt; **All**.

2.  Open any IMS record.

    If there are no records, create a dummy record.

3.  Click **Additional actions** &gt; **Configure** &gt; **Related lists**.

4.  Select the **HR Workspace** view.

5.  Locate the **Interaction Related Record -&gt; Interaction** option and add it to the **Selected** list.

    \[Omitted image "voice-related-list.png"\] Alt text: Select the HR Workspace view and the Interaction Related Record - Interaction option

6.  Click **Save**.
