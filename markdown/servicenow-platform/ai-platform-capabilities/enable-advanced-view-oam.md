# Enable Advanced view for Outlook Actionable Message definition

Enable the advanced view to customize the actionable message as per the requirement for Microsoft Outlook.

## Before you begin

Role required: oam\_admin

## Procedure

1.  Navigate to **All** &gt; **Actionable Messages** &gt; **OAM Definition**.

2.  Choose one of the following:

    |Option|Action|
    |------|------|
    |Request|Click **OAM Request Approval**|
    |Request Item|Click **OAM RITM Approval**|
    |Change|Click **OAM Change Approval**|

3.  Click **Advanced OAM** under **Related Links**.

4.  In the OAM Definition form, perform the following actions:

    1.  Modify the script as required.
    2.  Click **Update**.
