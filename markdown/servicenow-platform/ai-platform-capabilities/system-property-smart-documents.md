# System property for Smart Documents skill

Configure and set the system property to use Smart Documents skill.

## System Property for Smart Documents skill

Role required: Admin

**Note:** To view the System Properties table, enter `All Properties` in the navigation filter.

|System Property|Description|
|---------------|-----------|
|**glide.ui.seismic.attachment.dialog.renderInAppShell**|Set the value to `True` to use Smart documents skill.|
