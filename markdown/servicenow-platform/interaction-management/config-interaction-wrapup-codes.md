# Create interaction wrap-up codes

Wrap-up codes enable agents to quickly wrap up and close interactions. Agents can select wrap-up codes from the interaction wrap-up modeless dialog.

## About this task

Users with the system administrator role can create wrap-up codes for agents to use during the wrap-up period. When interaction wrap-up codes are enabled, agents can select wrap-up codes from the interaction wrap-up modeless dialog.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Interaction** &gt; **Wrap Up Codes**.

2.  Select **New**.

3.  Fill in the fields on the Interaction Wrap Up Code form.

    |Field|Description|
    |-----|-----------|
    |Code|The name of the wrap up code.|
    |Application|The application that contains this record. The default is CSM and FSM Configurable Workspace Foundation.|
    |Active|Indicates whether the interaction wrap-up configuration is active. The default value is true.|
    |Description|A brief description of the wrap-up code.|

4.  Select **Submit**.


## What to do next

After creating wrap-up codes, system administrators can enable wrap-up codes in the interaction wrap-up configuration record and select the wrap-up codes that are available to agents. For more information, see [Create an interaction wrap-up configuration](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/servicenow-platform/interaction-management/create-interaction-wrap-up-config.md).
