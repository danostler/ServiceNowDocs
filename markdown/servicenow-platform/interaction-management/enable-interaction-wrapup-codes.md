# Enable interaction wrap-up codes

Enable interaction wrap-up codes and select the wrap-up codes that are available to agents in the interaction wrap-up modeless dialog.

## About this task

Users with the system administrator role can enable wrap-up codes for agents to use during the wrap-up period. When enabled, agents can select wrap-up codes from the configured list in the interaction wrap-up modeless dialog.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Interaction** &gt; **Wrap Up Configuration**.

2.  Select a configuration record.

3.  In the Duration related list, enable the following fields.

    -   **Enforce wrap up duration**
    -   **Show duration to agent**
4.  Enter a value in the **Duration in seconds** field.

5.  In the Codes related list, enable the **Enable wrap up codes** field.

6.  In the **Allowed wrap up codes** field, select the desired codes and move them from the Available column to the Selected column.

7.  Select a **Default wrap up code**.

8.  Select **Update**.
