# Define a data source

The data source is an entity that has associated data that you intend to display. For example, a community post or a user.

## Before you begin

Role required: admin or evam\_admin

## Procedure

1.  Navigate to **All** &gt; **Entity View Action Mapper \(EVAM\)** &gt; **Data sources**, and then select **New**.

2.  On the form, fill in the fields.

</th><th>

</th></tr></thead><tbody><tr><td>

</td><td>

</td></tr><tr><td>

Name of the data source.

</td></tr><tr><td>

</td></tr></tbody>
</table>3.  Select **Submit**.
