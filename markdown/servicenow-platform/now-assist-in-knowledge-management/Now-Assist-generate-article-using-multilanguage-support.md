# Generate a Knowledge article using multi-language support

Now Assist in Knowledge Management provides multi-language support for Knowledge article generation. Generate knowledge articles in multiple languages with the Now Assist multi-language support for Knowledge Management. As an agent, you can generate articles from incidents, cases and other task types that are documented in languages other than English by using a language selector configured by the admin.

## Before you begin

Languages available in the language selector are determined by:

-   The language package installed for the Knowledge Base
-   The language preference selected by the admin in the knowledge settings

Role required: author or agent

## Procedure

1.  Navigate to **All** &gt; **Knowledge** &gt; **Articles** &gt; **All**

2.  Open a case or an incident assigned to you that is documented in a language other than English.

3.  Select **Create Knowledge**.

    \[Omitted image "NA-multi-language-1.png"\] Alt text: Create a new Knowledge article from a non-English incident, case or other task type.

4.  In the Use Al to draft this article? modal, select **Yes, draft with Now Assist**.

    \[Omitted image "NowAssist\_CoreUI.png"\] Alt text: Draft the article using Now Assist.

5.  In the Select Options modal select the **Knowledge base** in which you would like to publish the article and the **Article language** to be used to generate the article.

    \[Omitted image "NA-multi-language-2.png"\] Alt text: Select the Knowledge Base and the Article language to generate the article.

6.  Select **Continue**.

    \[Omitted image "NA-multi-language-3.png"\] Alt text: Knowledge article generated by Now Assist in the selected language.

    Now Assist generates the Knowledge article in the language selected.
