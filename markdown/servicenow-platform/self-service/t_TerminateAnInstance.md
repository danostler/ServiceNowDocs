# Terminate an instance

Shut down an Amazon EC2 instance through the service catalog.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Service Catalog** &gt; **Routine Changes** &gt; **Terminate Amazon EC2 Instance**.

    A list appears displaying the instances that are assigned to you.

2.  Select an instance from the list and click **Order**.

    This list does not support multi-line selection, and there is no **undo** feature.
