# Add new request items

Use ServiceNow's web-based interface to add new items to a request.

## Before you begin

Roles required: Only an administrator or a user with the catalog\_admin role can add a new item to a submitted request.

## About this task

Modifying a request does not resubmit it to the approval process.

## Procedure

1.  Navigate to **All** &gt; **Service Catalog** &gt; **Open Records** &gt; **Requests** and open an existing request.

2.  Click **Add New Item**.

    **Note:** **The Add New Item** button appears only if the current **Request state** is **Pending Approval**. After the request is approved, this button is no longer available and you will not be able to add new items.

3.  In the dialog box that appears, choose an item and a quantity to add.

4.  Click **OK**.
