# Edit the workflow activity properties form

Customize which workflow variables appear on an activity properties form and how the variables are arranged on the form.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **Workflow** &gt; **Administration** &gt; **Activity Definitions**.

2.  Scroll to the activity that you want to work with and click the activity name.

3.  On the Workflow Activity Definition form, click the **Edit Variables Layout** related link.

4.  On the Form Design page for the Activity Properties form, add and remove activity variables.

    The activity variables appear as separate items that you can rearrange on the form.

    For more details on using the form design interface, see [Using the form designer](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/platform-administration/ai-platform-administration/c_FormDesign.md).
