# Run Script workflow activity

The **Run Script** activity runs the specified script in the scope of the workflow version.

**Note:** All changes to `current` are automatically updated. There is no need to call current.update\(\)

## Input variables

Input variables determine the initial behavior of the activity.

|Field|Description|
|-----|-----------|
|Script|Script to execute.|
