# Download the Password Reset Windows Application installer

Download the Password Reset Windows Application installer.

## Before you begin

Role required: any user

## Procedure

1.  Log in to the ServiceNow Store Publisher Portal.

2.  Click your user menu and select **Store**.

    \[Omitted image "pwr-winapp-user-store.png"\] Alt text: Store option on the portal

3.  Set the filter for the **Ancillary Software** app type.

    \[Omitted image "pwr-winapp-filter-ancillary.png"\] Alt text: Select the Ancillary Software App Type

4.  Click **Password Reset Windows Application**.

    The application download location appears.

    \[Omitted image "pwr-winapp-app-downld-loc.png"\] Alt text: Password Reset Windows Application download location

5.  Click **Download**.
