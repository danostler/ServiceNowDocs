# Enroll for the Password Reset program using SMS codes

To verify your identity while resetting your password, you can use a code \(the SMS code\) that was delivered to your mobile phone or device. You can enroll on a mobile device.

## Before you begin

Role required: none

## Procedure

1.  Navigate to **All** &gt; **Password Reset** &gt; **Enroll** and then click the **SMS Verification** tab.

    If your organization requires you to enroll for the SMS verification method, then the tab is marked with an asterisk \(\*\).

2.  To add and verify a device, click **Add Device** and then follow the instructions.

    **Note:** If you had previously added an SMS-enabled device to your Password Reset profile, the device is listed.

3.  Provide a meaningful name for each device, and if you want to add multiple devices, then repeat the process.

4.  After you verify each device, select the **Authorized** check box for each device that you expect to use to change or reset your password.

    **Note:** You can deauthorize a device at any time by clearing the **Authorized** check box.

5.  Select **Submit**.

6.  Enroll for an additional identity verification using any of the other methods that your organization offers.
