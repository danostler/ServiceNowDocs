# Enroll for the Password Reset program using Soft PIN

Verify your identity while resetting your password by using a Soft PIN. To use it, you must enroll for the verification using a Soft PIN.

## Before you begin

Role required: none

## Procedure

1.  Navigate to **All** &gt; **Password Reset** &gt; **Enroll**.

2.  Select the **Soft PIN Verification** tab.

    **Note:**

    If your organization requires you to enroll for the Soft PIN verification, then the tab is marked with an asterisk \(\*\).

3.  In the **Enter the Soft PIN** field, enter a six-digit number of your choice.

    Make sure that you follow these rules while enrolling for the Soft PIN verification.

    -   The length of the Soft PIN must be six digits.
    -   The soft PIN repetition threshold is two.
    -   The soft PIN sequence threshold is two.
    -   You can't use your previous five Soft PINs.
    **Note:** Based on the configuration set by your admin, these rules might change.

4.  Select **Submit**.

    You're enrolled for the Soft PIN verification.
