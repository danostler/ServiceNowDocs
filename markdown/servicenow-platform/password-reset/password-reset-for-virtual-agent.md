# Password Reset for Virtual Agent

Augment an end-user experience with Password Reset for Virtual Agent that handles common support issues, using pre-built conversations.

For more information about the Virtual Agent, see [Virtual Agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/conversational-interfaces/virtual-agent/virtual-agent-landing-page.md).

Your end users can perform the following actions through the Virtual Agent conversations:

-   Reset a password
-   Change a password
-   Unlock an account
-   Enroll to reset passwords

The end users can reset passwords and unlock accounts even if they aren’t logged in. When the users are logged in, they can do all the actions including changing passwords. After changing or resetting a password, the users can’t change or reset the password for the next 24 hours.
