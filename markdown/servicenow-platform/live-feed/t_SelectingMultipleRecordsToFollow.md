# Follow a record feed from a list

You can follow records that have a record feed associated with them when viewing a list of records. This method allows you to select multiple records to follow.

## Before you begin

Role required: none

## Procedure

1.  Navigate to a list of records \(for example, **Assigned to me**\).

2.  Select the **Action** check box beside each record to follow.

3.  In the **Actions** choice list, select **Follow on Live Feed**.
