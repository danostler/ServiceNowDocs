# Manage Live Feed message content

Users can remove their own messages from feeds. If necessary, administrators can remove inappropriate messages that are posted by any user.

## Before you begin

Role required: live\_feed\_admin and admin

## About this task

For security reasons, any HTML code is automatically stripped out of a message before it is posted. This measure ensures that users cannot modify any page settings by posting a message.

## Procedure

1.  Navigate to **Live Feed** &gt; **Feed Administration** &gt; **Messages**.

2.  Open the message to be removed.

3.  In the **State** field, select **deleted**.

    You may need to configure the form to add this field.

4.  Click **Update**.
