# Email notifications installed with Live Feed

The Live Feed plugin includes the following email notifications.

|Email notification|Description|
|------------------|-----------|
|Live Feed Feed Member Accepted|Sends an email to the administrator when a user has accepted to join the feed.|
|Live Feed Feed Member Declined|Sends an email to the administrator when a user has declined to join the feed.|
|Live Feed Feed Member Invited|Sends an email to a user when they have been invited to join a feed.|
|Live Feed Feed Member Left|Sends an email to the administrators when a user has left the feed.|
|Live Feed Feed Member Removed|Sends an email to user when they have been removed from the feed.|
|Live Feed Feed Member Request|Sends an email to administrators when an user requests to join a feed.|
|Live Feed Feed Member Request Accepted|Sends an email to the user when their request to join a feed is accepted.|
|Live Feed Feed Request Rejected|Sends an email to the user when their request to join a feed has been rejected by an administrator.|
|Live Feed MsgReply Subscription|Sends an email when a reply or reply\_to\_reply message is inserted into the thread of a message in a feed.|
|Live Feed Team Member Accepted|Sends an email to administrators when a user accepts an invitation to join a team.|
|Live Feed Team Member Declined|Sends an email to administrators when a user declines an invitation to join a team.|
|Live Feed Team Member Invited|Sends an email when a user is invited to join a team.|
|Live Feed Team Member Left|Sends an email when a user leaves a team.|
|Live Feed Team Member Removed|Sends an email when an administrator has removed a user from a team|
|Live Feed Team Member Request|Sends an email when a user requests to join a team.|
|Live Feed Team Member Request Accepted|Sends an email when a user's request to join a team is accepted.|
|Live Team Member Request Rejected|Sends an email when a user's request to join a team is rejected.|
|Live Message All Subscription|Sends an email to subscribed users when any message \(new or reply\) is posted.|
|Live Message Feed Subscription|Sends an email when a new message is inserted into a feed.|
|Live Message Liked|Sends an email to the creator of a message when another user likes the message.|
|Live Message Mention|Sends an email to a user when that user is mentioned in a message.|
|Live Message New Posts Subscription|Sends an email to subscribed users when a new \(not reply\) message is posted.|
|Live Message Reply|Sends an email to all users in a feed thread when a user posts a reply \(live\_message.replied event\).|
