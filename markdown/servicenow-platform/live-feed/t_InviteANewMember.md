# Invite a new member

You can invite another user to join a team.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **Live Feed** &gt; **Live Feed**.

2.  In the sidebar, click your profile picture, then select the **Teams** tab.

3.  From the team list, select the team name.

4.  Click the **Pending members** tab.

5.  In the **Add User** field, enter the name of the user you want to invite.

    The invited member's picture displays.
