# Control how Live Feed is enabled for records

A property called **glide.ui.show\_live-feed\_activity** controls whether Live Feed is automatically enabled for all records.

This glide record is disabled by default. If the property is enabled, a record feed is automatically created whenever a record is viewed, and you become a member of that feed.

A second property, **glide.live-feed.auto\_join\_document\_group**, enables automatic membership to a record feed whenever the associated document is visited and the user clicks **Show Live Feed**.
