# Remove a hashtag from a posted message

As a participant in a conversation, you can remove hashtags from messages that have already been posted.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to the posted message.

2.  Click **Hashtags** in the message posting.

    The hashtags associated with the message display, along with the removal icon.

3.  Click the removal icon.
