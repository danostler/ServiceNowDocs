# Script includes installed with Live Feed

The Live Feed plugin installs the following script includes.

|Name|Description|
|----|-----------|
|LiveMsgUtil|Provides helper functions for working with live\_message records.|
|LiveFeedFilter|Queries filters for working with live\_feed records.|
|LiveFeedUtil|Provides helper functions for working with live\_feed records.|
