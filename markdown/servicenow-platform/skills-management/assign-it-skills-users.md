# Assign skill proficiency levels for skills

Add a skill and assign skill levels to users to facilitate the automatic assignment of skill-based tasks to agents.

## Before you begin

Role required: skill\_manager or skill\_admin

## About this task

Only skill administrators can create or edit skills. Skill managers can assign skills to users.

## Procedure

1.  Navigate to **All** &gt; **Skills** &gt; **Manage IT Skills**.

2.  Assign a skill level to a user or add a skill along with the skill level.

1.  Locate the user to whom you want to add a skill.
2.  Select **Add** in the skill field for the user for whom you want to add the skill proficiency level.

**Note:** When you select a skill, up to five skill levels display for each skill. If a user has a skill without a skill level, it displays as **Acquired**.

3.  Select a level.
4.  Select **OK**.
The skill level for the skill is assigned to the user. You can also select an existing skill level and select a higher or lower skill level for the user.

**Create new skill**

</td><td>

1.  Select **Add Skill**.
2.  In the **Name** field, enter a name for the skill.
3.  In the **Level type** field, select the type to which the skill belongs. For example, Language or Technical.
4.  In the **Skill categories** field, select one or more categories to which the skill belongs. For example, Language Skills and Reporting.
5.  Select **Submit**.
