# Create skill categories using sample data

Onboard your team quickly using the skill taxonomy framework provided with your application.

## Before you begin

Role required: skill\_admin

## About this task

The sample data provides 150 records that you can use to get started building your skill taxonomy.

## Procedure

1.  Activate the Skills Library Data for Skills Management \(com.snc.skills\_management.seed\_data\) plugin.

2.  Navigate to **Skills** &gt; **Skill Taxonomy**.

3.  In the **Skill Categories** list, review each category and set the **Add skills** field to **true** for all skill categories you want to add to your application.
