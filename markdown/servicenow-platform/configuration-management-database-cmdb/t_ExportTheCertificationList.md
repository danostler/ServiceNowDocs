# Export the certification list

Users with the certification\_admin role can export the certifications list and save the list in Excel, CSV, XML, or PDF format. This list is useful when you have a long list of certification elements or if many different users are assigned to certify elements on a single certification schedule.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Data Certification** &gt; **Tasks** &gt; **All Tasks**

2.  Open a task.

3.  Open any column context menu in the certification data list and complete the export.
