# Indicators used in the CMDB success advisor for Data Foundations dashboard

Indicators enable viewing of high-level metrics that highlight data quality, completeness, and integration coverage issues across principal classes and configuration items \(CIs\).

## CMDB success advisor for Data Foundations dashboard indicators

-   **DF CIs by principal class**

    Total number of CI records across principal classes, measured daily and grouped by CI class and data integration source.

-   **DF CIs by discovery source**

    Total number of CI records from principal classes, measured daily and grouped by the discovery source populating each CI.

-   **DF CIs missing name**

    Total number of CI records from principal classes missing a name, measured daily, where the CI does not have a name specified.

-   **DF CIs missing location**

    Total number of CI records from principal classes missing a location, measured daily, where the CI does not have a location assigned.

-   **DF CIs missing managed by group**

    Total number of CI records from principal classes missing a managed by group, measured daily, where the CI does not have a managed by group assigned.

-   **DF duplicate CIs**

    Total number of CI records within principal classes identified as duplicates based on key matching attributes, measured daily.

-   **DF CIs not updated in last 7 days**

    Total number of CI records from principal classes not updated in the last seven days, measured daily, where the update timestamp is older than seven days.

-   **DF CIs not updated in last 14 days**

    Total number of CI records from principal classes not updated in the last 14 days, measured daily, where the update timestamp is older than 14 days.

-   **DF CIs not updated in last 30 days**

    Total number of CI records from principal classes not updated in the last 30 days, measured daily, where the update timestamp is older than 30 days.

-   **DF CIs not updated in last 60 days**

    Total number of CI records from principal classes not updated in the last 60 days, measured daily, where the update timestamp is older than 60 days.

-   **DF CIs not updated in last 90 days**

    Total number of CI records from principal classes not updated in the last 90 days, measured daily, where the update timestamp is older than 90 days.
