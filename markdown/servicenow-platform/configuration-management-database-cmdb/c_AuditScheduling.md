# Schedule and run an audit

The system performs audits automatically from the schedule you configure.

Users with the certification\_admin or admin role can generate on-demand audits directly from the Audit form by clicking **Run Audit**. When an audit runs, ServiceNow populates the **Audit Results** related list in the form and shows follow-on tasks, if any, in the **Follow On Tasks** related list. Click **Preview Audit Results** to generate an audit preview that tests your template conditions without generating any audit results.
