# Configure Slack for a communication task

Configure Slack as a mode of communication for a communication task to send direct message to the users.

## Before you begin

Role required: sn\_comm\_management.comm\_plan\_admin or admin

Install the Collaboration Services for Task Communications Management plugin \(sn\_tcm\_collab\_hook\) from ServiceNow Store.

## Procedure

1.  Navigate to **All** &gt; **Incident** &gt; **Communication Plan Definitions**.

2.  Open the communication plan where you want to add Slack as a communication channel for a communication task.

3.  Open the communication task definition for which you want to add Slack as the mode of communication.

4.  Click the **Add Channel – Slack** related links.

5.  Click **Submit**.

    Slack gets added as a communication channel for the communication task.
