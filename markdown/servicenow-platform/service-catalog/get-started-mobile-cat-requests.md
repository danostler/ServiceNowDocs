# Get started with the mobile app for Approvals

Access the My Approvals application instance on your mobile app to manage approval requests using your mobile device.

## Before you begin

Role required: business\_stakeholder or approver\_user

## About this task

Download the ServiceNow® mobile application on an iOS platform from the Apple App Store or on an Android platform from the Google Play Store.

## Procedure

1.  Open the mobile app and tap the plus icon \(\[Omitted image "mobileAddInstance.png"\] Alt text: plus icon\).

2.  Add a ServiceNow instance. For more information on adding the instance, refer to Add a ServiceNow instance.

3.  Tap **My Approvals** to start managing your requests.
