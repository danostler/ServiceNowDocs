# Debugging Service Catalog

Debug user criteria of catalog items, UI customizations for Service Catalog pages, actions on catalog item variables, performance diagnostics of a catalog item, and so on.

-   **[Debug actions on catalog item variables](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/servicenow-platform/service-catalog/debug-actions-on-catalog-item-variables.md)**  
Analyze the client-side actions affecting the state and values of all variables. You can also get insights into the performance of each script working on the variables.
-   **[Debug user criteria of a catalog item](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/servicenow-platform/service-catalog/debug-user-criteria-cat-item.md)**  
Debug whether a user can access a catalog item based on the matched user criteria and fix any issues.
-   **[Debug UI customizations for a Service Catalog page](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/servicenow-platform/service-catalog/debug-ui-customization-catalog-page.md)**  
Perform the health check of a Service Catalog page. The UI Page Customization Diagnostics window displays the UI macros applicable for the Service Catalog page and their corresponding states. For issues on a Service Catalog page due to customized UI macros, you can identify the source of broken functionality.
-   **[Debug performance diagnostics of a catalog item](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/servicenow-platform/service-catalog/debug-perf-diagnostics.md)**  
Analyze the impact of the variable setup in a catalog item on its runtime performance, and identify any issues. You can review the processing time of the catalog item and its variables based on the triggered SQL queries.
-   **[Debug a Service Catalog variable](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/servicenow-platform/service-catalog/debug-a-service-catalog-variable.md)**  
 Monitor changes in the state and the value of a Service Catalog variable due to catalog client scripts, catalog UI policies, and catalog data lookups. You can diagnose and resolve issues related to such changes, and track client-side scripting errors on this tab.
-   **[Item diagnostic report](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/servicenow-platform/service-catalog/c_ItemDiagnostic.md)**  
For a catalog item data and configuration, you can identify possible violations from the ServiceNow recommended practices.
-   **[Service catalog UI policy](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/servicenow-platform/service-catalog/c_ServiceCatalogUIPolicy.md)**  
UI policies can be used to define custom process flows for tasks.
