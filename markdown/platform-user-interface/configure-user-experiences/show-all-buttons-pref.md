# Show all buttons without the need to hover preference

Learn more about the Show all buttons without the need to hover accessibility preference.

\[Omitted image "coral-pref-no-hover.png"\] Alt text: Show all buttons without the need to hover accessibility preference.

The purpose of this preference is to improve discoverability and usability.
