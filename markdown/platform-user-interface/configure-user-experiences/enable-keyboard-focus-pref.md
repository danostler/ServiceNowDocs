# Enable keyboard focus on text that displays tooltip

Learn more about the Enable keyboard focus on text that displays tooltip accessibility preference.

\[Omitted image "coral-pref-truncated-text.png"\] Alt text: Enable keyboard focus on text that displays a tooltip accessibility preference.

The purpose of this preference is to provide access to truncated text.

Learn more about this preference from the following tutorial:

Enable keyboard focus on text that displays tooltip 

<table id="table_jbq_dgy_hgc"><thead><tr><th>

Detail

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Feature

</td><td>

Makes hidden text accessible when navigating with a keyboard. Ensures that full context is available without relying on mouse device hover.

</td></tr><tr><td>

Who benefits

</td><td>

-   Keyboard-only users
-   Screen reader users
-   Anyone preferring to tab quickly through interfaces

</td></tr><tr><td>

Visual representation

</td><td>

**Activated:** Truncated text is accessible via keyboard focus, revealing full text.

 **Deactivated:** Truncated text requires mouse device hover to view full text.
