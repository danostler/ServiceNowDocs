# Create a Next Experience Menu Application

Create and configure application menus and modules directly within the All menu.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **+**.

    \[Omitted image "neame-create-menu.png"\] Alt text: Arrow pointing the create menu icon in the all menu.

2.  On the form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Title|Enter a Title of your Next Experience Application Menu.|
    |Roles|Select which roles can access the application menu. If no roles are set, all users can see the menu when it's active.|
    |Description|Provide a description of the application menus purpose.|

3.  Select **Create**.

4.  Select the modules that you would like to add to the application menu by pointing a module and selecting **Add**.

    **Note:** You can add multiple of the same module to an application menu.

5.  Select **Done**.

    The Next Experience All Menu Editor opens.
