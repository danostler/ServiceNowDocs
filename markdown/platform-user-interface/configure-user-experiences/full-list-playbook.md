# Open full lists within playbook

Open a full list within playbook cards to view and update list items.

To open a full list within playbook, click the **Open List** icon \(\[Omitted image "playbook-open-list.png"\] Alt text: Open list icon\).

\[Omitted image "playbook-open-list-card.png"\] Alt text: Playbook card displaying a Open List button

A new tab opens to show a full list.
