# Add favorites in Next Experience

Find and go to your most used pages quickly by adding them to the Favorites menu in the Next Experience Unified Navigation.

## Before you begin

Role required: none

## Procedure

1.  Add a favorite in one of the ways listed in the following table.

<table id="choicetable_dg5_wsn_vvb"><tbody><tr><td id="d74217e75">

**From any page**

</td><td>

From any page, select the star icon in the contextual app pill.

 \[Omitted image "pol-nav-pill-favorite.png"\] Alt text: Favorite star in the contextual app pill

</td></tr><tr><td id="d74217e94">

**From the All menu**

</td><td>

-   To add a single module: In the application navigator, select the star icon next to the module name.\[Omitted image "next-exp-save-favorites.png"\] Alt text: All menu single module saved as favorite.
-   To add all the modules under an application: In the application navigator, select the star icon next to the application name.


</td></tr><tr><td id="d74217e117">

**From a list or record in the classic environment**

</td><td>

1.  Open a list or record in the classic environment.
2.  Select the context menu icon \[Omitted image "MenuIconUI14.png"\] Alt text: next to the list or record title.
3.  Select **Create Favorite**.
 You can add a different favorite for each view of a list or record.

</td></tr><tr><td id="d74217e149">

**From a knowledge base article**

</td><td>

1.  Navigate to **All** &gt; **Self-Service** &gt; **Knowledge**.
2.  From the Knowledge Homepage, select a knowledge article.
3.  In the header, select the star icon \(If the knowledge article has multiple versions, only the currently viewed knowledge article version is added to favorites\).


</td></tr></tbody>
</table>2.  In the **Favorite added** dialog box, modify the favorite.

    A sample initial dialog box is shown in the following example.

    \[Omitted image "quick-add-favorite.png"\] Alt text: Favorite added dialog box

    1.  In the **Name** field, edit the name that appears in the Favorites menu.

    2.  In the **Location** field, if you have created groups for the Favorites menu, select a group for the favorite to be listed in.

        For information about creating groups to organize the Favorites list, see [Organize favorites in Next Experience](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/configure-user-experiences/organize-favorites-next-experience.md).

    3.  Either save your changes, make further modifications, or remove the favorite.

        -   To make no further modifications, select **Done**.
        -   To modify the favorite's icon and its color, select **More**, select an icon and its color, and select **Save edits**.
        -   To remove the favorite, select **Remove**.
