# CSS theme support

Themes give the user interface a specific look and feel by using different color combinations.

System administrators can create themes or customize the existing themes for an instance. To see existing themes, navigate to **System UI** &gt; **Themes**.

-   **[Activating CSS theme support](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/configure-user-experiences/r_ActivatingCSSThemeSupport.md)**  
There are different plugins that activate CSS themes support.
-   **[Create or customize a theme](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/configure-user-experiences/t_CreateOrCustomizeATheme.md)**  
You can create themes or customize existing themes.
-   **[Default CSS styles](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/configure-user-experiences/r_DefaultCSSStyle.md)**  
You can use CSS styles to control the appearance of elements in a theme.
