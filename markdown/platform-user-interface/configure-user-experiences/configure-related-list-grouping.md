# Group a related list by a field

Configure a related list to group by a field.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Now Experience Framework** &gt; **System UI** &gt; **Related Lists**.

2.  From the Related Lists list, select a related list.

    A Related List record opens.

3.  From the Related List Entries related list, select an entry.

    A Related List Entry record opens.

4.  In the Group By Field field, select a field.

5.  Select **Update**.
