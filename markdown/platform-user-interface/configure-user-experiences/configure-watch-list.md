# Configure a watch list

Configure a watch list where you can add yourself, remove yourself, add multiple users, and add multiple records.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Now Experience Framework** &gt; **Declarative Actions** &gt; **Field Decorators**.

2.  Configure the action to add yourself to a watch list.

    1.  From the Action Assignments list, select **Add current user**.

        An Action Assignment record opens.

    2.  From the Action Configurations related list, select **Edit**.

3.  Configure the action to remove yourself from a watch list.

    1.  From the Action Assignments list, select **Remove current user**.

        An Action Assignment record opens.

    2.  From the Action Configurations related list, select **Edit**.

    3.  Move your workspace's action configuration record from the Collection list to the Action Configurations list.

    4.  Select **Save**.

4.  Configure the action to add multiple people to a watch list.

    1.  From the Action Assignments list, select **Add multiple users**.

        An Action Assignment record opens.

    2.  From the Action Configurations related list, select **Edit**.

    3.  Move your workspace's action configuration record from the Collection list to the Action Configurations list.

    4.  Select **Save**.

5.  Configure the action to add multiple records to a watch list.

    1.  From the Action Assignments list, select **Add multiple records**.

        An Action Assignment record opens.

    2.  From the Action Configurations related list, select **Edit**.

    3.  Move your workspace's action configuration record from the Collection list to the Action Configurations list.

    4.  Select **Save**.
