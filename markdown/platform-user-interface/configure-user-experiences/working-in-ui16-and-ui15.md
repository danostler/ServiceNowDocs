# Working in Core UI

Learn about the features of the Core UI that predated Next Experience.

-   **[Core UI](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-user-interface/configure-user-experiences/c_UI16.md)**  
The Core UI predates the Next Experience UI.
-   **[Working in the classic environment](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-user-interface/configure-user-experiences/working-in-classic-lists-and-forms.md)**  
In the classic environment, use lists and forms to get your work done.
-   **[Common UI elements](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-user-interface/configure-user-experiences/p_CommonUIElements.md)**  
This page lists common UI elements for the standard user interface.
-   **[Browser support](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-user-interface/configure-user-experiences/browser-support.md)**  
Browser support varies for each version of the user interface \(UI\). Some features have additional browser requirements, which are noted in the appropriate documentation.
-   **[Core UI keyboard shortcuts](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-user-interface/configure-user-experiences/r_KeyboardShortcuts.md)**  
You can use keyboard shortcuts to quickly perform common actions in the user interface.
-   **[User interface configuration](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-user-interface/configure-user-experiences/p_NavigationAndUIConfiguration.md)**  
Change appearance, navigation menus, CSS, and utilize cutting edge interface tools.
