# Use tags in the Activity stream

Use tags to filter through the Activity stream.

## Before you begin

Role required: workspace\_user

## About this task

Use tags to filter activities in the Activity stream by dynamic and static states that are not offered in filter lists.

**Important:** Tags are enabled and created by your administrator. For instructions, see [Configure tags for the Activity stream](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/configure-user-experiences/tags-activity-stream-admin.md).

## Procedure

1.  Open a record in your instance.

2.  From the Activity stream, select the filter icon \(\[Omitted image "filter-icon-new.png"\] Alt text: Filter icon\).

3.  Select the **Tags** filter, and select tags related to activities you want to view.


## Result

Activities with the tags you selected are displayed in the Activity stream.
