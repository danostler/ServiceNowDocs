# Tile icon illustrations in Theme Builder

Tile icon illustrations are used to communicate a tile’s purpose using visual metaphors.

## Default modal illustrations

A tile icon illustration is a small, purpose-driven graphic that visually represents the content or function of a tile, making navigation intuitive, reducing reliance on text. Tile icon illustrations are theme-able, and adapt to the theme colors of your instance.

**Important:** Tile icon illustrations apply to Workspaces and are not supported in the Core UI.

\[Omitted image "tb-tile-icon-illustration.png"\] Alt text: Accessibility tile icon illustration.

## Custom images

Upload your custom images to Theme Builder to replace the default tile icon illustrations.

To override the default tile icon illustrations with custom images, upload each image individually. The size and format limitations for custom images are as follows:

|Field|Description|
|-----|-----------|
|Dimensions limit|w48 × h48 px|
|Format|SVG|
|File size limit|2 MB|
