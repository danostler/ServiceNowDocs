# Dark mode in Theme Builder

Learn how to switch to dark mode in Theme Builder.

Theme Builder includes a dark mode option, offering an eye-friendly alternative to the default light mode. You can enable dark mode by selecting the Choose theme icon \[Omitted image "tb-choose-theme-icon.png"\] Alt text: in the application banner.

**Note:** Dark mode applies to the Theme Builder interface only and does not affect the appearance of individual themes.
