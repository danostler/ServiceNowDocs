# Guided tours in Theme Builder

Learn about Theme Builder guided tours, including how to access and take them to build your knowledge of Theme Builder.

## What guided tours are

Guided tours are part of the default ServiceNow AI Platform. There are tours available to take and you can use the Guided Tours application to create custom tours yourself in your own instances.

Tours contain interactive steps to help train and teach users. Some tours show various features in the user interface, such as an overview of a homepage. Other tours help complete a task, such as creating an experience or previewing an experience page. For more information about what guided tours are and how to create them, see [Exploring Guided Tours](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/adoption-services/exploring-guided-tours.md).

## How to access and use Theme Builder guided tours

Guided tours about Theme Builder may launch automatically from the Editor page. You can also start them manually from the Editor page.

When a guided tour starts automatically, an introduction is displayed. Select **Get started**.

\[Omitted image "tb-guided-tour-get-started.png"\] Alt text: Theme Builder guided tour modal.

To begin a tour manually, navigate to the Editor page and select the Show guided tours icon \[Omitted image "tb-guided-tour-lightbulb-icon.png"\] Alt text: in the banner. From there, select which guided tour you prefer.

\[Omitted image "tb-guided-tour-selection-modal.png"\] Alt text: Theme Builder guided tour selection modal.

After a tour starts automatically or manually, you're led through a series of steps. Read the text for each step and perform the required actions or select **Next** to continue the tour.

To stop a tour at any time, select **Skip tour** on a tour step. At the end of the guided tour, select **Done**.
