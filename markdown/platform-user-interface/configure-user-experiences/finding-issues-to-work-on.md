# Finding issues to work on in your Workspace

Use workspace to find the issues to work on.

\[Omitted image "agent-workflow-whole-1.png"\] Alt text: Agent workflow

The first step in your workflow is finding issues to solve. You can find issues to work on in the following ways:

-   Landing page—Provides a high-level overview of tasks that need to get done.

    \[Omitted image "landing-page-KB-article.png"\] Alt text: Landing page

    Clicking a link takes you either to list view or a specific record. You get to the landing page by clicking the home icon \(\[Omitted image "home-icon-smaller.png"\] Alt text: Home icon\). For more information, see [Use the landing page to find issues to work on](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-user-interface/configure-user-experiences/use-landing-page.md).

-   List view—Displays lists of records to work on. Records are grouped by the table they're in, for example, Task, SLA, Incident.

    Clicking a link takes you to the record so you can work on it. You get to list view by clicking the list icon \(\[Omitted image "IconList.png"\] Alt text: list icon\). For more information about list view, see [Use lists to find work to do](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-user-interface/configure-user-experiences/list-menu.md).

-   Chat inbox—Interact in real time with requesters using chat.

    Accepting a chat request automatically creates a record for you to record the work you do to solve the issue. You get to chats by clicking the agent inbox icon \(\[Omitted image "ChatIcon.png"\] Alt text: Inbox icon.\). For more information about chatting with requesters, see [Getting work from chats](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-user-interface/configure-user-experiences/live-agent-overview.md).

-   Phone inbox—Interact in real time with requesters by phone.

    Accepting a phone call automatically creates a record for you to record the work you do to solve the issue. You answer and make phone calls by clicking the phone icon \(\[Omitted image "IconPhone.png"\] Alt text: Phone icon\). For more information about using phones in workspace, see [Get work by phone in workspace](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-user-interface/configure-user-experiences/answer-call-agent-workspace.md).

-   Share a record—See a record another agent is working on so you can help.
