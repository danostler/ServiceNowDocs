# Card illustrations in Theme Builder

Card illustrations are used to improve the clarity or scannability of a feature.

## Default card illustrations

Card illustrations are visual elements that highlight key information, enhancing scannability and facilitating action. Card illustrations are theme-able, and adapt to the theme colors of your instance.

**Important:** Card illustrations apply to Workspaces and are not supported in the Core UI.

\[Omitted image "tb-card-illustration.png"\] Alt text: Visualize data card illustration.

## Custom images

Upload your custom images to Theme Builder to replace the default card illustrations.

To override the default card illustrations with custom images, upload each image individually. The size and format limitations for custom images are as follows:

|Field|Description|
|-----|-----------|
|Dimensions limit|w453 × h453 px|
|Format|SVG|
|File size limit|2 MB|
