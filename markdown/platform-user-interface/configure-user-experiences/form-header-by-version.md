# Form headers for UI versions

Core UI and UI15 each have a different form header that offers different navigation icons.

Learn about form header navigation by selecting the form header topic that corresponds to the UI version your organization uses.

**Parent Topic:**[Forms in the classic environment](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-user-interface/configure-user-experiences/c_UsingForms.md)

## Core UI form header

The form header displays the title of the form \(table\) and provides several controls.

The Core UI form header includes the following controls.

\[Omitted image "FormHeaderUI16.png"\] Alt text: Form header Core UI

|Control|Icon|Description|
|-------|----|-----------|
|Back|\[Omitted image "FormBackUI15.png"\] Alt text: back icon|Navigates to the previously viewed page without saving changes.|
|Form context menu|\[Omitted image "MenuIconUI14.png"\] Alt text: form context menu icon|Appears when a user clicks the menu icon beside the form title or right-clicks the form header. This menu is also called a right-click menu.|
|Attachments|\[Omitted image "FormAttachmentsUI15.png"\] Alt text: attachments icon|Allows users to view and add attachments to the record. For more information, see [Add and manage attachments](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-user-interface/configure-user-experiences/t_AddingAnAttachment.md).|
|Show activity stream|\[Omitted image "FormLiveFeedUI15.png"\] Alt text: show activity stream icon|Moves focus to the journal entry section of the form. If the journal fields are not on the form, the activity stream opens in a fly-out window.|
|Personalize form|\[Omitted image "Personalize\_Form\_Icon\_Helsinki.png"\] Alt text: personalize form icon|Opens the form personalization menu. For more information, see [Personalize a form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-user-interface/configure-user-experiences/t_PersonalizeAForm.md).|
|More options\[Omitted image "IconUI16MoreOptions.png"\] Alt text: More options icon|
|Email|\[Omitted image "FormEmailUI15.png"\] Alt text: email icon|Opens an email window.|
|Template bar|\[Omitted image "IconToggleTemplateBar.png"\] Alt text: edit tags icon|Opens a bar at the bottom of the form and lists available templates.|
|Annotations|\[Omitted image "FormToggleAnnotUI15.png"\] Alt text: annotations icon|Displays on-form annotations. For more information, see Administering form annotations.|
|Tags|\[Omitted image "FormEditTagsUI15.png"\] Alt text: edit tags icon|Displays the option to create custom tags and categorize documents. For more information, see [Tags](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-user-interface/configure-user-experiences/c_Tags.md).|
|Remaining controls|
|Submit or Update| |Saves changes and returns to the previously viewed page.|
|Related actions| |Provides standard actions in the form header for some tables, such as Close Incident or Resolve Incident in incident.|
|Delete| |Deletes the record from the list.|
|Previous and Next|\[Omitted image "FormPreviousUI15.png"\] Alt text: previous icon \[Omitted image "FormNextUI15.png"\] Alt text: next icon|Opens the previous and next record on the list from which the record was accessed.|
