# Inline edit one or multiple cells in a list

Inline edit one or multiple cells from the same column of a list to simultaneously change the value of the selected cells.

Inline editing enables you to change a field in a list instead of opening the record and changing it on the form. Edited cells must be in the same column when editing multiple cells.

-   Edited cells must be in the same column when editing multiple cells.
-   Edit the following fields inline:

    -   Boolean
    -   Choice
    -   Date
    -   Date/time
    -   Decimal
    -   Duration
    -   Float
    -   Integer
    -   Phone
    -   Phone e164
    -   Reference
    -   String
    -   Table name
    -   Tags
    -   URL

## Inline edit using a mouse

Inline edit one or multiple cells from the same column using a mouse.

### Before you begin

Role required: none

### Procedure

1.  Edit a single cell.

    1.  Double click a cell.

    2.  Enter the value you want and press **Enter**.

        To cancel the inline edit, press **esc** or **Cancel** on the keyboard, or click outside the cell.

2.  Select consecutive cells.

    1.  Place focus on a cell.

    2.  Hold **shift** on the keyboard and select the final cell you want to change.

    3.  Double click the cell.

    4.  Enter the value you want.

        To cancel the inline edit, press **esc** or **Cancel** on the keyboard, or click outside the cell.

    5.  Press **Enter**.

    \[Omitted image "list-inline-editing-consecutive.png"\] Alt text: Inline editing consecutive.

3.  Select non-consecutive cells in a column

    1.  Place focus on a cell.

    2.  Hold down the **command** key \(or **windows** key\) and select the cells you want to edit.

        The cells must be in the same column.

    3.  Double click the cell.

    4.  Enter the value you want.

    5.  Press **enter**.


### Result

The selected cells display the new field value.

\[Omitted image "list-inline-editing-consecutive-result.png"\] Alt text: Inline editing consecutive result.

## Inline edit using a keyboard

Inline edit one or multiple cells from the same column using a keyboard.

### Before you begin

Role required: none

### Procedure

1.  Edit a single cell.

    1.  With focus on a cell, press **shift** and **Enter** on the keyboard.

    2.  Enter the value you want.

        To cancel the inline edit, press **esc** or **Cancel** on the keyboard, or click outside the cell.

    3.  Press **enter**.

2.  Select consecutive cells.

    1.  With focus on a cell, hold **shift** on the keyboard.

    2.  Use the arrow keys to select consecutive cells in a column.

    3.  Press **shift** and **Enter** on the keyboard.

    4.  Enter the value you want.

        To cancel the inline edit, press **esc** or **Cancel** on the keyboard, or click outside the cell.

    5.  Press **Enter**.

    \[Omitted image "list-inline-editing-consecutive.png"\] Alt text: Inline editing consecutive.


### Result

The selected cells display the new field value.

\[Omitted image "list-inline-editing-consecutive-result.png"\] Alt text: Inline editing consecutive result.
