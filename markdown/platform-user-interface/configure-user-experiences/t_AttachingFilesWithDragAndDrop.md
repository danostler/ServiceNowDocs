# Attach files with drag-and-drop

You can drag files from your local computer into your browser window to attach them to the current record.

## Before you begin

Role required: admin

## About this task

This functionality is supported in Firefox 3.6 or later and Chrome. Support will be added for other web browsers as they implement the HTML5 specification.

## Procedure

1.  Navigate to the record.

2.  On your computer, browse to the files to attach.

3.  Select the files in the file browser, and then drag them over the form.

4.  Release the mouse button to begin the upload.

    When the upload is complete, the file name is added to the attachments header. Upload speed depends on the file size and the speed of your network connection.

    **Warning:** Do not navigate away from the record while an upload is in progress. The upload must be completed for the file to be attached.
