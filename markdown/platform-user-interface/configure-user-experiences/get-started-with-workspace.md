# Learn how to use your Workspace

Use the tutorials to walk through many parts of workspace that you will use on a regular basis.

The following tutorials are included for you to get a sense of how workspace works:

-   [Work on issues that are already in the database.](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-user-interface/configure-user-experiences/workspace-tutorial-for-agents.md)
-   [Work on issues that come in through phone calls or chats.](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-user-interface/configure-user-experiences/workspace-tutorial-agents-chat.md)

    **Note:** UI may vary between classic Workspace and UIB Workspace, but the functionality of the features is the same.


Once you complete both tutorials, you should have a feeling for what workspace does. The topics after the tutorials provide detailed instructions about how to use all of the functionality in workspace.
