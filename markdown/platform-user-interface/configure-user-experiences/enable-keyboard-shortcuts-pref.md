# Enable keyboard shortcuts preference

Learn more about the Enable keyboard shortcuts accessibility preference.

\[Omitted image "coral-pref-shortcuts.png"\] Alt text: Enable keyboard shortcuts accessibility preference.

The purpose of this preference is to improve efficiency and productivity.

<table id="table_hyl_q2y_hgc"><thead><tr><th>

Detail

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Feature

</td><td>

Navigate quickly on a page without relying on a mouse device or touchpad. Keyboard shortcuts speed up workflow, reduce strain, and make navigation more precise.

</td></tr><tr><td>

Who benefits

</td><td>

-   Users with mobility impairments
-   Users who want faster navigation
-   Anyone working in high-efficiency environments

</td></tr></tbody>

Deactivated
