# Reduce motion preference

Learn more about the Reduce motion accessibility preference.

\[Omitted image "coral-pref-reduce-motion.png"\] Alt text: Reduce motion accessibility preference.

The purpose of this preference is to minimize distractions and improve usability.

<table id="table_ybb_4fy_hgc"><thead><tr><th>

Detail

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Feature

</td><td>

Reduces the speed of animations when switching between screens, mainly affecting login animations. Reducing motion helps people who experience discomfort or nausea from animated effects.

</td></tr><tr><td>

Who benefits

</td><td>

-   Users with vestibular disorders or motion sensitivity
-   Users who experience migraines triggered by animations
-   Anyone preferring faster interaction

</td></tr><tr><td>

Visual representation

</td><td>

**Activated:** Animations are slower or turned off.

 **Deactivated:** Animations occur at their normal speed.

</td></tr></tbody>
</table>
