# Working with Component styles

Component styles enable you to fine-tune the appearance of individual UI elements such as buttons, form controls, and navigation.

The Component styles panel provides control over how specific elements like buttons, inputs, and navigation items look and behave across different states such as hover, focus, and disabled. This panel enables tailoring the look of specific UI elements without affecting the broader theme.

A search and filter bar at the top of the panel lets you narrow down components by name or category.

Each component is represented as a tile and grouped by function such as Navigation, Messaging, Forms. Selecting a component tile opens a side panel with editable properties for that component and its subcomponents.
