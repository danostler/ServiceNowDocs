# Edit your Workspace list columns and save the list

Edit which columns appear in your list and how they're ordered, and save the list to My Lists.

## Before you begin

Role required: agent

**Note:** Column personalization must be enabled for agents.

## Procedure

1.  In a Workspace list, select the List Actions Icon \(\[Omitted image "list-actions-icon.png"\] Alt text: List actions icon.\).

2.  Select **Edit columns** from the menu.

3.  In the Edit List modal, move the columns over that you want to display from Available columns to Selected Columns.

    The selected columns display from left to right with the first column displaying on the far left.

4.  Select the List Actions Icon \(\[Omitted image "list-actions-icon.png"\] Alt text: List actions icon.\).

5.  Select **Save**.

    Your list configuration is now accessible in the **My Lists** tab. You can rename the list or delete the list from the options in the List Actions menu.

    \[Omitted image "lists-mylists.png"\] Alt text: My Lists.
