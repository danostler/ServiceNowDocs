# Create a theme variant

Create a theme variant from the Coral theme. A theme variant uses customizable styling CSS variables and branding options to tailor the visual experience for your users.

## Before you begin

Role required: admin

## About this task

Create a theme variant 

## Procedure

1.  Navigate to **Service Portal** &gt; **Themes**.

2.  In the themes list, select **Coral** theme.

3.  On the **Theme Variant Mappings** tab, select **New**.

    \[Omitted image "theme-variant-mappings-tab.png"\] Alt text: Theme Variant Mappings tab.

4.  In the **Name** field, enter a name for the theme variant, such as Coral Blue.

    \[Omitted image "theme-variant-mapping-fields.png"\] Alt text: Theme Variant Mappings fields.

5.  In the **Theme variant** field, choose a theme on which to base the new theme, such as **Coral Dark**.

6.  In the **Theme variant** field, select the information icon \[Omitted image "icon-information.png"\] Alt text: Information icon to change the CSS variables, description, logo, and logo alt text.

7.  In the **Matching Next Experience Theme Variant** field, match the new theme variant to an existing Next Experience theme, such as Dark.

    **Note:** This option is available if you have Next Experience themes.

8.  In the Virtual Agent branding field, select the associated branding theme that displays when the theme variant is selected in Service Portal.

    **Note:** This option is available if you have Virtual Agent branding.

9.  Select **Submit**.
