# Associate a portal with a knowledge base

You can associate a portal with any knowledge base. Multiple knowledge bases can be associated with a portal.

## Before you begin

Role required: admin or sp\_admin

## Procedure

1.  Navigate to **All** &gt; **Service Portal** &gt; **Portals**.

2.  In the **Knowledge Bases** related list, select **Edit**.

    **Tip:** If you don't see the **Edit** option, confirm that you have the correct application scope selected in the application picker.

3.  Select one or more knowledge bases to add to the portal.

    The knowledge bases are displayed in Service Portal page listed in the **KB Home page** for the portal. If no knowledge bases are selected, all knowledge bases are available in the portal.

4.  Select **Update**.
