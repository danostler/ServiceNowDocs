# Use condition builder in reference lists

Use condition builder in reference lists to filter results.

## Before you begin

Role required: workspace\_user

## Procedure

1.  Select **Search for Record** icon \[Omitted image "IconSearch.png"\] Alt text: icon search. in the record field.

2.  Select \[Omitted image "condition-builder-filter.png"\] Alt text: condition builder filter icon.

3.  Add the conditions you would like to filter your record search by.\[Omitted image "condition-builder-filter-display.png"\] Alt text: Condition builder filter fields.

4.  To add additional conditions, select **+ New condition set**.

5.  Select **Run**.
