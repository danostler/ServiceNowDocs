# Customize Next Experience keyboard shortcuts

Customize Next Experience keyboard shortcuts to streamline your workflow and boost productivity. Keyboard shortcuts can also improve accessibility by reducing reliance on cursor navigation.

## Before you begin

Role required: none

## About this task

Customizations apply to your user profile and persist across sessions.

Learn more about customizing keyboard shortcuts from the following tutorial:

Customize the Next Experience keyboard shortcuts 

## Procedure

1.  Navigate to **User Menu** &gt; **Keyboard shortcuts**.

    The keyboard shortcuts modal opens.

    \[Omitted image "pol-keyboard-shortcut-modal.png"\] Alt text: Keyboard shortcuts modal.

2.  Select **Remap keyboard shortcuts**.

3.  Find a keyboard shortcut to customize by scrolling or entering text.

    -   Scroll vertically through the list of available keyboard shortcuts.
    -   Enter text into the **Search** field to search through the entire list of available keyboard shortcuts.
    **Note:** The pencil icon \[Omitted image "pencil-icon.png"\] Alt text: appears for those keyboard shortcuts that are customizable. System-defined shortcuts can't be customized.

4.  Select the pencil icon \[Omitted image "pencil-icon.png"\] Alt text: to edit the keyboard shortcut.

5.  Enter the new key combination, then select the check mark to save your change.

    **Note:** You don't need to delete the existing shortcut. Entering a new key combination replaces it. Keyboard shortcuts must include at least one modifier key, for example, Ctrl, Command, or Alt.

    \[Omitted image "pol-remap-keyboard-shortcuts-checkbox.png"\] Alt text: Accept shortcut changes check mark.

    **Note:** If the key combination is already in use, the system displays a notification.

    \[Omitted image "pol-keyboard-shortcut-exists.png"\] Alt text: Keyboard shortcut exists message.

6.  Select **Close** to exit the modal.

7.  To restore all remapped keyboard shortcuts to their original values, select **Restore Defaults**.

    A confirmation dialog opens.


## Result

The keyboard shortcut is saved and ready to use.
