# Add a page to an application

Homepages and content pages are not added automatically to update sets and applications. They must be manually added.

## Before you begin

Role required: content\_admin or admin

## Procedure

1.  Navigate to **All** &gt; **Content Management** &gt; **Pages**.

2.  Right-click a content page record.

3.  Select **Unload Portal Page**.

    The page is added to the current application and to the current update set.
