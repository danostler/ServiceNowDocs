# Assign a page to a site

If you have created multiple sites, you can add pages from one site to another site.

## Before you begin

Role required: content\_admin or admin

## Procedure

1.  Navigate to **All** &gt; **Content Management** &gt; **Sites**.

2.  Select a site.

3.  In the **Pages** related list, click **Edit**.

4.  Select other pages to include in the site.

5.  Click **Save**.

6.  **Update** the site.
