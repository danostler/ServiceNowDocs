# Menu types

By changing the **Type** field on the navigation menu block, you can format the same menu in different ways.

|Type|Image|Description|
|----|-----|-----------|
|Super Menu for Headers|\[Omitted image "CMSMenuSuper.png"\] Alt text: CMS super menu|The super menu is a hybrid between the drop-down menu and the tabbed system. The user can use a super menu to create a menu from any number of menus.|
|Drop-Down Menu for Headers|\[Omitted image "CMSMenuDropdown.png"\] Alt text: CMS menu dropdown|The drop-down menu renders the menu sections as drop-down list. Use the mouse to point to the menu name and view the menu items.|
|Tab Menu for Headers|\[Omitted image "CMSMenuHeadertabs.png"\] Alt text: CMS menu header tabs|The tab menu renders the menu sections as tabs. Use the mouse to click the tab and view the menu items.|
|Horizontal Blocks|\[Omitted image "CMSMenuHorizontalblocks.png"\] Alt text: CMS menu horizontal blocks|The horizontal blocks menu renders the menu sections as block headings with menu items as links within the blocks. The blocks are arranged horizontally.|
|Tab Content Block|\[Omitted image "CMSMenuHorizontaltabs.png"\] Alt text: CMS menu horizontal tabs|The tab content block \(horizontal\) menu renders the menu sections as tabs with menu items as links within the block.|
|Vertical List|\[Omitted image "CMSMenuVerticallist.png"\] Alt text: CMS menu vertical list|The vertical list menu renders the menu sections as headings with menu items as links below them.|
|Vertical Blocks|\[Omitted image "CMSMenuVerticalblocks.png"\] Alt text: CMS menu vertical blocks|The vertical blocks menu renders the menu sections as block headings with menu items as links within the block. The blocks are arranged vertically.|
