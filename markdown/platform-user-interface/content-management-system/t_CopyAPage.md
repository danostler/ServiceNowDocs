# Copy a page

Copying pages is an efficient way to avoid duplicating the same work and to create pages quickly from a guiding base template.

## Before you begin

Role required: content\_admin or admin

## Procedure

1.  Navigate to **All** &gt; **Content Management** &gt; **Sites** and select the site.

2.  Select the page to copy.

3.  Click **Copy**.

4.  Rename the page.

    **Note:** Do not use **service\_catalog**to rename a page. It is already a valid page in the system.

5.  Edit the page fields.

6.  Click **Update**.
