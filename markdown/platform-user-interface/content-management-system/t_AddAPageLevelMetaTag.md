# Configure a page level meta tag

A page level tag is a meta tag defined on a specific page and included on only that page.

## Before you begin

Role required: content\_admin or admin

## About this task

If an individual page has a specific tag with the same name as the site, the page tag takes precedence.

## Procedure

1.  Navigate to **All** &gt; **Content Management** &gt; **Pages**.

2.  Open the page.

3.  If not already included, add the **Meta Tags** related list to the form.

4.  In the **Meta Tags** related list, click **New**.

5.  Type a **Name** and **Content** for the tag.

6.  Click **Submit**.
