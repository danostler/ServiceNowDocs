# Configure a site level meta tag

A site level tag is a meta tag is defined on a site and included on every page within that site.

## Before you begin

Role required: content\_admin or admin

## About this task

If an individual page has a specific tag with the same name as the site tag, the page tag takes precedence.

## Procedure

1.  Navigate to **All** &gt; **Content Management** &gt; **Sites**.

2.  Open the site.

3.  If not already included, add the **Meta Tags** related list to the form.

4.  In the **Meta Tags** related list, click **New**.

5.  Enter a **Name** and **Content** for the tag.

6.  Click **Submit**.
