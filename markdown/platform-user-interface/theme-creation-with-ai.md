# Theme creation with AI

Use AI to generate themes from images directly in Theme Builder or from the Now Assist panel. This feature is available for instances running Zurich Patch 7 or later.

## Configuring Theme Builder AI theme creation

The requirements for AI theme creation depend on where you start the workflow.

In order to start using AI theme creation from within Theme Builder, you must install Now Assist for Creator. See [Install Now Assist for Creator](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/build-workflows/now-assist-for-creator/install-now-assist-for-creator.md). After you have installed Now Assist for Creator, see [Create a theme with AI in Theme Builder](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-user-interface/tb-create-a-theme-ai.md).

In order to start using Theme Builder AI theme creation workflow from the Now Assist panel, you must install Now Assist for Creator and turn on the Theme Creation Workflow in AI Agent Studio. For more information, see [Turn on theme creation workflow for use in the Now Assist panel](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-user-interface/turn-on-theme-creation-workflow.md). After you have enabled the Theme Creation Workflow, see [Create a theme using Now Assist](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-user-interface/create-theme-now-assist.md).

-   **[Turn on theme creation workflow for use in the Now Assist panel](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-user-interface/turn-on-theme-creation-workflow.md)**  
Turn on the theme creation workflow to generate themes using the Now Assist panel.
-   **[Create a theme using Now Assist](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-user-interface/create-theme-now-assist.md)**  
Use Now Assist to generate themes from your brand image or brand guidelines. This AI-powered workflow extracts color palettes automatically, creating visually consistent themes without requiring brand color codes.
-   **[Create a theme with AI in Theme Builder](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/platform-user-interface/tb-create-a-theme-ai.md)**  
Use an AI-powered theme creation from within Theme Builder to generate themes from brand images, preview, and refine results all within the updated Create a theme wizard.
