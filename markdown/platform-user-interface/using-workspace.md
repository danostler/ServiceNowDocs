# Starting in your Workspace

Use workspace to find and resolve internal or external requester issues.

The following video shows an example of how to resolve an incident using workspace.

Your general workflow using workspace is:

1.  Find an issue to work on.
2.  Work on the issue to find a solution.
3.  Communicate the resolution to the requester.
