# Attach a file to a record

Attach a file to a Configurable Workspace record. The Attachments panel provides a list of attached files.

## Before you begin

Role required: none

## Procedure

1.  Open a Configurable Workspace record in your instance.

2.  Open the Attachments panel from the sidebar by selecting the Attachment icon \(\[Omitted image "icon-attachments-panel.png"\]\).

3.  Select the Add file icon \(\[Omitted image "icon-attachments-add-file.png"\]\) and open a file from your local computer.

    You can also drag and drop the file into the Attachments panel.

4.  In the Upload a file modal, review the file details.

    You can edit the file name and add an encryption module to the attachment.

5.  Select **Upload**.
