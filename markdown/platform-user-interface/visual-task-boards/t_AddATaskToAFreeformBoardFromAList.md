# Add a task to a freeform Visual Task Board from a list

You can add a task card to a freeform board from any table that extends Task. These cards represent records on whichever table you add them from.

## Before you begin

Role required: none

## About this task

You can add cards from multiple task tables, such as Incident or Problem, to the same freeform board.

## Procedure

1.  Navigate to a list of task records, such as **All** &gt; **Incident** &gt; **Open**.

2.  Perform either of the following actions.

<table id="choicetable_vpk_rrn_jq"><tbody><tr><td id="d54622e90">

**Add one task**

</td><td>

Right-click a record and select **Add to Visual Task Board**.

</td></tr><tr><td id="d54622e102">

**Add multiple tasks**

</td><td>

1.  Select the check boxes beside the records you want to add.
2.  In the column context menu, select **Add to Visual Task Board**.


</td></tr></tbody>
</table>3.  Click the name of the board you want to add the card to.

    A confirmation message appears.

4.  To view the board, click the board name in the confirmation message.

    The task appears in the first lane by default.
