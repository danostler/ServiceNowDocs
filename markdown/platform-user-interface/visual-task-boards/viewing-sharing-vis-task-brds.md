# Viewing and sharing Visual Task Boards

View details about Visual Task Boards, view them as a list, and share them with others.

-   **[View Visual Task Boards details](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/visual-task-boards/t_ViewBoardInformation.md)**  
You can display the key information about any Visual Task Board.
-   **[View a Visual Task Board in a list](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/visual-task-boards/view-board-list.md)**  
You can view a Visual Task Board in the list view using the board information option.
-   **[Add a Visual Task Board as a favorite](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/visual-task-boards/favorite-task-boards.md)**  
Add Visual Task Boards you access frequently to the application navigator using the favorite option.
-   **[Share a Visual Task Board in a Connect conversation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/visual-task-boards/t_ShareVTB.md)**  
You can share a Visual Task Board in a Connect Chat or Connect Support conversation.
-   **[Create a Connect conversation from a Visual Task Board](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-user-interface/visual-task-boards/t_ConversationVTB.md)**  
Create a Connect Chat conversation from a Visual Task Board to collaborate with board members and keep track of activity as it happens.
