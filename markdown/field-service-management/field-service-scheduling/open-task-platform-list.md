# Open tasks from the task list

Quickly open tasks from the Work Order Task list and see them in Dispatcher Workspace to act on them without searching.

## Before you begin

Role required: admin, wm\_dispatcher

## About this task

You can open up to 10 work order tasks simultaneously in Dispatcher Workspace. Each task opens in a new tab, and your selection generates a unique URL that you can save and reuse. The tasks that you select generate a unique URL, so you can save the URL and open the same tasks you selected when you load that URL.

When you open multiple tasks, Dispatcher Workspace displays today's date, not the tasks' scheduled dates. When you open a single task, Dispatcher Workspace displays that task's scheduled date.

## Procedure

1.  Navigate to **All** &gt; **Field Service** &gt; **Work Order** &gt; **All Work Order Tasks**.

2.  Select the check box next to the work order tasks you want to open.

    **Note:** You can select up to 10.

3.  Select **Actions on selected rows** &gt; **Open in Dispatcher Workspace**.

    \[Omitted image "open-from-platform.png"\] Alt text: opening work order tasks from the list

    The tasks open in Dispatcher Workspace. You can save the URL to open the same work order tasks in Dispatcher Workspace later.
