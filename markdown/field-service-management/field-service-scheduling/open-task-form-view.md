# Open tasks from the form

Quickly open tasks from the form and see them in Dispatcher Workspace to act on them.

## Before you begin

Role required: admin, wm\_dispatcher

## About this task

You can open up to 10 work order tasks simultaneously in Dispatcher Workspace. Each task opens in a new tab, and Dispatcher Workspace numbers each tab to indicate how many tasks it contains. Your selection generates a unique URL that you can save and reuse. The tab in Dispatcher Workspace has a number next to it that corresponds with the number of work order tasks open in that tab.

When you open multiple tasks, Dispatcher Workspace displays today's date, not the tasks' scheduled dates. When you open a single task, Dispatcher Workspace displays that task's scheduled date.

## Procedure

1.  Navigate to **All** &gt; **CSM/ FSM Configurable Workspace**.

2.  Select the **List** \[Omitted image "coral-list-icon.png"\] Alt text: List icon.

3.  Select **Work Order Tasks** &gt; **My Work Order Tasks**.

4.  Select a Work Order Task to open its form.

5.  Select **Open in Dispatcher Workspace** from the **More Actions** menu.
