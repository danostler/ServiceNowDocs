# Add work notes or comments with the activity stream

Quickly add work notes or comments to a work order task using the contextual side panel. Save time and add information while staying in Dispatcher Workspace.

## Before you begin

Role required: wm\_dispatcher

## Procedure

1.  Navigate to **All** &gt; **Field Service** &gt; **Dispatching** &gt; **Dispatcher Workspace**.

2.  Select **Dispatcher Workspace**.

3.  Select the info icon on a work order task card.

4.  From the contextual side panel, select the **Activity** tab.

5.  Choose from the following options

    |Selection|Action|
    |---------|------|
    |**Work Notes**|View work notes that have been added to the task or add work note.|
    |**Comments**|View comments that have been added to the task or add comment.|

    **Note:** Select **Stacked view** to see the work notes and comments on top of each other instead of as separate tabs.

6.  Select **Summarize** to get a summary of the activity stream from Now Assist.

    **Note:** The task summarization skill must be enabled for the user to see this option.
