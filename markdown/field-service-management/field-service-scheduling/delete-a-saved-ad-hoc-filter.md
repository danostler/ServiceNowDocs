# Delete a saved ad hoc filter

Delete ad hoc filters saved in Dispatcher Workspace.

## Before you begin

Role required: wm\_dispatcher

## Procedure

1.  Navigate to **All** &gt; **Field Service** &gt; **Dispatching** &gt; **Dispatcher Workspace**.

2.  Select **List**.

3.  Under **Lists**, navigate to **Saved Filters** &gt; **Task Filter**.

4.  Select a filter.

5.  Select the **More Actions** \(\[Omitted image "more\_actions1.png"\] Alt text: More actions icon.\) icon.

6.  Select **Delete**.

7.  Select **OK**.


## Result

The filter is removed from the saved filters.
