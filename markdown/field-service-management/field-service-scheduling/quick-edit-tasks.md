# Quickly edit work order tasks

Quickly edit work order tasks directly from Dispatcher Workspace without opening the work order task form.

## Before you begin

Role required: wm\_dispatcher

## Procedure

1.  Navigate to **All** &gt; **Dispatcher Workspace**.

2.  Select **Dispatcher Workspace**.

3.  Open Quick Edit using one of the following methods

    |Location|Action|
    |--------|------|
    |**Task panel**|Select the **More actions** icon on a task card and select **Quick Edit**.|
    |**CSP**|Select the **Overflow actions** icon and select **Quick Edit**.|
    |**Calendar**|Select a task and select **Quick Edit**.|
    |**Map**|Select a work order task and select **Quick Edit**.|
    |**Map \(collocated pin\)**|Select a collocated map pin, select the **Overflow actions** icon, and select **Quick Edit**.|

4.  Edit the work order task from the CSP.

5.  Select **Save**.


## Result

The work order task is updated with the changes you made.
