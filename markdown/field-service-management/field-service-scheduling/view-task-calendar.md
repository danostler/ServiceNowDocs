# Viewing tasks or events on the calendar

Dispatchers can view any task on the calendar to navigate to the scheduled time and date of the task they chose.

When you select **View on calendar**, the calendar displays the agent assigned to the task in the **Added resources** section at the top. You can view up to 10 tasks simultaneously.

You can view events assigned to a single person only. **View on calendar** is unavailable for events with multiple attendees.

For repeating events, such as scheduled lunch breaks, the calendar displays the specific occurrence you selected. For example, if you select a lunch event scheduled for November 21 from a daily recurring series, the calendar shows the November 21 occurrence.

If you have the calendar sorted in a non-default way, when you select View on calendar the sort changes to default.

Dispatchers can view tasks on the calendar in any of the following states:

-   Assigned
-   Accepted
-   Scheduled
-   Work in Progress
-   Closed complete
