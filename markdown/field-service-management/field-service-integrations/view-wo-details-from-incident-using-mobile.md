# View work order details from incident record using the Mobile Agent application

View the work order details for the related incident record from your Mobile Agent application.

## Before you begin

Roles required: wm\_agent, sn\_change\_read

## Procedure

1.  Navigate to the Mobile Agent application.

2.  Tap **My Work**.

3.  In the **My Tasks** section, open the task.

4.  In the Details tab, tap **Initiated from**.

5.  Tap **Details** to view the incident record details.
