# Clear Dispatcher Workspace resource filters

If a dispatcher experiences loading issues in Dispatcher Workspace, you can clear the Dispatcher Workspace resource filters and that can help resolve issues with loading.

## Before you begin

Role required: admin.

## Procedure

1.  Navigate to **All** &gt; **User administration** &gt; **User preferences**.

2.  Search for dispatcher\_workspace\_resource\_filters.

3.  Select the check box next to dispatcher\_workspace\_resource\_filters.

4.  Select the **Actions on selected rows** drop-down list.

5.  Select **Delete**.
