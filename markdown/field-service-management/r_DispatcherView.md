# Map iconography in Dispatcher Workspace

Unique map pins and icons in the dispatch map that mark the location of tasks and agents.

|Icon|Description|
|----|-----------|
|\[Omitted image "agent-location.png"\] Alt text: Agent location marker.|An agent's location.|
|\[Omitted image "agent-static-state.png"\] Alt text: Agent static state.|An agent's last known location when their live location can't be determined. For example, if the agent has lost reception.|
|\[Omitted image "agent-at-home.png"\] Alt text: Agent home location.|The agent is at their home location.|
|\[Omitted image "multiple-agents.png"\] Alt text: Multiple agents marker.|Multiple agents are at this location.|
|\[Omitted image "assigned.png"\] Alt text: Assigned map marker.|The task is accepted, assigned, or scheduled.|
|\[Omitted image "assigned-multiple.png"\] Alt text: Assigned multiple.|Multiple tasks have been accepted, assigned, or scheduled at this location.|
|\[Omitted image "assigned-P1.png"\] Alt text: Assigned P1.|A priority-1 task has been accepted, assigned, or scheduled.|
|\[Omitted image "route-marker.png"\] Alt text: Colored marker with 1 on it.|The assigned work order task displayed while viewing agent's route. The number indicates the order in which the task will be completed.|
|\[Omitted image "needs-attention.png"\] Alt text: Needs attention marker.|The task is in the pending dispatch or draft state.|
|\[Omitted image "needs-attention-multi.png"\] Alt text: Needs attention multiple.|Multiple tasks are in the pending dispatch or draft state.|
|\[Omitted image "needs-attention-P1.png"\] Alt text: Needs attention P1.|A priority-1 task is in the pending dispatch or draft state.|
|\[Omitted image "in-progress.png"\] Alt text: In progress.|An agent is on the way to the task, work on the task is paused, or the task is in progress.|
|\[Omitted image "in-progress-multiple.png"\] Alt text: In progress multiple.|An agent is on the way to a location with multiple tasks, work on a task is paused, or the task is in progress.|
|\[Omitted image "in-progress-P1.png"\] Alt text: In progress P1.|An agent is on the way to a priority-1 task, work on the priority-1 task is paused, or the priority-1 task is in progress.|
|\[Omitted image "complete.png"\] Alt text: Complete.|A work order task is complete.|
|\[Omitted image "complete-multiple.png"\] Alt text: Complete multiple.|Multiple tasks are complete at the same location.|
|\[Omitted image "complete-P1.png"\] Alt text: Complete P1.|A priority-1 task is complete.|
|\[Omitted image "incomplete.png"\] Alt text: Incomplete.|A task was closed as incomplete or canceled.|
|\[Omitted image "incomplete-multiple.png"\] Alt text: Incomplete multiple.|Multiple tasks were closed as incomplete or canceled.|
|\[Omitted image "incomplete-P1.png"\] Alt text: Incomplete P1.|A priority-1 task was closed as incomplete or canceled.|
|\[Omitted image "home-location.png"\] Alt text: Home location tasks.|A task is assigned at the agent's home location.|
|\[Omitted image "home-location-multi.png"\] Alt text: Home location task.|Multiple tasks are assigned at the agent's home location.|
|\[Omitted image "colocated-multi.png"\] Alt text: Co-located multiple.|More than one task, agent, or crew is at this location.|
|\[Omitted image "colocted-P1.png"\] Alt text: Co-located P1.|More than one task, agent, or crew is at this location. At least one of the tasks at this location is a priority-1 task.|
|\[Omitted image "completed-route.png"\] Alt text: Completed route.|The lighter-colored line on the dispatch map shows the route that a Field Service agent has completed.|
|\[Omitted image "pending-route.png"\] Alt text: Pending route.|The darker-colored line on the dispatch map shows the route that a Field Service agent is planning to complete.|
