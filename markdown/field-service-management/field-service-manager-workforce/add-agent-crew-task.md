# Add a resource to a crew task

Add resources that aren't part of a crew, like agents or pieces of equipment, if you realize they're required after the task has been created.

## Before you begin

Role required: wm\_dispatcher, wm\_manager

## Procedure

1.  Navigate to **All** &gt; **Field Service** &gt; **All Work Order Tasks**.

2.  Select the work order crew task for which you want to add a resource to.

3.  In the **Task Assignees** tab, select **Add**.

4.  On the form, fill in the fields.

    |Field|Input|
    |-----|-----|
    |Requirement|Name the requirement.|
    |Resource type|Set to **Agent** or **Equipment**.|
    |Assignee|Search for and add the agent that you want to add to the crew task.|

5.  Select **Submit**.

    The resource is added to the crew.
