# Add an equipment skill

Designate skills as equipment-related skills so that when a task that requires equipment skills is assigned, the correct equipment skills can be selected.

## Before you begin

Role required: wm\_admin

## Procedure

1.  Navigate to **All** &gt; **Skills** &gt; **All Skills**.

2.  Select the skill that you want to designate as an equipment skill.

3.  Under **Contains Skills**, select **Edit**.

4.  Under **Collection**, search for and select Equipment Skills.

5.  Select the Add \(\[Omitted image "add-greater.png"\] Alt text: add\) icon.

6.  Select **Save**.
