# Assign an Agent Efficiency value to agents

Assign a Field Service Agent Efficiency criteria and efficiency value to agents to more accurately calculate work duration.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Agent Efficiency** &gt; **Agent Efficiency Values**.

2.  Select **New**.

3.  On the form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Agent|Name of the agent.|
    |Agent Efficiency Criteria|Agent Efficiency criterion.|
    |Efficiency|Agent Efficiency value ranging from 0-2.|
    |Active|Option to activate the Agent Efficiency criterion and its value.|

4.  Select **Submit**.

5.  View the record by navigating to **All** &gt; **Agent Efficiency** &gt; **Agent Efficiency Values**.


## Result

The Agent Efficiency value is assigned to the agent.
