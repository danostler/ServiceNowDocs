# Workforce system properties

Modify the Workforce configuration by setting system properties to enable visibility for both group and territory views.

</th><th>

Description

team\_calendar.enable\_for\_grpmember

</td><td>

-   Location: **All** &gt; **System Properties** &gt; **All Properties**

</td></tr><tr><td>
