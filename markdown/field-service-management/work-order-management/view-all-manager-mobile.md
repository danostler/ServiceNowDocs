# View all tasks in Field Service Manager Mobile

Manager’s can view all the tasks assigned to assignment groups or territories they manage to get a complete view of the team’s workload.

## Before you begin

Role required: wm\_manager, sn\_fsm\_tp.fsm\_territory\_resource\_manager

## Procedure

1.  Open Field Service Manager Mobile.

2.  Tap **Tasks**.

3.  Choose from the following
