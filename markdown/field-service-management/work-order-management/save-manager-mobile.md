# Save a task from Field Service Manager Mobile

Save a task to review it or check on it later.

## Before you begin

Role required: wm\_manager, sn\_fsm\_tp.fsm\_territory\_resource\_manager

## Procedure

1.  Open Field Service Manager Mobile.

2.  Open a task.

3.  Select the Save icon \[Omitted image "mobile-save.png"\] Alt text: save icon.

    **Note:** Select the Unsave icon \[Omitted image "mobile-unsave.png"\] Alt text: unsave icon to remove the task from the Saved tasks list.

4.  Change the name of the task you want to save if you want to save it as something other than the work order task number.

5.  Select **Done**.

    The task shows on the home page of Field Service Manager Mobile under **Saved tasks**.
