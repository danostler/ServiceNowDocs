# Field Service Manager Mobile

Field Service Manager Mobile is a centralized place for managers to do anything related to Field Service Management while on their mobile device.

Field Service Manager Mobile is available for any user who has the wm\_manager, or the territory\_manager role assigned to them. When a user with the wm\_manager, or the territory\_manager role downloads the Now Mobile Agent application and signs in, they see the Field Service Manager Mobile interface. Managers can only view and edit agent information and work order tasks for agents that are part of assignment groups or territories that they manage. The Field Service Manager Mobile plugin must be installed to use Field Service Manager Mobile.

## Using Now Assist on the Field Service Manager Mobile

If your organization has Now Assist for Field Service Management \(FSM\) installed you can access your virtual agent by tapping the **More actions** icon in the top right, then selecting Ask Now Assist. For more information on using Now Assist on your mobile device, see [Mobile Now Assist display experience for Virtual Agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/mobile/mobile-platform/mobile-na-display-experience-va.md). For more information on Now Assist for Field Service Management \(FSM\), see [Now Assist for Field Service Management \(FSM\)](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/now-assist-for-field-service-management-fsm/now-assist-fsm.md).

## Offline mode

Offline mode is available for Manager Mobile and functions the same way that it does on the Now Mobile Agent App. For more information, see [Working offline on ServiceNow Agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/work-offline-fieldservice-mobile.md).

Managers can perform the following actions on the manager mobile application.

-   [Manage Field Service agents from Field Service Manager Mobile](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/mange-agents-mobile.md)

    Enables managers to see work assigned to Field Service agents, agent schedules, and contact information.

-   [View the tasks assigned to an agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/view-tasks-for-agent.md)

    See what tasks are assigned to an agent for the rest of the day.

-   [View all tasks in Field Service Manager Mobile](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/view-all-manager-mobile.md)

    See all the tasks assigned to technicians you manage.

-   [Search in Field Service Manager Mobile](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/search-manager-mobile.md)

    Search for a technician or a task.

-   [Save a task from Field Service Manager Mobile](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/save-manager-mobile.md)

    Save a task so you can review it later.

-   [View agent information](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/view-agent-manager-app.md)

    Quickly see information related to any of the Field Service agents that you manage.

-   [View agents' agenda in Field Service Manager Mobile](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/agenda-manager-mobile.md)

    View the agenda that a technician has scheduled.

-   [View task SLA status](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/view-sla-manager.md)

    View the SLA status for the tasks assigned to technicians you manage.

-   [View analytics from Field Service Manager Mobile](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/view-analytics-manager.md)

    View analytics about the tasks assigned to technicians you manage.

-   [Create a work order task](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/create-task-manager-mobile.md)

    Create a task and assign it to a technician.

-   [Reassign a task to another agent](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/reassign-task-manager.md)

    Reassign a task if something happens and the original technician can't complete the task.

-   [Filter agents in Field Service Manager Mobile](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/filter-agents-manager.md)

    Filter the technicians that you manage.

-   [Filter an agent’s tasks in Field Service Manager Mobile](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/filter-task-manager.md)

    Filter the tasks that are assigned to technicians you manage.

-   [View your profile in Field Service Manager Mobile](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/view-profile-manager-app.md)

    View your manager profile.

-   [Show that an agent is busy with a non-work order event on Field Service Manager Mobile](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/event-manager-mobile.md)

    Block a technician's calendar for a personal event.

-   [Delete or edit a non-work order event on Field Service Manager Mobile](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/delete-edit-event-manager.md)

    Make changes to a personal event.
