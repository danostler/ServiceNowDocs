# Modify or delete time worked entries

Agents can modify and delete time worked entries before a time sheet is approved.

## Before you begin

Role required: wm\_agent

## About this task

Modifying or deleting entries updates existing time cards and generates an updated time sheet.

**Note:** Do not manually edit time cards. Instead, edit the original time worked entries, which generates updated time cards and time sheets.

## Procedure

1.  Navigate to **All** &gt; **Time Sheets** &gt; **Time Recording** &gt; **My Time Worked**.

2.  Select the time worked entry you want to modify or delete.

3.  Either modify or delete the entry.

    -   To modify the entry, make changes to the desired fields and select **Update**.
    -   To delete the entry, select **Delete**.
