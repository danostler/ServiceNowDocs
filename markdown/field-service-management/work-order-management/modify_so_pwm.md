# Reschedule the planned work order

Reschedule planned work orders by advancing or delaying the occurrence date of the schedule.

## Before you begin

Role required: sn\_fsm\_planned\_wm.planned\_work\_admin

## About this task

Avoid generating duplicate work orders for overlapping schedule occurrences by changing the start or end date, either advancing or delaying the maintenance work as needed.

**Note:** You can modify a schedule occurrence only if either no work orders are generated for it or the work orders generated for it are in the Draft state.

## Procedure

1.  Navigate to **All** &gt; **Field Service Management** &gt; **Planned Work Management** &gt; **Plans**.

2.  Open your work plan.

3.  In the Planned Work Schedules related list, select a schedule for which you want to review the schedule occurrences.

4.  In the Schedule Occurrences related list, select a schedule occurrence that you want to modify.

5.  Update the **Earliest start date** or **Requested due by** field to change the date and time of the existing schedule occurrence.

6.  Select **Update**.
