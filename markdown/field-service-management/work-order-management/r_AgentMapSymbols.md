# Agent map symbols

The agent map shows your location, the tasks assigned to you for the current day, tasks that have been accepted, and other tasks that have not yet been assigned.

Task icons are color coded and display automatic sequence numbers indicating the order in which the dispatcher has scheduled the tasks initially.

|Icon|Title|Description|
|----|-----|-----------|
|\[Omitted image "Person2.png"\] Alt text: Agent map marker.|Agent|Your current location on the map.|
|\[Omitted image "TaskBlue.png"\] Alt text: Blue map marker.|Assigned|Assigned tasks that have not yet been accepted.|
|\[Omitted image "TaskGreen.png"\] Alt text: Green map marker.|Accepted|Tasks that have been accepted or tasks that are in progress.|
|\[Omitted image "TaskRed.png"\] Alt text: Red map marker.|Unassigned|Unassigned tasks that are awaiting dispatch. These tasks are available for an agent to accept to fill gaps in a schedule.|
|\[Omitted image "MultiTask.png"\] Alt text: Red marker.|Multiple tasks|Multiple tasks clustered by proximity. the ServiceNow system gathers tasks together by proximity into a cluster as you increase the altitude of the map view. Counters in the cluster icon indicate the number of tasks rolled up as the perspective changes.|

\[Omitted image "WMAgentMap1.png"\] Alt text: Agent map showing location and status of tasks. Additionally, the Edit tab is open to display date, assigned, and unassigned filters.
