# Search in Field Service Manager Mobile

Use search to quickly find work order tasks, field service agents, assignment groups, or territories.

## Before you begin

Role required: wm\_manager, sn\_fsm\_tp.fsm\_territory\_resource\_manager

## Procedure

1.  Open Field Service Manager Mobile.

2.  Tap the search bar at the top of the screen.

3.  Search for work order tasks, field service agents, keyword, assignment groups, or territories.

    **Note:** Keywords return all work order tasks, agents, or assignment groups/ territories that contain the keyword. For example, if you search using the keyword `repair`, then all the tasks that contain `repair` are displayed.

4.  Once you start typing, tap the work order task, Field Service agent, keyword, assignment group, or territory your searching for shows on the screen.

5.  Filter your results by tapping one of the filter bubbles at the top of the page.
