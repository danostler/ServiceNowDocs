# Review time recorded for a task

Agents can review the time recorded for work order tasks.

## Before you begin

Role required: wm\_agent

## About this task

The **Time Worked** related list on the Work Order Task form displays the Time Worked records that have been created for the work order task. From this related list, field service agents can review their recorded time and also delete records.

## Procedure

1.  Navigate to the desired work order task.

2.  Go to the **Time Worked** related list.

    This list displays the Time Worked records that have been created for this work order task.

3.  To view a record, click the **Category**.
