# View sibling tasks related to a work order

View details of sibling work order task that are related to the same work order in Field Service Contractor Portal.

## Before you begin

Role required: wm\_ext\_agent and wm\_ext\_manager

## Procedure

1.  Navigate to **All** &gt; **Field Service Contractor Portal** &gt; **Homepage**.

2.  Select **My Lists** &gt; **My Tasks** &gt; **All Tasks**.

3.  Open a work order task from the list.

4.  Click the Sibling Tasks related list in the **Work Order Task** form.

5.  From the **Service Order Task** list, open a sibling work order task and review the task details.

    The sibling work order task provide details, such as parent work order task number, required asset, location, and so on.
