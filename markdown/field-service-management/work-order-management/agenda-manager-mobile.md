# View agents' agenda in Field Service Manager Mobile

Manager’s can see the agenda of tasks that agents' have assigned to them so they can plan for the coming days.

## Before you begin

Role required: wm\_manager, sn\_fsm\_tp.fsm\_territory\_resource\_manager

## Procedure

1.  Open Field Service Manager Mobile.

2.  Select **Schedules**.

3.  Choose from the following

    |Agenda selection|Definition|
    |----------------|----------|
    |**Select Today**|To view the tasks that agents you manage have scheduled today.|
    |**Select Tomorrow**|To view the tasks that agents you manage have scheduled tomorrow.|
    |**Select This week**|To view the tasks that agents you manage have scheduled for the rest of the week.|

    **Note:** The numbers on each of the sections indicate the number of tasks for each day or week.

    The list of assigned tasks opens.

4.  Select the assignment group or territory under **Schedules by team** to see the schedules for any of the agents you manage.

5.  Select an agent under the **Today’s Roster** section to see the work schedule of the agents that are working today.

    **Note:** By default only five agents show, to see all the agent’s with work today select **See all**.
