# View transfer order lines for requested parts

View all transfer order lines for the parts that are requested by you.

## Before you begin

Role required: wm\_agent or wm\_dispatcher

## About this task

Transfer orders move necessary parts between stockrooms or to a location where an agent can receive them. The transfer order lines are created as following:

-   For consumable parts, only one transfer order line is created for all requested parts.
-   For hardware or non-consumable parts, one transfer order line is created for each asset of the requested parts.

## Procedure

1.  Open the ServiceNow Agent application.

2.  View the transfer orders.
