# View agent information

View information related to Field Service Management agents in Field Service Manager Mobile, giving managers the ability to see agent information no matter where they are.

## Before you begin

Role required: wm\_manager, sn\_fsm\_tp.fsm\_territory\_resource\_manager

## Procedure

1.  Open Field Service Manager Mobile and tap **Teams**.

2.  Tap the assignment group or territory that the Field Service agent you want to manage is part of.

3.  Tap the agents that you want to view information of.

    The agent's contact information shows on the screen.

4.  Tap **Related** and choose from the following.

    |Selection|Action|
    |---------|------|
    |**Tap recently closed tasks**|To see the tasks the agent recently closed.|
    |**Tap skills**|To see the skills that are applied to the agent.|
    |**Tap current location**|To see where the agent is currently located.|
    |**Tap agent shift schedule**|To see the schedule assigned to the agent.|
