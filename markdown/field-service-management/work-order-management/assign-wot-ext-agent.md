# Assign outsourced work order tasks

Managers of contractor companies can assign outsourced work order tasks to be performed at the customer location to their agents or to themselves.

## Before you begin

Role required: wm\_ext\_manager

## About this task

Outsourced work order tasks are always automatically assigned to the managers of the contractor companies.

## Procedure

1.  Navigate to **All** &gt; **Field Service Contractor Portal** &gt; **Homepage**.

2.  Click **My Lists** &gt; **My Tasks** &gt; **All Tasks**.

3.  Click a work order task in the list.

4.  On the Work Order Task form, assign the work order task.

    -   To work on the task yourself, click **Accept**.
    -   To assign the task to an agent, select one from the **Assigned to** list.
    -   To cancel the task, click **Reject**.

        If you reject working on the task, its state changes to pending dispatch or pending assignment so it can be reassigned to another qualified group.

5.  Click **Save**.


## Result

If you have assigned the task to an agent, the agent receives an email notification about the assignment.
