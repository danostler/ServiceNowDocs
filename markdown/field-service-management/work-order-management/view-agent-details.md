# View an agent profile on the Field Service Contractor Portal

As a manager of a contractor company, you can view the profile information of all the agents in your teams.

## Before you begin

Role required: wm\_ext\_agent and wm\_ext\_manager

## About this task

Agents can view and edit only their own profile information.

## Procedure

1.  Navigate to **All** &gt; **Field Service Contractor Portal** &gt; **Homepage**.

2.  Click **Agents**.

3.  From the **Service Organization External Staffs** list, click the agent profile that you want to review.

4.  Review the agent profile information.
