# Filter an agent’s tasks in Field Service Manager Mobile

Managers can filter the tasks assigned to an agent so they can quickly find the task they want to change.

## Before you begin

Role required: wm\_manager, sn\_fsm\_tp.fsm\_territory\_resource\_manager

## Procedure

1.  Open Field Service Manager Mobile and Tap **Teams**

2.  Select the assignment group or territory that the Field Service agent you want to manage is art of.

3.  Tap the clipboard icon \(\[Omitted image "manager-clipboard.png"\] Alt text: clipboard\) next to the agent whose assigned tasks you want to see.

4.  Tap filter icon \(\[Omitted image "filter-manager-app.png"\] Alt text: filter\).

5.  On the form, fill in the fields.

6.  Tap **Apply**.
