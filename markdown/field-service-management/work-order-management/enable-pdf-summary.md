# Enable PDF summaries at work order task level

Generate Signed PDF summaries after completing a work order task that includes details of the work order task, related work order, responses submitted in the smart assessment questionnaire associated with it, and the assessment score.

## Before you begin

Role required: wm\_agent

## Procedure

1.  Navigate to **All** &gt; **Field Service** &gt; **Configuration**.

2.  On the Field Service Configuration page, select the **Add-ons** tab.

3.  In the **Signature Capture And PDF Order Summary** panel, turn on the **Signature Capture**, **PDF Order Summary**, and **Use Document Template to generate PDF Summary** options.

    The **Work Order Document Template** and **Work Order Task Document Template** fields are displayed.

4.  Select the required template to generate the PDF from the **Work Order Task Document Template** field.

5.  Tap **Save**.


## Result

The configuration to generate signed PDF summaries at the work order task level is enabled. For more information, see [Close a work order task](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/field-service-management/work-order-management/close-wo-wot-mobile.md).
