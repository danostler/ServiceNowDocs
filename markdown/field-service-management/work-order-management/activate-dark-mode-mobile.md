# Activate dark mode

Turn on dark mode in ServiceNow Agent to alleviate eye strain if the default is too bright.

## Before you begin

Role required: wm\_agent

## Procedure

1.  Open the ServiceNow Agent application

2.  Tap **Settings**.

3.  Tap **Appearance**.

4.  Tap **Dark theme**.
