# Upload a document from Shopping Hub Home

Complete the task that your procurement specialist has assigned to you by uploading a document, directly from Shopping Hub Home. This action is listed as a task under **My to-dos**.

## Before you begin

Role required: sn\_shop.shopper

## Procedure

1.  Navigate to **All** &gt; **ShoppingHub** &gt; **ShoppingHub Home**.

2.  Select the user icon and select **My to-dos**.

3.  Select the **Other to-dos** tab.

4.  Select the task that you want to work with.

5.  View details of the task, such as the task number, due date, if any.

6.  Select **Select** to select one or more documents from your local drives, as required.

    Alternatively, you can drag and drop documents from your local drives to attach them.

7.  In the **Upload a file** dialog box, select **Upload** to attach the documents.

    Once attached, you have the option to download, remove, or rename the documents, if required.

8.  Select **Mark as complete**.
