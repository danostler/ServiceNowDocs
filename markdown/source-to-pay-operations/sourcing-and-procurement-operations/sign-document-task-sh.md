# Sign a document using DocuSign from Shopping Hub Home

Provide your consent by signing a document using DocuSign, directly from Shopping Hub Home. This action is listed as a task under **My to-dos**.

## Before you begin

Role required: sn\_shop.shopper, supplier

## Procedure

1.  Navigate to **All** &gt; **ShoppingHub** &gt; **ShoppingHub Home**.

2.  Select the user icon and select **My to-dos**.

3.  Select the **Other to-dos** tab.

4.  Select the task that you want to work with.

5.  View details of the task, such as the task number, due date, if any.

6.  On the displayed document, select **Continue**.

7.  Select **Start**.

8.  Follow the arrowhead and select **Sign**.

9.  Select **Finish**.
