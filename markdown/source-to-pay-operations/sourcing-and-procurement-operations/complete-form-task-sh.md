# Complete a form from Shopping Hub Home

Fill in a form that your procurement specialist has sent to you, and submit it directly from Shopping Hub Home. This action is listed as a task under **My to-dos**.

## Before you begin

Role required: sn\_shop.shopper

## Procedure

1.  Navigate to **All** &gt; **ShoppingHub** &gt; **ShoppingHub Home**.

2.  Select the user icon and select **My to-dos**.

3.  Select the **Other to-dos** tab.

4.  Select the task that you want to work with.

5.  View details of the task, such as the task number, due date, if any.

6.  On the displayed form, fill in the details as required.

7.  Select **Save as** if you want to save a draft in your requests and plan to complete it later.

8.  Select **Submit**.
