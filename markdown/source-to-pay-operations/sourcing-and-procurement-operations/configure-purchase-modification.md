# Configure purchase modification record producers

Configure the conditions to make the purchase modification records available for the shopper to edit or cancel. You can leverage the condition builder to set specific conditions from the ShoppingHub Configuration table.

## Before you begin

Role required: sn\_shop.shopping\_hub\_admin or sn\_shop.procurement\_administrator

## Procedure

1.  Navigate to **All** &gt; **ShoppingHub** &gt; **Administration** &gt; **ShoppingHub Configuration**.

2.  Open the user interface configuration record you want to set conditions for.

3.  In the **Appearance conditions** section, add or remove the desired conditions with values that must be met for the record to be available to the shopper to edit or cancel.

4.  Select **Update**.
