# Mark an exception task as complete from Employee Center

Mark an assigned invoice exception task as complete from Employee Center when you have finished working on the task.

## Before you begin

Role required: sn\_shop.invoice\_owner or sn\_ap\_cm.task\_owner

## Procedure

1.  Navigate to **All** &gt; **Employee Center** &gt; **My Tasks**.

2.  On the **Open** tab, select an exception task.

3.  Select **Mark Complete**.

    The Marking the task complete dialog box is displayed.

4.  Enter your comments and select **Mark Complete**.

    The status of the exception task changes to Closed complete.
