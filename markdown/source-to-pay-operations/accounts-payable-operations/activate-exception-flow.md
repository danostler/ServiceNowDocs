# Activate the Exceptions Monitoring Scheduler

Activate the **Exception Monitoring Scheduler** to run the exception engine on a regular schedule to identify invoice exceptions.

## Before you begin

Role required: admin

## About this task

The default automatically runs every 5 minutes after you activate it.

## Procedure

1.  Navigate to **All** &gt; **System Definition** &gt; **Scheduled Jobs**.

2.  Search for and select the **Exception Monitoring Scheduler** to open it.

3.  Select the**Active** check box.

    You will notice the**Repeat interval** set to five minutes.\[Omitted image "apo-exceptions-monitor.png"\] Alt text: Activate the Exceptions Monitoring Scheduler

4.  Select **Update**.


## Result

The Exception Monitoring Scheduler is activated.
