# Troubleshooting

Diagnose and resolve common issues in Accounts Payable Operations features using troubleshooting steps and diagnostic guidance to restore normal invoice processing

## Invoice status is bypassed

An Invoice stage record shows status as bypassed, indicating that the standard DocIntel processing path was skipped.

### Symptom

An Invoice stage record shows status as bypassed, indicating that the standard DocIntel processing path was skipped.

The `bypassed` status \(sequence 45 in the status choices\) indicates that processing was bypassed rather than completing normally.

### Resolution

Confirm when and how the `bypassed` status is set, and what action \(if any\) is required from the user when an Invoice Stage is bypassed.
