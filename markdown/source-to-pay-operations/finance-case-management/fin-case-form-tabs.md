# Finance Case form tabs

The Finance case form includes tabs that store finance case information that a finance specialist can use to perform related tasks.

\[Omitted image "fin-ops-case-form-tab.png"\] Alt text: Tabs on the Finance Case form

<table id="table_op1_2ml_w2c"><thead><tr><th>

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Details

</td><td>

Details about the case.

</td></tr><tr><td>

Tasks

</td><td>

Tasks that the finance specialist created for this case.

</td></tr><tr><td>

Emails

</td><td>

Emails sent or received while resolving this case.

</td></tr><tr><td>

SLAs

</td><td>

Service level agreements \(SLAs\) associated with the case. -   For assigning finance cases, SLA is 4 hours duration, between 08:00 to 17:00 on weekdays, excluding holidays.
-   For processing finance cases, SLA is 16 hours duration between 08:00 to 17:00 on weekdays, excluding holidays.

Finance admins can view and create SLAs by navigating to **All** &gt; **Finance Operations** &gt; **Administration** &gt; **Finance case SLA Definitions**.

</td></tr><tr><td>

Related cases

</td><td>

Other cases related to this case.

</td></tr><tr><td>
