# Manage documents from My Company page

You can upload, update, and download documents and document templates from the document widget on the My Company page.

## Before you begin

Role required: sn\_slm.contact

## About this task

Access to the Documents widget on the My Company page is controlled by the user criteria admins set on the Manage access tab when creating the supplier document configuration. For more information, see [Create a supplier document configuration](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/source-to-pay-operations/supplier-lifecycle-operations/create-doc-config.md).

## Procedure

1.  Navigate to the Supplier Collaboration Portal home page by accessing your instance URL and adding a /supplier suffix.

    For example, `https://example.com/supplier`.

2.  In the portal header, select **My Company**.

3.  Select the supplier from the list of suppliers, if applicable.

4.  Go to the Documents widget.

    You can now upload, download, maintain versions, and manage permissions for documents as follows.\[Omitted image "doc-widget-my-company.png"\] Alt text: Documents widget on the My Company page
