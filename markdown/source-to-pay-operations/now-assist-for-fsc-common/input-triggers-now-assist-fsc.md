# Skill inputs for Purchase order summarization for fulfillers skill

You can configure some of the inputs for a generative AI skill. Inputs permit you to determine how and when a skill is used.

## Overview of skills

Inputs identify the data used for a skill. Inputs include the table and fields used to generate a record summary.

You can modify inputs, but you can't modify a skill's data source. The data source contains the tables and fields that the skill relies on.

## Now Assist skills for Now Assist for Common Finance and Supply Chain features

The Now Assist for Common Finance and Supply Chain features includes the Purchase order summarization for fulfillers skill.

## Purchase order summarization for fulfillers skill

Inputs for the Purchase order summarization for fulfillers skill identify the table and fields used when the summary is generated for a purchase order.
