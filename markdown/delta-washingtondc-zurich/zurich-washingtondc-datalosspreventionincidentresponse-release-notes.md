-   **[Create a Data Loss Prevention Incident Response SLA trigger](https://servicenow-staging.fluidtopics.net/access?context=sla-records&family=yokohama&ft:locale=en-US)**

Enable prompt and efficient responses to incidents by creating SLA triggers.

-   **[Create a Data Loss Prevention Incident Response SLA definition](https://servicenow-staging.fluidtopics.net/access?context=dlp-sla-definitions&family=yokohama&ft:locale=en-US)**

Outline the conditions and duration for responding to data breaches by creating Data Loss Prevention Incident Response SLA definitions.

-   **[Create an Application in Proofpoint and Obtain Client Credentials](https://servicenow-staging.fluidtopics.net/access?context=create-application-proofpoint-dlp&family=yokohama&ft:locale=en-US)**

Create an application in Proofpoint and configure the required settings to obtain client credentials. These credentials enable secure access to the Proofpoint API for seamless integration and automation.

-   **[Internet Content Adaption Protocol \(ICAP\) integration for DLP IR](https://servicenow-staging.fluidtopics.net/access?context=icap-dlp-integration&family=yokohama&ft:locale=en-US)**

Integration supports the ingestion of Data Loss Prevention Incident Response alerts, allows the fetching of match content, and evidence files from Amazon S3 created on the ICAP supported Data Loss Prevention Incident Response deployment.

-   **[Configure evidence file storage](https://servicenow-staging.fluidtopics.net/access?context=configure-evidence-file-storage&family=yokohama&ft:locale=en-US)**

Store evidence files directly in your ServiceNow instance with Proofpoint integration, by enhancing the ability to manage and track evidence files within the platform.

-   **[Configure evidence file storage](https://servicenow-staging.fluidtopics.net/access?context=config-evidence-file&family=yokohama&ft:locale=en-US)**

Symantec DLP supports evidence file storage to securely store the evidence files for the DLP Incidents.

-   **[Preview Evidence files for DLP incidents of type Exchange Online, OneDrive, and SharePoint](https://servicenow-staging.fluidtopics.net/access?context=preview-file-dlp-microsoft&family=yokohama&ft:locale=en-US).**

With the DLP Microsoft integration, preview evidence files in the DLP Workspace in the Microsoft OneDrive, Microsoft Exchange Online, and Microsoft SharePoint formats. You can preview and download evidence files directly from the preview interface, simplifying evidence review and retrieval.

-   **Netskope integration: [Preview evidence files](https://servicenow-staging.fluidtopics.net/access?context=preview-files-netskope&family=yokohama&ft:locale=en-US) and [Download evidence files](https://servicenow-staging.fluidtopics.net/access?context=download-files-netskope&family=yokohama&ft:locale=en-US)**

With Netskope integration you can preview and download evidence files directly from the preview interface, simplifying evidence review and retrieval.

-   **[Preview evidence files](https://servicenow-staging.fluidtopics.net/access?context=preview-evidence-files&family=yokohama&ft:locale=en-US)**

Preview evidence files and download them directly from the preview interface, simplifying evidence review and retrieval.

-   **[Playbook for Data Loss Prevention Incident Response](https://servicenow-staging.fluidtopics.net/access?context=playbook-for-dlp&family=yokohama&ft:locale=en-US)**

Introduced playbooks in the DLP Workspace to enhance operational efficiency.

-   **[Create incident consolidation rules](https://servicenow-staging.fluidtopics.net/access?context=configure-incident-consolidation-rules-to-consolidate-your-dlp-incidents&family=yokohama&ft:locale=en-US)**

Parent incident is always assigned the highest priority among consolidated incidents, enhancing incident management accuracy.
