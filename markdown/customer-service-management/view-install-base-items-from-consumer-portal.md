# Access your list of install base items from the Consumer Portal

View install base items registered to your account from the Consumer Portal, including details for each device or service.

## Before you begin

Install Customer Service Install Base Management \(com.snc.install\_base\) app from [ServiceNow store](https://store.servicenow.com/sn_appstore_store.do#!/store/home).

Role required: none

## Procedure

1.  Log in to the Consumer Portal.

2.  Navigate to **My hub** &gt; **Manage install base items**.

    In the header on the Consumer Portal, you can access a list of the install base items that are registered to you.

    In the list layout for an install base item, the details that are related to consumers in the **Household** and **Consumer** fields are visible.

    **Note:** Only consumers with the appropriate roles can see the install base option in the support menu.

3.  From the list, open an install base record to see all the details and cases for that install base item.
