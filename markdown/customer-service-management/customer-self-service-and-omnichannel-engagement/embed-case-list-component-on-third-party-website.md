# Embed the Case list component instance on the third-party website

Embed the global code and component code of the Case list component instance that you configured so that you can enable your customers to start using the Case list component instance on your third-party website.

## Before you begin

You must configure the Case list component in your instance. For more information, [Configure the Case list component instance](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/customer-service-management/customer-self-service-and-omnichannel-engagement/configure-case-list-component-on-instance.md).

Role required: None

## Procedure

1.  On your third-party website, in the website's HTML file, paste the global code that you copied.

    **Note:** The global code only needs to be added once to your third-party website.

2.  On your website’s HTML file, paste the component code that you copied.

3.  Open the website to which you embedded the component to verify that it launches the Case List component instance.


## Result

The Case list component instance appears on your third-party website.
