# Properties installed with Web Embeddables

Configure the properties available with Web Embeddables. Properties are added with the activation of the Web Embeddables plugin.

All system properties are located in the System Properties \[sys\_properties\] table. To access table, navigate to **All** &gt; **System properties** or enter `sys_properties.list` in the navigation filter.

<table id="table_wkj_blc_25b"><thead><tr><th>

Property

</th><th>

Description

</th></tr></thead><tbody><tr><td>

\[glide.uxf.lib.embeddables.enabled\]

</td><td>

Enables the rendering of web components and assets in an embeddables context. You must enable this sys property to allow Web Embeddables to function on third-party website.By default, the property is set to true.

</td></tr><tr><td>

\[sn\_cb\_experiences.uib.enable.cb\]

</td><td>

Enables the component builder experience in UI Builder.By default, the property is set to true.

</td></tr></tbody>
</table>
