# Edit a custom component

Edit a custom component to further customize its design and properties as per the requirement. You can only edit a custom component that is in a deactivated state. Prebuilt components cannot be modified.

## Before you begin

Beginning with the Zurich release, Component UI Builder is enabled by default in your instance. Existing customers on release versions before Zurich release, you must enable Component UI Builder in your instance to edit a component. Make sure you’re referring UI Builder version 27.2 or above for editing component.

Make sure you’re referring UI Builder version 27.2 or above for component creation.

Role required: sn\_embeddable\_core.emb\_admin

## Procedure

1.  Navigate to **All** &gt; **Web Embeddables** &gt; **Homepage**.

2.  In the Manage components section, select Action \(⋮\) on the component.

    **Note:** You can only edit components that are labeled as **Custom**.

3.  Select **Edit**.

4.  Modify the following field values.

    |Field|Description|
    |-----|-----------|
    |Name|Name of the custom component.|
    |Component tag|The tag used in the embed code helps identify the component.|
    |Description|Description of your component.|
    |Icon|Thumbnail image of the custom component.|

5.  Select **Save and close**.

6.  Select **Edit component**.

    **Note:** You will be redirected to the UI Builder application where you can modify the component.
