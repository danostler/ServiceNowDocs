# Embed ServiceNow components instance on the third-party website

Embed the global code and component code of the components that you configured in your instance so that you can enable your customers to start using component instance on your third-party website.

## Before you begin

At least one ServiceNow component should be configured in your instance. For more information, see [Configure the ServiceNow component instance](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/customer-service-management/customer-self-service-and-omnichannel-engagement/configure-servicenow-web-components-instance.md).

Role required: none

## About this task

This procedure outlines the high-level steps that you must follow to embed Web Embeddables component instance on your third-party website.

## Procedure

1.  On your third-party website, in your website’s HTML file, paste the global code that you copied.

    **Note:** The global code only needs to be added once to your third-party website.

2.  On your website’s HTML file, paste the component code that you copied.

3.  Open the website in which you embedded the component instance to verify that it launches the component.


## Result

The ServiceNow component instance you’ve embedded appears on your third-party website.
