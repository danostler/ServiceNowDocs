# Mobile-responsive Web Embeddables

Web Embeddables components automatically adapt their layout and controls for mobile devices to provide an optimized experience on phones.

## Adaptive mobile interface

When you embed ServiceNow Web Embeddables components on your website or portal, they detect the device type and adjust their interface for mobile screens. This mobile-responsive design provides touch-friendly controls, reorganized layouts, and optimized interactions for smaller screens without requiring any configuration.

The mobile-responsive experience activates automatically when users access your embedded components from mobile devices. Desktop users continue to see the standard layout, while mobile users get an interface specifically designed for touch interaction.

The following Web Embeddables components include mobile-responsive designs:

-   Knowledge article view: Read knowledge articles with a mobile-optimized layout, touch-friendly language and version selectors, and content that reflows for narrow screens.
-   Catalog item: Order catalog items using forms that adapt field layouts for mobile, with drop-down menus that open as bottom sheets and one-tap copy functionality for reference numbers.
-   Case create: Create cases with mobile-optimized forms, touch-friendly controls, and easy-to-use confirmation screens with an option to copy case numbers for reference.
