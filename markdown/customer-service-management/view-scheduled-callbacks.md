# View scheduled callbacks

View all scheduled callbacks in a list view and access individual callback records to reschedule or cancel them.

## Before you begin

Role required: sn\_customerservice\_agent

## About this task

The Scheduled Callbacks list view displays all scheduled callbacks with details such as callback task number, contact or consumer name, callback number, scheduled start time, and reason for the call. Only callbacks with a type as Scheduled and the state as Open are displayed.

## Procedure

1.  Navigate to **Workspaces** &gt; **CSM/FSM Configurable Workspace.**.

2.  Select the List icon \(\[Omitted image "list-icon-callback.png"\] Alt text: List icon to view callback list\)

3.  In the Callbacks section, select **Scheduled callbacks**.

4.  Select a scheduled callback to open the callback task record page.


## Result

You can view all scheduled callbacks and access the callback task record page to reschedule or cancel them.
