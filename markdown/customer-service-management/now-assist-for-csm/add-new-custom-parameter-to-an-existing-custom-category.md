# Add custom parameter to an existing custom category

Create and save a custom parameter within a category that supports custom parameters.

## Before you begin

Role required: admin

## About this task

Use this procedure to add custom parameter to an existing custom category. All fields in the form must be completed before saving.

**Note:** You can’t add custom parameters to the base system categories.

## Procedure

1.  In the **Define scoring parameter** step, select **Add new** &gt; **Parameter** to add parameter.

2.  In the pop-up window, select the radio button **Add parameter** to existing category.

3.  Enter a **Category**.

    Only custom categories are available in the category drop-down. The error message is seen at field level as soon as the user types a parameter name that is the same as other parameters.

4.  Fill in all the required form fields for the new parameter.

5.  Select **Save**.


## Result

The new custom parameter is added to the selected custom category. You get a confirmation message confirming the action.
