# Edit custom category

Update the details of an existing custom category when changes are required.

## Before you begin

Role required: admin

## About this task

Use this task to edit an existing custom category. You can modify all category fields. Only applicable for cloned skill. The admin isn’t permitted to edit a new category for base system skill.

## Procedure

1.  In the **Define scoring parameter** step, select a custom category to edit.

    The system opens the selected category record in a modal form.

2.  Update the value in the **Name** field.

    The name must be unique. Duplicate names result in a warning message.

3.  Modify any additional fields as needed.

    All fields in a custom category are editable. Verify that all changes conform with field-format requirements shown in the form.

4.  Select **Save**.

    The system validates the updated category information.


## Result

If validation succeeds, the category is updated in the category list. If validation fails, the system displays error messages that must be corrected before saving again.
