# Add new category

Create a category by entering the required details and saving the record.

## Before you begin

Role required: admin

## About this task

Use this task to create category by entering all the required fields. Only applicable for Cloned skill. Admin isn’t permitted to add category for the base system skill.

## Procedure

1.  In the **Define scoring parameter** step, select **Add new** &gt; **Category**.

    The system displays the form for adding a category.

2.  Enter a value in the **Name** and **Short description** fields.

    **Note:** The name must be unique. Duplicate names trigger a warning message.

3.  Select **Save**.

    The system validates the new category information.


## Result

If validation is successful, the category is added to the category list. If validation fails, the system displays error messages that must be corrected before saving again. The new category appears in the category list.
