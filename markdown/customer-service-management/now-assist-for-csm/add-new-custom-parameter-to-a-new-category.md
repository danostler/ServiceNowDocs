# Add custom parameter to a new category

Create a custom category and add a custom parameter to it.

## Before you begin

Role required: admin

## Procedure

1.  In the **Define scoring parameter** step, select **Add new** &gt; **Parameter** to add parameter.

2.  In the pop-up window, select the radio button **Create new category &amp; add parameter** to existing category.

3.  Enter a category name and a short description.

4.  Select **Next**.

5.  Add new parameter details to the new category.

    The error shows at field level as soon as the user types the name that is the same as an existing parameter.

6.  Select **Save**.


## Result

The new parameter is added to the newly created category. You get a confirmation message that a new category and parameter are added.
