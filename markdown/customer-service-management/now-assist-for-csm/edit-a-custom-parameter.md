# Edit a custom parameter

Modify an existing custom parameter by editing it and resolving naming conflicts if needed.

## Before you begin

Role required: admin

## Procedure

1.  In the **Define scoring parameter** step, select a custom parameter to open it for editing.

2.  Edit any field as needed.

    For example, you can edit the **Exclusion criteria**. Parameters are grouped under categories. If the parameter name isn’t unique, you see the error message: 'Parameter name exists. Choose a unique name.'

3.  Select **Save**.


## Result

The custom parameter is updated and saved successfully.
