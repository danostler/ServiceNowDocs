# Edit a parameter in the base system

Update the weight of a parameter and enable or disable it.

## Before you begin

Role required: admin

## About this task

Use the parameter editor to modify the weight of a selected parameter. You can activate or deactivate the parameter, but all other details are read-only.

## Procedure

1.  In the **Define scoring parameter** step, select a parameter to edit.

    The parameter details appear in the edit panel.

2.  Edit the Weight field.

    Enter a new numeric weight value to adjust the parameter’s impact.

3.  To enable or disable the parameter, select or clear the **Active** check box.

    Enabled parameters are included in scoring. Inactive ones are ignored.

4.  Review the remaining fields.

    All other fields are read-only and can’t be modified.

5.  Select **Save**.

    You see an alert confirming that the parameter has been saved and updated.


## Result

The parameter is updated with the new weight and activation status.
