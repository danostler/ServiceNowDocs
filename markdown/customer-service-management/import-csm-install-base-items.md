# Import install base items with guided setup

Install base items represent the instances of the product that have been configured for a customer.

## Before you begin

Role required: csm\_guided\_setup\_user or admin

## About this task

Each install base item references the configuration item record for the customer.

## Procedure

1.  Navigate to **All** &gt; **Customer Service** &gt; **Administration** &gt; **Guided Setup** and click **Get Started**.

2.  In the Foundation Data category, click **Get Started** and then click **Import Install Base Items**.

3.  Upload data from an external data source into an import set.

4.  Create a transform map.

5.  Execute the transform map to transfer the data.

6.  Verify that the data records are imported into the target table.
