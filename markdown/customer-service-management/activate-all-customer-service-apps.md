# Install all Customer Service Management applications

Install all eligible Customer Service Management \(CSM\) applications from a single ServiceNow Store product. Use this procedure to find and install the CSM apps your organization is entitled to in one step.

## Before you begin

Role required: admin

## About this task

A ServiceNow Store product groups multiple Store applications into one installable bundle. With a Store product, customers can:

-   Install multiple apps in a single select.
-   See which apps they're entitled to, based on their licenses.
-   Choose which apps to include at install time.

CSM capabilities are delivered through many individual applications. Currently, customers must locate and install each app one at a time through the ServiceNow Store or Application Manager — a fragmented experience that increases the risk of missing apps. Because of the volume of apps, customers also frequently miss newly released CSM applications they're entitled to use.

The [Install all CSM Apps](https://store.servicenow.com/store/product/04085a37938bba501caaf76e1dba10fa) Store product consolidates the experience: customers see every CSM app they qualify for in one list and install them together.

## Procedure

1.  Log in to your ServiceNow instance.

2.  Navigate to **All** &gt; **Application Manager**.

3.  Search for the Install all CSM Apps Store product.

4.  Select **Install**.

5.  Review the list of eligible apps and select the apps to install.

6.  Select **Confirm**.
