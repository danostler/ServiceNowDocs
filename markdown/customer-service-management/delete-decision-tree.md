# Delete a decision tree

Delete a decision tree and all the related version records. All related records such as decision tree versions, nodes, and paths are deleted.

## Before you begin

Role required: admin, sn\_gd\_core.decision\_tree\_author

## Procedure

1.  Navigate to **All** &gt; **Guided Decisions** &gt; **Decision Trees**.

2.  Select the decision tree from the list.

3.  Select **Delete**.
