# Configure component attributes for the Customer History view

Set the Customer History view component attributes, as required.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Customer Central** &gt; **Administration** &gt; **Related Items**.

2.  Select **Customer History**.

3.  Select **Advanced View**.

4.  Select the Component Attributes tab.

5.  Set the following component attributes, as required.

<table id="table_zjn_kc4_nlb"><thead><tr><th>

Property

</th><th>

Description

</th></tr></thead><tbody><tr><td>

childFacetsListSize

</td><td>

Number of child facets to display.Default value is 3.

</td></tr><tr><td>

activitiesListSize

</td><td>

Number of activities to display in the activity feed.Default value is 10.

</td></tr></tbody>
</table>    **Note:** Avoid making changes to the other component attributes in this list.
