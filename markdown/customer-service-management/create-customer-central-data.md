# Create data for Customer Central

When upgrading from an existing release, customers must run a fix script to create data for Customer Central.

## Before you begin

Role required: admin

## About this task

This task applies to customers who are upgrading their release version and installing Customer Central for the first time. Running the fix script creates data in the Activities table for Customer Central.

## Procedure

1.  Navigate to **All** &gt; **System Definition** &gt; **Fix Scripts**.

2.  Select the **Create activities for last N days data** fix script.

3.  Select **Run Fix Script**.
