# Add and Manage Document Reference in Task Plan Templates

Add documents to task plan template items through extensible feature tables. Supported file types include PDFs, Word documents, PowerPoint files, or images.

## Before you begin

Roles required: admin, sn\_task\_plan.admin, sn\_task\_plan.creator, sn\_task\_plan.writer

Plugin Id: com.snc.platform\_document\_management.

## Procedure

1.  Navigate to **All** &gt; **Task Plan Template.**.

2.  Select a template which is in **Draft** state

3.  Select the template item to which you want to add documents from the **Template Items** related list.

4.  From the **Task Plan Template Document** related list, select **New**.

    **Note:** The **New** option is visible only for templates in **Draft** state, not for those that are **Published**.

5.  Confirm that the following information is auto-populated in the form.

    -   Select Document
    -   Task plan feature auto-populates to Documents and corresponding task plan template is also auto-populated.
    -   Template item is auto-populated from the form.
6.  Select **Submit**.
