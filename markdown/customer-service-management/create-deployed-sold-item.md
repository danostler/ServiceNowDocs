# Create installed products

Create an installed product to create an association between sold products and install base items. Installed products provide information on the instances that a sold product is deployed on.

## Before you begin

Role required: sn\_customerservice\_manager or admin

Ensure that the Customer Service Install Base Management plugin \(com.snc.install\_base\) is installed.

## Procedure

1.  Navigate to **All** &gt; **Customer Service** &gt; **Products** &gt; **Installed Products**.

2.  Click **New**.

3.  Fill out the fields on the form.

    |Field|Description|
    |-----|-----------|
    |Install Base Item|Instance the sold item is deployed on.|
    |Sold Product|Sold product deployed to the customer.|

4.  Click **Submit**.
