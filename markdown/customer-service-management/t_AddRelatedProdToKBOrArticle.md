# Add a related product to a knowledge base or knowledge article

Add a product to a knowledge base or knowledge article using the **Related Products** field on the corresponding form.

## Before you begin

Role required: admin

## About this task

To access knowledge bases and articles associated with a product, a user must have read or contribute access and have the product included in the user's account as an asset. For more information about assets, see [Configure assets](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/customer-service-management/configure-csm-assets.md).

## Procedure

1.  To add a related product to a knowledge base:

    1.  Navigate to **Knowledge** &gt; **Administration** &gt; **Knowledge Bases**.

    2.  Select the desired knowledge base.

    3.  Add one or more products in the **Related Products** field.

    4.  Select **Update**.

2.  To add a related product to a knowledge article:

    1.  Navigate to **Knowledge** &gt; **Articles** &gt; **All**.

    2.  Select the desired knowledge article.

    3.  Add one or more products in the **Affected Products** related list.

    4.  Select **Update**.
