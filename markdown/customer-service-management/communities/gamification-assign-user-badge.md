# Award a badge to a user

Award gamification badges to community users to recognize their community activities.

## Before you begin

Role required: sn\_gamification.moderator, sn\_gamification.admin

## About this task

A badge can be awarded to a user only once. If a badge is removed from a user, it can then be awarded to that user again.

## Procedure

1.  Navigate to **All** &gt; **Community** &gt; **Gamification Data** &gt; **Profile Badges**.

2.  Click **New**.

3.  Select a **Badge** from the Gamification Badges list.

4.  In the **Live Profile** field, add the name of the user.

5.  Click **Submit**.

    The badge is assigned to the user and it appears on the community portal.
