# Revoke a user ban

Revoke a user ban to allow a user to create and edit content again.

## Before you begin

Role required: sn\_communities.admin, sn\_communities.moderation\_admin, or sn\_communities.community\_moderator

## Procedure

1.  Perform either of the following actions.

    -   Navigate to **Community** &gt; **User Profile** &gt; **Community Profiles**.
    -   Navigate to **Community** &gt; **Moderation** &gt; **Users Banned by Moderation**.
2.  Click the Preview icon next to the user and then select **Revoke User Ban**.

    The user ban is revoked. In the Community Profiles list, the value in the **Banned** column is set to **false**.
