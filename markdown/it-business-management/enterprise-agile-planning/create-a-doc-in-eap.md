# Create a Doc in EAP

Store information related to your work, iterations, and teams, and collaborate in real-time using Docs in Enterprise Agile Planning workspace.

## Before you begin

Role required: sn\_apw\_advanced.eap\_user

## About this task

Create a Doc at the team-level or planning item level.

## Procedure

1.  Navigate to **Workspaces** &gt; **Strategic Planning Workspace** &gt; **Enterprise Agile Planning**.

2.  Navigate to a team or planning item that you would like to create a Doc for.
