# Personalize and filter data in EAP Backlog

View the information that is most relevant to you and your team on the Backlog tab by choosing the columns to be displayed and applying filters to narrow down the data on the page.

## Before you begin

Role required: sn\_apw\_advanced.eap\_user

## Procedure

1.  Navigate to **Workspaces** &gt; **Strategic Planning Workspace** &gt; **Enterprise Agile Planning**.

2.  From the Agile structure section of the left navigation panel, choose your EAP team.

3.  Add or remove columns and change their order of display on the Backlog page.

    1.  Select **Edit columns** \(\[Omitted image "eap-backlog-edit-columns.png"\] Alt text: Edit columns icon.\).

    2.  From the Available columns section, select the columns that you want to be displayed on the Backlog.

    3.  From the Selected columns section, Remove the columns that you don’t want on the Backlog.

    4.  From the selected columns section, rearrange the columns to change the order of their display on the Backlog.

    5.  Select **Apply**.

        \[Omitted image "eap-edit-columns.png"\] Alt text: Personalize column display for Backlog.

4.  Apply filters to work items to narrow down the data that's displayed on the Backlog.

    1.  Select **Filter** \(\[Omitted image "eap-filter.png"\] Alt text: Filter icon.\).

    2.  Add conditions to filter the work items.

        You can add multiple conditions. These conditions apply to the **Type** of work item that is displayed in the Backlog page.

    3.  Select **Save filter** so that you need not recreate the same conditions to apply them to a different set of data.

    4.  Select **Apply**.

5.  For an ART, you can personalize the grouping of the work items by Planning Interval or Team.

    \[Omitted image "eap-backlog-art-personalize.png"\] Alt text: Group items on ART Backlog by team or planning interval.
