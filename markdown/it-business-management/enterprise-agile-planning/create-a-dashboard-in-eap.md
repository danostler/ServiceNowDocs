# Create a dashboard in EAP

Create a dashboard to track key metrics for your custom Agile configuration by creating a dashboard or duplicating and editing a default EAP dashboard.

## Before you begin

[Create or update a configuration in EAP](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-business-management/enterprise-agile-planning/create-eap-configuration.md)

Role required: sn\_apw\_advanced.eap\_admin

## Procedure

1.  Navigate to **All** &gt; **Platform Analytics** &gt; **Dashboards**.

2.  Create a dashboard or modify one of the default dashboards for your Agile configuration.

    -   To create a dashboard, select **Create new dashboard**. For more information, see [Create a dashboard with the in-line editor](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/now-intelligence/create-db-in-ac.md).
    -   To modify an existing dashboard, create a copy of it so that you can the widgets that you need. For more information, see [Duplicate a Platform Analytics dashboard](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/now-intelligence/duplicate-db.md).

        **Note:** The **Duplicate** option is available only for an Inline dashboard.

3.  Add visualizations and widgets.

    For more information, see step 6 through 10 in [Create a dashboard with the in-line editor](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/now-intelligence/create-db-in-ac.md).

    For information about the default indicators, see [Enterprise Agile Planning Indicators](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-business-management/enterprise-agile-planning/eap-indicators.md).

4.  Select **Save**.
