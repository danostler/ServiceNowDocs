# Export a Doc in CWM to a PDF file

Use the Docs offline, and share with teams or stakeholders outside Collaborative Work Management \(CWM\) by exporting Docs as PDF.

## Before you begin

[Create a Doc in CWM](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-business-management/collaborative-work-management/create-a-doc-in-cwm.md)

Role required: sn\_cwm.cwm\_user

## Procedure

1.  Navigate to **Workspaces** &gt; **Collaborative Work Management**.

2.  From a Space, select the Doc that you want to export.

3.  From the Doc header, select the More actions icon \(\[Omitted image "cwm-icon-more-actions-dots.png"\] Alt text: More actions icon.\), and then select **Export as PDF**.

    \[Omitted image "cwm-doc-export-pdf.png"\] Alt text: Export as PDF.


## Result

The Doc is exported and saved in your local system as a PDF file.
