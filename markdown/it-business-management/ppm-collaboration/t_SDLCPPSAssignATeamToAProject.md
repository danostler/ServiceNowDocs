# Assign a group to a project

Assign a group to a project so that the group later works on the tasks of the assigned project.

## Before you begin

Role required: it\_project\_manager

## Procedure

1.  NavNavigate to **All** &gt; **Project** &gt; **Projects** &gt; **All**.

2.  Select your project to open its form.

3.  From the **Assignment group** field, select a group to assign this project.

    If you don't see the group that you need, work with your admin to create it. See [Define an Agile group](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-business-management/ppm-collaboration/t_SDLCPPSCreateATeam.md).

4.  Save the project form.
