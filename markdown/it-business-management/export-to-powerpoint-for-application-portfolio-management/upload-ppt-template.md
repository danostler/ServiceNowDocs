# Upload your Microsoft PowerPoint template

Upload the latest template to your instance to generate and download the status report as a Microsoft PowerPoint file.

## Before you begin

Role required: sn\_ppt\_export.ppt\_user

## Procedure

1.  Navigate to **All** &gt; **PowerPoint Management** &gt; **PowerPoint Templates**.

2.  Select a record to attached the latest template.

3.  Select the manage attachments icon \(\[Omitted image "icon-attachment.png"\] Alt text: Manage attachments icon.\).

4.  Select **Choose file** to browse and upload the template.

    **Note:** You can use only one template \(`.pptx`\) at a time with a maximum file size of 15 MB. Delete the existing template to upload and use a new one.

5.  Close the dialog box, and select **Update** on the form.
