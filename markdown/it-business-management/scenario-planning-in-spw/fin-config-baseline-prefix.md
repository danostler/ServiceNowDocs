# Create a custom prefix for baseline

Modify the prefix to customize the name of the baselines for your planning items.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** and type in **sys\_properties.list** in the filter box.

2.  Filter the name filed to locate and open **sn\_invst\_pln.baseline\_prefix**.

3.  Modify the **Value** field to create a custom prefix for your baselines.

    If you’re working on a sales portfolio, you may change the **Value** field to `Sales baseline on`.

4.  Select **Update**.


## Example

Assume you are creating a baseline for a sales project using the above prefix on December 04, 2023. Name of the new baseline will be **Sales baseline on 2023-12-04**.
