# Choice map form

Understand the fields of the choice map form. This form is used to create choice maps.

|Field|Description|
|-----|-----------|
|Execution table column|Field column of the selected execution table that needs to be mapped. The value in this field is automatically added based on the selection made in the field map configuration.|
|Execution choice|Select the choice field value of the selected execution table column field.|
|Alignment table column|Field column of the selected alignment table that needs to be mapped. The value in this field is automatically added based on the selection made in the field map configuration.|
|Alignment choice|Select the choice field value of the selected alignment table column field.|
