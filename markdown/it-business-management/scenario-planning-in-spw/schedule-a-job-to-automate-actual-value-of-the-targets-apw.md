# Update a schedule job to automate Actual value of the targets

Use the Goal Framework for SPM application to modify the run time for the scheduled job according to your preference to auto-update the actual value of the targets for which a target source has been configured.

## Before you begin

Role required: sn\_gf.goal\_admin

## About this task

By default, the scheduled job runs daily at 00:00:00 AM \(System Time Zone\).

## Procedure

1.  Navigate to **All** &gt; **System Definition** &gt; **Scheduled Jobs**.

2.  Search for and click the **Update Actual value of the targets using Goal Framework for SPM** scheduled job.

3.  Ensure that the **Active** check box is selected to run the job at a scheduled time.

4.  Edit the details of the **Run** and **Time** fields according to your preference.

5.  Click **Update**.

    To learn more about scheduled jobs, see Scheduled jobs.
