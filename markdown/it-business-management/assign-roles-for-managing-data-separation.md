# Assign roles for managing Data Separation

Assign Data Separation roles to user groups and to individual users based on user activities and responsibilities.

## Before you begin

Role required: admin

## About this task

You can assign roles to individual users or groups. Applying roles to a group gives access to all the members of the group. For more information on roles installed with Data Separation, see the Roles installed table in [Components installed with Data Separation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-business-management/installed-with-data-separation.md).

## Procedure

1.  Navigate to **All** &gt; **Data Separation** &gt; **Guided Setup**.

2.  Select **Get Started**.

3.  Select **Assign roles** from the tasks list.

4.  Select the required role \(sn\_ds.ds\_admin or sn\_ds.ds\_privileged\_user\), select **Configure**, and assign the role to a user or user group.


## What to do next

Configure a data separation hierarchy for enabling Data Separation. For more information, see [Configure a lens for enabling Data Separation](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-business-management/configure-lens-for-data-separation.md).
