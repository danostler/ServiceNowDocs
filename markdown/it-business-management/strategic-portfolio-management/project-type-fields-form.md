# Project Type Fields form

Field descriptions for the Project Type Fields form. Use this reference when entering key project dates for a retail store project.

## New store opening project type form fields

|Field|Description|
|-----|-----------|
|Store go-live date|Date on which the retail store is scheduled to open and begin operations.|
|IT installation date|Date on which IT systems and technology infrastructure are scheduled to be installed at the store.|
|Site approval date|Date on which the store site receives formal approval to proceed with the project.|
|Construction start date|Date on which construction or refurbishment work at the store site is scheduled to begin.|

## Store relocation project type form fields

|Field|Description|
|-----|-----------|
|Relocation approval date|Date on which the store relocation receives formal approval to proceed.|
|Current store closure date|Date on which the current store location is scheduled to close.|
|New site readiness date|Date on which the new store site is expected to be ready for operations.|
|Relocation completion date|Date on which the relocation of the store is scheduled to be complete.|

## Store refurbishment project type form fields

|Field|Description|
|-----|-----------|
|Refurbishment approval date|Date on which the store refurbishment receives formal approval to proceed.|
|Decommissioning start date|Date on which decommissioning of the existing store setup is scheduled to begin.|
|Fit-out completion date|Date on which the store fit-out work is scheduled to be complete.|
|Re-opening date|Date on which the refurbished store is scheduled to reopen for operations.|

## Store technology refresh project type form fields

|Field|Description|
|-----|-----------|
|Refresh approval date|Date on which the technology refresh receives formal approval to proceed.|
|Installation scheduled date|Date on which the installation of new technology is scheduled to begin.|
|Installation completion date|Date on which the technology installation is scheduled to be complete.|
|Store handover date|Date on which the store is handed over after the technology refresh is complete.|

## Store closure project type form fields

|Field|Description|
|-----|-----------|
|Closure decision date|Date on which the decision to close the store is formally made.|
|Last trading day|Date on which the store is scheduled to trade for the last time.|
|Asset recovery completion date|Date on which the recovery of store assets is scheduled to be complete.|
|Closure confirmation date|Date on which the store closure is formally confirmed.|
