# Configure Retail Strategic Portfolio Management Suite

Configure Retail Strategic Portfolio Management Suite \(SPM Retail\) to manage store life cycle projects such as new store openings, closures, refurbishments, relocations, and technology refreshes. Follow the configuration overview or use guided setup to get started.

1.  Install the Retail Strategic Portfolio Management Suite from the ServiceNow Store.

    For more information, see [Install Retail Strategic Portfolio Management Suite](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-business-management/strategic-portfolio-management/install-spm-r-store-app.md).

2.  Add members to Retail HQ Group who can receive playbook stage approval requests.

    For more information, see [Add members to HQ group to complete playbook](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-business-management/strategic-portfolio-management/add-members-to-hq-group-spmr.md).
