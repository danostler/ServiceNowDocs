# Enter project type fields

Define the key dates to categorize and manage retail projects using the Retail Strategic Portfolio Management Suite.

## Before you begin

Role required: sn\_spm\_retail.project\_manager

## Procedure

1.  Navigate to **Workspaces** &gt; **Project Workspace** and open an existing retail project.

2.  Select **Details** tab from the side pane and select the **Project Type** related list.

3.  Enter the dates for the selected project type.

    Each project template displays fields specific to that project type. For more information about the fields and their descriptions, see [Project Type Fields form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-business-management/strategic-portfolio-management/project-type-fields-form.md).

4.  Select **Apply** or **Save** to save the entered dates.


## What to do next

-   Manage tasks and track project health using baselines and status reports. For more information, see [Managing projects with Project Workspace](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-business-management/project-workspace/use-projects-pw.md).
-   Assign tasks and work items to resources by creating resource assignments. For more information, see [Resource planning with Project Workspace](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-business-management/project-workspace/use-resource-mgmt-prj-wksp.md).
-   Create and track financial records for your project expenses using the Financials tab. For more information, see [Managing financials for your projects](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-business-management/project-workspace/using-financials-prj-wrkspc.md).
