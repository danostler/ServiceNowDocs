# Enable detailed project logging in Jira Integration with Agile Development

Understand how to enable integration project logging in Jira so that you can view details of the project logs.

## Before you begin

Role required: sn\_jira\_int.admin \(with edit access\) or sn\_jira\_int.user \(with read access\).

## About this task

Enable the project logging property in the Agile Jira Integration application.

## Procedure

1.  Navigate to **All** &gt; **Agile Jira Integration** &gt; **Properties**.

2.  Click the **Enable Integration Project Logging** check box.

    \[Omitted image "enable-logging-jira.png"\] Alt text: Click the Yes\|No check box to enable project logging.

3.  Click **Save**.
