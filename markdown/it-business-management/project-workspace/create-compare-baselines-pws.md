# Create and compare financial baselines for your projects

Create and compare financial baselines to get a snapshot of the project expenses and compare the planned costs with the actual expenses.

## Before you begin

For more information regarding financial baselines and how the comparison works, see [financial baselines](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/it-business-management/project-workspace/using-financials-prj-wrkspc.md).

Role required: it\_project\_manager

## Procedure

1.  Navigate to **Workspaces** &gt; **Project Workspace** and open a project.

2.  Select **Financials** from the left menu.

3.  Create or compare baselines.
