# Create a custom prefix for baseline

Modify the prefix to customize the name of the baselines for your projects.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **System Properties** &gt; **All Properties**.

2.  Filter the name filed to locate and open **sn\_invst\_pln.baseline\_prefix**.

3.  Modify the **Value** field to create a custom prefix for your baselines.

    If you’re working on a sales portfolio, you may change the **Value** field to `Sales baseline on`.

4.  Select **Update**.
