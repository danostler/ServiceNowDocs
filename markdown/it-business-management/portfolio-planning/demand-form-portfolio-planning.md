# Demand planning item form

Learn about the fields of the demand form. Use this form to create or edit the details of a planning item of the type Demand in Portfolio Planning.

These fields are displayed on the full details page of the planning item.

|Planned end date|Date on which the work for this planning item is planned to be complete. This value is copied from the **Approved end date** field.|

|Field|Description|
|-----|-----------|

|Business application|Business application that this planning item is related to.|

|Business capabilities|Business capabilities of this planning item.|
