# Create custom mapping configurations

Create mapping configurations for planning items that are not created by default such as features, stories, etc.

As an admin, you can choose to create table maps, field maps, and choice maps for the planning items that are not created by default. For more information, see:

1.  [Create a table map](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-business-management/portfolio-planning/create-custom-table-maps-for-custom-planning-items-ppw.md).
2.  [Create a field map](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-business-management/portfolio-planning/create-field-map-ppw.md).
3.  [Create a choice map](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-business-management/portfolio-planning/create-choice-map-ppw.md).
