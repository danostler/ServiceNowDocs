# Change the default expense type in Portfolio Planning

Configure the default expense type to create or edit cost plans for your planning items.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** and type in **sys\_properties.list** in the filter box.

2.  Filter the name to locate and open the **sn\_plng\_att\_core.default.expense\_type** table.

3.  Change the Value field to the required expense type and select **Save**.

    Opex is the default value.
