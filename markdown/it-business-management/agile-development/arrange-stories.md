# Arrange Agile Development 2.0 stories in Agile Board using the keyboard

Learn how to use the keyboard to move and rearrange Agile Development 2.0 stories in the Backlog and Sprint Planning tabs of your Agile board.

## Before you begin

Role required: scrum\_admin, scrum\_master, scrum\_product\_owner, or scrum\_sprint\_planner

## Procedure

1.  Reorder a single story:

    1.  Highlight the desired story by pressing the Tab key.

    2.  Highlight the context menu \(\[Omitted image "plan-sprint-activity.png"\] Alt text: Context menu icon\) icon on the story by pressing the Tab key again.

    3.  Select the story by pressing the Enter key.

    4.  Move the story to the desired position using the up and down arrow keys.

    5.  Fix the position of the story by pressing the Enter key.

2.  Reorder multiple stories:

    1.  Highlight the desired story by pressing the Tab key.

    2.  Select the story by pressing the Enter key.

    3.  Select multiple stories using the up and down arrow keys and the Shift or Control keys.

        -   To select consecutive stories, use the down arrow key to highlight the last story that you want to select, and then press the Shift and Enter keys.
        -   To select stories that are in different positions on the page, use the down arrow key till you go to the next story that you want to select. Then, press the Control and Enter keys.

            Repeat this process to select other stories.

    4.  Highlight the context menu \(\[Omitted image "plan-sprint-activity.png"\] Alt text: Context menu icon\) icon on the story by pressing the Tab key again.

    5.  Press the Enter key to group all the selected stories.

        Multiple stories that you selected appear grouped.\[Omitted image "stories-grouped-using-keyboard.png"\] Alt text: Image showing multiple stories selected on the Agile Board.

    6.  Move the stories to the desired position using the up and down arrow keys.

    7.  Fix the position of the stories by pressing the Enter key.
