# Create a time sheet from your mobile device

Create a time sheet to group all your time cards for the given week and submit them in a single step from the Now Mobile application.

## Before you begin

Role required: timecard\_ user

## Procedure

1.  Log in to your instance using the Now Mobile application.

2.  Tap the plus icon located on the top-right corner.

3.  In the **Select a day to create a time sheet** field, select the starting date of the week for which the time sheet is being created.

4.  Tap the forward icon.


## What to do next

Add time cards to the time sheet.
