# Resource Event form

Use the Resource Event form to update resource even which is part of resource plan.

<table id="table_sx1_ldt_bt"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

The name of the event.

</td></tr><tr><td>

Type

</td><td>

The type of event:-   **Time off**
-   **Appointment**
-   **Meeting**
-   **Phone call**
-   **Task**

</td></tr><tr><td>

When/To

</td><td>

The start and end date of event.

</td></tr><tr><td>

All day

</td><td>

If the event is an all-day event as specified by the schedule.

</td></tr><tr><td>

Repeats

</td><td>

If the event repeats **Daily**, **Weekly**, **Monthly**, or **Does not repeat**.

</td></tr><tr><td>

Task

</td><td>

The task associated with this event.

</td></tr><tr><td>

User

</td><td>

The user associated with the event.
