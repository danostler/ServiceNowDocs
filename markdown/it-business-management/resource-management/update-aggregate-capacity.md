# Update the resource aggregates

From the Resource Manager view, you can update the resource aggregates for a specific time frame.

## Before you begin

Role required: pps\_admin

## About this task

You can use the **Update Resource Aggregates** related link in the following scenarios:

-   If you notice a discrepancy in aggregates and want to generate correct data.
-   If you want to synchronize aggregates with the Resource Allocation Daily \[resource\_allocation\_daily\] entries for a specific time frame.

## Procedure

1.  Navigate to **All** &gt; **Resource Management** &gt; **Resources** &gt; **Users**.

2.  Select a **User ID**.

3.  Select the **Update Resource Aggregates** related link.

4.  In the window, provide a start date and end date.

5.  Select **OK**.
