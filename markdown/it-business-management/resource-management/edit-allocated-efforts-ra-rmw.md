# Edit allocated efforts for tasks with actual hours

Edit allocated efforts for tasks which have partial actual efforts logged directly from the grid view. This keeps reported time accurate without rolling back time entries through other channels.

## Before you begin

Role required: resource\_manager, it\_project\_manager

## About this task

The total allocated hours for a task appear in the Task effort column of the grid view for the corresponding period. You can edit these values directly in the grid, including the assignments for which partial actual efforts have been logged.

## Procedure

1.  Navigate to **Workspaces** &gt; **Resource Management Workspace**.

2.  Select the Resource cards icon \(\[Omitted image "rmw-resource-cards-L1-icon.png"\] Alt text: Resource cards icon.\) from the menu and open a resource card.

3.  Select the chevron icon \(\[Omitted image "rmw-chevron-image.png"\] Alt text: Chevron icon.\) to expand the resource view in the top tray.

4.  Select the cell for the required period in the **Task effort** column to change the efforts and enter the updated hours.


## Result

The allocated hours for the selected period are updated. If there are any actual efforts captured for the past fiscal periods, they won't be affected with this change.
