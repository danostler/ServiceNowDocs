# Create and manage resource assignments for a demand

As a resource requester, find the availability of the resources, and create and manage the resource assignments for a demand.

## Before you begin

Role required: it\_demand\_manager

## Procedure

1.  Navigate to **All** &gt; **Demand** &gt; **Demands** &gt; **All**.

2.  Open the required demand form.

3.  Select the **Resource assignments** related list and select **New**.

4.  On the Resource assignment new record form, fill in the fields.

    For a description of the field names, see [New Resource Assignment form](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-business-management/resource-management/create-ra-form-rmw.md).

5.  Select **Submit**.
