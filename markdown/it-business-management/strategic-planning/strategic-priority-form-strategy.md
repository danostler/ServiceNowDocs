# Strategic Priority form

Use the Strategic Priority form to orient your organization strategic plan toward your goal.

<table id="table_demand_benefit_plan_form"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Name of the strategic priority.

</td></tr><tr><td>

Parent

</td><td>

Name of the parent strategic priority that this strategic priority contributes to.

</td></tr><tr><td>

Owner

</td><td>

Owner of the strategic priority. By default, the name of the user creating the strategy is populated.

</td></tr><tr><td>

Start date

</td><td>

Start date for the strategic priority.

</td></tr><tr><td>

End date

</td><td>

End date for the strategic priority.

</td></tr><tr><td>

Type

</td><td>

Strategic priority type. The available options are:-   **Key Initiative**
-   **Shared Vision**
-   **Strategic Target**
-   **Operating Principle**
-   **Relationship Strategy**
-   **Environmental**
-   **Social**
-   **Governance**
-   **Artificial Intelligence**

</td></tr><tr><td>

Status

</td><td>

Status of the strategic priority. Status can be **Red**, **Yellow**, **Green**, or **None**.

</td></tr><tr><td>

Strategic Plan

</td><td>

Name of the strategic plan for which this strategic priority is created.

</td></tr><tr><td>

Description

</td><td>

A detailed description of the strategic priority.

</td></tr></tbody>
