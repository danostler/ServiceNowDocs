# Configuring SPW Jira Integrations

Configure the set up and implementation of SPW Jira Integrations. You can either follow the tasks listed in the configuration or use Guided Setup.

## Configuration overview

1.  [Install SPW Jira Integrations](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-business-management/strategic-planning/install-spw-jira-integrations.md).
2.  Assign the Jira integration admin role. See [Components installed with SPW Jira Integrations](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-business-management/strategic-planning/components-installed-with-spw-jira-integrations.md).
3.  [Set up a Jira instance](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-business-management/strategic-planning/setting-up-jira-instance-spw.md).
4.  [Connect SPW to Jira](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-business-management/strategic-planning/connect-spw-to-jira.md).
5.  [Import Jira projects to SPW](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-business-management/strategic-planning/imports-jira-projects-to-spw.md).
6.  Configure Jira Project settings.
    1.  [Associate Jira projects with EAP teams](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-business-management/strategic-planning/create-team-integration-settings-for-jira-project-in-spw.md).
    2.  [Review default mapping configuration for Jira projects in SPW](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-business-management/strategic-planning/review-default-map-config-jira-projects-spw.md).
    3.  \(Optional\) [Create custom mapping configuration](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-business-management/strategic-planning/custom-mapping-configuration-spw-jira.md).
    4.  [Enable bidirectional sync between SPW and Jira](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-business-management/strategic-planning/enable-bidirectional-sync-spw-jira.md).
7.  [Import issues from Jira to SPW](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-business-management/strategic-planning/import-issues-from-jira-to-spw.md).
8.  Repeat steps 6 and 7 for all imported Jira projects.
9.  [Activate Jira configuration in Enterprise Agile Planning](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-business-management/strategic-planning/activate-jira-configuration-in-eap.md).
10. [Define team structure in Enterprise Agile Planning](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-business-management/strategic-planning/define-agile-structure-eap-teams-spw-jira-integrations.md).

## Using guided setup to implement SPW Jira Integrations

Guided setup provides a sequence of tasks that help you configure SPW Jira Integrations on your ServiceNow instance. To open guided setup for SPW Jira Integrations, navigate to **All** &gt; **Strategic Planning** &gt; **Jira integration** &gt; **Guided Setup**.
