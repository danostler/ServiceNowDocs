# Delete a test case

Delete a test case and all the tests associated to it.

## Before you begin

Role required: tm\_test\_manager

## Procedure

1.  Navigate to **All** &gt; **Test Management** &gt; **Test Repository** &gt; **Test Cases**.

2.  Open the test case that you want to delete.

3.  Click **Delete**.

4.  Read the warning message in the confirmation pop-up window.

5.  Click **Delete** to confirm the deletion.


## Result

The test case is removed from the list and all the associated tests are deleted.
