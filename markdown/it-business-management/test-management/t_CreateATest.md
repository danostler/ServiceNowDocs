# Create a test

Create a test to see if your features are working correctly.

## Before you begin

-   [Create a test case](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/zurich/markdown/zurich/it-business-management/test-management/t_CreateANewTestCase.md).
-   Role required: tm\_test\_manager or tm\_tester

## Procedure

1.  Navigate to **All** &gt; **Test Management** &gt; **Test Repository** &gt; **Test Cases**.

2.  Open a test case to which you want to add a test.

3.  From the Tests related list, click **New**.

4.  On the form, fill in the fields.

## What to do next
