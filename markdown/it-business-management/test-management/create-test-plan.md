# Create a test plan using Test Board in Test Management 2.0

Define how to test a product or feature by creating a test plan.

## Before you begin

Role required: sn\_test\_management.test\_manager

## Procedure

1.  Navigate to **All** &gt; **Test Management 2.0** &gt; **Test Board**.

2.  Click **Create Test Plan**.

3.  In the form, fill in the fields:

    |Field|Value|
    |-----|-----|
    |Name|Unique name for the test plan.|
    |Planned start date|Planned date and time for starting the test plan execution.|

4.  Click **OK**.

    The **Details** tab opens.

5.  Provide, review, and edit the test plan details.
